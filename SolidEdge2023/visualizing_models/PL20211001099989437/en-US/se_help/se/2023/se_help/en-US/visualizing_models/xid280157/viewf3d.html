Lights tab (View Overrides dialog box)

Formats a view by setting options for light sources. You can specify the color, angle and intensity for each of the light sources.

Lights

Defines the color, intensity and position of the eight directional light sources. You can enter values to change the properties of the active light. You can copy the properties of one light source to another using Drag and Drop.

Light Angle

Specifies the light angle for the active light source.

Ambient

Defines the minimum illumination of any element regardless of which lights illuminate the element. You can adjust the ambient light value.

Red, Green, Blue

Defines a light color using sliders that control the amount of red, green, and blue that make up the light color. These controls also affect the settings for the Hue, Saturation, Intensity sliders.

Hue, Saturation, Intensity

Defines a light color using sliders that control the amount of hue, saturation, and intensity that make up the light color. These controls will affect the settings for the Red, Green, Blue sliders.

Preview

Shows you the result of your settings before you apply them.

Description

Displays a description of the formatting options.
