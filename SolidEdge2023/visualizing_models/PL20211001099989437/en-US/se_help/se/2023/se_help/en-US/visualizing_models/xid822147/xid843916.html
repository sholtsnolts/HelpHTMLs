Workflow for animating with KeyShot

This workflow shows you how to use Solid Edge animation events in KeyShot.

  1. Define the assembly features and configuration information such as motors and exploded views.

  2. Choose Tools tab→Environs group→ ERA environment, and then select Animation Editor.

  3. Add events to the timeline such as explosion durations or motor durations.

  4. From Animation Editor, choose the KeyShot Animate command.

  5. In the KeyShot Options dialog box , review the settings and adjust any parameters, and then click OK.

    The KeyShot application starts.

  6. At the bottom of the window, select the Animation command. A timeline with similar controls to the Solid Edge animation timeline appears.

  7. In the timeline, make additional adjustments

    Tip:

    KeyShot changes are not stored in the Solid Edge file.

  8. Switch back to the Solid Edge application and make any changes that should be retained in the file.

  9. Use the KeyShot Update command to send any animation updates to the active KeyShot session.

    Note:

    Animation updates only occur if the Solid Edge Animation Editor is open.

  10. In KeyShot, select the Render command then select the Animation option to define the parameters for generating the output file.

    You can change settings for the following items:

    • Resolution

    • Time range

    • Video output

    • Frames Output

    • Render Mode

  11. After you are satisfied with all the parameter settings, select the Render button.

Note:

Solid Edge supports KeyShot 5 or later. If you have an earlier version or do not have KeyShot installed, Solid Edge automatically installs it for you.
