Add Frames Dialog Box

This dialog box is available when adding frames or time to a event duration bar in the right pane of the Animator Editor tool.

Length

Specifies the amount of time or number of frames you want to add to the selected event duration bar. When you type a value in one of the boxes, the value in the other box also updates.

Frames

Specifies the number of frames you want to add.

Seconds

Specifies how much time you want to add. Type the amount of time you want.

Position

Specifies at what location on the event duration bar you want to add frames or time. When you type a value in one of the boxes, the value in the other box also updates.

Before

Specifies that the frames or time is added before the value displayed.

After

Specifies that the frames or time is added after the value displayed.

Frame

Specifies the frame number position at which you want to add frames. Type the frame number in the box.

Second

Specifies at which time position in the duration bar you want to add frames. Type the time number in the box.
