Add Events to or Remove Events from an Event Group

You can add events to or remove events from an event group in the Explode PathFinder tab.

To

Do this

Remove an event from an event group in Explode PathFinder.

In an event group in Explode PathFinder, select the part you want to remove, then on the shortcut menu, click Remove Event from Group.

Add an event to a different event group in Explode PathFinder.

In an event group in Explode PathFinder, select the part you want to remove, then on the shortcut menu, click Add Event to Group.

In Explode PathFinder, select a part in a different event group.
