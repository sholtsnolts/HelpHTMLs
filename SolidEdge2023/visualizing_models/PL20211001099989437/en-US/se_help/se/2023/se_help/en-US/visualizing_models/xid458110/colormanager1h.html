Set the part color display

  1. Choose View tab→Style group→Color Manager.

  2. In the Color Manager dialog box, do one of the following:

    • To display part colors using the colors defined on the Colors tab of the Solid Edge Options dialog box, select Use Solid Edge options color settings.

    • To display part colors defined using the Base Styles options, select Use individual part styles.

      Tip:

      Also select the Use individual part styles option when you want to change the colors of model elements using the Part Painter command.
