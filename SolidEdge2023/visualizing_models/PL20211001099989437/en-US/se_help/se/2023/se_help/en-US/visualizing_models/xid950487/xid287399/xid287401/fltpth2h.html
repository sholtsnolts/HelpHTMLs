Create an animation using the Animation Editor

  1. On the Animate toolbar, click the Animation Editor button .

  2. On the Animation Editor tool, click the New Animation button .

  3. On the New Animation dialog box, type the animation name, frames per second, and animation length you want, then click OK.

  4. On the Animation Editor tool, in the left pane, do one or more of the following:

    • To define a camera path, right click the Camera Path entry in the Event Category list, then click Edit Definition.

    • To add an existing motor, right-click the Motors entry, then click Edit Definition.

    • To add an existing exploded view, right-click the Explosion entry, then click Edit Definition.

    • To add an appearance event, on the Animation Editor tool, click the Appearance button , then use the Appearance Event command bar to define an appearance event.

    • To add a motion path event on the Animation Editor tool, click the Motion Path button , then use the Motion Path Event command bar to define a motion path event.

    • On the Animation Editor tool, click the Save Animation button .

Tip:

  • You can only define one camera event and one exploded view per animation.

  • You can define as many motor events, appearance events, and motion events as needed for an animation.

  • You can use the Save As Movie command on the Animation Editor tool to save your assembly animation as an AVI file.
