Render Scene command

Use the Render Scene command to render the active window.

Depending on the type of Solid Edge license you have, you may have advanced rendering capabilities available to you in the Explode-Render-Animate application in the Assembly environment. For example, if you have a Solid Edge Classic license, the KeyShot Render command is available. If you have a subscription license, the Render Scene command is available through the application program interface.
