Rotational Motor command bar

Steps
Select Moving Part Step

Specifies the part which will be moved by the motor. You can select any underconstrained part in the assembly.

Motor Axis Step

Defines the axis about which the part will rotate.

Cancel/Finish

This button changes function as you move through the motor definition process. The Finish button defines the motor feature using input provided in the other steps. After you finish the feature, you can edit it by re-selecting the appropriate step on the command bar. The Cancel button discards any input and exits the command.

Options
Flip Direction

Switches the direction of the motor movement. A graphic representation of the movement direction is displayed in the graphic window.

Motor Value

Specifies the rate at which the motor will move. The units available depend on the motor type you defined, and the working units in the document.

You can set the working units for the angular and linear velocity of a motor using the Advanced Units button on the Units tab of the File Properties dialog box.

Limit

Specifies a value to limit motor movement. For example, with a rotary motor, you may want to specify that the motor only rotates a particular number of degrees.

Name

Specifies the name assigned to the feature.
