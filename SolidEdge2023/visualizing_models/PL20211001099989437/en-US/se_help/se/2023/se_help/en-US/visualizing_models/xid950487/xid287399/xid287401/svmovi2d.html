Save As Movie Options dialog box

View Style

Lists the view styles you can use for creating the movie.

Compression Codec

Lists the available compression codecs you can use for creating the movie. The codecs in the list are the codecs that are currently registered for your computer. The Full Frames (Uncompressed) option is selected by default. This codec is a lossless format, which should be playable on any system, however the AVI file can become large. Some codecs create a higher quality animation, but this typically results in an increased file size. You may want to test several of the options to determine the output option that best suits your needs.

Although a codec is in the list, it does not mean that the codec will work properly. Some codecs are decoders only, and the encoders are fee-based.

A selection of codecs can be found by searching for codecs at the Microsoft website (www.microsoft.com), or at other internet sites.

Quality

Sets the quality of the movie.

Output Range

Specifies what part of the animation you want to create a movie of. You can specify that you want to create a movie of the entire timeline or only a range of frames you can define.

Output Size

Specifies the size of the movie in pixels. You can use the preset size or specify a custom size.

Preset

Sets the movie size to one of the preset values in the Width X Height list.

Custom

Sets the movie size to the width and height values you type.

Maintain Aspect Ratio

Maintains the window aspect ratio between the window height and width. If this box is checked, any changes in the Height or Width box will cause an automatic proportional change in the other box.
