Display draft face colors on a part

To turn on or turn off draft face colors:

  • Choose Inspect tab→Analyze group→Draft Face Analysis→Show.

To define the draft base plane, draft angle, and draft face colors:

  1. Choose Inspect tab→Analyze group→Draft Face Analysis Settings.

  2. On the Draft Face Analysis command bar, click the Options button to display the Draft Face Analysis dialog box, and set the draft angle and draft colors you want.

  3. In the graphic window select a planar face or reference plane that is perpendicular to the direction that the part will be pulled from the mold or die.

  4. Position the cursor so that the dynamic arrow points in the direction you want. This direction defines the Positive Face direction.

  5. Click Finish.

Tip:

  • To view draft face colors on a model you must also shade the active window using the Shaded or Shaded With Visible Edges commands.

  • You can use the Draft Face Analysis dialog box to control the draft face settings, such as the colors for the negative and positive faces.
