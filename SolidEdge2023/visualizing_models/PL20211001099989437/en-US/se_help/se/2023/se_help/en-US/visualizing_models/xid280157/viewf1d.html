Background tab (View Overrides dialog box)

Use the settings on this tab to define the background that is displayed in the document. Three options are available:

Solid

Defines a single solid color to use for the background. When you use a solid color, the selected color preference becomes the default color for all documents on your system. If you open a document from another user who has saved it using a different solid color background, your default color is displayed.

Gradient

Specify the two colors to use for the gradient and one of the gradient-type options. If you select the Square Spot or Circular Spot gradient-type options, you can define where the center point of the spot is located by clicking in the Background thumbnail area.

When you use a gradient for the background the defined gradient is saved in the document. If you open a document from another user that has a different gradient defined, the gradient that is saved in the document is displayed.

Color 1

Displays a selection of colors from which you can choose.

Color 2

Displays a selection of colors from which you can choose.

Image

Defines an image to use as the background. When you use a image for the background the defined image is saved in the document. If you open a document from another user that has a different image defined, the image that is saved in the document is displayed.

File name

Current defined path of the image file.

Browse

Accesses a dialog box that allows you to search for a document.

Mirror up/down

Mirrors the background about the x axis. Available for Gradient and Image options.

Mirror left/right

Mirrors the background about the y axis. Available for Gradient and Image options.

Thumbnail view

Displays a thumbnail of the current background selection.

Preview

Shows you the result of your settings before you apply them.

Description

Displays a description of the formatting options.
