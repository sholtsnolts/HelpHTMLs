Exploding, rendering, and animating assemblies

The Explode-Render-Animate application in the Assembly environment allows you to create exploded views, renderings, and animated presentations of your Solid Edge assemblies.

To access the commands for creating these types of assembly presentations, choose Tools tab→Environs group→ERA in the Assembly environment.

For more information on exploding assemblies, see the Creating Exploded Views of Assemblies Help Topic.

For more information on rendering assemblies, see the Rendering parts and assemblies Help Topic.

For more information on animating assemblies, see the Creating Assembly Animations Help Topic.
