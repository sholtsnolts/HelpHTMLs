Change the current view settings

  1. Choose View tab→Style group→View Overrides .

  2. In the View Overrides dialog box, do any of the following:

    • To render a view, set the options you want on the Rendering tab.

    • To change the window background color, use the Background tab.

    • To set lights for a view, set the options you want on the Lights tab.

    • To specify the image you want to see reflected on the face of the model, use the Reflection Box tab.

  3. To save the view style changes that you make as a new style:

    1. In the View Overrides dialog box, click Save As.

    2. In the New Style from Overrides dialog box, type a name for the new style.

    The new style is added to the 3D View Styles list in the Style dialog box.

Tip:

To select and apply a newly created view style to another model:

  1. Choose View tab→Style group→Styles command.

  2. In the Style dialog box, in the Style type box, click 3D View Styles.

  3. In the Styles box, click the name of the view style you created.
