Automatic Explode command bar

Steps
Select Step

Selects the assembly to be exploded. You can specify that you want to explode the top-level assembly or subassemblies below the top-level assembly.

Settings Step

Specifies options for the explosion.

Explode/Finish/Cancel

This button changes function as you move through the automatic explosion process. The Explode button explodes the parts based on the criteria you specified in the previous steps. The Finish button completes the explosion using input provided in the other steps. After you explode the feature, you can edit it by re-selecting the appropriate step on the command bar. The Cancel button discards any input and exits the command.

Select Step Options
Selection Type

Specifies the assembly explosion method.

  • Top-Level Assembly—Specifies that the top-level assembly and all its subassemblies are exploded.

  • Subassembly—Specifies that only the subassemblies you select are exploded. You can select the subassemblies in Assembly PathFinder or the graphic window.

Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selection.

Settings Step Options
Automatic Explode Options

Displays the Automatic Explode Options dialog box so you can set the explode options you want.

Automatic Spread Distance

Automatically determines the spread distance for the explosion when set. This option uses the spread distance calculation method available prior to version 19 of Solid Edge.

  • To have the spread distance calculated automatically, set the Automatic Spread Distance option.

  • To specify the spread distance yourself, clear the Automatic Spread Distance option, then type the value you want in the Distance box.

Distance

Specifies the spread distance for the exploded parts. You can type the value you want, then click the Explode button to see the result. To try another spread distance, type a new value, then click the Explode button again.
