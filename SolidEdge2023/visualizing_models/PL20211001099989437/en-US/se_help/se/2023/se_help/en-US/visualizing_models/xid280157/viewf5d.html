Rendering tab (View Overrides dialog box)

Formats a view by setting options such as rendering and perspective.

Perspective

Applies perspective to the view. When perspective is applied, objects that are farther away appear smaller. You can choose from a list of values that equate to the focal lengths of a 35 mm camera.

  • None - No perspective is applied.

  • Portrait (85 mm)

  • Normal (50 mm)

  • Wide (35 mm)

Antialias Level

Determines whether the view image is improved by removing aliasing. Applying antialiasing will remove the jagged appearance from the angular edges of parts. When no antialiasing is applied, view shading will complete faster.

Render Mode

Lists and applies the rendering method for the elements in the view.

Hidden Lines

Specifies the format for hidden lines in the view.

Dim By %

Specifies how intense hidden lines appear for vector-hidden line rendering. The higher the value, the dimmer the lines.

Silhouettes*

Creates silhouettes when you set the Render Mode to Shaded with VHL Overlay mode. The checkbox is inactive until you select Shaded with VHL overlay as the Render Mode.

Facet Edges

Determines whether facet edges are displayed.

Textures

Determines whether textures are generated when a view is shaded. When this option is cleared, no textures are generated and view shading will complete faster.

Reflections

Determines whether reflections are generated when a view is shaded. When this option is cleared, no reflections are generated and view shading will complete faster.

Floor reflection*

Provides a reflective appearance underneath the part or assembly.

Depth Fading*

Determines whether the display intensity uses the display depth. When depth fading is set, the intensity of the element varies with its display depth. If elements are closer, they appear at full intensity. If elements are farther away, the elements appear at a lower intensity, or darker.

High-Quality*

Applies high quality rendering to the view. This setting will improve the appearance of low face count models and will enable display of bump maps and panoramic images. This setting impacts performance compared to standard display settings.

Physically based rendering*

Uses advanced physical-based rendering techniques to enhance the quality of the display. The option is disabled if High Quality is not selected.

Cast Shadows*

Determines whether shadows are displayed on the part or assembly when you shade a view. When you clear this option, no shadows display and view shading is faster.

Floor Shadow*

Displays a shadow beneath the part or assembly when you shade a view.

Ambient shadows*

Improves the quality of shading used on the model. The model displays gradient shadow edges and a more realistic appearance.

Preview

Shows you the result of your settings before you apply them.

Description

Displays a description of the formatting options.

Note:

*These options are disabled when using Large Assembly Performance mode.
