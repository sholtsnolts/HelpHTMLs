Motor Group Properties dialog box

The Motor Group Properties dialog box is opened when you select the Simulate Motor command (Home tab→Motors group→Simulate Motor) or when you right-click the Motors line in the Animation Editor when using the ERA tool (Tools tab→Environs group→ERA).

Use the Motor Group Properties dialog box to set the following options when adding motors to the animation time line.

No Analysis

Allows you to move underconstrained parts and observe the results. This is the only valid option when using a Variable Table Motor.

Detect Collisions

Detects collisions during motor animation.

Physical Motion

Simulates physical motion between parts. This option detects contact between unconstrained surfaces and applies temporary constraints between the contacting surfaces. This makes it possible to analyze motion in mechanisms that contain gears and other forms of sliding contact.

Motor Duration

Specifies how the motor duration is defined.

Use motor limits as duration if defined

Specifies that the motor limits define the duration. If a limit of motion was placed on the motor when it was defined activate this option to honor the limit in the timeline.

Default motor duration

You use this field to specify duration for the motors when placed in the timeline.

Available Motors

Lists the available motors that have been defined. You can use the Add and Remove buttons to add or remove motors from the Motors in Animation list.

Motors in Animation

Lists the motors that will be used in the animation. You can use the Add and Remove buttons to add or remove motors from the Motors in Animation list.

Add

Adds the highlighted motor in the Available Motors list to the Motors in Animation list.

Remove

Removes the highlighted motor from the Motors in Animation list to the Available Motors list.
