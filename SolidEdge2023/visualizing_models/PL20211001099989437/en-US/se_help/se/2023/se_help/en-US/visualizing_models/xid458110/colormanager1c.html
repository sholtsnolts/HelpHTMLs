Color Manager command

Allows you to switch between color display modes. You can specify that color display is based on the settings on the Colors tab of the Solid Edge Options dialog box or on the current style settings for the part set on the Color Manager dialog box.

When you set the Use individual part styles option, you can set the base styles you want to use for part, construction, threaded cylinders and weld bead elements. This option also enables you to use the Part Painter command to paint individual elements in the model.
