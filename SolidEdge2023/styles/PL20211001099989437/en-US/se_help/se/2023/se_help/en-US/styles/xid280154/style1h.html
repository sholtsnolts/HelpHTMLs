Apply a style

  1. Select the element you want to format.

  2. Choose View→Style group→Style.

  3. On the Style dialog box, in the Styles list, click the style you want to apply.

  4. Click the Apply button.

Note:

If you have formatted an element with the Properties command or by setting options on the active command bar, these formats are removed when you apply a style.
