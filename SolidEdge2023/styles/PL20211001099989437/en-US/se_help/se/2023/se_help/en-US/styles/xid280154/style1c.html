Styles command

Modifies, creates, deletes, or applies styles using the Style dialog box.

You can copy styles from one document to another document using the Style Organizer option. This option displays the Style Organizer dialog box.

For more in-depth information on working with styles, see Applying formats with styles.

Creating new styles

The following dialog boxes are available for creating a new style.

  • New Dimension Style dialog box

  • New Drawing View Style dialog box

  • New Faces Style dialog box

  • New Fill Style dialog box

  • Hatch Style dialog box

  • New Line Style dialog box

  • New Table Style dialog box

  • New Text Box Style dialog box

  • New 3D View Style dialog box

Managing styles

In the Style dialog box, you can select the Style Organizer button to open the Style Organizer dialog box, where you can copy styles between documents, delete a style, and rename a style. For example, you can share a custom fill style created in a part model with a drawing of the part.

Modifying existing styles

The following dialog boxes are available when you are modifying an existing style.

  • Modify Dimension Style dialog box

  • Modify Drawing View Style dialog box

  • Modify Faces Style dialog box

  • Modify Fill Style dialog box

  • Modify Hatch Style dialog box

  • Modify Line Style dialog box

  • Modify Table Style dialog box

  • Modify Text Box Style dialog box

  • Modify 3D View Style dialog box
