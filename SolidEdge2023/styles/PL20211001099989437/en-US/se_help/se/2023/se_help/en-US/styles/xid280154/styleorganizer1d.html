Style Organizer dialog box

Use the Style Organizer dialog box to copy style definitions from one file to another, to delete a style from a document, and to rename a style. You can open this dialog box by selecting the Style Organizer button in the Style dialog box.

Styles Available In:

Specifies the document you want to copy styles from.

Browse

Accesses a dialog box that allows you to search for a document.

File Name

Displays a list of styles found in the document from which you can copy styles. For example, Materials.mtl.

Active File Name

Displays a list of styles found in the active document. For example, Widget.asm.

Copy

Copies the selected style from one file to the other. You can copy styles to or from the active Solid Edge document.

Delete

Deletes the selected style from the active document.

Rename

Renames the selected style from the active document.

Description

Displays a full text description of the selected style.

Close

Closes the Style Organizer dialog box.
