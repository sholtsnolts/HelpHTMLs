Create or modify a table style

Use the options in the Table Style dialog box to define a new table style or to change an existing table style. You can specify the appearance of lines and text in a table style.

  1. Choose View tab→Style group→Styles .

  2. In the Style dialog box, set the Style Type to Table.

  3. Do one of the following:

    • (To define a new table style) Click New.

    • (To modify an existing table style) From the Styles list, select the style to modify and then click Modify.

  4. (For a new table style) In the New Table Style dialog box, on the Name page, do the following:

    • In the Name box, type a name for the new table style.

    • From the Based On list, select a style to use as a template for the new table style.

  5. On the Lines page, choose the line styles that will be used to display the table.

    To learn how, see the Help topic, Specify the appearance of table lines.

  6. On the Text page, select the text styles that will be used to display the table.

    To learn how, see the Help topic, Specify the appearance of table text.

  7. To apply the current table style selections to a table on the drawing, in the Table Style dialog box, click OK.

  8. In the Style dialog box, click Close.

Tip:

You can create and name a unique table style for each type of table you use. To apply a style you have created, select the style name from the Table style list on the command bar, or from the Table style list on the General page of the Table Properties dialog box.
