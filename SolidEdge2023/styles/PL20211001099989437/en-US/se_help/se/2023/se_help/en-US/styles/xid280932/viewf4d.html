Reflection Box tab (View Overrides dialog box)

Formats a view by setting the reflection box options. You can define reflection box options using either a view style or a face style.

Type

Specifies how the reflection box is defined. Choose from:

Reflection box

Specifies a six-sided environment in which the model resides.

Single image

Specifies a planar image which the model will reflect.

Sphere map

Specifies a hemi-spherical shaped image which the model will reflect.

Panoramic

Specifies a panoramic image which the model will reflect.

Orient

Accesses the Orient Reflection Box dialog box.

File name

Displays the name of the reflection box file.

Clear

Clears the image from the selected reflection box.

Clear All

Clears the images from all the reflection boxes.

Browse

Accesses a dialog box that allows you to search for images. The default displays the contents of folder containing High Dynamic Range (HDR) panoramic images.

Preview

Shows you the result of your settings before you apply them.

Description

Provides a description of the selected type.
