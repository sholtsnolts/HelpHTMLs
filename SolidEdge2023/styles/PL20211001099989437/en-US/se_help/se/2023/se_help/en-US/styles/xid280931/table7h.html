Specify the appearance of table lines

You can set display options for each of these linear table elements: table border, table separators (title and horizontal/vertical title headers), and grid lines (horizontal and vertical). As you make adjustments, the Preview pane shows how they affect the appearance of a table.

  1. In the New Table Style dialog box or the Modify Table Style dialog box, click the Lines tab.

  2. To change color—On the row representing the linear element you want to change, click the Color cell and then select a color from the Color list.

    The default is Black/White.

  3. To change line type—On the row representing the linear element you want to change, click the Type cell and then select a line type from the list.

    The default is a continuous line.

  4. To change line width—On the row representing the linear element you want to change, click the Width cell and then select a line width from the list.

  5. In the Table Style dialog box, click OK.

  6. In the Style dialog box, click Close.

Tip:

  • The line style changes are applied to any existing tables on the drawing, as well as to new tables you create that use this line style.

  • To make changes to the appearance of a single table without changing the table style, right-click the table and choose Properties.

  • To make changes to the table text, see Specify the appearance of table text.
