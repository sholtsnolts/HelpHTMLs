Map dimension styles

  1. On the Dimension Style tab of the Solid Edge Options dialog box, select the element for which you want to set a default dimension style.

  2. From the Style list, select a dimension style.

  3. Repeat as needed for other elements.

Tip:

  • You can use the Set All Styles To list to set all elements in the list to the same style. Select the style you want, and then click Apply.

  • When the Use Dimension Style Mapping option is selected, Solid Edge applies the styles in the dimension style table to elements as they are placed.  

    When this option is cleared, Solid Edge applies the style and units shown in the Active dimension style list as you place elements.

  • The units referenced in the Active dimension style list are defined on the Units tab in the Solid Edge Options dialog box.
