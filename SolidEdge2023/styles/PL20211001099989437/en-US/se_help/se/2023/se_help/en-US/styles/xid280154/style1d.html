Style dialog box

Specifies the name of the style, the formats defined for the style, and other options for working with styles.

Style type

Specifies the type of style that you want to create or modify.

Example:

  • Select the Dimension Style type when you want to change the font size in the ANSI, ISO, or DIN style used to format all dimensions and annotations.

  • Select the Face Styles type when you want to change the appearance of the material or color applied to model faces and features. The face style includes parameters for edges, faces, textures, bump maps, shininess, shadowing, and reflections.

If you select an element and then access the Style dialog box, the style type of the element is displayed in this list. If you select a group of elements from more than one style type, the list is blank. If you do not select an element, the last active style type is displayed in the list.

Show in Styles list

Specifies which styles appear in the Styles list. You can list only the styles in use, styles that you defined, or all styles.

Style Organizer

Opens the Style Organizer dialog box for you to add or remove styles from the current document. Before clicking the Style Organizer button, from the Style type list, select the type of style that you want to organize.

Styles

Selects a specific style to change. An enhanced preview image is displayed to the right of the Styles list.

If you select an element and then access the Style dialog box, the style of the element is selected in the list. If you do not select an element, the last active style is highlighted in the list.

New

Opens one of the following New Style dialog boxes. Before clicking the New button, from the Style type list, select the type of style you want to create.

  • New Dimension Style dialog box

  • New Faces Style dialog box

  • Fill Style dialog box

  • Hatch Style dialog box

  • New Line Style dialog box

  • New Table Style dialog box

  • New Text Box Style dialog box

  • New 3D View Style dialog box

Modify

Opens one of the following Modify Style dialog boxes. Before clicking the Modify button, from the Style type list, select the type of style you want to modify.

  • Modify Dimension Style dialog box

  • Modify Faces Style dialog box

  • Fill Style dialog box

  • Hatch Style dialog box

  • Modify Line Style dialog box

  • Modify Table Style dialog box

  • Modify Text Box Style dialog box

  • Modify 3D View Style dialog box

Delete

Deletes the style that is selected in the Styles list.

Description

Lists all of the parameters set for the selected style.

Apply

Applies the style selected in the Styles list as the current style.
