Delete a face style from a document

  1. Choose View tab→Style group→Styles.

  2. On the Style dialog box, set the Style type to Face Styles.

  3. Click Style Organizer.

  4. On the Style Organizer dialog box, click the style in the File Name list that you want to delete.

  5. Click Delete.
