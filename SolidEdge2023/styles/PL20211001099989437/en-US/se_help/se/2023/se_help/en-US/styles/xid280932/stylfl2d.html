Face Style Name Tab

Name

Displays the name of the new or current style. Style names can contain up to 253 characters and can include any combination of characters and spaces, except backslashes (\), semicolons (;), and braces ({ }). Style names are not case sensitive and must be unique.

Based On

Displays the name of the style that the current style is based on. You can select or type a new style.

Preview

Shows you the result of your settings before you apply them.

Preview Render Mode

Sets the rendering method to be used in the Preview box. You can preview a rendering using the Wireframe, Outline, Vector Hidden Line, or Smooth Shaded methods.

Preview Object

Sets the object to be used in the Preview box. You can preview a rendering using a Cube, Sphere, or Tea Pot.

Description

Displays a description of the formatting options.
