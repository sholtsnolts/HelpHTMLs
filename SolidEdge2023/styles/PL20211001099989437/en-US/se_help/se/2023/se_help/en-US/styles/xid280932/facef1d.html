Appearance Tab

Formats a part by setting options for the appearance of the part's faces.

Shininess

Controls the amount of specular reflection from a light source. If the shininess value is high, a part appears to be highly polished or glossy. If the shininess value is low, the part appears dull.

Reflectivity

Specifies how reflective the part is when a reflection box has been defined.

Opacity

Controls the degree to which a face appears transparent or opaque. The lower the opacity value, the more transparent the face appears.

Refraction

Specifies how refractive the part is.

Type

List of face styles that are exported for use by third party renderers.

Casts Shadows

Specifies that the part will cast shadows on other parts which have the Accepts Shadows option specified.

Accepts Shadows

Specifies that the part will accept shadows cast onto it from other parts which have the Casts Shadows option specified.

Preview

Shows you the result of your settings before you apply them.

Preview Render Mode

Sets the rendering method to be used in the Preview box. You can preview a rendering using the Wireframe, Outline, Vector Hidden Line, or Smooth Shaded methods.

Preview Object

Sets the object to be used in the Preview box. You can preview a rendering using a Cube, Sphere, or Tea Pot.

Description

Displays a description of the formatting options.
