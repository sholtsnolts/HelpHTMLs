Face Overrides dialog box

Formats a part in your assembly by setting options for the color and display characteristics of the edges and faces on a part. You can format individual parts in an assembly to have different color and display characteristics.

Tabs

  • Edges

  • Faces

  • Texture

  • Bump

  • Appearance

  • Reflection Box

Note:

You can use the Save As button on the Face Overrides dialog box to save the overrides to the current face style as a new face style. To apply and modify the newly created style, select View tab→Style group→Styles command→Face Styles style type.

How do I
Search for a style in the Style Palette
Apply a face style to a part with the Style Palette
Apply detailed stylistic changes to assemblies with Face Overrides

Delete a face style from a document
