Download high-quality visual styles .zip file

Due to the large file size of high-quality visual styles, only a lower-fidelity set of material images is delivered with the Solid Edge setup. Designers should find the quality of the styles delivered sufficient to verify the visual details of the product during the design phase. However, if you send the Solid Edge model to professional renderers, such as Keyshot, you may want to use higher-fidelity images. The full-fidelity images (about 800 MB) are captured in a .zip file and are available for download from Support Center.

Note:

Solid Edge uses the default zip creation process provided by Windows. For more information, see the Microsoft support site.

To download the full suite of high-fidelity images:

  1. Access Siemens Support Center at https://support.sw.siemens.com.

    Note:

    Sign in using your Support Center login information.

  2. Near the top of the page, click Downloads→ Solid Edge, and then set your version to Solid Edge 2023.

  3. Click Solid Edge 2023 Add-ons.

  4. From the left column, click High Quality Styles.

  5. Download Solid_Edge_High_Quality_Styles_2023_2210.zip to your local machine.

  6. Unzip the full-fidelity visual styles to either of the following locations:

    • The Solid Edge installation folder under \Preferences\Materials\HighQualityStyles.

    • The location defined in Settings→Options→File Locations for Material Table Folder .

    Note:

    The .zip file must be expanded while maintaining the existing sub-folder structure. With either of these options, the files are copied to the folder location you specify overwriting existing user files.

  7. To use the full-fidelity materials, after the .zip is expanded, restart Solid Edge.
