Custom Line Type dialog box

Specifies options for creating custom line types.

Line Type

Lists the available line types, including custom line types and line types delivered with Solid Edge.

Pattern

Displays the line type pattern for each entry in the Line Type list.

Copy

Copies the selected line type.  You can copy both custom line types and line types delivered with Solid Edge.

Edit

Edits the selected line type.  You can edit custom line types, but you cannot edit line types delivered with Solid Edge.

Delete

Deletes the selected line type.  You can delete custom line types, but you cannot delete line types delivered with Solid Edge.  You cannot delete a custom line type that is currently being used.

Pattern Sequence

Patterns always consist of couplets of dashes and gaps.  The length can be either fixed or not fixed.  For a pattern of fixed length, dashes or gaps are not scaled to fit the specified distance.

Name

Specifies the name of the current custom line type.

Repeat

Specifies that the pattern is repeated to fit the specified length.

Center

Specifies that the pattern is centered between the specified distances.  The Center option scales the non-fixed length pattern types to fit the specified distance.  To use the Center option, at least one dash or gap must have a non-fixed length.

Show % of Start and End Dash

Specifies the percentage of the dash length used for the first and last elements of a pattern.

Type

Specifies the dashes in the current pattern.

Length

Specifies the dash length.

Fixed

Specifies whether the dash length is fixed (cannot be scaled).

Type

Specifies the gaps in the current pattern.

Length

Specifies the gap length.

Fixed

Specifies whether the gap length is fixed (cannot be scaled).

Delete Row

Deletes the selected row from the table.

Preview

Previews the line type with the specified settings.

Create

Creates a new line type with the specified settings.

Update

Updates the selected line type with the specified settings.
