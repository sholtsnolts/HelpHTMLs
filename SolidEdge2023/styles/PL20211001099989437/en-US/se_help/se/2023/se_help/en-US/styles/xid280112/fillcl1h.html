Create a custom fill color

  1. Select a fill.

  2. On the Fill command bar, click Solid Color or Pattern Color.

  3. At the bottom of the palette, click More.

  4. In the Colors dialog box, click Define Custom Colors and set the options you want to create a new color.

  5. Click Add to Custom Colors.

  6. In the Custom Color Name dialog box, type the name that you want for the color you created.

    This name appears on the palettes for Solid Color or Pattern Color on the Fill command bar.
