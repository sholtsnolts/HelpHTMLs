Properties tab (Fill Style dialog box)

Sets the properties for new or modified fill styles. The changes to the style definition will be visible the next time the style is selected and applied to a drawing element with the Fill command.

Style

Displays the style name. To change the name, use the Name tab.

Solid Color

Applies a background color to the fill. All system colors are selectable, as well as any custom colors you defined with the Solid Color box on the Fill command bar.

Preview

Shows you the result of your settings before you apply them.
