Format a geometric element

  1. Select a geometric element, such as a line, arc, or circle.

  2. Do one of the following:

    • On command bar, set the options you want.

    • From the shortcut menu, choose Properties. Then, on the Properties dialog box, set the options you want.

    Tip:

    • Changing the current style settings on command bar or in the Properties dialog box overrides the line style formats.

    • You can change the definition of a line style in the Style dialog box. Choose the Styles command to access the dialog box.
