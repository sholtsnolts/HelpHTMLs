Create a custom line type

  1. Use the Styles command to open the Modify Line Style dialog box.

    To learn how to do this, see Change the formats of a style.

  2. Click the General tab in the Modify Line Style dialog box.

  3. From the Type list, select More.

  4. In the Custom Line Type dialog box, specify the parameters you want for the custom line type.

  5. Click Create.

  6. Click Close.

    The custom line type is now available for use.

Tip:

  • You also can open the Custom Line Type dialog box by right-clicking an element and choosing Properties. Then, on the Format tab of the Element Properties dialog box, from the Type list, click More.

    You also can select More from the Line Type list on the Line command bar.

  • You can also use the Custom Line Type dialog box to edit custom line types that you have already created. After you make changes to an existing custom line type, click Update to save them.

  • You can copy both custom line types and line types delivered with Solid Edge. However, you can edit or delete only custom line types.
