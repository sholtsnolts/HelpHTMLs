Create as Construction command

Use the Create as Construction command to change between profile and construction modes when creating sketch geometry. While the option is on, all sketch elements are created as construction elements. While the option is off, all sketch elements are created as profile elements.

Use the Construction command to change existing sketch geometry to either construction or profile elements.
