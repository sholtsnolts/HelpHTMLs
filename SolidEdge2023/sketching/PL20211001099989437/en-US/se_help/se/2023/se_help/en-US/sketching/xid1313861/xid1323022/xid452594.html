Clean Sketch command

It is possible for a sketch to have redundant geometry consisting of identical superimposed elements. Duplicate sketch elements can cause undesirable effects for further processes that use the sketch geometry. Non-native geometry imported into Solid Edge, such as data originating in DXF and DWG file formats, sometimes import with duplicate and overlapping geometry. The Clean Sketch command is used to move or delete redundant and unwanted elements from the sketch.

Valid input for identical elements are:

  • Point elements

  • Lines

  • Circles

  • Arcs

  • Ellipses

  • Partial ellipses

  • B-spline curves

Note:

  • Duplicate elements can be deleted or moved to a different sketch layer.

  • The Clean Sketch command does not move or delete sketch elements with relationships assigned to them.

  • Arcs and lines less than a user-defined length can be moved or deleted.

  • To select a synchronous sketch, select a sketch element or lock the sketch plane to the desired sketch.
