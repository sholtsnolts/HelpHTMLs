Create a model sketch from 2D geometry

The results of the Create 3D command vary depending upon the model type and whether it contains synchronous objects, ordered objects, or both.

  1. Select the Tools tab→Assistants group→Create 3D .

  2. In the Create 3D dialog box, do either of the following:

    • (To create a new model file):

      1. Select the Add to new file option.

      2. From the File list, select a template for the model you want to create.

    • (To update a model file):

      1. Select the Add to existing file option.

      2. Use the Browse button to locate and select the assembly, part, or sheet metal file you want to update.

  3. Click Next.

  4. To define the primary (first) sketch view, do all of the following:

    1. In the Create 3D dialog box, under View Properties:

      1. Select Fold principal views.

      2. From the Primary view list, select the view orientation you want to create (front, back, right, left, top, or bottom).

    2. On the drawing, select the 2D elements to define the primary view orientation. You can drag a box around the elements to include, or press the Shift key and click each element.

      To include dimensions in the sketch, you must also select them in the drawing view.

  5. To define another view, click the Next button and select the elements to include.

    Tip:

    You can use the Fold principal views, Fold auxiliary views, or Copy view(s) options to define additional sketches.

  6. Repeat the previous step until you have defined all of the required sketches.

  7. Click the Finish button.

    This opens the model document with all of the sketches placed in it.

  8. (Optional) Use the 3D steering wheel to adjust the position of the sketches and associated PMI dimensions in the model document:

    1. In PathFinder, right-click the Create 3D user-defined set that contains the sketch you want to move. To move all of the sketches as a group, right-click the last Create 3D user-defined set in the list.

    2. Reorient the steering wheel to move the sketches and associated dimensions.

    3. When the sketches are positioned correctly, right-click the user-defined set again and choose Attach PMI.

Tip:

  • After you select the geometry for the second view, you can click the Set Fold Line button to define a line or point in an orthogonal or auxiliary view about which to fold the view. This insures proper orientation of the sketches.

  • You can change the Scale value for the model geometry created from each drawing view.

  • You can click the Options button to change the default setting for drawing view projection angle, and to specify the dimension types you want to be able to include in the sketch.

  • The environment setting for opening part and sheet metal documents (on the Helpers page (Solid Edge Options dialog box) determines the default settings for creating a new file using the Create 3D dialog box.
