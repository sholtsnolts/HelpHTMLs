Draw an assembly layout

Note:

  • To define an assembly sketch or layout in the top-level assembly, start with the Sketch command.

  • To draw a part sketch when in-place activated from an assembly, you can use the 2D drawing commands on the Home tab or the Sketching tab using the workflow in Draw an ordered sketch of a part.

  • For examples of how you can use assembly sketches, see Using assembly sketches.

  1. Choose Features tab→Sketch group→Sketch .

  2. Select an assembly reference plane, part face, or part reference plane.

    A sketch view is displayed, oriented to the reference plane or face you selected. Sketch commands are displayed.

    Note:

    Assembly sketch geometry can be associatively linked to drive the creation of ordered part geometry only.

  3. Use the sketch commands to draw the assembly sketch elements you want.

  4. Choose Home tab→Close group→Close Sketch.

For information about how you can use assembly sketches, see the following assembly topics:

  • Designing in the context of an assembly

  • Assembly layouts
