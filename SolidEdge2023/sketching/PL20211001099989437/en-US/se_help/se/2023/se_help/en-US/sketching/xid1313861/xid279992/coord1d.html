Coordinate System Options dialog box

In the ordered environment, use this dialog box to specify how you want to define the location of the custom coordinate system.

Orient By:

Sets the method for defining the coordinate system.

Key-in (relative to another Coordinate System)

Specifies that you want to define the orientation of the new coordinate system relative to an existing coordinate system.

Geometry

Specifies that you want to define the orientation of the new coordinate system relative to geometry on the model. When you define a coordinate system relative to the model, the coordinate system is associative to the model geometry. If the model geometry changes orientation, the coordinate system also updates.

Show this dialog box when command begins

Displays the Coordinate System Options dialog box when you select the Coordinate System command.
