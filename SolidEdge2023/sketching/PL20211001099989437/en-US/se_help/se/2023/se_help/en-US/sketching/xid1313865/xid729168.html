Move 3D sketch elements

You can move and rotate selected 3D sketch elements by choosing the Move command . There are two ways to access the Move command.

  • Selecting a 3D sketch in PathFinder starts the Move command and displays the steering wheel to control movement.

  • Selecting 3D sketch elements in the part window starts the Modify command and displays the Modify command bar. The Move command is located on the Modify list.

You can use any keypoint in the model to define movement. Movement can only occur in unconstrained directions. Check the 3D relationships on the sketch if movement is constrained. You may have to delete some relationships to perform a move.
