Change peer edge locatability

  • When editing a profile or sketch, choose Tools tab→Edge Locate group→Peer Edge Locate . This command is an on and off setting.

Tip:

  • This command is not available unless the Sketch Relationships to Peer Geometry option on the Inter-Part tab on the Solid Edge Options dialog box is set.

  • Holding the Shift key allows locking to peer faces and planes for sketch creation.

  • When peer edge locate is active, you can place dimensions and relationships between sketch geometry and the edges of other parts in the assembly, but you cannot locate peer edges when creating new sketch geometry. For example, to place a line to the endpoint of a peer edge, you must first place the line near the endpoint of the peer edge, then use the Connect command to add a connect relationship between the endpoint of the line and the endpoint of the peer edge.

  • You can use the Select All Peer Relationships command on the shortcut menu to select all the dimensions and geometric relationships in the current sketch that reference other parts in the assembly.
