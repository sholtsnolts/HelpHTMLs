Change the active layer

  1. Click the Layers tab to list all layers in the document.

  2. Double-click the layer name that you want to set as the active layer.

    Tip:

    • You can also activate a layer using the Make Active command on the shortcut menu.

    • You can both activate and display a layer using the Show Only command on the layer shortcut menu.
