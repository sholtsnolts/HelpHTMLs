Create a reference plane normal to a curve

  1. Choose Home tab→Reference group→More Planes→Normal to Curve .

  2. Select a curve or part edge near the end where you want the reference plane to be.

    If the edge is a closed curve, the plane will automatically be placed at the curve's start point.

  3. Define the position for the new reference plane using the cursor or by entering a value in the Position or Distance boxes.

    Tip:

    • You can use QuickPick to locate hidden part edges.

    • You can use an upper or lower case N to redefine the X-axis direction of the reference plane.
