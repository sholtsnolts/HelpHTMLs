Sketch display

In some modeling scenarios, the number of sketch elements, colors, styles, and annotations can be rather significant. This may be the case when using blocks in the design. You may need a method of controlling the display of these sketches.

In the ordered environment, the sketch display can be controlled with the following options found on the Sketch shortcut menu:

  • Single color sketch display

  • Multi-color sketch display

  • Accelerated sketch display

  • Show Annotations (also available in synchronous environment)
