Move an element to another layer

  1. On the drawing sheet or a sketch, select one or more elements.

  2. On the Layers tab, click the Move Elements button .

  3. In the Move Elements dialog box, select the layer that you want to move the elements to, and then click OK.

    Example:

    You can create a new layer, rename it Dimensions, and then move all of your dimensions onto this layer.
