Peer Edge Locate command

The Peer Edge Locate command makes edges of other parts locatable or non-locatable in a sketch window, and while in-place activated and working within the context of an assembly.

You can press the Shift key to locate and lock to a plane in surrounding assembly components for sketching. The Peer Edge Locate command does not need to be turned on for this behavior to occur while in-place activated.

When peer edge locate is active, you can place dimensions and relationships between sketch geometry and the edges of other parts in the assembly, but you cannot locate peer edges when creating new sketch geometry.

For example, to place a line to the endpoint of a peer edge, you must first place the line near the endpoint of the peer edge, then use the Connect command to add a connect relationship between the endpoint of the line and the endpoint of the peer edge.

This command is not available unless the Sketch Relationships to Peer Geometry option on the Inter-Part tab on the Solid Edge Options dialog box is set.

You can use the Select All Peer Relationships command on the shortcut menu to select all the dimensions and geometric relationships that reference edges of other parts in the assembly. This can make it easier to visualize the relationships or to delete them without affecting other geometry.

Note:

  • Edges of inactive parts in the assembly are not locatable regardless of the setting of this command.

  • The Peer Edge Locate command setting will remain set to on during a Solid Edge session and when Solid Edge is restarted.
