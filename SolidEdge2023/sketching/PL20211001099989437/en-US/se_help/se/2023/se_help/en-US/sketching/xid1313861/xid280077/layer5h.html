Find the layer of an element

  1. On the drawing sheet or sketch, click the element that you want to find the layer it is placed on.

  2. Right-click to display the shortcut menu, and then choose Properties.

    Note:

    The name of the layer where the element is located is displayed in the Layer box on the Info tab in the Element Properties dialog box.
