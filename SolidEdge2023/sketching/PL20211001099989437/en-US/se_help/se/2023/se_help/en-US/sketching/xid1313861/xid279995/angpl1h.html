Create an angled reference plane

  1. Do one of the following:

    • To create a global reference plane, choose Home tab→Reference group→More Planes→Angled Plane.

    • To create a local reference plane,, on the command bar for the current feature, in the Plane Options list, click the Angled Plane option.

  2. Do one of the following:

    • To create the angled plane by selecting a planar plane first:

      1. Select a planar face.

      2. Define a rotation axis by doing one of the following:

        • Select a part edge.

        • Select a plane that intersects the plane you selected in the previous step.

          The intersection of the two planes is the rotation axis.

      3. Define the origin of the new reference plane by clicking an endpoint on the rotation axis.

        Note:

        The command displays a dynamic representation of the new reference plane. As you move the cursor, the new reference plane moves about the rotation axis.

      4. Define the rotation angle for the new reference plane and the side you want to rotate toward by doing one of the following:

        • Move the cursor to position the plane where you want it, and then click.

        • Type an angle value in the Angle box on the command bar. Move the cursor to define the side to rotate toward and click.

    • To create the angled plane by selecting a reference plane first:

      1. Select a reference plane.

      2. Select a planar face or reference plane.

        The rotation axis is the intersection of the reference plane and planar face or reference plane.

      3. Define the rotation angle for the new angled reference plane by doing one of the following:

        • Move the cursor to position the plane where you want it, and then click.

        • Type an angle value in the Angle box on the command bar. Move the cursor to define the side to rotate toward and click.

Tip:

  • You can use QuickPick to locate hidden part faces.

  • As you move the cursor, the rotation angle of the dynamic reference plane changes in increments based on the Step value on the command bar.

  • You can locate keypoints in 3D space to position the new plane accurately. Use the Keypoints list on the command bar to specify whether you want to locate the end point of a line or part edge, the midpoint of a circle or arc, or a tangency point on a curved analytic face.

    If there is more than one possible tangency between the plane and a curved face, a bold line shows which tangency is active as you move your cursor over the face.
