Edit Reference Plane command bar

This command bar is available when a reference plane is selected for editing.

  • Available Actions

    Lists alternate actions which are available for the selected element. The options in the Available Actions change base on what you have selected.

    • Move

      Specifies that you want to move or rotate the selected element.

    • Resize

      Displays the handles for the reference plane so you can edit its size.

    • Relate

      Specifies that you want to define a geometric relationship between the selected element with another element.

  • Solve Type

    Specifies the type of face solving you want to perform when faces are selected.

    • Simple

      Modifies the model by trimming and extending adjacent faces during the move/rotate operation.

    • Advanced

      Modifies the model by allowing you to specify relationship controls between the select set and the rest of the model.

  • Copy

    Specifies that you want to copy the selected faces during the move/rotate operation when set.

  • Detach

    Specifies that the selected faces are detached from the model during the move/rotate operation.

  • Keypoints

    Sets the type of keypoint you can select to define the feature extent. This allows you to define the feature extent using a keypoint on other existing geometry. The available keypoint options are specific to the command and workflow you use.
