Delete a sketch or sketch elements

Delete a sketch

  1. In PathFinder, expand the Sketches collection.

  2. Position the cursor over a sketch name so that the sketch geometry highlights in the graphics window.

  3. In PathFinder, right-click the sketch name and choose Delete.

    All sketch geometry is removed from the graphics window, and the sketch name and node are deleted from PathFinder.

    Tip:

    You can use the Undo command located on the Quick Access toolbar to restore the sketch.

Delete sketch elements

  1. To delete one or more sketch elements, select them using the Select command .

    There are a variety of selection filters available.

  2. Press the Delete key.

    Tip:

    If you delete one element that forms a closed region, the entire region is deleted.
