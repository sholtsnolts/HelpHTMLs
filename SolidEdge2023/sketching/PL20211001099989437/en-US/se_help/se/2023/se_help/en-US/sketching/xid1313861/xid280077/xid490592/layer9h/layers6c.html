Show Only command (Layers shortcut menu)

Activates and displays the selected layer or layers, turning off all other layers on the current drawing sheet. If you select multiple layers, the first layer in the list becomes the active layer, unless there is already an active layer in the selection list. In that case, the active layer in the list remains active.

Note:

In profile, the Show Only command does not activate parts on other layers.
