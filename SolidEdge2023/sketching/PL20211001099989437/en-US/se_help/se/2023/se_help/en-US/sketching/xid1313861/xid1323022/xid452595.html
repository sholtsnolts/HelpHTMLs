Clean Sketch command bar

Sketch Selection

To select a synchronous sketch, select a sketch element or lock the sketch plane to the desired sketch.

Clean Sketch Options

Displays the Clean Sketch Options dialog box.

Main Steps
Select elements to clean

Press Shift or Ctrl before clicking to select multiple elements or click and drag to fence. Press the S key to select all elements in the sketch.

Deselect

Deselects all of the currently selected elements.

Finish/Cancel

This button changes cleans the sketch using the options set. The Cancel button discards all input and exits the command.
