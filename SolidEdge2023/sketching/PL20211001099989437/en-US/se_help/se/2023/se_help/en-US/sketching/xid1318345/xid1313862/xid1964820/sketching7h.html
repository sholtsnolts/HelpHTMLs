Restore a used sketch

You can restore a used sketch to its original location in the model so that it can be used to construct a new feature. Dimensions are not restored with the sketch geometry.

  1. In PathFinder, expand the Used Sketches collection.

  2. Position the cursor over the sketch name you want to restore.

    Tip:

    Used sketches do not highlight in the graphics window.

  3. In PathFinder, right-click the sketch name and choose Restore.

    • The sketch is removed from the Used Sketches collection and placed in the Sketches collection in PathFinder.

    • The sketch geometry is restored to its original location in the model. You may need to use QuickPick to select it, as it may be obscured by model faces.
