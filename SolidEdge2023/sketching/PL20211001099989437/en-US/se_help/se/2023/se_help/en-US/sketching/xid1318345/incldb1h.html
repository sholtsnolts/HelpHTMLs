Project part edges onto a sketch

  1. Do one of the following:

    1. In the synchronous environment, choose Sketching tab→Draw group→Project to Sketch , and then click a planar face or reference plane to lock sketch input to the plane onto which you want to project the elements.

    2. In an ordered sketch, choose Home tab→Draw group→Project to Sketch .

  2. If the Project to Sketch Options dialog box displays, set the options you want, then click OK.

  3. Select one or more elements to project onto the sketch plane, and then click Accept.

Tip:

  • You can project edges from the active part, an assembly sketch, or the other parts in the assembly. When you project edges from an assembly sketch or other parts in the assembly, the edges are associatively copied if the Allow Inter-Part Links Using: Project to Curve Command From Part and Assembly Sketches on the Inter-Part tab on the Solid Edge Options dialog box is set.

  • When you use the Inter-Part Copy command to copy faces from other parts in an assembly into the active document, these faces become part of the active part, and you can use the Project to Sketch command to associatively copy their edges.

  • You can extend the ends of open projected elements.

  • You can project edges that are curves.

  • You cannot project an element that is perpendicular to the sketch plane.

  • You can control whether or not silhouette edges of parts are locatable with the Silhouette Edge Locate command.
