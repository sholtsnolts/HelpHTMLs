Construction command

Use the Construction command to change a sketch element into a construction element. You can use construction elements to help you draw and constrain the profile, but they are not used to construct the surfaces for a feature.

When you select a profile element, it changes to a construction element. When you select a construction element, it changes to a profile element. You can also use a fence to select multiple elements. There is a Chain option on the Construction command bar that allows you to select a chain of elements. If Chain is set and you use a fence for selection, any portion of a chain of elements that is in the fence causes the chained elements to change.
