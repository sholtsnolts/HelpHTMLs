Copy Sketch command bar

Steps
Select Source Occurrence

Defines the document that contains the sketch you want to copy.

Select Sketch

Specifies the sketch you want to copy.

Select Target

Defines the document to which you want to copy the sketch. You can use the Copy Sketch To Target dialog box to specify the target document(s) and whether you want to copy the sketch associatively or non-associatively. The target document must be a part or subassembly within the active assembly.

command bar Options
Construction Display

Specifies which construction elements are displayed. For example, you can select the source occurrence and use the Construction Display options to display the reference planes and sketches for the source part.
