Copy a sketch to another document

  1. Choose Home tab→Sketch group→Copy Sketch.

  2. Select the occurrence that contains a sketch you want to copy.

  3. Select the sketch that you want to copy.

  4. On the Copy Sketch To Target dialog box, set the options you want.

Tip:

  • You can copy a sketch to the active document or to another document in the active assembly.

  • You can specify whether the copied sketch is associative or non-associative to the parent sketch.
