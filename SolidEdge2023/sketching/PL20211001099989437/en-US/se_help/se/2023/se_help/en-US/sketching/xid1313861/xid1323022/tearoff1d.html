Tear-Off Sketch Options dialog box

Defines the options for copying and moving sketch elements from one reference plane to another. The option you select remains set while you are in the current command session.

Copy Elements (Associative)

Specifies that you want to copy the sketch elements associatively from one reference plane to another.

Copy Elements (Non-Associative)

Specifies that you want to copy the sketch elements non-associatively from one reference plane to another.

Move Elements

Specifies that you want to move the sketch elements from one reference plane to another.
