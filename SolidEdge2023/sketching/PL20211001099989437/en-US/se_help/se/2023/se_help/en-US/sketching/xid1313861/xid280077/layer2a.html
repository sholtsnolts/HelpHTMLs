Layer status icons

The Layers tab in the Library or in PathFinder displays a list of the layers in the document. You can see which layer is active, change the active layer, and create a new layer. You also can identify hidden and visible layers, select layers to show and hide, rename layers, and add properties to layers.

The following table explains the symbols used on the Layers tab:

Note:

On the sheet view of the Layers tab, the active layer name is displayed in boldface (L1).

Layer names below the drawing views symbol do not indicate an active layer. For drawing views, the active layer is displayed only when you are using the Draw in View command.
