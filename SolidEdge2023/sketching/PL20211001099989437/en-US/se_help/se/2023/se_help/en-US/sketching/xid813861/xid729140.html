3D IntelliSketch Options command

Use the 3D IntelliSketch command to display the IntelliSketch dialog box.

Use the dialog box to control the following cursor options:

  • Locate zone pixels

  • Relationship cursor size

  • Option to display tooltips for keypoints

  • Keypoint size

  • Intent zone pixels

How do I
Lock an element or keypoint
Apply a rigid set relationship to 3D sketch elements
Make elements symmetric in a 3D sketch
