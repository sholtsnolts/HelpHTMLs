Copy Dimensions to Model command

The Copy Dimensions to Model command is only available in synchronous modeling. If selected, when you select a dimensioned region, the dimensions get attached to the feature. The Copy Dimensions to Model command is located on the shortcut menu when you right-click on a 3D sketch in PathFinder.
