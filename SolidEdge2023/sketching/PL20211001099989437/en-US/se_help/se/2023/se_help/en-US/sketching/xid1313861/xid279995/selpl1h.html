Create a coincident reference plane by axis (ordered modeling)

Note:

This command exists only in the ordered modeling mode.

  1. Do one of the following:

    • To create a global reference plane, in the Part environment, choose Home tab→Planes group→More Planes→Coincident By Axis.

    • To create a global reference plane, in the Assembly environment, choose Features tab→Planes group→More Planes→Coincident By Axis.

    • To create a local reference plane, on the command bar for the current feature, in the Plane Options list, choose the Coincident Plane By Axis option.

  2. Select a planar face or reference plane. If you select a reference plane, a new reference plane is created, coincident with the selected reference plane. Otherwise, proceed with step 3.

  3. Define the base of the new reference plane by clicking a part face, a part edge, or another reference plane.

    Note:

    If you select a reference plane, the new reference plane’s orientation is identical to the selected reference plane.

  4. Define the origin of the new reference plane by clicking near one end of the displayed reference axis.

    Tip:

    You can use QuickPick to locate hidden part faces.
