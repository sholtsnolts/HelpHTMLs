Create a coordinate system (ordered)

  1. Do one of the following:

    • Choose Home tab→Planes group→Coordinate System .

    • In an assembly document, choose Features tab→Planes group→Coordinate System .

  2. Specify the origin for the new coordinate system.

    You can o this by selecting a keypoint or by using the X, Y, and Z boxes to entenr coordinates as offsets from the based coordinate system.

  3. Specify the vector and axis direction for the first axis of the new coordinate system.

  4. Specify the vector and axis direction for the second axis of the new coordinate system.

  5. On the command bar, click Finish.

    Tip:

    • You can use the Coordinate System Options dialog box to specify whether you want to define the new coordinate system using geometry on the model or by key-in relative to another coordinate system. Coordinate systems defined relative to model geometry are associative to the model geometry.

    • In the Assembly environment, the Coordinate System command is on the Features tab.

    • When you select a coordinate system, you can use the commands on the shortcut menu to show, hide, edit, rename, and delete the coordinate system.
