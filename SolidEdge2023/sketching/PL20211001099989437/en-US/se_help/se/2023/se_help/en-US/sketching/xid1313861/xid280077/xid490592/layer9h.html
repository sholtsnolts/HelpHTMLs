Display a layer or layers and hide all others

To show only the objects on a selected layer or layers while hiding all other layers:

  1. Click the Layers tab to list all layers in the document.

  2. On the Layers tab, select the layer or layers that contain the objects you want to show.

  3. On the layer shortcut menu, choose Show Only.

    Tip:

    When nested blocks are created using layers, and a block on one layer contains block graphics that reside on another layer, then the Show Only command does not display the geometry on the other layers. You can use the Show All Layers shortcut command to ensure that the block, and the block geometry that it references, are visible. Or you can use the Unblock command to unblock the blocks before selecting the Show Only command.
