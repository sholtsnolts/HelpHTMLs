Workflow for moving or copying a sketch

  1. Select sketch elements to copy or move in the part window individually or with a select box. If the sketch elements form a region, disable regions for the sketch before using the select box.

  2. Choose the Move command .

  3. Select point (on any of the selected sketch elements) to move or copy from. You can use keypoints to define the move or copy from point.

  4. Notice that there are options available in the Move command bar.

    If you want to move a copy, select Copy . You can enter the X, Y distances to move or copy to. You can also enter a step distance in the Step field.

  5. Click a point to move or copy to. If you are moving, then the command ends once a to point is clicked. The select set is still active. If you are copying, each click places a copy. A right-click during a copy will end the command with the select set still active.

  6. Press the Escape key to clear select set.
