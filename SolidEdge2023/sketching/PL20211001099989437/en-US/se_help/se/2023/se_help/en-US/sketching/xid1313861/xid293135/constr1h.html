Change a profile or sketch element into a construction element

  1. Choose Home tab→Draw group→Construction .

  2. Click the profile element or sketch element that you want to make a construction element.

    Tip:

    • When you change a profile element or sketch element into a construction element, the line type changes to the double chain line style.

    • You can also use the Construction command to change a construction element into a sketch element.

    • Some elements cannot be changed to sketch elements. For example, points, hole circles, and elements drawn to define the range of a circular pattern are always construction elements.

    • You can select multiple sketch elements to change using fence select.
