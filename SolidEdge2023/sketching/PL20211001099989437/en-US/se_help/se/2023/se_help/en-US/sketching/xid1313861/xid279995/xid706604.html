Change reference plane name display

  1. Choose Home tab→Select group→Select .

  2. Do the following:

    1. Right-click the plane in PathFinder.

    2. Select the Plane Name Display command.

  3. In the Plane Name Display dialog box, make changes to any of the following:

    • To turn the plane name display on or off, use the Show reference plane names option.

    • To change the color of the plane name, use the Color list.

    • To change the height of the plane name, change the value in the Height list.

    • To change the font used to display the plane name, use the Font list.

    • To show or hide the rectangle on the plane that represents the plane orientation, use the Show orientation option.

    Tip:

    You can also open the Plane Name Display dialog box from the Solid Edge Options dialog box, on the General tab.
