Line and Arc command bar

Type

Sets the drawing mode to Line (l), Arc (a), or Symmetric (s). You can also use the L, A, or S keys on the keyboard.

Properties
Length

Sets the length of a line during an edit.

This option is displayed when you are in edit line mode.

This option is greyed out during creation. The line length is controlled by the edit value box on the dynamic line during creation.

Angle

Sets the orientation angle of the element during an edit. Zero degrees is horizontal to the x axis. The angle increases in the counterclockwise direction.

This option is displayed when you are in edit line mode.

This option is greyed out during creation. The line angle is controlled by the edit value box on the dynamic line during creation.

Radius

Sets the radius.

This option is displayed when you are drawing in arc mode.

Sweep

Sets the sweep angle.

This option is displayed when you are drawing in arc mode.

Style
Style

Sets the active style.

Color

Sets the drawing color. You can click the More option to define custom colors with the Color dialog box.

Width

Sets the line width.

Type

Sets the drawing line type and style.

Projection
Turn on projection line

Specifies that the selected line is a projection line.

Type

Specifies the line type for projection lines.
