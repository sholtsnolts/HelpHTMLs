Arrange 2D elements

You can arrange the display order of 2D elements using the set of commands in the arrange group located on the Sketching (or Home) tab —> Arrange group.

Bring to Front command

You can use the Bring to Front command to move the selected elements to the top of the display stack of overlapping elements.

Send to Back command

You can use theSend to Back command to move the selected elements to the bottom of the display stack of overlapping elements.

Pull Up command

You can use thePull Up command to move the selected elements up one position in the display stack of overlapping elements.

Push Down command

You can use thePush Down command to move the selected elements down one position in the display stack of overlapping elements.

Note:

Bring to Front and Send to Back commands also exist in the shortcut menus for dimensions, annotations, and text boxes
