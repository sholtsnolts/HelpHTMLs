Split an element

Split an open element

  1. Select the Split command .

  2. Select the element you want to split.

  3. Click a point on the element where you want to split it.

A closed element requires you to define two points to split it.

Split a closed element

  1. Select the Split command .

  2. Select the element you want to split.

  3. Click the first point on the element where you want the split to begin.

  4. Click the second point on the element where you want the split to end.

Tip:

The Split command applies appropriate geometric relationships at the split point automatically. In some cases, this deletes an existing relationship.
