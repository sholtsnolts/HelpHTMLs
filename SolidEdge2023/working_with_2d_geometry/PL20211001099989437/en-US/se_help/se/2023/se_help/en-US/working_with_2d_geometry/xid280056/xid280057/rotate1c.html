Rotate command (2D elements)

Rotates one or more 2D elements a precise distance or angle about a specified point.

Note:

In the Draft environment, you can use this command to rotate drawing views. When you rotate a drawing view it is no longer aligned with the other drawing views. You can use the Align command on the shortcut menu to return the drawing view to its original orientation and align it with the other drawing views.
