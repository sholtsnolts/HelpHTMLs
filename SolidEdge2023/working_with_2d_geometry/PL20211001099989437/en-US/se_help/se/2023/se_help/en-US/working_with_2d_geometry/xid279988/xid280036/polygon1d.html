Polygon by Center command bar

Style

Sets the active style.

Line Color

Sets the drawing color. You can click the More option to define custom colors with the Color dialog box.

Line Type

Sets the drawing line type and style.

Line Width

Sets the line width.

By Vertex

Specifies that the polygon is drawn by center and vertex. This option circumscribes a circle about the polygon.

By Midpoint

Specifies that the polygon is drawn by center and midpoint of a side. This option inscribes a circle in the polygon.

Sides

Specifies the number of sides for the polygon.

Distance

Specifies the distance between the center of the polygon and the vertex or midpoint.

Angle

Specifies the angle of the polygon. Zero degrees is horizontal to the x axis. The angle increases in the counterclockwise direction.
