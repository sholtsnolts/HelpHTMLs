Circle command bar

Type

Sets the circle type to Tangent (t), By center (n), or By 3 points (p). You can also use the T, N, or P keys on the keyboard.

Properties
Diameter

Sets the diameter of the circle.

Radius

Sets the radius of the circle.

Style
Style

Sets the active style.

Color

Sets the drawing color. You can click the More option to define custom colors with the Color dialog box.

Width

Sets the line width.

Type

Sets the drawing line type and style.

How do I
Draw a circle by center point
Draw a circle by defining 3 points
Draw a circle tangent to one or two elements
