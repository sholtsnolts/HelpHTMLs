Parallel Relationship command

Use the Parallel Relationship command to make two lines parallel.

Note:

You can apply a parallel relationship to multiple elements in a single operation. After selecting an element to apply a parallel relationship to, you can press Shift+Select to select more elements or deselect selected elements. You can also use a Fence to select elements to apply a parallel relationship to. You can press Shift+Fence to add more elements to the set.
