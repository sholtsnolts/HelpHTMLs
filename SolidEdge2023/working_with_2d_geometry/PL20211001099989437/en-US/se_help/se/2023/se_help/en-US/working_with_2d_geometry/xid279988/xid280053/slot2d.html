Style

Sets the active style. This option is available only when working with a sketch.

Line Color

Sets the drawing color. You can click the More option to define custom colors with the Color dialog box.

Line Type

Sets the drawing line type and style. This option is available only when working with a sketch.

Line Width

Sets the line width. This option is available only when working with a sketch.

Symmetric Offset Options

Accesses the Symmetric Offset Options dialog box.

Select Step

Specifies the elements you want to offset to construct the symmetric offset.

Select

Sets the tool selection method for the symmetric offset operation.

  • Chain—Allows you to select a chain of elements to create the symmetric offset.

  • Single—Allows you to select a single element to create the symmetric offset.

Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selection.
