Draw connected lines and arcs with FreeSketch


  1. Choose the Home tab (or Sketching tab)→Draw group→Line list→FreeSketch command .

  2. On the FreeSketch command bar, click the Line and Arc element type buttons and clear the other element type buttons.

  3. Drag to draw a line or arc. When you stop moving the cursor, do not release the mouse button.

  4. Drag to draw a series of line segments and arcs, continuing to hold the mouse button as you draw each line and arc.

    Tip:

    If you release the mouse button while you are drawing, you can start drawing at the last end point you entered. Make sure the IntelliSketch End Point option is set.

  5. When you have finished, release the mouse button.

    Tip:

    • The Adjust option specifies whether the geometry is adjusted as you draw.

      • When the Adjust option is off, the software draws the exact movements of your cursor.

      • When the Adjust option is on, the software interprets the lines as horizontal or vertical and the arcs as tangent to the elements they are connected to. IntelliSketch places relationship handles.

    • IntelliSketch recognizes relationships at the start point and end point of the line. IntelliSketch places relationship handles.

    • You can use the boxes on the FreeSketch command bar and the commands on the shortcut menu to edit a line or an arc.
