Format a fill

  1. Select a fill.

  2. On the Fill command bar, click the settings that you want to update the fill.

Tip:

  • You can format a fill before you place it by setting options on the command bar or with the Properties command.

  • You can apply several formats at once by clicking a new fill style in the Style list box on the command bar.

  • You can modify an existing fill style or create a new one with the Style command.
