Apply a Rigid Set relationship to a set of elements

  1. Choose Home or Sketching tab→Relate group→Rigid Set .

    Note:

    In the Draft environment, the Rigid Set button is located in the Sketching tab→Relate group.

  2. Click the elements to add to the set.

    Note:

    You can hold the Shift button and click an element to remove it from the set.

  3. On the Rigid Set command bar, click the Accept button.

    Tip:

    • You cannot locate 2D elements that have an include, offset, include + offset, or include + rigid set relationship.

    • You cannot locate existing dimensions and annotations.

    • You can locate elements that have a rigid set relationship. Locating an element in a rigid set removes it from the existing set and places it in a new set.
