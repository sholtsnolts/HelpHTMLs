Simplify a curve

  1. Select the Simplify Curve command from the command bar or right-click a curve element, and then choose Simplify from the shortcut menu.

  2. In the Simplify Curve dialog box, use the Increase and Decrease arrow buttons to change the number of edit points and control vertex points.

  3. Click OK to update the curve.
