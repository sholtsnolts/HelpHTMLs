Convert analytic geometry to a b-spline curve

  1. Choose the Convert to Curve command .

  2. Select the geometry you want to convert.

Tip:

  • You can convert a single element or multiple connected elements.

  • You can drag the mouse to fence elements or press the Shift key and click each element to select more than one element.

  • The command also works in object/action mode which means you can select the elements you want to convert and then select the Convert to Curve command.
