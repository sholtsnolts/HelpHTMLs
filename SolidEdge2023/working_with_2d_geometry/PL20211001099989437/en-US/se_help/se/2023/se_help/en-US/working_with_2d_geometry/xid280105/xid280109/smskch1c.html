IntelliSketch command

Customizes the operation of IntelliSketch . The relationships selected on the IntelliSketch dialog box are recognized as you draw. To turn IntelliSketch off, clear all relationships.

Note:

IntelliSketch does not interrupt the active command. For example, if you are in the middle of drawing a profile and click IntelliSketch to set the relationship options, the drawing command does not terminate. Once you finish with IntelliSketch, you return to the drawing command in the same state as it was when you left.
