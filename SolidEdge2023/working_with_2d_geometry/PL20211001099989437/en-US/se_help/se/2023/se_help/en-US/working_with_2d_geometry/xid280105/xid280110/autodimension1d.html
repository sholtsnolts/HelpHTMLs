Auto-Dimension page (IntelliSketch dialog box)

The Auto-Dimension page of the IntelliSketch dialog box sets automatic dimensioning options.

Automatically Create Dimension For New Geometry

Specifies that dimensions should automatically be created for new geometry.

When Geometry Is Drawn

Automatically creates dimensions only when geometry is drawn, as opposed to when being keyed in.

Only When Geometry Is Created With Keyed-In Values

Automatically creates dimensions only when dimension values are keyed in.

Do Not Create Dimensions For Fully Defined Geometry

Does not create dimensions for fully defined geometry.

Create Driven Dimensions For Fully Defined Geometry

Automatically places driven dimensions for fully defined geometry.

Use Dimension Style Mapping

Specifies that automatically placed dimensions inherit the settings on the Dimension Style page of the Options dialog box.
