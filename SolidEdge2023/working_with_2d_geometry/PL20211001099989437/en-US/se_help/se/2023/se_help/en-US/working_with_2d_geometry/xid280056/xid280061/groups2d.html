Group command bar

Style

Defines a line style for the selected group.

Line Color

Defines a line color for the selected group.

Line Type

Defines a line type for the selected group.

Line Width

Defines a line width for the selected group.

Select

Selects the elements you want to add to or remove from the group.

Group Properties

Accesses the Properties dialog box for the group.

Name

Defines the name of the group. Group names do not have to be unique if they are not on the same drawing sheet.

Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selection.
