Draw command (sketch)

The Draw command draws using a pen, finger, or mouse to create a freehand drawing that is converted into precise, accurate geometry. This command is enabled only on Windows 10, version 1607 and beyond.

Use the Create Geometry option to set the pen mode to geometry creation. Create geometry by dragging your pen, finger or mouse.

Use the Trim Geometry option to set the pen mode to trim geometry. Trim or delete geometry by dragging your pen, finger or mouse.
