Rectangular Pattern command bar (Draft)

Options

Displays the Rectangular Pattern Options dialog box.

X Count

Sets the number of pattern members along the x axis of the pattern rectangle.

Y Count

Sets the number of pattern members along the y axis of the pattern rectangle.

X Offset

Sets the distance between adjacent rows in the pattern, measured along the x axis of the pattern rectangle.

Y Offset

Sets the distance between adjacent rows in the pattern, measured along the y axis of the pattern rectangle.

Angle

Sets the rotation angle of the pattern rectangle.
