Draw a circle with FreeSketch

Note:

The IntelliSketch options Point On Element or End Point must be set to draw circles that are tangent to other elements.

  1. Choose the Home tab (or Sketching tab)→Draw group→Line list→FreeSketch command .

  2. On the FreeSketch command bar, click the Circle element type button and clear the other element type buttons.

  3. Drag to sketch the shape of the circle or the diameter of the circle.

    Tip:

    • The Adjust option specifies whether the geometry is adjusted as you draw.

      • When the Adjust option is off, the software draws the exact movements of your cursor.

      • When the Adjust option is on, the software interprets your cursor movements as a circle.

    • IntelliSketch places relationship handles.

    • You can use the options on the FreeSketch command bar and the commands on the shortcut menu to edit a circle.
