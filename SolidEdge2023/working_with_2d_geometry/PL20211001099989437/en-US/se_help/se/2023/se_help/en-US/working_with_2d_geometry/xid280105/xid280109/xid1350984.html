Locate Background command

In the Draft environment, you can use the Sketching tab→IntelliSketch group→Locate Background command for precise positioning of objects on the working sheet with respect to 2D elements on the background sheet. Use this command when you want to draw new elements or align existing objects by locating keypoints on background sheet geometry, such as the drawing border or the title block.

For example, you can:

  • Draw a line that is exactly horizontal or vertical to geometry on the background sheet.

  • Locate a keypoint in the title block when placing a callout to extract the title of the document.

  • Position a table so that its origin point is precisely aligned to a corner of the drawing border.

Note:

The IntelliSketch settings and the Alignment Indicator command control which keypoints can be located and whether alignment lines are displayed. Keypoints and alignment lines are displayed for reference only. No associative relationships are created.
