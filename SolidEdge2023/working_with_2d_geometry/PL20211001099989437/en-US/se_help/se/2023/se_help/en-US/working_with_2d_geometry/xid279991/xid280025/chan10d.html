Tangent command bar (sketch environment)

Relationship

Sets the tangent option you want. The option you set determines what types of elements are available.

  • Tangent—Specifies that you want two elements tangent. The first element you select can be linear or nonlinear.

  • Tangent + Equal Curvature—Specifies that you want two elements tangent and with an equal radius of curvature. The first element you select must be a curve, ellipse or partial ellipse. The elements can be 2D or 3D.

  • Parallel Tangent Vectors—Specifies that you want the tangent vectors for the two elements to be parallel. The first element you select must be a curve, ellipse or partial ellipse. The elements can be 2D or 3D.

  • Parallel Tangent Vectors + Equal Curvature—Specifies that you want the tangent vectors for the two elements to be parallel and with an equal radius of curvature. The first element you select must be a curve, ellipse or partial ellipse. The elements can be 2D or 3D.
