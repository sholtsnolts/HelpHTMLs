Info page, Element Properties dialog box (Curve)

Type

Displays the type of the selected element.

Sheet

Displays the name of the drawing sheet the element is on.

Layer

Sets the layer the element is on. This information is only displayed in the Draft environment.

Curve Type

Displays the type of curve (open or closed).

Degree

Displays the number of degrees of a curve.

Edit Points

Displays the coordinates of the edit points on the curve.

Points

Lists the edit points on a selected curve. When you select an edit point from the list, its x and y values are displayed. The values are relative to the current user defined origin of the sketch that owns the geometry.

X

Displays a value for the x coordinate.

Y

Displays a value for the y coordinate.

Length

Displays the length of a curve.

Area

Displays the area of a closed curve. Although you cannot edit the Area, the box automatically updates when you edit the geometry.
