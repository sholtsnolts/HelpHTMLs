Move command bar

Specifies the from point and to point when you move objects or elements.

Copy

Copies the elements in the selection set. This option is not selectable if a drawing view is in the selection set.

Step Value

Increments or decrements the value displayed in the command bar boxes. For example, typing a step value of 0.25 and moving the cursor away from the from point would increment the distance from 0.25 to 0.5, 0.75, and so forth.

X Coordinate

Sets a value for the x coordinate. You can use this option by itself or with the Y option.

Y Coordinate

Sets a value for the y coordinate. You can use this option by itself or with the X option.
