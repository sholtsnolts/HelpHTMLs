Relationship Assistant command bar

Options

Opens the Relationship Assistant dialog box.

Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selection.

Show Variability

Shows how a profile that needs additional relationships or dimensions applied can change based on the current relationships and dimensions.

To use this option, first drag a fence around the profile or related sketch elements, and then click the Accept button. When you click the Show Variability button, the number of relationships needed is displayed, and a temporary display of the profile using the highlight color illustrates how the profile can change. You can click this button repeatedly to illustrate other possible variations.

Note:

Show Variability shows the degrees of freedom for one set of constrained elements at a time. It is not intended to work with multiple sets of unrelated, independent elements.

Relationships Needed

Displays the number of relationships needed to fully constrain the selected profile elements.
