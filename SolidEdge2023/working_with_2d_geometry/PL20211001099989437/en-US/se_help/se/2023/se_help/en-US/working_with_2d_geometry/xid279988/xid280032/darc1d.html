Arc command bar

Type

Sets the the drawing mode to By center (n), By 3 points (p), or Tangent (t). You can also use the N, P, or T keys on the keyboard.

Properties
Radius

Sets the radius.

This option is displayed when you are drawing in arc mode.

Sweep

Sets the sweep angle.

This option is displayed when you are drawing in arc mode.

Style
Style

Sets the active style.

Color

Sets the drawing color. You can click the More option to define custom colors with the Color dialog box.

Width

Sets the line width.

Type

Sets the drawing line type and style.
