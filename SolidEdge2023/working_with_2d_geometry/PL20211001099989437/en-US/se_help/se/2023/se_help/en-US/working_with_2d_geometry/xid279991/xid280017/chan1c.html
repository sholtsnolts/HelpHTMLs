Concentric command

The Concentric command makes an arc or circle concentric with another arc or circle.

Note:

You can apply a Concentric relationship to multiple elements in a single operation. After selecting an element to apply a concentric relationship to, you can press Shift+Select to add more elements to the set or deselect elements from the set. You can also use a Fence to select elements to apply a concentric relationship to. You can press Shift+Fence to add more elements to the set. Shift+Fence does not deselect selected elements.
