Select all the peer relationships in a sketch

  1. In the graphics window when nothing is selected, right-click to display the shortcut menu.

  2. On the shortcut menu, choose Select All Peer Relationships.

    Tip:

    You can use this command to select all the dimensions and geometric relationships in the current sketch that reference other parts in the assembly.
