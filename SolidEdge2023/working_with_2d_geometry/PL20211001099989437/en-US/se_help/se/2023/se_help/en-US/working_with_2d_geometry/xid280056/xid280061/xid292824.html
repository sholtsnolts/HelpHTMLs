Ungroup All command

Removes all elements from all groups, including:

  • Top level groups

  • Nested groups

  • All groups on all sheets (working, background, and 2D Model)

A dialog box prompts you to confirm or reject the action. You also can use the Undo command to restore all nested groups.

Tip:

You can use the Ungroup All command, along with the Unblock All command, before you copy and paste imported AutoCAD geometry to a Solid Edge sketch. These commands simplify nested groups and nested block occurrences by reducing them to their lower level geometry.
