Saved Settings

Lists saved bead settings. You can access the saved settings by selecting them from the list. The settings on the dialog box display the characteristics of the bead you select.

You can type a name in the box to name a group of settings. You can then use the Saved Settings list on the Bead Options dialog box or the Bead command bar to select a saved setting later in any Solid Edge document where you can construct bead features. The saved settings are added to the Custom.xml file in the C:\Program Files\Siemens\Solid Edge 2023\Preferences folder. You can also use the File Locations tab on the Solid Edge Options dialog box to specify a different folder for the Custom.xml file.
