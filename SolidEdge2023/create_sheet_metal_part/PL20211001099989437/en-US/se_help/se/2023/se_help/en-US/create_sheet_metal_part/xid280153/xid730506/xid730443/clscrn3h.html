Edit an ordered closed 2 bend corner

  1. Select a closed corner.

  2. Select the edit definition button.

  3. On the Close 2–Bend Corner command bar, click the Select Bends step.

  4. Use the command bar optons to edit the close corner and press Enter.

  5. On the command bar, click Finish to save the edits.

Edit a synchronous closed 2–bend corner

  1. Select a closed corner.

  2. Select the edit definition handle.

  3. Use the command bar or dynamic edit control to edit the close corner and press Enter.

  4. Click to save the edits.
