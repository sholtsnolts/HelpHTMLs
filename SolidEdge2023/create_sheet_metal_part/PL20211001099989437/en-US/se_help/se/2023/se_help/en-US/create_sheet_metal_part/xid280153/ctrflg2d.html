Contour Flange Options dialog box

Defines the bend radius, bend relief, and corner relief options for the flange. You can use the default values, which are located on the Options dialog box, or enter new values.

You can also add a miter to the ends and close the interior corners of a contour flange using the Miters and Corners tab.

Dialog Box Options
Save Default

Saves the current settings as the default. Saved defaults will be used the next time you use this command. If you do not save the current settings as defaults, the previous default settings will be used.
