Convert a part to sheet metal

  1. Choose Tools tab→Transform group→Part to Sheet Metal.

  2. Click the linear edges to create the bends for the sheet metal part.

  3. (Optional) Click the direction arrow to change the thickness side.

  4. (Optional) Type a value to change the face thickness.

  5. (Optional) Override the corner value.

  6. (Optional) Override the bend values.

  7. Right-click to convert the part.

Note:

You can also select a partially cylindrical face on a part model to define the bend for the sheet metal part.
