Construct a contour flange

You can construct a contour flange as the first feature in a sheet metal document or add a contour flange to an existing tab or flange.

Construct a contour flange in the ordered environment

  1. Choose Home tab→Sheet Metal group→Contour Flange .

  2. Define the profile plane.

  3. Draw an open profile or copy a profile into the profile window.

  4. Choose Home tab→Close group→Close.

  5. Define the extent of the contour flange.

  6. Finish the feature.

Construct a contour flange in the synchronous environment

  1. Choose Home tab→Sheet Metal group→Contour Flange .

  2. Click a sketch to define the contour flange.

  3. (Optional) Use the options on the QuickBar to define parameters for the contour flange.

  4. Define the extent of the contour flange.

  5. Click the handle to define the direction for the contour flange.

  6. Type an extent for the contour flange.

  7. Click to finish the feature.

Tip:

You can also define the material thickness of a contour flange by typing a value in the dynamic edit control displayed when defining the extent. You can use the Tab button to switch between the extent distance and material thickness.
