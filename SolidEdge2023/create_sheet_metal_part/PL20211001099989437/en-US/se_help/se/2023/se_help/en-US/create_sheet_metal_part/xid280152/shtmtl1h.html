Construct a sheet metal base feature

Part features add material to or remove material from a base feature, a solid shape that defines the part's basic shape.

When you create a new document in the Sheet Metal environment, the Tab command is active by default, so you can construct a base feature.

  1. On the Home tab, in the Sheet Metal group, click one of the following buttons:

    • Tab

    • Contour Flange

    Note:

    Note:

    When creating a tab, you can also right-click a region and select the command from the ribbon or the context toolbar.

  2. Define the profile plane.

  3. Draw a profile or copy a profile into the profile window.

    Note:

    You must use an open profile when constructing a base feature with the Contour Flange command. You must use a closed profile when constructing a base feature with the Tab command.

  4. Choose Home tab→Close group→Close Sketch to close the profile view.

  5. Click to define the side of the profile you want to add material to.

  6. Finish the feature.
