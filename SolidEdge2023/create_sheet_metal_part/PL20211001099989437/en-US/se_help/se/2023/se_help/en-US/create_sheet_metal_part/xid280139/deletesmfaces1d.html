Delete Relief Faces command bar

Main Steps
Select Step

Specifies the faces you want to delete from the model. Solid Edge automatically selects all faces created by zero-radius bends and system-generated bend relief. You can use the Select Zero-Radius Faces and Select Bend Relief Faces buttons to toggle the selection of these faces.

Preview/Finish/Cancel

This button changes function as you move through the feature construction process. The Preview button shows what the constructed feature will look like, based on the input provided in the other steps. The Finish button constructs the feature. After previewing or finishing the feature, you can edit it by re-selecting the appropriate step on the command bar. The Cancel button discards all input and exits the command.

Select Zero-Radius Faces

Selects the small faces created for zero-radius bends.

Select Bend Relief Faces

Selects the small faces created by system-generated bend relief.

Name

Displays the feature name. Feature names are assigned automatically. You can edit the name by typing a new name in the box on the command bar or by selecting the feature and using the Rename command.
