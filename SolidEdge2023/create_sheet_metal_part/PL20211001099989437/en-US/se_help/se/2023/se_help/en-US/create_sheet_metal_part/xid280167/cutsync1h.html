Construct a sheet metal cutout from a closed profile

You can construct sheet metal cutouts using the Cut command or the Select tool. Both workflows are explained in this topic.

Construct a sheet metal cutout using the Cut command

  1. Choose Home tab→Sheet Metal group→Hole list→Cut .

    Note:

    You can also right-click a region and select the command from the ribbon or the context toolbar.

  2. Select the region(s) to define the cutout.

  3. Type a value for the cutout extent or right-click for a dynamic preview of the cutout.

  4. (Optional) Click the handle to change the cutout direction.

  5. (Optional) Use the command bar to change the cutout options.

  6. Right-click to place the cutout.

Construct a sheet metal cutout using the Select tool

  1. Select the region(s) to define the cutout.

  2. Click the handle to specify the cutout direction.

  3. (Optional) Use the command bar to change the cutout options.

  4. Right-click to place the cutout.
