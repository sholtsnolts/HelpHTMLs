Thin Part to Sheet Metal Options dialog box

Tab Options
Bend Relief

Specifies that you want to apply bend relief to the source face from which the sheet metal feature is constructed. When you set this option, you can also specify whether the bend relief is round or square, and whether the bend relief applies to only the material adjacent to the bend or to the entire face.

This option is not available with the Transform to Synchronous Sheet Metal command.

Square

Specifies that the internal corners contain square bend relief.

Round

Specifies that the internal corners contain round bend relief.

Depth

Specifies the depth of the bend relief.

Use Default Value

Uses the default value specified on the Options dialog box.

Enables the associated value list so you can override the default value specified on the Gage tab of the Material Table dialog box.

Width

Specifies the width of the bend relief.

Save Default

Saves the current settings as the default. Saved defaults will be used the next time you use this command. If you do not save the current settings as defaults, the previous default settings will be used.
