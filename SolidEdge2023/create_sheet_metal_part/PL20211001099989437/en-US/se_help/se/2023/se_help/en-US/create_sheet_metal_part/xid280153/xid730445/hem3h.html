Move, rotate, or edit a hem

In the ordered environment, you can edit the hem feature to change such things as the hem type and material setback.

You can move a hem along the face, rotate a hem, or edit the hem feature.

Edit a hem feature in the ordered environment

  1. Select the hem.

  2. Click the Edit Definition button.

  3. Use the Hem command bar and Hem Options dialog box to make changes to the hem.

  4. Click the Finish button.

Move a hem in the synchronous environment

  1. Select the hem.

  2. Click the steering wheel axis shown.

  3. Move the cursor until the face is positioned where you want it, then click.

Rotate a hem in the synchronous environment

  1. Select the hem.

  2. Click the steering wheel torus shown.

  3. Move the cursor until the face is positioned where you want it, then click.

Edit a hem feature in the synchronous environment

  1. Select the hem.

  2. Click the edit handle.

  3. Use the Hem QuickBar and Hem Options dialog box to make changes to the hem.

  4. Click to save the edit.
