Contour Flange command

Use the Contour flange command to construct a contour flange by extruding a profile that represents the edge of the contour flange.

Creating a contour flange in the context of an assembly

When in place activated into a synchronous sheet metal document, a contour flange can be created from a chain of edges from adjacent parts. The contour flange must be the first feature in the sheet metal document.
