Gusset Option

Accesses the Gusset Options dialog box.

Saved Settings

Lists saved gusset settings. You can access the saved settings by selecting them from the list. The settings on the dialog box display the characteristics of the gusset you select. You can type a name in the box to name a group of settings.

Keypoints

Sets the type of keypoint you can select to define a feature extent or to position a new reference plane. With keypoints, you can define the feature extent or the location of the reference plane using a keypoint on other existing geometry. The available keypoint options are specific to the command and workflow you use. This option is available only when creating a single pattern gusset.

Selects any keypoint.

Selects an end point.

Selects a midpoint.

Selects the center point of a circle or arc.

Selects a tangency point on an analytic curved face such as a cylinder, sphere, torus, or cone.

Selects a silhouette point.

Selects an edit point on a curve.
Patterning

Specifies the type of gusset pattern to be created.

  • Single—You specify a keypoint or an offset distance for the gusset.

  • Fit—You specify the number of occurrences of the gusset and the software calculates the spacing. If the length of the parent bend changes, a fit patterned gusset recomputes.

  • Fill—You specify the spacing and the software calculates the number of occurrences. If the length of the parent bend changes, a fill patterned gusset recomputes.

  • Fixed—You specify the number of occurrences and the spacing.
