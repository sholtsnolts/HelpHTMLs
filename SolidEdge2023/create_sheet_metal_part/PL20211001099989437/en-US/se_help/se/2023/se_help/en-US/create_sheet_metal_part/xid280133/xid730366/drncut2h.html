Construct a drawn cutout

In the ordered environment, you can construct drawn cutouts using the Drawn Cutout command.

In the synchronous environment, you can construct drawn cutouts using the Drawn Cutout command or the Select tool. Both workflows are explained in this topic.

Construct a drawn cutout in the ordered environment

  1. Choose Home tab→Sheet Metal group→Dimple list→Drawn Cutout.

    Note:

    You can also right-click a region and select the command from the ribbon or the context toolbar.

  2. Define the profile plane.

  3. Draw a profile or copy a profile into the profile window.

  4. Choose Home tab→Close group→Close.

  5. Do one of the following:

    • If you drew a closed profile, skip to step 6.

    • If you drew an open profile, click to define the material side.

  6. Click to define the extent.

  7. Finish the feature.

Construct a drawn cutout using the Drawn Cutout command in the synchronous environment

  1. Choose Home tab→Sheet Metal group→Dimple list→Drawn Cutout .

  2. Select the region(s) to define the drawn cutout.

  3. Type a value for the drawn cutout extent or use the mouse wheel for a dynamic preview of the cutout.

  4. (Optional) Click the handle to change the drawn cutout direction.

  5. Right-click to place the cutout.

Construct a drawn cutout using the Select tool in the synchronous environment

  1. Select the region(s) to define the drawn cutout.

  2. Choose Home tab→Sheet Metal group→Dimple list→Drawn Cutout .

  3. Type a value for the drawn cutout extent or use the mouse wheel for a dynamic preview of the cutout.

  4. (Optional) Click the handle to change the drawn cutout direction.

  5. Right-click to place the cutout.
