Gusset Options dialog box

Saved Settings

Lists saved gusset settings. You can access the saved settings by selecting them from the list. The settings on the dialog box display the characteristics of the gusset you select. You can type a name in the box to name a group of settings.

Save

Saves the current settings to the name you type.

Delete

Deletes the saved settings selected in the Save Settings box.

Gusset Profile
Depth

Specifies the depth of the gusset.

Rounding
Include Rounding

Specifies that the edges of the gusset are rounded.

Punch Radius

Specifies the radius value at the bottom of the gusset.

Die Radius

Specifies the radius value for the top of the gusset.

Gusset Shape

Specifies the shape for the gusset.

Round

Specifies that you want the gusset to be round.

Square

Specifies that you want the gusset to be square.

Radius

Specifies the gusset radius.

Taper Angle

Specifies the taper angle for the gusset.

Width

Specifies the width of the gusset.
