Construct a dimple

In the ordered environment, you can construct a dimple using the Dimple command. In the synchronous environment, you can construct dimples using the Dimple command or the Select tool. Both workflows are explained in this topic.

Construct a dimple in the ordered environment

  1. Choose Home tab→Sheet Metal group→Dimple list→Dimple button .

    Note:

    You can also right-click a region and select the command from the ribbon or the context toolbar.

  2. Define the profile plane.

  3. Draw a profile or copy a profile into the profile window.

  4. Choose Home tab→Close group→Close.

  5. Do one of the following:

    • If you drew a closed profile, skip to step 6.

    • If you drew an open profile, click to define the material side.

  6. Click to define the extent.

  7. Finish the feature.

Construct a dimple using the Dimple command in the synchronous environment

  1. Choose Home tab→Sheet Metal group→Dimple list→Dimple button .

  2. Select the region(s) to define the dimple.

  3. Type a value for the dimple extent or use the mouse wheel for a dynamic preview of the dimple.

  4. (Optional) Click the handle to change the dimple direction.

  5. Right-click to place the dimple.

Construct a dimple using the Select tool

  1. Choose Home tab→Select group→Select .

  2. Select the region(s) to define the dimple.

  3. Choose Home tab→Sheet Metal group→Dimple list→Dimple button .

  4. Type a value for the dimple extent or use the mouse wheel for a dynamic preview of the dimple.

  5. (Optional) Click the handle to change the dimple direction.

  6. Right-click to place the dimple.
