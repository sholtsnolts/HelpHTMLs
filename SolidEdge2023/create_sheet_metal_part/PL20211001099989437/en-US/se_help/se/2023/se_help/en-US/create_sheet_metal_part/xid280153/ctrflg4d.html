Miters and Corners tab

Tab Options
Start End

Sets the miter options for the start end of the contour flange.

Miter

Miters the end of the contour flange when checked.

Angle

Sets the miter angle for the specified end of the contour flange. A negative value will miter the flange inward and will typically remove material. A positive value will miter the flange outward and will typically add material.

With some complex contour flanges, material can be both added or removed when an end is mitered.

Normal to Thickness Face

Miters the end of the contour flange perpendicular to the thickness face.

Normal to Source Face

Miters the end of the contour flange perpendicular to the source face.

Finish End

Sets the miter options for the finish end of the contour flange.

Miter

Miters the end of the contour flange when checked.

Angle

Sets the miter angle for the specified end of the contour flange. A negative value will miter the flange inward and will typically remove material. A positive value will miter the flange outward and will typically add material.

With some complex contour flanges, material can be both added or removed when an end is mitered.

Normal to Thickness Face

Miters the end of the contour flange perpendicular to the thickness face along a planar tab.

Normal to Source Face

Miters the end of the contour flange perpendicular to the source face.

Interior Corners

Specifies whether or not to close the interior corners of the feature and specify the type of corner  treatment you want. You can close the corners for both two bend and three bend corners.

Close Corner

Specifies that you want to close the corners for the interior corners. Bend relief is applied when you close interior three bend corners.

Treatment

Specifies the treatment you want to apply to the bent faces of the flanges. You can apply no treatment to the bent faces, close the bent faces until the edges intersect, or apply a circular cutout. This option is only available if you select the Close Corner option.

Gap

Specifies a gap between the faces. The gap distance cannot be negative and cannot be larger than the material thickness.

Diameter

Specifies the diameter for the circular cutout. This option is only available when you set the Treatment option to Circular Cutout.

Miter Bend Edges That are Larger Than Default Bend Radius

Specifies that the corners of the contour flange that have a non-default bend radius are mitered.

Miter Using Normal Cutout Method

Specifies that the corners of the contour flange are mitered normal to the thickness face. Setting this option constructs miters that are closer together. Setting this option may also allow you to unbend a contour flange that would not unbend previously.

Tip:

Use this option when you want to use the Close 3 Bend Corner command to close a contour flange that contains three corners.
