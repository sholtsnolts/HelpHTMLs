Edit a gusset

Edit a gusset in the ordered environment

  1. Click the gusset to edit.

  2. Select the Edit Definition button

  3. On the command bar, click Options and use the Gusset Options dialog box to edit the gusset parameters.

  4. On the Gusset Options dialog box, OK.

  5. Click to save the change.

Note:

You can also select a gusset, then select the Dynamic Edit button to reposition the gusset along the bend.

Edit a gusset in the synchronous environment

  1. Click the gusset to edit.

  2. Click the gusset edit handle.

  3. Use the command bar options to edit the gusset.

  4. Click to complete the edits.
