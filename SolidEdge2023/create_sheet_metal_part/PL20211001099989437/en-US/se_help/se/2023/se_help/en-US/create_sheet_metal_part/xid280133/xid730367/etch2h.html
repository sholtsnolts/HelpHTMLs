Move or edit an etch

In the ordered environment, you can dynamically edit an etch, edit the etch feature to change such things as the etch color or line width, or you can edit the etch profile to make changes to the sketch element used to create the etch.

In the synchronous environment, you can move an etch along the face, move an etch to another face, edit the etch feature, or you can edit the etch profile to make changes to the sketch element used to create the etch.

Dynamically edit an etch in the ordered environment

  1. Choose Home tab→Select group→Select .

  2. Select the etch.

  3. Click the Dynamic Profile button.

  4. Drag the etch to dynamically change the etch and click to make the change.

  5. Click to save the edit.

Edit an etch feature in the ordered environment

  1. Choose Home tab→Select group→Select .

  2. Select the etch.

  3. Click the Edit Definition button.

  4. Use the options to edit the etch definition.

  5. Click to save the edit.

Edit an etch profile in the ordered environment

  1. Choose Home tab→Select group→Select .

  2. Select the etch.

  3. Click the Edit Profile button.

  4. Make changes to the existing etch profile.

  5. Click the green Accept button .

  6. Click to save the edit.

Move an etch along the same face in the synchronous environment

  1. Choose Home tab→Select group→Select .

  2. Select the etch.

  3. Click the steering wheel axis shown.

    .

  4. Drag the etch to a new location and then click.

  5. Click to save the move.

Move an etch to a new face in the synchronous environment

  1. Choose Home tab→Select group→Select .

  2. Select the etch.

  3. Click the edit handle (1).

    .

  4. Click the Edit Face Definition button.

    .

  5. Drag the etch to a new location and then click.

  6. Right-click to exit the edit mode.

  7. Click to save the move.

Note:

You cannot move etch features to faces that are added to the model after the etch, such as a bend or jog.

Edit an etch feature in the synchronous environment

  1. Choose Home tab→Select group→Select .

  2. Select the etch.

  3. Click the edit handle (1).

  4. Use the options to edit the etch attributes and then right-click.

  5. Click to save the edit.

Edit an etch profile in the synchronous environment

  1. Click the etch to display the edit handle.

  2. Click the edit handle (1).

  3. Click the feature profile edit handle (2).

  4. Make changes to the existing etch profile.

  5. Click the green Accept button .

  6. Right-click to exit the Edit Profile mode.

  7. Click to save the edit.
