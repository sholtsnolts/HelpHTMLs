Create a sheet metal part (workflow)

Use the following process to create a sheet metal part.

  1. Create the sheet metal document.

    Create a document using the iso sheet metal.psm template.

  2. Construct the base feature.

    Construct the base feature with the Tab, Contour Flange, and Lofted Flange commands.

  3. Define the sheet metal part properties.

    Use the Gage tab on the Material Table dialog box to define properties such as sheet metal gage, material thickness, and bend radius.

  4. Add flanges to the sheet metal part.

    Use the Flange command to add flanges to the sheet metal part.

  5. Add features to the sheet metal part.

    Construct additional features such as tabs, bends, and jogs.

  6. Add deformation features to the sheet metal part.

    Construct features that are manufactured with metal deformation techniques, such as deep drawing and coining. These features include such things as dimples, louvers, gussets, and beads.

  7. Apply bend and corner relief.

    Use the Flange Options dialog box to apply bend and corner relief.

  8. Add cutouts to the part.

    Add the Cutout command or the Normal Cutout command to add cutouts to the sheet metal part.

  9. Flatten the sheet metal part.

    Use the Flat Pattern command or Save as Flat command to create a flat pattern of the sheet metal part.

  10. Save the sheet metal part
