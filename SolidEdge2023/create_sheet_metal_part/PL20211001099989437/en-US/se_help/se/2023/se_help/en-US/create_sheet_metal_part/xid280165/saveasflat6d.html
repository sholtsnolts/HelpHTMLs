Font Mapping tab (Save As Flat DXF Options dialog box)

Defines the mapping between AutoCAD and Solid Edge fonts.

Delete

Deletes the selected font.

AutoCAD Font

Displays a list of available AutoCAD fonts. The list of fonts is defined by the active configuration file. The list also includes AutoCAD fonts found in the AutoCAD file being translated. Fonts found in the AutoCAD  file precede the list of fonts found in the Configuration file.

To change the font mapping, click the AutoCAD font you want to change the mapping for and type the full name for the new font, including the extension.

Solid Edge Font

Displays the fonts used in Solid Edge sheet metal documents. You cannot select the fonts that are listed.
