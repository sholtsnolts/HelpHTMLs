Blank Options dialog box

Cut Size

Specifies information about the cut size of the blank. These options are not available when creating a blank surface.

Length (X)

Displays the cut length of the blank.

Width (Y)

Displays the cut width of the blank.

Show Cut Size Range and Dimensions

Displays a range box for the blank along with the dimensions for the current flat pattern cut size. The dimensions match the Length (X) and Width (Y) values displayed in the Current Size section of the Flat Pattern Options dialog box.

Formability Properties

Specifies information used to control the formability of the blank.

Strain hardening exponent

A materials property that is used when calculating stress-strain behavior in work hardening. The value of the strain hardening exponent is between 0 and 1. A value of 0 means that a material is a perfectly plastic solid, while a value of 1 represents a 100% elastic solid. Most metals have a value between 0.10 and 0.50.

Strength coefficient

A materials property that is used along with stress and strain on a material to determine how much a material will deform as it changes shape.

Alarm

Specifies information used for setting alarms regarding the size of the blank. These options are not available when creating a blank surface.

Length alarm

Enables the Length alarm.

Length (X)

Specifies the limit for the length of the blank size. If the actual blank size exceeds this length, an alarm icon is displayed adjacent to the blank entry in PathFinder.

Width alarm

Enables the Width alarm.

Width (Y)

Specifies the limit for the width of the blank size. If the actual blank size exceeds this length, an alarm icon is displayed adjacent to the flat pattern entry in PathFinder.

Use Default Values

Specifies that you want to use the default values that are set in the Flat Pattern Treatments tab on the Solid Edge Options dialog box.
