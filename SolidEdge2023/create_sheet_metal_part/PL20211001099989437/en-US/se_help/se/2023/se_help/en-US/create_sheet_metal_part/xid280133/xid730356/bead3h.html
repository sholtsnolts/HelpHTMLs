Move or edit a bead

In the ordered environment, you can dynamically edit a bead , edit the bead feature to change such things as the bead direction and extent, or you can edit the bead profile to make changes to the sketch element used to create the bead.

In the synchronous environment, you can move a bead along the face, edit the bead feature to change such things as the bead direction and extent, or you can edit the bead profile to make changes to the sketch element used to create the bead.

Dynamically edit a bead in the ordered environment

  1. Choose Home tab→Select group→Select .

  2. Select the bead.

  3. Click the Dynamic Profile button.

  4. Drag the bead to dynamically change the bead and click to make the change.

  5. Click the Finish button to save the edit.

Edit a bead feature in the ordered environment

  1. Choose Home tab→Select group→Select .

  2. Select the bead.

  3. Click the Edit Definition button.

  4. Use the controls on the command bar to edit the bead definition.

  5. Click the Finish button to save the edit.

Edit a bead profile in the ordered environment

  1. Choose Home tab→Select group→Select .

  2. Select the bead.

  3. Click the Edit Profile button.

  4. Make changes to the existing bead profile.

  5. Click the green Accept button .

  6. Click the Finish button to save the edit.

Move a bead in the synchronous environment

  1. Choose Home tab→Select group→Select .

  2. Select the bead.

  3. Click the steering wheel axis shown.

    .

  4. Drag the bead to a new location.

  5. Click to save the edit.

Edit a bead feature in the synchronous environment

  1. Select the bead.

  2. Click the edit handle.

    The command bar, feature origin, bead direction arrow, and feature profile edit handle are displayed.

    You can click the arrow to change the bead direction, use the command bar to change bead options, or the edit handle to edit the feature profile.

  3. Click the direction arrow to change the bead direction.

  4. Right-click to save the edit.

Edit a bead profile in the synchronous environment

  1. Click the bead to display the edit handle.

  2. Click the edit handle.

    The command bar, bead direction arrow, and feature profile edit handle are displayed.

  3. Click the feature profile edit handle.

  4. Make changes to the existing bead profile.

  5. Click the green Accept button .

  6. Right-click to save the edit.
