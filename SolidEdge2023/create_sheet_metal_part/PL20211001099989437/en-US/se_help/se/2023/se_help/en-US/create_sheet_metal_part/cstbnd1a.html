Sheet Metal Bend Formulas

When you flatten a sheet metal part, Solid Edge uses the active settings for the Bend Equation on the Gage tab of the Solid Edge Material Table dialog box. You can use the standard formula delivered with Solid Edge, one of the example formulas delivered in the Solid Edge/Custom/Sheet Metal folder, or your own custom formulas.

Standard Bend Formula

The standard sheet metal bend formula delivered with Solid Edge is:

  • PZL = * (BR + (NF * THK)) * BA / 180

    Where:

    PZL = Plastic Zone Length

    BR = Bend Radius

    NF = Neutral Factor

    THK = Material Thickness

    BA = Bend Angle

Modifying Example Formulas

To modify one of the example formulas delivered with Solid Edge, you must use Visual Basic 4.0 or higher. The README file provided with each example contains important information about the program.

Developing Custom Formulas

To develop your own formulas, you must use a Windows object-oriented programming language that is capable of creating an OLE server DLL, such as Visual Basic or Visual C++. The comments contained in the example files can also be useful if you are developing your own programs.

Before you can use a custom formula you must register the program that contains it. For example, to register a program named BendMetal.dll, type the following command at an MS-DOS prompt:

regsvr32 BendMetal.dll

To unregister a program, type the following command at an MS-DOS prompt:

regsvr32 /u <Program Name>

After you have registered the program you want to use, you must supply the Program ID and Class Name to the Sheet Metal environment. Use the following syntax in the ProgramID.ClassName box on the Gage tab of the Solid Edge Material Table dialog box:

ProgramID.ClassName
