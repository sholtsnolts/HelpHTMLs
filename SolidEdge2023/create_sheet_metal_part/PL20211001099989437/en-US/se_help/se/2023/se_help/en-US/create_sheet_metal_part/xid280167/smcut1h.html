Construct a normal cutout in a sheet metal part

  1. Choose Home tab→Sheet Metal group→Normal Cutout.

    Note:

    You can also right-click a region and select the command from the ribbon or the context toolbar.

  2. Do one of the following:

    • If you want to draw a profile, select a planar face or reference plane.

    • If you want to select a profile from an existing sketch, on the command bar, on the Create-From Options list, click the Select From Sketch option. This option is not available if there are no sketches in the document.

  3. Draw or select an open or closed profile. The ends of an open profile are extended infinitely—an arc with open ends is extended to form a circle.

  4. Do one of the following:

    • If you drew or selected a closed profile, go to the next step.

    • If you drew or selected an open profile, click to define the side of the profile you want to remove material from.

  5. Define the extent of the material you want to remove.

    How To

  6. Finish the feature.
