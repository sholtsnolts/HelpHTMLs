Jog command

Use the Jog command to construct two bends to add a jog to a planar face of a sheet metal part. In the ordered environment, the profile for a jog feature must be a single linear element. In the synchronous environment, the sketch element used to construct the jog must be a single line that is coplanar with the face being bent. The jog can be minimal: for example, a slight offset or step to provide clearance or rigidity to a part.
