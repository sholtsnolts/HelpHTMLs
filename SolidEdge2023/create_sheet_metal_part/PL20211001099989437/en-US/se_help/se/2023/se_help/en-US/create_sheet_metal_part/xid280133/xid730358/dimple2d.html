Dimple Options dialog box

Taper Angle

Specifies the taper angle for the dimple or drawn cutout.

Rounding

Sets the dimple or drawn cutout punch and die rounding options.

Include Rounding

Specifies that the edges of the dimple are rounded.

Punch Radius

Specifies the radius value at the bottom of the dimple or drawn cutout.

Die Radius

Specifies the radius value for the top of the dimple or drawn cutout.

Include Punch-Side Corner Radius

Specifies that you want to apply a radius to any punch side corners in the profile.

Radius

Sets the radius.

Save Default

Saves the current settings as the default. Saved defaults will be used the next time you use this command. If you do not save the current settings as defaults, the previous default settings will be used.
