Move or edit a dimple

You can move a dimple along the face, edit the dimple feature to change such things as the dimple direction and extent, or you can edit the dimple profile to make changes to the sketch element used to create the dimple.

Move a dimple

  1. Choose Home tab→Select group→Select .

  2. Select the dimple.

  3. Click the steering wheel axis shown.

    .

  4. Drag the dimple to a new location.

  5. Click to save the edit.

Edit a dimple feature

  1. Click the dimple to display the edit handle.

  2. Click the edit handle.

    The dimple direction arrow, dynamic edit box, and feature profile edit handle are displayed.

  3. Type a value for the dimple extent.

  4. Click the handle to change the dimple direction.

  5. Right-click to save the edits.

Edit a dimple profile

  1. Click the dimple to display the edit handle.

  2. Click the edit handle.

    The dimple direction arrow, dynamic edit box, and feature profile edit handle are displayed.

  3. Click the feature profile edit handle.

  4. Make changes to the existing dimple profile.

  5. Click the green Accept button .

  6. Right-click to save the edit.
