Close the corner where two flanges meet

In the ordered environment, you can close the corners where two flanges meet or overlap the corners where the flanges meet.

In the synchronous environment, you can specify whether you want to close the corners where the flanges meet or overlap the corners where the flanges meet.

Close the corner where two flanges meet in the ordered environment

  1. Choose Home tab→Sheet Metal group→Corners list→Close 2-Bend Corner .

  2. Select the first bend.

    Note:

    The bend angle must be less than 90 degrees.

  3. Select the second bend.

  4. Set the treatment for the corner. You can set the treatment type to closed, open, circular cutout, u-shaped, v-shaped, square, or miter.

  5. Use the command bar boxes to define the characteristics such as gap, diameter, and offset of the feature.

    Note:

    The options that appear depend on the type of treatment you select. Enter appropriate values for the options that appear.

  6. Finish the feature.

Overlap the corners where two flanges meet in the ordered environment

  1. Choose Home tab→Sheet Metal group→Corners list→Close 2-Bend Corner .

  2. On the command bar, elect the Overlap button .

  3. Set the treatment for the corner. You can set the treatment type to closed, open, circular cutout, u-shaped, v-shaped, square, or miter.

  4. Select the first bend.

  5. Select the second bend.

  6. On the command bar, enter a gap value. The gap value cannot be negative or greater than the material thickness.

    Note:

    Depending on the treatment you select, you may have to add additional values for Diameter. Offset, and Angle.

  7. On the command bar, click Preview, and then click Finish.

Close the corners where two flanges meet in the synchronous environment

  1. Choose Home tab→Sheet Metal group→Corners list→Close 2-Bend Corner .

  2. Set the treatment for the corner. You can set the treatment type to closed, open, circular cutout, u-shaped, v-shaped, square, or miter.

  3. Select the first bend.

  4. Select the second bend.

  5. Type a gap value. The gap value cannot be negative or greater than the material thickness.

    Note:

    The options that appear depend on the type of treatment you select. Enter appropriate values for the options that appear.

  6. Press Enter to close the corner.

Overlap the corners where two flanges meet in the synchronous environment

  1. Choose Home tab→Sheet Metal group→Corners list→Close 2-Bend Corner .

  2. Select the Overlapping Corners button .

  3. Set the treatment for the corner. You can set the treatment type to closed, open, circular cutout, u-shaped, v-shaped, square, or miter.

  4. Select the first bend.

  5. Select the second bend.

  6. Type a gap value and overlap ratio. The gap value cannot be negative or greater than the material thickness.

    Note:

    If the Treatment is set to circular cutout, you need to type a diameter value also.

  7. Press Enter to overlap the corners.

Tip:

You can click the Flip button on the command bar to reverse the flange overlapping.
