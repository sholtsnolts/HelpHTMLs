Construct a sheet metal cutout from an open profile

  1. Choose Home tab→Sheet Metal group→Hole list→Cut .

  2. On the Cut QuickBar, set the Profile option to Open .

  3. Select the region(s) to define the cutout.

  4. (Optional) Click the directional arrow to change the feature direction.

  5. Type a value for the cutout extent or right-click for a dynamic preview of the cutout.

  6. Change the settings to update the preview or right-click to place the cutout.
