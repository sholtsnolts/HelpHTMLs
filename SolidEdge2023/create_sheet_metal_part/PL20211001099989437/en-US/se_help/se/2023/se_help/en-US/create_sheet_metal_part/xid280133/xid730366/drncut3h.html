Move or edit a drawn cutout

You can move a drawn cutout along the face, edit the drawn cutout feature to change such things as the cutout direction and extent, or you can edit the drawn cutout profile to make changes to the sketch element used to create the cutout.

Move a drawn cutout

  1. Select the drawn cutout.

  2. Click the steering wheel axis shown.

  3. Drag the cutout to a new location.

  4. Click to save the edit.

Edit a drawn cutout feature

  1. Select the drawn cutout.

  2. Click the edit handle.

    The cutout direction arrow, dynamic edit box, and feature profile edit handle are displayed.

  3. Type a value for the cutout extent.

  4. Click the handle to change the cutout direction.

  5. Right-click to save the edits.

Edit a drawn cutout profile

  1. Select the drawn cutout.

  2. Click the edit handle.

    The cutout direction arrow, dynamic edit box, and feature profile edit handle are displayed.

  3. Click the feature profile edit handle.

  4. Make changes to the existing cutout profile.

  5. Click the green Accept button .

  6. Right-click to save the edit.
