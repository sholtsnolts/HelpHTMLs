Louver Options dialog box (synchronous environment)

Saved Settings

Displays louver settings saved in the Saved Settings list. You can access the saved settings by selecting them from the list. The settings on the dialog box display the characteristics of the louver you select.

You can type a name in the box to name a group of settings.

You can then use the Saved Settings list on the Louver Options dialog box or the Louver command bar to select a saved setting later in any Solid Edge document where you can construct louver features. The saved settings are added to the Custom.xml file in the \Program Files\Siemens\Solid Edge 2023\Preferences folder. You can also use the File Locations tab on the Solid Edge Options dialog box to specify a different folder for the Custom.xml file.

Save

Saves the current settings to the name you type.

Delete

Deletes the saved settings selected in the Saved Settings box.

Formed-end Louver

Constructs the louver ends so that they are formed, or connected to the part.

Lanced-end Louver

Constructs the louver ends such that they are lanced, or separated from the part.

Length

Specifies the louver length.

Depth

Specifies the louver depth.

Height

Specifies the louver height.

Rounding

Sets the louver rounding options.

Include Rounding

Specifies that the edges of the louver will be rounded. This option is not recommended when using the louver to construct a pattern feature that contains a large number of louvers.

Die Radius

Sets the radius.

Feature Origin Offset:

Specifies an X and Y value to reposition the feature origin location.
