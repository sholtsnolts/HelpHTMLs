Apply a patch to relief in a flat pattern

When applying patch relief in a flat patten, you can:

  • Apply a patch to an individual relief

  • Apply a patch to a fence selected area

  • Apply a patch to all selected reliefs

  • Apply a patch to a drawn profile

Apply a patch to an individual relief

  1. Choose Tools tab→Flat group→Relief Patch

  2. Click on a bend edge within a relief area.

  3. Continue to click bend edges within a relief area.

  4. On the Relief Patch command bar, click Accept .

    Note:

    You can click the Draw Profile button to open the Sketch environment and edit the default sketch used to patch the relief.

  5. Click Finish.

Apply a patch to a fence selected area

  1. Choose Tools tab→Flat group→Relief Patch .

  2. To select a fenced relief area, drag a fence around the relief areas.

  3. On the Relief Patch command bar, click Accept .

    Note:

    You can click the Draw Profile button to open the Sketch environment and edit the default sketch used to patch the relief.

  4. Click Finish.

Apply a patch to all selected reliefs

  1. Choose Tools tab→Flat group→Relief Patch .

  2. To select all the eligible reliefs, press the A key

  3. On the Relief Patch command bar, click Accept .

    Note:

    You can click the Draw Profile button to open the Sketch environment and edit the default sketch used to patch the relief.

  4. Click Finish.

Apply a patch to a drawn profile

  1. Choose Tools tab→Flat group→Relief Patch .

  2. On the Relief Patch command bar, click the Draw Profile button .

  3. Use the commands on the Draw tab to create a profile around relief areas.

  4. Click the Close Sketch button .

  5. On the Relief Patch command bar, click Accept , and then click Finish.
