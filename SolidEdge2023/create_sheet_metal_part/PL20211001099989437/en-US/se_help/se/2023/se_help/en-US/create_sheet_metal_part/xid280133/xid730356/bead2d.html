Bead Options dialog box

Lists saved bead settings. You can access the saved settings by selecting them from the list. The settings on the dialog box display the characteristics of the bead you select.

You can type a name in the box to name a group of settings.

You can then use the Saved Settings list on the Bead Options dialog box or the Bead QuickBar to select a saved setting later in any Solid Edge document where you can construct bead features. The saved settings are added to the Custom.xml file in the C:\Program Files\Solid Edge 2023\Preferences folder. You can also use the File Locations tab on the Solid Edge Options dialog box to specify a different folder for the Custom.xml file.

Saves the current settings to the name you type.

Deletes the saved settings selected in the Saved Settings box.

Cross Section

Sets the bead cross section options.

Circular

Specifies that the bead cross section is circular.

U-Shaped

Specifies that the bead cross section is U-shaped.

V-Shaped

Specifies that the bead cross section is V-shaped.

Height

Specifies the height of the bead. The bead height is measured from the planar face closest to the top of the bead to the top of the bead.

Radius

Specifies the radius of the bead. This option is not available for u-shaped bead cross sections.

Width

Sets the width of the bead. This option is not available for circular and v-shaped bead cross sections.

Angle

Sets the angle of the bead. This option is not available for circular bead cross sections.

End Conditions

Sets the bead end condition options.

Formed

Constructs the ends of the bead such that they are connected to the part.

Lanced

Constructs the ends of the bead such that they are separated from the part.

Punched

Constructed the ends of the bead such that they are separated from the part. When you select the punched option, you can also specify the gap distance you want.

Rounding

Sets the bead rounding options.

Include Rounding

Specifies that the edges of the bead are rounded. This option is not recommended when using the bead to construct a pattern feature that contains a large number of beads.

Punch Radius

Specifies the radius at the bottom of the bead.

Die Radius

Specifies the radius value for the top of the bead.

Preview

Shows you the result of your settings before you apply them.

Save Default

Saves the current settings as the default. Saved defaults will be used the next time you use this command. If you do not save the current settings as defaults, the previous default settings will be used.
