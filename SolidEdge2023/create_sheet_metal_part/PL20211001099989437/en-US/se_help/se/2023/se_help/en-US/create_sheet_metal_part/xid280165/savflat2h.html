Save and flatten a sheet metal part

  1. On the File menu, click Save As→Save As Flat.

  2. In the graphics window, select the face to be oriented upward in the flat.

  3. Click an edge to define the X axis and origin.

  4. In the Save as Flat dialog box, select the output file type you want.

  5. Type the file name you want.

Note:

  • You can use this command to flatten a sheet metal part to a .par, .psm, or a .dxf file. The bend lines are automatically created in a .dxf file.

  • The flattened part is not associative to the folded part. If you want to maintain associativity to the folded part, use the Home tab→Clipboard group→Part Copy command on the Insert menu or the Tools tab→Model group→Flatten command.

  • The Save as Flat command does not automatically create bend lines. You can use the Part Copy command to insert a part copy into a sheet metal (.psm) file if you want to display the bend lines in a drawing view.

  • You can use the Save as Flat DXF Options dialog box to define such things as the line type, line color, and layer display in the dxf file.

  • You can use the Flat Pattern Treatments page in the Solid Edge Options dialog box to specify if want to use the folded model or an existing flat pattern to define the flat pattern.
