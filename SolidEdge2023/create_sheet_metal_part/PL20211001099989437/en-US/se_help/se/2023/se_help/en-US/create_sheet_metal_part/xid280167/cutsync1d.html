Cut command bar

Face Normal Cut Types

Switches between normal cut types.

Thickness Cut

Creates a cutout that compensates for the material thickness of the part.

Mid-Plane Cut

Creates a cutout based on the mid-plane of the part.

Nearest Face Cut

Creates a cutout based on the nearest face of the part.

Extents

Specifies the extent type for the cut.

Finite

Extends the cut a user-defined depth in the part.

Through All

Extends the cut through all the faces encountered on the part.

Through Next

Extends the cut through the next face.

From-To

Extends the cut by specifying a from face and to face.

Symmetric

Creates the cut symmetrically about the region.

Keypoints

Sets the type of keypoint you can select to define a feature extent or to position a new reference plane. You can then define the feature extent or the location of the reference plane using a keypoint on other existing geometry. The available keypoint options are specific to the command and workflow you use.

Selects any keypoint.

Selects an end point.

Selects a midpoint.

Selects the center point of a circle or arc.

Selects a tangency point on an analytic curved face such as a cylinder, sphere, torus, or cone.

Selects a silhouette point.

Selects an edit point on a curve.

Sets the no keypoint option.
Wrapped Cut

Specifies that the affected bends are flattened during profile edit.

Profile

Specifies profile type.

Open

Specifies an open profile that creates an area by excluding adjacent model edges.

Closed

Specifies a closed profile that creates an area by including adjacent model edges.

Selection Type

Specifies the sketch selection type.
