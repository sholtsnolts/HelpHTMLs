Create a blank body

  1. Select Tools tab→Model→Flatten.

  2. Select Tools tab→Flat group→Blank Body.

  3. In the Solid Edge Material Table dialog box, select the Material name you want from the list, and then click Apply To Model.

    Note:

    If a material is already applied to the model, the Solid Edge Material Table dialog box is not displayed.

  4. Click to select a tangent face chain.

    Note:

    You can press the S key to select individual faces.

  5. Click the handle to define the draw direction for the blank.

  6. On the Blank Body command bar, click Preview to display a preview of the blank along with a dialog box containing information about the surface area of the selected faces and the surface area of the computed blank.

  7. On the Blank Body command bar, click Finish to create the blank.
