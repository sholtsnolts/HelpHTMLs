Create Report dialog box

Important information

Report Title

Type the report title you want to use.

Author

Type the author name you want to use.

Company

Type the company name to use.

Date

Type the date you want to use.

Introduction

Type the introduction you want to use.

Conclusion

Type the conclusion you want to use.

Logo

Displays a preview of the logo image file you selected. You can use the Browse button to locate and select a logo image file.

Browse

Displays the Insert Logo dialog box, so you can locate and select a logo image file. The Insert Logo dialog box is similar to the Open dialog box.

Report location

Specifies a location where the report is produced. The default location is $Temp. You can use the Browse button to find or create an alternate folder location.

Document

Specifies the output format for the Simulation report. The available options depend on what programs are installed on your computer.

Web page (.html)

Creates an online report in .html format for display in your Internet browser.

Word 97–2003 (.doc)

Creates a Microsoft Word document in .doc format.

You can add information to the report and customize it per company requirements.

Word 2007/2010 (.docx)

Creates a Microsoft Word document in (.docx) format.

You can add information to the report and customize it per company requirements.

Adobe PDF (.pdf)

Creates a document in Adobe Portable Document Format (.pdf), which can be opened in the free Adobe Reader.

A PDF report cannot be edited, so it is presumed to be accurate.

Create Report

When you click the Create Report button, the report is displayed.

After the report is created, you also can use the Save As command to save the report to a location and file name you define.
