Tutorial: Simplify a model and apply a load using X, Y, Z components

To get the best results from Solid Edge Simulation, you can optimize the geometry in your part, sheet metal, and assembly models. There are several techniques for doing this, such as:

  • Using a simplified model

  • Removing geometry flaws and small entities

  • Offsetting edges around openings, holes, and fasteners

For information about these and other techniques, see Assembly best practices for simulation.

To speed meshing times in an assembly model, you can use the simplified representation of part and sheet metal geometry in a study. In this tutorial, you learn how to simplify a bicycle handlebar using the Loads command bar→Direction Type list, Component option .

  1. Open the file named FE_bar-XYZ.par. This model represents a bicycle handle bar.

    Simulation models are delivered in the \Program Files\Siemens\Solid Edge 2023\Training\Simulation folder.

  2. Simplify the model.

    Assume that forces are applied equally by each arm to the bar. By taking advantage of symmetry you speed up processing of the finite element model by cutting the bar in half.

    Note:

    This change affects the design part; for your own production models, it is recommended that you copy a model for this type of simplification.

    1. In PathFinder, turn on the Front (xz) base reference plane.

    2. Also in PathFinder, turn on the Base coordinate system.

    3. Sketch a circle by center point centered on the origin of the base coordinate system.

    4. Define a circle of diameter 75 mm.

    5. Select the region encompassed by the circle. On the Extrude command bar, from the Add/Cut list, choose Cut .

    6. Extrude the circle 375 mm to the right.

    7. In PathFinder, turn off the Base coordinate system and the Front (xz) reference plane.

  3. Create a static study.

    1. Select Simulation tab→Study group→New Study.

    2. Choose the Linear Static study type and Tetrahedral mesh type. Select OK.

  4. Add a force.

    1. Select Simulation tab→Structural Loads group→Force.

    2. On the Loads command bar, from the Direction Type list, select Components .

    3. Also on the command bar, from the Select Coordinate System list, select right hand CS .

    4. Select the inner and outer cylindrical faces of the bar. In the Fy component box, type and enter a force of 334 N (75 lbf). Set to zero the Fx and Fz components.

    5. Right-click to accept and click to finish.

  5. Define a fixed constraint on several faces.

    1. Select Simulation tab→Constraints group→Fixed.

    2. Select the inner and outer faces at the right end of the bar.

    3. Right-click to accept and click to finish.

  6. Select the Mesh command, and in the dialog box click the Mesh & Solve button.

    After a few moments, the part is meshed and solved.

    The results are displayed on the deformed model in the Simulation Results environment.

  7. Rotate the part to view the area of maximum stress.

  8. Close the file.
