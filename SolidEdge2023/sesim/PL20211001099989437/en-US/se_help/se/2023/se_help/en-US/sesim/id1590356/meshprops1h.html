Override mesh properties

To provide better control over the mesh that is generated or sized on a surface body or on united bodies, you can use the Override Property command to change the material properties assigned to individual faces or surfaces.

  1. In a part, sheet metal, or assembly model, choose the Simulation tab→Geometry group→Override Property command .

  2. Select one or more faces on a surface body where you want to change the material or material thickness.

    You can use QuickPick, the Select tool, or you can drag a box around the faces to create the selection set.

  3. On the Override Property command bar, do any of the following:

    • Choose a different material from the Material List.

    • Type a different thickness value in the Thickness box.

  4. Click Accept, press Enter, or right-click to apply the properties.

  5. (Optional) Select another face or another set of faces to apply another override property.

  6. Press Esc to end the command.

  7. Select the Mesh and Solve commands to update the study.

Tip:

An Override Property node is displayed in the Simulation pane when you create the first override property. A different Property node is displayed for each set of faces to which you apply properties.

You can modify override properties using the Edit Definition command on the shortcut menu of a selected Property node, or by double-clicking the Property node.
