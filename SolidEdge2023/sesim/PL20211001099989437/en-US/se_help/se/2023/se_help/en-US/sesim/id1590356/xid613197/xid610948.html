Control Parameters (Optimization dialog box)

Maximum number of iterations

Specifies the maximum number of iterations that the design optimization is allowed to run, regardless of whether the optimization has converged on a solution.

The valid range of values is –1 to 1,000,000,000.

Save results for all iterations

When this option is cleared, only the final iteration in the optimization is saved.

The final iteration is not necessarily the optimum iteration.

Note:

If you save the results for all iterations, it increases the results file size (.ssd) and increases the clutter in the Simulation pane.
