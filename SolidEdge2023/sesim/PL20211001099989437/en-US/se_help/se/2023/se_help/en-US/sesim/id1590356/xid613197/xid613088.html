Convergence status dialog box

A successful optimization is when a converged solution is achieved and a convergence status message is displayed in a dialog box.

If optimization processing does not complete because you cancel the optimization process or if an error message is displayed, all valid iterations that have been completed are saved.

You can use the Control Parameters page in the Optimization dialog box to increase or decrease the number of processing iterations that the simulation is allowed to run.

You can use the Convergence Parameters page in the Optimization dialog box to allow some flexibility in the convergence values found for optimization design variables.
