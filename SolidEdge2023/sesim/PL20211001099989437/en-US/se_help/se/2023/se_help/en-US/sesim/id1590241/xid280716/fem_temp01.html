Activity: Apply a body temperature load

  1. Open the file FE_Pump_Casing.par.

    Simulation models are delivered in the \Program Files\Siemens\Solid Edge 2023\Training\Simulation folder.

  2. Create a simulation study:

    1. Select Simulation tab→Study group→New Study.

      Note that the pump casing material is already defined.

    2. Choose the Linear Static study type and Tetrahedral mesh type.

    3. Click OK.

  3. Define a temperature load:

    1. Select Simulation tab→Body Loads group→Body Temperature .

    2. Type a temperature value of 148 C.

      This is equivalent to 300 degrees Fahrenheit.

    3. Right-click in space to accept the temperature load. Click in space to finish.

      4. You can see the Simulation pane entry for this load.

  4. Apply fixed constraints:

    1. Select Simulation tab→Constraints group→Fixed.

    2. On the Constraints command bar, change the Selection Type to Feature.

    3. Within PathFinder, in the Features collector, click Protrusion 4, Protrusion 5 and Protrusion 6 to select all faces of the mounting plate.

    4. Right-click to accept and click to finish.

  5. Mesh and solve the study:

    1. In the Simulation pane, turn off constraints and loads.

    2. Select the Mesh command. Choose Mesh & Solve.

      After a few moments, the part is meshed and solved.

  6. View the results:

    1. In the Simulation Results environment, select Home tab→Deformation group→Actual.

    2. Rotate and zoom to view the areas of highest stress.

  7. On the ribbon, click Close Simulation Results.

  8. On the Quick Access toolbar click Save, and then click the X button on the document tab to close this file.
