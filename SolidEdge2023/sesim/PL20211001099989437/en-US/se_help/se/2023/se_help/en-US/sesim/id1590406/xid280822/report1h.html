Report simulation results

You can generate a report of analysis results in the Simulation Results environment. You can choose to produce the report in several different formats.

Generate an analysis report

  1. In the Simulation Results environment, choose the Home tab→Output Results group→Create Report command.

    Tip:

    You also can right-click the study node in the Simulation tree-view, and then choose the Create Report command.

  2. In the Create Report dialog box, edit the information as you want it to appear in the report.

    In addition to the pre-filled information, you can type an introduction and a conclusion. These appear on separate pages in the report.

  3. (Optional) To insert a logo at the top of the title page of the report, click the Browse button and then select an image file.

  4. (Choose report format) In the Create Report dialog box, under Document format, choose a report output format. The options that are available depend upon the programs installed on your computer.

  5. (Specify output location) In the Report location text box, type a report folder location, or use the Browse button to find a folder.

  6. In the dialog box, click the Create Report button.

    The simulation results report is created in the location you specified. If you selected the .html report format, the report is displayed automatically in your Internet browser window.

    Tip:

    If a popup is displayed stating that sections of the study may be out of date, you may want to update the study before producing the report.

    To learn more, see Reviewing the status of existing studies, and Using the Simulation tree-view pane.
