Saving Solid Edge Simulation studies to Femap

You can export a Solid Edge Simulation analysis study for use in the Femap finite element analysis program using the Save Femap Model File command on the selected study shortcut menu in the Simulation pane.

You may want to make Solid Edge Simulation documents available to Femap when:

  • You have a model that requires more advanced finite element analysis capabilities than Solid Edge Simulation can provide.

  • You want to share data with an analyst.

The Save Femap Model File command saves an existing Solid Edge Simulation analysis study—including geometry, material, loads, constraints, and mesh controls—to the Femap (.modfem) format. If the study contains a mesh and analysis results, they also are saved to the .modfem file.

No NX Nastran files are saved.

For more information, see the help topics, Save a study to Femap and Setting units for files opened in Femap.

To learn about the differences in units in Solid Edge Simulation and in Femap, try the practice activities, Using Solid Edge Simulation files in Femap.

Special cases of Femap file output

Sometimes cylindrical geometry is converted differently for use in Femap than in Solid Edge Simulation. For example:

  • When meshing splits cylinders into two surfaces, nodes or elements align along the split.

  • When a load is placed on a cylindrical face that must be split in Femap, the Save Femap Model File command generates split geometry as well as extra loads, constraints, and connectors.

Saving result plots as images

You can save individual result plots as images in the Simulation Results environment.

After analysis processing, and while the Probe command is active, you can preserve the image and temporary annotations for documentation and review using the Home tab→Output Results group→Save As Image command. Individual result plots can be saved in any of these formats: .jt, .bmp, .jpg, .tif or .wrl.
