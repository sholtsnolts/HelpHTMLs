Tutorial: Simulate stresses on a valve handle

In this tutorial, you learn how you can use a cylindrical constraint and a force load to simulate linear static stresses on a handle.

  1. Open FE_valve_handle.asm.

    Simulation models are delivered in the \Program Files\Siemens\Solid Edge 2023\Training\Simulation folder.

  2. Create an analysis study.

    1. Select Simulation tab→Study group→New Study.

    2. Choose the Linear Static study type and Tetrahedral mesh type, and then click OK.

      The Define command in the Geometry group is active.

      You are prompted to select the occurrences (parts) that you want to include in the study.

    3. Select the valve handle and valve shaft, and then click Accept.

      Tip:

      To make a part available for selection, you can click it in the graphics window and it will activate automatically. You also can use the Home tab→Select group→Activate command on the ribbon.

      The new study is listed in the Simulation pane.

  3. Define a force on the handle.

    1. Select Simulation tab→Structural Loads group→Force.

    2. Select the side face of the handle.

    3. In the Value box, type 44482.216 mN. This is the equivalent of a 10 pound force, converted to milliNewtons.

    4. On the Loads command bar, click Flip Direction to reverse the direction of the force.

    5. Right-click in space to accept; click to finish.

  4. Add a cylindrical constraint to the shaft.

    1. Select Simulation tab→Constraints group→Cylindrical .

    2. Select the cylinder of valve_shaft. Right-click in space to accept it and click to finish.

      3. You can see the finished constraint on the model.

  5. Define an assembly connector between the shaft and handle.

    1. Select Simulation tab→Connectors group→Auto.

    2. On the Auto command bar, do both of the following:

      1. Select Glue from the Connector Type list.

      2. Verify that One connector per component pair is selected.

    3. Select the shaft and handle. You can do this by selecting them in the graphics window or by selecting the Select All button on the command bar.

    4. Click Accept on the command bar.

      The Results dialog box lists the connector.

    5. Click the Create Connectors button in the Results dialog box to apply it. Click in space to finish.

      You can see the connector in the assembly model as well as in the Simulation pane.

  6. Mesh and solve the assembly.

    1. Select the Mesh command, and in the Mesh dialog box, use the slider to define a subjective mesh size of 3.

    2. Click the Mesh button.

      The entire assembly is meshed.

    3. Select the Simulation tab→Solve group→Solve command.

      After a few moments, the results are displayed in the Simulation Results environment.

      Note:

      Optionally, you can select the Mesh dialog box option, Mesh and Solve, to do both processes in one step. Be aware that the processing time increases.

  7. View the area of maximum stress.

    1. Turn off the loads, constraints, and connectors in the Simulation pane.

    2. Rotate the view and zoom in so you can see the highest stress on the underside of the handle-shaft interface.

    3. The Von Mises stress results are displayed on the deformed model. Change the display to show both the deformed and the undeformed model. Choose Home tab→Show group→Display Options , and then select the Undeformed Model check box.

    4. From the Color Bar tab→Show group, select the Max Marker check box.

  8. Close this file.
