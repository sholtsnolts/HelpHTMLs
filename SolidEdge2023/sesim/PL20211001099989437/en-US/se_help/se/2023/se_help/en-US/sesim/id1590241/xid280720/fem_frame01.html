Tutorial: Simulate stresses on a bridge

In this tutorial, you explore how a load applied to a bridge will affect its structure.

  1. Open FE_Bridge.asm.

    Simulation models are delivered in the \Program Files\Siemens\Solid Edge 2023\Training\Simulation folder.

  2. Open the Frame environment: Select Tools tab→Environs group→Frame .

  3. Create a structural frameworks analysis study.

    1. Select the Simulation tab→Study group→New Study command.

    2. Choose the Linear Static study type and Beam mesh type, and then click the Options button.

    3. Under Results Options, in the Nodal section, select Beam end reactions.

    4. Click OK to close the Create Study dialog box.

  4. The Define command in the Geometry group is active. Select the bridge frame.

  5. Accept the frame.

    The blue curves represent beam curves.

  6. Define a force on the top beams.

    1. Select the Simulation tab→Structural Loads group→Force command.

    2. On the command bar, change the selection filter to Curve.

    3. Select the four beams on the top of the frame.

    4. The default force direction vector points up. Select the Flip Direction button to change the force direction so that it points down onto the frame.

    5. Type 100 kN in the value box. This is equal to a 22481 pound force load.

    6. Right-click to accept and click to finish.

      The force is placed.

  7. Add a fixed constraint to the base of the frame.

    1. Select the Simulation tab→Constraints group→Fixed command.

    2. On the command bar, change the selection filter to Node.

    3. Select the beams shown below, near the end points closest to joints. When you are finished, you should see constraint nodes at the six beam joints along the base of the frame.

      When using the Nodes filter, click along a beam and the closest end is selected.

    4. Right-click to accept, and then click to finish.

      You can use the Symbol Size/Spacing button on the command bar to increase the constraint symbol size so they are easier to see.

    5. In the Simulation pane, turn off loads and constraints.

  8. Mesh the frame.

    1. Select the Mesh command, and on the Mesh command bar, use the slider to define a subjective mesh size of 3 or 4.

    2. Click the Mesh button.

      The entire assembly is meshed.

  9. Solve the study—Select the Simulation tab→Solve group→Solve command.

    After a few moments, the results are displayed in the Simulation Results environment.

  10. Display beam cross section properties.

    1. On the Simulation pane, expand the Geometry node.

    2. Select a frame component for which you want to display the beam cross section.

      Individual beams are highlighted in the graphics window as you hover over frame components in the Simulation pane.

    3. Right-click and select the View Beam Properties command.

      4. For information about this panel, see Beam properties.

  11. Display beam end reaction plots.

    1. On the Home tab→Data Selection group, select:

      Result Type=Beam End Reactions

      Result Component=Beam Plane1 Moment.

      • The color bar header is turned off in this image.

      • The units for moment values in the image are newton meters.

    2. Change the Result Component list selection to Beam Pl1 Shear Force.

      The units for shear force values in the image are millinewtons.

  12. Display beam diagram plots for beam end reactions and stress.

    1. Select the Home tab→Show group→Display Options command.

    2. On the Display Options list, clear the Beam Cross Section check box, and select the Beam Diagram check box.

      The beam diagram plot is displayed for Result Component=Beam Pl1 Shear Force.

    3. On the Home tab→Data Selection group, change the following:

      Result Type=Beam End Reactions

      Result Component=Beam Plane1 Moment

      The beam diagram plot for Result Component=Beam Plane1 Moment shows the bending moments.

    4. On the Home tab→Data Selection group, change the following:

      Result Type=Stress

      Result Component=Beam Factor of Safety

      Note:

      To learn more about beam plots:

      • Beam properties

      • Beam reaction component plots

      • Beam stress component plots

      • Beam diagrams

  13. Close this file.
