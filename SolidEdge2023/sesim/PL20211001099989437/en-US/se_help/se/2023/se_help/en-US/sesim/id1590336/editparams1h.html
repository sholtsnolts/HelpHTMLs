Edit the study processing parameters

  1. In the Simulation pane, right-click the study name and choose the Modify Study command.

  2. In the Modify Study dialog box, click Options to expand the dialog box.

  3. (To modify processing options) Under Advanced Options, the most frequently used processing settings are shown. You can change these, or you can type command-line-only keywords in the box provided.

    Example:

    Some NX Nastran solver errors, like this one, can be corrected in Advanced Options: Solve failed due to Mesh size and/or geometry complexity. Change the Mesh size or modify the Study by setting the "Geometry check" option to "Warning only" and then solve again.

    Advanced users can click the NX Nastran Options button to review the default options and make changes using the NX Nastran keyword editor.

    Tip:

    Refer to these help topics:

    • NX Nastran processing errors, for some typical errors and suggestions for resolving them.

    • NX Nastran processing options, for an overview.

  4. In the Modify Study dialog box, click OK to update the study definition.

  5. To regenerate the study with the new settings, do one of the following:

    • If it is available, choose the Solve command from the ribbon or the study shortcut menu.

    • If the study status indicator shows that the mesh is out of date , choose the Mesh command and select Mesh & Solve from the command bar.

      To learn more, Reviewing the study status.
