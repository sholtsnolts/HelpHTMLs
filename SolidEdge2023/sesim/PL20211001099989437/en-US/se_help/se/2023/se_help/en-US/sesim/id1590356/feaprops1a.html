Simulation materials and properties

To get the best results from Solid Edge Simulation, you can review and modify properties.

  • Change the study material or thickness at the object level.

    To learn more, see Using materials in studies

  • Override mesh properties on faces or surfaces using the Override Properties command.

    To learn more, see Mesh material properties.

  • Review stress recovery locations in a beam cross-section.

    See Beam properties.
