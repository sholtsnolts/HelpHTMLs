Set units for .modfem files opened in Femap

You can use this procedure to set consistent units in Femap after you Save a study to Femap.

Note:

This procedure requires that you have a separate installation and license for Femap.

  1. In Femap, choose File→Preferences.

  2. On the Geometry/Model tab, from the Solid Geometry Scale Factor list, set the scale factor to 1.0 (Meters).

    Note:

    The available scale factors are Inches, Meters, and Millimeters. When the Solid Edge model file (in meters) is opened in Femap, Femap applies the appropriate factors to display the analysis data in SI units that are consistent with what you select here.

  3. Close Femap, and then restart Femap.

  4. Choose File→Open to open the .modfem file you saved in Solid Edge Simulation.

    At this point, all of the Femap units are set to the same geometry units (meters) as the Solid Edge .modfem file, and the analysis data is set to the SI standard, which is also based on meters.

  5. (Optional) Select Tools tab→Convert Units to change the .modfem file to whatever unit system you want to use in Femap.

    You can use the dialog box to:

    • Convert all quantities by converting basic units. For example, you can use millimeters instead of meters.

    • Convert specific entities with individual conversion factors. For example, you can use megapascals instead of pascals.

Tip:

In Femap, you can use the File→Notes command to document the current unit system for future reference. This also adds a comment in the Femap analysis file.
