Solid Edge Simulation licensing

Solid Edge Simulation is installed automatically with Solid Edge. The availability of Solid Edge Simulation functionality is based on license levels.

License-based functionality in Solid Edge Simulation

When creating and modifying a study, the study types, study options, and availability of other simulation commands are based on the following license types, where (1) is all-inclusive and (3) is the least inclusive.

  1. Solid Edge Simulation—Advanced (SEADVSIMULATION)

    This comprehensive license includes all study types and study optimization.

  2. Solid Edge Simulation—Standard (SESTDSIMULATION)

    This license excludes the following study types:

    • Steady State Heat Transfer

    • Steady State Heat Transfer + Linear Static

    • Steady State Heat Transfer + Linear Buckling

    • Transient Heat Transfer

    • Dynamic Frequency Response

  3. Solid Edge Premium (and lower)

    This license also excludes the following types of studies, as well as study optimization:

    • Linear Buckling

    • Normal Modes—Excludes modal frequencies beyond the four basic modes (Mode 1, Mode 2, Mode 3, Mode 4).

To obtain a Solid Edge Simulation license, contact your Solid Edge reseller. You also can access Siemens Support Center at https://support.sw.siemens.com/en-US/.

Availability in common product configurations

Solid Edge Simulation functionality is also available in the following product configurations (licenses):

  • As an add-on module to Solid Edge Classic and Solid Edge Foundation.

  • Packaged with Solid Edge Classic, Wiring Harness, and XpresRoute (Solid Edge Premium)

  • Bundled with Teamcenter Rapid Start [Solid Edge Premium (Teamcenter Rapid Start Edition)].

To see what type of license you have:

  1. Click the File tab.

  2. At the bottom of the File menu, click About Solid Edge to open the About Solid Edge dialog box.

Activating Solid Edge Simulation in Solid Edge

Once you have a license, you may need to activate the license for use. This depends upon whether the license is floating or node locked.

  • If you have a premium or node-locked Solid Edge Simulation license, then all of the Solid Edge Simulation functionality associated with that license level is available by default when you open a part, sheet metal, or assembly model.

  • If you have a floating Solid Edge Simulation license, then you need to check out a floating Solid Edge Simulation license for use on your computer. (You do not do this if you have a floating Solid Edge Premium license.)

Note:

For more information, see the Solid Edge Installation and Licensing guide. The guide is also available for download from the Support Center website, https://support.sw.siemens.com.
