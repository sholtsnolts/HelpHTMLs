Tutorial: Select additional Simulation Results options

In this activity you will select and review several additional options in the Simulation Results environment. For more information about the plots shown, see the following help topics:

  • Plotting results

  • Contour styles

  • Primary, deformed, and undeformed display

  1. Open FE_study_results.par.

    Simulation models are delivered in the \Program Files\Siemens\Solid Edge 2023\Training\Simulation folder.

  2. Solve the existing study.

    1. Choose the Simulation tab→Solve group→Solve command.

      The deformed Von Mises stress results are displayed.

    2. Select Home tab→Close group→Close Simulation Results.

  3. Modify the existing study to obtain nodal results options.

    1. Right-click Static Study 1 and from the menu, select Modify Study.

      The Modify Study dialog box is displayed.

    2. Click the Options button to expand the Modify Study dialog box.

    3. In the Results Options portion of the dialog box, select all of the available Nodal options; click OK.

    4. Reprocess the study.

      Choose the Simulation tab→Solve group→Solve command to reprocess the study.

    5. Observe displacement results.

      1. In the Home tab→Data Selection group, select Displacement from the Result Type list.

        Total Translation results are displayed; this is the default displacement result plot.

      2. Use the Result Component list to change the results to T1 Translation.

      3. Experiment with the other results by selecting different results from the Result Component list.

    6. Observe constraint force results.

      1. Turn off the display of loads and constraints.

      2. From the Home tab→Data Selection group→Result Type list, select Constraint Force.

      3. Zoom the area around the socket to view the area of maximum force due to the constraint.

      4. Experiment with viewing the other results by selecting different Result Components from the list.

        Note:

        This model does not manifest any moment results.

      5. Select Home tab→Close group→Close Simulation Results.

  4. Modify the existing study to obtain elemental results options.

    1. As in Step 4, right click Static Study 1 and from the menu, select Modify Study.

    2. In the Results Options portion of the dialog box, clear all of the options under Nodal. Select all of the Elemental options, and then click OK.

    3. Solve the study.

    4. In the Simulation Results environment, turn on the display of loads and constraints.

    5. Observe stress results.

      Von Mises Stress results are displayed by default.

      Note:

      If they are not, from the Home tab→Data Selection group, select Stress from the Result Type list.

    6. Use the Result Component list to change the results to Intermediate Principal Stress.

    7. Zoom the area around the socket to view the areas of stress.

    8. Experiment with viewing the other results by selecting different Result Components from the list.

  5. Observe strain results.

    1. Turn off display of loads and constraints.

    2. In the Home tab→Data Selection group, select Strain from the Result Types.

    3. Display the Von Mises Strain result component.

      Notice that the result values are different than for the Von Mises Stress results.

    4. Experiment with viewing the other results by selecting them from the Result Component list.

  6. Close this file.
