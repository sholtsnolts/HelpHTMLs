Simulation Results Display Settings dialog box

Saves your analysis results display settings for reuse within the current document.

Setting Name

In the upper box, type a name that you want to assign to the current analysis results display settings.

In the lower box, select from the list of saved analysis results display settings.

Note:

The currently selected and active display setting name appears in the Home tab→Settings group on the command ribbon.

Apply

Applies the display settings associated with the name selected in the lower box to the document.

Save

Saves the current display settings to the name you type in the upper box. Settings are saved within the current document, not to an external file.

Note:

Use the Save Results Display Globally command to save the display results locally into the user’s registry settings. These will be read in upon solve and can be used in other studies in other documents.

Delete

Deletes the saved setting(s) selected in the lower box.

How do I
Display the Simulation Results environment
Save changes to the simulation results display settings
Apply results display settings to the simulation results
