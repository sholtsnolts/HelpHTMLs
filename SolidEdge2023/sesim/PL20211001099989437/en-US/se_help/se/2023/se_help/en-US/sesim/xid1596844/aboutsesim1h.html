Check out a floating Solid Edge Simulation license

You only need to do this once per session, and only if you use a floating Solid Edge Simulation license.

  1. Open a Solid Edge model document.

  2. On the ribbon, choose the Simulation tab→License group→Get License command .

    Note:

    If this command is not available, then you already have a floating license checked out to you.

    The commands on the Simulation tab are now available for you to use them.

Tip:

The floating license remains checked out for all of your documents until you select the Simulation tab→License group→Release License command , or until you close Solid Edge.
