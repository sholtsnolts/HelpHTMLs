Display a beam cross section

  1. In the Frame environment, on the Simulation pane, expand the Geometry node.

  2. Right-click a frame component for which you want to display the beam cross section and select the View Beam Properties command.

  3. Use the Beam Properties dialog box to review beam properties for a beam cross section to understand the results of a frame model analysis. For more information, see Beam properties.

Tip:

Solid Edge Simulation displays the stress recovery points that are calculated by Femap. In Femap, you can modify the stress recovery and reference points on the cross section:

  • In the Stress Recovery section of the Cross Section Definition dialog box.

  • In the Stress Recovery section of the Define Property dialog box.
