Design Objective (Optimization dialog box)

The Design Objective page in the Optimization dialog box assigns a single parameter as the objective you want to achieve. Optimization can be used to solve for a specific value. You also can choose a general quality that you want to optimize for, such as mass.

Define Objective

Opens the Design Objective dialog box for you to select a design objective for the optimization.

Only variables that are valid for selection based on model type and study type are listed in the dialog box. The variables are categorized as follows:

  • Physical Properties, such as mass, volume, or surface area.

  • Natural Frequency (for normal modes studies)

  • Buckling Load Factor Eigen Values (for linear buckling studies)

  • Simulation Study Results, such as stress, safety factor, strain, or temperature

    Note:

    For study results to be available for selection, the study must be solved. If a particular study result type is displayed but is not selectable, then you can modify the study and update the selected result types under Results Options in the Create Study (or Modify Study) dialog box.

For more information, see Assigning design optimization variables.

Maximum

When the selected objective is a Simulation Study Result component (for example, Total Translation or Von Mises Stress), specifies that the optimization process uses the maximum value from the results to define the design objective.

Minimum

When the selected objective is a Simulation Study Result component (for example, Total Translation or Von Mises Stress), specifies that the optimization process will pick the minimum value from the results to define the design objective.

Value

Displays the current value of the selected objective.

If Not Applicable is displayed in the Value box, this means the result is unprocessed, the study is not solved, or the geometry is not included in the study.

Name

Displays the name of the selected design objective.

Current Value

Shows the current value of the design objective variable.

Objective Type

Selects the range of values for the objective. The options are Minimize, Maximize, and Target.

Target Value

When the Objective Type is set to Target, specifies the desired value for the design objective variable. Any integer or fractional number is allowed.
