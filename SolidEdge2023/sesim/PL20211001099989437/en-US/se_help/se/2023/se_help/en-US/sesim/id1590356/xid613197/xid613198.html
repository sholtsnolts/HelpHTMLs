Design Limit/Design Variable Rule Editor

Defines the rules for a design limit or a design variable selected in the Design Optimization dialog box.

Design limit (or Design variable)

Displays the name of the variable for which you are defining rules.

Current value

Lists the current value of the variable.

Minimum limit/Maximum limit

Specifies the limits you want to define for the variable by defining a range of values that are valid for the variable. You can use the Minimum limit, the Maximum limit, or both.

You can specify that the variable minimum and maximum values are greater than or equal to a value, less than or equal to a value, or that there is no maximum or minimum limit. The following examples illustrate possible uses for a min/max limit:

  • Example 1: Greater than or equal to 15 mm AND less than or equal to 30 mm. Valid values are within a defined range.

  • Example 2: Greater than or equal to 30 mm AND less than or equal to 15 mm. Valid values are outside of a defined range.

  • Example 3: None AND less than or equal to 30 mm. Valid values are within a defined range.

    Minimum limit

    Specifies the lower value for the design limit as one of the following:

    • None (no lower limit)

    • Greater than or equal to (the specified value)

    Value

    Specifies the value that the Minimum limit is greater than or equal to.

    Maximum limit

    Specifies the upper value for the design limit as one of the following:

    • None (no upper limit)

    • Less than or equal to (the specified value)

    Value

    Specifies the value that the Maximum limit is less than or equal to.
