Tutorial: Simplify a sheet metal assembly

In this activity, you use the Mid-Surface, Extend, and Unite Body commands on the Simulation Geometry tab to simplify an assembly for simulation.

For information about these and other techniques, see Assembly best practices for simulation.

  1. Open the file Unite_body.asm.

    Simulation models are delivered in the \Program Files\Siemens\Solid Edge 2023\Training\Simulation folder.

  2. Choose File menu→Save As→Save As command, and give this file a unique name. You will use the saved copy of the file to create a second study later in this activity.

  3. Create mid-surfaces.

    1. Select Simulation Geometry tab→Surfaces group→Mid-Surface.

    2. Select the vertical tab.

    3. On the Mid-Surface command bar, click Offset From Side 1, and ensure Offset Ratio is set to 0.500.

      You can see that the part is automatically selected, and the mid-surface is displayed in green.

    4. Click the Preview button, and then click Finish.

    5. Repeat for the other two tabs.

      6. Note:

      Several Mid Surface nodes are displayed on the Simulation pane but not on the Assembly PathFinder.

  4. Expose the mid-surfaces.

    1. In the empty space around the view, right-click and select Show/Hide All Components.

    2. In the Show All/Hide All dialog box, select the Show All check box to the right of Surfaces, and select the Hide All check box to the right of Design body.

      3. This exposes just the mid-surfaces.

  5. Extend these surfaces.

    1. Select View tab→Views group→Left View to rotate the view.

      Notice the gaps between the mid-surfaces of each tab. You need to extend each mid-surface.

    2. Press Ctrl+I to rotate to an isometric view.

    3. Choose Simulation Geometry tab→Surfaces group→Extend .

    4. Extend the first edge—

      1. Select and accept the edge shown.

      2. On the command bar, click Extend to, and then select the vertical face.

      3. Click Finish to complete Extend Surface_1.

    5. Repeat the previous steps to extend the mid-surfaces. Extend the two edges of the triangular plate so they intersect the surfaces.

    6. Click Cancel to end the Extend command.

    7. To check your work, rotate the model so you can see all the edges intersect the surfaces and leave no gaps.

      8. Note:

      Several Extend nodes are displayed on the Simulation pane but not on Assembly PathFinder.

      Tip:

      If you extend a surface using the Finite Extent option, you can specify the distance to extend it, and then use the Simulation Geometry tab→Surfaces group→Trim command to trim away the excess so the surfaces just intersect.

  6. Create a united body.

    1. Choose Simulation Geometry tab→Surfaces group→Unite Bodies.

    2. Select all three faces.

    3. Right-click to accept.

      4. Note:

      A Unite Bodies node is displayed in the Simulation pane, but not on the Assembly PathFinder.

  7. Create a new study.

    1. Choose Simulation tab→Study group→New Study.

    2. Choose the Linear Static study type and Mixed and General Bodies mesh type. Select OK.

    3. The Define command is active. Select and accept the united body as the geometry to simulate.

  8. Mesh the model.

    Choose Simulation tab→Mesh group→Mesh. Using the default mesh size, select the Mesh button.

    Note:

    Notice the consistent, even distribution of nodes and elements between surfaces. Specifically, note how nodes on each element precisely match up.

  9. Save and close this file.

  10. Create a second study and mesh the model using the separate mid-surfaces as the geometry.

    1. Open the unmodified copy of the original file Unite_body.asm.

    2. As before, select the Save As command and give this file a unique name.

    3. Prepare the mid-surfaces.

      1. Repeat the previous steps to create, expose, and extend the mid-surfaces.

      2. Create a new study.

        1. Choose Simulation tab→Study group→New Study.

        2. Choose the Linear Static study type and Surface mesh type. Select OK.

        3. Select all three surfaces as the geometry to simulate.

      3. Select Simulation tab→Mesh group→Mesh. Using the default mesh size, select the Mesh button.

        Note:

        Notice the independent element distribution between surfaces. There are inconsistent shapes and node mismatches.

      4. Note:

      Solving a study using the collection of individual surfaces requires edge connectors between the surfaces; this impacts processing time. Also, a less accurate solution can result due to inconsistent element distribution and node mismatches. For sheet metal parts, the united body method may provide a more accurate set of results in less time.

  11. Close this file.
