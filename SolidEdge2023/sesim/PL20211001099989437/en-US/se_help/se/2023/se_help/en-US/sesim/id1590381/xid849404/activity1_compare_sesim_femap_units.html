Activity 1: Compare Solid Edge Simulation file units with Femap units

In this activity you will learn how units are written to the Femap model file from Solid Edge Simulation.

Note:

This activity requires that you have a separate installation and license for Femap.

Create a simulation study In Solid Edge

  1. Create a new part.

    1. On the File menu, click New to create a new part file using an iso metric part.par template.

      Note:

      The unit system for this template is measured in millimeters, kilograms, and seconds. You can verify this by opening the Units tab in the Solid Edge Options dialog box.

    2. Create a 10x10x100 mm protrusion, with 10x10 mm on the Z-Y plane and 100 mm along the X axis.

    3. Choose File menu→Info→Material Table.

    4. Select Aluminum 1060 for the material and click Apply to Model.

      Tip:

      You also can select and apply a material using the Material List or the Material Table button on the Simulation tab→Study group.

    5. Choose Simulation tab→Study group→New Study .

  2. Create a new linear static study.

    1. In the Create Study dialog box, select the following:

      • Study type = Linear Static

      • Mesh type = Tetrahedral

    2. Choose Simulation tab→Geometry group→Define , and then accept the selected part.

  3. Define loads and constraints:

    1. Choose Simulation tab→Structural Loads group→Force .

    2. Select the 10x10 mm face on the right, and apply a force of 1000 N along the X-axis positive direction. Right-click to accept.

      Note:

      The value converts to millinewtons when it is applied. This is consistent with the Force units in the metric template, iso metric part.par.

    3. Choose Simulation tab→Constraints group→Fixed .

    4. Select the 10x10 mm face on the left end of the part, and right-click to accept it.

  4. Mesh and solve the study:

    1. Choose Simulation tab→Mesh group→Mesh .

    2. In the Tetrahedral Mesh dialog box, click Mesh & Solve.

    3. When processing is finished, the model is displayed in the Simulation Results environment. Observe that the maximum Von Mises Stress is 10.4 MegaPa.

      Note:

      This matches the Stress units on the Units tab in the Solid Edge Options dialog box for the metric template, iso metric part.par.

    4. Choose Home tab→Close group→Close Simulation Results.

  5. Generate an output file to open in Femap:

    1. In the Simulation pane, right-click Static Study 1, and choose Save Femap Model File .

    2. In the Save Femap Model File dialog box, click Save.

    3. Close Solid Edge.

Set and review units in Femap

  1. Start the Femap program from the desktop or from the Start menu.

    Note:

    Do not use the Femap command on the Tools tab→Environs group in Solid Edge to run Femap.

  2. Choose File→Preferences.

  3. In the Preferences dialog box, click the Geometry/Model tab.

  4. From the Solid Geometry Scale Factor list, select 1. Meters.

    This specifies the internal storage units for your geometry in Parasolid.

  5. Close Femap.

  6. Open the analysis file in Femap:

    1. Restart Femap.

    2. Choose File→Open, and then select and open the .modfem file that you previously created in Solid Edge Simulation.

    3. Click OK to dismiss the Neutral File Read Options dialog box.

    4. In the Model Info pane, review the material properties.

      • Young's Modulus is 6.8948E+10. This is consistent with pascals.

      • The density is 2712, consistent with kg/m^3.

    5. Measure the distance of one of the long edges of the model. Observe that it is 0.1.

      Note:

      This is because the Femap model size is always written using SI base units (meters), when the Femap model file is created in Solid Edge Simulation. 

    6. Choose Tools→Check→Sum Forces.

      Observe that the total force (the sum of the applied loads) is 1000 newtons.

    7. Adjust the units as needed. If the units do not meet your requirements, then select the Tools→Convert Units command to convert the quantities of basic units or apply conversion factors to specific entities.

    8. Close Femap without saving the file.
