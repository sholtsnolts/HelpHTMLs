Selecting convergent bodies for studies


You can create a study for a part or sheet metal model with convergent bodies. The process of selecting a convergent body to use in a study differs for part and sheet metal models.

Selecting convergent bodies in a part model

You can select a model for the study using the Simulation tab → Geometry group → Define command . Select the part in the graphics window or in PathFinder, and then choose Accept on the Define command bar.

Selecting convergent bodies in a sheet metal model

When working with sheet metal models (.psm), you must select a mid-surface feature. For more information, see Selecting surface geometry for studies.

For more information about creating and solving a study using convergent bodies, see Create and solve a study using a convergent model.
