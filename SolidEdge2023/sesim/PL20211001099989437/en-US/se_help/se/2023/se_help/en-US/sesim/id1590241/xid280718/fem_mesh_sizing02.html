Activity: Size the mesh on an edge

For information about meshes, mesh quality, and how you can refine the mesh to improve results, see the following help topics:

  • Meshes

  • Mesh sizing.

  1. Open the file FE_bearing_edge_size.par.

    Simulation models are delivered in the \Program Files\Siemens\Solid Edge 2023\Training\Simulation folder.

  2. Notice that Static Study 1 was created and is the active study.

  3. Mesh the bearing.

    1. Choose Simulation tab→Mesh group→Mesh.

    2. In the Tetrahedral Mesh dialog box, click the Mesh button.

      The NX Nastran mesh processing starts and displays a progress meter.

      Note:

      In the absence of loads and constraints, you cannot solve the study, but you can mesh the model.

      After a short period of time, the bearing displays with a tetrahedral mesh.

  4. Rotate the part so you can view the rear, and zoom in on the main bore.

  5. Change (refine) edge sizing.

    1. Choose Simulation tab→Mesh group→Edge Size.

    2. Select the circular edge of the bore.

    3. In the Edge Size dialog box, type 50 in the Number of elements box, and then click Accept

    4. Mesh the part again.

      Note the increased number of elements in the region around the selected edge.

      5. Observe that the edge sizing you applied is shown in the Simulation pane as Edge Size 1.

  6. Change the number of edge sizing elements.

    1. Double-click the Edge Size 1 entry.

      The Edge Size dialog box appears, showing the current sizing value for the number of elements applied to the edge.

    2. Change the Number of elements to 75 and click Accept.

    3. Mesh the part again. In the Simulation pane, right-click the Mesh node and choose the Mesh command from the shortcut menu.

      Now, you see a significant change around that edge.

      Note:

      Increasing the number of elements in a single, highly-stressed region of a part can improve the accuracy of the analysis without greatly impacting the processing time within NX Nastran.

  7. Close this file.
