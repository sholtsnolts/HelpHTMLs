Tutorial: Create and simulate a roof truss framework

In this activity, you will explore how to construct and simulate a framework of roof trusses.

  1. Open the file FE_Roof.asm.

    Simulation models are delivered in the \Program Files\Siemens\Solid Edge 2023\Training\Simulation folder.

  2. If the Frame environment is not opened already, select Tools tab→Environs group→Frame .

  3. Create the initial truss.

    1. On the ribbon, choose Home tab→Frame group→Frame .

    2. In the Frame Options dialog box, click OK.

    3. In the Frame dialog box, open the ANSI folder, then the Rectangular Tubing folder. Select Rectangular Tubing 2x1 1I2x.12.par.

    4. Click OK on any notification dialog box.

    5. Select the first truss components as shown below.

    6. Click the Accept button, and then click Finish.

    7. In the Frame Options dialog box, choose OK.

  4. Create the remaining trusses.

    1. The Frame command is still active. Select the next truss components as shown.

    2. Click the Accept button, and then click Finish.

    3. In the Frame Options dialog box, choose OK.

    4. Repeat for the remaining trusses until the entire framework is created.

    5. Press Esc to exit the Frame command.

      Tip:

      As an alternate method of creating the framework you can drag a select box around all of the paths and all of the trusses will be created in one step.

  5. Select the Solid Edge File tab, and then choose Save.

  6. Observe the Frame on PathFinder.

    1. Note that a single Frame Components feature exists on PathFinder.

    2. Expand this group and you can see all of the frames you created in the previous steps. Place your cursor over each frame and observe that it highlights in the view.

    3. Expand each frame and the individual beams are displayed. Place your cursor over each beam and observe that it highlights in the view.

      Note:

      If you used the single selection of all paths at one time as explained in the previous tip, PathFinder displays just one frame containing all of the beams.

  7. Create a study.

    1. Select the Simulation tab→Study group→New Study command.

    2. In the Create Study dialog box:

      • Choose the Linear Static study type and Beam mesh type, and then click Options.

      • Under Connector Options, ensure the Maximum rigid link length check box is selected. Do not change the default value.

      • Under Results Options, in the Nodal section, select Beam end reactions.

      • Click OK.

    3. The Define command in the Geometry group is active. Drag a selection box around the entire frameworks.

    4. Accept the frame.

      The blue curves represent beam curves.

      In the Simulation pane, notice the Rigid Links collector is now populated with the rigid links that were generated automatically between beams when the study and study geometry were defined. For more information, see Rigid link connectors.

  8. Place loads on the roof.

    1. Select the Simulation tab→Structural Loads group→Force command.

    2. On the command bar, change the Selection Filter to Curve. Select the beams on the top of the frame.

      Note:

      There are 18 beam curves.

    3. Select the Flip button on the Loads command bar to flip the load direction.

    4. Type 1000 lbf in the Value box.

    5. Right-click to accept, and then click to finish.

      6. The force is placed.

  9. Add a fixed constraint to the outside members of the framework.

    1. Select the Simulation tab→Constraints group→Fixed command.

    2. On the command bar, change the Selection Filter to Curve.

    3. Select the four outside beams shown below.

    4. Right-click to accept, and then click to finish.

  10. In the Simulation pane, do the following:

    1. Place your cursor over the Geometry node and right-click. Select Hide Beam Curve.

    2. Turn off the display of the loads and constraints by clearing the check boxes in front of them.

  11. Mesh and solve the study.

    1. Select the Mesh command, and in the Mesh dialog box, use the slider to define a subjective mesh size of 3 or 4.

    2. Click the Mesh and Solve button.

      3. After a few moments, the results are displayed in the Simulation Results environment.

  12. Display beam reaction plots.

    1. In the Home tab→Data Selection group:

      1. From the Result Type list, choose Beam End Reactions.

      2. From the Result Component list, choose Beam Plane1 Moment.

        Note:

        • The color bar header is turned off in these images.

        • The units for moment values are inches-pounds force.

    2. Without changing the Result Type =Beam End Reactions selection, review the other results that are available in the Result Component list.

      3. Note:

      If your plot results do not look like the images shown above, do the following:

      1. Choose Home tab→Show group→Display Options .

      2. Verify that the Contour, Deformation, Color Bar, and Beam Cross Section options are selected, and click Close.

      3. From the Display tab→Primary Display group→ Edge Style list, select External Mesh.

      4. From the Color Bar tab→Format group→ Format list, select Floating Point.

  13. Display a beam diagram.

    1. Select Home tab→Show group→Display Options, and select the Beam Diagram check box.

    2. Verify that Beam End Reactions is still selected in the Result Type list. From the Component Type list, select Beam Plane1 Moment.

    3. Compare the beam cross section results, shown above, with the beam diagram results for the beam end reactions, shown below.

      Note:

      To learn more about beam plots:

      • Beam reaction component plots

      • Beam stress component plots

      • Beam diagrams

      • Beam properties

  14. Close this file.
