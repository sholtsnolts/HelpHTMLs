Design Limits (Optimization dialog box)

The Design Limits page in the Optimization dialog box defines the condition that the optimization must satisfy. You can impose multiple design limits on an optimization.

For more information about defining the problem, see Optimizing study results. For more information about selecting variables, see Defining design optimization parameters.

Add Limit

Opens the Add Design Limits dialog box for you to select a design limit for the optimization.

Only parameters that are valid for selection based on model type and study type are listed in the dialog box. These are the same properties that are valid for the design objective, but you cannot select the same property for both.

If you want to use a simulation study result such as stress as a limit on the optimization, you must solve the study first to make the simulation variables available for selection in the dialog box.

Example:

To set a design limit of Stress<Yield Stress/3, you can use the Add Design Limits dialog box to select the simulation result, Von Mises Stress, for optimization.

For more information about defining the problem, see Optimizing study results.

Maximum

When the selected limit is a Simulation Study Result component (for example, Total Translation or Von Mises Stress), specifies that the optimization process uses the maximum value from the results to define the design limit.

Minimum

When the selected limit is a Simulation Study Result component (for example, Total Translation or Von Mises Stress), the optimization process uses the minimum value from the results to define the design limit.

Value

Displays the current value of the selected limit.

If Not Applicable is displayed in the Value box, this means the result is unprocessed, the study is not solved, or the geometry is not included in the study.

Delete Limit

Deletes the associated limit when a cell or a row is selected on the Design Limits tab.

Name

Displays the name of the design limit.

Current Value

Shows the current value of the design limit variable.

Rule

Opens the Design Limit Rule Editor dialog box for you to specify minimum or maximum conditions or a range for the limit.

Example:

To set a design limit of Stress<Yield Stress/3, you can use the Design Limit Rule Editor dialog box to specify a Maximum limit less than or equal to the value for Yield Stress/3.

Limit Value

Displays the value specified in the Design Limit Rule Editor dialog box.
