Save a study to Femap

The study geometry, material, loads, constraints, and mesh controls are saved to the Femap .modfem file, as are the mesh, analysis results, and other data that may exist in Solid Edge Simulation.

  1. In the Simulation pane, right-click the name of the study you want to export, and then choose the Save Femap Model File command .

  2. In the Save Femap Model File dialog box, select the folder where you want to create the file.

  3. (Optional) In the File Name box, type a new name for the document.

  4. Click Save.

Tip:

  • The .modfem file is an executable file, which is ready to open in Femap.

  • For information about how Solid Edge models and simulation data units are handled in Femap, see Setting units for files opened in Femap. You also can try the hands-on activities in Practice: Setting units for Femap.
