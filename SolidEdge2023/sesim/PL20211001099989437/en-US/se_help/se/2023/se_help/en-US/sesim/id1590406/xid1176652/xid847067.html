Apply results display settings to the simulation results

You can apply previously saved results display settings to the solved study currently shown in the Simulation Results environment. These settings may contain such parameters as model display state, contour style, color bar style, deformation, face and edge color, and font specifications. They do not include study-dependent options set on the Home tab→Data Selection group, or the scale options set on the Color Bar tab→Scale group.

  • In the Simulation Results environment, do one of the following:

    • From the Home tab→Settings group→Current Named Display Settings list, choose a results display settings name.

      Tip:

      If the results saved settings have been saved globally using the Save Results Display Global command, then choosing the default,username setting will retrieve the global values.

    • On the Home tab→Settings group, click Results Display Settings .

      The Simulation Results Display Settings dialog box is displayed.

      1. From the lower box, select the setting name you want to use.

      2. Click Apply.

Tip:

  • If global display settings have not been saved, then default values are used.

  • You can update your currently applied saved settings at any time by clicking the Save Results Display Settings button in the Settings group.
