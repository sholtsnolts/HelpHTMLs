Design Variables (Optimization dialog box)

The Design Variables page in the Optimization dialog box specifies the changeable dimensions or other variables of the model.

Add Model Variable

Opens the Variable Table for you to select one or more model variables that you want to allow to change during optimization. For example, you can select model dimensions or simulation variables representing plate thickness or loads.

Use the Add button in the Variable Table to add a selected variable to the Optimization dialog box.

For more information about selecting design variables, see:

  • Defining design optimization parameters

  • Best practices for design optimization

Add Peer Variable

If you are optimizing a study of an assembly model, you can add variables from any of the part or subassembly models in the study using the Add Peer Variable button. This starts the Peer Variables command, which prompts you to select the part or subassembly model whose variables you want to use in the optimization. When the Variable Table for the selected model is displayed, you can choose a variable to add to the Design Variables page in the Optimization dialog box.

Delete Variable

Deletes the associated variable when a cell or a row is selected on the Design Variables tab.

Type

Displays the type of design variable selected. These may be user variables, dimensions (2D), simulation variables, physical properties, or PMI dimensions.

Name

Displays the name of the design limit.

Current Value

Shows the current value of the design limit variable.

Rule

Opens the Design Variable Rule Editor dialog box for you to specify minimum or maximum conditions or a range for the limit.

Range

Displays the value range defined in the Design Limit Rule Editor dialog box.
