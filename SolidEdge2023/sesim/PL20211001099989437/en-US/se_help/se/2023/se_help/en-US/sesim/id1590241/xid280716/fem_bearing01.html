Activity: Apply a bearing load

  1. Open the file named Mower_pulley.asm.

    Simulation models are delivered in the \Program Files\Siemens\Solid Edge 2023\Training\Simulation folder.

    Note:

    You can analyze a single part while its parent assembly is displayed.

  2. Choose Simulation tab→Study group→New Study.

  3. Choose the linear static study type and tetrahedral mesh type. Select OK.

  4. To satisfy the Define command, select the wheel as the part to activate.

    Note:

    The Define command allows you to analyze a single part while working within the context of an assembly.

  5. Click Accept on the command bar.

  6. Hide the display of some fasteners:

    1. Rotate and zoom the view to see the underside of the wheel.

    2. Select washer.par. Also select the two occurrences of nut.par within the lever_arm.asm.

    3. Right-click and choose Hide.

  7. Add a bearing load:

    1. Select Simulation tab→Structural Loads group→Bearing Load

      The Loads command bar is displayed.

    2. Select the cylindrical face shown.

    3. Type 50 for the load value and 135 for the angle value, and then press Tab.

    4. Rotate the view so you can look straight down into the center of the wheel. Select the origin of the directional steering wheel, align it with the center of the bore and ensure the primary axis points along the direction of the lever arm.

    5. To learn how to use the steering wheel to change the load direction, see Loads directional steering wheel.

    6. Right-click in space to accept, and then click to finish.

    7. Hide bolt.par to see the bearing load symbol.

  8. Define a fixed constraint on several faces.

    1. Choose Simulation tab→Constraints group→Fixed.

    2. Rotate the view in order to see a side of the wheel. Select the faces in the center of the wheel.

    3. Right-click in space to accept, and then click to finish.

    4. Press Ctrl+I to set an isometric view.

  9. Mesh and solve the study:

    1. Select the Mesh command. Choose Mesh & Solve.

      After a few moments, the part is meshed and solved. The results are displayed in the Simulation Results environment.

    2. Select Home tab→Deformation group→Actual. Zoom to see the highest stress concentration.

  10. Close this file.
