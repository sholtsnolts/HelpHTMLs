Assign units in studies

Default analysis units, such as the force and pressure units used to define structural loads, or the default temperature units applied to body loads, are defined at the document level based on the template you are using. You can review the units and make changes to them on the Units tab in the Solid Edge Options dialog box. The units you assign are used by the study during processing, and they are used by default to display results in the Simulation Results environment.

  1. From the File menu, choose Info→File Units.

    This displays the Units tab in the Solid Edge Options dialog box.

  2. On the Units page, scroll down to the Base Units table and the Derived Units table.

  3. For each analysis property unit listed in the Name column that you want to change, do the following:

    1. Click the cell in the Value column, and then select the default units from the list.

      Only valid units are displayed.

      Example:

      For Pressure, you can choose Pa, mPa, kPa, N/mm^2, psi, ksi, psf, ksf, bar, mbr, and atm.

    2. Click the cell in the Precision column, and then select the default number of decimal places from the list.

      Note:

      In the Simulation Results environment, you can change the precision value used in plots and reports on the Color Bar tab→Format group.

Tip:

If your model was developed using a metric template whose Length unit is millimeters or meters, and if you want to open a Solid Edge Simulation study in Femap, then you must set consistent units in Femap for your Solid Edge models and simulation data.

For more information, see Setting units for files opened in Femap.
