Create and solve a study using GD optimized model


This example shows how you can create and solve a linear static study for optimized geometry model from generative design. It demonstrates how to define geometry, create mesh, apply constraints, and solve the simulation.

  1. In the Generative Design pane, under the Results node, right-click a result name and choose Detach from Study.

  2. In PathFinder, under the Construction Bodies node, right-click a solid mesh body name and choose Toggle Design/Construction. The construction body is converted to a design body. The new design mesh body is listed under the Design Bodies pane.

    Note:

    To get accurate results, we recommend that; you convert the construction body to a design body.

  3. Create a simulation study.

    1. On the Simulation tab → Study group, select the material from the Material List.

    2. Select Simulation tab → Study group → New Study .

    3. For this example, choose the Linear Static study type and Tetrahedral mesh type, and then click OK.

    The new study is listed in the Simulation pane.

  4. Select a convergent geometry to include in the study.

    When creating a new study for a solid part model with a Mixed or Tetrahedral mesh type, the Simulation tab → Geometry group → Define command runs automatically. Select the design body and click Accept on the Define command bar.

  5. Define the load condition.

    For this example, the pressure load is applied to the top face using the Pressure command in the Structural Loads group.

  6. Apply a fixed constraint.

    A fixed constraint is applied to the bottom face using the Fixed command in the Constraints group.

    Note:

    The constraint types Cylindrical and Sliding Along Surface cannot be applied when you convert the solid design body to a mesh body.

  7. Mesh and solve the study.

    1. From the Simulation tab → Mesh group, select the Mesh command .

    2. In the Tetrahedral Mesh dialog box, click Mesh & Solve to solve the study.

    The part is meshed and solved. The solution is displayed in the Simulation Results environment.

    Note:

    To improve the mesh for a complex part, we recommend that; you select the Body Mesh option available in the Mesh dialog box.
