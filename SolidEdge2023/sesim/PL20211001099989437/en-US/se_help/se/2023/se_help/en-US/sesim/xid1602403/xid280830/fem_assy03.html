Tutorial: Using bolted connections

Bolt connectors transfer tensile, bending, and thermal loads from one part to another. You can use the Simulation tab→Connectors group→Bolt command to simulate bolted connections between two part faces, without having to model the fastener itself.

Most of the information required to create a bolt connector is inferred from the elements you select. If your 3D model contains bolts and other fasteners, you can hide them in PathFinder and just use the part holes to generate the bolt connectors in the FEA model. Selecting holes rather than modeled bolt components yields a much faster processing time for meshing and solving.

Example:

If you select a hole, the software determines the connected parts by finding cylindrical holes along a shared axis in other parts. The top and bottom faces are determined from the connected face(s) at either end of the hole. The type of bolt defaults to no-press fit. The diameter of the head and nut are calculated from the diameter of the selected hole.

In this tutorial, you explore how to apply a bolt connection within an assembly, and review the stresses associated with the connectors in the Simulation Results environment.

  1. Open FE_pipe_connect.asm.

    Simulation models are delivered in the \Program Files\Siemens\Solid Edge 2023\Training\Simulation folder.

  2. Define the study.

    1. Select the Simulation tab→Study group→New Study command.

    2. In the Create Study dialog box, select the following:

      1. Study type=Linear Static

      2. Mesh type=Tetrahedral

      3. Click Options.

      4. In the Results Options section, under Elemental, select Force.

    3. Notice the Define command in the Geometry group is active.

      • Select both parts and click Accept.

  3. Define a force on the pipe flange.

    1. Select Simulation tab→Structural Loads group→Force.

    2. Select the end face of the pipe.

    3. Type 1334 N for the force value. This is equal to a 300 pounds-force load.

      You may need to select the Flip button to ensure the force direction is downward.

    4. Right-click to accept and then click to finish.

      The force is placed.

  4. Add fixed constraints to the flange.

    1. Rotate the view to see the back side of the mating flange.

    2. Select Simulation tab→Constraints group→Fixed, and then select the outer cylindrical and end faces.

    3. Right-click to accept, and then click to finish.

  5. Define assembly connectors between the pipe flange and the mating flange.

    1. Choose Simulation tab→Connectors group→Manual command .

    2. On the command bar, select No penetration as the connector type.

    3. Use QuickPick to select the front face of the pipe flange in pipe_flange.par as the target face, and then right-click to accept.

      Note:

      You also can turn off the display of the mating flange mating_flange.par.

    4. Use QuickPick again to select the inside face of the mating flange in mating_flange.par as the source face, and then right-click to accept.

      Note:

      You also can turn on the mating flange and turn off the pipe flange pipe_flange.par. Rotate the model so you can view the front face of the mating flange.

    5. Click to finish.

  6. Define a bolt connection.

    1. Choose Simulation tab→Connectors group→Bolt command .

    2. On the command bar, ensure the Selection Type is set to Single Hole .

    3. Select the first bolt hole in the pipe flange as shown below.

      For this activity, the default values are acceptable for bolt and nut diameter and for prestress force.

    4. On the command bar, click Multiple Bolts .

      Select the three remaining holes in the flange.

      Tip:

      If your part has many holes of the same size, you can use the Multiple Bolts option to select them. This also makes it easy to edit all of the bolt connectors at once.

    5. Right-click to accept and click to finish.

  7. Mesh the assembly—Select Simulation tab→Mesh group→Mesh, and click the Mesh button.

    The entire assembly is meshed.

  8. Solve the study—Select the Simulation tab→Solve group→Solve command.

    After a few moments, the results are displayed in the Simulation Results environment.

    Note:

    If the study does not solve and an NX Nastran warning message is displayed, edit the study processing parameters using the Modify Study dialog box. Change the Geometry check option to Warning only and then solve again. For more information about this option, see Advanced Options for studies.

  9. Review the bolt connector force results—In the Simulation Results pane, under the Results node, double-click Bolt Connector Forces.

    The Bolt Connector Forces Results table is displayed. Because you used the Multiple Bolts option to create the bolted connections, the table identifies all of the bolts in the Connector Name column as Bolt Connection 1, but it reports the unique stresses for each of the simulated bolt connections.

    For more information, see Bolt connector simulation results.

  10. Close this file.
