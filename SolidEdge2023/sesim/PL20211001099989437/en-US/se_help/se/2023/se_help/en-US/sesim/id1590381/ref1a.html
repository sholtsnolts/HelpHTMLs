Displaying processing logs

Solid Edge Simulation employs the NX Nastran solver to process your finite element analysis studies. There are two ways to display and review NX Nastran solution processing information.

  • When you choose the Solve command, the Simulation Progress dialog box is displayed to show the solution processing status. During processing, you can use the options in this dialog box to display and review the files associated with the NX Nastran solver: .log, .f04, and .f06.

  • After solving, you can review the processing files for the active study in the NX Nastran Log Viewer. Open the viewer by selecting either of the following:

    • Simulation tab→Study group→Nastran Log Viewer command .

    • View tab→Show group→Panes command →NX Nastran Log Viewer .

    The NX Nastran Log Viewer contains information about the .log, .f04, and .f06 processing files:

    .log

    The .log file contains system environment information for running NX Nastran and for file linking.

    .f04

    The .f04 file contains information about the files accessed, disk space, modules used, and more. This file is useful for debugging.

    .f06

    The .f06 file is an ASCII file that contains processing errors.

Solid Edge Simulation options

The Simulation page on the Solid Edge Options dialog box enables you to tailor many aspects of your Solid Edge Simulation environment. This includes setting unique colors for each type of load and constraint, and for setting default solution processing options.

In addition, the File Locations page on the Solid Edge Options dialog box is where you can specify an NX Nastran scratch processing directory for analysis files.
