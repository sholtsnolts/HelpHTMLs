Overview of Solid Edge Simulation finite element analysis

Solid Edge Simulation overview

Solid Edge Simulation is a finite element analysis (FEA) application designed specifically for engineers and designer-analysts who use Solid Edge. In a structural simulation, FEA helps you visualize where structures bend or twist, and it indicates the distribution of stresses and displacements. This information helps you minimize weight, materials, and costs.

FEA enables entire designs to be constructed, refined, and optimized before the design is manufactured. This reduces the time-to-market and the costs incurred in rework.

Solid Edge Simulation comes with dedicated, integrated, user assistance—both help and tutorials—to get you started quickly.

Getting started

If you have a Solid Edge Simulation license and a model document open, you should see the Simulation tab on the command ribbon. That is your starting point for finite element analysis.

  • You can select the New Study command on the Simulation tab and then press F1 to open the help instructions, Analyze a model.

  • You can practice using Solid Edge Simulation using the Solid Edge Simulation tutorials. See Practice creating and using studies for a complete list of tutorials provided.

If you do not see the Simulation tab on the command ribbon, then you need a Solid Edge Simulation license. There are three levels of Solid Edge Simulation licenses available: Premium, Standard, and Advanced. For more information, see Solid Edge Simulation licensing.

Relationship to Femap and NX Nastran

Solid Edge Simulation employs Femap pre- and postprocessing resources—plus the industry-standard NX Nastran solver—to analyze part, sheet metal, assembly, and structural frame models. Whereas Femap is targeted toward dedicated analysts who typically spend most of their day doing finite element analysis, and who have more in-depth knowledge of FEA, Solid Edge Simulation provides basic, integrated analysis for Solid Edge users. Solid Edge Simulation replaces the complex Femap user interface with one that has been simplified and integrated with Solid Edge.
