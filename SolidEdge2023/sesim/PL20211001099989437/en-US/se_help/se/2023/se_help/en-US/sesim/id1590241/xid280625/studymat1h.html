Verify simulation material properties

  1. Open a model with a defined Solid Edge simulation study, and activate a study. The study does not have to be solved.

  2. Select Simulation tab→Study group→Material Table .

  3. In the Material Table dialog box, select the Material Properties tab, and then locate and click a material in the left pane. You can select the material assigned to the model, or you can select a different material that you want to apply to the model.

  4. On the right side of the Material Table, click the Highlight Simulation Properties button .

  5. In the Property Name column, review the highlighted properties, and verify that there are valid values for each in the Value column.

    The properties that are highlighted are required for the type of study you created. The values listed are based on the selected material.

    Tip:

    Highlighted properties are color-coded:

    Required properties are used for solving the simulation study.

    Secondary properties are used for calculating results if specific loads are defined in the study, or for viewing reports and results.
