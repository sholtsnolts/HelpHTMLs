Create a symbol

  1. Choose Home tab→Clipboard group→Copy to Library.

  2. Click the elements to define the symbol, and then click the Accept (check mark) button on the command bar.

  3. On the command bar, in the Name box, type a name for the symbol.

  4. Click to define an origin point for the symbol.

    Note:

    For Solid Edge data management users who selected the option Automatically name files via Document Number and Revision, the system displays a dialog box where you can assign common properties, such as document number and revision number, as explained in Create documents with unique properties.

Tip:

  • When you use the Copy to Library command to create a symbol, a copy of the graphics is used to create the symbol.

  • You can also create a symbol by dragging the graphics you want into the Library tab. When you create a symbol using this method, the original graphics are used to create the symbol, and Solid Edge defines the name and origin.
