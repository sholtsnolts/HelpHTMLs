Add Block View command


Defines another geometry representation of a source block in the active document. Use this command to create unique configurations or views of an object within the same block. The source block under which the view is created is set automatically to be the default block. It appears in boldface in the Block Selection Pane.

To select another block as the basis for a new block view, first place the block in the document, and then select the Unblock command to make the graphics selectable.

For more information about using a.dft, .dxf, or .dwg file as the source of block view geometry, see Using blocks.
