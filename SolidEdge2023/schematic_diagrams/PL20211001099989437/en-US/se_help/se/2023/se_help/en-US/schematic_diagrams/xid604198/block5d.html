Block Label command bar

The Block Label command bar is displayed when you select the Sketching tab→Blocks group→Block Label command and when you click a block label on the drawing sheet so that you can edit it.

Dimension Style Mapping

Specifies that the setting on the Dimension Style tab of the Options dialog box determines the dimension style. When the Dimension Style Mapping option is set, the individual Dimension Style control is unavailable.

Dimension Style

Lists and applies the available styles. This control is unavailable when Dimension Style Mapping is enabled.

Properties

Displays the Block Label Properties dialog box, which defines the block label name, behavior, and properties.

Block Label Name

Displays the selected block label name. The block label name is assigned in the Block Label Properties dialog box and cannot be changed on the command bar.

X:

Sets the default X coordinate for the block label origin point where it is attached to the block. The block label handle also is located at the specified (X, Y) point.

Y:

Sets the default Y coordinate for the block label origin point where it is attached to the block. The block label handle also is located at the specified (X, Y) point.
