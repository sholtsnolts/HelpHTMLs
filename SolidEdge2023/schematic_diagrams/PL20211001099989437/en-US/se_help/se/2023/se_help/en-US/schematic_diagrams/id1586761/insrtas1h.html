Set the symbol placement method

Set symbol placement method

  1. Click the Library tab.

  2. On the Library page, with nothing selected, right-click to display the shortcut menu.

  3. On the shortcut menu, point to Insert Symbol As, and then click the insert option you want.

    Tip:

    To learn about the different methods for placing symbols, see the Choosing a symbol placement method section in the Symbols Overview Help topic.

When the Show Blocks option is set in the Library, then symbol functions are not available.

Enable the options on the Insert Symbol As shortcut menu

  1. Click the Library tab.

  2. On the Library page, in the row of command buttons at the top, deselect the first one, Show Blocks: .

    You are now able to select the options on the Insert Symbol As shortcut menu.
