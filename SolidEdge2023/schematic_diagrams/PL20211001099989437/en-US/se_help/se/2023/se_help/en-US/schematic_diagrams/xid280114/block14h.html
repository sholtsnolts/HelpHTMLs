View and modify block scale

Set the scale of new blocks

  1. Place a block.

  2. In the Block Scale box on the command bar, type a new value, and then press the Enter or Tab key to update the graphics.

  3. Click to position the block.

  4. Right-click to end the command.

    Tip:

    • While dragging a file, if you change the scale value in the Block Scale box and press Enter or Tab, you can change the size of the red frame that represents the geometry so that it is fully visible on the drawing sheet. Then you can position it exactly as you want it before clicking to place it on the sheet.

    • If you do not know what scale to use, click to place the geometry using the default scale, and then click the Fit button located on the bottom frame of the graphics window. This will make everything visible on the sheet.

Modify the scale of block occurrences

  1. Click an existing block in the active document.

    The Select Tool command bar is displayed.

  2. In the Block Scale box on the command bar, do one of the following:

    • Type a new scale value, and then press Enter or Tab to update the block graphics. The scale value can be any positive number.

    • Select a previously used scale value from the list.
