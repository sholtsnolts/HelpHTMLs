Behavior Tab

Displays and allows you to set symbol behavior options.

Insertion Type

Specifies the insertion type for a symbol. Once you set an insertion type for a symbol you can not change it.

Update

Specifies how the symbol can be updated. Options are Automatic, Frozen, and Manual.

Scale Factor

Specifies the scale for a symbol.

Scale Dimensions And Annotations

Specifies that dimensions and annotations that are contained within the symbol are scaled with the symbol. When this option is set, the annotations and dimensions display at the active Scale Factor for the symbol. When this option is cleared, annotations and dimensions display and print at paper scale.

Scale Lock

Specifies whether the symbol can be scaled. When this option is set, the symbol can not be scaled. When this option is cleared, the symbol can be scaled.

Nested Display

Defines the behavior of nested symbols. The options are Public and Private. When you open a document that contains nested symbols, the public option displays them and the private option hides them.
