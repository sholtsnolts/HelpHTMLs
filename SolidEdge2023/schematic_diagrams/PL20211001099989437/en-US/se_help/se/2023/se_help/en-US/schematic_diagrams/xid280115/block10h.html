Edit block graphics

When you open a source block, block view, or block occurrence for editing, the graphics display in a window superimposed on the active sheet, similar to Draw In View.

  1. On the drawing sheet or on the Block Library Selection Pane, right-click the block to edit, then click Open on its shortcut menu. Alternatively, you can double-click a block on the drawing sheet to open it for editing.

    • If you open a block occurrence on the drawing sheet, any edits you make to this occurrence will be applied to the source block in the Block Library as well as to the other occurrences.

    • If you open a block occurrence of a multi-view block, the edits will update the view of the source block.

  2. For Nested Blocks—To edit a block nested within the currently open block, double-click the block graphics again to open the next level of graphics in another window. You can continue in this way until you have opened the nested block that you want to modify.

  3. Add or edit graphics in the block as needed. For example, you can:

    • Use the drawing tools to add lines, circles, arcs, or other 2D elements to the block.

    • Select the handles on individual graphical elements to resize them, delete them, or otherwise change their color, line type, or thickness using the options on the command bar.

    • Copy/paste graphics from another sheet into the editing window. You can switch to other windows within the active document. Unsupported objects are not copied to the block.

    • Drag another block onto the current block you are editing. Unblock it to make its individual elements selectable.

    • Group or ungroup elements within the block. Use these to create a new block within the current block.

    • Add an attribute definition, called a block label, to the block.

  4. Click Return to save your edits and update the source block definition and all occurrences in the active document. If you have opened a nested block for editing, click Return until the highest level block is closed and the Return button is no longer displayed.

Note:

You cannot place, copy/paste, or create a block of the same name as the one you are editing.
