Edit a connector

  1. On the drawing, click the connector to edit.

    This displays the Connector command bar and highlights the connector segment handles as well as any connected parent elements.

  2. Do one of the following:

    • Change the connector shape, orientation, and other properties using options in the Connector command bar. Connector modifications made using the command bar are updated dynamically.

    • Select Properties from the connector's shortcut menu to display the Connector Properties dialog box.

Tip:

For specific examples of the ways that you can edit connectors, see Using connectors.
