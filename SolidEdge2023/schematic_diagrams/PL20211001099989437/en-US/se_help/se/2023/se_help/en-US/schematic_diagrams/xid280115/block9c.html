Unblock command

Drops a selected block or blocks occurrence to its base elements, similar to the way the Ungroup command works for groups. Use this command to unblock a block or blocks so that it can be selected as the graphics for a new source block or block view.

The Unblock command is available only from a selected block's shortcut menu in the active document.
