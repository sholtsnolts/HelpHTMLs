Delete (Block) command

Deletes the selected source block or block view from the active document. If you delete the last view from a block, the block is deleted, too.

All occurrences of the block in the active document, including all views of the block, are deleted with it.

You also can cut, copy, and paste blocks between documents using those commands from the selected block's shortcut menu in the document.
