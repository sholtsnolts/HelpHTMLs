Add a label to a source block or block view

You can define a block label and then include it in a source block or block view. Or you can copy and paste a label from one block to another.

  1. On the Block Selection Pane, right-click the source block or block view to add the label to, then click Open on its shortcut menu.

  2. Choose Sketching tab→Blocks Group→Block Label to define and place a label.

    Tip:

    (Copy and Paste Technique) You can select the Copy command on the shortcut menu of a label in another block on another sheet, and then select Edit→Paste to place the label into this one. All label properties are copied to the current block.

  3. Click Close Block on the command bar.

    The block label is saved with the source block definition and the editing window is closed. You can verify that the label has been included in the source block by looking at the block image in the Block Preview Pane.
