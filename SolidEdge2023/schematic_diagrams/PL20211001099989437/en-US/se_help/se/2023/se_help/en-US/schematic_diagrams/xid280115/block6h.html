Delete a block

Source blocks and block views can be deleted from the block's shortcut menu in the Block Selection Pane. To delete a block occurrence, delete the source block.

  1. Click the block name you want to delete.

  2. Right-click to display the shortcut menu.

  3. Select the Delete command.

Tip:

You also can cut, copy, and paste blocks between documents using those commands from the selected block's shortcut menu in the document.
