Unblock All command

Drops all block occurrences to their base elements, including:

  • Top level block occurrences

  • Nested block occurrences

  • All block occurrences on all sheets (working, background, and 2D Model)

The original block definitions are not deleted from the file.

A dialog box prompts you to confirm or reject the action. You also can use the Undo command to restore all nested blocks.

Tip:

You can use the Unblock All command, along with the Ungroup All command, before you copy and paste imported AutoCAD geometry to a Solid Edge sketch. These commands simplify nested block occurrences and nested groups by reducing them to their lower level geometry.
