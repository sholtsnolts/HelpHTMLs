Edit block properties

You can assign or edit block properties for existing blocks from the selected source block's shortcut menu in the Block Library or from a selected block occurrence in the drawing. To define properties for new blocks, click the Block Options button on the Block command bar.

If there is a block label included in the block definition, then additional label options are displayed for editing.

  1. On the drawing or on the Block Library Selection Pane, select the block to edit.

  2. Right-click to display the shortcut menu.

  3. Select the Properties command.

  4. Edit Block Graphics Properties—On the Block Properties dialog box, in the Properties area, add or edit property names and values that refer to the block's graphical representation.

  5. Edit Block Layer—You can move the block occurrence to another layer by selecting a layer setting from the dropdown list.

  6. Set or clear the Scale Dimensions and Annotations option.

  7. Edit Block Origin Point—The X, Y coordinates of the block origin point can be changed by typing new values in the X and Y text boxes. Press ENTER to apply the change.

  8. Edit Block Label Properties—If a label is included in the block definition, the Block Properties dialog box expands to display name-value options under the Block Label area. You can confirm or edit the label value, as well as any of the text display settings. To learn more about the block label options, see Block Label Properties dialog box.
