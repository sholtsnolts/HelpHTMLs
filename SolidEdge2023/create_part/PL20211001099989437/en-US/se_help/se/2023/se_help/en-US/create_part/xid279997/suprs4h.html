Suppress an assembly relationship

  1. Click the PathFinder tab.

  2. In the top pane of PathFinder, select the part which you want to suppress a relationship.

    Note:

    The part highlights in the assembly window and the bottom pane of PathFinder shows the relationships applied to the part you selected.

  3. In the bottom pane of PathFinder, click the relationship you want to suppress.

    Note:

    The faces that were used to apply the relationship highlight in the assembly window to show which relationship will be suppressed.

  4. Right-click to display the shortcut menu, then click Suppress.

Note:

  • When you suppress an assembly relationship, a symbol is added adjacent to the assembly relationship in the bottom pane of PathFinder to indicate that the relationship is suppressed. The symbol in the top pane of PathFinder also changes to show that the part is no longer fully positioned and the part is put in the Error Assistant dialog box.

  • You can use the Unsuppress command on the shortcut menu to unsuppress the relationship later.

  • You can use the Suppression Variable command to add a suppression variable to the variable table. Conditional statements in the formula field can be used to control whether the relationship is suppressed or unsuppressed. A value of 1 (one) indicates that the relationship is suppressed. A value of 0 (zero) indicates the relationship is unsuppressed.

  • Formulas in the variable table can be used to suppress assembly relationships when the value of the variable meets the criteria defined by the formula. See the Add Suppression Variable command topic for more information about suppression variables.
