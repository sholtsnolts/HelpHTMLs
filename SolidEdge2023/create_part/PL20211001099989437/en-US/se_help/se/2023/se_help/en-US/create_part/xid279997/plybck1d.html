Feature Playback tab

Allows you to playback the construction of a part model feature by feature so you can visualize the steps used to build the part. This can be useful when working with a part you are unfamiliar with.

Rewind

Sets playback mode to the base feature of the model.

Play

Plays back the model construction one feature at a time. You can also control how fast you want the playback to proceed.

Step

Steps through the model construction one feature each time you click the button.

Stop

Stops the playback at the current feature.

Seconds Between Features

Sets the speed of the playback.
