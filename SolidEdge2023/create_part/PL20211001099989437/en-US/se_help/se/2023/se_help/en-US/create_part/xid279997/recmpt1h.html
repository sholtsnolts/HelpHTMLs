Recompute a feature

  1. Display the PathFinder tab.

  2. On the PathFinder tab, click the name of the feature you want to recompute.

    Note:

    To find the feature you are interested in, move the cursor over the feature names in PathFinder. Each feature is highlighted in the graphics window when you position the cursor over its name.

  3. Right-click to display the shortcut menu, then click Recompute.

Tip:

  • You can also select the feature you want to recompute in the graphics window.

  • When you use the Recompute command, the dimensions and geometric relationships for the selected feature and subsequent features are recalculated to ensure that the features are up-to-date.

  • You can only recompute features when the Automatic Update command is set.
