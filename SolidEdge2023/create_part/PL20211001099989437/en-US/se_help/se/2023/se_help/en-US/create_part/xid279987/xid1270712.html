Construct a cutout (ordered)

Note:

The Cut command is available through customization. Use the Cut option for the Extrude command to create cutouts. For information on adding the command to the ribbon bar, see Customize the command ribbon.

  1. On the Home tab→Solids group, click the Cut command .

    Note:

    You can also right-click a region and select the command from the ribbon or the context toolbar.

  2. Do one of the following:

    • If you want to draw a profile, select a planar face or reference plane.

    • If you want to select a profile from an existing sketch, on the command bar, on the Create-From Options list, click the Select From Sketch option.

      This option is not available if there are no sketches in the document.

  3. Draw or select an open or closed profile.

    Note:

    If no profile exists, see Draw a profile. For examples, see Profile-based ordered features.

  4. If the profile you selected is an open profile, click to define the side of the profile you want to remove material from.

  5. Define the extent of the material you want to remove using the options on the Cut command bar.

  6. Click Finish to finish the feature, and then click Cancel to exit the command.

Tip:

  • The ends of an open profile are extended infinitely—an arc with open ends is extended to form a circle.

  • The Treatment Step on the command bar allows you to define draft and crown parameters on a protrusion or cutout feature.
