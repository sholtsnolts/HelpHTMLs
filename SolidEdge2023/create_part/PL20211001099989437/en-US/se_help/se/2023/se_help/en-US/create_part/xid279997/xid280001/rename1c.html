Rename command

The Rename command is available on the shortcut menu of an object selected in PathFinder or any docking pane in Solid Edge. Type the new name within the field provided and press Enter. For example, you can rename a feature, assembly occurrence, assembly pattern. You also can rename PMI model views and individual PMI elements, such as annotations and dimensions.

When renaming assembly occurrences, only the occurrence name within PathFinder is renamed. The document name that the occurrence represents is not renamed.

The Rename command is available in all environments.
