Group command (PathFinder)

Binds entries in PathFinder so you can locate, select, and manipulate them as a unit. Grouped entries are usually related, such as the sketches, cutouts, and hole features that make up the area where another part would mount.

Dimensions and annotations can also be grouped together. PMI groups exist in the PathFinder PMI collector.

Grouping similar PathFinder entries together also reduces the space requirements for a part with many features. A group entry can be expanded and collapsed using the plus and minus symbols adjacent to the group entry in PathFinder, or using the Expand and Collapse commands on the shortcut menu.

You can add any entry type in PathFinder to a group: features, sketches, design bodies, surfaces, and reference planes. Dimensions and annotations can be added to a PMI group. After you define a group for a select set of PathFinder entries, you can use the Rename command on the shortcut menu to rename the group entry to a more logical name.

You can use the Ungroup command to unbind a previously defined group of features.

Note:

You can only group contiguous Ordered entries in PathFinder. Synchronous entries do not need to be contiguous.
