Profile Error Assistant dialog box

The Profile Error Assistant dialog box for the Draft Environment is the same as for Profile, except for the addition of the Exit control.

Validate

Validates any updates you make to the profile. If the updated profile is valid, the dialog box dismisses and you are returned to the next feature creation step. If the updated profile contains errors, the profile error description is refreshed.

Zoom to Error Elements

Zooms to the element(s) that are selected in the table.

Zoom to Error Position

Zooms to the location of the error instance. If the error is an open profile error with two disconnected locations, QuickPick is displayed and lists one error element for each non-connected position.

Delete Error Elements

Deletes the profile geometry in the select set.

Convert To/From Construction

Changes the selected profile geometry to or from construction geometry.

Line Color

Allows you to change the color of the selected profile geometry.

Undo All

Undoes all actions performed in sketch. This option is only available if you set the Enable "Undo All" For Profile/Layout Modifications option on the General tab of the Options dialog box.

Error Description

Displays a description of the profile error. You can click the error description to select the invalid profile geometry. Once selected, you can zoom to the element, delete the element, change the color of the element, change the element to construction geometry, or save the geometry as a sketch or failed feature.

Save As
Failed Feature

Dismisses the dialog box and saves the profile as a failed feature.

Sketch

Dismisses the dialog box and saves the profile as a sketch.

Exit

Closes the dialog and exits the command. The button is available only in the Draft environment and is enabled when a "no geometry" error occurs in the Cutting Plane and Broken-Out Section View commands for which this applies are Cutting Plane and Broken-Out Section.

Shortcut menu commands

The following shortcut menu commands are available. The shortcut menu commands are context-sensitive. In other words, the commands which are available change based on what is selected in the table area.

Columns

Displays the Format Columns dialog box so you format the active column.

Fonts

Displays the Font dialog box so you can format the fonts used in the dialog box.

Search

Displays the Find dialog box so you can define a search string.

Sort

Allows you to sort the error descriptions in ascending or descending order.

List Error Elements

Displays a QuickPick list of all elements related to the selected profile error.

Delete Error Elements

Deletes the profile geometry in the select set.

Zoom to Error Elements

Zooms to the element(s) that are selected in the table.

Zoom to Error Position

Zooms to the location of the error instance. If the error is an open profile error with two disconnected locations, QuickPick is displayed and lists one error element for each non-connected position.

Convert To/From Construction

Changes the selected profile geometry to construction geometry.
