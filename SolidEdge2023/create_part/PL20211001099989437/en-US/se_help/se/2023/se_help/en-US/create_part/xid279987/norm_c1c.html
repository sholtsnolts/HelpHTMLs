Normal Cutout command (Part environment)

The Normal Cutout command constructs a cutout that is normal to a part face by extruding a closed curve that lies on the face. This command is useful for constructing a feature on a non-planar face using a text profile or other sketch elements.  

Unlike the Cutout command, the Normal Cutout command is not profile-based. To construct a normal cutout on a non-planar face, you must first project sketch elements onto the face using the Wrap Sketch or Project Curve commands.

During the Side Step, you can specify whether the normal cutout feature is constructed by:

  • removing material inside the wireframe elements,

  • or removing material outside the wireframe elements.
