Show All Toggle Axes command

Displays the reference axis for a cylindrical face or a revolved feature you are not currently editing. (The reference axis of a feature you are editing is always displayed.)

Displaying reference axes allows you to construct more than one cylindrical feature with the same center line. It also allows you to establish relationships based upon the center line of a cylindrical feature.

Note:

Reference axes are available only in the ordered environment.

Show All Toggle Axes command, Solid Edge 2023
