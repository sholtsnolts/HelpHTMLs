Ordered to Synchronous Converter dialog box

Displays information about the documents selected for conversion, along with buttons that start and end the conversion process, save information about the conversion process, or dismiss the dialog box.

Warning:

Since the conversion process is irreversible, you should create backup copies of your documents before starting the conversion.

List

Displays information about the conversion process including the document name, documents selected for conversion, document status, conversion status, and comments about the documents being converted.

Name

Displays the name of the document. The tooltip displays the full path for the document.

Convert

Controls whether or not a document is converted. You can select multiple rows, change the value for one row, and then right-click to copy and paste the value into the other rows.

File Status

Displays the status for the document. The file must have write-access before it can be converted.

Conversion Status

Displays the conversion status for the individual documents. There are three conversion status values that change and update during the conversion process.

  • Scheduled – Document is in the list to be converted.

  • Completed – Document conversion is complete.

  • Cannot Schedule – Document failed to convert and was not saved.

Comments

Displays information about the individual documents.

Process

Starts the document conversion process. As documents are converted, they are saved to disk.

Stop

Ends the document conversion process. When clicked, the process is stopped after the current document is processed.

Close

Closes the dialog box. This button is disabled while documents are being processed.

Report

Displays the Save As dialog box, so you can save the information about the conversion to a .txt file.
