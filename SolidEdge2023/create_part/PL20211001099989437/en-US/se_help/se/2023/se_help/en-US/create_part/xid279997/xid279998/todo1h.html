Track and fix failed features

  1. Choose Tools tab→Errors.

    The Error Assistant tracks features that have failed to recompute in part and sheet metal models. The Error Assistant command is active only when there are features in the part model that did not recompute properly.

  2. In the Error Assistant dialog box, select the feature you want to work with by clicking the row of the feature.

  3. On the Error Assistant dialog box, do one of the following:

    • For profile-based features or treatment features, to activate the command bar used to construct the feature, click the Edit Definition button. You can then return to any step in the construction process and make changes.

    • For profile-based features, to edit the profile, click the Edit Profile button. The profile view is displayed so you can add, remove, or modify profile elements.

    • For profile-based features or treatment features, to dynamically edit the profile or its dimensions, click the Dynamic Edit button. This displays the feature dimensions so you can select them and edit their values.

    • To delete the feature, click the Delete button.

    • To suppress the feature, click the Suppress button.

Tip:

  • You can also use the Investigate Error command on the PathFinder shortcut menu to fix a failed feature.

  • You can also use the Dynamic Edit option to drag a dimension to a new location or drag an under-constrained profile element to a new location to change the size or shape of the profile.

  • Check the Errors command periodically as you construct and edit parts. If the Errors command button is active, there are failed features in the list.
