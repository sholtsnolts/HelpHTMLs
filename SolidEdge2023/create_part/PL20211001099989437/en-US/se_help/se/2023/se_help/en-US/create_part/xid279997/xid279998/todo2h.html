Track and fix failed parts in an assembly

  1. Choose Tools tab→Errors.

    The Errors command in the Assembly environment tracks parts that are not fully positioned, parts whose positions have failed to recompute, and failed assembly features. The Errors command is active only when these types of problems occur.

  2. In the Error Assistant dialog box, select the part you want to work with by clicking the row of the part.

  3. Do one of the following:

    • To display the part in the graphics window, click the Highlight button on the Error Assistant dialog box.

    • To edit the selected part's relationships, click the Edit button on the Error Assistant dialog box. The Assemble command bar is displayed so you can edit the relationships for the part. If your edits allow the position of the part to be successfully recomputed, it is removed from the list.

Tip:

  • The position for a part might fail to recompute because you deleted, suppressed or edited a part feature that was used to position the part in the assembly.

  • If you are not sure of the name of the part you want to edit, highlight it before selecting the Edit button.

  • Check the Errors command periodically as you build and modify an assembly. If the Errors command button is active, there are failed parts in the list.
