Find Feature dialog box

Search String

Allows you to type the search string for the feature you are trying to find. For example, if you are looking for a particular cutout, but are not sure which one, you can type: cutout. When you click the Find Next button, each feature whose name begins with that string will be highlighted in PathFinder and the graphics window.

Find Next

Highlights the next feature based on the search string you typed.
