Error Assistant dialog box (Part and Sheet Metal environments)

Options
Edit Definition

Displays the command bar for the feature so you can edit any of the inputs for the feature.

Edit Profile

Displays the profile for the selected feature so you can edit it. You can perform all profile modification options including adding, deleting and modifying profile geometry, dimensions, and relationships.

Dynamic Edit

Allows you to modify the selected feature without rolling back the model. You can use the Dynamic Edit option to edit the dimensional value of a dimension, drag a dimension handle to reposition the dimension or the dimension text, or to drag an under-constrained profile element's handle to change the physical size and shape of the profile. To add or delete profile elements you must click the Edit Profile button.

You can clear the Enable Dynamic Edit of Profile and Dimensions option on the General tab on the Options dialog box if you want to only allow the dimensional value of dimensions to be edited with this option.

Delete Error

Deletes the feature that is selected in the list.

Zoom to Error

Reduces or enlarges the display of geometry so you can easily visualize the error.

Suppress Feature

Suppresses the feature that is selected in the list.

Print Errors

Displays the Print dialog box so you can print the contents of the Error Assistant dialog box.

Column names and functions
Name

Specifies the name of the feature that failed to recompute.

Status

Lists the failure status for the feature.

Date

Specifies the date when the recompute was attempted (or when a change was made that invalidated the listed feature).

Error

Describes the problem that prevented the feature from recomputing.

Corrective Action

Suggests what actions you can take to correct the error.

Creator

Lists the user name of the person who created the document.

Last Recompute

Lists the user name of the person who performed the last recompute on the document.

Shortcut menu commands

The following additional shortcut menu commands are available.

Columns

Displays the Columns dialog box so you format the columns. For example, you can specify which columns you want to display.

Font

Displays the Font dialog box so you can format the fonts used in the dialog box.

Sort

Allows you to sort the error descriptions in ascending or descending order.
