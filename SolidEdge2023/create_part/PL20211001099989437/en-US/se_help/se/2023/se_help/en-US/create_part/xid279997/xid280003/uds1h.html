Create, add to, or dissolve a user-defined set

You can create, add to, or dissolve user-defined sets using the commands on the shortcut menu.

Create a user-defined set

  1. Select one or more faces, features, or sketches in the graphics window, or in PathFinder.

  2. Right-click to display the shortcut menu, then click Create User-Defined Set.

Add elements to an existing user-defined set

  1. Select a valid element in PathFinder or the graphics window.

  2. Right-click to display the shortcut menu, then click Add to User-Defined Set.

  3. In PathFinder, select an existing user-defined set.

Dissolve an existing user defined set

  1. Select an existing user-defined set in PathFinder.

  2. Right-click to display the shortcut menu, then click Dissolve User-Defined Set.
