Search for a model based on shape

You can search for 3D models (JT, part, sheet metal, assembly or weldment) based on the shape of a new or existing model.

Note:

The requirements for using Shape Search are detailed in Using Shape Search to locate models.

Using Shape Search with Active Workspace (Teamcenter-managed environment)

  1. Create a new model, activate an existing model, or select a model in PathFinder.

  2. With Active Workspace enabled, choose Teamcenter tab→Document group→Shape Search.

  3. View the results of the search in Active Workspace.

    Tip:

    For best results, pin the Active Workspace docking pane.

  4. (Optional) To refine your search results, click the Shape Search Filter and set the options to ensure the results match your model as closely as possible.

Using Shape Search with a web browser

  1. Create a new model, activate an existing model, or select a model in PathFinder.

  2. Choose File menu→Tools tab→Shape Search.

  3. Log in to the Geolus server by providing your user information and password.

  4. View the results of the search in the Geolus web client.

  5. (Optional) To refine your search results and ensure the results match your part as closely as possible, set Attribute Filters and Geometric Filters.
