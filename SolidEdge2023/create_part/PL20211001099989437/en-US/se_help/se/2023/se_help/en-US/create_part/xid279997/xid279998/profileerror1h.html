Analyze and edit failed profiles with Profile Error Assistant

You can use the Profile Error Assistant dialog box to analyze and edit failed profiles.

To select an error element(s)

  • In the Profile Error description, click the error description for the invalid element(s).

To zoom to an error element(s)

  1. In the Profile Error description, click the error description for the invalid element(s).

  2. Click the Zoom to Error Elements button.

To zoom to an error position

  1. In the Profile Error description, click the error description for the invalid element(s).

  2. Click the Zoom to Error Position button.

To delete error element(s)

  1. In the Profile Error description, click the error description for the invalid element(s).

  2. Click the Delete Error Elements button.

To convert an error element(s) to construction geometry

  1. In the Profile Error description, click the error description for the invalid element(s).

  2. Click the Convert To/From Construction button.

To change the colors of an error element(s)

  1. In the Profile Error description, click the error description for the invalid element(s).

  2. Click the Line Color button.

  3. Click a new color for the element.

To save an error element(s) as a failed feature

  1. In the Profile Error description, click the error description for the invalid element(s).

  2. Click the Failed Feature button.

To save an error element(s) as a sketch

  1. In the Profile Error description, click the error description for the invalid element(s).

  2. Click the Sketch button.

To list error element(s)

  1. In the Profile Error description, right-click the error description for the invalid element(s).

  2. On the shortcut menu, click List Error Elements.

To undo all actions in the profile

  • On the Profile Error Assistant dialog box, click the Undo All button.
