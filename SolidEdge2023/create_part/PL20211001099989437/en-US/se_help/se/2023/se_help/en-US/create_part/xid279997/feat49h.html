Reorder features

  1. Click the PathFinder tab.

  2. On the PathFinder tab, click the name of the feature you want to reorder.

    Note:

    To find the feature you are interested in, move the cursor over the feature names in PathFinder. Each feature is highlighted in the graphics window when you position the cursor over its name.

  3. Drag the selected feature to its new position.

    Note:

    If you reorder one feature in a way that invalidates other features, the other features are placed on the Error Assistant dialog box.
