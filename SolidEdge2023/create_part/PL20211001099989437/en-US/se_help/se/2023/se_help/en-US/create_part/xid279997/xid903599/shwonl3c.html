Show Only Command (Features)

Shows only the selected elements in the view. You can use this command to quickly hide reference planes, surfaces, and curves that are not the focus of the current task. You can also use this command to hide the solid model. For example, you can select one or more surfaces, then use this command to hide the solid model and all the reference planes.

You cannot use this command to hide individual features on a solid model.
