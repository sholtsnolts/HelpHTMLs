PathFinder tab


Provides alternate ways of viewing part features.

For a more in-depth explanation of the capabilities of PathFinder, see the Using PathFinder in a Part model Help topic.

Note:

When no feature is selected, you can use the commands on the shortcut menu to change how the feature list is displayed. When a feature is selected, you can use the commands on the shortcut menu to delete, rename, suppress, and insert features.

List Area

Displays the features that make up the part in the order the features were constructed.
