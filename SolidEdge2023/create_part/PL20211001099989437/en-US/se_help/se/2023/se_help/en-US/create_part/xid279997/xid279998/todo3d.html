Error Assistant Dialog Box (Exploded Views)

Options
Ground

Grounds the selected part in its current position.

Explode

Runs the Explode command so you can manually explode the part.

Column Names and Functions
Object

Specifies the name of the object that failed to recompute.

Date

Specifies the date when the recompute was attempted (or when a change was made that invalidated the listed object).

Description

Describes the problem that prevented the object from recomputing.

Ground

Grounds the selected object in its current position.

Explode

Runs the Explode command so you can redefine the selected object's parent.

Shortcut menu commands

The following additional shortcut menu commands are available.

Columns

Displays the Columns dialog box so you format the columns. For example, you can specify which columns you want to display.

Font

Displays the Font dialog box so you can format the fonts used in the dialog box.

Sort

Allows you to sort the error descriptions in ascending or descending order.
