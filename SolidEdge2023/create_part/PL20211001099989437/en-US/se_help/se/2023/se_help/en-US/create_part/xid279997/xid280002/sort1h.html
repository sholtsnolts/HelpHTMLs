Sort the entries in a PathFinder collection

You can the sort the entries in a PathFinder collection by entry name or by entry type.

  1. Position the cursor over a collection title, then right-click to display the shortcut menu.

  2. Do one of the following:

    • To sort the collection by entry name, choose Sort by Name.

    • To sort the collection by entry type, choose Sort by Type.
