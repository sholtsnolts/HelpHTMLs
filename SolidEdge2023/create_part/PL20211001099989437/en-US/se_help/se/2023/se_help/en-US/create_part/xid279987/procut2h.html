Construct an extrusion or cutout: subsequent features

You can construct extruded features using the Select tool or the Extrude command. Both workflows are explained in this topic.

Construct an extrusion or cutout using the Select tool

  1. Choose Home tab→Selection group→Select command .

  2. Position the cursor over a sketch region, and then click to select it.

    The extrude handle (A) is displayed.

  3. Position the cursor over the extrude handle, and then click to select it.

  4. Reposition the cursor to define the direction of the material you want to add or remove.

    • If you click away from the model body, material is added.

    • If you click into the model body, material is removed.

  5. Click to both position and finish the feature.

Construct an extrusion or cutout using the Extrude command

  1. Choose Home tab→Solids group→Extrude command .

  2. On command bar, do one of the following:

    • Select the Face option, position the cursor within a sketch region, and then click.

    • Select the Chain option, position the cursor over a connected chain of sketch elements, and then click to accept the selection.

    • Select the Single option, select one or more connected elements, and then right-click to accept the selection.

      The extrude handle (A) is displayed.

  3. Position the cursor over the extrude handle, and then click to select it.

  4. Reposition the cursor to define the direction of the material you want to add or remove.

    • If you click away from the model body, material is added.

    • If you click into the model body, material is removed.

  5. Click to both position and finish the feature.

Tip:

To define draft and crown parameters for the protrusion or cutout, use the Treatment Step options on the command bar. See the help topic, Applying draft angle and crowning.

Note:

To remove material in a multi-body model, see the help topic Multi-body cut.
