Convert ordered features to synchronous features

  1. With no document open, click the File tab located at top-left of the application window .

  2. From the File menu, choose Tools→Convert .

  3. On the Select Documents for Conversion dialog box, from the Look In list, select the folder and document(s) to convert.

  4. Click the Open button.

  5. On the Ordered to Synchronous dialog box, use the Convert column to specify the part and assembly documents to convert.

  6. On the Ordered to Synchronous dialog box, click the Process button to convert the documents.
