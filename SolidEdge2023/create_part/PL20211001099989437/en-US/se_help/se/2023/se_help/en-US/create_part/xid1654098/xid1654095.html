Using Shape Search to locate models

Searching for an existing model, or creating a new model similar to an existing one can be time-consuming. Using Shape Search, you can locate 3D models (JT, part, sheet metal, assembly or weldment) based on the shape of the active or selected model and view the results of the search in either the Geolus web client or Active Workspace.

To use Shape Search:

  • Java (JRE 8 or later) is required. The Java installation location must be defined by the JAVA_HOME environment variable.

  • A Geolus server must be installed and available.

  • If you work in a Teamcenter-managed environment, Teamcenter must be configured to work with Geolus. For example, the preference, GeolusServer is used to specify the URL of the Geolus server.

  • If you do not have content managed by Teamcenter, you must provide the location of the Geolus search server on the Solid Edge Options→Shape Search page.

Note:

Solid Edge file names with non-ASCII characters are not supported by Geolus 10 or older.

For details, refer to Geolus documentation at https://support.industrysoftware.automation.siemens.com/docs/geolus/9/index.htm. A WebKey is required to access the site.

Using Shape Search with Active Workspace

When you use Shape Search in the Teamcenter-managed environment, and your Teamcenter environment is configured to use hosted Active Workspace 4.1 or later with Geolus, Shape Search results display in the Active Workspace docking pane. You can interact with the results of Shape Search in the same way you would with any other item in hosted Active Workspace. For example, you can drag documents shown in Active Workspace into Solid Edge. For the best experience, pin the Active Workspace docking window.

For configuration information, see Active Workspace Help at https://docs.plm.automation.siemens.com/tdoc/aw/4.1/aw_html_collection#search:page=1;rpp=10;query=geolus. A WebKey is required to access the site.

Using Shape Search with a web browser

You can take advantage of Shape Search functionality by viewing the search results in the Geolus web client.

To use Shape Search with the web browser, add the Open in Solid Edge command to the Geolus web client as described in Add Open in Solid Edge to the Geolus web client.
