Display reference elements for a part

Note:

The reference elements that are available are based on whether you are working in the synchronous or ordered environment. For example, reference axes are not available in the synchronous environment.

To

Do this

Display all global reference planes.

Right-click to display the shortcut menu, point to Show All, then click Reference Planes.

Turn off the display of all reference planes.

Right-click to display the shortcut menu, point to Hide All, then click Reference Planes.

Display a local reference plane

Edit the feature that the reference plane belongs to.

Toggle the display of one reference axis on or off

Right-click to display the shortcut menu, point to Show All, then click Toggle Axes. Click the cylindrical face or revolved feature whose reference axis you want to display.

Display all reference axes (see note below table)

Right-click to display the shortcut menu, point to Show All, then click Reference Axes.

Turn off the display of all reference axes

Right-click to display the shortcut menu, point to Hide All, then click Reference Axes.

Display all sketches

Right-click to display the shortcut menu, point to Show All, then click Sketches.

Display one sketch

In PathFinder, select the sketch you want to display, then click the Show command on the shortcut menu.

You can also click the check box adjacent to the sketch to display a sketch.

Turn off the display of all sketches

Right-click to display the shortcut menu, point to Hide All, then click Sketches.

Turn off the display of one sketch

In PathFinder, select the sketch you want to hide, then click the Hide command on the shortcut menu.

You can also click the check box adjacent to the sketch to hide a sketch.

Display all construction curves.

Right-click to display the shortcut menu, point to Show All, then click Construction Curves.

Turn off the display of all construction curves.

Right-click to display the shortcut menu, point to Hide All, then click Construction Curves.

Display all construction surfaces.

Right-click to display the shortcut menu, point to Show All, then click Construction Surfaces.

Turn off the display of all construction surfaces.

Right-click to display the shortcut menu, point to Hide All, then click Construction Surfaces.

Display all coordinate systems.

Right-click to display the shortcut menu, point to Show All, then click Coordinate Systems.

Turn off the display of all coordinate systems.

Right-click to display the shortcut menu, point to Hide All, then click Coordinate Systems.

Display the design body

Right-click to display the shortcut menu, point to Show All, then click Design Body.

Hide the design body

Right-click to display the shortcut menu, point to Hide All, then click Design Body.
Tip:

  • The Show All Reference Axes command does not display the axes of non-revolved features, such as holes, except for axes you have previously displayed with the Toggle Axes command on the Show All shortcut menu.

  • You can also use the Construction Display command on the View tab, in the Show group to control the display of reference elements.
