Unsuppress a feature

  1. In PathFinder, click the Feature Library tab.

  2. On the Feature PathFinder tab, click the feature you want to unsuppress.

  3. Right-click to display the shortcut menu.

  4. On the shortcut menu, click Unsuppress.

    The symbol in PathFinder changes to reflect that the feature has been unsuppressed.
