Select Documents for Conversion dialog box

Look In

Displays the drive and folder in which you want to search for the document or image you want to open

The system stores the last location you browsed in the Look In list.

Go to Last Folder Visited

Returns to the last folder that you visited. This option is not available when working with Teamcenter.

Up One Level

Accesses the parent folder.

Create New Folder

Displays the Create New Folder dialog box, so you can create a new folder. This option is not available with Teamcenter.

View Menu

Controls the display method for the listed documents. This option is not available when working with Teamcenter.

  • Thumbnails—Displays a thumbnail-sized image of the contents of each document or folder. If the folder contains multiple Solid Edge documents with graphical contents, then up to four graphical images are displayed on the folder thumbnail. Click the folder and then click the Open button to see additional thumbnails of individual documents in the folder.

  • Large Icons—Displays large icons to represent each document type.

  • Small Icons—Displays small icons to represent each document type.

  • List—Lists the names of the documents in columns.

  • Details—Displays a detailed view of the folder contents. The columns displayed include Name, Size, Type, and Modified.

Name

Lists the folders and documents you can open.

Preview

Displays a bitmap image of the document if a preview image was saved for the file.

Properties

Displays the properties of the highlighted document in the right pane.

File Name

Specifies the name of the file you want to open.

Note:

In the split message field at the bottom of the dialog box, a message displays the availability of the currently highlighted file. The availability may be: Read Write, or Read Only. The message changes depending on the current status of the file.

Files Of Type

Specifies the type of files that you want to list. Only part and assembly documents can be converted. You can select multiple part documents within a single folder. Only one assembly document can be converted at a time, but the assembly may contain multiple subassemblies. Assembly documents will use links to resolve the location of the component documents so the component documents do not have to be in the same folder as the assembly.

Options

Displays the translation options. This is not available when working with Teamcenter.

Search

Displays the Search dialog box so you can define advanced search criteria for the document you want to open.

If you are working with Teamcenter, the Search dialog box for Solid Edge Embedded Client is displayed.
