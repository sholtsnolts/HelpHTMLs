Rename a feature or an assembly occurrence

Rename a feature

  1. On the PathFinder tab in a part, click the name of the feature you want to rename.

    Note:

    To find the feature you are interested in, move the cursor over the feature names in PathFinder. Each feature is highlighted in the graphics window when you position the cursor over its name.

  2. Right-click to display the shortcut menu.

  3. On the shortcut menu, choose Rename.

  4. Type the new name.

  5. Press the Enter key.

Tip:

You can also select the feature you want to rename in the graphics window.

Rename an assembly occurrence

  1. On the PathFinder tab in an assembly, click the name of the assembly occurrence you want to rename.

  2. Right-click to display the shortcut menu.

  3. On the shortcut menu, choose Rename.

  4. Type the new name.

  5. Press Enter.

Tip:

When you rename an assembly occurrence, the document name is not changed.
