Shape Search command

Searches for models similar to the shape of the active or selected model. You can search based on any 3D model whether it is a foreign body (JT), part, sheet metal, assembly or weldment. Results display in either Active Workspace or the Geolus Search web client.

Note:

Java (JRE 8 or later) is required. Additional requirements are specified in Using Shape Search to locate models.
