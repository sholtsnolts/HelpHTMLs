Add Open in Solid Edge to the Geolus web client

Administrators of the Geolus server can use the following steps to configure the Geolus web client to include the command, Open in Solid Edge.

  1. Determine the values for the protocol, name, port, and context root of the Geolus server.

    <geolus protocol>, <geolus host>, <geolus port>, <geolus>

  2. If you are working in a Teamcenter-managed environment, determine the values for the protocol, name and port of the Teamcenter server.

    <tc protocol>, <tc host>, <tc port>

  3. Generate the custom URL for the Geolus server configuration.

    • For Teamcenter-managed users, replace the entries in the following URL with the information obtained in steps 1 and 2.

      <geolus protocol>://<geolus host>:<geolus port>/<geolus>/geolusCustomAppLauncher?Item ID={01}&ItemRevisionUID={00}&Item Name={02}&Item Revision={03}&Owner Username={04}&Server\='<tc protocol>://<tc host>:<tc port>/'

      Example:

      http://my_geolus:8080/geolus/geolusCustomAppLauncher?ItemID={01}&ItemRevisionUID={00}&Item Name={02}&Item Revision={03}&OwnerUsername={04}&Server='http://my_teamcenter:8080/'

    • If your documents are not managed by Teamcenter, replace the entries in the following URL with information obtained in step1.

      <geolus protocol>://<geolus host>:<geolus port>/<geolus>/geolusCustomAppLauncher?path={<FilePath>}{<DocName>}

      For <FilePath> and <DocName>, enter the generic attribute numbers associated with the attribute names on the Geolus server.

      Example:

      http://my_geolus:8080/geolus/geolusCustomAppLauncher?path={00}{17}

  4. On the Geolus server, run the Server Configuration tool.

  5. Navigate to the Web Page Links page.

  6. Add a new Text Link with the display name of Open in Solid Edge, or your preferred localized display name.

  7. Enter the URL determined in step 3 as the Text Link URL.

  8. Save and then exit the Server Configuration tool.

  9. Replace the settings in the URL below with the values determined in step 1.

    <geolus protocol>://<geolus host>:geolus port>/<geolus>/admin?CMD=REREAD

  10. To reread properties on the Geolus server, in your browser, enter the URL determined in step 9.

    Note:

    You may need to clear your browser cache before the new Open in Solid Edge link is visible in the Geolus web client.

  11. Start the Geolus web client and select the search result you want to open. Click Open in Solid Edge.

    Note:

    The Solid Edge file type .geolusAttributes is downloaded by your browser. Opening this file accesses the document you requested from Geolus. Refer to your browser's documentation to enable automatic opening of this file type.
