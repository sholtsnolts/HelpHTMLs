Investigate Error dialog box

Failure Message

Describes the problem that caused the failure.

Corrective Action

Suggests what actions you can take to correct the error.

Edit Definition

Displays the command bar for the feature so you can edit any of the inputs for the feature.

Edit Profile

Displays the profile for the selected feature so you can edit it. You can perform all profile modification options including adding, deleting and modifying profile geometry, dimensions, and relationships.

Dynamic Edit

Allows you to modify the selected feature without rolling back the model. You can use the Dynamic Edit option to edit the dimensional value of a dimension, drag a dimension handle to reposition the dimension or the dimension text, or to drag an under-constrained profile element's handle to change the physical size and shape of the profile. To add or delete profile elements you must click the Edit Profile button.

You can clear the Enable Dynamic Edit of Profile and Dimensions option on the General tab on the Options dialog box if you want to only allow the dimensional value of dimensions to be edited with this option.
