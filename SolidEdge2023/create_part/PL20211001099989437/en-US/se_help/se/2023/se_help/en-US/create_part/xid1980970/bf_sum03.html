Lesson summary

  • When constructing synchronous parts, you typically use the principal planes of the base coordinate system to draw 2D sketches in 3D space.

  • You can use a coordinate system to position a part in an assembly. You can measure distances relative to a coordinate system with the Measure Distance and Measure Minimum Distance commands. You can display and hide the base coordinate system. Coordinate systems are displayed in the Coordinate System collection in PathFinder.

  • Pierce and silhouette points can assist in connecting curves to off-plane geometry.
