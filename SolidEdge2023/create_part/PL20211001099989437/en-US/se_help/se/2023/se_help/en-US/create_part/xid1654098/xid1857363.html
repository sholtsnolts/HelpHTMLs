Use the Shape Search Batch Indexer

Administrators of the Geolus server can use the following steps to configure the Shape Search Batch Indexer for use in Solid Edge when documents are not managed by Teamcenter.

  1. In Solid Edge, choose File menu→Settings→Options→Shape Search, and type the URL of your Geolus Server.

    When you run ShapeSearchIndexer.exe, the URL of your Geolus Server is displayed at the top of the Solid Edge Shape Search Indexer dialog box.

  2. From the \Solid Edge 2023\Program folder, run ShapeSearchIndexer.exe.

  3. Enter your Geolus web services administrator username and password.

  4. Ensure your attribute mapping file displays in the Attribute Mapping File box.

  5. If you are a Solid Edge Data Management (SEDM) user, select Use Solid Edge Data Management vault location.

    Otherwise, click Add, and then locate the folder that contains the CAD files you want to index.

    The folders you select for indexing are shown in the Folder Path area.

    If you want assembly files to be indexed, select Include assembly files.

    Note:

    By default, all files are indexed except assembly and draft files.

  6. To save your settings, click OK.

  7. To begin indexing, click Index.

    The first time you run the indexer, it indexes all the 3D documents it finds. As you work (add, remove, update files), you must run the indexer again to pick up your changes.

Tip:

You can use the command line to run the indexer unattended.

ShapeSearchIndexer.exe -u=<admin username> -p=<admin password>
