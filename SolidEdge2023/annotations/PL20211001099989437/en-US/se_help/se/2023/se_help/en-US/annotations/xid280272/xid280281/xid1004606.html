Edit associative text in a text box

Use this procedure to edit property text and reference text in a text box.

  1. In a text box, double-click the associative text that you want to edit.

    Tip:

    If you do not know which associative text you want to edit, first right-click the text box and choose Show Property Text Code. This displays the codes associated with all of the property text and reference text, such as %{Datum Frame Text|Ref:A746}.

  2. In the Insert Property Text dialog box, click the property text string. If it is eligible for formatting, the property text string changes to red underlined text.

  3. Click the Format button.

  4. In the Format Values dialog box, select the formatting option you want to apply to the output value or output text. Click OK to close the dialog box and continue.

    The format codes to modify the reference or property text output are shown in the Insert Property Text dialog box.

  5. Click OK to update the text box annotation.

Tip:

  • To hide the property text codes in a text box, right-click the text box and clear the check mark in front of the Show Property Text Code command.

  • You can copy and paste associative text using Ctrl+C and Ctrl+V, but the text then loses its associativity.
