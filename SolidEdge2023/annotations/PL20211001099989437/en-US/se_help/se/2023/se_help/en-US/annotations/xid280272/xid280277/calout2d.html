Preview

Shows or hides the preview window on the right side of the Callout Properties dialog box.

As you make changes to the callout using the options on the dialog box tabs, you can see the effect of your changes before you place the annotation in the document. Both the callout graphics and the extracted text are displayed and updated as you continue to work.

OK

Accepts the current annotation settings and closes the dialog box, so you can place the callout.

Cancel

Cancels the current input and closes the dialog box.
