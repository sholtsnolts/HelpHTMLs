Show/Hide Annotation Alignment Shape command

If annotations have been aligned using the Annotation Alignment Shape command, then the Show Annotation Alignment Shape command displays the geometric shape used to arrange them.

This option is on by default to alert you to the presence of one or more alignment shapes. You can use existing alignment shapes to align newly created annotations and to reposition an existing annotation.

Note:

Alignment shapes do not print.
