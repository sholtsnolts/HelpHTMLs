Technical Requirements Properties dialog box

Tabs

Purpose

General

Defines the content of a technical requirements list. Numbers are assigned automatically. You can reference the numbers from other annotations.

Location

Sets the initial placement options for a technical requirements list based on its border.

Text Format

Primarily specifies the appearance of non-title text. However, the options in the Text Control section also affect the title text.

Title

Specifies the title text and the title appearance for the technical requirements list. There is one title per list.

Indents and Spacing

Adds indentation and spacing between the items in the technical requirements list. The options apply to all lines in the list, not to individual lines.
