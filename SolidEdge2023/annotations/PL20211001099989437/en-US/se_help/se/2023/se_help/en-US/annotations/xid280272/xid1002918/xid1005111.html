Location tab (Technical Requirements Properties dialog box)

Sets the initial placement options for a technical requirements list based on its border.

Note:

All of the options on this tab are stored in Saved settings.

Anchor corner

Specifies a corner of the technical requirements border as the anchor point for placing it on the drawing sheet. The options are Top-Left, Top-Right, Bottom-Left, and Bottom-Right.

This also determines how the list changes size. For example, when the anchor point is on the left, the list expands to the right. When the anchor point is on the bottom, the list adjusts by expanding up.

Enable predefined origin for placement

When this option is selected prior to placing a technical requirements list, you can use the Position coordinates to define where the Anchor corner is located on the sheet.

Position
X origin

Specifies the X coordinate of the anchor corner point on the sheet.

Y origin

Specifies the Y coordinate of the anchor corner point on the sheet.
