General tab (Datum Frame Properties dialog box)

Text

Specifies the letters for the datum frame.

You can create or modify a datum frame name when automatic name generation is not being used.

Shape

Specifies whether the datum frame shape is a rectangle or circle.

Dashes on datum text

When selected, dashes are displayed before and after the datum text, for example, –A–.
