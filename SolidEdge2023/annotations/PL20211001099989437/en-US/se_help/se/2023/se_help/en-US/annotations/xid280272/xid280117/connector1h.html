Add connector annotations

Connectors are a type of annotation that can be added between blocks and other 2D graphic elements to depict relationships in schematic diagrams, flow charts, and other drawings.

  1. Choose Home tab→Annotation group→Connector .

  2. (Specify connector start point) Click a point on the first object that you want to connect to.

    This point can be on a 2D graphic element, on a block or group, on a grid, on another annotation or connector, or in free space. See Tips, below.

  3. (Select shape type) From the Shape list, select a connector shape type that you want to add. Alternatively, press S on the keyboard to scroll through the different shape types.

  4. (Select start and end connector terminators) You can choose different terminator types for each end of the connector.

    1. From the Start list, select a terminator type for the start of the connector.

    2. From the End list, select a terminator type for the end of the connector.

  5. (Adjust the connector segment) Before the second connector point is specified, you can make orientation adjustments to a connector segment.

    • To manually flip a connector, click the Flip button on the command bar or press F on the keyboard. You can flip a Jump, Step, or Corner connector. Also see Auto Flip in Tips, below.

    • To adjust the size of a Jump arc or the span of a Gap connector, type a new value in the Jump radius/gap text box on the command bar and press Enter.

  6. (Specify connector end point) Click a point on the second object to which you want to connect.

  7. Continue adding connectors by clicking subsequent connection points.

    You can adjust the connector shape options on the command bar before you click the next point.

  8. To exit connector placement mode, click the Select tool or press the Esc key.

Tip:
Connection points

  • To maintain connectivity between the connected objects and the connector line segment, the connection points should be keypoints on elements or objects, a point on another annotation, or a point on another connector. The eligible points on a connector include any midpoint or start/end vertex.

  • If you click a free space point for one end of the connector, and then you move the connected object at the other end, the entire connector line segment moves with the object.

Auto flip

  • If you are connecting to a horizontal or vertical line element or connector with a Step or Corner connector, the connector flips automatically so that the first segment of the connector being placed is collinear with the selected line or connector. You can reset the automatic flip using the manual Flip button or F key, and Solid Edge will retain this orientation until you restart the Connector command. For more information about default connector orientation, see the Connector command bar Help topic.

Changing middle handle location

  • To move the midpoint of a Jump arc, a Gap, or a Step, you have to exit connector placement mode. Then select the midpoint handle and drag it along the line segment until it is positioned as needed.

  • You can change the connector line display properties using any of the button settings on the command bar before you place the connector. You also can edit the connector properties in the Connector Properties dialog box after you place a connector.

Editing connector properties

  • You can change the connector line display properties using any of the button settings on the command bar before you place the connector. You also can edit the connector properties in the Connector Properties dialog box after you place a connector.
