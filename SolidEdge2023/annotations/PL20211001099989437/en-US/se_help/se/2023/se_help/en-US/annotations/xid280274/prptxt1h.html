Create property text

These instructions apply to selecting property text to display in annotations, dimension prefixes, drawing view captions, tables, and other objects, such as blocks and block labels.

  1. While creating or editing an annotation, a table title, a block, or a column in a parts list, click the Property Text button in the Properties dialog box or on the command bar.

  2. In the Select Property Text dialog box, select a property text source, such as From Graphic Connection to Part.

    This sets the source, or reference, from which the text will be extracted. It also determines which text properties are displayed in the Properties list. For examples, see Selecting a property text source.

  3. From the Properties list, click the property you want to use, and then click the Select button.

    The correct syntax for the property appears in the Property Text box at the bottom of the dialog box.

  4. Continue selecting properties from the Properties list until all of the properties you want are displayed in the Property Text box.

  5. When you are done selecting properties, do one of the following:

    • Click OK to place the property text at the cursor location in the annotation, table, parts list, or other place of reference.

    • Click Copy to copy the property text to the Clipboard.

Tip:

  • You can select properties from more than one source. You can select In Active Document as the source, for example, and then select the Title and Project properties from the properties list. Then you can change the source to Named Reference and select the Material and Volume properties from the property list.

  • You can edit property text at any time by placing the cursor at the insertion point in the Property Text box and adding or deleting text.

  • You can apply formatting to the evaluated value of a property text string using the Format Values dialog box.

    Example:

    You can use the Format Values dialog box to do any of the following:

    • Specify date and time properties

    • Set precision for numeric properties

    • Choose not to display unit properties

    For more information, see Format property text values.
