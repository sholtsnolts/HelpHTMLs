General tab

The General tab lets you manage formatting for notes, add references and special characters, save them for quick reuse, and preview them.

Saved Notes

Lets you use the saved notes from the Saved Notes list or specify the name for the saved note. When you enter the name for the note, the Save and Delete buttons are activated.

When you save the notes, by default, they are saved with their formatting properties at C:\Program Files\Siemens\Solid Edge XXXX\Templates\Reports\.

Note:

To modify the existing saved notes:

  • Select the note from the Saved Notes list.

  • Modify the note content and its formatting and click Save.

    To save the note with these changes, you can add a new name in the Saved Notes box.

Note

Lets you add the text displayed in the work area.

Reference
Property Text

Selects property text to be displayed in text annotations of parts. For more information about property text, see Using property text.

For a complete list of property text and format codes, see the following topics:

  • Basic property text rules

  • Property text codes

  • Format codes to modify property text output

Insert Character

Provides access to the Microsoft character mapping function, which inserts a character into a text box using a different font. You can also insert special characters that do not have corresponding keys on the keyboard. For more information, see Character Map dialog box.

Select Symbols and Values

Opens the Select Symbols and Values dialog box to generate the appropriate symbols (± (plus minus), ° (degree), and ∅ (diameter)) and model-derived values (hole references, bend data, and weld beads) without having to type the property text codes.

Format Values

Modifies the format of the property text string output value or text at the current cursor position in the notes.

Displays the Format Values dialog box to choose the formatting. You can apply formatting to each valid property text string. The type of formatting you can apply depends on your cursor position when you select the Format button. For property text that extracts the file name or document title, you can change the text case to all UPPERCASE letters. For property text that extracts the date created, modified, saved or originated, you can apply month-day-time formatting.

For more information, see Format property text values.

Special Characters

Adds special font characters, such as diameter, counter bore, or depth, to the text annotation. Hover over a symbol to see what it represents.

Show this dialog when the command begins

Select this check box to display the Text Box Properties dialog box when you select the Text command. When cleared, right-click the text annotation and select Properties.
