Save a feature control frame

  1. Choose the Feature Control Frame command .

  2. On the Feature Control Frame command bar, click the Properties button to open the Feature Control Frame dialog box.

  3. In the Feature Control Frame Properties dialog box, type the name you want in the Saved Settings box.

  4. Set the options you want to save for reuse with the feature control frame.

  5. Click the Save button.
