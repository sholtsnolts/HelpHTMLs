General tab (Edge Condition Properties dialog box)

Saved settings

Specifies the name for a new saved setting. Selecting the arrow lists the previously saved setting names.

Saved settings enable you to save content and formatting properties. You can apply saved settings to quickly format new edge condition symbols and to modify existing edge condition symbols.

Save

Saves the information in the Direction tolerance 1, Upper tolerance, Lower tolerance, Direction tolerance 2, No text on symbol, and All around symbol with leader boxes to the name displayed in the Saved settings box.

Saved settings for edge condition symbols are available for both the Current ISO/DIN standard and the Previous ISO/DIN drawing standards.

Delete

Deletes the saved setting information associated with the name currently displayed in the Saved settings box.

Note:

Saved settings are saved to text files in the ..\Program Files\Siemens\Solid Edge 2023\Template\Reports folder, so that you can share these files with other users. The text files are named for the annotation type, for example, DraftEdgeCondition.txt.

Current ISO/DIN symbol options
Direction Tolerance 1

Sets the direction 1 tolerance value for edge condition symbols.

Upper Tolerance

Sets the upper tolerance value.

Lower Tolerance

Sets the lower tolerance value.

Direction Tolerance 2

Sets the direction 2 tolerance value.

No Text on Symbol

Disables the Direction Tolerance 1, Direction Tolerance 2, Upper Tolerance, and Lower Tolerance boxes on the dialog box. When this option is set, you can only place a basic edge condition symbol.

All Around Symbol

Displays the all around symbol on the leader.

Previous ISO/DIN symbol options
Upper Tolerance

Sets the upper tolerance value.

Lower Tolerance

Sets the lower tolerance value.

Show this dialog when the command begins

When set, displays the Edge Condition Properties dialog box automatically when you select the Edge Condition Symbol command. When cleared, you have to use the Properties button on the command bar to open this dialog box.

The default is to show the dialog box.

Note:

You can place edge condition symbols using the current or previous ISO/DIN standard for edge condition symbols. You set the edge condition symbol standard on the Drawing Standards page of the Solid Edge Options dialog box.
