Automatically add balloons to a part view

This procedure explains how you can add item balloons automatically to a drawing view using the Parts List command with or without adding a parts list.

If you place an auto-ballooned part view without a parts list, you can select the Annotation Alignment Shape command to automatically select and arrange the balloons around the view. To learn how to do this, see Align annotations using the Annotation Alignment Shape command.

  1. Choose one of the following commands from the Home tab→Tables group:

    • Parts List

    • Family of Assemblies Parts List

  2. On the drawing sheet, select a drawing view of a 3D model.

  3. On the Parts List command bar, do the following:

    1. Select the Auto-Balloon option .

    2. Click Properties.

  4. On the Balloon tab (Parts List Properties dialog box), do the following:

    • Specify the auto-balloon appearance and text size.

    • Clear the check box, Create alignment shape.

  5. Do one of the following:

    • To add the balloons with the parts list:

      1. Set the Place List option on the command bar .

      2. Move the cursor until the parts list is positioned as you want it, and then click to place it.

    • To add the balloons without the parts list:

      1. Clear the Place List option on the command bar .

      2. Click anywhere in the graphics window.

Tip:

  • You can reposition the parts list by dragging it to a different location or by moving it to another sheet using the Sheet list on the General tab of the Drawing View Properties dialog box of the selected drawing view.

  • You can save your parts list and balloon formatting for reuse by assigning a name in the Saved settings box on the General tab of the Parts List Properties dialog box.

    You can quickly reapply the settings using the Saved settings list on the Parts List command bar.
