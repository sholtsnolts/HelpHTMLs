Automatically remove centerlines and center marks from a drawing view

  1. Choose Home tab→Annotation group→Automatic Centerlines .

  2. On the Automatic Centerlines command bar, set the options you want.

  3. Select the Remove Lines and Marks option on the command bar.

  4. Click the drawing view.

Tip:

  • All automatically-created centerlines and marks are removed from the drawing view.

  • You can also delete individual centerlines and marks that were originally placed automatically using the Delete command on the shortcut menu.
