Add technical requirements to a drawing

  1. Choose Home tab→Annotation group→Text command list →Technical Requirements .

  2. On the command bar, click Properties.

  3. On the General tab (Technical Requirements Properties dialog box), do the following:

    1. In the large input box, type the text of the first technical requirement, and then click Insert.

      This transfers the information from the input box to the first blank row in the Content list.

    2. Repeat the previous step for each line of information that you want to enter in the technical requirements list.

      Tip:

      • To insert property text, reference text, symbols, and values, use the buttons in the Reference section.

      • To copy and paste information into the input box, use Ctrl+C and Ctrl+V.

    3. From the Style list, select the alphanumeric system you want to use.

      Example:

      Plain numbers (1, 2, 3, ...)

      Uppercase Roman numerals (I, II, III ...)

      Lowercase Roman numerals (i, ii, iii ...)

      Uppercase English letters (A, B, C ...)

      Lowercase English letters (a, b, c ...)

    4. From the Format list, choose how you want the numbers in the list to be formatted.

      Example:

      1

      1.

      1)

      (1)

    5. (Optional) Adjust the order of the items in the list by selecting a row and then clicking Move Up or Move Down to move it.

  4. Click the Title tab (Technical Requirements Properties dialog box), and then do the following:

    1. In the Title text box, type the title text as you want it to appear on the top line of the technical requirements list.

    2. From the Justification list, choose an option to apply to the title.

  5. Click the Location tab (Technical Requirements Properties dialog box), and then choose an option from the Anchor corner list.

  6. If you want to be able to reuse the technical requirements list, click the General tab, type a name in the Saved settings box, and then click Save.

  7. Click OK to close the dialog box, and then click the drawing sheet where you want to place the list.

Tip:

  • When you click a valid property text or reference text string in the input box, you can use the Format button to apply formatting to the evaluated output value. For more information, see Format property text values.

  • You can make text larger by typing a value that is greater than 1.00 in the Text size box on the command bar.

  • You can choose an option from the Text Control list on the command bar to control how lines of text fit within the border around the technical requirements list. If you choose the Fixed—Wrap Text option or the Fixed—Adjust Aspect Ratio option, you can drag the edit handle to change the width of the technical requirements list.

  • To modify an existing technical requirements list, right-click the list and choose Properties.

  • If you make any changes to the information or formatting or style of an existing technical requirements list using the Technical Requirements Properties dialog box, be sure to click the Update button to apply the changes back to the list on the sheet.

  • To create a second technical requirements list, you have to change to a different working sheet.
