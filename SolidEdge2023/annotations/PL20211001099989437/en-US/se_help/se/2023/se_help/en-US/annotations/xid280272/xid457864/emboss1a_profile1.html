Creating profiles with text

Use the Text Profile command in part, sheet metal, assembly, and draft to draw a profile or sketch that includes text. The Text dialog box is where you create the text and specify its characteristics, such as the font and size. You can use the Text Profile command bar to choose an anchor point that provides the best alignment of the text.

When creating text features that go all the way through a part, such as with sheet metal parts, you should use the Solid Edge Stencil font. This font was designed for these situations, and ensures that the part can be manufactured.
