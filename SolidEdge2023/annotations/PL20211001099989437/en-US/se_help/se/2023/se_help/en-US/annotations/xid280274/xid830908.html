Referencing properties

You can create associative references to text, document and model properties, and values in Solid Edge. You also can insert symbols into text fields.

You can create references to these properties using the following command buttons, which are located on various dialog boxes. You also can type property text directly into some text fields.

Property Text

Select Property Text dialog box—Accesses the general properties in the current document, as well as the properties in models attached to the current file.

Example:

File property text
%{Revision number}

Select Symbols and Values

Select Symbols and Values dialog box—Inserts annotation and dimension symbols directly into an annotation or dimension. You also can insert references that derive their values by property text to extract bend data, hole data, weld beads, and drawing views.

Example:

Symbols and values in the form of 2-letter property text codes.
%FL

Reference Text

Select Reference Text dialog box—Extracts text fields from one annotation or drawing view and displays them in another annotation.
Example:

A specific text field in a drawing view caption, such as
%{Drawing View Scale|Ref:A962}.

Format

Format Values dialog box—Applies formatting to the extracted property text output value.
Example:

Formatting instructions that specify the title text is all uppercase.
