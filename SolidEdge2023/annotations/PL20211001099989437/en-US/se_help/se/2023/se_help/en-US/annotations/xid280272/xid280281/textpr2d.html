Info tab

The Info tab applies character and line spacing when creating or modifying text.

Sheet

Shows which drawing sheet an element is on.

Layer

Shows which layer an element is placed on in a drawing sheet. This option is available only in the Draft environment.

Origin X, Y

Specify the coordinates, or location, of a text box along the X and Y axes.

Angle

Specifies the angle of the text box. For example, if you enter 90, then the text box is rotated 90 degrees counterclockwise.

Text Orientation

Sets the orientation of the text in an annotation.

Horizontal

Specifies that the text is oriented horizontally in the document.

Vertical

Specifies that the text is oriented vertically in the document.

Vertically oriented text is displayed so that it can be read from top to bottom. Each character in the text box is displayed on a line by itself.

When you use vertically oriented text, you should adjust the Text Control setting to ensure that the text is displayed vertically, as intended, and does not wrap or scale.

Text Control

You can select one of the following options to specify how text size is managed within the text box border.

Fit width to content

Automatically sizes the width of the text box to its contents.

Note:

When you select the Text command and then click and type, the text box is defined as Fit width to content by default.

Fixed Width, Height

Specifies the exact width and height of the text box. Text size is adjusted automatically to maintain the fixed width and height using one of the options below.

Note:

When you select the Text command and then click and drag to draw the box, it is defined as Fixed Width and Height by default.

Automatically adjust aspect ratio

When one or more characters exceed the specified width, adjusts the width of all of the characters in the text, without changing their height.

Calculated aspect ratio

Shows the actual aspect ratio used to maintain the width. It is displayed for reference only.

Wrap text

Automatically word-wraps the text that extends beyond the specified line width. A new line is started each time the characters in a word exceed the width of the text box.

Set Textbox as Watermark

Changes the textbox into a watermark.
