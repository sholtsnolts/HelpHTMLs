Line Up Text command bar

Select Base Element

Defines the linear dimension or annotation with which the other text elements are lined up.

Vertical Left

Lines up the left edges of the text elements vertically, using the left-most point of the range box of the base element. Left-justifies the range boxes.

Vertical Center

Lines up the centers of the text elements vertically, using the midpoint of the range box of the base element. Center-justifies the range boxes.

Vertical Right

Lines up the right edges of the text elements vertically, using the right-most point of the range box of the base element. Right-justifies the range boxes.

Horizontal Top

Lines up the top edges of the text elements horizontally, using the topmost point of the range box of the base element. Top-justifies the range boxes.

Horizontal Middle

Lines up the centers of the text elements horizontally, using the midpoint of the range box of the base element. Center-justifies the range boxes.

Horizontal Bottom

Lines up the bottom edges of the text elements horizontally, using the bottommost point of the range box of the base element. Bottom-justifies the range boxes.

Vertical Break Point

Aligns the leader break points of the selected elements vertically. If a leader does not have a break line, then the point where the leader attaches to the element is used for alignment.

Horizontal Break Point

Aligns the leader break points of the selected elements horizontally. If a leader does not have a break line, then the point where the leader attaches to the element is used for alignment.

Align Offset

Defines an offset distance with which the text elements are aligned.
