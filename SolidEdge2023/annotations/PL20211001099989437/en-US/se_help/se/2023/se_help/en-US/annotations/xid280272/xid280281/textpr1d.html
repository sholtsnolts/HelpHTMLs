Border and Fill tab

The Border and Fill tab is available in the Text Box Properties dialog box and the Watermark Properties dialog box when creating or modifying an annotation created with the Text command or the Watermark command.

Border

Specifies how a border appears around an annotation.

Show Outline

Displays the outline of the text box.

Type

Specifies the border type for the text box.

Color

Specifies the color of the border.

Width

Specifies the border width.

Note:

You must set Display As Printed on the View tab of the Solid Edge Options dialog box to display a text box border greater than 2 mm.

Offset

Specifies how much the border is offset from the text inside a text box.

Fill Color

Fills the text box boundary with the selected color. Transparent is the default.

You also can review and change the fill color using the Text command bar.

Shadow

Places a shadow around the borders of text in a text box.
