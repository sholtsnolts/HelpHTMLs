Insert a font character into a text box

You can create a new text box and add some text to it, or you can edit an existing one.

  1. Use the Text command to place a text box or text string. Type the text that you want it to contain.

  2. On the Text command bar, click Insert Symbols, Characters, or Property Text .

  3. From the list, select Insert Character .

  4. In the Character Map dialog box, from the Font list, select the font you want to use.

  5. In the font character grid, click the character you want to insert.

  6. Click Select.

    Tip:

    You also can double-click the character in the grid to add it to the Characters To Copy box without having to click Select.

  7. Continue to click the characters you want, clicking Select after you click each character.

  8. In the Character Map dialog box, click Copy to copy the characters to the Clipboard.

  9. On the drawing sheet, click inside the text box where you want to paste the characters.

  10. Press Ctrl+V to paste the characters into the draft document.

    Tip:

    • The Paste command does not work when pasting Unicode characters from the Clipboard.

    • If you do not see the character you are looking for, you can click the Advanced View option in the dialog box and then use the additional filter selection and search features to find it. To learn how to use these options, click the Help button on the Character Map dialog box.
