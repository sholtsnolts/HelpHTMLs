Line Up Text command

The Line Up Text command lines up text elements with a base text element that you choose. You can specify alignment and justification of the text, as determined by its range box, or alignment of the leader break points.

You can use the Line Up Text command to align linear dimension text and annotations, such as balloons, callouts, datum frames, edge condition symbols, feature control frames, surface texture symbols, text boxes, and weld symbols.
