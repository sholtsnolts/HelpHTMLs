Reference an annotation from a technical requirements list

When creating or editing technical requirements, you can reference information in another annotation.

Example:

When authoring the technical requirement, Machine parts #XX and #YY separately, you can reference balloon item numbers to fetch the correct part numbers for XX and YY. When a balloon item number changes due to parts list renumbering, the technical requirement also updates.

  1. Right-click a technical requirements list and choose Properties.

    The General tab (Technical Requirements Properties dialog box) is displayed.

  2. In the Content section, click the row that you want to edit.

    This displays the existing information in the edit box at the top of the General tab.

  3. Position your cursor within the edit box where you want to insert associative reference text, and click the Reference Text button .

  4. In the Select Reference Text dialog box, do the following:

    1. From the Sheet list, select the draft sheet where the annotation is located.

    2. From the Source type list, choose the annotation type, such as Balloon.

    3. From the Field list, choose the text field within the annotation that you want to reference.

      Example:

      To reference a balloon item number, you can choose the Upper or Lower fields.

    4. In the table grid, click a row and then see what is highlighted on the sheet. If this is what you want to reference, click the Select button.

      Repeat this step for each additional balloon item number that you want to insert into the same technical requirement.

    5. Click OK to close the Select Reference Text dialog box and continue.

  5. On the General tab in the Technical Requirements Properties dialog box, verify that the reference text is inserted where you want it, and then click the Update button.

    This updates the information in the Content box in the technical requirements list.

    Example:

    If the current balloon item numbers are 10 and 20, then the technical requirements list displays this:

    Technical Requirements

    (1) Machine parts #10 and #20 separately.

    If the balloon item numbers change, then the part numbers in the technical requirements list update, too.

Tip:

If you are editing a saved technical requirements list, click Save to update the Saved settings before you close the Technical Requirements Properties dialog box.
