Centerline command bar

Dimension Style Mapping

Specifies that the setting on the Dimension Style tab of the Options dialog box determines the dimension style. When you set this option, Dimension Style is unavailable.

Dimension Style

Lists and applies the available dimension styles. This option is unavailable when Dimension Style Mapping is enabled.

Properties

Opens the Center Line & Center Mark Properties dialog box.

Line

Specifies a straight line as the centerline type.

Arc

Specifies a curved line as the centerline type.

By Center Axis

Places a centerline using a 3D axis on a cylinder or hole feature. For more information, see Placing a center axis on a cylinder or hole feature.

Placement Options

Specifies lines, arcs, or key points as the connection elements for the centerline. The available placement options are based on the centerline type selected.

  • By 2 Lines—Places a Line centerline midway between two lines.

  • By 2 Points—Places a Line centerline midway between two points.

  • By 2 Arcs—Places an Arc centerline midway between two concentric arcs.

  • By Center Point—Places an Arc centerline by specifying a center point, a start point, and an end point.

Note:

After placing an Arc centerline, you can use its edit handles to adjust the sweep of the centerline.
