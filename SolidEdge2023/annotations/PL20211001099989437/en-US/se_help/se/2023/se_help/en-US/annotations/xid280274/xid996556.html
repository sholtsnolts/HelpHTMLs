Selecting a property text source

When inserting property text using the Select Property Text dialog box, use the following table to determine which Source type to select. This is based on where you want to use the property text and what type of information you want to extract.

Reference a specific part, component, or subassembly within a top level assembly.

Named reference to the model file currently selected in the Parts pane in the Select Property Text dialog box.

Retrieve the File name property for a specific part, component, or subassembly and display it in a drawing view caption. For an example, see Create assembly part views with the View Wizard.

When you add and edit property text in a user-defined table, select the Named reference source to reference custom properties, file properties, and variables exposed in the Variable Table in the model document.

Title blocks in a multi-sheet drawing showing a different model on each sheet.

Index reference, which is based on the sequence of model drawing views created in the draft document, as shown in the Parts pane.

When predefining the title block on each sheet, reference the unique File name property for the model shown on the first sheet, and then the model shown on the second sheet, and then the model shown on the third sheet.
