Place a datum target

  1. Choose the Datum Target command.

  2. On the Datum Target command bar, click the options you want to use.

  3. Click an element or click in free space.

  4. Click where you want to place the datum target.

Tip:

  • You can select the Use Drawing View Scale option on the command bar to associate datum area position and datum area size with a selected drawing view scale. This ensures that the datum area moves and scales with the drawing view.

  • To place a model PMI annotation that references multiple elements instead of a single element, click Referenced Geometry on the command bar. To learn how to use this option, see Select multiple reference elements.
