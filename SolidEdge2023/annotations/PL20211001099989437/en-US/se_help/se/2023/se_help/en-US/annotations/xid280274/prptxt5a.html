Property text from drawing views

In the Select Property Text dialog box, when the property text source is From drawing view, you can choose from the list of properties defined in this table. You can retrieve the derived property text values into a drawing view caption, for example, or a callout.

Property text string

Definition

%{Annotation Sheet Number|DV}

Annotation Sheet Number—For a given drawing view, displays the sheet number where the corresponding view annotation exists.

%{Primary Scale|DV}

Primary Scale—For a given drawing view, displays the first number in the drawing view scale.

%{Secondary Scale|DV}

Secondary Scale—For a given drawing view, displays the second number in the drawing view scale.

%{View Angle|DV}

View Angle—Displays the rotation angle for the drawing view.

%{View Sheet Number|DV}

View Sheet Number—For a given view annotation, displays the sheet number where the corresponding drawing view exists.

View annotation types include cutting plane, viewing plane, and detail envelope.
