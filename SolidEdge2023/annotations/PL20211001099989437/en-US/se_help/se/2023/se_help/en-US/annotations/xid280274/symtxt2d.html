Select Symbols and Values dialog box

The Select Symbols and Values dialog box selects, previews, and inserts annotation symbols and model-derived values directly into an annotation or dimension, without having to type the property text codes yourself.

The symbols and values in the Select Symbols and Values dialog box are cross-referenced in the property text codes tables.

Note:

The content of this dialog box varies with annotation type. For some annotations, it contains a comprehensive list of drawing symbols and model-derived values, such as the values extracted for holes, bends, weld beads, and drawing views. For text box annotations, only symbols are available.

The list of values shown in this dialog is the complete list. All the values in the list are not available for all annotations. The values list is not reduced by the annotation edited when the dialog was entered. Example: Display of the Weld Bead values is limited to the GOST weld symbol. Callouts do not have access to these values. These values display blank in callouts.

Symbols and Values

Lists available symbols and model-derived values, organized into one or more of the following categories. You can expand the individual categories by clicking the + button. You can expand all categories at once by pressing Shift while clicking the + button.

Symbols

The Solid Edge fonts—ANSI GDT Symbols, ISO GDT Symbols, and GOST1 Symbols—are the source of the symbols in these categories.

  • Geometric Characteristics

  • Material Condition

  • Tolerance Zone

  • Dimensioning

  • GOST Weld Symbols

  • Other Symbols—This category includes the Centerline, Parting line, and Oval, as well as the ESKD and GB drawing standard symbols for view rotation.

Values

Lists the model- or drawing-derived data that you can extract into the annotation or dimension.

  • Feature Reference

  • Bend

  • Drawing View—This category includes view scale, angle of rotation, and drawing view sheet number.

  • Weld Bead

  • Wiring Harness

Other

Inserts miscellaneous characters, such as a hard carriage return into a block of text.

Copy

Copies the property text codes displayed in the Text box to the clipboard. You can then paste the contents into another application, such as Microsoft Word or Excel.

Select

Adds the property text code of the selected symbol or value to the Text box. You also can double-click a symbol or value to add it to the Text box.

Text

Shows the current string of property text codes that have been added using the Select button or by double-clicking a symbol or value in the list.

Preview

Provides a graphical preview of the symbols you have selected. For property text based values, the Preview box displays angle brackets around the content to be derived from the model when the annotation or dimension is placed.

Example:

Symbol code preview: ±

Value code preview: <Bend Radius>

OK

Copies the property text codes currently displayed in the Text box to the current cursor location.

For example, if you opened the Select Symbols and Values dialog box from a callout or from a text box created with the Text command, then the property text codes are copied to the cursor position in the annotation.
