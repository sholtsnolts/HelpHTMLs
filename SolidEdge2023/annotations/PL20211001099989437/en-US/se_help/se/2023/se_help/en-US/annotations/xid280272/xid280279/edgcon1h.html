Place an edge condition symbol

  1. In a draft document, choose the Home tab→Annotation group→Edge Condition command.

  2. In the Edge Condition Properties dialog box, specify the options you want for the symbol.

  3. In the graphics window, select the element on which you want to place the terminator end of the symbol.

  4. In the graphics window, click in space to position the symbol.

Tip:

  • You can place edge condition symbols using the current or previous ISO/DIN standard for edge condition symbols. You set the edge condition symbol standard on the Drawing Standards page of the Solid Edge Options dialog box.

  • To place a model PMI annotation that references multiple elements instead of a single element, click Referenced Geometry on the command bar. To learn how to use this option, see Select multiple reference elements.
