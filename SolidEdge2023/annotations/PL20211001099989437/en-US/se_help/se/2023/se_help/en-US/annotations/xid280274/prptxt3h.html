Convert all property text in a drawing or model

  1. Choose Home tab→Property Text group→Convert All

    Tip:

    If you are working with PMI elements, this command is on the PMI tab.

  2. In the displayed dialog box, click Yes.

All property text in the current document is converted into plain text.
