Character Map command

Provides access to the Microsoft character mapping function, which inserts a character into a text box using a different font. You can also insert special characters that do not have corresponding keys on the keyboard. You can use this command when you type or edit text in a text box.

How do I
Place a text box or text string
Create a text box that references another annotation
Edit associative text in a text box
Insert a font character into a text box
Set text annotations parallel to the screen
