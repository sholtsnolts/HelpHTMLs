Select Reference Text dialog box

You can use the Select Reference Text dialog box to extract text from annotations for the purpose of inserting it into other annotations. The reference text updates whenever the source text changes.

Not all options are available in all environments.

Sheet

Selects the 2D Model sheet, a drawing sheet, or a background sheet that contains the information you want to reference.

Source type

Specifies the source of the reference text. The Source types and Field names that you can reference are listed in the following table.

Displays the Primary caption text and Secondary caption text that is defined on the Caption tab in the Drawing View Properties dialog box.

Scale

View Scale displays the composite scale value, for example 1:2. Primary Scale displays the numerator in the scale ratio; Secondary Scale displays the denominator in the scale ratio.

View Annotations

On a drawing view where a clipping plane, auxiliary view plane line, or detail envelope is defined, identifies the view annotation label name, the sheet number where the view annotation is located, and the sheet number where the derived view is located.

Note:

The selected Source type controls all of the available options in the lists below.

Field

Specifies which text field can be extracted. It also populates the Source column and the Value column in the table grid in the dialog box.

Source and Value

Lists all available object names and values based on your Sheet, Source type, and Field selections.

If there are no values for a selected Field item, then the Value column remains blank. For example, if you are referencing balloon suffix text, but there is no suffix defined in any balloon in the document, then the Value column is blank.

Note:

  • You can select items in the Source column to see the item you are referencing highlighted in the draft document or in the graphics window (for PMI).

  • You can use shortcut menu commands to sort the contents of a column and to find a specific value or object name. You also can double-click a column header to sort its contents.

Copy

Copies the content displayed in the Reference text box to the clipboard.

Select

Does all of the following:

  • Displays a preview of the evaluated text in the Preview pane.

  • Adds the Source and Value text selected in the table grid to the Reference text box at the bottom of the dialog box.

Note:

You also can double-click the reference text source to add it to the list.

Preview

Displays the selected reference text as it will appear in the annotation.

Reference text

Shows the reference text string (in property text format) to be resolved. If multiple reference text values are selected, they are displayed in the box on one line. You can separate them with spaces or other punctuation.
