Center Line and Center Mark Properties dialog box

This dialog box modifies the display properties of a selected centerline or center mark. The options available are based on document type (draft or model), annotation type, and whether dimension mapping is applied.

Dimension style

Lists and applies the available dimension styles.

For center marks, the dimension style defines line type, line width, line extension, and center mark size. For centerlines, the dimension style defines line type and line width.

Color

In a draft document, shows the annotation color based on the dimension style.

In a model document, shows the color of PMI annotations as defined by the Sketch setting on the Colors page of the Solid Edge Options dialog box.

Line type

Lists and applies the available line types.

Line width

Sets the line width.

Line extension

Specifies the line extension length.

Center mark size

Specifies the height and width of a center mark. This value is a ratio of the dimension text size.

Text scale

In a draft document, text scale is applied to the value in the Center mark size box to specify the size of center marks on the current sheet. The default value is 1.0.

Note:

In a model document, the size of PMI is controlled by the Model Size PMI and Pixel Size PMI options on the PMI tab on the ribbon. For more information, see Change PMI text size.
