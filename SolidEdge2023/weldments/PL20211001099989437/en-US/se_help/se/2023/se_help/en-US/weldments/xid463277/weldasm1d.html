Weldment Assembly Dialog Box

Mark This Assembly As a Weldment

Specifies that the assembly document is considered a weldment. When you set this option, the Bead Material, Bead Density, Bead Style, and Construction Style options are activated.

Component Material

Allows you to type the component material name you want to use. The Component Material name you type can be used in the Draft environment with the property text functionality to read the component material name you specify. For example, you can use the Callout command in the Draft environment to place the property text string for Material: %{Material|R1}, in the title block for a welded assembly drawing. The name you type in the Component Material box is then inserted in the title block.

This can be useful for companies that specify the component material name in the welded assembly drawing using property text. For more detailed information on property text and callouts, see the Using Property Text and Callout command Help topics.

Bead Material

Allows you to type the bead material name you want to use.

Bead Density

Allows you to type the bead density value you want to use. The bead density value you type is used when performing a physical properties calculation for the assembly using the Physical Properties command. The bead density is used for weld bead features and protrusion features you construct.

Bead Style

Sets the bead face style. You can set the bead face style on this dialog box, or on the Color Manager dialog box.

Construction Style

Sets the construction face style. You can set the construction face style on this dialog box, or on the Color Manager dialog box.
