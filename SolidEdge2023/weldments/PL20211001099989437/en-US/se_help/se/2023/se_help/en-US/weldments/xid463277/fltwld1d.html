Fillet Weld command bar

Steps
Select Base Step

Defines the faces you want to add to the first select set. The faces can be from one or more components. Faces on existing fillet weld features are valid for selection.

Select Target Step

Defines the faces you want to add to the second select set. The faces can be from one or more components. Faces on existing fillet weld features are valid for selection.

Finish/Cancel/Preview

This button changes function as you move through the feature construction process. The Finish button constructs the feature using input provided in the other steps. Once you construct the feature, you can edit it by re-selecting the appropriate step on the command bar. The Cancel button discards any input and exits the command.

command bar Options
Fillet Weld Options

Displays the Fillet Weld Options dialog box so you can define the characteristics of the weld.

Select

Allows you to specify the selection criteria. For this feature, you can specify whether individual faces or chains of faces are selectable.

Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selection.

Name

Displays the feature name. Feature names are assigned automatically. You can edit the name by typing a new name in the box on the command bar or by selecting the feature and using the Rename Feature command on the shortcut menu.

Select Base Step Options
Base Thickness

Sets the thickness of the fillet weld on the base thickness side.

Select Target Step Options
Target Thickness

Sets the thickness of the fillet weld on the target thickness side. You can set different values for the base and target thickness. This option is available when you set the Type option on the Fillet Weld Options dialog box to Unequal Setbacks.
