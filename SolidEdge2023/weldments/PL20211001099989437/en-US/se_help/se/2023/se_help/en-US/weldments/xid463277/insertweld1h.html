Insert a weldment in an Assembly

  1. Click Home→Insert→Insert Weldment.

  2. On the Insert Weldment command bar, click the Select Step button.

  3. On the Select File dialog box, select the weldment file you want to copy as the base feature and click Open.

  4. On the Insert Weldment command bar, click the Parameters Step button.

  5. On the Weldment Parameters dialog box, specify the parameters for the part copy and click OK.

  6. On the Insert Weldment command bar, click Finish.
