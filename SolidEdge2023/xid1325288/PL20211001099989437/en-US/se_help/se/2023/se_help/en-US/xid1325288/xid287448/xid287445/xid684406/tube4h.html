Create a bend table (XpresRoute)

  1. In XpresRoute, on the Tools menu, click Bend Table.

  2. Do one of the following:

    • If you want to select all tubes in the file, click the Select All Tubes button.

    • If you want to select an individual tube in the files, click the tube.

  3. Click the Accept (check mark) button or the Finish button on the command bar.

  4. On the Bend Table dialog box, in the Save In box, enter the folder for the output files and click the Save button.
