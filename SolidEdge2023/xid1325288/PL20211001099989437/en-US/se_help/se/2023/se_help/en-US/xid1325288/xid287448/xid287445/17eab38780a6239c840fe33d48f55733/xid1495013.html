Redefine a tube path

  1. Select the tube in the top level assembly.

  2. Click the Edit Definition button.

  3. On the Tube command bar, click the Deselect button .

  4. Select the new path for the tube.

  5. On the Tube command bar, click the Accept button and then click the Finish button.

  6. Close XpresRoute.

    The tube is created along the new path.
