Piping Report dialog box

Retrieves and displays information for pipes and fittings in the assembly document. Use the Format button to define the report format.

You can display the Report Output dialog box by clicking the OK button.

Note:

The actual report output dialog box name is based on the report type you select and the assembly document name.

Among

Defines which pipes you want to include in the report:

  • All the pipes in the assembly

  • Only the currently selected pipes

Pipes

Specifies the type of piping report you want to generate.

Individual Cut Lengths

Groups pipes with the same diameter, type, and length together and displays the quantity for each group in the reports.

Total Length

Groups all pipes with the same diameter and type together and sums up the total lengths for each group of pipes.

Fittings

Determines whether or not the pipe fittings should be included in the report.

Include Fittings

Specifies that you want to include the fittings for the selected pipes in the report.

Use assembly generated item numbers

When checked, specifies that item numbers displayed in the report are derived from the item numbers in the assembly structure. When unchecked, numbers are generated on-the-fly by the assembly Reports command.

Note:

The Item Numbers page (Solid Edge Options dialog box) controls:

  • Whether item numbers are generated in the assembly document

  • The starting number and the increment value

  • The item numbering schema, for example, top level, atomic, and exploded.

Format

Displays the Format Report dialog box for the report type you specified.
