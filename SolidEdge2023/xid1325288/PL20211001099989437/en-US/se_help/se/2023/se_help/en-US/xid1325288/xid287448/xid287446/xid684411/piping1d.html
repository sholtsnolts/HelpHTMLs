Piping Options dialog box

Saved settings

Lists saved pipe settings, such as pipe size, standard, class, and end treatment. You can access the saved settings by selecting them from the list. The settings on the dialog box display the characteristics of the pipe you select. You can type a name in the box to name a group of settings.

Save

Saves the current settings to the name you type.

Delete

Deletes the saved settings selected in the Saved Settings box.

Pipes

Specifies information about the fittings you want to place in the pipe.

Pipe Location

Specifies that you want to automatically select and orient fittings based on the selected piping route and the attributes associated with the fittings.

Select from Standard Parts Library

Specifies that you want to select the pipe from the Standard Parts Library.

Browse for Pipes

Specifies that you want to select the pipe from a managed or unmanaged location that is outside the standard parts library.

Select File

Displays a dialog box to browse for the pipe location.

Recently used

Displays a list of recently used pipe files.

Standard

Specifies the standard to be followed when creating the pipe. This list is based on the list of fittings stored in the standard parts library.

Diameter

Specifies the diameter for the pipe. This list is based on the list of fittings stored in the standard parts library.

Material

Specifies the material for the pipe. This list is based on the list of fittings stored in the standard parts library.

Fittings

Specifies information about the fittings you want to place in the pipe.

Fittings Location

Specifies that you want to automatically select and orient fittings based on the selected piping route and the attributes associated with the fittings.

Select from Standard Parts Library

Specifies that you want to select the fitting from the standard parts library.

Browse for fittings

Specifies that you want to select the fitting from a managed or unmanaged location that is outside the standard parts library.

Automatically place fittings

Specifies that you want to automatically select and orient fittings based on the selected piping route and the attributes associated with the fittings.

Verify availability of fittings

Specifies that you want to search the database based on the specified fitting standard and diameter for each entry in the Pipeclass.txt file. Only entries that contain fittings based on the defined search criteria appear in the fittings class list.

Select preferred class fitting

Displays the Standard Parts dialog box so you can select a preferred fitting from the available fittings in the standard parts library. If there are multiple fittings available, the fittings appear in a list and you can chose the fitting you want.

Standard

Specifies the standard to be followed when placing the fitting. This list is based on the list of fittings stored in the standard parts library.

Class

Specifies the class to be followed when placing the fitting. This list is based on the list of fittings stored in the standard parts library.

Threaded fittings

Specifies that the fitting end conditions are threaded.

Socket weld fittings

Specifies that the fitting end conditions are socket welded.

Butt weld fittings

Specifies that the fitting end conditions are butt welded.

Flanged fittings

Specifies that the fitting end conditions are flanged.

Pipe penetration cuts

Specifies that the fitting end conditions are pipe-to-pipe penetrations.

Companion

Displays a list of companion flange fittings that you need to place with flanged fittings. This list is based on the list of fittings stored in the standard parts library.

When you place a flanged fitting or replace a non-flanged fitting with a flanged fitting from the standard parts library, the Companion Flanges dialog box appears so you can select the companion flange you want to place.

When you replace a flanged fitting with a non-flanged fitting, the companion flanges associated with the fitting are automatically removed.

After you place a companion flange, you can select the flange and replace it with another companion flange.

Note:

You can use the Orientation option on the Piping command bar to rotate a companion flange.

Show this dialog when the command begins

Specifies that you want to display the dialog box every time you start the command.
