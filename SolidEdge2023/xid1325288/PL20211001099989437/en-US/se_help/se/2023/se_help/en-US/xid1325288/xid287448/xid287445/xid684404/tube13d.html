End Treatment Options dialog box

Saved settings

Saves the End Treatment options settings to the C:\Program Files\Siemens\Solid Edge 2023\Preferences\Custom.xml file. The name entered in the saved settings field is the SavedEndTreatmentForTube attribute name in Custom.xml.

End Treatment Type

Specifies the type of end treatment you want to apply. The list includes None, Expand, Reduce, Close, and Flange.

Inside Diameter

Specifies the inside diameter of the end treatment. This option is not available if the End Treatment Type is set to Reduce.

Outside Diameter

Specifies the outside diameter of the end treatment. This option is only available if the End Treatment Type is set to Reduce.

Depth

Specifies the depth for the end treatment. This option is not available if the End Treatment Type is set to Close or Flange.

Angle

Specifies the angle of the end treatment. This option is not available if the End Treatment Type is set to Close.

Radius

Specifies the radius of the end treatment. This option is not available if the End Treatment Type is set to Close.

Preview

Displays a bitmap image of the end treatment.

Save Default

Saves the current settings for later use.
