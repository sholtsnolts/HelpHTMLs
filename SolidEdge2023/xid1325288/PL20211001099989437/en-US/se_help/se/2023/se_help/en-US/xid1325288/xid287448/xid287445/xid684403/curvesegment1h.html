Create a curve segment

  1. Choose Home tab→Segments group→Curve Segment .

  2. Select the path segments that you want the curve to follow.

  3. Select the Curve Fit for the curve to follow.

  4. Select the path.

  5. Click the Accept (check mark) button on the command bar.

  6. Click the Preview button on the command bar.
