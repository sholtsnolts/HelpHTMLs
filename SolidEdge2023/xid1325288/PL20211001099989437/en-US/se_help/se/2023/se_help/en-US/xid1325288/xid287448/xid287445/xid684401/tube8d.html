PathXpres command bar

Previous

Displays the previous path choice.

Next

Displays the next path choice. The order of the paths goes from the simplest path, with the least number of segments, to the most complex path. The maximum number of segments in a path that PathXpres generates is five.

Cancel/Finish

Cancels the command or finishes the feature.

Port Segment Length

Specifies the length of the line segment at the end of the ports.
