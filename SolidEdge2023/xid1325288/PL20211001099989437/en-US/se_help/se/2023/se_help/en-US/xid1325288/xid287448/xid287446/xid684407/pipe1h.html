Create a piping route

When creating a piping route, you can:

  • Place pipes and fittings located in a standard parts library.

  • Place pipes and fittings located outside a standard parts library.

    Note:

    You can also combine these two methods. For example, you can place a pipe from a standard parts library and the fittings from a location outside the library or vice versa.

  • Place pipes and fittings along a gradient pipe route.

Note:

For Solid Edge data management users who selected the option Automatically name files via Document Number and Revision, the system displays a dialog box where you can assign common properties, such as document number and revision number, as explained in Create documents with unique properties.

Place pipes and fittings located in a standard parts library

  1. While working in XpresRoute, choose Home tab→Piping group→Piping Route .

  2. On the Piping Options dialog box, select the Select from Standard Parts Library options for both Pipes Location and Fittings Location.

  3. Set the other options on the dialog box to place the appropriate pipe and fittings.

  4. Select the path to define the piping route.

  5. Click the Accept button .

  6. Click the Preview button.

  7. Click the Finish button.

Place pipes and fittings located outside a standard parts library

  1. While working in XpresRoute, choose Home tab→Piping group→Piping Route .

  2. On the Piping Options dialog box, select the Browse for Pipes and Browse for Fittings options.

  3. On the Piping Options dialog box, click the Select Folder button .

  4. On the File Open dialog box, select the pipe part you want to place and click Open.

  5. On the Piping Options dialog box, click OK.

  6. Select the path to define the piping route.

  7. Click the Accept button .

  8. Select the keypoint to place the fitting.

  9. On the File Open dialog box, locate and select the fitting you want to place, and then click Open.

    The fitting is placed at the selected keypoint.

  10. Repeat the previous two steps until all fittings are placed.

  11. Click the Preview button.

  12. Click the Finish button.

Place pipes and fittings along a gradient pipe route

  1. While working in XpresRoute, choose Home tab→Piping group→Piping Route .

  2. On the Piping Options dialog box, select the Select from Standard Parts Library options for both Pipes Location and Fittings Location.

    Note:

    You can also place the pipes and fittings from a location outside the standard parts library.

  3. Set the other options on the dialog box to place the appropriate pipe and fittings.

  4. Select the path to define the piping route.

  5. Click the Allow Gradient button .

  6. Type a value for the maximum gradient you want to allow.

    The maximum value you can enter is 20 degrees.

  7. Click the Accept button .

  8. Click the Preview button.

  9. Click the Finish button.
