Curve Segment command bar

Main Steps
Path Step

Specifies the segments for the curve segments to follow.

Preview/Finish/Cancel

This button changes function as you move through the feature construction process. The Preview button shows what the constructed feature will look like, based on the input provided in the other steps. The Finish button constructs the feature. After previewing or finishing the feature, you can edit it by re-selecting the appropriate step on the command bar. The Cancel button discards all input and exits the command.

Curve Fit

Specifies how the keypoints are extracted from the selected segment. These keypoints are used to create the curve. The curve always starts and ends at the start and end points of the first and last segments. The midpoints of the first and last segments are never used and the curve is always tangent to the first and last segments in the path.  The keypoint option setting does not affect this process.

  • Mid Points—Uses the mid points of the input segments to create the curve.

  • End Points—Uses the start points of the input segments to create the curve.

  • All Points—Uses start points and mid points of the input segments to create the curve.

Select

Sets the path segment selection criteria.

  • Chain—Allows you to select a chain of path segments.

  • Single—Allows you to select a single path segment.

Accept (check mark)

Accepts the selection.

Deselect (x)

Clears the selection.

End 1

Controls the magnitude of tangency for the first endpoint.

End 2

Controls the magnitude of tangency for the second endpoint.
