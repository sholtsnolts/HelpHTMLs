Create a pipe report

  1. In XpresRoute, choose Tools tab→Assistants group→Pipe Reports.

  2. On the Piping Report dialog box, specify which pipes you want to include in the report. You can include all pipes in the assembly or only the pipes you have selected.

    Note:

    To create a report on selected pipes, you must select the pipes before selecting the Pipe Reports command.

  3. On the Piping Report dialog box, select the report type you want. You can create a report that displays the individual cut lengths or a report that displays a total length for the cuts.

  4. On the Piping Report dialog box, select the Include Fittings option if you want to include the fitting for the selected pipes in the report.

  5. Click the Format button.

  6. On the <Report Type> Format dialog box, select the formatting options you want. For example, you can choose a font, set the justification, or display a grid around the report.

  7. Click the OK button. A report output dialog box is displayed. The dialog box title is based on the report type and document name.

  8. On the report output dialog box, select the output option you want. For example, you can print the report, save the report as a document, copy the report to the Clipboard, or create a new report.

Tip:

  • You can add and remove column titles based on available properties using the Options button on the Format Report dialog box.

  • You can also create a report from Windows Explorer, without running Solid Edge. Right-click the assembly document and select the Reports command on the shortcut menu. The report includes all parts in the selected assembly. If the assembly has been converted to an alternate assembly, you can use the FOA Member Names dialog box to specify which member you want to create a report for.
