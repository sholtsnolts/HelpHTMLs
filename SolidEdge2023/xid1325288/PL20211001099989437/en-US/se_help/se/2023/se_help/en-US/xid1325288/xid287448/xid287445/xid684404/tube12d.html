Tube command bar

Main Steps
Path Step

Specifies the 3-D path that you want to create the tube along.

Extent Step

Defines the extent of the tube ends.

Preview/Finish/Cancel

This button changes function as you move through the feature construction process. The Preview button shows what the constructed feature will look like, based on the input provided in the other steps. The Finish button constructs the feature. After previewing or finishing the feature, you can edit it by re-selecting the appropriate step on the command bar. The Cancel button discards all input and exits the command.

Path Step Options
Tube Options

Accesses the Tube Options dialog box.

Name

Specifies the name of the tube part.

Select

Sets the path segment selection criteria.

  • Chain—Selects a chain of path segments.

  • Single—Selects a single path segment.

Extent Step Options
End 1

Defines the extent for the first end of the tube. You can increase the extent with a positive number or decrease the extent with a negative number.

End 2

Defines the extent for the second end of the tube. You can increase the extent with a positive number or decrease the extent with a negative number.

End Treatment Options

Accesses the End Treatment Options dialog box.

Other command bar Options
Accept (check mark)

Accepts the selection.

Deselect (x)

Clears the selection.
