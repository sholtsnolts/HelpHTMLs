Curve Segment command

Note:

To access XpresRoute, click Tools→Environs→XpresRoute.

Creates a 3D curve based on a set of endpoint-connected path segments. The curve is always tangent to the first and last segments in the select set and it passes through the first and last points of the path.  There is an option that allows you to use the midpoints, endpoints, or all points of the input segments.
