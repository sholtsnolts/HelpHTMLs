Mesh Options dialog box (mesh modeling)

Use the options on the Mesh Options dialog box to control how a model document is translated to a mesh format when you use the Convert to Mesh Body command .

Tolerance Options

These options are used to specify the tessellation tolerance when converting the solid model into a mesh.

Smaller conversion tolerance values and smaller angles result in greater accuracy of the whole part, at the expense of larger file sizes.

Custom

Select this option to specify custom conversion tolerance and surface plane angle values. You can enter a value or use the Conversion tolerance or Surface plane angle sliders to change the values.

Fine

Selecting this option sets the Conversion tolerance to .1 mm (.004 inches), and the Surface plane angle value to 30 degrees.

You cannot change the values for the Conversion tolerance or Surface plane angle.

Coarse

Selecting this option sets the Conversion tolerance to 2.5 mm (.098 inches), and the Surface plane angle value to 60 degrees.

You cannot change the values for the Conversion tolerance or Surface plane angle.

Conversion tolerance

Use this slider to dynamically adjust the value for convergence tolerance. It is only available when the Custom tolerance option is selected.

Move the slider to increase or decrease the conversion tolerance shown in the Conversion tolerance box. You can also manually enter a specific value into the field.

  • When the Conversion units option is set to Millimeters, the valid range is from .100 mm to 2.5 mm.

  • When the Conversion units option is set to Inches, the valid range is from .004 inches to .98 inches.

Note:

If a value is entered and the field turns yellow (above 2.5 mm or .98 inches), a conversion is attempted but may not be successful.

Surface plane angle

Use this slider to dynamically adjust the value for the Surface plane angle. It is only available when the Custom tolerance option is selected.

Move the slider to increase or decrease the value shown in the Surface plane angle box. You also can manually enter a specific value into the box.

Selected tolerances are the maximum value the user allows the software to use for the conversion. Conversion Surface plane angle may use a smaller value on specific facets to mesh the geometry.  If you define 60 degrees, it can use 45, 59, or 60, but not 61. 

  • The slider range for the surface plane angle is from 1 to 60 degrees.

  • You can type a value greater than 60 degrees and less than 90 degrees. The field will turn yellow, but the conversion is still attempted.

  • If you enter a value of 90 degrees or greater, the field turns orange and a conversion is not attempted.
