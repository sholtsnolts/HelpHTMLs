Deviation Analysis command bar

Settings

Displays the Deviation Analysis Settings dialog box so you can control the settings for the deviation analysis.

Select Reference Object

Specifies the faces, regions, features, or bodies you want to use as reference objects for analysis.

Select Test Object

Specifies the faces, regions, features, or bodies you want to use as test objects for analysis.

Results

Displays the deviation analysis results on the model.

Cancel/Preview/Finish

This button changes function as you move through the feature construction process. The Finish button constructs the feature using input provided in the other steps. After you construct the feature, you can edit it by re-selecting the appropriate step on the command bar. The Cancel button discards any input and exits the command.

Select

Specifies the type of element you want to select.

  • Face

  • Feature

  • Body

Accept

Accepts the current input.

Deselect

Deselects the current input.

Show Max and Min markers

Displays the maximum and minimum value markers on the model.

Show Results Information Dialog

Displays the Deviation Analysis dialog box that contains information from the analysis, such as the minimum deviation, maximum deviation, and total number of samples. The information is available in text so you can copy it and use it in reports.
