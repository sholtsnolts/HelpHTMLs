Create a sketch from a section curve

  1. Use the Surfacing tab→Curves group→Section Sketches command to draw a sketch from a section curve.

  2. Select a body or multiple bodies.

    Note:

    Mesh bodies must be divided into regions or faces for sketch entities to be recognized.

  3. Right-click to accept.

  4. Click a reference plane or planar face.

  5. On the Section Sketches command bar, enter the number of planes and the offset distance.

  6. Use the handle to specify the direction for creating the plane pattern.

  7. Right-click to accept, and then on the Section Sketches command bar, click Finish.

    8. Note:

    The sketches are not automatically constrained. You must create the constraints manually before using them to create B-rep models or bodies.
