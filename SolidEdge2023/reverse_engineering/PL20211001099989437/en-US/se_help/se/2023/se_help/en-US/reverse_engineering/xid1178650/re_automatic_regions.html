Automatic regions command

The features and options of the Automatic regions command are used to identify the geometric regions of the mesh file. The different types of geometric regions are identified by color.

The type of geometric construction surface extracted from a contiguous region is based on the color assigned to a region.

The Automatic regions command bar has the following options:

Accuracy

Adjust the accuracy associated with identifying the different types of regions in the mesh file by increasing or decreasing the numerical value. After adjusting the value, select Accept to view the results. Repeat this process to adjust the region assignment until you are satisfied with the results. Then select Accept again to assign your selection.

Undo

If you do not want the assigned selection, select Undo to restore the original mesh.

Accept

This option serves two functions. After assigning an Accuracy value, select this option to view the results. To re-evaluate the identified regions, change the Accuracy value and select this option again. Repeat this process to identify the most desired results. To assign the regions, select this option again. Regions can be further refined using the Manual regions command and the Fit command.

Deselect all

Select this option to deselect all selected mesh facets.
