Manual regions command

Use the Manual regions command to manually identify and refine the extent of a geometric region of the mesh file by painting a selected color over sub-mesh areas, or filling in an entire sub-mesh with a selected color.

If a mesh area is painted one of the pre-defined region colors, the Extract command will attempt to extract the region based on the applied color.

If a non-recognized color is painted onto a mesh, no assignment is made during extraction. A geometric region can then be applied and resolved later using the Fit.

If a white color is assigned to a mesh surface, the mesh area is designated as non-extractable.

In either instance, geometric surfaces can be assigned to the un-extracted mesh areas using the Fit command.

The Manual command bar has the following options:

Paintbrush

Paints the selected color over areas of sub-meshes. The painted color is selected using the Pick Color option or the Color option.

Fill

Fills the entire sub-mesh with the selected color. The fill color is selected using the Pick Color option or the Color option.

Pick Color

Picks a color from already identified regions. Selecting a recognized color will result in attempting to assign the default geometry associated with the selected color when the surfaces are extracted.

Color

Opens a color picker dialog box that is used to select a color to use for the Paintbrush or Fill tools. Pick a recognized region color to attempt to identify an area as a known geometric region type. All other colors are ignored during the Extraction process.

Non Extractable

Used to paint selected sub-mesh areas white which is a non-extractable color. Geometric surfaces can be assigned to the un-extracted mesh areas using the Fit command.

Undo

If you do not want the assigned selection, select Undo to restore the mesh region to its initial color.

Accept

Select Accept, right click or press Enter to accept your changes.

Reset

Select Reset to reset the all assigned colors.
