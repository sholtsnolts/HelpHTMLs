Remesh Options dialog box

Minimum face quality

Specifies the face quality of the triangle. Values range from 0 to 1 with 0 producing the worst quality and 1 producing perfect quality. The default value is .05, with recommended values being .01 to .10.

Surface growth rate

Specifies the rate at which a face edge size varies from one face to adjacent faces. You will only see this effect when there is a transition from an area of refinement to the rest of the surface mesh.

The default value is 1.3, which means that a face edge can be 1.3 times its adjacent edge length when growing from a short to a long edge length.

The recommended range for the surface growth rate is 1.1 - 1.6.

Sharp edges angular tolerance

Sets the angle tolerance for determining the sharpness of an edge.

Compatibility refinement

When selected, imposes a surface growth rate of 2 to limit the difference in face sizes that extend across a gap. If the gap between the faces is small, the relative size of the faces on either side of the gap is close to one another.

Proximity refinement

Specifies a mesh size that is based on the gap size and the number of points inside the gap.

Reset

Resets the values for the dialog box.
