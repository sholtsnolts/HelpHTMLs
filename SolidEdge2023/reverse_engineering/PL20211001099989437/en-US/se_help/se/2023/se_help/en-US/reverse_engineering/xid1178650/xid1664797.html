Align a mesh body to a coordinate system

You can align a mesh body by specifying the "From" coordinate system in the following ways:

  • Using principal axes

  • Using a best fitting bounding box

  • Using a custom fitting bounding box

  • Using geometry references

Use principal axes to align a mesh body

  1. Choose Reverse Engineering tab→Cleanup Mesh group→Align .

  2. Select the body or bodies to align.

  3. On the Align command bar, click Accept .

    A coordinate system is created at the center of mass of the selected body.

    Note:

    You can use the keyboard shortcuts to orient the coordinate system.

  4. On the Align command bar, set the From Coordinate System Creation option to Use Principal Axes.

  5. Click Accept .

  6. Click the coordinate system to which you want to align.

  7. Click Accept .

  8. Click Finish.

Use a best fitting bounding box to align a mesh body

  1. Choose Reverse Engineering tab→Cleanup Mesh group→Align .

  2. Select the body or bodies to align.

  3. On the Align command bar, click Accept .

  4. On the Align command bar, set the From Coordinate System Creation option to Use Best-fit bounding box.

    A bounding box that fits the body is created.

  5. Place and orient the coordinate system on the vertex, edge, or face of the bounding box.

    Note:

    You can use the keyboard shortcuts to orient the coordinate system.

  6. Click Accept .

  7. Click the coordinate system to which you want to align.

  8. Click Accept .

  9. Click Finish.

Use a custom bounding box to align a mesh body

  1. Choose Reverse Engineering tab→Cleanup Mesh group→Align .

  2. Select the body or bodies to align.

  3. On the Align command bar, click Accept .

  4. On the Align command bar, set the From Coordinate System Creation option to Use Custom bounding box.

    In this example, we are using the Coincident Plane option.

  5. Click a plane to define the orientation of the bounding box.

  6. Place and orient the coordinate system on the vertex, edge, or face of the bounding box.

  7. Click Accept .

  8. Click the coordinate system to which you want to align.

  9. Click Accept .

  10. Click Finish.

Use geometry reference to align a mesh body

  1. Choose Reverse Engineering tab→Cleanup Mesh group→Align .

  2. Select the body or bodies to align.

  3. On the Align command bar, set the From Coordinate System Creation option to Use Geometry References.

  4. Click a point on the mesh body to define the origin of the coordinate system.

  5. Click a point on the mesh body to define the X-axis vector.

  6. Click a point on the mesh body to define the Y-axis vector.

  7. Orient the coordinate system to the appropriate position. You can use the shortcut keys to control the orientation of the coordinate system.

  8. On the Align command bar, click Accept .

  9. Click on the coordinate system to which the body will align using the "From" coordinate system.

  10. Click Accept .

  11. Click Finish.
