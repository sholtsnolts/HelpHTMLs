Replace duplicates in multiple assemblies

If you found a duplicate file in one assembly, you can verify if that file exists in other assemblies, and fix the duplicate in those assemblies as well.

  1. In the Design Manager document list, select the file whose duplicates you just found.

  2. Choose Home tab→Assistant group→Where Used .

  3. Click Next, and then click Finish.

  4. In the document list, right-click the file and choose Update Document.

  5. Click Perform Actions .
