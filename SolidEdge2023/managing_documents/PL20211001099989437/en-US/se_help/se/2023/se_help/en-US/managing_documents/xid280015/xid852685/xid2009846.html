Revert documents to the version in Teamcenter

The Revert command returns a document to the latest version in Teamcenter. You can select single or multiple documents. Selecting an assembly reverts the entire structure. When no document is selected, the entire assembly is closed and reopened using the version from Teamcenter.

Note:

To use the Revert command, the assembly must be saved to cache, but not uploaded to Teamcenter.

  1. Open an assembly in Solid Edge.

  2. Make changes to the design.

  3. Save the assembly, but do not upload it to Teamcenter.

  4. Select the part or subassembly that you want to revert to the latest version in Teamcenter.

    Note:

    If no documents are selected, the direct document is reverted.

  5. From the Teamcenter ribbon, click Revert.

    The check-out of the selected document is cancelled and the version from Teamcenter is reloaded into the assembly. Reverting the direct document closes the direct document and reloads the latest version from Teamcenter.
