Assigning common properties

Solid Edge displays this common properties dialog box because you selected the option Automatically name files via Document Number and Revision. For more information, see Creating documents with unique properties.

Paste

Pastes the contents of the clipboard to the selected cell.

Cut

Cuts the selection and places it on the clipboard.

Copy

Copies the selection to the clipboard.

Find

Finds text, or finds and replaces text.

Columns

Shows or hides columns, changes column order, or sets column width. The list of columns includes the registered custom properties you defined in your templates or in the propseed.txt file. It does not include the properties defined in the Custom tab of the Document Properties dialog box.

Auto Fit

Automatically fits the column width to the longest string in each column.

Browse

Displays a dialog box where you can select a new location.

Assign All

Automatically generates and assigns a document number. The system uses the prefix you defined in the Document Naming Rule tab of the Solid Edge Options dialog box→Manage tab, and the number from the NumberGenerator.dat file. For more information, see Manage tab, Solid Edge Options dialog box.

Validate

Verifies that your entries are correct. If the system identifies an error, it displays an error icon in the Issues column. You can hover over the icon to see the associated error message.

Save

Saves the new document.

When saving a new draft file in Solid Edge data management, the common properties dialog box displays the 3D file along with the draft file. Saving the draft file commits the changes to the draft and to the 3D file.
