Basic window operations

In Structure Editor, you see a hierarchical structural view of the item you have selected.

The window is divided into four panes.

  • The upper-left pane of the window displays your item using the Revision Rule you selected when you opened the item. This pane is sometimes referred to as the source pane.

  • The upper-right pane of the window initially displays your item in the same view as the source. However, this pane reflects any changes you make to the item while in Structure Editor. You can consider it the target pane.

  • If an assembly is saved with a preview, the preview image is displayed in the lower-left pane. It can be thought of as the preview pane.

  • The lower-right pane displays property information from the database and is considered the properties pane.

You can expand each level of an item using the + to the left of the item or you can use the Expand All command on the View menu to expand all levels.

Right-clicking an occupied row or cell in the grid displays a shortcut menu to change the font, sort the properties, find and replace text, format columns, change column width, and hide columns.

Additionally, you can use commands on the View menu to turn on the Preview and Properties display.

Tip:

Turning off the preview and properties panes enhances the performance of the top panes of the Structure Editor.

You can minimize an open Structure Editor window and display it as an icon in the application work space.
