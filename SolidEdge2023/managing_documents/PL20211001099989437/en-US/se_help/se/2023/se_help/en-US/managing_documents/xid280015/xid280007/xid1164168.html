Revise command (SEEC)

Displays the Revise Document dialog box that enables you to create a new revision for the direct document or documents within a selection set. The next revision is automatically presented for you.

Translated files (Parasolid, STEP) that are part of the active document's dataset are not saved to new documents created with the Revise command.
