Select files

If you want to select a group of files based on file types or actions, use the Select Files command.

When you select an assembly or subassembly, Design Manager does not select its children. Therefore, any actions you perform apply only to the selected document. To perform an action on the entire assembly, use the Select All command to select all documents in the display list. To perform an action on a set of documents, multiple-select those documents.

  1. Choose Home tab→Select group→Select Files .

  2. To select all files of a given file type, in the File(s) box, enter the file type of interest, and then click Select.

  3. To select all files of a given file type and with actions pending in the Action column, in the File(s) box, enter the file type of interest, and then click With Actions.

  4. To clear the selection of all files of a given type, click Deselect.
