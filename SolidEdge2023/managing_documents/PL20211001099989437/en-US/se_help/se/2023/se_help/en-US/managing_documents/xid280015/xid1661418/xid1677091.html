Extract translated files from Teamcenter

Translated files stored in Teamcenter can be collected for sharing with vendors. You can collect these files while working in Solid Edge, or while no document is open.

Note:

The Teamcenter preference, SEEC_Save_Translation determines the behavior for translated files. For a full description of the preference, see the Teamcenter Preferences section of the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.

  1. Do one of the following:

    • With a Teamcenter-managed document open in Solid Edge, open the Extract Translated Files pane from the View tab→Show group→Panes list.

    • With no document open, choose File menu→Manage tab→Extract Translated Files.

  2. If you have a Solid Edge document open, select any objects you want to have considered during the extraction process.

    The extraction of files includes both the selected objects and the objects you specify using the options in the docking pane.

    Note:

    When no document is selected, the direct document is considered.

  3. In the Expand Selection section, select the check boxes that correspond to the objects you want included in the list of Item Revisions to be extracted.

    Note:

    • If an assembly is selected, objects in the assembly can be included using the option, Include components from selected assemblies.

    • When the option to include part copy is selected, source components are included.

  4. Filter the objects in your list of Item Revisions by selecting the Solid Edge file types you want included in your list of objects to be extracted.

    For example, if you only want part documents extracted from Teamcenter, click .

  5. Further reduce the number of Item Revisions in your list by setting additional filters on the items in the selection set based on translation file type, relation type, and reference type.

    For example, set the file type for filtering to STEP documents (*.step; *.stp), the relation type to IMAN_manifestation, and the reference type to Image.

  6. Browse to the unmanaged folder where you want the files extracted.

  7. Click Extract.

    You can repeat the extraction process multiple times if necessary to obtain the desired translated documents.

    Tip:

    Extracting the translated files does not create a package or zip of the files. It only places the files in the location you specify.
