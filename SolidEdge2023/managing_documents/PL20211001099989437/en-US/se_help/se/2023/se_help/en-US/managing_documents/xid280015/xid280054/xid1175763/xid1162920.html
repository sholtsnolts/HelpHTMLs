Create a package of the cache workspace

Note:

Prior to creating a package of files to send to a collaboration partner, check out the documents you intend the design partner to modify.

  1. On the Cache Assistant ribbon, click Create Package.

    Note:

    This command is enabled using the Teamcenter preference SEEC_Enable_CreatePackage.

  2. Browse for the unmanaged folder location where you want to save the compressed file that contains the contents of the current cache workspace.

  3. Define the file name for the compressed file.

    Note:

    A file compression utility such as 7-Zip or WinZip is required.

  4. Click Create.

    The resulting compressed file can be sent to a collaboration partner.

    Use Add to Teamcenter-Interactive to reimport the data to the Teamcenter-managed environment. Documents that were checked out before package creation are automatically set to overwrite after being modified by the design partner.

Tip:

You can view a summary of the contents of the workspace and what was written to the compressed file by clicking the Packaging Activity Log command.
