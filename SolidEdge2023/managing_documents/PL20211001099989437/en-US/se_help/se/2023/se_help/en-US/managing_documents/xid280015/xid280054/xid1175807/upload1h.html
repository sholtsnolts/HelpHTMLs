Upload a document

Saves the selected document to a document library, but keeps it checked out to you. The command is not available if the document is not checked out.

From the File menu:

  • Choose Manage→Upload.

In a Teamcenter-managed document:

  • In PathFinder, select the document you want to upload to Teamcenter.

  • Choose Teamcenter tab→Manage group→ Upload.

    • If no files are selected in PathFinder, the active document is uploaded.

    • If any files are part of a select set, all selected files are uploaded.
