Cache Assistant command

Manages your local cache. You can remove documents you no longer need, check out documents from the managed location, check in any documents you have checked out, synchronize any out-of-date documents, or load a document from the managed location.

How do I
Switch to Offline Mode
Filter the display of cache contents
Change the default Teamcenter project
Change User Group or Role
Create a new cache location
Modify cache workspace description
