Filter the display of cache contents

  1. In the Cache Assistant dialog box, click inside the cell in the first row of the document list you want to filter by and display the available filter values.

    Note:

    If no value has been set, All Values is displayed.

  2. Select a value from the list you want to filter by.

    Tip:

    • Setting the value for all columns to All Values displays all documents in the cache.

    • Setting the Cache Status to Checked Out To You to displays documents you have checked out, including documents with a status of Modified as documents have to be checked out to be modified.

    • Setting the Cache Status to Checked Out To Others displays all documents checked out to someone other than yourself.
