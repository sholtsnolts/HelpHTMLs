Working with hosted Active Workspace

After you define your Teamcenter environment to use an Active Workspace enabled server, hosted Active Workspace is available in the quick access toolbar in Solid Edge , or from the File menu→Manage page.

Use Active Workspace to search for your content. Then you can use additional capabilities such as:

  • Open in Solid Edge

    The object is opened in Solid Edge using the Revision Rule displayed in Active Workspace or the current saved setting for that document type.

  • Open

    The Open File dialog box lets you select the options for opening the document in Solid Edge. The results display in the Look In list in the folder Open in Solid Edge.

  • Drag a component into a Solid Edge document.

  • Copy and paste a component into a Solid Edge document.

  • Access Teamcenter information about your direct document using the Teamcenter Information command available in the Teamcenter tab → Active Workspace group on the Solid Edge ribbon.

  • Use cross-highlighting functionality to select an object in Active Workspace and have the corresponding object selected in Solid Edge.

    Use the Select in Solid Edge and Select in Active Workspace commands on the Teamcenter tab → Active Workspace group of the ribbon in Solid Edge to highlight content on-demand.

    Note:

    Automatic highlighting must be disabled to use these two commands.

    Automatic highlighting from Solid Edge to Active Workspace and Active Workspace to Solid Edge creates bi-directional cross-highlighting between the two products. This functionality is enabled or disabled using options in the Hosted Active Workspace cross highlighting section of the Assembly tab, Manage page (Solid Edge Options) dialog box. You can also set the option to Automatically expand PathFinder and scroll to the first object in the select set to locate the object in PathFinder.

    Tip:

    For bi-directional selection to be most effective, similar BOMs should be displayed in both Solid Edge and Active Workspace.

    Follow the steps in Customize the command ribbon to add the commands Auto-Select in Active Workspace and Auto-Select in Solid Edge to your Teamcenter command ribbon.
