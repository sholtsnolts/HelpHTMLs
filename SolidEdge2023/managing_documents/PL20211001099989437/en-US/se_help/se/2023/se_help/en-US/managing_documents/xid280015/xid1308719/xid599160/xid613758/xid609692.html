Query dialog box

Finds documents using the search criteria you define.

Query name

Specifies a name for the query.

Query scope

Defines the scope of the query. For example, you can specify that the query scope includes all the parts in the assembly, only parts that are currently displayed, and so forth.

Search subassemblies

Specifies that you also want to search subassemblies within the active assembly.

Load subassemblies

Specifies that you want to load and search any hidden subassemblies.

Query Criteria

Defines the specific query criteria you want to use.

Property

Lists the properties you can search against. Examples include Address, Filename, Item ID, Item Type, and others.

Condition

Defines the conditional argument for the property being searched. For example, you can define a query where the Condition option is set to contains and the Property option is set to Filename and the Value is set to steel. All parts whose filename property contains the word steele is selected. You can choose the conditions: contains, is (exactly), or is not.

Value

Defines the value to search for in the specified property. You can type any text you want to search for. You can search for multiple values in one operation by separating the values with a semicolon (;). For example, you can search for a Item ID property of either 0008 or 0009 by typing: 0008; 0009.

Add to List

Adds the specified search criteria to the Find Items That Match These Criteria list.

Find items that match these criteria

Displays the criteria to be used in the search.

Remove

Removes selected entries from the Find Items That Match These Criteria list.
