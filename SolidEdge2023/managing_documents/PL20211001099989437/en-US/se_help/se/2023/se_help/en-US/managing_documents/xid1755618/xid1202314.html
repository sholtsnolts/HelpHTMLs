Check in documents

Use the Check In command to release the checkout lock, allowing other users to modify the document.

The Check In command applies the Available status to documents that have In Work status.

  1. In PathFinder, select the document you want to check in.

  2. On the Data Management tab, click Check In .

    If you turned on the display of document status, the PathFinder displays:

    • The Available icon to the left of the document name.

    • The document name in black text.

    • When you hover over a checked-in document, the system displays additional check-in information, such as the check-in date and the name of the user who checked in the document.
