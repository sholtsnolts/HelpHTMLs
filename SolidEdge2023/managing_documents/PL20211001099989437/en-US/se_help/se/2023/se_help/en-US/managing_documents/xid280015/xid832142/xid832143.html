Defining the Teamcenter Item type list

Common property dialog boxes within the Teamcenter-managed environment display Item Types based on your configuration of the following Teamcenter preferences.

  • SEEC_ItemTypeList_SE Part

  • SEEC_ItemTypeList_SE Assembly

  • SEEC_ItemTypeList_SE Draft

  • SEEC_ItemTypeList_SE SheetMetal

  • SEEC_ItemTypeList_SE Weldment

  • SEEC_ItemTypeList_SE Component

  • SEEC_ItemTypeList_SE Connection

  • SEEC_ItemTypeList_DirectModel

  • SEEC_ItemTypeList_Image

  • SEEC_ItemTypeList_MSExcel

  • SEEC_ItemTypeList_MSExcelX

  • SEEC_ItemTypeList_MSWord

  • SEEC_ItemTypeList_MSWordX

  • SEEC_ItemTypeList_PDF

  • SEEC_ItemTypeList_StructureEditor

  • SEEC_ItemTypeList_UGMASTER

By default, each preference has a default value of Undefined which provides Item as the only available Item Type.

You can control the list of Item Types displayed by editing the preferences’ values. For each preference, provide a list of Item Types you want to use to manage the Solid Edge Dataset type. For example, use SEEC_ItemTypeList_SE Part to define the Item Types you want available in the Solid Edge Part environment. Some examples of most commonly used Item Types are: Item, Design, Document, and Drawing.

For more information on the preferences, see the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.
