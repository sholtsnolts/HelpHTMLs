Find a part using Query

  1. With a document open in Structure Editor, select Tools→Query.

  2. Set the Property option you want to query for. For example, you can set the property to Action, Checked Out By, Dataset Description, and so forth.

  3. Set the Condition option you want. You can set the condition to Contains, Is (Exactly), or Is Not.

  4. Type the value of the property you are searching for.

  5. Click Add to List to enter the information in the Find Items That Match These Criteria portion of the dialog box.

    The parts that match the query properties you defined are highlighted in the Structure Editor window.

Tip:

You can use the Scope dialog box to specify whether you want to search subassemblies.
