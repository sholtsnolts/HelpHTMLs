Where Used Results dialog box

Displays where selected document(s) are used in other documents.

You can right-click on an occupied row in the grid to display a shortcut menu that allows you to do such things as change the font, format columns, sort the properties, expand the where-used list, open and preview documents in the where-used list, and perform a where-used search on unmanaged documents in the list. You can double-click near a column divider to do a best fit on the grid. Any changes you make to the columns are stored for future sessions.

Note:

Because you cannot make changes to the data in the list, the Save, Cut, Copy, Paste, and Restore commands are disabled on this dialog box.

Save

Saves changes you have made on the Where Used Results dialog box.

Print

Sends a copy of the active document to a specified plotter, printer, or file. Options are available for defining the printing area, range, number of copies, and other printing characteristics.

Before using this command, you must install and select a printer. For help installing a printer, see the Windows documentation.

Cut

Cuts the current selection and pastes it into the Clipboard.

Copy

Copies the current selection to the Clipboard.

Paste

Inserts the Clipboard contents at the current cursor location.

Restore

Restores the information to the previous saved configuration. A dialog box is displayed to confirm that you want to discard the changes you have made.

Select by Type

Displays the Select by Type dialog box, in which you can specify the file types to select in the Where Uses Results dialog box.

Use Precise

Finds the BOM view revision where the current document is placed precisely.

Revision Rule

Displays where the current document is used based on the revision rule selected. This option is not available if you have selected the Use Precise check box.
