Find and replace duplicates in an assembly

  1. In the Design Manager document list, select the Solid Edge file or files for which you want to check for duplicates.

  2. Choose Home tab→Assistant group→Find Duplicates .

    The system searches the Solid Edge vault. If you did not define a Solid Edge vault, the system searches all indexed locations on the client machine. Files are ranked and listed based on the property match. For information on defining a Solid Edge vault, see Define a Solid Edge vault.

  3. (Optional) To modify the view of the resulting list, right-click any file in the list and use the Freeze Columns, Sort, Column Width, or Font options.

  4. In the Find Duplicates dialog box, select the file with which you want to replace the duplicate file.

  5. Click Replace, and then click Finish.

  6. Click Perform Actions .
