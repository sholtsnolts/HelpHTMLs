Sort multiple columns

You can use the Sort command in the shortcut menu to sort lists, including indented lists, based on the contents of the column.

  1. Click a column header to select it.

  2. Right-click and select Sort.

  3. Determine the sort order - Ascending or Descending.

  4. Determine the type of sort - Alpha & Numeric or Alphabetical.

  5. To identify secondary columns for sorting, Shift+Double-click additional column headers.

Note:

  • Following an initial sort of the contents, look for the sort indicator to determine the primary sort column. The sort indicator ˄ denotes ascending sort order. The sort indicator ˅ denotes descending sort order.

  • In Structure Editor, column data is string data which can affect sorting.

  • You can sort data in ascending or descending order within each sort column.

  • Blank rows in a table may be affected by sorting.
