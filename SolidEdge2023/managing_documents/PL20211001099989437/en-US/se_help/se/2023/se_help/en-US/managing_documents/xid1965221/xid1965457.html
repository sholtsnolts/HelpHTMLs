Using the Teamcenter Integration for Solid Edge

Description

The following document management activities introduce you to using Solid Edge Embedded Client (SEEC), Solid Edge’s integration to Teamcenter, to create, revise, and manage data in the context of your daily tasks.

Information on Teamcenter instructor-led or online courses is available from http://training.plm.automation.siemens.com/country/en-ams/index.shtml

Objectives

These activities demonstrate how to use the transparent integration between Solid Edge and Teamcenter, and are designed to be completed in succession. After completing the activities, you will be able to:

  • Understand the relationship between Solid Edge, the Solid Edge Embedded Client, and Teamcenter.

  • Understand and use Solid Edge Embedded Client workflows and common property dialog boxes.

  • Move unmanaged documents into a Teamcenter managed environment.

  • Create and modify Solid Edge documents managed in the Teamcenter environment.


How to use the activities

The format of each activity is consistent. Steps are numbered and specify what you should accomplish at any point in the exercise. Below each step are specific instructions for the individual actions you must take to accomplish the step.

Each activity ends with a review that emphasizes concepts introduced in the exercise.

Software requirements

The activities assume you have successfully loaded Solid Edge and are licensed to use the Teamcenter Integration for Solid Edge (SEEC).

For software version compatibility information, see the SEEC_Readme.htm file located in \Program Files\Siemens\Solid Edge 2023.

Activity data set and configuration requirements

To successfully complete the activities:

  • Obtain the training files required for completion of the activities.

    Extract the activity data set spse01424.zip to a folder named \SEEC Class Files.

  • Load ISO templates.

  • Set the user interface theme to Balanced (Solid Edge Default).

  • Add the Start menu entries for SEEC using the Choose PDM Integration tool.

  • Set the Solid Edge Options→Helpers→Starting Solid Edge option to Start with a Blank page.

Teamcenter server configuration assumes default preference values, particularly:

  • SEEC_Property_Item_Name=null

  • WsoInsertNoSelectionsPref=1
