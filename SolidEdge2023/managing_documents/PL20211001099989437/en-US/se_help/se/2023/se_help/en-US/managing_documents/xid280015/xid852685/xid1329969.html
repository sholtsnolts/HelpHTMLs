Save to Existing command

Saves a new Solid Edge document to an existing Teamcenter item that does not contain a Part, Sheet Metal or Assembly document. When you provide the Item ID of an existing item, the item is located in Teamcenter and property information for the existing item is displayed in the New Document dialog box. The font for the Item ID is displayed in italics and the associated tooltip indicates the item you have chosen exists in Teamcenter.

When there is more than one match for the Item ID you provide, the Select Item dialog box displays so that you can choose from the list of items with the matching Item ID. If no matching Item ID is found, the Select Item dialog box is displayed to assist you in locating existing Teamcenter items.

Note:

This command operates on one row of the New Document dialog box at a time.
