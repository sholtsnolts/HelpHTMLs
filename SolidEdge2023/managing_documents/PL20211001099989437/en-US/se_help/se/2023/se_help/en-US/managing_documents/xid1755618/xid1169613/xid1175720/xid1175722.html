View a document in the application used to create it

To view a document in the application used to create it, you use the Editor command.

  1. In the Design Manager document list, click the Current Filename cell for the document of interest.

    The Editor command is not available when you select multiple documents.

  2. Choose Home tab→Open in group→Editor .
