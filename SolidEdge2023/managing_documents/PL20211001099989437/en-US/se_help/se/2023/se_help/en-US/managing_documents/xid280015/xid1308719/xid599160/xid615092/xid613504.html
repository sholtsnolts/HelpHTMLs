Report Duplicates command

Displays a list of objects in the Error Assistant pane that have been identified as having either a duplicate file name or duplicate document numbers. Duplicate file names are highlighted in red indicating they must be resolved before continuing. Duplicate document numbers are highlighted in yellow, indicating further investigation should be taken before continuing.
