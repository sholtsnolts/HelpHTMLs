Activate a PDM integration

Each of the product data management (PDM) integration clients are installed with Solid Edge. You need only activate the integration you want to use (provided you have a license for it), and you can change active PDM clients at any time.

  1. Choose Start menu→Siemens Solid Edge 2023→Choose PDM Integration.

    The program determines the available PDM integration clients based on your license.

  2. Select the PDM integration you want as your active client when you start Solid Edge.

    • Solid Edge with Data Management

      Included with each base Solid Edge license.

    • Teamcenter Integration for Solid Edge (SEEC)

    Your active client is displayed under the ribbon. Only one PDM client is active at a time.

  3. (Optional) To add or remove Start menu entries, click the Home tab, and from the Start Menu group, select the PDM integration for which you want Start menu entries.

    You can choose to perform other functions, including starting applications, using the options in the command ribbon. For more details, see PMD Client - Application Launcher dialog box.

  4. Click Exit.
