Modify single sign-on Teamcenter environments

Configure Teamcenter Client Communication System (TCCS) and Teamcenter Security Services (TcSS) before running Solid Edge.

  1. Access the Define Teamcenter Environments dialog box by selecting Start→Siemens Solid Edge 2023→Choose PDM Integration→Define Teamcenter Environments.

  2. Click the TCCS button to retrieve information regarding servers that have Teamcenter Client Communication System enabled.

    Server information is returned to the table.

    Note:

    If no server information displays, either there are no servers configured for single sign-on, or TCCS is not configured properly. Ensure that TCCS starts and then confirm your configuration through the Teamcenter Environment Manager (TEM).

  3. Specify an environment description for the URL you specified.

    Note:

    At least one Teamcenter environment must be defined with a corresponding environment description.

  4. In the Teamcenter Security Services (SSO) section of the dialog box, select the Enable check box.

    The Teamcenter environment URL you specified is added to the list and is displayed in green for easy identification.

    Note:

    Only one Teamcenter environment URL that is single sign-on enabled can be active at a time. Choose the URL from the list and select Activate to make an environment active.

  5. (Optional) Use Move Up or Move Down to rearrange the order of display of the registered environments in the Current Teamcenter Environment Selection list.
