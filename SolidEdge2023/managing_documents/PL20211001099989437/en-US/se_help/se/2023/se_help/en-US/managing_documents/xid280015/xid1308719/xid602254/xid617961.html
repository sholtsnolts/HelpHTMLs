Add documents to the unmanaged list

  1. From the toolbar, click Add Document .

  2. On the Open dialog box, select the documents you want to add to the list of documents to be imported to Teamcenter.

    You can select a single document or multiple documents from the Open dialog box.

    Note:

    Folders, .xls, and .csv files are not acceptable input.

  3. Click Open.

    The left pane of the Add to Teamcenter Interactive window displays each document expanded to one level. Existing property data is retrieved based on the currently displayed columns.
