Making the most of the Structure Editor help system

Structure Editor help is a highly navigable information system consisting of right-at-your fingertips information on how to use Structure Editor commands and dialog boxes. The content is integrated with Solid Edge help and is fully searchable to assist you in finding just what you need when you need it.

In most cases, topics that apply to the active command or dialog box are available simply by pressing the F1 key or by clicking a Help button.

Using the search feature

Structure Editor help supports advanced full-text searches. The following search options are available: basic search, wildcard search, nested search, or boolean search. You can also search for similar words, or you can search through a previous results list.

The basic rules for formulating queries are as follows:

  • Searches are not case-sensitive, so you can type your search in uppercase or lowercase characters.

  • You can search for any combination of letters (a-z) and numbers (0-9).

  • Punctuation marks such as the period, colon, semicolon, comma, and hyphen are ignored during a search.

  • Group the elements of your search using double quotes or parentheses to set apart each element. You cannot search for quotation marks.
