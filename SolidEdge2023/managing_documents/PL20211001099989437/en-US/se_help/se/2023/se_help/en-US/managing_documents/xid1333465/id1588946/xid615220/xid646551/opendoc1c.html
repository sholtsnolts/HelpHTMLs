Open command

Opens an existing Solid Edge document in Structure Editor. The Open command focuses on an Item-centric view, displaying all the Items in the database, including those with no dataset. You can use the View controls to determine if you want to see a Basic, Intermediate, or Full display of the objects in the database in the Look In portion of the Open dialog box. See the Detail portion of the Help topic, Open a File dialog box for a description of the three views.

When you open an empty object (one the does not contain a dataset), you have the opportunity to create structure. This enables the top-down design approach for the dataset to be defined later.

You can use the Revision Filter to display All, Latest, Released, Latest Released, or Latest 3 revisions of documents.

After you use the Open File dialog box to open a document from the Teamcenter database, the application title displays as Structure Editor — Teamcenter — [formula].

Note:

For a list of all available filter types, see the Files of Type list box on the Open File dialog box.
