Take Ownership dialog box

Displays a new revision of the documents selected with the Take Ownership command along with their property values retrieved from Teamcenter. You can set the value for the individual property being uploaded to Teamcenter.

Files Of Type

Specifies the type of files that are listed in the Filename column.

Save

Saves changes you have made to properties.

Print

Sends a copy of the data to a specified plotter, printer, or file. Options are available for defining the printing area, range, number of copies, and other printing characteristics.

Before using this command, you must install and select a printer.

Cut

Cuts the current selection and pastes it to the Clipboard.

Copy

Copies the current selection to the Clipboard.

Paste

Inserts the Clipboard contents at the specified insertion point.

Restore

Sets all property cells back to the previous values.

Assign All

Assigns Teamcenter-generated values to the properties displayed in the dialog box.

Active Project

Displays the name of the project you are working in.

Document property definition area

Lists each document you have selected for taking ownership. Specific properties for each document are defined in each cell shown in this area of the dialog box. Required properties are identified with a red asterisk.
