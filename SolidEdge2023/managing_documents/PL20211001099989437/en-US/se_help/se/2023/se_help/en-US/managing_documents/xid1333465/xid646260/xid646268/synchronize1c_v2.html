Synchronize command

Synchronizes an out-of-date document in your local cache by automatically downloading the latest version of the document from Teamcenter. All related parts and assemblies that may have been added to the structure by another user out of context are downloaded to ensure the active document's structure and properties are up-to-date. If no document is selected, the command acts on all indirect documents that are not checked out.

Note:

The Synchronize command does not operate on the direct document. The direct document (top-level document) is only synchronized when the document is opened.

With a document open in the Teamcenter-managed environment, use this command to get the latest Teamcenter release status, replace the document with the latest version, and synchronize out of date properties. BOM structures are synchronized when the Teamcenter preference SEEC_BOM_Synchronize is set to True.
