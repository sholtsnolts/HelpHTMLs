Importing Solid Edge templates into Teamcenter

Regardless of the method you use to create a document, a document template is used as a starting point. The template provides default settings for text, formats, geometry, dimensions, units of measurement, and styles that are used to produce a new document. Document templates are delivered with Solid Edge in the \Program Files\Siemens\Solid Edge 2023\Template folder and they can be imported into the Teamcenter-managed environment using the SEEC→Import Solid Edge Templates program.

To run the program, from the Start menu, choose Programs→Siemens Solid Edge 2023→SEEC→Import Solid Edge Templates. The Import Solid Edge Templates dialog box displays. You only need to provide the location of your unmanaged Solid Edge templates. Once the import is run, your Solid Edge templates are imported into Teamcenter.

As a best practice, maintain any changes to the templates in the unmanaged folder, then upload the changes into the managed environment.
