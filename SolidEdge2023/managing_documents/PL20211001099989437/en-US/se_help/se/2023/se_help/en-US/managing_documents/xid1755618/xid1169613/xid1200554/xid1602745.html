Automatically name files via Document Number and Revision

Selecting this option displays the following information in the Design Manager document list:

This column in the document list

Displays

New Filename

The document number, the delimiter, and the revision number. This column is read-only.

Doc Number

Displays the current document number.

Revision

Displays the current revision number. For new files, it displays the initial value, as specified in the Solid Edge Options dialog box→Manage tab→Document Naming Rule tab.
Automatically revise or copy drawings that use the selected 3D document

Automatically copies and revises drawings that use the selected 3D document. If there are multiple drafts, the system adds an increment ID between the document number and revision number. For example, test1-1.dft, test1-01-1.dft, test1-02-1.dft, and so on.

Life Cycle

Show below statuses in Design Manager and Property Manager

  • Available

  • Released

  • In Work

  • Obsolete

  • In Review

  • Baselined

Specifies which statuses to display.

Link Management

Automatically update links in all where-used results when existing file name or location changes

Updates all links in where-used results as you change file names or locations.
