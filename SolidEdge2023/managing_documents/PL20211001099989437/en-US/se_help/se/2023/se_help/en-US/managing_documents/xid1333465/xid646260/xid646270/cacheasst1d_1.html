Cache Assistant dialog box

Manages documents in local cache.

Update Status Info

Updates information for all documents in the local cache. It retrieves document information, such as check out status and revision information, from the database and document status information from the local cache.

Synchronize

Updates out-of-date documents in your local cache by downloading the latest version of the documents from the database.

Check In All

Checks in all checked out documents in your local cache to the database.  This is helpful if you need to check in documents when you are manually managing the cache or trying to clean up the local system.

Download

Loads a document from the database to your local cache.

Create Package

Displays the Create Package dialog box for compressing documents in the workspace into a file to share with external collaboration partners. This option is available when the Teamcenter preference SEEC_Enable_CreatePackage is enabled.

Packaging Activity Log

Displays the contents of the package activity log file following the creation of a package. The command is available after the Create Package command creates the compressed file. This option is available when the Teamcenter preference SEEC_Enable_CreatePackage is enabled.

Me

Shows only the documents checked out to you.

All

Shows all documents in the workspace.

Others

Shows documents that are checked out to those other than you.

Auto-Update Cache Status

Updates the cache status box automatically when the button is selected.

Columns

Displays the Format Columns dialog box so that you can determine the display of property information within the table.

Preview

Displays a preview image of the document in a separate window. To create a preview image, use the options on the Preview page of the File Properties dialog box as described in the Help topic, Set document preview options.

Auto Fit

Performs a Best Fit to resize each column based on the longest string in each column.

Clear Cache

Removes from the cache all documents in the current workspace.

Workspace Name

Displays the name of the current workspace and lists the workspaces available on the client.

Workspace Description

Provides information about the current workspace.

Edit

Displays the Modify Workspace Description dialog box so that you can edit or modify a workspace description.

New Workspace

Displays the Create New Workspace dialog box so that you can define a new workspace and description.

Delete

Displays the Delete Workspace dialog box and informs you that deleting the workspace from the local cache performs the following actions:

  • Checks in to Teamcenter all checked out documents.

  • Deletes from cache this workspace and the associated documents.

Delete All

Displays the Delete All Workspaces from Cache dialog box and informs you that deleting all workspaces from the local cache performs the following actions:

  • Checks in to Teamcenter all checked out documents.

  • Deletes from cache all of the local workspaces and associated documents.

  • Creates a new workspace named Default.

Open

Opens the selected document in Solid Edge using the settings you saved on the Open dialog box. You can also choose to open the document read-only or open the version from your cache.

Close

Exits the Cache Assistant dialog box.

User Session Information

Displays your last name, first name or login name, the current group, current role, the default project ID and project name, and current database when working in a managed environment. Selecting the link displays the User Settings dialog box.

Documents

Displays a list of documents in your local cache along with file property information. The columns displayed include Item Type, Item ID, Revision, Item Name, Dataset Name, Dataset Description, and Cache Status.

The possible values for the Cache Status column include:

  • Newer Version Available, indicating a newer version of the document exists on the server. The font displays blue.

  • Newer Revision Available, indicating a newer revision of the document exists on the server. The font displays black.

  • Newer Version and Revision Available, indicating a newer version and revision of the document exists on the server. The font displays blue.

  • New, indicating the managed document has not been uploaded or checked in to the server.

  • Modified, indicating the document is checked out, waiting to be uploaded, and it is newer than the version on the server. The font displays green.

  • Unknown, indicating the document is not checked out and you have not queried the server.

  • Checked Out To <username>, indicating another user has the document checked out and a new revision exists. The font color displays dark orange, and is underlined.

  • Checked Out To <username>, indicating another user has a document checked out and a new version exists. The font color displays blue.

  • Deleted from Teamcenter, noting that the document you have selected has been removed from the Teamcenter database. You must delete the document from the cache.

Summary information

Displays the number of documents, free disk space, and space used relative to the documents displayed in cache.

  • When no documents are selected in Cache Assistant, the number of Solid Edge documents in cache displays with the amount of free disk space and the amount of disk space used by the documents in cache.

  • If any documents are selected in Cache Assistant, the summary information displays the number of files selected and the disk space used.

  • When the view of documents in Cache Assistant is filtered, the summary displays the number of files shown in the filter and the total number of Solid Edge files in the folder along with the disk space used by the documents in the view.
