Document status types

Every document in Design Manager has an initial status and can be assigned a different status. The initial status for all documents is Available.

You use the settings in the Options dialog box, on the Manage tab, to determine which status types that are available in Design Manager.

Status option

Description

Available

You can work with this document and you can also change its status.

In Work

The document is being revised by another user. Only the user who set the status to In Work, can change it.

In Review

The document is being reviewed by another user.

Released

You cannot make changes to a released document. You can, however, make a revision of the document and work with that.

Baselined

You cannot make changes to a baselined document.

Obsolete

The document is no longer being used.
