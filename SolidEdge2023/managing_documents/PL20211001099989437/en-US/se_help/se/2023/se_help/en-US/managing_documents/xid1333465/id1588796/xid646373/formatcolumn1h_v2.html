Specify the appearance of columns (SEEC)

You can set display options for the columns appearing in dialog boxes such as the Open, New Document, Upload Document, and Cache Assistant dialog boxes within Solid Edge Embedded Client.

  1. In the dialog box, right-click and choose Columns.

    The Format Columns dialog box is displayed.

  2. Do one or more of the following:

    To change

    Do this

    The column display

    Select the check box adjacent to the column name to restore a hidden column. Cleared check boxes adjacent to the column name indicate the column is hidden from display. The absence of a check box beside the column name indicates the column is required and its display cannot be hidden.

    The order of the column headers

    Click the header you want to move and click up or down to change its position in relation to the other column headers.

    The column width

    Click the header you want to change and click either Specific Width to define the column width by specifying a number of characters or click Best Fit to let the application determine the width of each column.

    The alignment of the column headers

    Click the header you want to change and click either left, center, or right for the alignment of the header within the cell of the column.

  3. To apply the style selections, click OK.

  4. Click Close.
