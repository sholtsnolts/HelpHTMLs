Viewing documents

When working in Design Manager, you have several options for viewing your documents. You can:

  • View a preview of your document

    Use this option if you want to quickly see how the part or assembly of interest looks and see its properties. Note, however, that not all documents have a preview image associated with them.

    To preview a document, you use the Preview command .

  • View a document in the application used to create it

    This option enables you to open a document in its associated editor and make changes. For example, you can open a .par file in Solid Edge, modify it, save it, and work with the new part in Design Manager.

    Design Manager uses the Windows file type associations to determine the appropriate editor for any given file. For example, when you choose to open a .par file in an editor, Design Manager launches Solid Edge.

    Note:

    If no application is registered to open a specific file type, you can specify the program from the Open With dialog box and selecting the check box, Always use selected program to open this kind of file.

    To open any object in its associated editor, use the Editor command.
