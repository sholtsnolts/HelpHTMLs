Push files to a different Concept Workspace

Documents in a Concept Workspace can be pushed to a different Concept Workspace.

Note:

The Concept Workspace command is enabled by the Teamcenter preference SEEC_Enable_ConceptWorkspace.

  1. From the Teamcenter ribbon, click Concept Assistant.

  2. Select the documents you want to push to a different Concept Workspace.

    You can select documents in the Concept Assistant and in PathFinder. The Push command supports all selected documents and the dependents of the selected documents. If no documents are selected, the direct document and its dependent documents are pushed to a different Concept Workspace.

  3. Do one of the following:

    • In the Clone group of the ribbon, click the down arrow to view and select the workspace to which you want to push the files.

    • In the Clone group of the ribbon, click the down arrow and select Create New Workspace to define a new workspace as the recipient of the files being pushed.

  4. Click Push.
