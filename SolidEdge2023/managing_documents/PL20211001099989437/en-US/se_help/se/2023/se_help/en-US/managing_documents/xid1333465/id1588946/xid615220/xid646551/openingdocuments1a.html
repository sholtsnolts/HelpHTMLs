Opening documents

The Open command opens existing documents. The software keeps track of the documents you worked on last, so you can open them quickly. These documents are listed at the bottom of the File menu. Just click the name of a document on the list to open it.

If you need to search for the document that you want to open, you can click the Search button on the Open a File dialog box to display the Search dialog box. You can use the Search dialog box to search by document name, document properties, or the creation or modification date.

Opening specific types of documents

There are some options on the Open a File dialog box that enable you to only open specific types of Solid Edge documents.

When opening draft documents, for example, you can specify that files displayed in the Open a File dialog box are of the type Draft document (*.dft). This filter helps you locate and open files quickly.

Using revision rules

You can use the Revision Rules option on the Open a File dialog box to determine which item revisions are displayed for an item. These rules are very helpful in managing documents throughout the life of a project. A few of the most frequently used Revision Rules are:

Any Status; No Working

Loads the latest item revision with a release status of Released. In this instance, you only view item revisions that are released.

Any Status; Working

Loads the latest revision.

Latest Working

Opens the latest item revision regardless of the release status.

Precise Only

Loads the specific item revision that was in effect the last time the item revision was saved.

Version from Cache

Opens the item you have selected using the version already in your local cache.

This is very beneficial if you work with the same data all day and know the information is up-to-date. The version information is validated and the documents are checked out, but no file transfer takes place. This improves performance.

Note:

Your system administrator configures the revision rules for your site and they may be different from what is listed here.

The current revision rule is displayed in the Revision Rule list box on the Structure Editor toolbar. Changing the revision rule of an open document will result in a prompt to perform any actions that have not been processed.

  • Choosing No will result in the loss of any actions that have not been processed.

  • Choosing Cancel will cancel the change of the revision rule.

  • Choosing Yes will process any outstanding actions and change the revision rule.

Searching for documents to open

The Search button on the Open a File dialog box allows you to search for documents. You can search any folder in your managed library. You can search for documents using a number of criteria, including mathematical operators and standard Boolean expressions. For a list of valid expressions, see Search criteria list. To learn more, see Searching for documents.

Opening documents containing non-Solid Edge files

When you open a document that references other files not created in Solid Edge, such as an AutoCAD .dwg or .dxf file, a MicroStation .dgn file, or a Pro/E part or sheet metal file, they display in Structure Editor using the Teamcenter representation for a revision. The filename is not shown.

Opening documents in Solid Edge

Once a document is displayed in Structure Editor, it can be opened in Solid Edge by choosing the Open In Editor command from the File menu.

Note:

You do not have to close Structure Editor to open a file in Solid Edge. However, if you want to make changes to the file, you should close Structure Editor before doing so.
