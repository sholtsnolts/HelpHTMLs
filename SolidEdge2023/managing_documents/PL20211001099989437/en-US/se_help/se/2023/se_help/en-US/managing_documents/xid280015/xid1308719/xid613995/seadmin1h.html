Use Solid Edge Administrator to set user privileges

Solid Edge Administrator is designed for an administrator to set privileges for other Solid Edge users in the group.

  1. Access Support Center.

  2. Log on using your WebKey username and password.

  3. From the list of products, select Solid Edge.

  4. Click Downloads.

  5. Select the version of Solid Edge and then click Solid Edge 2023 Add-ons.

  6. Select SptTools→SptTools_2023.zip.

  7. Unzip SptTools_2023.zip to any location.

  8. From the SEAdmin folder, copy SEAdmin.exe to your \Program Files\Siemens\Solid Edge 2023\Program folder.

  9. To start Solid Edge Administrator, in the \Program Files\Siemens\Solid Edge 2023\Program folder, double-click SEAdmin.exe.

    Note:

    If you have not changed the default location of the Options.xml file on the File Locations page of the Solid Edge Options dialog box, the system informs you that no path has been set for the file. Click OK to dismiss the error message and open Solid Edge Administrator. After you save your changes in Solid Edge Administrator, the File Locations page updates to reflect the new name and location of the Options.xml file.

  10. In the Value cell that corresponds to the option you want to set, enter or change the value.

  11. In the Allow Override cell that corresponds to the option you want to set, click Yes or No to give users the appropriate override privileges.

  12. To save your changes, click File→Save.

  13. In the Save As dialog box, specify the folder for the new configuration file. The location must be a URL, UNC, or shared network connection.

  14. In the File name box, type a new name for the file.

  15. Click Save.

  16. Notify users to use the File Locations page to update the Solid Edge Administrator file location to point to the appropriate configuration file.

Tip:

To update the location, on the Solid Edge Options→File Locations page:

  1. Select Solid Edge Admin, and then click Modify.

  2. Specify the location of the file, Options.xml.

  3. To accept the change and return to the File Locations page, click Open.

  4. To save the changes, select Solid Edge Admin again, and then click Update.
