Synchronize all out-of-date documents in the local cache

In Cache Assistant

  • On the Cache Assistant dialog box, click Synchronize All .

In a Teamcenter-managed document

  • Choose Teamcenter tab→Update group→Synchronize.

Tip:

  • This command may take a long time to process if you have a large cache and a large number of out-of-date documents.

  • You can also select this command on the Status shortcut menu in Assembly PathFinder when nothing is selected.
