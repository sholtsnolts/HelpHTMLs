Use naming rules

Properties that have a naming rule applied appear in Solid Edge common property dialog boxes with a button on the left side of the property cell. The following is an example workflow of using a naming rule that is applied to the Item Name.

  1. Choose the Save or Save As command to save a Solid Edge Teamcenter-managed document.

  2. In the New Document dialog box, choose an Item Type.

    A naming rule button displays on the left side of the cell of the property where a naming rule is applied.

  3. Click the naming rule button to view the valid patterns for the property.

  4. Enter the property value.

    You can click Assign All or use the Assign command to assign the appropriate value. Teamcenter generates the next number in the appropriate sequence and the cell updates with the generated value.

  5. Click OK on the dialog box.

    Your input is validated against the list of available patterns prior to submitting the information to Teamcenter. If your input does not match one of the patterns, you are notified with a message box that contains information about the error.

    Note:

    Your input is validated against all available patterns, not just the one you select. You can click the information button to see a list of patterns.

Tip:

To remove an assigned value for the Item ID and Revision properties, right-click and choose Clear Contents. You can then reassign a new value.
