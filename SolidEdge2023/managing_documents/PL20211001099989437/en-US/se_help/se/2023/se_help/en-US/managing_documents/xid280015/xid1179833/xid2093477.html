Using the Duplicate command in Active Workspace

Both users and non-users of Solid Edge can clone existing document structure in Teamcenter Active Workspace using the Duplicate command. After the Teamcenter preference SEEC_Support_TCOperations is set, the Duplicate command is available to create cloned copies of your top-level document. The newly created structure gets a new Item Revision and new BOM based on your input.

Note:

  • Linked 3D documents such as FOP Master, FOP Member, Part Copy, and Interpart links are not processed by the Duplicate command.

  • Solid Edge 2023 does not support the use of the Duplicate command in Structure Manager.

  1. Do one of the following:

    • Access the hosted Active Workspace client through the quick access toolbar in Solid Edge , or from the File menu→Manage page.

    • In your browser, log into standalone Active Workspace.

      Note:

      The standalone Active Workspace Client and Solid Edge do not share a Teamcenter server. Each client prompts for a login unless Teamcenter Security Services and Teamcenter Communication Services are installed and configured.

  2. Search for the Teamcenter-managed Solid Edge assembly you want to duplicate.

  3. In Active Workspace, click Duplicate.

  4. Set the BOM lines you want cloned with the assembly structure to Save As.

    Note:

    Solid Edge 2023 does not clone BOM lines containing Solid Edge documents with linked 3D documents. For example: FOP Master, FOP Member, Part Copy, and Interpart links. Draft files and non-Solid Edge documents in the same Item Revision are supported.

  5. Clear the option, Run in Background.

    While the Duplicate command can run in the background, Teamcenter Dispatcher must be configured before running the command.

  6. Determine how you want the new object identified.

    • Default ID's - Assigns an ID for you.

    • ID Naming - Lets you specify a prefix or suffix to use with your naming convention.

  7. When you download the duplicated assembly using Cache Assistant, or open the duplicated assembly in Solid Edge, you can click the View List command to view a log file and examine the actions taken by the Duplicate command.

    GRMs are not supported. Any 3D document links such as FOP Master, FOP Member, Part Copy, and Interpart links are not duplicated. The summary of actions take by the command are saved in a log file in \Users\<username>\AppData\Roaming\Siemens\Solid Edge\Version 223\Log Files.
