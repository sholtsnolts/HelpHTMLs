Save an existing design to a different Item type (SEEC)

  1. From an existing Teamcenter-managed Solid Edge document, choose Teamcenter tab→Document group→Save As list→Save As - New Item.

    Tip:

    This command is also available from File menu→Save As→Save As - New Item.

  2. Choose the Item Type and complete the New Document dialog box.

  3. Click Perform Actions.
