Concept Assistant command

Displays the Concept Assistant so that you can interact with documents created or modified within an active Solid Edge session in Concept Workspace. The Concept Assistant is only available for the direct (top-level) document, and is accessed from the Teamcenter tab when you work in a Concept Workspace. Use the Concept Assistant to compare what is being changed to what is in Teamcenter. Think of it as a companion to PathFinder that works as a dashboard for interacting with your conceptual design. Concept Assistant simplifies the display to show new and modified components and gives you access to tools like Compare Models, Revert, and Clone/Push that specify how the components are committed to Teamcenter.

Note:

Concept Workspace functionality is enabled using the Teamcenter preference SEEC_Enable_ConceptWorkspace.

You can use functionality within Concept Assistant to:

  • Filter the display of the contents.

  • View a simplified structure of new and modified content.

  • View or modify file properties of documents created or modified before publishing to Teamcenter.

  • Set actions for the documents in the workspace.

  • Publish the items to Teamcenter.

For more information, see Concept Assistant dialog box.
