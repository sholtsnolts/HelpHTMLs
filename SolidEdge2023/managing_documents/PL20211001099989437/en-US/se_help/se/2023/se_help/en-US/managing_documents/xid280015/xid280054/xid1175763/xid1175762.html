Reimport package file contents to Teamcenter

  1. Receive the compressed package collaboration file (.zip) from a design partner and extract the contents to an unmanaged folder.

  2. Verify the existence of the manifest file, for example: Manifest-<name>-<timestamp>.xml

    Caution:

    The manifest file is generated by the Create Package command and should not be edited or removed.

  3. Start Add to Teamcenter-Interactive. You have two options:

    • In Solid Edge, select File menu→Manage→Add Document to Teamcenter.

    • From the Start menu, select Solid Edge 2023→SEEC→Add to Teamcenter-Interactive.

  4. Select Start New Session and Add Document and log into Teamcenter.

  5. Browse to the unmanaged file location and select the first document.

    The Session History cell changes color to indicate there is a manifest for that object. A green cell indicates the document is found in Teamcenter and No indicates the documents has not yet been uploaded to Teamcenter.

  6. Add additional top-most documents using the toobar of Add to Teamcenter-Interactive to guide you through finding linked documents and running validations.

    You do not have to add any property data to the cells denoted as having manifest information.

    Caution:

    Do not edit the file name or the mandatory property data.

  7. Determine the appropriate action for the documents being reimported to Teamcenter.

    In the Teamcenter Action column, you can choose from:

    • Existing – Use Existing

      This action is automatically set for you if the document was previously imported and it has not changed since it was last imported.

    • Existing – Overwrite

      This action is automatically set for you if the document was previously imported but was modified since it was last imported. Choosing this command overwrites existing Teamcenter data.

    • Existing – Revise

      The document was previously imported and a new revision is needed.

    • Existing – Save As

      The document was previously imported and you wish to save a copy to a new item number, revision, and item name.

    • New

      The document was not previously imported.

  8. Select the Perform Actions command .

    A log file is generated that contains details about the transaction. The Progress Log summarizes whether the documents passed or failed reimport to Teamcenter.

Note:

For more information on adding unmanaged documents to the Teamcenter-managed environment, see Add to Teamcenter-Interactive help.
