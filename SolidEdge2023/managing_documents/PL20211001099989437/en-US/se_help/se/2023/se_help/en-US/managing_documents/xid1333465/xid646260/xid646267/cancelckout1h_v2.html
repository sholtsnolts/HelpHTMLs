Cancel the check out of a document

In Cache Assistant:

  1. On the Cache Assistant dialog box, select the document you have checked out.

  2. Right-click to display the shortcut menu.

  3. On the shortcut menu, click Cancel Check Out.

In a Teamcenter-managed document:

  1. In PathFinder, select the document you have checked out.

    • If no files are selected in PathFinder, the checkout of the active document is canceled.

    • If any files are part of a select set, the checkout of all selected files are canceled.

  2. Select Teamcenter tab→Manage group→Cancel Check Out.

Tip:

If you want to cancel the checkout of multiple documents, you can use the Ctrl key to select documents randomly, or the Shift key to select a sequential list of documents.
