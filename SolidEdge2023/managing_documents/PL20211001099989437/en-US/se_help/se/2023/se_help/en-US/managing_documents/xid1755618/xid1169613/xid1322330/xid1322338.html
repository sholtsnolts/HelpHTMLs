Access file property information

You can use the File Properties command to access information about the current document. Some of the information you can review and edit includes: the document summary, statistics, associated project, and status. You can also create custom properties.

  1. In the Design Manager document list, right-click the Current Filename cell for the document of interest, and then select File Properties.

  2. Select the tabs of interest:

    • General

    • Summary

    • Statistics

    • Project

    • Status

    • Custom

    Note:

    Not all options are available in all environments.
