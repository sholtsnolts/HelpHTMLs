Assign document and revision number automatically

If you have legacy data that does not have document and revision numbers, or you want to delete existing document and revision numbers and assign new numbers, you can do so in Solid Edge data management.

To automatically assign document and revisions numbers, the following must be true:

  • Your data must be stored in the Solid Edge vault. For more information, see Define a Solid Edge vault.

  • Solid Edge data management must be turned on. For more information, see Turn on Solid Edge data management.

  • The check box Automatically name files via Document Number and Revision must be selected. For more information, see Manage tab, Solid Edge Options dialog box.

  1. Open in Solid Edge the document for which you want to assign or replace document and revision numbers.

  2. Choose Data Management tab→Properties group→File Properties or Property Manager .

  3. Do one of the following:

    • If the document has document and revision numbers, clear the Document Number and Revision Number cells.

      • In File Properties, click Apply and Close, and then reopen File Properties and click the Assign button .

      • In Property Manager, click the Assign All button .

    • If the document doesn’t have document and revision numbers:

      • In File Properties, in the Document Number cell, click the Assign button .

      • In Property Manager, click the Assign All button .

    Solid Edge populates the Document Number and Revision cells with the document and revision numbers as defined on the Manage tab of the Solid Edge Options dialog box.
