Determine the issue for correction

  1. Click a yellow or red cell in the Issues column of either the upper-left or upper-right Add to Teamcenter Interactive application window.

  2. The Error Assistant displays the identified problem and suggests corrective action.

    The information continues to display until you click a different cell in the Issues column.

    Tip:

    Use either the Run Unmanaged Validations command or the Run Teamcenter validations command as often as necessary to update the indicator status. A green cell in the Issues column indicates no action is required.
