Select By Type dialog box

Specifies the Solid Edge file types to be used as input for input to Teamcenter.

The checkboxes are are enabled based on the contents in the unmanaged data shown in the upper-left pane of the Add to Teamcenter Interactive window.

Select the Following Document Types:

  • Assembly Documents (*.asm)

  • Part Documents (*.par)

  • Sheet Metal Documents (*.psm)

  • Weldment Documents (*.pwd)

  • Draft Documents (*.dft)

OK

Accepts the input for the dialog box and selects files that match the selected file type.

Cancel

Exits the Select By Type dialog box.
