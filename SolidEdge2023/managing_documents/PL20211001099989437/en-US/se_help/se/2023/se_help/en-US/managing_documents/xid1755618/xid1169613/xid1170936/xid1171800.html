Cancel your changes

To cancel the changes you made in the current session, use the Clear Action command.

  1. Do one of the following:

    • To cancel the changes made to one document, in the Design Manager document list, click the Current Filename cell for the document of interest.

    • To cancel the changes made to all documents, press Ctrl+A, to select all documents in the document list.

  2. Choose Home tab→Action group→Clear Action .

Note:

The Clear Action command does not clear invalid release requests, which are alerts in the Issues column. For more information, see Clear invalid release requests.
