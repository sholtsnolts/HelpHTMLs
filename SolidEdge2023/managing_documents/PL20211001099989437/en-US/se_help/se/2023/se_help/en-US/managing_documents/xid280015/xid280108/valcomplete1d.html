Validations Complete dialog box

Notifies you that your request to import unmanaged documents into the managed environment are either complete, complete with warnings, or complete with errors. You cannot start a new import of files into the managed environment until the previous one is completed or canceled.

View Log

Displays the log file.

Summary

Displays a quick summary of errors or warnings encountered during the import process.

Continue

Continues the import process and starts loading the files into the managed environment. This option is disabled if there are errors noted.

Cancel

Exits the process so you can start over. Any processing is discarded and the queue is cleared of documents being loaded into the database.

Suspend

Interrupts processing and exits the utility, but leaves the import queue intact. Data already created in the database is not deleted. When you later restart the import process using the Retry button, the files resume loading from this point.

Help

Displays help information regarding the dialog box options.
