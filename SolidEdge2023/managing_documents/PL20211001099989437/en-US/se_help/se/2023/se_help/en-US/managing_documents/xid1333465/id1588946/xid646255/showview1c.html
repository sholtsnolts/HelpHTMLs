Show View command

Determines how the document is displayed in Structure Editor. You can choose between Exploded or Part List views. Structure Editor – Teamcenter has a third option, Indented-ItemRev that is the default view. You can switch between the options while a document is displayed in Structure Editor. The actions and where used information is maintained.

The Show View list box is located on the Structure Editor toolbar.

Indented-ItemRev

Presents the components of a document such that for any parent document, all occurrences of the Item Rev are displayed once. If all occurrences use the same Find Number, that value is displayed as the Find No. If an occurrence’s Find Number is unique, the Find No. is displayed as Packed. The Quantity is defined as the sum of all occurrences. If an occurrence’s quantity is non-numeric, the Quantity cell displays as Packed. This view is the default view for Structure Editor – Teamcenter.

Exploded view

Displays the components of a document in an exploded, indented, hierarchical view. The first row of the source and target panes displays descriptions of the properties of the document.

Parts List view

Displays the components of a document in a flat list where a particular file is only shown once. You can sort entries in the active parts list view using shortcut commands. The first two rows of the parts list view displays descriptions of the property values. The third row displays filter options for filtering the contents shown.

You may want to filter the documents displayed in the Parts List view to make it easier to find the document you are looking for. The third row in the Parts list view is used to set filters on the documents displayed. Use the values associated with each property to display the documents that match the criteria you choose. For example, setting the Status to Checked Out To You displays files you have checked out The filter Checked Out To Others displays all files checked out to someone other than yourself.

Note:

If no filter has been set, All Values displays in each column of the Parts List view.
