Session overview

Getting started

Add to Teamcenter Interactive provides support for loading or reloading your unmanaged design data into Teamcenter.

You should consider the following questions before starting a new session:

  • Is the amount of data larger than a project or product in size or is it related to a migration to Teamcenter?

    If you are importing large amounts of data, use the Data Preparation Tools and Add to Teamcenter.

    See Solid Edge data preparation utilities and Working with Add to Teamcenter in Solid Edge help.

  • Are you starting Add to Teamcenter Interactive from Solid Edge?

    See Add Document to Teamcenter command.

  • Are you starting Add to Teamcenter Interactive from the Start menu?

    See Start a new session in Add to Teamcenter Interactive.

Use the following workflow whether your data is:

  • Completely new data that has never been managed.

  • Previously managed, exported from Teamcenter, modified by a collaborative partner, and being reimported into Teamcenter.

  • A combination of new data and previously managed data.

Session workflow

Use the following process to start a session in Add to Teamcenter Interactive and add unmanaged documents into the Teamcenter managed environment.

  1. Start the Add to Teamcenter Interactive application.

    You can start Add to Teamcenter Interactive from the Start menu or from Solid Edge.

  2. Start a new session.

    You can start a new session by selecting the Start New Session and Add Document option from the Add to Teamcenter Interactive startup screen, or you can select the New Session command from the File menu.

  3. Choose the documents you want to add to Teamcenter.

    On the Open dialog box, browse for your unmanaged documents.

    Note:

    Folders, .xls, and .csv files are not acceptable input.

    In the left pane of the Add to Teamcenter Interactive window each root-level document is expanded to one level. Existing property data is retrieved based on the currently displayed columns.

  4. Define the session settings.

    Access the Session Settings dialog box to define options specific to your working session. You can specify where to look for associated draft documents, the Teamcenter Revision Rule used for the session, and so on.

  5. Verify the unmanaged data (find linked documents, missing property data, and so on).

    The Issues column of the upper-left window pane indicates the status of your selected documents. Green indicates no action is required, yellow indicates caution should be taken before the documents are added to Teamcenter, and red indicates corrective action must be taken before adding the documents to Teamcenter.

  6. Resolve any issues reported in the Issues column.

    Error Assistant can help. Use the various commands and tools available to you to fix problems. For example, you can:

    • Use Solid Edge to visualize your designs and fix out of date issues.

    • Use Design Manager to move, copy replace or rename files.

  7. Retrieve Teamcenter data.

    Use the Retrieve Data from Teamcenter command to compare your unmanaged data with existing Teamcenter managed data.

  8. Supply required property information.

    Required property data is indicated by red asterisks.

  9. Add the documents to Teamcenter.

    Use the Perform Actions command to load the documents into Teamcenter and complete the session.
