PDM Client Application Launcher

While each of the product data management (PDM) integration clients are installed with Solid Edge, the PDM Client Application Launcher can be used to activate a product data management (PDM) client for which you are licensed. Optionally, you can control what Start menu entries you want and also start other applications using the options on the dialog box.

In the Home tab, you can:

  • Access the Software Licensing Tool for license borrowing.

  • Start the Solid Edge License Utility so that you can specify your Solid Edge license type.

  • Register your cloud-enabled license.

  • Add Start menu entries.

  • Start Design Manager or Solid Edge.

In the Teamcenter tab, you can:

  • Start the Solid Edge Data Preparation Utility.

  • Run the SEEC diagnostics tool.

  • Define your Teamcenter environments.

  • Import templates, run Add to Teamcenter, or run Add to Teamcenter-Interactive.

  • Start Structure Editor or Solid Edge.

Your active PDM client is displayed under the ribbon. Use the entries in Select Solid Edge PDM Integration Client, to set the PDM integration you want as your active client when you start Solid Edge:

  • Solid Edge with File Management

    Included with each base Solid Edge license.

  • Teamcenter Integration for Solid Edge (SEEC)

Only one PDM client is active at a time.
