Solid Edge Data Preparation Utility

The Solid Edge Data Preparation Utility assists you in analyzing and preparing your unmanaged Solid Edge documents before importing them into a Teamcenter-managed environment. The utility is delivered when you install Solid Edge.

In the utility, use the following workflow to prepare your unmanaged documents for importing into a managed environment:

Basic workflow

  • Specify the document management application for which you are preparing your data.

  • Set your working folder if you want to use a folder other than the default folder for log files.

  • Select files or folders for analysis, or use an existing ordered or unordered list of documents.

  • Analyze your unmanaged files to identify many of the document preparation tasks necessary for importing your data into the managed environment. The Generate Import List option:

    • Evaluates Solid Edge files for duplicate document number values, duplicate file names, empty revision entries, empty project names, and broken links.

    • Performs a file name length analysis.

    • Evaluates the Solid Edge property mapping.

    • Reports the results in spreadsheet form in an analysis report.

    Caution:

    If you use custom Solid Edge data properties, you must define the custom properties for use in Teamcenter using the spreadsheet datapreputilitiestemplate.xlsm located in \Program Files\Siemens\Solid Edge 2023 \Program. For instructions, see Define custom properties before analysis.

  • Fix broken links in Solid Edge files before you import them into the managed environment using options in the Fix Broken Links tab.

  • Update unmanaged Solid Edge files based on entries in the analysis report, and modify properties of the unmanaged Solid Edge files.

  • Use the Add to Teamcenter program to import your prepared unmanaged document to the managed environment.

Note:

You must evaluate and correct your data before importing, so you can have a correctly populated database from the start, which minimizes future efforts. Be sure to read Considerations before using the Data Preparation Utility.
