Revisions dialog box (SEEC)

Right-clicking a cell in the grid displays a shortcut menu to change the font, sort the information, find and replace text, and change column display.

Document list

Displays properties for each document shown in the Filename column. For each document, the property information displayed includes the Item Type, Item Display Name, Item ID, Revision, Item Revision Name, Folder, Project ID, and Teamcenter release status.

Note:

The folder name is not unique and the same item can exist in multiple folders.

Print

Sends a copy of the file property information to the printer. Options are available for defining the printing area, range, number of copies, and other printing characteristics.

Copy

Copies the current selection to the Clipboard.

New

Displays the Revise Document dialog box so that you can change the property values using the commands in the ribbon and assign a new revision to the selected document.  The new revision is created in the Folder you define.
