Create Package command

Supports collaboration with external partners by providing you the ability to create a compressed file of the contents of the cache workspace to send to a design partner. Once the external design collaboration is complete, the design data can be reintegrated into Teamcenter using Add to Teamcenter-Interactive.

This approach is particularly helpful when you need to:

  • Create an unmanaged copy for archival.

  • Create an unmanaged copy for a collaboration partner who performs work and then returns the data.

  • Populate a workspace so that the data can be used offline. For example, during a design review.

Note:

This command is enabled using the Teamcenter preference SEEC_Enable_CreatePackage.
