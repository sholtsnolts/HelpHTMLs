Setting obsolete status on released documents

You use the Obsolete command if you want to make a released revision obsolete.

When you select the Obsolete command, Design Manager updates the cells corresponding to the document of interest as follows:

  • Sets the Action cell to Change Status.

  • Sets the SE Status cell to Obsolete.

  • Updates the New Filename cell.

  • Displays the location where you chose to save the file in the New Location cell.

If you specified in the Solid Edge Options dialog box→Manage tab→Life Cycle page the location where you want to save documents with Obsolete status, Design Manager saves your documents in the specified location by maintaining the subdirectory structures. If you did not specify a location, Design Manager displays a dialog box where you can specify the location.
