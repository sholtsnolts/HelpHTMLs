Starting the application

Add to Teamcenter-Interactive can be started by choosing Start→Siemens Solid Edge 2023→Choose PDM Integration→Add to Teamcenter-Interactive, or from Solid Edge using the Add Document to Teamcenter command.

Note:

Access to the Add Document to Teamcenter command from Solid Edge is determined by Solid Edge Administrator.

The start-up screen

The Add to Teamcenter-Interactive start-up screen provides a product description, the commands to login or start a new session, and a toolbar with access to help. You can access a list of recently used documents from the File menu.
