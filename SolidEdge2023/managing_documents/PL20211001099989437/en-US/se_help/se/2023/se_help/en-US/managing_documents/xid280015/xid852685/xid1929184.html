Assign Project ID command

Assigns projects to the selected Solid Edge document or documents using the Select Projects dialog box.

When you work in an assembly, you can use the Assign Project ID All command to assign single or multiple projects to all components of the assembly. Within an assembly, you can make assignments to specific documents by selecting the documents in PathFinder and then choosing the projects you want associated with those specific documents without having to upload them to Teamcenter.
