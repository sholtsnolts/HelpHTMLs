Copy column settings to other users

You can copy the configuration of columns so that columns display predictably from one user to another.

  1. Define the column display using the commands on the Format Columns (SEEC) dialog box.

  2. Locate the <Teamcenter Database URL>_Custom_Column_Settings.xml file.

    Example:

    \Users\<username>\AppData\Roaming\Siemens\Solid Edge\Version 223\myserver_7001_tc_Custom_Column_Settings.xml

  3. Copy the file to the same location on the next user's computer.
