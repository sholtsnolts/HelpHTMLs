Working in sessions

Add to Teamcenter Interactive introduces the concept of working with your data in individual sessions. A session begins when you do one of the following:

  • Select the Start New Session and Add Document option from the Add to Teamcenter Interactive startup screen.

  • Select the New Session command from the File menu.

  • Select the Manage→Add Document to Teamcenter command from Solid Edge.

A session completes when documents are successfully imported into Teamcenter. The parameters of each session are defined by the options on the Session Settings dialog box. For each session, you can determine how links are enumerated, where to search for draft documents, the Teamcenter Revision Rule used, and so forth.

For more information see Session overview.

Note:

The Document Name Formula used in your Add to Teamcenter Interactive sessions is set in Solid Edge on the Helpers page of the Solid Edge Options dialog box.

Sessions that are interrupted before completion can be saved and then restarted later.

Planning your session

Before starting a session in Add to Teamcenter Interactive, determine your session settings in the Session Settings dialog box. The controls apply to each working session.

Starting a session

A session starts when you select either the New Session command or the Start New Session and Add Document option from the startup page. You begin by selecting the documents you want to add to the managed environment.

Restarting a session

During the Add to Teamcenter Interactive process, some requested actions may not complete, or they may complete with warnings. In either case, a notification dialog box is displayed. You cannot start a new import of files into the Teamcenter database until the previous one is completed or canceled.

The options on the notification dialog box determine how you proceed. You can view the log generated by the process, view a summary, continue, cancel, suspend, or view help regarding the upload process.

  • View Log — Displays the Add to Teamcenter log file.

  • Summary — Displays the Add to Teamcenter Interactive Information dialog box providing a quick summary of errors or warnings encountered during the process.

  • Continue — Continues the process and starts loading the files into the managed environment. This option is disabled if there are errors noted.

  • Cancel — Exits the process so you can start over. Any processing is discarded and the queue is cleared of documents being loaded into Teamcenter.

  • Suspend — Interrupts processing and exits the program, but leaves the import queue intact. Data already created in Teamcenter is not removed. When you later restart the Add to Teamcenter process using the Retry button, the files resume loading from this point.

  • Help — Displays help information regarding the dialog box.

If you previously suspended the session, the notification dialog box is displayed automatically when you next start the next session.
