Create a Concept Workspace

When no document is open, you can create and activate Concept Workspaces while online or offline.

Note:

The Concept Workspace command is enabled by the Teamcenter preference SEEC_Enable_ConceptWorkspace.

  1. Close any open Solid Edge documents and start Cache Assistant.

  2. On the Cache Assistant ribbon, expand the Workspace group and click Concept Workspace.

  3. In the New Concept Workspace dialog box, provide a name for the new workspace.

  4. In the Description box, describe the design data in the workspace.

  5. Click OK.

    The Workspace Name and Workspace Description boxes indicate your active workspace, and the title bar updates to indicate you are working in a Concept Workspace.

    Note:

    When you create a Concept Workspace, a folder is created in the default SEEC cache location defined in Settings→Options→File Locations. Do not attempt to modify the cache by manually changing data in the Windows folder on disk. Instead, use the options in Cache Assistant dialog box.
