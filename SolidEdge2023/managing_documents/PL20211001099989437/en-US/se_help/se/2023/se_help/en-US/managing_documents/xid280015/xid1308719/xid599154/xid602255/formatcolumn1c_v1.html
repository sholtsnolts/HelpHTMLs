Columns command (SEEC)

Presents the Format Columns dialog box providing you options for displaying property information in addition to what is present by default. Any property synchronized with Teamcenter can be displayed. Information is from the Teamcenter database and not from the local cache. Custom properties are displayed in the language and property display name used to create them in Teamcenter.
