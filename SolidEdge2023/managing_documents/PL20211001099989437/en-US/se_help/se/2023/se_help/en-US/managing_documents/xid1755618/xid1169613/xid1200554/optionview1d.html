View tab, Options dialog box

Controls the appearance of the software and document window, such as the display of the graph paper, ruler, and scroll bar.

Note:

Not all options are available in all environments.

Show ToolTips

Displays on-screen descriptions of a toolbar button when the mouse cursor pauses over it.

Show Status Bar

Displays the status bar of the application. You can choose to display the status bar at the top or the bottom of the window.

Application Color Scheme

Determines the window and background color. You can choose from Aqua, Black, Blue, Silver, and White.

2D Loader Preferences

Opens the 2D loader Preferences dialog box with tabs for CGM, DWG/DXF, DGN, HPGL and ME10 file format preference settings.
