Modify the display of columns

Out of the box, Design Manager displays several columns in the document list, such as Current Filename, Issues, and Action. Each column represents a document property, which include registered custom properties. You can choose to display more columns, as well as hide columns. You can also change the display order and modify column width and alignment.

You cannot hide the following columns:

  • Current Filename

  • Current Location

  • Issues

  • Action

  • New Filename

  • New Location

To learn more about the document properties, see Design Manager document properties.

  1. In the Design Manager document list, right-click any column header and choose Columns.

  2. Do any of the following, as needed:

    • To display a column, in the Column display and order list, select the check box to the left of the column of interest.

    • To hide the display of a column, in the Column display and order list, clear the check box to the left of the column of interest.

    • To change the display order, click the column you want to move and use the Up and Down buttons.

    • To change the width of the columns, modify the Width options.

    • To change the alignment of text in the columns, modify the Alignment options.
