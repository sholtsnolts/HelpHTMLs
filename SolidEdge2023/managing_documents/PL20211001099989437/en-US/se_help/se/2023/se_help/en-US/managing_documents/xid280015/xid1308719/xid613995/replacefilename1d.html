Document Name Formula dialog box

Property

Lists document properties that you can use for composing a formula to replace the display of the file name throughout Solid Edge.

For a list of properties available by document type, see the Help topic, Standard and Custom properties.

Note:

  • The Object property is from the Item Revision.

  • If the Occurrence property is part of the formula, it is not displayed for the top level assembly.

Add

Adds the selected property to the end of the formula displayed in the Formula field.

Formula

Displays the properties you select to create the document name formula. You can add a property that is not in the active document by typing [property name] in the Formula field.

Tip:

You can combine properties with additional characters to separate the two properties, such as [Document Number]-[Revision Number].

Clear

Removes the text from the Formula field.
