Set draft status automatically

You can now specify that you want to automatically set the draft status of the draft document associated with a Solid Edge CAD file. When you assign any status to a 3D file in Design Manager, its draft inherits the same status.

You set the option Automatically set the draft status to match the status of the part placed in the drawing in the Manage tab, Solid Edge Options dialog box.
