Open a Solid Edge document from Teamcenter rich client

The availability of the Start/Open a Solid Edge Document command in the Teamcenter rich client toolbar is dependent on the value defined for the Teamcenter preference SEEC_TcClient_Enable_Open_Cmd.

For more information about Teamcenter Preferences, see the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.

  1. In the Teamcenter rich client, select an Item, Item Revision, or dataset.

  2. From the rich client toolbar, select Start/Open in Solid Edge

    If you select an Item or Item Revision containing multiple SE Draft datasets where no Solid Edge model documents exist, you are prompted to select a dataset from a list of available Solid Edge Draft datasets.

    If you do not have write access to the Solid Edge document, the Read-Only Document Status Indicator is displayed in the graphics window and [Read-Only] is displayed in the title bar.

    To learn more about read-only documents, see the help topic, Working with read-only documents.

Note:

  • The cache influences how the document is opened. If the cache is empty, the revision rule and variant rule settings from the Solid Edge Open File dialog box are used. If the cache has content, the revision rule saved in the dataset is used.

  • If the selected Item or Item Revision has no Solid Edge datasets, Solid Edge opens and displays the New Template dialog box so you can choose a template.

  • If multiple Solid Edge datasets exist in an Item Revision, Solid Edge opens the associated Solid Edge dataset in the following order:

    • Assembly

    • Part

    • Sheet Metal

    • Weldment

    • Draft

    • Direct Model

    If none of these are found and the Item Revision contains a BOM View Revision, you are prompted to select an assembly template and the structure is added to the assembly.

    Solid Edge does not return the SE Component or SE Connection Solid Edge dataset types.
