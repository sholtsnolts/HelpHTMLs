Add to Teamcenter Complete dialog box

Notifies you that the Add to Teamcenter process has completed. If errors prevented all files from loading or if corrective action was taken on any files to load them, use the following options to complete the process.

View Log

Displays the SuccessFailureLog file. You must have Microsoft Excel or another .csv editor to view the log file.

Summary

Displays the SuccessFailureLog file providing a quick summary of the files that were imported and the ones that produced errors or warnings encountered during the Add to Teamcenter process.

Continue

Continues the Add to Teamcenter process and attempts to load the files into the managed environment.

Cancel

Exits the Add to Teamcenter process. Any files already loaded into Teamcenter remain in the database. The list of documents to be retried for input is cleared. You are returned to the Add to Teamcenter dialog box.

Suspend

Interrupts processing and exits the Add to Teamcenter process, but leaves the import queue intact. Data already created in Teamcenter is not deleted. When you later restart the Add to Teamcenter process using the Retry button, the files resume loading from this point.

Help

Displays help information regarding the dialog box.
