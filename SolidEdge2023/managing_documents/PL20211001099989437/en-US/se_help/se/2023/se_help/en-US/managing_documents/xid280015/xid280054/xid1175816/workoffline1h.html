Switch to Offline Mode

Before working offline, you should download all necessary documents to your cache.  You should check out any documents to which you will need write access.

Teamcenter-managed environment

  • On the Cache Assistant dialog box, click Work Offline.

    Notice the button changes to Working Offline and the user session information indicates you are working offline.

  • To work offline each time you start Solid Edge, in Solid Edge Options→Manage→General tab, select Always work offline.
