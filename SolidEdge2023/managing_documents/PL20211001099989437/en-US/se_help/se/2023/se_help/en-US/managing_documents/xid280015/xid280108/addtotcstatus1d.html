Add to Teamcenter Status dialog box

Provides details regarding the progress of importing unmanaged files into the Teamcenter-managed environment. Clicking Suspend interrupts processing at the end of a block of files being imported, so you can choose an action to perform before the next block is imported.

Files Per Block

Displays the maximum number of files that constitute a block for import into Teamcenter.

Current block

Displays file property information regarding the block of files actively loading into Teamcenter.

File Name

Lists the file name of the documents in the block that is loading into Teamcenter.

Item Number

Displays the document number property of each unmanaged file within the block.

Revision Number

Displays the revision number of each file in the block loading into Teamcenter.

Folder

Displays the unmanaged folder location of each file within the block being imported.

File Size (MB)

Displays the size of each file within the block in megabytes rounded to the nearest decimal point.

BOM Links

Displays the number of Bill of Material (BOM) links in each file being imported as part of the block.

Other Links

Displays the number of non–Bill of Material (BOM) links in each file within the block.

Progress Summary

Displays the total number of occurrences at start, the current block number, the number of files processed, and the number of files remaining to be processed. As each block of files is loaded, the data displayed is updated to indicate the occurrences of the following:

  • Part/Sheet Metal files

  • Assembly files

  • Draft files

  • Other (non-Solid Edge files)

  • BOM Links

  • Other Links

  • File Size (MB)

Note:

The number of each type of link is not stored in PassFail.csv or when you retry loading documents that failed to import. Therefore, when you use choose either of these as your input, the Totals At Start and Remaining data is not reported and Unknown is displayed for BOM links and Other Links.

History

Provides details regarding blocks of files previously loaded into Teamcenter.

Block

Displays the block number of files previously loaded into Teamcenter.

Part/Sheet Metal

Displays the number of Part and Sheet Metal files in the block.

Assembly

Displays the number of Assemblies in the block.

Draft

Displays the number of Drawings in the block.

Other

Displays the number of non-Solid Edge files in the block.

New

Displays the number new files processed in the block of files.

Overwrite

Displays the number of existing files overwritten during the processing of the block of files.

Warnings

Displays the number of files that produced warnings while being loaded. You are given the opportunity to view a summary of the warnings once the Add to Teamcenter process completes.

Errors

Displays the number of files within the block that did not load due to errors. You are given the opportunity to view a summary of the errors once the Add to Teamcenter process completes.

File Size

Displays the total size of all files in the block in megabytes.

BOM Links

Displays the total number of Bill of Material (BOM) links created for the block of files loaded.

Other Links

Displays the total number of other links created for the block of files loaded.

Time

Presents the total time recorded for processing the block of files. The time is presented in HH:MM:SS format indicating the hours, minutes, and seconds for processing the files.

Suspend

Interrupts the Add to Teamcenter process once the end of the block being imported is reached. You are presented the option to resume the process the next time you start Add to Teamcenter.

Note:

If the number of files you are importing is less than the file block size of 100, the Add to Teamcenter process continues until it reaches the end of the block before stopping.

Cancel

Stops further processing and discards any completed actions for the active block of data.

OK

Dismisses the Add to Teamcenter Status dialog box and returns you to the Add to Teamcenter dialog box.
