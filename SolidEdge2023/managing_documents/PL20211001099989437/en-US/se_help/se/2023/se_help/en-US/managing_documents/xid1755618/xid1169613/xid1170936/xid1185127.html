Using a wizard for basic operations

The Design Manager Assistant wizard guides you through the steps needed to perform several basic operations. The wizard is available from the Tools tab.

Using the wizard, you can:

  • Save as

  • Revise a file

  • Move a file

  • Replace a file

  • Search an assembly for a file or string

Note:

Only a document controller user can perform save as, rename, and move operations on standard parts. For more information, see Assign document controller privileges.

  1. Choose Tools tab→Assistants group→Design Manager Assistant .

  2. Click the arrow below Select an action, and choose the operation you want to perform.

  3. Do one of the following:

    • To continue with the chosen operation, click Next.

    • To return to a previous page and change your selections, click Back.

    • To cancel your selections and close the wizard, click Cancel.
