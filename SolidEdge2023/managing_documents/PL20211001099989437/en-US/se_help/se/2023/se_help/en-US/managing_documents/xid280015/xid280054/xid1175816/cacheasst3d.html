Cache Assistant dialog box (SEEC)

The Cache Assistant dialog box:

  • Manages documents in your local Cache Workspace or in a Concept Workspace when you work in a Teamcenter-managed environment.

  • Displays a summary of the number of documents, free disk space, and space used relative to the documents displayed in cache.

    • When no documents are selected in Cache Assistant, the number of Solid Edge documents in cache is displayed with the amount of free disk space and the amount of disk space used by the documents in cache.

    • If any documents are selected in Cache Assistant, the summary information displays the number of files selected and the disk space used.

    • When the view of documents in Cache Assistant is filtered, the summary displays the number of files shown in the filter and the total number of Solid Edge files in the folder along with the disk space used by the documents in the view.

Command ribbon

Update Status Info

Updates information for all documents in a Teamcenter-managed assembly. The command retrieves Teamcenter information, such as check out status and version information, from the server and document status information from the local cache.

You can update the status for all documents in the assembly or for a selected set of documents using shortcut commands.

Synchronize

Updates documents in your local cache by downloading the latest version of the documents from Teamcenter. Synchronizes all documents in the cache, and not just the ones displayed in the document list.

Check In

Checks in to Teamcenter all checked out documents in your workspace. This is helpful if you need to check in documents when you are manually managing the cache or trying to clean up the local system.

Check In New

Checks in to Teamcenter all new documents in your workspace. This is helpful when you have created new documents while working offline from Teamcenter.

Download

Displays the Download dialog box so that you can populate the cache by selecting the documents for download from Teamcenter to your workspace.

Create Package

Displays the Create Package dialog box for compressing documents in the workspace into a file to share with external collaboration partners.

Note:

This option is available when the Teamcenter preference SEEC_Enable_CreatePackage is enabled.

Packaging Activity Log

Displays the contents of the package activity log file following the creation of a package. The command is available after the Create Package command creates the compressed file.

Note:

This option is available when the Teamcenter preference SEEC_Enable_CreatePackage is enabled.

Working Online/Work Online

Functions both as a descriptive icon, stating your working mode (Working Online), and as a command button (Work Online) that reactivates your session with Teamcenter. For example, when you are working online, the Working Online button is highlighted and you can choose to work offline by clicking the adjacent Work Offline button.

Synchronize

Automatically synchronizes with Teamcenter the documents that are in your local cache when you choose to return to working online.

Check In All

Automatically checks in to Teamcenter all the documents that are in your workspace when you choose to return to working online.

Check In New

Automatically checks in to Teamcenter all new documents in this workspace when you return to working online.

Work Offline/Working Offline

Functions both as a command button (Work Offline) that deactivates your connection with Teamcenter, and as a descriptive icon stating your working mode (Working Offline). Working Offline enables you to work without access to Teamcenter. When you work offline, the Cache Assistant title bar indicates you are working offline and commands that interact with Teamcenter are disabled.

Synchronize

Automatically synchronizes with Teamcenter the documents that are in your local cache when you choose to work offline.

Me

Shows only the documents checked out to you.

All

Shows all documents in the workspace.

Others

Shows documents that are checked out to those other than you.

New

Shows documents that are new.

Auto-Update Cache Status

Updates the cache status box automatically when the button is selected.

Columns

Displays the Format Columns dialog box so that you can determine the display of property information within the table.

Preview

Displays a preview image of the document in a separate window. To create a preview image, use the options on the Preview page of the File Properties dialog box as described in the Help topic, Set document preview options.

Auto-Fit

Performs a Best Fit to the column width to resize each column based on the longest string in each column.

Workspace

Creates a defined location in your local cache for your designs.

Clear Cache

Removes from the cache all documents in the current workspace.

Workspace Name

Displays the name of the current workspace and lists the workspaces available on the client.

Workspace Description

Provides information about the current workspace.

Edit

Displays the Modify Workspace Description dialog box so that you can edit or modify a workspace description.

Cache Workspace

Displays the Create New Workspace dialog box so that you can define a new local cache workspace and description. To learn more about Cache Workspace, see Managing your local cache.

Concept Workspace

Displays the Create Concept Workspace dialog box so that you can define a new concept workspace with a name and description. To learn more about Concept Workspace, see Using Concept Workspace to explore design ideas.

Delete

Displays the Delete Workspace dialog box and informs you that deleting the workspace from the local cache performs the following actions:

  • Checks in to Teamcenter all checked out documents.

  • Deletes from cache this workspace and the associated documents.

Delete All

Displays the Delete All Workspaces from Cache dialog box and informs you that deleting all workspaces from the local cache performs the following actions:

  • Checks in to Teamcenter all checked out documents.

  • Deletes from cache all of the local workspaces and associated documents.

  • Creates a new workspace named Default.

Open

Opens the selected document in Solid Edge using the settings you saved on the Open dialog box. You can also choose to open the document read-only or open the version from your cache.

Close

Exits the Cache Assistant dialog box.

User Session Information

Displays your last name, first name or login name, the current group, current role, the default project ID and project name, and current database when working in a managed environment. Selecting the link displays the User Settings dialog box.

Document table

Displays the documents in your local cache that meet the criteria defined by the values in the filtering row.

Property row

For each document, the property information displayed includes the Item Revision Display Name, Item Type, Item ID, Revision, Item Name, Dataset Name, Dataset Description, Release Status, Cache Status, and Filename.

Filtering row

Lists values for filtering the documents displayed in the document list. Each column of the first row of the document list contains filtering information. If no filter has been set, All Values is displayed in each column of the filter row. The possible values for the Cache Status column include:

  • Up-to-date, indicating the documents in the cache and the server are the same.

  • Newer Version Available, indicating a newer version of the document exists on the server. The font is blue.

  • Newer Revision Available, indicating a newer revision of the document exists on the server. The font is black.

  • Newer Version Available, indicating a newer version and revision of the document exists on the server. The text is underlined and font is blue.

  • New, indicating the managed document has not been uploaded or checked in to the server.

  • Modified, indicating the document is checked out, waiting to be uploaded, and it is newer than the version on the server. The font is green.

  • Unknown, indicating the document is not checked out and you have not queried the server.

  • Checked Out To <username>, indicating another user has the document checked out and a new revision exists. The font color is dark orange, and is underlined.

  • Checked Out To You, indicating you have the document checked out and a new version exists. The font color is green.

  • Deleted from Teamcenter, noting that the document you selected has been removed from the Teamcenter database. You must delete the document from the cache.
