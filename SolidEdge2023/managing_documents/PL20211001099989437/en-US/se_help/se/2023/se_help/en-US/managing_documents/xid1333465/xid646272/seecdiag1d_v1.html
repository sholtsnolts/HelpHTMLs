SEEC Diagnostics dialog box

Settings

Includes controls to set the logging level, and define the location for the diagnostic package created by a scan. Also displays information regarding your version of Solid Edge.

Logging Level

Defines the logging level. All performance log files are saved to the C:\Users\<username>\AppData\Roaming\Siemens\Solid Edge\Version 223\Log Files folder with the exception of soaclient*.txt which is saved to %TEMP%.

Performance

Generates the Performance Journal Logging (PJL) file in the Solid Edge logs folder. This level of logging is the default.

Debug

Enhances the amount of information collected during logging. In addition to the PJL file, the following logs are generated:

  • PJL file

  • TALLOG file

  • SOAClient log file

  • TAL debug log file

  • Save as deep copy data

Apply

Sets the logging level selected.

Note:

If Solid Edge or a Teamcenter integration application such as Structure Editor is running when you make changes to the logging level, the application must be restarted for the changes to be implemented.

Diagnostic package location

Defines the location for the log files created by a diagnostic scan.

Solid Edge Version

Displays information regarding the release of Solid Edge installed on the machine.

Multiple versions found

Indicates more than one version of Solid Edge is found on the machine.

Legacy cache folders

Indicates the presence of legacy cache data.

Teamcenter

Accepts client configuration information input.

2–Tier

Indicates the Teamcenter Integration for Solid Edge (SEEC) and Enterprise tier (Teamcenter business logic) are installed locally on the client.

TC_ROOT

Displays the location of the Teamcenter installation on the local machine.

TC_DATA

Displays the location of Teamcenter data. Teamcenter data may be located on a remote system.

4–Tier

Indicates only the Teamcenter Integration for Solid Edge (SEEC) runs on the workstation. The Enterprise tier, database, volume, and other resource tier components run on the server.

URL

Displays the URL of the Teamcenter server.

Detail

Displays information retrieved during the scan. This portion of the dialog box is read-only.

FMS_HOME

Displays the value of the FMS_HOME environment variable. If the environment variable is not defined, Not Set is displayed.

TCCS_JAVA

Displays the value of the TCCS_JAVA environment variable. If the environment variable is not defined, Not Set is displayed.

JRE_HOME

Displays the value of the JRE_HOME environment variable. If the environment variable is not defined, Not Set is displayed.

JRE64_HOME

Displays the value of the JRE64_HOME environment variable. If the environment variable is not defined, Not Set is displayed.

JAVA_HOME

Displays the value of the JAVA_HOME environment variable. If the environment variable is not defined, Not Set is displayed.

SPLM_LICENSE_SERVER

Displays the value of the SPLM_LICENSE_SERVER environment variable. If the environment variable is not defined, Not Set is displayed.

UGII_LICENSE_FILE

Displays the value of the UGII_LICENSE_FILE environment variable. If the environment variable is not defined, Not Set is displayed.

TC_SOACLIENT_LOGGING

Displays the value of the TC_SOACLIENT_LOGGING environment variable. If the environment variable is not defined, Not Set is displayed.

Solid Edge install location

Displays the installation location of the latest version of Solid Edge installed on the computer.

SEEC cache location

Displays the folder defined in Solid Edge Options→File Locations for the SEEC Cache.

SEEC log file location

Displays the folder defined in Solid Edge Options→File Locations for your log files.

Scan

Starts the scan of the local system and generates the SEECDiagnostic log file.

Close

Closes the application.
