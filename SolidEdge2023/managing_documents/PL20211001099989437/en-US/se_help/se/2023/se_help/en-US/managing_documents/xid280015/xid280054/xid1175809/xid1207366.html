Open Version from Cache command

Opens the selected document directly from your local cache with no interaction with Teamcenter. If the document is not checked out to you, it opens read-only. Once opened, you can use the Update Status Info command to get information about the document.

Note:

This command behaves differently than opening a document using the Open File dialog box where you have the opportunity to select the document to be opened along with a revision rule and optionally, Version from Cache.
