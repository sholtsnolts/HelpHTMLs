Administering Add to Teamcenter Interactive

Application administrators are responsible for the operation and maintenance of an application. Their duties can range from installing software, loading product updates, controlling privileges, and performing backups. Solid Edge contains features to assist administrators with their tasks.

  • The Solid Edge readme.htm file in \Program Files\Siemens\Solid Edge 2023 contains system requirements, installation instructions, and additional information regarding the Solid Edge program.

  • The Solid Edge Administrator program gives you the opportunity to define policies and easily change the policies later.

  • You can use the settings within Add to Teamcenter Interactive Options to:

    • Determine the behavior when a managed document is closed.

    • Define assembly structure synchronization.

    • Control Teamcenter client communication system settings (TCCS).
