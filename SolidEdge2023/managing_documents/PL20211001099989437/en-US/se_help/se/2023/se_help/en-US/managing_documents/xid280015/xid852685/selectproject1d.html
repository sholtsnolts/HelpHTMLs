Select Projects dialog box

Displays the Teamcenter projects to which you have privileges to assign Solid Edge documents. Use this dialog box to assign one or more projects to your Teamcenter-managed documents.

Item Information

Identifies the item you are assigning to projects by displaying the icon associated with your item followed by the document name formula. When you have more than one item selected, you are notified that multiple items are selected.

Projects

Lists the projects you have you have access to. Displays the Project ID, Project Name, and Project Description properties.

Note:

Your project access is determined at the time you log into Teamcenter. If project access changes during the time you are logged in, the project list will not be updated until you reestablish your session by logging off and logging back in again.

Selection check boxes

Assigns the selected project to your item.

Check box selection

Definition

None of the objects are assigned to the available project.

All selected objects are assigned to the project.

Some of the selected objects are assigned to the available projects.
