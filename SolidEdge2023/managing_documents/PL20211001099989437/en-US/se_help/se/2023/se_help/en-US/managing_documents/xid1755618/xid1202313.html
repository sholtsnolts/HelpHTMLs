Check out documents

Use the Check Out command to lock a document for a longer period of time so that only you can modify it.

The Check Out command applies the In Work status to documents that have Available status.

Note:

If you use Solid Edge data management in conjunction with Xcelerator Share, Dropbox, or other file-sharing software, you must close and save the files you've checked out so the In Work status is synchronized.

  1. In PathFinder, select the document you want to check out.

  2. On the Data Management tab, click Check Out .

    If you turned on the display of document status, the PathFinder displays:

    • The In Work icon to the left of the document name.

    • The name of the document you checked out in green text.

    • When you hover over a checked-out document, the system displays additional check-out information, such as the check-out date and the name of the user who checked out the document.
