Teamcenter release status

Solid Edge displays icons in PathFinder, Read-Only Assistant, Cache Assistant, the Open File dialog box, and the File Properties dialog box to represent the status of documents within Teamcenter workflows. However, within Solid Edge, documents can be in a state not recognized by Teamcenter.

The following status icons are designated by Solid Edge to indicate specific conditions not found in Teamcenter.

Working

The dataset has not progressed through a workflow and has no status.

Has Status

The document has more than one status such that a specific status cannot be determined.

Custom status and images

In Teamcenter, you have the capability to create custom workflows along with customized status and images that can be displayed in Solid Edge. The Teamcenter preference SEEC_Object_Images enables the display of the customized icons both for release status and for workspace objects. Use the following steps to have your custom images displayed in Solid Edge:

  1. Create a zip file containing a flat list of the customized images you want to use in Solid Edge.

    Images must be on a 16x16 canvas where the image is 14x14 and in PNG format.

  2. In the Teamcenter rich application client (RAC), create a new dataset of type Zip.

    The Zip object type is delivered by the Teamcenter Foundation template.

    The dataset can reside anywhere in Teamcenter.

    All users must have access to the dataset.

  3. Upload the zip file containing the custom images to the dataset object that you created in the previous step.

  4. In Teamcenter, define the preference SEEC_Object_Images.

    The first line must contain the DatasetUID of the dataset object created in the first step.

    The UID is case-sensitive.

    Tip:

    To obtain the DatasetUID, in Teamcenter turn on Print Object. The UID is displayed at the bottom of the Print Object window.

    Each subsequent line must pair the real object name with the associated image file (.png). Define one pair per line.

    Example:

    DatasetUID=wCMAAAmRofPWZA

    Mse1_Dev=Mse1mse1Dev1.png

    Mse2_Test=Mse2mse2Test2.png

    Design=Design.png

    Folder=Folder.png

For additional information, see the Teamcenter Preferences section of the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.
