Import unmanaged Solid Edge templates into Teamcenter

  1. Start the Import Solid Edge Templates program by choosing Start→Siemens Solid Edge 2023→Choose PDM Integration→Templates.

  2. Click Login, and log in to Teamcenter.

    A message displays in the status bar at the bottom of the dialog box to indicate successful login to Teamcenter.

  3. Define the location of your unmanaged Solid Edge template folder.

    Note:

    You can type the location of your unmanaged templates or use Browse to identify the folder. Your templates can also be in a network location. Example: \\abc123\plan\Template_V110_ANSI_ISO.

  4. Click Import.

    Note:

    You must have write access to both the Teamcenter application directory (TC_ROOT) and the Teamcenter Data directory (TC_DATA) in order to import Solid Edge templates into Teamcenter.

  5. Click OK to dismiss the message indicating the Solid Edge templates were imported.
