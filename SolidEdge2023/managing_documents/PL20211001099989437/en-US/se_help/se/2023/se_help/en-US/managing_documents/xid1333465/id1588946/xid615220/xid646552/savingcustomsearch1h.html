Save a custom search

  1. In the Search dialog box, click in the Saved Searches box.

  2. Type a name for the search.

  3. In the Search dialog box, click an attribute box.

  4. Type or select the search criteria.

  5. Click Save to save the search criteria.

  6. In the Search dialog box, click the Search button to conduct the search.
