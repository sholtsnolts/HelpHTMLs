Printing documents

As you work with the structure of a document you may need to send a copy of it to a specified printer or file. The Print command prints the active Structure Editor window. With the Print command, you can:

  • Select or Find a printer.

  • Print to file.

  • Set printing options, such as the range of pages or number of copies to print.
