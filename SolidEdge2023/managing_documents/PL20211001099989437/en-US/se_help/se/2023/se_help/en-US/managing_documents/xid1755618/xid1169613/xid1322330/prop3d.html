Custom group, File Properties dialog box

Displays the custom properties for the document. Custom properties can added or removed using commands in the command ribbon or shortcut menu. You can customize lists for the properties by editing the propseed.txt file located in the Solid Edge Preferences directory.

Note:

If any of the entries in the propseed.txt file exceed the character length of the text box, you can position your mouse over the text box to view a tooltip that contains the full contents.

Name

Specifies the name of the property.

Type

Determines the data type for the property. You can click one of the following types of data: text, date, number, yes or no.

Value

Displays the value of the user-defined property.

Properties

Lists the custom properties defined for the document.
