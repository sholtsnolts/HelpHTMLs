Reload the latest data

If you encounter any of the following issues while working in Design Manager, you can use the Reload command to cancel and clear all changes, actions, and issue messages:

  • Other users checked out or locked files (marked Read-only and highlighted in orange).

  • There are missing links due to folder rename.

  • Concurrent design, assembly was updated.

  • You performed a where-used search, found no results, the system added some additional rows, and you want to return to the original state.

  • You tried to release an assembly that cannot be released, for example an assembly that has a part with Obsolete status or has parts that are not released, and Design Manager alerted you that it cannot release it. To read the associated error message, hover over the icon in the Issues column.

You can use the Reload command to cancel and clear all changes, actions you set, and issue messages. This action closes and reopens the file without leaving Design Manager.

To cancel and clear all changes, actions set, and issue messages, you can also use Home tab→Action group→Clear Action . For more information about the Clear Action command, see Cancel your changes.
