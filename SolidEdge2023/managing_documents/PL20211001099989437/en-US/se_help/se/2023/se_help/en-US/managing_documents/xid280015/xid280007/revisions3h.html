Revise a Teamcenter-managed document

Using the Revisions command:

  1. With your document open, choose Teamcenter tab→ Document group→ Revisions.

  2. Click New.

  3. If you want to edit a document property, click the property cell and edit the value in the text box.

    Note:

    The availability of the Revision column is based on the value of the Teamcenter preference SEEC_MakeReadOnly-Revisions.

  4. Click Perform Actions to create the new revision of the document and upload it into Teamcenter.

Using the Revise command:

  1. With your document open, choose Teamcenter tab→ Document group→ Revise.

    The Document Name defaults to the open document’s file name and the file properties are displayed with the next revision automatically presented in the Revision column.

  2. If you want to edit a document property, click the property cell and edit the value in the text box.

    Note:

    The availability of the Revision column is based on the value of the Teamcenter preference SEEC_MakeReadOnly-Revisions.

  3. Click Perform Actions to create the new revision of the document and upload it into Teamcenter.

    Note:

    • If no files are selected in PathFinder, the action applies to the active document.

    • If a file is selected in PathFinder, the action applies to the file.

    • If multiple files are selected, the command is disabled.

    • Translated files (Parasolid, STEP) that are part of the active document's dataset are not saved to new documents created with the Revise command.
