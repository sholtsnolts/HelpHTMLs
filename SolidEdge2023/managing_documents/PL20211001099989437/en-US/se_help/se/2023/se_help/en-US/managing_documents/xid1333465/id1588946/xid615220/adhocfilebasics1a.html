File operations

Structure Editor supports the same basic file operations you expect from Windows applications:

  • Opening documents

  • Searching for documents

  • Printing documents

  • Saving documents

  • Revising documents

As you work with your documents, use the same conventions that the file system supports for naming documents. Document names can consist of all Unicode characters except the following characters:
# : \ ? * < > % / | “ ~ !

When these characters are encountered during the assignment of an Item ID, the characters are removed from the filename and the Document Name cell is updated for you. For example:

Item ID

Filename

D/123/4/3

D12343.par

D/3456

D3456.par

D.123.34.567

D12334567.par

In the event that the Item ID has no valid filename characters, a message is displayed indicating that a there is a missing value for the required property.
