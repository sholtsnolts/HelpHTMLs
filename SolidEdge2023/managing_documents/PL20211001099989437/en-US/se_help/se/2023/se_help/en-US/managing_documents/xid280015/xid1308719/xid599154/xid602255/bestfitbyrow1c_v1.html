Best Fit by Row command

Fits the width of a column to the width of the largest field in the selected column of the dialog box.

In Structure Editor and Add to Teamcenter-Interactive, the command operates independently in each window pane. For example, selecting Best Fit by Row in the upper-right window pane does not affect columns in the upper-left window pane.
