Extract Translated Files command

Exports from Teamcenter a collection of translated files created in Solid Edge and places them in the unmanaged folder you specify. The files extracted are determined by options you define in the Extract Translated Files dialog box or docking pane. The files are not part of the local cache and are not intended for reimport to Teamcenter.

This command is particularly helpful when you need to create an unmanaged copy for a vendor or manufacturer.

Extracting the translated files does not create a package or zip of the files. It only places the files in the location you specify.
