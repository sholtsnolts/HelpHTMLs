Print dialog box (General)

Controls how a document is printed.

Select Printer

Specifies the printer you want to print to. You can select from a list of all the configured printers available for printing.

Status

Displays the current status of the selected printer. This area is read-only.

Print To File

Accesses the Print To File dialog box.

Preferences

Accesses the Printing Preferences dialog box.

Find a Printer

Accesses the Find Printers dialog box.

Page Range

Specifies which part of the document you want to print.

All

Prints all pages of the active window.

Selection

Prints the currently displayed portion of the active window.

Current Page

Prints the contents of the active window.

Pages

Prints the pages or range of pages you specify.

Number of Copies

Sets options for the number of copies you want to print.

Collate

Organizes sheets when you print multiple copies.  With this option enabled, if you print two copies of a 3-page document, then the sheets are organized 1,2,3; 1,2,3; with this option disabled, the sheets would be organized 1,1; 2,2; 3,3.
