Print to a file

You can generate a printer-ready file for printing at a later time.

Generate a print file

  1. On the File menu, click Print.

  2. In the Print dialog box, select the name of the printer that will print the document at a later time.

  3. In the Print dialog box, set the Print To File option.

  4. Click OK.

  5. In the Print To File dialog box, browse to the folder where you want to generate the print file, and then type the name of the document in the File Name box. For draft files, the file Save As Type is set automatically to Printer File (*.pri).

  6. Click Save to generate a printer-ready file.

Print a printer file

  1. To print a file generated with the Print To File option, the printer has to be local to the system.

  2. Open an MS DOS command prompt window, then change directory to the folder where the printer-ready file is located.

  3. Type the following at the command prompt, where <filename> is the name of the printer-ready file, including the file extension:

    copy <filename> lpt1: /b

  4. This will cause the file to print. If you have more than one printer on this system you may have to use lpt2: or lpt3:.
