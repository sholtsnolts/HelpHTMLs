Upload dialog box (SEEC)

Displays each property of the document being uploaded to Teamcenter. This dialog box is one of the common property dialog boxes that you use in the managed environment to define property data.

Right-clicking an occupied row or cell in the grid displays a shortcut menu to change the font, sort the properties, find and replace text, format columns, change column width, and hide columns.

Note:

The Find and Replace command affects all cells in the table that are not read only.

Restore

Returns the properties to the values shown when you opened the dialog box.

Print

Sends a copy of the data to a specified plotter, printer, or file. Options are available for defining the range, number of copies, and other printing characteristics.

Command ribbon

Paste

Inserts the contents of the Clipboard at the specified location. The shortcut command is Ctrl + V.

Cut

Removes selected elements from the table and places them on the Clipboard. The selected elements replace the previous contents of the Clipboard. The shortcut command is Ctrl + X.

Copy

Copies selected elements from the table and places them on the Clipboard. This command replaces the previous contents of the Clipboard with the new contents. The shortcut command is Ctrl + C.

Find

Displays the Find dialog box so that you can specify text to locate in the table.

Required

Displays only properties that are defined as required properties.

Editable

Displays only properties that you can modify.

Empty

Displays only the properties that have no value defined.

Text

Displays only the properties that contain the text you provide. The text filter acts on the entire table, or only on cells you highlight.

Does not contain

Displays only the properties that do not contain the text you provide.

Show/Hide Columns

Displays or hides columns within the table. This command is only active if a column is hidden using the Columns command. Property and Value columns cannot be hidden.

Clear Filters

Removes any filters applied in the Filter group of the dialog box.

Columns

Displays the Format Columns dialog box so that you can determine the display of property information within the table.

Preview

Displays a Preview pane containing an image of the active document. The pane contains the saved view of the document along with options for displaying an interactive view, storing the image in the file, and updating the image when the document is saved.

Auto-Fit

Performs a Best Fit to resize each column based on the longest string in each column.

Properties

Displays a vertical representation of document properties. You can define or modify property data including file property data.

Save to Existing

Saves the property data to an existing Teamcenter-managed document.

Apply

Saves the property changes to the file. Changes are saved to Teamcenter when you check the object in.

Assign All

Assigns Teamcenter-generated values to the properties displayed in the dialog box.

Assign Project ID

Displays the Select Projects dialog box for assigning projects to selected Solid Edge documents.

Upload All

Sets the action on all documents to Upload.

Check In

Sets the action on all documents to Check In.

Validate

Identifies any issues with property data. After making changes to the property information, use Validate to ensure that those changes are correctly entered and allowed.

Perform Actions

Accepts and performs the information supplied on the dialog box.

Cancel

Exits the dialog box without saving any of the information.

File properties table

Defines specific properties for each document. Properties are defined in each cell shown in this area of the dialog box. Required properties are identified by a red asterisk. Properties that comprise multifield keys are identified by a gold asterisk . You can show additional property columns using the shortcut command, Columns.

Type

Identifies the file type of the object. Icons and tooltips indicate part (.par), assembly (.asm), draft (.dft) , or sheet metal (.psm). Sort this column to group objects by file type.

Action

Displays the option to Upload or Check In the document to Teamcenter.

Issues

Indicates the status of properties following property data validation. When an error is detected, the cell in the Issues column is displayed with a red background and the following symbol , indicating you need to take action to resolve the error. When all cells in a row pass validation, the cell in the Issues column is displayed with a green background and the following symbol , indicating the properties have passed validation. Double-clicking a cell in the Issues column displays the property value errors.

Item Type

Identifies the type of property data for the identified property - Design, Item, Document, Drawing, and so on.

Item ID

(Required property) Unique identifier used in Teamcenter.

Revision

(Required property) Indicates the current revision of the document.

Item Name

(Required property) Defined to locate the document in Teamcenter.

Dataset Name

Displays the name assigned to the dataset.

Dataset Description

Displays up to 240 characters of information regarding the dataset.

Filename

Displays the name of the Solid Edge document.

Folder

Displays the name of the folder where the document is stored.
