Concept Assistant dialog box

Displays documents within the active Concept Workspace that have a status of New or Modified. You can filter the view of the contents based on options in the Select By group of the command ribbon. Items selected in Concept Assistant cross-highlight with items in PathFinder.

Right-clicking an occupied row or cell in the grid displays a shortcut menu to change the font, sort the properties, find and replace text, format columns, change column width, and hide columns.

Note:

The Find and Replace command affects all cells in the table that are not read only.

Command ribbon

Refresh

Updates the contents of the table.

File Properties

Displays the Files Properties dialog box. You can define or modify property data, including file property data.

Compare Models

Compares the version of the selected part or sheet metal model (single selection) in Concept Workspace to the version of the document in Teamcenter. The command evaluates shape, volume, and properties, and notes the differences.

Revert

Returns a document to the latest version in Teamcenter. You can select single or multiple documents. Selecting an assembly reverts the entire structure. When no document is selected, the entire assembly is closed and reopened using the version from Teamcenter.

Auto Fit

Performs a best-fit to resize each column based on the longest string in each column.

New Item

Selects only the files newly created in your active Concept Workspace.

Modified

Selects only files that have been changed.

All

Selects all new and modified documents in the active concept workspace.

Select By group

Uses four values to determine the action. Set the mode first (Select or Set Value), and then select the Action or Item Type.

Example:

Use Select to select all New objects, then use Set Value to give them an Item Type of Design.

Select

Selects documents based on assigned properties.

Action

Determines the action for Select or Set Value.

Item Type

Determines the Item Type for Select or Set Value.

Set Value

Sets property values for documents selected within the workspace.

Validate

Identifies problem areas that should be investigated before publishing the items to Teamcenter. The results are provided in the Issues column.

Save

Retains the properties or options set in Concept Assistant.

Publish

Uploads the selected new or modified contents of the workspace to Teamcenter.

Create New Workspace

Displays the Create a New Concept Workspace dialog box so that you can define a new concept workspace with a name and description.

Clone/Push

Copies selected files from the active concept workspace to the specified concept workspace. When no document is selected, the active document is copied to the selected concept workspace.

File properties table

Object

Displays the object name of new or modified content in the active Concept Workspace.

Action

Displays the options available for publishing each new or modified document in the workspace. Examples of options include: New Item, Upload, Save Item As, and Revise.

Issues

Indicates problem areas that should be investigated before publishing the items to Teamcenter. Each icon displays a tooltip with information needed to resolve the issue.

Item Type

Identifies the type of property data for the document - Design, Item, Document, Drawing, and so on. You can only specify an Item Type for documents with an Action of New Item, based on your preferences.

Concept Status

Displays the status of the document in the concept workspace compared to the document in Teamcenter. Some of the possible values for the Concept Status column include:

  • Up-to-date—Any Solid Edge document retrieved from Teamcenter that has not been modified.

  • Modified—Managed documents that have been changed or contain modified properties, and saved to disk.

  • New—Document created new or in-place within the workspace. The new status is not applied until the file is saved to disk. The Save As, Replace with Copy, Replace with New commands create documents with a status of New.
