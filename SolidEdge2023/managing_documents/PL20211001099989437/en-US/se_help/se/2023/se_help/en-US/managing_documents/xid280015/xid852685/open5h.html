Open a non-Solid Edge document in a Teamcenter-managed environment

  1. Choose File menu→Open.

  2. Select the STEP, IGES, or X_T file you want to open.

  3. Since no Solid Edge content exists in the Item Revision, choose a Solid Edge template to use with your file.

    The non-Solid Edge document (STEP, IGES, or X_T) is downloaded to cache and translated. The translated file is then opened in Solid Edge.

    Note:

    Retranslation of the same document shows only the valid Solid Edge templates since it already contains a Solid Edge file.

  4. Save the document.

    The New Document dialog box displays. The Solid Edge file is added to the same Item ID as the original foreign file. The Document Name matches the Item ID. If a folder or project is defined, the information is automatically populated for you.

    Note:

    You have the option to save the document with a different Item ID by right-clicking the Document Name and choosing Assign.

  5. Click OK on the New Document dialog box.

    The Upload dialog box displays all the indirect documents of an assembly.

  6. On the Upload dialog box, click Assign All to assign values to the indirect documents of an assembly.

    The Dataset Description automatically updates with the file name, and the folder and project information is automatically populated for you.

    Note:

    When you change an Item ID using the Assign command, the Item Name, Dataset Name, and Document Name update for you. Changing the Revision updates the Dataset Name.
