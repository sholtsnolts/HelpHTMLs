Synchronize an out-of-date document

In Cache Assistant:

  1. On the Cache Assistant dialog box, select the document you want to synchronize.

  2. Right-click to display the shortcut menu.

  3. Click Synchronize.

In a Teamcenter-managed document:

  1. In PathFinder, select the documents you want to synchronize with Teamcenter.

  2. Choose Teamcenter tab→Update group→Synchronize.

Tip:

  • If you want to synchronize multiple documents, you can use the Ctrl key to select documents non-sequentially, or the Shift key to select documents sequentially.

  • You can also select this command on the Status shortcut menu in PathFinder when a managed assembly document is selected.

  • If no files are selected in PathFinder, the action applies to all documents that are not checked out.
