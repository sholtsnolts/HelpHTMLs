Migrate a standard part to Teamcenter

  1. Back up your existing standard parts installation.

  2. Choose Start→Siemens Solid Edge 2023→Standard Parts→Configuration Manager.

  3. On the Standard Parts Configuration Wizard, set the Parts Storage option toTeamcenter.

  4. In the Item Type box, type a Teamcenter Item type.

    For Teamcenter Rapid Start, you could enter Standard and when standard parts are created, they will be created with a item type of Standard. For Teamcenter, you could enter Item.

    Note:

    In both Teamcenter Rapid Start and Teamcenter, you can build and use a custom Item Type under Design that represents a standard in your organization.

  5. On the Solid Edge Standard Parts Configuration Wizard, in the Database File Location, specify the a new name for the database.

  6. On the Solid Edge Standard Parts Configuration Wizard, click Save.

  7. Choose Start→Siemens Solid Edge 2023→Standard Parts→Administrator.

  8. On the Standard Parts Administrator dialog box, choose Database→Migrate to Teamcenter to move the unmanaged files to the Teamcenter folder.

  9. Use the Data Prep utilities to analyze the generated parts and make any needed modifications.

  10. Run Add to Teamcenter to upload the generated parts to Teamcenter.

  11. Use the Solid Edge Standard Parts Administrator to register the parts in the database.
