Pull files into a Concept Workspace

Documents are pulled from a Concept Workspace into the active Concept Workspace using the Teamcenter Parts Library or the Replace Part command.

Using the Teamcenter Library

  1. While working in a Concept Workspace, select the Teamcenter Parts Library tab from the margin of the design window.

  2. In the Teamcenter Parts Library pane, set your folder to the Concept Workspace that contains the document you want to pull into your active Concept Workspace.

  3. Select the document and drag it into the active Concept Workspace.

Using the Replace Part command

  1. While working in a Concept Workspace, select the Home tab→Modify group→Replace Part command.

    Tip:

    The command is also available from PathFinder by right-clicking a document and selecting More→Replace Part.

  2. In the Replacement Part dialog box, browse to the location of the Concept Workspace that contains the document you want to pull into your active Concept Workspace.

  3. Select the document you want to use as the replacement part and click Open.
