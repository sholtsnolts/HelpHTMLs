Session Settings dialog box

Controls settings specific to each working session of Add to Teamcenter Interactive.

Link Management

Defines how links are resolved during the session. The behavior is the same as Solid Edge which finds and lists all document links.

Note:

Solid Edge link management is defined on the File Locations page of the Solid Edge Options dialog box.

Where Used – Draft Documents

Specifies where to look for draft documents associated with the documents identified for the working session. You must select either:

  • Search for Draft documents in the same folder as their related models

  • Search for Drafts in the following folder

    You can specify the location by browsing for the folder or by typing the location into the text box.

Remove

Removes the selected folder from the list of specified folders.

Remove All

Removes all the selected folders from the list of specified folders.

Include subfolders

Expands the where-used search to the subfolders of the location specified in by either previous check box.

Teamcenter Revision Rule

Specifies the revision rule used to create the Bill of Materials (BOM) so the number of links in the managed document can be compared to the number of links in the unmanaged file.

Use Precise BOM View when adding assemblies

Sets the Bill of Materials (BOM) as fixed links. This option is not linked to the Revision Rule, and selecting this option does not change the Revision Rule to Precise Only.

Automatically retry documents that fail to load

Attempts to add files to Teamcenter from the group of files that failed to load on the previous attempt.
