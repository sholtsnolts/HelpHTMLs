Change the Document Name Formula

You can create a document name formula composed of document properties.

  1. Exit Add to Teamcenter Interactive.

  2. Start Solid Edge.

  3. Click the File tab and choose Manage→Teamcenter to turn off Teamcenter.

  4. Choose File menu→Settings→Options→Helpers tab.

  5. Locate the entry for Document Name Formula and click Change.

  6. Select a property from the Property list and click Add.

  7. Continue to select and add the properties you want to use as the formula, and then click OK.

    Example:

    You can combine properties with additional characters to separate the two properties, such as [Document Number]-[Revision Number].

  8. On the Solid Edge Options dialog box, click OK.

    Tip:

    Remember to enable Teamcenter the next time you want to use Solid Edge in the Teamcenter-managed mode.

  9. Start Add to Teamcenter Interactive.
