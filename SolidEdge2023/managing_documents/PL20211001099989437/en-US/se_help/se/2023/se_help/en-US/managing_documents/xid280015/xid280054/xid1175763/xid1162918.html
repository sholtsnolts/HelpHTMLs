Create Package dialog box (SEEC)

Manages the process of creating a compressed file of the cache workspace in a Teamcenter-managed environment.

Workspace Name

Displays the name of the current workspace.

Description

Provides information about the current workspace.

Unmanaged folder location

Defines the location for the unmanaged compressed file created by the Create Package command.

Filename (*.zip)

Defines the name of the compressed file that contains the contents of your current cache workspace.

Write file properties to documents

When checked, the file properties are written in the file equal to what you see in Teamcenter.

Create

Creates the compressed file.
