Modify Solid Edge files using command line options

The Modify program updates Solid Edge files based on entries in the analysis report spreadsheet. New property values are saved in the unmanaged Solid Edge files.

You can modify files using options in the Data Preparation Utility or from the command line by typing DataPrepUtility.exe /ModifyProperties and applying the following options:

Note:

The default location for the DataPrepUtility.exe program is \Program Files\Siemens\Solid Edge 2023\Program.

Example:

DataPrepUtility.exe /ModifyProperties /file "D:\new folder\dataPrepUtility.xlsm" /viewlog

Option

Use

/file <spreadsheet file>

Specifies the path of the spreadsheet file used as input.

/ShowDialog

Displays the Modify Files dialog box.

Specify this option when you want to see the Modify Solid Edge Files dialog box. The dialog box is displayed with all specified options selected. You must manually start the process by clicking OK in the dialog box.

/?

Displays help for command line options.

/viewlog

Options log file.

Log file is opened when processing completes.
