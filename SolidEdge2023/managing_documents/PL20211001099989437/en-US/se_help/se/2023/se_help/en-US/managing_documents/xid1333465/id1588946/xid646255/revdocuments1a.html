Revising documents

As you work within the structure of a document, you will need to update document revision levels for each file you change. There are several options for revising documents in Structure Editor.

  • Revise command

  • Revise All command

  • Revise Selected command

Note:

CAD documents not created or owned by Solid Edge (such as a driven reference) must be revised in the foreign CAD system. Alternately, you can Take ownership of the foreign CAD document so that Solid Edge becomes the owner of the item revision. Then the document can be revised.

For more information, see Multi-CAD in the Teamcenter managed environment.
