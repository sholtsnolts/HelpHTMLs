Activity: Create a drawing

Try this activity, to learn how to create a drawing in a managed environment using an existing assembly from Teamcenter.

After completing this activity, you will be able to:

  • Use the assembly to create a drawing.

  • Save and upload a drawing into Teamcenter.

Instructions for loading the activity files are found in Activity data set and configuration requirements. The activities assume ISO templates are loaded.

1: Verify Solid Edge is still running with Teamcenter enabled.

  • Verify that Teamcenter is displayed in the application title bar.

2: Search for the assembly you created in the first part of the activity.

Click Open .

In the Filter Options pane, set the Files of type to Assembly Documents (*.asm).

In the Advanced Search pane, from the Saved Searches list, select Item Name.

In the Criteria column, specify the Item Name you gave the assembly in the previous activity. Type Handle Assembly.

Click Search.

Select the item from the list and click Open.

3: Create a drawing.

On the File menu, choose New→Drawing of Active Model.

The Create Drawing dialog box is displayed.

Choose the Crate standard views option and click OK.

4: Save the drawing.

On the Quick Access toolbar, click Save .

The Item ID, Revision, and Item Name are assigned to the document.

Note:

By default, the draft document is created in the same Item Revision and gets the same Item ID as the first view on the drawing.

Change the Item Name to Drawing of Handle.

Enter a Dataset Description of Drawing Created in Activity 4.

Ensure the Action is set to Upload for both the draft document and assembly, and click Perform Actions.

5: Close the drawing and assembly.

Choose Teamcenter tab →Close.

In PathFinder, the status of the assembly is Checked Out To You because the file has not been closed.

Choose Teamcenter tab→Close →Close All.

This concludes the activity.

6: Activity summary

In this activity, you learned how to create a drawing using managed data, and recognize the relationship between the draft document and the Item Revision and Item ID of the first view on the drawing. You also learned to how to upload documents into Teamcenter.
