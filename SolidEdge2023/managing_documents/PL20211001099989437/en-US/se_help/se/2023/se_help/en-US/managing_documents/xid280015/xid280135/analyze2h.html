Define custom properties before analysis

If your Solid Edge files contain custom properties, define those properties before running the Data Preparation Utility and adding the file to Teamcenter.

  1. Access the Data Preparation Utility spreadsheet by opening datapreputilitiestemplate.xlsm, available in the \Program Files\Siemens\Solid Edge 2023\Program folder.

  2. Begin defining your columns for custom properties in the next available column in the spreadsheet. Use the custom property name as the column title.

    Example: SE_ASSEMBLY_QUANTITY_OVERRIDE

    Caution:

    In the Teamcenter worksheet, the column headings for columns A (Path) through L (Last Saved Version) are required and should not be edited or removed.

  3. Save and close datapreputilitiestemplate.xlsm.
