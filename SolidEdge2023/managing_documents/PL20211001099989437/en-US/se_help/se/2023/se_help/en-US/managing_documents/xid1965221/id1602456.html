Activity: Use the Hide All option

After completing this activity, you will understand how to use the Hide All option to effectively work with large assemblies in the Teamcenter-managed environment.

Instructions for loading the activity files are found in Activity data set and configuration requirements. The activities assume ISO templates are loaded.

1: Open the valve body assembly you created in the previous lesson and set Hide All to Yes.

Start Solid Edge with Teamcenter enabled.

Click Open .

Log in to Teamcenter when prompted.

The Open File dialog box displays and the Browse field is set to the location where it was last used.

In the Filter Options pane, set the Files of Type to Assembly documents (*.asm).

In the Load Options pane, set the Assembly Open As option to Small Assembly, and set Hide All to Yes.

Opening assemblies using the Hide All option enables you to effectively manage various components of an assembly. All components are hidden and only the direct first-level children of the assembly being opened are downloaded to cache.

Select the completed valve body assembly created in the previous lesson.

In the Open File dialog box, click Open .

2: Use the shortcut command, Expand, to show the Modified Handle Assembly.

  • In PathFinder, right-click Modified Handle Assembly and then click Expand.

    The branch of the parent assembly is downloaded to cache, enabling navigation at that level.

3: Show all components of Modified Handle Assembly.

  • In PathFinder, right-click Modified Handle Assembly, and click Show All.

    The modified handle assembly is displayed, enabling modification at that level without having to display the entire parent assembly.

4: Open and edit the subassembly.

In PathFinder, select Modified Handle Assembly, right-click, and select Open to edit the subassembly.

Right-click the assembly and select Activate.

Select the Item that is the handle cap, and in the View tab→Style group, set the Face Overrides to Black (clear).

Save your change.

5: Close the Modified Handle Assembly.

  • Choose Teamcenter tab→Close→Close.

6: View all components of the valve body assembly.

In PathFinder, select the Valve Body Assembly, right-click, and select Show All.

Fit the view and save the assembly.

7: Close the valve body assembly and check in the document.

Choose Teamcenter tab→Close→Close All.

In the Upload dialog box, set the action to Check In and click Perform Actions.

Click the Close button in the upper-right corner of this activity window.

8: Activity summary

In this activity, you learned how to use the Hide All command to open selected portions of assemblies, enabling you to work more efficiently.
