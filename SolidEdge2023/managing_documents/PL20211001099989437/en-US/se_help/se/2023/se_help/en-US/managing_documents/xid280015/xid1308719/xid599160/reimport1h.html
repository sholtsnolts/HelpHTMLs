Reimport documents to Teamcenter (workflow)

  1. Start the Add to Teamcenter-Interactive program and begin a new session.

    You have two options:

    • From the Start menu, select Siemens Solid Edge 2023→Choose PDM Integration→Add to Teamcenter Interactive. Then choose Start New Session and Add Document.

    • In Solid Edge, select File menu→Manage→Add Document to Teamcenter.

  2. On the Open dialog box, select the documents you want to reimport to Teamcenter.

    You can select a single document or multiple documents from the Open dialog box.

    Note:

    Folders, .xls, and .csv files are not acceptable input.

  3. Click Open.

    The upper-left pane of the Add to Teamcenter Interactive window displays each document expanded to one level. Existing property data is shown based on the currently displayed columns.

    Tip:

    To include additional columns, right-click and select the Columns command. The Format Columns dialog box provides options for displaying property information in addition to what is present by default.

  4. Determine the parameters for this session by clicking Session Settings , and then selecting the options from the Session Settings dialog box.

  5. Select Find All Linked Documents from the toolbar.

    The documents related to the documents you have selected as input to Teamcenter are added into the upper-left pane of the Add to Teamcenter Interactive window. The Progress Log summarizes the activity and displays the time it took to complete.

  6. Select Find All Related Drafts from the toolbar.

    The associated draft documents are inserted into the list of documents as children of the objects they reference.

  7. Evaluate the readiness of the documents you selected for reloading into Teamcenter.

    The Issues column displays a cell for each document in either green, yellow, or red to indicate the document’s readiness to be input to the managed environment. Yellow cells indicate caution and the issue should be investigated before importing the documents to Teamcenter. Red cells indicate problems that should be resolved before continuing with importing the documents to Teamcenter.

  8. Resolve any problems identified in the Issues column of the selected documents and click Run Unmanaged Validations to confirm the issues are resolved.

    Consult the Error Assistant to review and correct any reported problems.

    Tip:

    See the help topic Resolving problems identified in the Issues column for additional information.

  9. Save any modifications you made to the documents’ properties.

  10. Select Retrieve Data from Teamcenter from the toolbar.

    The unmanaged data is compared to the Teamcenter managed data and the existing managed information is displayed in the upper-right pane of the window.

  11. Determine the appropriate action for the documents being reimported to Teamcenter.

    In the Teamcenter Action column, you can choose from:

    • Existing – Use Existing

      This action is automatically set for you if the document was previously imported and it has not changed since it was last imported.

    • Existing – Overwrite

      This action is automatically set for you if the document was previously imported but was modified since it was last imported. Choosing this command overwrites existing Teamcenter data.

    • Existing – Revise

      The document was previously imported and a new revision is needed.

    • Existing – Save As

      The document was previously imported and you wish to save a copy to a new item number, revision, and item name.

    • New

      The document was not previously imported.

  12. Consult the Issues column to evaluate the readiness of the documents you selected for loading into Teamcenter.

    Consult the Error Assistant to review and correct any reported problems.

  13. Select the Run Teamcenter Validations command as often as necessary to help you clear any problems noted in the Issues column.

  14. Select the Perform Actions command .

    A log file is generated that contains details about the transaction. The Progress Log summarizes whether the documents passed or failed reimport to Teamcenter.

  15. Successful import of your documents into Teamcenter completes the session.
