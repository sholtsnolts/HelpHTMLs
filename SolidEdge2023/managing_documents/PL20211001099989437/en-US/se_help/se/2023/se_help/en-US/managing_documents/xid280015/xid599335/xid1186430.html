Data Preparation dialog box

Defines the parameters related to the evaluation of your Solid Edge files and produces an analysis report, a spreadsheet for reviewing your files before importing them into Teamcenter.

Prepare data for

Defines the managed application to which you want to import your unmanaged data. You can choose Teamcenter or Unmanaged.

Set Working Folder

Opens the Browse dialog box so you can define the location for the log files generated by the commands in the Data Preparation dialog box. The default folder is \Users\<username>\AppData\Roaming\Siemens\Solid Edge\Version 223\Log Files\Data Preparation.

Open Working Folder

Opens the folder you defined as your working folder.

Generate Import List

Performs a dry run of the data import to identify potential problems that should be addressed before loading the data into a managed environment. This process creates an ordered .xlsm list of your files.

Select Files/Folders

Opens the Data Integrity Check dialog box where you can select the files or folders you want to include in the dry run.

Use Existing List

If you already performed a dry run, and have an ordered or unordered csv or txt file, you can select it for inclusion in the new dry run.

View Properties

Defines the location for the text file containing the results of the analysis. The default file name of the report file is AnalysisOfSolidEdgefiles.txt

Broken links XML file

If the system found any broken links, it displays the path to the file listing the broken links, BrokenLinkLog<date_time>.xml.

Ordered/Unordered CSV file/text file

Displays the path to the ordered or unordered csv or txt file.

View Log

Displays the AnalysisOfSolidEdge*.txt file.

Show Link Information

Displays the DataPrep_Links*.xlsx file.

Generate Custom Properties Report

Displays the Custom_Properties*.txt file.

Analyze

The utility analyzes the properties and displays the PropertyList*.xlsm file.

Fix Broken Links

Fixes the broken links the utility found during the dry run.

Browse for spreadsheet

Displays the path to the spreadsheet containing broken/replacement links information.

Locate folder containing SE files

Displays the path to the folder containing the data you want to import.

Modify Properties

Applies the changes you made to properties.

Spreadsheet containing property information

Displays the path to the spreadsheet containing the data you want to import, PropertyList*.xlsm.

Modify Properties

The utility applies the property changes.

Cancel

Closes the Data Preparation dialog box.
