New Workflow Process dialog box

Displays the item revision of each document selected for the workflow. The item revisions are considered targets of the workflow process.

Right-clicking an occupied cell in the grid displays a shortcut menu to change the font, sort the information, find and replace text, format columns, change column width, and hide columns. Asterisks (*) indicate required fields.

Process Name

Specifies a name for the workflow. Initially, the item revision of the first object added as a target is used as the default process name.

Description

Provides an explanation of the workflow process.

Process Template Filter

Determines whether all Teamcenter process templates are shown in the process template list or only the ones assigned to you.

Process Template

Lists the process templates available in Teamcenter.

Refresh

Retrieves a current list of process templates from Teamcenter.

Cut

Removes the selected item revision from the target list. This option is not available when only one item revision is in the list.

Expand Selected Node

Expands levels of content of the selected item revision.

Expand All

Expands all levels of content in the target area of the dialog box.

Collapse All

Collapses all expanded levels in the target area of the dialog box.

Reset

Restores the entries in the dialog box to their original condition.

Target area

Displays the item revision selected for the workflow process. Additional information displayed includes:

  • The Process Stage and Release status for item revisions that are part of a workflow.

  • The target's description and the name of the person who has the document checked out.

  • Additional details relevant to items in the target list.

When the target is a driven reference, configured assembly, or has unconfigured BOM lines, the item revision is italicized and the Details column indicates the special condition.
