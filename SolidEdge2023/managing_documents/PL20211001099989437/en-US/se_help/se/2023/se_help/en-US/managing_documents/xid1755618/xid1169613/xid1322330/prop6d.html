Project Tab, Document Properties dialog box

Displays project information associated with the document. You can customize lists for the Document Number and Project Name by editing the propseed.txt file located in the Solid Edge Preferences directory.

Note:

If any of the entries in the propseed.txt file exceed the character length of the text box, you can position your mouse over the text box to view a tooltip that contains the full contents.

Document Number

Displays the drawing number. The list of available document numbers is stored in the text file, propseed.txt. You can edit this file to add more document numbers to the default list.

Revision Number

Displays the revision number of the document.

Project Name

Displays the name of the project for which the document is being created. The list of available project names is stored in the text file, propseed.txt. You can edit this file to add more project names to the default list.

Hardware part

Designates the part as hardware part.
