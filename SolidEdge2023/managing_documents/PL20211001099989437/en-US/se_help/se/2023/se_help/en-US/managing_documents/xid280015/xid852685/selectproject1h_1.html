Assign a Teamcenter project to an item


You can assign a project to a Teamcenter-managed document when you create a new document, update an existing document, or revise a document using the Select Projects dialog box.

Note:

Your project access is determined at the time you log into Teamcenter. If project access changes during the time you are logged in, the project list will not be updated until you reestablish your session by logging off and logging back in again.

New documents

  1. In the Teamcenter-managed environment, create and save a document.

  2. In the New Document dialog box, assign the mandatory properties (Item ID, Revision, Item Name) by clicking Assign All or by entering the values.

  3. To access the Select Projects dialog box and assign a project to the document, you can:

    • Double-click the Project ID cell.

    • Click the Assign Project ID command.

    Note:

    The Teamcenter preference, SEEC_ProjectID_Default determines the default project ID used when you upload a new document to Teamcenter.

  4. From the projects listed in the dialog box, select the check box next to the project to associate it with the selected object(s).

  5. Click OK to return to the New Document dialog box.

    The assigned projects are displayed in the Project ID column of the New Document dialog box.

  6. Click Perform Actions to create the item in Teamcenter with the properties and projects you assigned.

Existing documents

  1. In the Teamcenter-managed environment, open an existing document.

  2. Click the Assign Project ID command to display the Select Projects dialog box.

  3. From the projects listed in the Select Projects dialog box, you can:

    • Select the check box next to the project to associate it with the selected document.

    • Remove the project assignment by clearing the check box.

  4. On the Select Projects dialog box, click OK.

Multiple components within an assembly

  1. In the Teamcenter-managed environment, open an assembly.

  2. Within the context of an assembly, you can either:

    • Use the Assign Project ID All command to assign all documents within the assembly to the select project.

    • Use PathFinder to select individual or multiple documents within the assembly for project assignment.

  3. To select individual or multiple documents within the assembly for project assignment, click the Assign Project ID command.

  4. From the projects listed in the Select Projects dialog box, you can:

    • Select the check box next to the project to associate it with the selected document.

    • Remove the project assignment by clearing the check box.

  5. Use the following table to determine which projects are assigned to the selected documents.

    Check box selection

    Definition

    None of the objects are assigned to the available project.

    All selected objects are assigned to the project.

    Some of the selected objects are assigned to the available projects.

  6. On the Select Projects dialog box, click OK.
