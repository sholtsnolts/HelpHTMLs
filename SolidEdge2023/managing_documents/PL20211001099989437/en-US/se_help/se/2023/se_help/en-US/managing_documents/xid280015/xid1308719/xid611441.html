Add to Teamcenter-Interactive

The Add to Teamcenter-Interactive program is available for all users of the Teamcenter Integration for Solid Edge (SEEC).

This tool provides you an interactive process for adding Solid Edge 3D and draft data into Teamcenter. With a focus on working with unmanaged data that is of a product or project in size, it is particularly suited for supply chain collaboration and reintegration of formerly managed content into Teamcenter.

If Add to Teamcenter-Interactive is new to you, see Getting started with Add to Teamcenter-Interactive.
