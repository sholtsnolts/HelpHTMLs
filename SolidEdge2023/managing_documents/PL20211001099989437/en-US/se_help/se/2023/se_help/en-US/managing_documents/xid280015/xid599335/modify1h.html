Modify Solid Edge files using the Data Preparation Utility

Use this procedure to modify the properties of unmanaged Solid Edge files according to the specifications you enter in the analysis report, PropertyList<timestamp>.xlsm. You must have write access to the unmanaged files to save the properties.

  1. Choose Start→Siemens Solid Edge 2023→Choose PDM Integration.

  2. On the ribbon, click Utility.

  3. Click the Modify Properties tab.

  4. Verify that the analysis report displayed in the dialog box contains the unmanaged Solid Edge files to be modified along with the new property information.

    This file was generated with the Generate Import List option. The default location for the report is \Users\<username>\AppData\Roaming\Siemens\Solid Edge\Version 223\Log Files\Data Preparation\PropertyList_<timestamp>.xlsm.

  5. Click Modify Properties.

    The length of time required for processing depends on the number of files to process. A status notification is displayed to let you know when processing is complete. The results of processing the data are also written to the log file FileRename_<timestamp>.log.

  6. (Optional) Click View Log to view the contents of the log generated during modification of the unmanaged file properties.

    You can now add your unmanaged files into Teamcenter.
