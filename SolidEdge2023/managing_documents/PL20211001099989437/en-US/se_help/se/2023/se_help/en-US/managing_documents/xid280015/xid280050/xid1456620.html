Create new documents while working offline

  1. On the Cache Assistant dialog box, click Work Offline.

    Notice the button changes to Working Offline. The title bar and the user session information indicates you are working offline.

    Tip:

    To work offline each time you start Solid Edge, in Solid Edge Options→Manage→General tab, select Always work offline when Solid Edge starts.

  2. Create and save a new Solid Edge, Excel, Word, or image (bmp, jpg, png) document.

    A temporary filename is assigned to the document. The filename indicates the document was created while offline from Teamcenter.

    Example:

    offline-102831.par

    offline-102831.dft

  3. Close any open documents.

  4. To upload or check in to Teamcenter the documents you created when you return online, open the Cache Assistant dialog box and select either:

    • Check In New to have new documents created while you were offline from Teamcenter automatically checked in to Teamcenter.

    • Check In All to have all the documents in your cache automatically checked in to Teamcenter.

  5. Return online to Teamcenter by clicking Work Online.

    Notice the button changes to Working Online and the Upload dialog box displays the documents you created while offline.

  6. Set the Action to either Check In or Upload.

  7. Set the Item Type for each document, or click Assign All to have property data assigned for you.

  8. Click Perform Actions.
