Teamcenter - Save As Translated dialog box

Specifies the file type, dataset type, dataset name, and dataset description of a translated file when it is saved to Teamcenter. Options presented in this dialog box are defined by the Teamcenter preference SEEC_Save_Translation.

Note:

For a full description of the preference, see the Teamcenter Preferences section of the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.

Save as type

Displays the translation file types available to you as defined by the Teamcenter preference SEEC_Save_Translation.

Dataset type

Defines how your data is connected to the Item Revision in Teamcenter. The list of dataset types available is defined by the Teamcenter preference SEEC_Save_Translation.

Object name

Active document being translated.

Relation

Displays the relation defined by the data model in Teamcenter.

Dataset Name

Displays the name of the dataset.

Dataset Description

Provides an opportunity to describe the data contained in the dataset.

File name

Displays the name of the file as configured by the Teamcenter preference SEEC_Save_Translation.

Dispatcher

Defers the translation of the active document to a Dispatcher service request for completion outside of Solid Edge.

Translate

Performs translation of the active document while you are in Solid Edge.

Options

Displays the Export Options dialog box so that you can chose translation options for various output formats.
