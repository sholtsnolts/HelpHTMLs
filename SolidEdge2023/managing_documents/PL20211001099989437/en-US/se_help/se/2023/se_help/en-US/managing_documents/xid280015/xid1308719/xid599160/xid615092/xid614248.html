Resolving problems identified in the Issues column

Once you select unmanaged documents for adding to Teamcenter, the Issues column in the upper-left pane of the Add to Teamcenter Interactive window displays indicators regarding the preparedness of the documents you have selected.

Each cell can display green, yellow, or red indicating the severity of any action needed to ensure your documents are correctly added to the Teamcenter managed environment.

Green

No action required. Documents are ready for adding to Teamcenter.

Yellow

Investigation required. Caution should be taken before documents are added to Teamcenter.

Red

Action must be taken before the document can be successfully added to Teamcenter.

The same indicators display in the upper-right pane of the Add to Teamcenter Interactive window when you retrieve data from Teamcenter and run Teamcenter validations.

These indicators display based on identification any of the following:

  • Document Status not equal to Available

  • Duplicate file names

  • Duplicate Item IDs

  • Item IDs with more than one revision

  • Broken links

  • Problems with items that are a Family of Parts

  • Revision mismatches between drafts and linked 3D models with the same Item ID.

Each of these conditions should be addressed before continuing with adding your documents to Teamcenter.

As you click each cell of the Issues column, the Error Assistant provides suggestions for resolving each issue identified. The information continues to display until you click a different cell in the Issues column.
