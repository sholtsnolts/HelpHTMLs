Concept Workspace command

The Concept Workspace command creates a defined workspace for exploring design ideas without impacting the data stored in Teamcenter. When you create a Concept Workspace, a folder is created in the default SEEC cache location defined in Settings→Options→File Locations. You can create multiple Concept Workspaces, each of which you define using the Create Concept Workspace dialog box. When you complete your experimental design, you can integrate the design data can be integrated into Teamcenter or you can delete the workspace.

Note:

The Teamcenter preference SEEC_Enable_ConceptWorkspace enables the Concept Workspace command.
