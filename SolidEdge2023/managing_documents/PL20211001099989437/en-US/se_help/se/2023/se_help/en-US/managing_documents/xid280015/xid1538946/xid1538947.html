Generate logs for incident reporting

Enabling Debug level logging impacts performance and should only be used when you need to collect logs for incident reporting.

  1. Before you sign in to Teamcenter, choose File menu→ Settings→Options→Manage→ Login.

  2. In the Generate logs for Incident Report section of the dialog box, select the Enable debug logging level check box.

  3. Click Apply.

    All log files are saved to the location defined in Solid Edge Options→File Locations→Log files with the exception of soaclient*.txt which is saved to %TEMP%.

  4. Notice that the Syslog location is blank.

    Once you sign in to Teamcenter, the system retrieves the location of the Syslog file from your Teamcenter server and displays it in the dialog box.

  5. Click OK.

  6. To disable Debug level logging, exit and then restart Solid Edge.

  7. Choose File menu→ Settings→Options→Manage→ Login and clear the Enable debug logging level check box.
