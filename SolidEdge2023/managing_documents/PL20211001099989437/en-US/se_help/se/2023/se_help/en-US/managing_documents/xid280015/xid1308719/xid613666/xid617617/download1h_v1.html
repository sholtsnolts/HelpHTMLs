Download a document to the local cache

In a Teamcenter-managed document

  1. Access the Cache Assistant dialog box using one of the following methods:

    • Select File menu→Manage tab→Cache Assistant.

    • From an open document, choose Teamcenter tab→Cache group→Cache Assistant.

  2. Click Download.

  3. Browse for and select the document you want to download.

  4. Click Open.

Note:

The Download command is also available on the Assembly shortcut menu.
