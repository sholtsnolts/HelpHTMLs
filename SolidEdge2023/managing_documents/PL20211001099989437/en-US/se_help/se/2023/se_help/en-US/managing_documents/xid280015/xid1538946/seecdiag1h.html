Run a SEEC diagnostics scan

  1. Start the SEEC Diagnostic Application by choosing Start→Siemens Solid Edge 2023→SEEC→ Diagnostic Application.

    Caution:

    You should exit Solid Edge, Structure Editor, and Add to Teamcenter before running diagnostics.

  2. In the SEEC Diagnostics dialog box, verify the selection of the 2-tier or 4-tier button identifying your Teamcenter connection type.

  3. Supply the location of information associated with your specific configuration.

    For a 2-tier configuration, browse for the location of your TC_ROOT and TC_DATA information.

    For a 4-tier configuration, supply the URL of your server.

  4. In the Diagnostic Package Location, define a folder where the files created by the scan will be stored.

  5. Click Scan.

  6. Sign in to Teamcenter when prompted.

    The scan begins and you are notified when it completes.

  7. To dismiss the Scan Complete message box, click OK.

    The SEECDiagnostic_timestamp.txt log file is created in the folder you defined in the Diagnostic Package Location. The folder also contains the log files and other files generated by running the diagnostics application.

  8. Click Close.
