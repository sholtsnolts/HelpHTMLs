PDM Client - Application Launcher dialog box

Activates a product data management (PDM) client for which you are licensed. Optionally, you can control what Start menu entries you want and also start applications using the options on this dialog box.

For information specific to each tab, see:

  • Home

  • Teamcenter

Home

Borrowing

Displays the License Borrowing dialog box that provides information about the licenses available to you. Use this dialog box to borrow a license for Solid Edge.

Utility

Starts the Solid Edge License Utility so that you can specify your Solid Edge license type.

Cloud Gateway

Enables you to register your cloud-enabled license.

Refresh

Rereads your license file and updates the list of PDM integrations available so that you do not have to exit an application to change your active client.

Diagnostics

Starts the Solid Edge Data Management Diagnostic Application.

SEEC

Adds or removes Start menu entries related to the Teamcenter Integration for Solid Edge (SEEC).

Design Manager

Starts Design Manager.

Solid Edge

Starts Solid Edge.

Exit

Closes the PDM Client - Application Launcher dialog box.

Teamcenter

Utility

Starts the Solid Edge Data Preparation utility.

Diagnostics

Starts the SEEC Diagnostics tool.

Teamcenter Environments

Displays the Define Teamcenter Environments dialog box.

Templates

Starts the Import Solid Edge Templates utility.

Add to Teamcenter

Starts Add to Teamcenter.

Add to Teamcenter-Interactive

Starts Add to Teamcenter-Interactive.

Structure Editor

Starts Structure Editor.

Solid Edge

Starts Solid Edge.

Exit

Closes the PDM Client - Application Launcher dialog box.
