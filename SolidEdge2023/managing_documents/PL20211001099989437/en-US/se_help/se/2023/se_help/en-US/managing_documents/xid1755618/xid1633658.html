Release documents

In Solid Edge data management and in Design Manager, you can release documents using the One Step Workflow command . In Design Manager, you can also release documents using the Release command , as explained in Releasing documents using the Release command.

To learn more about releasing documents, see Releasing documents (Solid Edge data management) and Releasing documents (Design Manager).

Prepare to release documents

Before you can release documents using the One Step Workflow command, you must:

  • Enable Solid Edge data management, as explained in Turn on Solid Edge data management.

  • Set the vault location, as explained in Define a Solid Edge vault.

  • Add reviewers to the Workflow Reviewers list, as explained in Define workflow options.

Release documents in a single step

  1. In Solid Edge data management or Design Manager, open the document you want to release.

  2. Do one of the following:

    • In Solid Edge data management, choose Data Management tab→Workflow group→One Step Workflow .

    • In the Design Manager document list, click the Current Filename cell for the document of interest and then choose Home tab→Workflow group→One Step Workflow .

  3. In the Workflow Title box, enter a title for your workflow.

  4. (Optional) In the Workflow Description box, enter a description for your workflow.

  5. Set Workflow Type to Quick Release.

  6. (Optional) In the Workflow settings section, do any of the following:

    • To include draft files in your workflow, select Include Drawings. If you choose to include to include drawings, you can also choose to update drawings by selecting Update Drawings. Updating drawings impacts performance.

    • To create a report that lists all the parts in the assembly, select Generate Assembly Report (Part List).

      This option generates a report for the assembly and each subassembly.

    In the Workflow Action section, the Approved option is already selected.

  7. Click Start.

    If you defined locations for release management, the system saves the documents in the locations you specified. Otherwise, it saves the documents in the same location from which you opened them.

    The system sets the document status to Released.

Send documents to review

  1. In Solid Edge data management or Design Manager, open the document you want to release.

    Note:

    Make sure to open a document from a shared location (UNC path).

  2. Do one of the following:

    • In Solid Edge data management, choose Data Management tab→Workflow group→One Step Workflow .

    • In the Design Manager document list, click the Current Filename cell for the document of interest and then choose Home tab→Workflow group→One Step Workflow .

  3. In the Workflow Title box, enter a title for your workflow.

  4. (Optional) In the Workflow Description box, enter a description for your workflow.

  5. Set the Workflow Type option to One Step Release.

  6. Select a reviewer from the Reviewers list.

    The Reviewers list displays the names of the reviewers you added in the Solid Edge Options dialog box→Workflow tab. If the list is empty, you have not designated any users as reviewers.

  7. (Optional) In the Workflow settings section, do any of the following:

    • To include draft files in your workflow, select Include Drawings. If you choose to include to include drawings, you can also choose to update drawings by selecting Update Drawings. Updating drawings impacts performance.

    • To create a report that lists all the parts in the assembly, select Generate Assembly Report (Part List).

      This option generates a report for the assembly and each subassembly.

    • To send email notification informing colleagues of the workflow, select Email Notification and then add the email addresses of all the people you want to notify of the task, separated by commas (,).

  8. Click Start.

    The system informs you that it sent the document for review and set the status to In Review, and gives you the opportunity to view the associated log file.

    The designated reviewer can review the documents or delegate the workflow task.

View additional properties

When working with One Step Workflows, you can view the following additional properties:

  • SE Workflow Title

  • SE Workflow History (Workflow description)

  • SE Workflow Reviewer

  1. In Windows Explorer, navigate to the location of the Solid Edge vault (root folder).

  2. Right-click any column and select More.

  3. Scroll down and select the properties of interest.

Generate additional ouput files

After you completed a One Step Workflow, you can generate additional files. The supported file formats are:

  • *.asm, *.par, *.psm – JT, PDF (2D only)

  • *.dft – DXF, PDF (2D only)

  1. Specify the formats of interest in the file SEWorkflowTrans.ini, located in \Program Files\Siemens\Solid Edge 2023\Preferences.

  2. Run SEWorkflowTranslation.exe, located in \Program Files\Siemens\Solid Edge 2023\Program.

    Solid Edge saves the files in the same location as the vault.
