Open a Teamcenter-managed document

This procedure describes opening documents that have been added to Teamcenter.

  1. Choose File menu→Open tab.

  2. On the Open page, do one of the following:

    • Open one of the last documents you worked on by selecting it from the Recent Files list at the right side of the Open page.

    • Use the Open File dialog box (SEEC) to browse to a document to open, by clicking Open .

  3. In the Open File dialog box, set the Browse location to the folder containing the managed document you want to open.

    To narrow the list of files to those of a certain type, open the Filter Options docking window and set the Files of type buttons. You can filter on combinations of file types by selecting more than one of the Solid Edge file type buttons.

  4. Select the document to open from the list of documents.

    If you do not know the location of the document you want to open, you can use the Advanced Search docking window to define search criteria to locate the document. To learn about search capabilities, see Advanced Search dialog box (SEEC).

    Tip:

    To learn how to use docking windows, see Manipulate multiple- and single-page windows.

  5. (Optional) Set additional options for opening a document by selecting the docking windows:

    Open Options

    Contains options for Check Out Information, Revision Rules, and Download Options-Overrides.

    Load Options

    Includes options for Assemblies, Display Configuration and Zones, and opening Drawings.

    For example, to open the document for viewing, click the Open Options docking window, and then select the Read-Only check box.

    Tip:

    Documents opened as read-only display the Read-Only Document Status Indicator in the graphics window and [Read-Only] is displayed the title bar. To learn more about read-only documents, see Working with Read-Only documents.

  6. Click Open.

    When you open a document from Teamcenter, the document is checked out to you and copied to the local cache.
