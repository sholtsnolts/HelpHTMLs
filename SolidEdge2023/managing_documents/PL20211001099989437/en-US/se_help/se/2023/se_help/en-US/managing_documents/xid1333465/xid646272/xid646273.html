Administering Structure Editor

Application administrators are responsible for the operation and maintenance of an application. Their duties can range from installing software, loading product updates, controlling privileges, and performing backups. The following features are available to assist administrators with their tasks:

  • The Solid Edge readme.htm file in \Program Files\Siemens\Solid Edge 2023 contains system requirements, installation instructions, and additional information regarding Solid Edge.

  • The Solid Edge Administrator program gives you the opportunity to define policies and easily change the policies later.

  • You can use features on the Solid Edge Options dialog box to:

    • Set folder locations for log files.

    • Determine the location of your local cache.

    • Define the Document Name Formula.

  • You can use the settings within Structure Editor Options to:

    • Determine the behavior when a managed document is closed.

    • Define assembly structure synchronization.

    • Control Teamcenter client communication system settings (TCCS).
