Session page (User Settings dialog box)

In Cache Assistant

Sets the group or role if you are part of more than one role or group in a managed environment. Changes to role or group are not implemented until you select OK.

Person

Displays the name of the person logged in. This value is read-only.

User ID

Specifies your login name. This value is read-only.

Group

Displays the group to which you belong and displays a drop-down list of groups to which you may change. If you change your group, your role is changed to your default role.

Role

Displays the role you currently have within the group to which you belong and displays a drop-down list of roles to which you may change.

Volume

Displays the current library volume information.

Project ID

Displays the list of projects in Teamcenter. A blank entry indicates there is no default project defined. Use the Teamcenter preference SEEC_ProjectID_Default to determine whether the default project is the last Project ID specified in CPD, or the Project ID as defined in the User Settings dialog box in Solid Edge, Hosted Active Workspace, or Teamcenter RAC.

In a Solid Edge document

The session information available to you when a document is open in Solid Edge, is displayed in the User Session Info docking pane.

Teamcenter

Displays the URI and name of the database where you are connected.

Session

Presents information from the time of logon including: person, users id, group, role, volume, local volume, location code, and project ID.

To change session information, access the User Settings dialog box by clicking Start User Settings dialog.

Connection Information

Teamcenter Version, Site ID, Locale, Encoding and Syslog data.

The information displayed can vary depending on the appbar_title_contents Teamcenter preference definition. For additional information regarding Teamcenter preferences, see the Teamcenter Preferences section of the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.
