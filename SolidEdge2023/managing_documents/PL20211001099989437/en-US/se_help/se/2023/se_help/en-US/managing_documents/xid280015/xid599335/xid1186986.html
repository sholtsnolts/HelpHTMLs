Generate a list of reverse links

Use this procedure to generate a list of reverse links, such as inter-part copies and part copies, before importing your unmanaged documents into the managed environment.

  1. Choose Start→Siemens Solid Edge 2023→Choose PDM Integration.

  2. On the ribbon, click Utility.

  3. Select the document management application for which you are preparing your data:

    • Teamcenter

    • Unmanaged

  4. Define a location for the files generated as output of the analysis by clicking Set Working Folder.

    Note:

    By default, the folder, Data Preparation, is created in \Users\<username>\AppData\Roaming\Siemens\Solid Edge\Version 223\Log Files to maintain all files generated during the file analysis.

  5. In the Generate Import List tab, click Select Files/Folders.

  6. Select whether the files to be analyzed are located in a file or in a system folder.

    Use this option

    To do this

    Add Files

    Identify individual files for analysis or define the location of either the ordered or unordered CSV file containing the list of files for analysis.

    Add Folders

    Identify the location of the system folders containing the files for analysis.

  7. Browse for the files or folders specified in the previous step.

    Tip:

    • When you choose to Add Folders, you can select the Include Subfolders check box so that any subfolders are included for you.

    • You can click Types to filter your selection by file types (.dft, .asm, .par, .psm, .pwd or All documents including non-Solid Edge documents).

    • You can click Remove or Remove All to modify the list of files or folders.

  8. Click Generate Import List to start the analysis and generate the ordered list of files for input into the managed environment.

    The length of time required for processing depends on the number of files analyzed. When processing is finished, several files are generated for you, including:

    • AnalysisOfSolidEdgefiles_<timestamp>.txt

      Contains a summary of the file analysis.

    • DataPrep_Links_<timestamp>.xlsx

      Lists relation types if the relation type is a reverse link.

    • OrderedList_<timestamp>.CSV

      Contains the primary list of files to be imported into the managed environment in the order in which they are to be imported.

    • PropertyList_<timestamp>.xlsm

      Displays the results of the data analysis in a separate window. For more information, see Analysis report.

    • Unordered_<timestamp>.CSV

      Contains a list of files to be imported into the managed environment.

    The PropertyList_<timestamp>.xlsm spreadsheet is displayed in a separate window. This analysis report lists the files you analyzed. The file is saved to disk in the location you defined as your working folder.

  9. In the Data Preparation dialog box, click the View Properties tab.

  10. Provide the location of your OrderedList_<timestamp>.CSV file, if the location is not provided for you.

  11. Click Show Link Information.

    The file DataPrep_Links.xlsx is saved to the working folder you specified and the file is opened in a separate window. The file indicates the number of files processed, the number of reverse links found, and the name of the reverse links.
