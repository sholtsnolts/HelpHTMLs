Using full text search for drawings

In addition to the existing property-based search on all Solid Edge file types, you can also use full text search for drawings. You use the Windows Explorer search to find Solid Edge drawings in your vault locations based on any text used inside the drawing. For information on defining a vault, see Define a Solid Edge vault.

For example, if you have a drawing that has a callout Adjusted, you can search in Windows Explorer for contents:Adjusted. Every search must be preceded by contents:.

If your search takes a long time, in Windows Explorer verify that the system uses the optimal search option. In Windows 10, from the Tools tab→Folder Options→Search tab, verify that the option When searching non-indexed locations is not set to Always search file names and contents (this might take several minutes).
