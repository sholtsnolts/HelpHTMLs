Check out a document

In Cache Assistant

  1. On the Cache Assistant dialog box, select the document you want to check out.

  2. Right-click to display the shortcut menu.

  3. On the shortcut menu, click Check Out.

In a Teamcenter-managed document

  1. In PathFinder, select the document you want to check out.

    Note:

    • If no files are selected in PathFinder, the active document is checked out.

    • If any files are part of a select set, all selected files are checked out.

  2. Choose Teamcenter tab→Manage group→Check Out.

    The Check Out command is not available if the document is already checked out.

Tip:

  • You can set the Always Synchronize Cache to Get Latest Version of The Document option on the Manage tab of the Solid Edge Options dialog box to automatically download the latest version when you open a managed document.

  • If you check out a direct document, Solid Edge downloads the associated linked (indirect) documents to your local cache.

  • If you in-place activate a part in an assembly, Solid Edge checks out the part.
