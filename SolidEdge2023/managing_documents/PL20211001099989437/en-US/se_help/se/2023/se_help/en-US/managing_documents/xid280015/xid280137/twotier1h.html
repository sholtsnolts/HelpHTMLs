Configure Solid Edge Embedded Client for two-tier deployment

If a Solid Edge Embedded Client two–tier deployment is required, Solid Edge Embedded Client should be configured after installation is complete.

  1. On your client, select Start→Siemens Solid Edge 2023→SEEC→Define Teamcenter Environments.

  2. Select the 2–Tier check box under the Teamcenter Environment URL, type a database description, and click Add.

    The Define Teamcenter 2–Tier Server dialog box is displayed.

  3. Define your Teamcenter application and data folder information.

    1. Type the location of your Teamcenter two–tier rich client application folder (TC_ROOT).

    2. Type the location of your Teamcenter data folder (TC_DATA).

    3. Click OK.

    A new row is added to the table.

  4. Click OK to save the information.

    When you are prompted to log on to Teamcenter, choose the two–tier connection from the list of environments on the Login dialog box.

Note:

Solid Edge Embedded Client can run as either a four–tier or two–tier client. However, the cache is on the selected Teamcenter Environment URL. Therefore, a four–tier connection does not use the same physical cache as a two–tier connection for the same Teamcenter environment and user ID.
