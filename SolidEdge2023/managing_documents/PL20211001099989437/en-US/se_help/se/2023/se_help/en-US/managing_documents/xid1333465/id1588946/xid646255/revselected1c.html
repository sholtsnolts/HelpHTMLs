Revise Selected command

Revises selected files within a Solid Edge structure of any file type to a new revision.

Setting an action of Revise Selected on one occurrence of a file within the structure sets the same action for all occurrences within the structure. The structure does not have to be displayed in expanded form in Structure Editor.

Note:

Solid Edge Embedded Client supports only one revision of an item within the product structure.

Links within the structure are updated to point to the new filenames. Since the files are being copied, they are not checked out of the Teamcenter database.

When a part is selected for the Revise Selected command, the action for its parent document is set to Update. This action updates the link in the parent document to point to the new file if the parent is not selected for revision. If the parent document is being revised, the link is updated in the revised copy of the parent and the original document is not modified.

When the revised documents are displayed in the Structure Editor window, the Revision field is blank and flagged with a red asterisk, indicating a required field. You can manually update the revision or you can use the auto-assign feature to generate the next available revision ID.
