General tab, Solid Edge Options dialog box

Note:

Not all options are available in all environments.

Change text to past tense when action is complete

Changes the entry in the Action column to the past tense and displays the date and time of the action. For example, suppose you check this option and revise a part. When you run the Perform Actions command with this option enabled, the entry in the Actions column is Revised <date & time>.

Update duplicate documents in tree

Updates all instances of a document in the tree. If this option is not checked, only the selected document is updated.

Revise and Increment name

Parameters for revising and incrementing file names.

Revision delimiter

Specifies the delimiter character for use with the Revisions command to separate the document name from the revision number. The default character is a dash (-).

Starting revision value

Specifies the start character or number for revisions.

Maximum length

Specifies the maximum number of characters allowed for a revision. Valid values are 1 through 5.

Excluded characters

Specifies the characters that are not allowed in a revision value.
