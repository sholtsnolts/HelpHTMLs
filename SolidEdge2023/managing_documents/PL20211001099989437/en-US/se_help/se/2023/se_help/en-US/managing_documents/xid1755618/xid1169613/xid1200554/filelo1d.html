File Locations tab, Options dialog box

Specifies the default location for documents, templates, and other objects you create or use in Solid Edge.

Note:

Not all options are available in all environments.

File types

Indicates the file type for which you want to specify a location.

Location

Displays the designated location for each file type.

Link Mgmt File

LinkMgmt.txt is an ASCII text file that contains folder information and a search algorithm that defines the folders to be searched when resolving links. The search paths are processed in no particular order, so you should not assume any order when adding the search paths to the list.

Search Scope

SearchScope.txt is an ASCII text file that is used to define locations for the Where Used and Search commands.

Backup Folder Location

Specifies the folder for the backup files.

External BOM Files

Specifies the location of the external BOM files. These are files you can create and later reference when you use the External System revision rule to open a managed document that contains links.

Solid Edge Admin

Specifies the folder and name of the options configuration file which is used by the Solid Edge Administrator to define corporate policies.

Update

Updates any changes you make to the Options.xml file. You must click Update any time you make changes to the Options.xml file.

Modify

Allows you to change the location of a folder or file.

Reset

Sets the selected file location path to the default setting.

Reset All

Sets all selected file location paths to the default setting.
