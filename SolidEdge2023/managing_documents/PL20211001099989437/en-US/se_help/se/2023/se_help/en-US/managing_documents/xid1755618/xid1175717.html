Search for documents used in other documents

To find where documents are used in other documents, you can use the Where Used command. Use options in the Where Used dialog box to search for documents anywhere on your network. After you perform a where-used search on a document, you can expand the nodes in the search results until you get the information you need.

  1. You can search for documents using the Where Used command as follows.

    • In Solid Edge or Solid Edge with data management:

      1. Open a document.

      2. Choose Data Management tab→Document group→Where Used .

        When you are a Solid Edge data management user performing a where-used search, the system might display a dialog box where a red folder indicates that the location is not indexed. Your system can very quickly search indexed locations. For information on indexing files, see Define a Solid Edge vault.

    • In the Teamcenter-managed environment:

      1. Open a document.

      2. Choose Teamcenter tab→Document group→Where Used .

    • In Design Manager:

      1. In the Design Manager document list, click the Current Filename cell for the document of interest.

      2. Choose Home tab→Assistant group→Where Used .

  2. Next:

    • If you are a Solid Edge data management user and have defined a vault containing your Solid Edge files, the system automatically searches your specified vault locations for any Solid Edge files.

      The system displays information about how many documents it processed and how many documents it found.

    • If you work in a Teamcenter-managed environment, the Where Used Results dialog box displays information for Teamcenter-managed files and files created while working offline from Teamcenter.
