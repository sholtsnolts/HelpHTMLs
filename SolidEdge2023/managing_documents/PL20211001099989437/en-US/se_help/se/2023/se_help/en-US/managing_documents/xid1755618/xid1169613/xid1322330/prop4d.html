General group, File Properties dialog box

Displays general information about the current document.

Template

Displays the Solid Edge template used to create the document.

Filename

Displays the name of the document.

Location

Displays the folder containing the document.

Type

Displays the type of document. The type is based on the application used to create the document.

MS-DOS Name

Displays the name of the document as it would appear in a File Allocation Table file naming system. These types of documents have an eight-character name and three-character extension.

Size

Displays the size of the document in bytes, kilobytes, and megabytes.

Created

Displays the date and time the document was created or copied. This information is read from the file system and is populated when the General group is expanded.

Note:

Each Solid Edge document maintains an Origination Date property, which is defined from the system creation date when a new file is first written to disk.  Explorer reads this property from the file and populates the field when the General group is expanded. If you use the Solid Edge Save As command or the Design Manager, Set Action to Copy command to create a new file based on an existing file, the creation date of the new file will display the date and time when the file was created. The creation date of the original file remains unchanged.

Modified

Displays the date and time when the document was last modified. This information is read from the file system and is populated when the General group is expanded.

Note:

Each Solid Edge document stores a Modified Date property, and it is updated each time the file is saved. Explorer reads the property from the Solid Edge document and the field is populated when the group is expanded.  

Accessed

Displays the date when the document was last opened or copied. This information is read from the file system and is populated when the General group is expanded.

Version Created

Displays the software version used to create the document.

Version Modified

Displays the software version used to modify the document.
