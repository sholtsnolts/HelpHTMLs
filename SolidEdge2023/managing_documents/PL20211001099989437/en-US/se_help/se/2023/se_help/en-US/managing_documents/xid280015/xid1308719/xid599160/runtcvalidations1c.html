Run Teamcenter Validations command

Use this command when you have content you want to validate before selecting Perform Actions to add the content to Teamcenter.

In Add to Teamcenter-Interactive

This command assists with identifying any issues with managed documents. The command:

  • Verifies required properties are present.

  • Checks for duplicate Item IDs.

  • Checks for valid Item ID, Revision, and Item Name length.

  • Checks for invalid characters in the Item ID.

  • Validates the naming rule for Item ID and Revision.

  • Verifies number of working revisions.

  • Verifies permissions are correct to perform the requested action.

Note:

To select all rows in the table for verification, use the Select All command. To clear your selected items, use the Clear All command.

In the Concept Assistant Publish

The Run Teamcenter Validations command assists with identifying any issues with documents in Concept Assistant. Issues are noted when any of these conditions occur:

  • Copy permission is denied.

  • Revise permission is denied.

  • Maximum number of revisions is exceeded.

  • Newer version exists in Teamcenter.

  • Write permission is denied.

  • File is currently checked out by another user.
