Using Download Options - Overrides

When you access the Open File dialog box to open Teamcenter-managed documents, or the Download dialog box in Cache Assistant, you are presented with the opportunity to override specific settings for populating your cache. The contents of your cache is very important when you work with collaborative partners who rely on receiving packages of your data generated by the Cache Assistant→Create Package command.

The override options you can choose from are dependent on the value of the Teamcenter preference SEEC_Property_Synchronize, the document type you select to open, and whether drawings are opened inactive. For example, selecting an assembly document presents a different set of options than when you select a part document.

Note:

  • The override options impact the amount of data downloaded to your cache which can impact the time it takes to open the document.

  • Download Options - Overrides apply only to the document type that is selected. For example, Synchronize assembly structures has no affect when you download a Draft document.

  • The override options are not retained with the Save As Default command.

  • Override options are not available in Structure Editor or Add to Teamcenter-Interactive.

Assembly

Synchronize document properties

When the Teamcenter preference SEEC_Property_Synchronize value is set to false, you can choose to override the setting and synchronize the properties for all linked documents by selecting the check box.

Download linked documents

Linked documents created by Insert Assembly Copy, Insert Part Copy and assembly constructions are downloaded to cache. OLE and BOM links are included as part of regular expansion. Inter-part copies are not considered. Assembly options for simplifying assemblies and hiding all levels can impact BOM expansion.

Download linked drawings

Performs a Where Used for the list of documents included in the assembly and finds its related drawings. This operation is not recursive.

Download linked documents

Downloads to the cache part copies, family of parts, and other similarly linked documents when Download Linked Drawings is selected. OLE links are part of the default implementation.

Download Simulation results file (*.ssd)

Downloads to the cache the Solid Edge Simulation analysis file (.ssd) along with the part, sheet metal or assembly document.

Synchronize assembly structures

Performs BOM synchronization when you download the files to cache. This option is only available in the Download dialog box in Cache Assistant.

Draft

Synchronize document properties

When the Teamcenter preference SEEC_Property_Synchronize value is set to false, you can choose to override the setting and synchronize the properties for all linked documents by selecting the check box.

Download linked documents

Downloads to the cache part copies, family of parts, and other similarly linked documents when Download Linked Drawings is selected. OLE links are part of the default implementation.

Part and Sheet Metal

Synchronize document properties

When the Teamcenter preference SEEC_Property_Synchronize value is set to false, you can choose to override the setting and synchronize the properties for all linked documents by selecting the check box.

Download linked drawings

Performs a Where Used of the list of documents included in the assembly and finds its related drawings. This operation is not recursive.

Download linked documents

Downloads to the cache part copies, family of parts, and other similarly linked documents when Download Linked Drawings is selected. OLE links are part of the default implementation.

Download Simulation results file (*.ssd)

Downloads to the cache the Solid Edge Simulation analysis file (.ssd) along with the part, sheet metal or assembly document.
