Format Columns dialog box (SEEC)

Column display and order

Lists the column names that display in many dialog boxes within Solid Edge Embedded Client. Cleared check boxes adjacent to the column name indicate the column is hidden from display. Hidden columns can be restored by selecting the check box. The absence of a check box beside the column name indicates the column is required and its display cannot be hidden.

Up

Moves the selected column header up in the order it displays in dialog boxes.

Down

Moves the selected column header down in the order it displays in dialog boxes.

Width

Specifies the width of the column either by number of characters or by best fit.

Alignment

Specifies the alignment of the header within the column. The header can be left-justified, centered, or right-justified.
