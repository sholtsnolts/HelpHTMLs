Print to File dialog box

Saves a document to a printer-ready file instead of routing it directly to a printer or plotter. This allows you to print from a computer that does not have the application installed, or to print to a printer other than the one you currently have installed.

Save In

Select a folder location where the printer-ready file is to be generated.

File Name

Type the name for the printer-ready file to be generated. This can be a different name than the source document name.

Save As Type

Specifies the file name extension for the printer-ready document to be generated.
