Upgrade your options.xml file

When you upgrade between major versions of Solid Edge, you need to run SEAdmin.exe to also upgrade your Options.xml file.

  1. Install the new version of Solid Edge on one client machine.

  2. Install SEAdmin, open your existing Options.xml file, and then save it.

    Note:

    The Options.xml file can contain option settings for multiple Solid Edge releases so you don't have to create a new options.xml file for each new release.

  3. After updating and saving the Options.xml file (even if you don't make changes), upgrade your other clients to the new version of Solid Edge.

    You can specify the location of your Options.xml file during setup or manually update your Options.xml file location after installation.
