Finding file duplicates

One of the challenges of working in an unmanaged environment is avoiding file duplicates. File duplicates can share the same file name, document number and revision, or geometric versions with other files.

The Find Duplicates functionality allows you to find and replace duplicate Solid Edge files within a Solid Edge vault. Note, however, that you cannot replace families of parts, divide parts, multibodies, where-used documents, or documents that have a change action status.

For each duplicate file, the system provides a ranking to indicate how likely it is that a given file is a duplicate.

To help you better understand the results, the Find Duplicates dialog box displays the following visual cues:    

  • Bold text indicates matching properties.

  • A yellow icon to the left of a file name indicates that there is an issue with the file.

  • A green icon indicates that the file does not have any duplicates.

Note:

For information about defining a vault, see Define a Solid Edge vault.
