Select in Solid Edge command

Highlights in Solid Edge, the object that is selected in Active Workspace.

Note:

This command is not available when the option to automatically select the object in Solid Edge is enabled in the Hosted Active Workspace cross highlighting section of the Assembly tab, Manage page (Solid Edge Options) dialog box.
