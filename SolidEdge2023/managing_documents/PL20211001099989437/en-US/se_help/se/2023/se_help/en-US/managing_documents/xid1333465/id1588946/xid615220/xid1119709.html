User Session Information

User session information is readily available in the Structure Editor startup page after you log on to Teamcenter.

The session information available to you indicates:

(user (userid) - group/role - [site] [ current project] [location code] [change notice]) .

Note:

You can customize the information displayed using the Teamcenter preference appbar_title_contents. For additional information regarding Teamcenter preferences, see the Teamcenter Preferences section of the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.

User Session information is displayed as a hyperlink to the User Settings dialog box where you can change your Session and Person information.
