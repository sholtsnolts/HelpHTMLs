Extract Translated Files dialog box (SEEC)

Determines the translated files extracted from Teamcenter to an unmanaged location.

Expand Selection

Determines files extracted in addition to the components of the selected file.

Include components from selected assemblies

Specifies that files that are part of the BOM are extracted.

Include components linked with part copy

Includes files linked by the Part Copy command.

Filter by associated Solid Edge file type

Filters the type of files that you want to have extracted. Buttons for each Solid Edge file type (Assembly, Part, Sheet Metal, Draft, Weldment) can be selected in combination, or individually to select a single file type.

Filter by relation, reference, file type

Reduces the selection set by refining the number of files based on the settings:

File type filter

Specifies the file type for filtering, for example STEP documents (*.step; *.stp).

Relation filter

Specifies the relation type for filtering, for example IMAN_manifestation.

Reference filter

Specifies the reference type for filtering, for example, Image.

Export Location

Defines the unmanaged folder location for the extracted files. Extracting the translated files does not create a package or zip of the files. It only places the files in the location you specify.

Extract

Gathers the translated files specified and places them in an unmanaged folder.
