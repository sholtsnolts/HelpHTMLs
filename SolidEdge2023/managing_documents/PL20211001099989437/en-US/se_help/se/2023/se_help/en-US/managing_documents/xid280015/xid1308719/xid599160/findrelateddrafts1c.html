Find All Related Drafts command

The Find All Related Drafts command performs a Where Used search to discover the drafts that are related to the documents chosen as input into Teamcenter. The draft documents are inserted into the list of unmanaged documents as a child of each object that it references. The parent object is expanded to one level so you can see the added draft documents.

By default, Find All Related Drafts searches for draft documents in the same folders as the 3D documents. You can specify additional folders on the Session Settings dialog box.

The Find All Related Drafts command is available on the toolbar at the top of the application window.
