Open drawings

You can quickly find and open drawings using the Open Drawing command .

To use the Open Drawing command, you must be a Solid Edge data management user and have vault locations defined for your Solid Edge files. For more information, see Getting started with Solid Edge data management.

  1. Do one of the following:

    • To find drawings for the open document, go to step 2.

    • To find drawings for a specific part in an assembly, select that part in PathFinder, and then go to step 2.

  2. On the Data Management tab, click Open Drawing .

    Solid Edge searches the vault and does one of the following:

    • If no drawings are available, it informs you that it did not find any drawings.

    • If one drawing is available, it opens the drawing.

    • If more than one drawing is available, it displays a list of drawings from which you can choose the ones you want to view.

  3. If Solid Edge found more than one drawing, select the drawing or drawings of interest and then click OK.
