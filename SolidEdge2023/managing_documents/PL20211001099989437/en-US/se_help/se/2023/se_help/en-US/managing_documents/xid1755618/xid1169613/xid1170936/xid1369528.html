Assign document controller privileges

To protect your data, you can use the document controller option to give specific users additional privileges. Document controllers can set the status to Obsolete on released revisions, as well as rename, save as, and move standard parts in Design Manager.

Assign one or more users as document controllers

  1. Choose File tab→Settings→Options→Manage.

  2. In the Document Controller (Windows user account) box, enter your user name or the user names of interest, separated by semicolons (;).

    Tip:

    To verify your user name, right-click the Windows status bar and select Start Task Manager. Click the Users tab and note the user name.

  3. Click OK.

Assign one or more users as document controllers and prevent changes to the document controller option

  1. In Windows Explorer, copy SEAdmin.exe (Solid Edge Administrator) from the \Solid Edge\SptTools\SEAdmin folder on the Solid Edge DVD to the \Program Files\Siemens\Solid Edge 2023\Program folder.

    Note:

    For more information about Solid Edge Administrator, see Managing user privileges and document locations.

  2. In the \Program Files\Siemens\Solid Edge 2023\Program folder, double-click the SEAdmin.exe file to start Solid Edge Administrator.

  3. Scroll down to the Document Controller Users option.

  4. Enter the user names of the users of interest, separated by semicolons (;).

  5. Make sure that Allow Override is set to No.
