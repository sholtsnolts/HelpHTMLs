Exclude Draft option

Excludes Solid Edge Draft documents when you create a new Item from an existing Teamcenter-managed document using the Save As, Save As - New Item, and Save Copy As commands. Select the Exclude Draft option and then the command so that the newly created Item contains only the Solid Edge 3D content, and no draft documents are carried forward to the newly created document.

Note:

The No Draft option does not apply to Revise.
