Preview dialog box

On

When set, specifies that a graphical preview of the document is displayed.

Parts

Sets the preview options for parts.

Bitmap

Specifies that the preview image is a static bitmap.

Interactive

Specifies that the preview image is an interactive image, which you can zoom, rotate, and pan. Pause the cursor over the image to display a tooltip that explains which keys you can press to manipulate the image.

Assemblies

Sets the preview options for assemblies.

Bitmap

Specifies that the preview image is a static bitmap.

Interactive

Specifies that the preview image is an interactive image, which you can zoom, rotate, and pan. Pause the cursor over the image to display a tooltip that explains which keys you can press to manipulate the image.

Note:

With large assemblies, setting this option can affect performance.
