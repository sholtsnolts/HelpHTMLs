Determine your Teamcenter connection type

  1. Start the Define Teamcenter Environments application by choosing Start→Programs→Siemens Solid Edge 2023→SEEC→ Define Teamcenter Environments.

  2. From the Current Teamcenter environment selection list, click the URL of the environment for which you want to determine the connection type.

  3. In the Define Teamcenter Environments dialog box, notice the state of the 2-tier check box.

    If a check mark appears in the 2-tier check box, the connection type is 2-tier.

    If no check mark appears in the 2-tier check box, a URL string displays and indicates the connection type is 4-tier.

Note:

In the event a single server has both a 2-tier and 4-tier connection, run the diagnostics scan using each setting.
