User assistance

Structure Editor user assistance makes command information available as you perform tasks. You can access command, conceptual, reference, and instructional information any time during a session.

User interface help features

  • The Structure Editor startup screen provides access to Help, and basic operations, such as logging in and opening existing documents.

  • ToolTips help you find command names. When you point the cursor at the command button on the toolbar, a label displays the command name and details about the command.

Help

A structured table of contents and full-text search capabilities provide easy access to Help topics. Press F1 any time you need Help. When a command is active, the Help topic for that command appears. If no command is active, then the table of contents for the Help topics appears.

Tip:

You can also access the table of contents by clicking Structure Editor Help on the Help menu.

When you use the search capabilities:

  • Searches are not case-sensitive.

  • Punctuation marks such as the period, colon, semicolon, comma, and hyphen are ignored during a search.

  • You can group elements of your search using double quotes or parentheses to set apart each element. You cannot search for quotation marks.

  • You can use wild card expressions to search for words or phrases. Wild card expressions allow you to search for one or more characters using a question mark or asterisk. For example, the search string "dimension*” would display topics that contain the term "dimension," "dimensional," and so on.

Learning tools

  • The description on the startup screen provides you with a brief explanation of the product.

  • An activity, Use the Structure Editor, gives you the opportunity to practice using the tool.

  • A overview video is included in Structure Editor overview.

  • You can use the About Structure Editor command on the Help menu to see the software version and copyright information.
