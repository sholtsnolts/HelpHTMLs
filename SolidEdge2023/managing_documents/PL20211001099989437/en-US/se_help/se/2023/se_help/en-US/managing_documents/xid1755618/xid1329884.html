Solid Edge data management

If you are not licensed to use Solid Edge with the product data management (PDM) client, Teamcenter, Solid Edge provides data management functionality that is integrated into Solid Edge. If you are not licensed for Teamcenter and you also choose not to use the Solid Edge data management functionality, you are working with completely unmanaged data.

We refer to users who choose to use Solid Edge data management as Solid Edge data management users.

For information about setting up Solid Edge data management, see Getting started with Solid Edge data management.

In addition to Solid Edge data management, you can use Design Manager to manage document links. Launched as a separate application from the Data Management tab, Design Manager provides additional functionality, such as Rename, Search For Broken Links, Redefine Links, and Search for Duplicate Document Names. For more information, see Welcome to Design Manager.
