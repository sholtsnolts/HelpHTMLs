Select documents for a workflow process

  1. With a document open in Solid Edge, specify documents for the workflow process by selecting one of the following:

    • The File menu→Manage→New Workflow Process.

      The direct document is added as a target for the workflow.

    • The New Workflow Process from the Teamcenter tab.

      The active document is added as a target for the workflow.

    • A single occurrence or multiple occurrences from PathFinder in the assembly environment, and then right-click to select the shortcut command, New Workflow Process.

    Tip:

    Any document that is in the active session that has been saved to Teamcenter is a candidate for a workflow process.

  2. Define the Process Name for the workflow.

    This is a required field. By default, the process name is the item revision of the first document in the target list.

  3. (Optional) You can specify a Description for the workflow process.

  4. Choose one of the following:

    • All - To view all process templates.

    • Assigned - To view only the templates assigned to you.

  5. From the Process Template list, select the workflow process you want to use.

    Tip:

    You can click Refresh to retrieve a current list of process templates from Teamcenter.

  6. Click OK.
