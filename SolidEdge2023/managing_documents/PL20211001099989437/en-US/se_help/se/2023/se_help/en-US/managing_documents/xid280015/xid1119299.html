User Session Information

In a Teamcenter-managed environment, your user session information is readily available:

  • In the Solid Edge start page after logging in to Teamcenter.

  • When a Solid Edge document is open, select the View tab→Show group→Panes command, and then choose User Session Info.

    Note:

    To learn how to use tool panes, see the help topic Manipulate multiple- and single-page windows.

Solid Edge start page

The session information available to you from the start page indicates:

(user (userid) - group/role - [site] [ current project] [location code] [change notice]) .

You can customize the information displayed using the Teamcenter preference appbar_title_contents.

User session information is displayed as a link to the User Settings dialog box where you can change your Session and Person information.

Solid Edge document

The session information available to you when a document is open in Solid Edge, is displayed in the User Session Info docking pane:

  • Teamcenter information - the URI and name of the database where you are connected.

  • Session information - time of logon, person, users id, group, role, volume, local volume, location code, and project ID.

  • Connection information - Teamcenter Version, Site ID, Locale, Encoding and Syslog data.

You can use Switch View to change the information from displaying in a docking pane to displaying in a horizontal bar at the top of the Solid Edge workspace.

The information displayed can vary depending on how the appbar_title_contents Teamcenter preference is defined.

For additional information regarding Teamcenter preferences, see the Teamcenter Preferences section of the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.
