Replace dialog box

Finds and replaces existing text. You can replace all occurrences at once or confirm the ones you want to replace by viewing each occurrence individually.

Find What

Specifies the text for the search.

Replace With

Specifies the replacement text.

Find Next

Locates the next occurrence of the text.

Replace

Replaces the selected text.

Replace All

Replaces all occurrences of the text.

Match Whole Word Only

Searches for instances of text where only the entire word matches the text specified.

Match Case

Searches for case-sensitive occurrences of a word or phrase. This limits the search so it only locates text that matches the upper- and lowercase characters you enter.
