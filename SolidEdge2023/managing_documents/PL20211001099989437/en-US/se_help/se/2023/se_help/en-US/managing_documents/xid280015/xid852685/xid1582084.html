Hide CPD option

Bypasses the display of common property dialog boxes when you upload or check in existing Teamcenter-managed documents. All documents being saved (direct and indirect) must have been previously saved to Teamcenter. If any new documents are discovered, the New Document dialog box is displayed.

The Hide CPD option (Teamcenter ribbon→Manage group) and the option Do not show common property dialog (Solid Edge Options→Manage page→General tab) are synchronized.

Note:

The Limited Save command changes the behavior of Save when you work in an assembly. Only files that have been directly opened or edited-in-place are saved when the assembly is saved.
