Download dialog box

Look In

Displays the drive and folder in which you want to search for the document you want to download. The system stores the last location you browsed in the Look In box.

Go to the Last Folder Visited

Returns to the last folder that you visited.

Up One Folder

Moves the Look In folder up one level.

View Type

Controls the display method for the listed documents.

Large Icons

Displays large icons for the documents.

Small Icons

Displays small icons for the documents.

List

Lists the names of the documents in columns.

Details

Displays a detailed view of the folder contents. The columns displayed include Name, Size, and Modified.

Add

Adds the selected documents you want to download to the Documents to Download box.

Remove

Removes the selected documents from the Documents to Download box.

Remove All

Removes all folders and documents from the Documents to Download box.

Documents to Download

Lists the folders and documents you want to download from the library.

Check Out Selected Documents

Checks out the documents you have selected for download, which gives you write access to the documents.
