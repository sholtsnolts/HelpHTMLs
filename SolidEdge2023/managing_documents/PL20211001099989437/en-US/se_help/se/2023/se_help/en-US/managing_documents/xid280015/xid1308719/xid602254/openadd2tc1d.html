Open dialog box

Displays documents that can be selected for adding to Teamcenter.

Navigation pane

Lists the folders. On first run, displays My Documents.

Library pane

Lists the contents of the location selected in the navigation pane.

Views

Controls the display method for the listed documents.

Extra Large Icons

Displays very large icons for the documents.

Large Icons

Displays large icons for the documents.

Medium Icons

Displays medium icons for the documents.

Small Icons

Displays representative icons for the documents.

List

Lists the names of the documents in columns.

Details

Displays a detailed view of the directory contents. The columns include Name, Date Modified, Type, and Size.

Tiles

Displays thumbnail sized icons for the documents.

Content

Displays each document on a single line along with the date modified.

Show Preview Pane

Toggles the display of a document preview pane on or off.

File name

Lists documents to open. You can select a single document or multiple documents.

Document Filter

Specifies the type of Solid Edge documents that you want to see in the library pane. You can select from All Solid Edge documents, Assembly, Part, Draft, Sheet Metal, or Weldment documents.
