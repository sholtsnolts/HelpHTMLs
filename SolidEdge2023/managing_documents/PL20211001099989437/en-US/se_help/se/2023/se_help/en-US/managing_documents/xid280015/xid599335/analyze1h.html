Perform a Solid Edge file analysis

Use this procedure to analyze your files and prepare the primary list of files for import into the managed environment.

Note:

If your Solid Edge files contain custom properties, define those properties before running the Data Preparation Utility. For details, see Define custom properties prior to analysis.

  1. Choose Start→Siemens Solid Edge 2023→Choose PDM Integration.

  2. On the ribbon, click Utility.

  3. Select the document management application for which you are preparing your data:

    • Teamcenter

    • Unmanaged

  4. (Optional) To change the default location for the log files generated as output of the analysis, click Set Working Folder.

    Note:

    By default, the utility saves log files in \Users\<username>\AppData\Roaming\Siemens\Solid Edge\Version 223\Log Files\Data Preparation.

  5. (Optional) To view the contents of the working folder click, Open Working Folder.

  6. In the Generate Import List tab, click Select Files/Folders.

  7. Select whether the files to be analyzed are located in a file or in a system folder.

    Use this option

    To do this

    Add Files

    Identify individual files for analysis or define the location of either the ordered or unordered CSV file containing the list of files for analysis.

    Add Folders

    Identify the location of the system folders containing the files for analysis.

  8. Browse for the files or folders specified in the previous step.

    Tip:

    • When you choose to Add Folders, you can select the Include Subfolders check box so that any subfolders are included for you.

    • You can click Types to filter your selection by file types (.dft, .asm, .par, .psm, .pwd or All documents including non-Solid Edge documents).

    • You can click Remove or Remove All to modify the list of files or folders.

  9. To start the analysis, click Generate Import List.

    The length of time required for processing depends on the number of files analyzed. When processing is finished, several files are generated for you:

    • AnalysisOfSolid Edgefiles_<timestamp>.txt

      Contains a summary of the file analysis.

    • DataPrep_Links_<timestamp>.xlsx

      Lists relation types if the relation type is a reverse link.

    • OrderedList_<timestamp>.CSV

      Contains the primary list of files to be imported into the managed environment in the order in which they are to be imported.

    • PropertyList_<timestamp>.xlsm

      Displays the results of the data analysis in a separate window. For more information, see Analysis report.

    • Unordered_<timestamp>.CSV

      Contains a list of files to be imported into the managed environment.

  10. To add property data or correct problems identified during the analysis, use the tools in the analysis report to apply corrections to the files listed.

    Examples of the type of corrections you might need to apply include changing the Project ID, changing revisions, or renaming existing unmanaged files.

    Tip:

    • If broken links are identified in the file analysis, use the options in the Fix Broken Links tab to repair them. To learn how to do this, see Fix Links in Solid Edge Files.

    • To modify the properties of your unmanaged Solid Edge files with the changes you specify in the analysis report, click the Modify Properties tab.

  11. Exit the analysis report, PropertyList_<timestamp>.xlsm.

    Note:

    The commands for making changes to your Solid Edge files rely on macros. If you close and reopen the spreadsheet, macros may be disabled due to your Microsoft Office security settings. Discuss your security policy with your system administrator before making changes to your security settings.
