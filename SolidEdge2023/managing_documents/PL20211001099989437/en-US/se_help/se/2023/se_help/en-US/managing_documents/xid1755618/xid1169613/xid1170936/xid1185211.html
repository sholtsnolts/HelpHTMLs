Manage document properties

Design Manager provides several document properties, but you can also create custom properties, as needed. You can also choose to display or hide any of the properties.

  1. In the Design Manager document list, right-click the Current Filename cell for the document of interest, and then select Show Properties.

  2. To display properties in the Available properties list, click the arrow under Select available properties from, and select the group of properties you want to display.

    You choose properties from the list of available properties to display in the Design Manager document list.

  3. Do any of the following:

    • To display a property in the Design Manager document list, in the Available properties list, select a property, and then click Add.

    • To remove a property from the Design Manager document list, from the Show these properties in this order list, select a property, and then click Remove.

    • To change the property display order, click the property you want to move and use the Move up and Move down buttons.

    • To create a custom property, click New.

      1. In the Name box, enter a name for your property.

      2. From the Type menu, select the type of property.

      3. Click OK.

  4. Click OK.
