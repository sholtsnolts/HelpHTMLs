Reorient command bar

Printer Settings

Displays the printer settings dialog box.

Settle

Settles the model on the printer bed using the closest planar face.

Auto Orient

Orients the model to the next best fit.

Scale to Fit

Scales the model to fit the printer bed using the current orientation.

Cancel

Cancels the Reorient command.

Reset Orientation

Resets the model to its original orientation.

Undo

Undoes the last action.

Redo

Undoes the previously undone action.

Move list

Select the action: Move or Rotate

Move

Enter the move distance in the X, Y, or Z fields. Press Enter to move. The move uses the X, Y, or Z field that has focus. Pressing Enter again performs another move of the same distance. Selecting the Move action list resets the field values.

Rotate

Enter the rotation angle in the X, Y, or Z fields. Press Enter to rotate. The rotate uses the X, Y, or Z field that has focus. Pressing Enter again performs another rotate of the same angle. Selecting the Move action list resets the field values.
