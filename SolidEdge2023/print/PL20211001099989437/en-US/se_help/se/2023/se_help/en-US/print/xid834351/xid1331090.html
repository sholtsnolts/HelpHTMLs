Open a Solid Edge file in 3D Builder

You can use the Microsoft 3D Builder application to view and edit your STL or 3MF file before you send it to a 3D printer. For example, you can emboss text, split the model, smooth a 3D object, merge two 3D objects, and subtract intersecting geometry. These capabilities are not available in Solid Edge for STL and 3MF files.

To begin using the 3D Builder application:

  1. Open a model document in Solid Edge.

  2. Select the Application→3D Print page→3D Builder command .

  3. In the Microsoft 3D Printing Export dialog box, choose the file type that you want to export, and then select Save.

    Your .stl or .3mf file is opened in a new 3D Builder application window on top of Solid Edge.

To learn how to use the 3D Builder application, refer to the tutorials and the 3D Builder User's Guide available on the Help menu in the 3D Builder application. You also can find this information at the Microsoft Developer Resources website→Create and print 3D objects page (https://developer.microsoft.com/en-us/windows/hardware/3d-print/3d-builder-resources).

Note:

The 3D Builder application is installed by default with Windows 10.
