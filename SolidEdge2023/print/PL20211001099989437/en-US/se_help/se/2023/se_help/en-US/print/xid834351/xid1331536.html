Scale a model for 3D printing

To scale a model in Solid Edge so that it fits on the platen of a 3D printer, you can use the Scale Body command or the Part Copy command, as explained below.

Scale a model using the Part Copy command

  1. Select the File menu→New command to create a new part file (.par) file.

  2. Select the Home tab→Clipboard group→Part Copy command .

  3. In the Select Part Copy dialog box, locate and open the model to scale.

  4. In the Part Copy Parameters dialog box, use the X, Y, and Z boxes to specify a Scale factor.

    For example, to scale the model uniformly by 50 percent, enter .5 in the X box and select the Uniform scale check box.

Tip:

You also can scale a model in the Microsoft Print Manager dialog box, when you Print directly to a 3D printer using the 3D Print command.
