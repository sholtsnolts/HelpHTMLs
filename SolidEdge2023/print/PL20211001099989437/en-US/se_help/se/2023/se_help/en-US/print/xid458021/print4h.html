Print at a 1:1 scale


  1. Choose File menu→Paper Print.

  2. From the Printer list, select the name of the printer to print the document.

  3. Select what you want to print:

    • To print multiple working sheets, select the All sheets option.

    • To print only the 2D Model sheet at a 1:1 scale, select the Active sheet option.

    • To print multiple sheets, select the Sheets option and enter the sheet numbers in the Sheets box.

    Note:

    To print at a 1:1 scale, the Current display option must be cleared. Otherwise, the print area is set to the window size and not the actual sheet size.

  4. Click the Settings button, and then under the Scale options, select the Manual scale option and enter 1 : 1.

  5. Click Print.
