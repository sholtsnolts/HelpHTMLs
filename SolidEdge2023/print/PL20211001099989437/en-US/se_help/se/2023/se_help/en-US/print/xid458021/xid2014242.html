Print to Adobe PDF

To print a Solid Edge model or draft document as a PDF file, you can:

  1. Use the Save As command to do either of the following:

    • Save a document to Adobe PDF.

    • Save a document as 3D Adobe PDF

  2. For a draft document, use the PDF Export Options dialog box to select options for saving the PDF file.

  3. Open the PDF document in Adobe Reader, and then use the Adobe File→Print command to print it.
