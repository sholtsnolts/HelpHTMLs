Print Drawings - Select Documents dialog box

The Print Drawings - Select Documents dialog box is displayed when you select the Print Drawings command and you do not have any draft documents open. Use this dialog box to choose the managed and unmanaged files that contain drawings you want to print.

You also can reopen this dialog box using the Browse button on the Print Drawings - Select Sheets dialog box.

Look In

Displays the drive and folder in which you want to search for the document or image you want to open.

The system stores the last location you browsed in the Look In box.

If you are opening a document that is managed by Teamcenter, this option displays the Teamcenter folder structure and does not show the cache path.

Go to Last Folder Visited

Returns to the last folder that you visited. This option is not available when working with Teamcenter.

Up One Level

Accesses the parent folder.

Create New Folder

Displays the Create New Folder dialog box. This option is not available with Teamcenter.

View Menu

Controls the display method for the listed documents. This option is not available when working with Teamcenter.

  • Thumbnails—Displays a thumbnail-sized image of the contents of each document or folder. If the folder contains multiple Solid Edge documents with graphical contents, then up to four graphical images are displayed on the folder thumbnail. Click the folder and then click the Open button to see additional thumbnails of individual documents in the folder.

  • Large Icons—Displays large icons to represent each document type.

  • Small Icons—Displays small icons to represent each document type.

  • List—Lists the names of the documents in columns.

  • Details—Displays a detailed view of the folder contents. The columns include Name, Size, Type, and Modified.

Preview

Displays a bitmap image of the selected document if a preview image was saved for the file.

Properties

Displays the properties of the selected document in the right pane.

File

Specifies the name of the file you want to open.

Note:

In the message field at the bottom of the dialog box, a message displays the availability of the currently selected file. The availability may be: Read Write, Read Only, or Unknown. The message changes depending on the current status of the file.

Files of type

The documents you can choose from are restricted to Solid Edge Draft files (*.dft).

Open as read-only

Opens a document with read-only privileges. For Teamcenter-managed documents, use the Open as Read-Only options under Check-Out Information.

First level read-only

Sets the direct document with read-only privileges. All indirect documents are opened with write privileges.

All levels read-only

Sets all levels of the document with read-only privileges.

Revision rule

Specifies a revision rule for updating links when you open a managed document.

  • As Saved - Opens the document and updates the links as it was last saved.

  • Latest - Opens the document and updates the links to the latest revision in the system.

  • Latest Released - Opens the document and updates the links with the latest released revision.

  • External System - Opens the document and updates links with revisions specified by an external system.

  • Version from Cache - Opens the document from your cache. No document transfers take place between the document library and your local cache resulting in optimized performance. When Version from Cache is chosen in conjunction with the Solid Edge Option, Always Synchronize to Get Latest Version, the latest version of the direct document is copied from the server to the cache and indirect documents are opened directly from the cache.

The default value is As Saved. This control is not available if you select an unmanaged document, a document that is checked out, or if you are working offline.

Note:

If you are opening a document that is managed by Teamcenter, you will be able to select from all of the revision rules available in Teamcenter in addition to those listed here.

Stop open if rule not matched

Specifies that you want to stop the open process if the system does not find a revision that matches the specified rule. This option is not available with Teamcenter.

Options

Displays the translation options. This is not available when working with Teamcenter.

Search

Displays the Search dialog box so you can define advanced search criteria for the document you want to open.

If you are working with Teamcenter, the Search dialog box for Solid Edge Embedded Client is displayed.

Variant rule

Specifies the variant rule used when opening an assembly from Teamcenter. If no variant was saved with the assembly, the option is grayed out on the Open dialog box. The variant rule is assembly-specific and is cleared if you select a different assembly.

Check-out information

This portion of the dialog box is only available when working with Teamcenter.

Open as read-only

Specifies that you want to open a read-only copy of the selected document.

Reason for check out

Allows you to type a reason you are checking out the document. The option is only available when working with Teamcenter.

Change ID

Displays the Change ID for the selected document. The option is only available when working with Teamcenter.
