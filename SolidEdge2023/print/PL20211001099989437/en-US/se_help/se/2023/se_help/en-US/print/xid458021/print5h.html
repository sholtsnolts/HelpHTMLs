Print a composite drawing

  1. In the active document window, display the draft document that contains the drawing that you want to print.

    Tip:

    You also can print drawings from the File menu without opening any documents.

  2. On the File menu, choose Paper Print→Print Drawings .

    Tip:

    You can choose additional drawings that you want to print using the Browse button in the Print Drawings dialog box.

  3. If the Print Drawings - Select Documents dialog box is displayed, select one or more Solid Edge draft documents (.dft) that you want to print.

    Tip:

    To select multiple documents, press the Ctrl key while you select.

  4. In the Print Drawings - Select Sheets dialog box, do all of the following:

    1. Choose what you want to print by clearing or setting the check marks in the Print column. You can select the entire drawing or individual sheets.

      Tip:

      To see all of the sheets in multiple drawings at once, press the Ctrl key while you click the document names in the first column, and then right-click and choose Expand All.

    2. From the Printer list, choose the printer you want to use.

      Tip:

      Largest sheet size shows the paper size needed to print the largest sheet in your print list. Choose a printer that uses that paper size.

    3. Under Layout, choose the Multiple sheets per page option.

    4. (Specify sheet options) Click the Options button, and then specify scale, margins, and whether to print cut lines in the Options-Multiple Sheets per Page dialog box.

  5. Click Preview.

  6. If a dialog box appears indicating that <filename.dft> contains one or more sheets that are too large for the current print size, select one of the following options to specify how to handle the sheet:

    • Skip the document—If you choose this option, none of the sheets in the document will be printed. The drawing will remain checked in the Select Sheets dialog box, as a reminder that it was not printed. You can submit the document in a separate print job using a different paper size.

    • Modify the Print Drawings options—If you choose this option, the Options dialog box is displayed for you to make changes to sheet scale or margin settings.

    • Scale to fit—Automatically scales the sheet so it fits on the page.

    • Apply to all documents—For the current print job, specifies that this is how to handle any other oversized sheets that do not fit on the page.

  7. In the Preview dialog box, use the Next Page and Previous Page buttons to display and review each page of the drawing.

    The sheet and page order shown is the order that the drawing pages will be printed.

  8. If you are satisfied with the layout of the composite drawing, click Print.

Tip:

  • In the Print Drawings - Select Sheets dialog box, you can:

    • Choose additional documents to include in the composite drawing by selecting the Browse button.

    • Set printer-specific options, such as duplex printing, by selecting the Properties button.

    • Print to a file instead of a printer by selecting the Print to file option.

  • When printing a single sheet per page rather than multiple sheets per page, you can use the Options-Single Sheet per Page dialog box to control the positioning of the sheet.
