Preview dialog box

The Preview dialog box shows the arrangement of multiple drawing sheets on the paper prior to printing it. If there are multiple pages in the drawing, use the Next Page and Previous Page buttons to preview the layout of each page.

You can use the Cancel button to make changes to the print job specifications.

Preview

The preview image shows the layout and relative size of all of the sheets selected to print on the current page of the drawing.

Report
Printer

Shows the currently selected printer.

Number of page(s)

If the composite drawing contains multiple pages, then the total number of pages is shown.

Actual paper size

Shows the selected paper size in inches or millimeters. Paper units are specified on the Options - Multiple Sheets per Page dialog box.

Page Details
Number of sheets

Shows the number of sheets arranged on the current drawing page.

Usage

As a percent, shows how much of the drawing page space is utilized by the sheet arrangement.

Zoom slider

Dynamically zooms in or out of the drawing sheet preview. Sliding the arrow to the right zooms in on the preview and sliding the arrow to the left zooms out on the preview.

Previous page

Shows the print layout of the previous page in the drawing.

Next page

Shows the print layout of the next page in the drawing.

Jump to page

Shows the print layout for the page number you select.

Print

Prints the current drawing.

Cancel

Displays the Print Drawings - Select Sheets dialog box.
