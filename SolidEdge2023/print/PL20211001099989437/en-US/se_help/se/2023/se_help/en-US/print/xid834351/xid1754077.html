Print Material command

Use the 3D Print tab→Prepare group→Print Material command to assign a material to the part to print.

Many different materials can be used for 3D printing, such as ABS plastic, PLA, polyamide (nylon), glass-filled polyamide, stereolithography materials (epoxy resins), silver, titanium, steel, wax, photopolymers, and polycarbonate.

Each material supports a different thickness. Use the 3D Printing material library to assign a material for the print job. The library provides the minimum thickness it can support.

Learn how to Create a 3D Print material.

With the 3D print material defined, you can check your model readiness for 3D printing.

On the Print Material command bar, assign a material (1) or apply a color (2).
