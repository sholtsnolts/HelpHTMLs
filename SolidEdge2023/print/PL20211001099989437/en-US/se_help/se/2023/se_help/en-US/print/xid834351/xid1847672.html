Validate a 3D print model

  1. Open a .par, .asm, or .psm document to validate.

  2. Choose the 3D Print tab→Validate group→Print Validations command .

    The Print Validations dialog box displays.

  3. If a 3D print material is already applied to the part, the same material is contained in the Material list. If not, apply any 3D Print Material (optional).

    Note:

    Do not expect to see all materials from the Material Table. The materials which are marked as “3D Print”=Yes are shown in the list.

  4. Apply a display resolution. This sets the tolerance level when converting the model into an STL format.

  5. Select one or more validation checks.

  6. Click Compute.

    Observe the Results text box displays results for each selected check.

  7. In the Results box, click to expand the regions. Select a region and then click Fit to zoom in to the area selected.
