Print Validations dialog box

Model Options
Materials

Materials marked as "3D Print"=Yes in the Material Library are shown in the list.

For more information, see Create a 3D Print material..

Display Resolution

Tolerance levels used to convert model into STL format.

Checks
Wall Thickness

The distance between one surface of the model and the opposite sheer surface.

Thickness value

Defines the minimum thickness to validate.

Color list

Defines the color to display on the problem areas identified by the wall thickness check.

Overhang

When printing an object that has slanted surfaces without support material under it can produce bad results. This usually happens when trying to 3D print overhangs exceeding an angle of about 30-40 degrees. The print starts to curl upwards and because the following layers are then also printed upwards, it becomes an exponential problem.

Overhang angle value

Defines the overhang angle to validate.

Color list

Defines the color to display on the problem areas identified by the overhang check.

Degenerate

A degenerate triangle is the one with less area than the defined tolerance. These are typically in areas around sharp edges.

Tolerance value

Defines the tolerance value to validate.

Color list

Defines the color to display the problem areas identified by the degenerate triangle check.

Intersection

Intersecting triangles usually lead to failures during slicing.

Color list

Defines the color to display the problem areas identified by the intersection check.

Non Manifold

Checks if the object is non-maniflod.

Color list

Defines the color to display on non-manifold problem areas.

Compute

Invokes the validation process.

Reset

Clears all checks and returns the dialog box to the default settings.

Results

Displays the results for each selected check, and shows the corresponding colors on the model.

Fit

Fits the display to a selected region.
