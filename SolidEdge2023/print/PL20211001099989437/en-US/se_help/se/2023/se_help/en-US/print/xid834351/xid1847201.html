Create a 3D Print material

Materials designated as 3D print materials are suitable for additive manufacturing. They are applied at the minimum thickness that your design can support, reducing the cost of 3D printing.

  1. Open the Material Table.

  2. Create a New Group or select an existing group.

    Click Plastics for this example.

  3. Right-click on the Plastics group and then choose New Material.

  4. Name the new material PLA (or any name of your choice).

  5. Right-click on the material entry PLA and then choose Add Custom Property.

  6. In the Add/Edit Custom Property dialog, add the following:

    Name: 3D Print

    Type: Select "Text" from the list.

    Value: Yes

    Note:

    Materials which are marked as "3D Print"=Yes will be shown in the Print Material command bar Material list and in the Print Validations dialog box Material list.
