Print Area dialog box

Specifies the area that you want to print.

Print Range

Specifies which part of the document you want to print or plot.

X

Displays a value for the x coordinate for the area you want to print.

Y

Displays a value for the y coordinate for the area you want to print.

Origin

Adjusts the location of the print area on the printer paper.

X

Displays a value for the x coordinate for the area you want to print.

Y

Displays a value for the y coordinate for the area you want to print.

Center

Centers the print area on the printer paper.

Scale

Controls the scale for the print area in a document.

Fit To Printer Margins

Scales the selected drawing sheets or print area to fit the configured device printer paper.

Manual Scale

Specifies the scale value you want to use during printing. The value you type is the ratio of the paper to the design.

Paper Length

Specifies the paper length for the document you want to print.

Design Length

Specifies a design length.

Line Width Scale Factor

Applies a scale factor to the screen resolution line weight increasing the number of pixels representing lines sent to the printer. This option is not available in the Draft environment.

Preview

Shows the specified plotting area in relation to the effective plotting area. By default, the specified plotting area is displayed as a blue rectangle.

Effective Plotting Area

Specifies the plotting area for the document.

XY Range

Clears the previous defined print area and prompts you to define a new print area.
