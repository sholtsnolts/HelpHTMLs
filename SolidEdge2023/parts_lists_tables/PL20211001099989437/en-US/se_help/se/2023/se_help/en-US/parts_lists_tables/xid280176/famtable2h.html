Link family of parts variables to a drawing

When you create or edit a family of parts table on a drawing, you can link one or more variables in the table with driven dimensions in a drawing view. This automatically copies the dimension variable name to the drawing. It also ensures that the drawing dimensions are associative to the member data in the family of parts.

Link table variables to drawing dimensions

  1. On the drawing sheet, click a family of parts table to activate the Family Of Parts Table command bar.

  2. On the command bar, click Link Variable .

  3. On the command bar, from the Variables list, select a variable name.

  4. In the drawing view, click the dimension that matches the variable.

  5. Continue selecting variables and then dimensions for each drawing dimension you want to link to a table variable.

  6. On the command bar, click Finish.

Unlink drawing dimensions from table variables

  1. On the drawing sheet, click a family of parts table to activate the Family Of Parts Table command bar.

  2. On the command bar, click Unlink Variable .

  3. In the drawing view, click the dimension that displays a variable name you want to unlink.

    To unlink more than one dimension, press and hold the Ctrl key while you click dimensions.

  4. On the command bar, click Finish.
