Block Table command

The Block Table command creates a block table similar to a parts list from all of the visible blocks on the active sheet, in a selected drawing view, or from manually selected blocks. You can use the Selection Method option on the Block Table command bar to specify which method you want to use.

The Auto-Balloon option on the command bar generates balloons that identify the item number and quantity for each block referenced in the block table.

You can specify the type of block list that you want to produce using the List Options on the Options tab in the Block Table Properties dialog box. For example, you can use the Block Table command to:

  • Generate a block list or block view list from the components on a schematic diagram.

  • Produce a nested parts list from nested blocks that simulates the structure of parts in an assembly.
