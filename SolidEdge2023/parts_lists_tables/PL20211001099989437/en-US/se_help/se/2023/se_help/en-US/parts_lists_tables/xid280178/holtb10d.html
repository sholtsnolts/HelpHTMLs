Callout tab

Use the Callout tab to define the information you want to show in the selected Hole Callout column in the hole table. For more information, see Define a smart hole table callout column.

Hole callout column

Specifies one of four available columns for hole callout data The options are Hole Callout 1, Hole Callout 2, Hole Callout 3, and Hole Callout 4.

Simple

Displays callout data for all simple holes in the hole table.

Counterbore

Displays callout data for all counterbore holes in the hole table.

Countersink

Displays callout data for all countersink holes (chamfered holes) in the hole table.

Counterdrill

Displays callout data for all counterbore holes with angled bottoms. These are defined in the part model using the V bottom angle check box in the Hole Options dialog box.

Simple (Threaded)

Displays callout data for all simple threaded holes in the hole table.

Counterbore (Threaded)

Displays callout data for all counterbore threaded holes in the hole table.

Countersink (Threaded)

Displays callout data for all countersink threaded holes (chamfered holes) in the hole table.

Counterdrill (Threaded)

Displays callout data for all threaded counterbore holes with angled bottoms. These are defined in the part model using the V bottom angle check box in the Hole Options dialog box.

Symbols and Values

Opens the Select Symbols and Values dialog box where you can generate the appropriate symbols and model-derived values without having to type the property text codes yourself. Examples of the types of symbols you can select include ± (plus minus), ° (degree), and ∅ (diameter). Examples of model-derived values include hole feature references, such as chamfer angle.

Format

Modifies the format of the property text string output value at the cursor position in any of the text boxes above.

Displays the Format Values dialog box for you to choose the formatting. You can apply formatting to each valid property text string. The type of formatting you can apply depends on your cursor position when you select the Format button.

For more information, see Format property text values.

Special characters

Specifies mechanical font characters for the text fields. The available special characters are diameter, square, counterbore, countersink, depth, initial length, arc length, plus/minus, degree, between, statistical tolerance, and carriage return.

Hole reference

Specifies hole and thread reference attributes for the text boxes above. The available attributes are Hole Size, Hole Depth, Counterbore Size, Counterbore Depth, Countersink Size, Countersink Angle, Thread Size, and Thread Depth.

Smart depth

The two buttons in this section reference the information on the Smart Depth tab. You can use this relationship to customize the hole table callout. This ensures that you see your custom content in the hole callout, no matter which hole feature was created in the part model.

Smart Depth

Specifies hole depth data from the Smart Depth tab in the Hole Table Properties dialog box.

Smart Thread Depth

Specifies thread depth data from the Smart Depth tab in the Hole Table Properties dialog box.
