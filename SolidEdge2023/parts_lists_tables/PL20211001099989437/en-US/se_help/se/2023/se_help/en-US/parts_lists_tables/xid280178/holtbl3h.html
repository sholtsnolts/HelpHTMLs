Change hole table labels

You can edit the label letter or number used to identify the holes listed in the Hole column. Label changes also appear on the hole callout annotation on the drawing.

Example:

Change the label letter from A→AA.

Change the label numbers from 1.1→5.1.

Change to an alphanumeric label, such as 1.1→1-A.1

Assign different labels to different sets of holes, such as A1 to A20 and B1 to B5. You can do this, for example, when you list holes by size and also list each hole size.

  1. Right-click the hole table you want to edit and choose Properties.

  2. Click the Hole Number tab.

  3. From the Origin/Size box, select the number corresponding to the set of holes you want to relabel. If there is only one origin defined for the drawing view, then select the number 1.

    The hole data for the selected hole set is shown in the grid on the Hole Number tab.

  4. In the Origin/Size box, type the alphanumeric characters you want to assign, and then press Tab or Enter.

  5. To relabel a second set of holes, select the next origin number from the Origin/Size list, for example, 2.

  6. In the Origin/Size box, type the alphanumeric characters you want to assign to the second set of holes, and then press Tab or Enter.

  7. In the Hole Table Properties dialog box, click Apply to update the table and the hole callout annotations.
