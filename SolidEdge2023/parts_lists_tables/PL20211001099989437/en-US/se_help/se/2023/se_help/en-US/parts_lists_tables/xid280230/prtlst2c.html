Convert To Table command

Converts a parts list to a user-defined table format. This retains the table grid lines, column headings, table titles, and other display features of the original parts list. Property text is converted to simple text. You can modify the table format using the Properties command on the shortcut menu.

When you convert a parts list to a table, you can no longer make associative updates to the parts list information.
