Family Of Parts Table command bar

Table Style

Lists and applies the available table styles.

Table Properties

Displays the Family Of Parts Table Properties dialog box. You can make changes to the drawing table format and content using the options on the General, Title, Sorting, Data, and Variables pages on this dialog box.

Link Variable

Creates a link between a variable in the family of parts table and a dimension on the drawing.

Cancel

Cancels the command without placing the table on the drawing.

Finish

Applies all currently selected table properties and applies the variable links to dimensions on the drawing.

Variables

Lists the variables that can be linked to dimensions on the drawing view using the Link Variables button on the command bar.

Note:

You can use the options on the Variables page of the Family Of Parts Table Properties dialog box to add and remove variables from this list.

Unlink Variable

Removes the link between a variable in the family of parts table and a dimension on the drawing.
