Bend Table dialog box

Displays information about the bends in the sheet metal part. You can use the buttons to change the bend sequence, and you can right-click in the table to display a shortcut menu to:

  • Manage bend table column display and display order.

  • Create and manage user-defined columns.

  • Search for information, and copy, cut, and paste it.

  • Change the font.

Columns...

Displays the Format Columns dialog box when you select Columns... from the shortcut menu.

In the Format Columns dialog box, you can use the check boxes to choose the predefined sheet metal bend data columns to show in the table. You can use the Up and Down buttons to change the display order of the columns in the sheet metal bend table. You also can change the column width and alignment.

New Column...

Displays the New Column dialog box when you select New Column... from the shortcut menu.

In the New Column dialog box, you type a title to assign to a new column. You can use this type of column for comments and any other user-defined information you want.

Delete Column

Removes a user-defined column when you right-click the column header.

You cannot remove the shaded columns, which are associative to the model.

Move Up

Moves the selected bend feature up one spot in the bend sequence. When you change the bend sequence, the balloons automatically update to reflect the change.

Move Down

Moves the selected bend feature down one spot in the bend sequence. When you change the bend sequence, the balloons automatically update to reflect the change.
