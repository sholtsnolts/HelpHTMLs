Display assembly occurrences in a drawing view or parts list

If parts are missing in a parts list or a drawing view for an assembly, verify that the missing parts are not turned off in the assembly document Occurrence Properties dialog box.

  1. Open the assembly document.

  2. In PathFinder, right-click the part and choose Occurrence Properties.

  3. In the Occurrence Properties dialog box, use the horizontal slider to scroll the table until you see these two columns: *Assembly Reports and *Drawing Views.

  4. The table cells in these columns must be set to Yes for the part occurrence to display. If a table cell in these columns is set to No, click the cell and change it to Yes.

    Note:

    At the bottom of the Occurrence Properties dialog box, observe the following message: *Selecting 'No' for Drawing Views or Assembly Reports will remove the occurrence from the Draft Parts List.
