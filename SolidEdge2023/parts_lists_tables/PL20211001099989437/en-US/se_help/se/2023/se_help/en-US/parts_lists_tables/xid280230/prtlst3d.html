Parts List Properties dialog box

On this tab

You can do this

General tab

Specify the table style and the maximum table height or number of data rows.

Location tab

Specify the placement location on the sheet, and whether to place it on the current sheet or a new sheet.

See the help topic, Defining table size and location.

Title tab

Specify the text, formatting, and positioning of table titles and subtitles.

See the help topic, Using the Title tab.

Columns tab

Specify the column content and the initial column formatting. You define content by selecting the properties that you want to extract to display in the columns. You can combine multiple properties in each column, and you can add simple text strings to any column.

See the help topic, Using the Columns tab.

Member Selection tab

Includes and excludes FOA members from a family of assemblies parts list.

This tab is available in the Parts List Properties dialog box only when you select the Family of Assemblies Parts List command .

Data tab

Add and remove columns and rows, and where you edit extracted information displayed in individual cells. See the help topic, Using the Data tab.

Sorting tab

Specify that the parts list is sorted based on the document number of the part in the assembly, by component type, material, quantity, and title.

See the help topic, Using the Sorting tab.

Groups tab

For a parts list that spans multiple table sheets, you can use the Groups tab to define table subheadings to categorize table data and keep like items together. See the help topic, Grouping data in tables.

Options tab

Assign the parts list item number format, and specify whether to produce a cut length or total lengths part list for pipe and frames

See the help topic, Using the Options tab.

Item Number tab

After the parts list is created, edit the item numbers displayed in the table and in the balloons.

See the help topic, Item numbers in parts lists.

List Control tab

For each part and subassembly, specify the granularity of the parts list (top level assembly, subassembly, exploded, or atomic).

Balloon tab

Specify the appearance of the balloons and how they are numbered. Also specifies automatic balloon stacks for fastener system components.
