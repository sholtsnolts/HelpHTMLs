Member Selection tab (Family of Assemblies Parts List Properties dialog box)

Selects one or more family of assembly members to include in the family of assemblies parts list.

The Member Selection tab is available in the Parts List Properties dialog box when you select the Family of Assemblies Parts List command . For more information, see Create a family of assemblies parts list.

Select members to include in table
Select all

When checked, selects all members in the FOA source assembly.

☑ <Member Name>

Identifies the name of the FOA member in the drawing view that you selected to create the parts list.

You can add more FOA members to the parts list by checking the box preceding the FOA member name. You can exclude a FOA member from the parts list by clearing its check mark.

OK

Confirms the members to be included in the FOA parts list. The OK button is available before the parts list is placed when at least one FOA member is selected.

Apply

The Apply button is available when you are modifying the properties of an existing table on the drawing sheet. It provides immediate visual feedback of your changes to the table.
