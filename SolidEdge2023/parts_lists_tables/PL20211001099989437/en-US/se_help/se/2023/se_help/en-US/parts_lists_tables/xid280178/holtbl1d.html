Hole Table Properties dialog box

On this tab

You can do this

General tab

Specify the table style and the maximum table height or number of data rows.

Location tab

Specify the placement location on the sheet, and whether to place it on the current sheet or a new sheet.

See the help topic, Defining table size and location.

Title tab

Specify the text, formatting, and positioning of table titles and subtitles.

See the help topic, Using the Title tab.

Columns tab

Specify the column content and the initial column formatting. You define content by selecting the properties that you want to extract to display in the columns. You can combine multiple properties in each column, and you can add simple text strings to any column.

See the help topic, Using the Columns tab.

Data tab

Insert blank columns and rows, and delete columns and rows. Apply text formatting to the data cells. See the help topic, Using the Data tab.

Sorting tab

Specify that the table is sorted based on the columns included in the hole table.

Options tab

Choose whether holes are arranged by size or by origin in the hole table. Also specify whether holes are renumbered when a hole is deleted or the table is updated.

Set the default hole annotation options, including the default location of the hole callout, and center mark, leader line, and break line display.

Hole Number tab

Reorder and relabel holes in the table according to origin number or hole size category.

Units tab

Define primary units formatting for numeric distance, size, and location values in the hole table.

Callout tab

Specify the content that you want to display in the Hole Callout column.

Smart Depth tab

Specify the smart hole information you want to fetch from the model and display in the Hole Callout column.
