Apply a different table style



To change the appearance of text and lines in a table on the drawing, you can apply a different table style. You can do this for many types of tables, including parts lists, bend tables, and user-defined tables.

  1. On the drawing, right-click the table and choose Properties.

  2. In the Properties dialog box, click the General tab.

  3. On the General page, from the Table style list, select a different style name.

Tip:

For some table types, you can identify the table style associated with the table when you click a table on the drawing. The Table style list on the command bar shows the currently applied style name.
