Text tab (Table Style dialog box)

Selects the text style for the title, column header, and column data in a table. Also displays the Text Box Style dialog box for you to create or modify a text style.

Style

Displays the name of the table style.

Title style

Specifies the style for the table titles.

Column header style

Specifies the style for the table column headers.

Column data style

Specifies the style for the table column data.

Text styles

Lists the text styles available in the file.

Description

Displays a description of the text style that is currently selected in the Text Styles field.

New

Displays the New Text Box Style dialog box.

Modify

Displays the Modify Text Box style dialog box.
