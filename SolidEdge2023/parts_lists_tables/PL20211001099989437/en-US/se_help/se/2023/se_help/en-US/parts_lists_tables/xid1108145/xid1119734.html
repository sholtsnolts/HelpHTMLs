Add or remove dimensions in a tolerance table

  1. Select the tolerance table that you want to modify.

    The method that was used to create the tolerance table is shown on the command bar in the Selection Method list.

    Notice the following:

    • If the table was created with the By User Selection method or the By Active Sheet method, then all of the fit dimensions associated with the table are highlighted.

    • If the table was created with the By Drawing View method, then the drawing view associated with the table is highlighted.

  2. On the command bar, click Add or Remove Dimensions .

  3. (Optional) Choose a different option from the Selection Method list.

    Example:

    If the table was created using the By Active Sheet method, and you want to add all of the new fit dimensions on the sheet to the table, then do not change the selection method. But if you want to selectively add or remove dimensions from the table, then choose By User Selection from the list.

    Example:

    If the table was created using the By Drawing View method, and you want to include another drawing view with fit dimensions in the tolerance table, then do not change the selection method.

  4. Do one of the following:

    • If the table was created using the By User Selection method, then you can click individual dimensions to add them to the table, or you can Shift+click individual dimensions to remove them from the table.

    • If the table was created using the By Drawing View method, then you can click other drawing views to add their dimensions to the existing tolerance table. To remove dimensions from the table, Shift+click individual drawing views.

    • If the table was created using the By Active Sheet method, then do nothing. If fit dimensions were added or removed after the table was created, then the border around the table indicates it is out of date.

  5. On the command bar, click Accept .

    The table updates to show the following:

    • A row is added for every class fit dimension added to the table based on user selection, drawing view selection, or by active sheet selection.

    • A row is removed for every class fit dimension that was removed.
