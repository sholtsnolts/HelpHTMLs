Hole Table command

The Hole Table command creates a hole table that displays information about a selected set of holes on a drawing view.

Example:

Instead of placing individual hole callout annotations on a drawing with the Callout command, you can use the Hole Table command to present the same information about a large number of holes in a clean, uncluttered format.

Before you place the hole table on the drawing sheet, you can use the Hole Table Properties dialog box to format it the way you want.

Note:

To learn how to use the Hole Table command, see the tutorial Creating a hole table on a drawing.
