Bend Table command (Draft)

Creates a bend table on a drawing sheet. Before you place the bend table, you can:

  • Use the Auto-Callout option on the Bend Table command bar (Draft) to automatically create bend callouts for the selected flat pattern drawing view.

  • Use the Bend Table Properties dialog box to select the information you want to place in the bend table and specify how to format it.

To learn more, see Workflow to produce a drawing with a bend table.

To learn how you can change the appearance of individual elements of a table—titles, columns, headers, and data cells—without changing the table style, see the Help topic, Manipulate table rows and columns.
