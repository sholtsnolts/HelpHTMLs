Place a bend table on a drawing sheet

  1. Choose Home→Tables group→Part List split button→Bend Table .

  2. On the drawing sheet, select a flat pattern drawing view.

    Tip:

    To learn how to generate a flat pattern drawing view, see Workflow to produce a drawing with a bend table.

  3. To show bend callouts on the selected flat pattern drawing view, on the Bend Table command bar (Draft), select Auto-Callout.

    See Tips, below, to learn how you can customize the information displayed in the bend callouts.

  4. On the command bar, click the Properties button.

  5. In the Bend Table Properties dialog box, do the following:

    1. On the Bend Callouts tab, select the bend reference information to display in the callouts, such as internal and external angle, bend radius, bend direction, and bend sequence.

    2. Use the other tabs to set the content and formatting options you want to use.

      See the following Help topics to learn more:

      • Defining table size and location

      • Using the Title tab

      • Using the Columns tab

      • Using the Data tab

      • Using the Sorting tab

  6. Click to place the bend table on the drawing sheet.

Tip:

  • To make changes to the bend table content or formatting after you place it, right-click the table and choose Properties.

  • You can specify the information to display in the bend callouts using the Callout Properties dialog box. On the General tab, click the Bend buttons corresponding to the bend information you want to display: Bend Angle, Bend Radius, Bend Direction, Bend Sequence, Bend Quantity, and Included Angle. To make changes to bend callouts, you can do either of the following:

    • To modify one callout, right-click the callout and choose Properties.

    • To modify all callouts on the active sheet, Select similar elements with SmartSelect.

  • You can use the Derive bend direction from drawing views option on the Annotation tab (Solid Edge Options dialog box) to specify that the bend direction is based on the up and down direction of the flat pattern drawing view rather than the model. This is useful when you do not know which side of the flattened model is designated the top.
