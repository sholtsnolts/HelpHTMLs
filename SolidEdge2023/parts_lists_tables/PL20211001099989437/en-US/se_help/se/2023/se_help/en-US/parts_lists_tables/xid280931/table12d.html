Lines tab (Table Style dialog box)

Modifies the lines used to create borders, separators, and grids in the table. The Preview pane shows the effect of changing each option in the Border and Grid Properties list.

Style

Displays the name of the table style.

Border and Grid Properties

Displays a list of the different styles for the different components in the table.

Preview

Displays an example of the table based on the styles specified in the border and grid properties.
