Change the active parts list

  1. Right-click the parts list that you want to make the active parts list.

  2. On the shortcut menu, click Make Active.

Tip:

  • When you create a new parts list, it becomes the active parts list.

  • When you create a new parts list that you want to share the same item numbers with an existing parts list, link it to the active parts list with the Link To Active button on the Parts List command bar.
