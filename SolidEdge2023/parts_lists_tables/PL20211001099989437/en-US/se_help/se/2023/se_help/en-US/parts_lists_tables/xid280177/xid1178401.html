Combine properties in a column

In an associative table, such as a parts list or bend table, you can show the values of multiple properties derived from property text in a single column.

  1. In the Properties dialog box for the selected table, click the Columns tab.

  2. In the Columns list, click the column name to which you want to add properties.

    The property text for the selected column is displayed in the Property text box.

  3. In the Properties list, click the name of the property that you want to add to the selected column, and then click the Add Property button.

    The property text for the selected property is added to the column property text in the Property text box.

  4. Repeat the previous step for each additional property that you want to add to the selected column.

  5. (Optional) Add spacing and punctuation as needed to separate the property text strings by typing directly in the Property text box.

  6. (Optional) Apply formatting to the output text or value—Click within a property text string in the Property text box, and then click the Format button to open the Format Values dialog box.

  7. Click Apply to update the table.
