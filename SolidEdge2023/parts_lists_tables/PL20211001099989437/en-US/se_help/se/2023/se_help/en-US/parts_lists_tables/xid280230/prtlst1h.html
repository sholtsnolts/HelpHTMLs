Renumber a parts list

  1. On the drawing sheet, select the parts list.

  2. On the Parts List command bar, click the Properties button.

    The Parts List Properties dialog box is displayed.

  3. (Optional) Click the Sorting tab, and then select the Sort Criteria options you want to use.

  4. Click the Options tab, and then do the following:

    1. Type a number in the Start with Item Number box.

    2. Type a number in the Increment By box.

    3. Select the Renumber Items/Balloons According to Sort Order option.

Tip:

  • When you delete parts from an assembly, you can use this procedure to renumber the parts list so that no numbers are skipped.

  • If you used automatic ballooning when you created the parts list, renumbering the list also renumbers the balloons.
