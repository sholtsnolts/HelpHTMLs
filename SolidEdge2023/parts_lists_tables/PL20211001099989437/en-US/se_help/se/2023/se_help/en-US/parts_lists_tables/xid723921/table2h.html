Manipulate table rows and columns

You can use the Table Properties dialog box to format table data columns, reorder columns and rows, and show or hide columns and rows. You also can insert and delete columns and rows.

With the exception of moving rows and columns, you also can make these changes by editing a table directly on the drawing sheet, bypassing the Table Properties dialog box altogether.

Insert a column or a row

You can insert columns to contain user-defined data, and you can insert empty cells to provide white space and to group table data.

  1. On the Data tab in the Table Properties dialog box, click a column or row in the table to activate it.

  2. (Insert a column)

    1. Click Insert Column .

    2. In the Insert Column dialog box, type the number of columns you want to add, and specify if the columns should be inserted before or after the currently selected column.

    3. Click OK to insert the column.

  3. (Insert a row)

    1. Click Insert Row .

    2. In the Insert Row dialog box, type the number of rows you want to add, and specify if the rows should be inserted before or after the currently selected row.

    3. Click OK to insert the row.

Tip:

  • Blank rows may be moved if you sort the table using the Sorting tab.

  • You also can right-click a column and choose Insert Columns on the shortcut menu to display the Insert Columns dialog box. You also can right-click a row and choose the Insert Rows command.

  • You can extract property text from the model and display its value in a user-defined column. See Extract model information using property text.

Delete a column

  1. On the Data tab in the Table Properties dialog box, click the column you want to delete.

  2. Click Delete Column .

Tip:

  • You can also right-click the column you want to delete, and then choose Delete Column from the shortcut menu.

  • To select more than one column, hold down the Ctrl key, and then click the columns you want.

  • To select a range of columns, click the first column, hold down the Shift key, and then click the last column.

Change the column order

On the Data tab, you can select and drag any column except the first column to change its display order in the table.

  1. On the Data tab in the Table Properties dialog box, click the column header to select it.

  2. Drag the column to the new location within the table and release the cursor.

Hide a column in a table

Use the Show Column option on the Format Column dialog box to hide and unhide a table column.

  1. On the Data tab in the Table Properties dialog box, right-click the column, and then click Format Column.

  2. In the Format Column dialog box, clear the Show Column check box.

  3. Click OK to update the drawing table.

Tip:

To show a column that currently is hidden, reset the Show Column option on the Format Column dialog box.

Change the column format

Use the Format Column dialog box to change the appearance of derived columns and of columns you insert.

  1. On the Data tab in the Table Properties dialog box, click the column you want to format.

  2. Click Format Column .

  3. In the Format Column dialog box, use the options to make your format changes. You can do such things as add, align, and position headers, hide columns, change column width, and align data within the table.

Tip:

  • You can place a header at the top of the column, at the bottom of the column, at both the top and bottom, or neither.

  • Each column header may have multiple lines of text.

  • Changing the column format will change all pages.

  • The alignment of each column header and data can be defined separately. For example, column 1 might have left justification for both the header and the data, while column 2 has center justification for the header and right justification for the data.

  • You can extract property text from the model and display its value in a user-defined column.

Hide or show rows

To hide and to show rows, use the Hide command on the shortcut menu of the selected row.

  • To hide a row so that it is not visible in the table, on the Data tab in the Properties dialog box, right-click the row and choose Hide, and then click OK to update the drawing table.

  • To show a row that is invisible in the table, right-click the row and choose Hide again, and then click OK to update the table.

    This removes the check mark in front of the word Hide.

Move rows in a table

  1. On the Data tab in the Table Properties dialog box, click the row you want to move.

  2. Specify the direction you want to move the row by clicking one of the following buttons:

    • Move Row Down

    • Move Row Up
