Item Number tab

Use the Item Number tab to edit the item numbers associated with each item in the parts list or block table. To control the formatting of item numbers, use the Options tab.

Only the cells in the first Item Number column may be edited. All other columns display the layout and content of the list based on what is defined on the Columns tab and the Data tab of the Properties dialog box. These columns are provided for reference and are read-only. They also show how the list will appear when it is placed or updated on the drawing.

Item numbers do not update on the drawing until you select Apply or OK.

Item Number (first column)

Edits the item number on the selected row in the parts list when you double-click the data cell.

If the cells in this column are gray shaded, the parts list is sharing the item numbers assigned in the assembly model. These cannot be edited.

  • Valid values are positive integers.

  • You cannot have duplicate item numbers. If you enter a duplicate item number, the text of the changed item and the item in conflict turn red. You must edit one of the item numbers so they are not duplicates before you can update the item numbers.

    Note:

    When you specify that you want duplicate item numbers to be assigned to a block view list in a block table on the sheet, only the first instance of a duplicate number is shown on the Item Number tab. The other instances of the same item number are hidden. If you edit the item numbers to remove duplicates, be sure to unhide the other duplicates so they can be changed, too.

Note:

  • You cannot edit parts list item numbers if either of these check boxes is selected on the Options tab (Parts List Properties dialog box):

    • Renumber Items/Balloons According To Sort Order

    • Use assembly generated item numbers

  • You cannot edit item numbers for a total length parts list.
