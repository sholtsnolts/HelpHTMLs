Create a bend table for the part

  1. In the sheet metal model, select either of these options in the Tools tab→Model group:

    • Synchronous

    • Ordered

  2. Choose Tools→Assistants→Bend Table .

Tip:

The Bend Table dialog box is displayed so you can set the bend sequences for the part. Balloons are displayed on the bends while the Bend Table dialog box is open.

To learn more, see Working with sheet metal bend tables.
