Table cell color and value editing

The background color of a table cell indicates its edit state.

A table cell with this color

Indicates

(white)

Eligible for value and format editing.

Selected for value editing.

(medium gray)

Selected for format editing.

(dark gray)

Disabled for value editing in this state. The value is derived from the model.
Note:

  • When a cell is eligible for editing, you can double-click the cell, type the new information, and then press Tab.

    You also can edit the cell value on the Data tab in the Table Properties dialog box.

  • If the cell is dark gray (disabled for editing), you can use the Allow Cell Overrides command on the cell shortcut menu to edit the cell value or its formatting.

  • You can use the Clear Cell Overrides command on the shortcut menu to return an edited cell to its original value.
