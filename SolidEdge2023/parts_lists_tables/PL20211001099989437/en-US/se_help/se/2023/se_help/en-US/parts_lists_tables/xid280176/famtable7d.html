Variables tab (Family Of Parts Table Properties)

Selects the family driven variables to be shown in the family of parts table on the drawing.

Variables

In alphabetical order, lists all of the variables found in the Variable Table for the selected family of parts, less those variables that are listed in the right-hand pane of this dialog box.

Note:

If a variable has been hidden in the table using the Show Column option on the Format Column dialog box, then it is also not shown in this list.

Add

Moves a variable selected in the left-hand pane to the table list in the right-hand pane.

Remove

Moves a variable selected in the right-hand pane to the variable list in the left-hand pane.

Variables Shown In Table

Lists all of the variables to be shown in the family of parts table on the drawing. The list order is based on the order of the columns on the Data page.

The default list contains all of the family-driven dimensional and positional variables used by any member of the family of parts.

You can remove variables from the table using the Remove button. You can add variables to the table using the Add button.
