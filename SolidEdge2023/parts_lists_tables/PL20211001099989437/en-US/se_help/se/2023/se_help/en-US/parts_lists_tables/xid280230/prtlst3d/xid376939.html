Location tab

Sets the sheet placement and sizing options for parts lists and tables. To learn how you can use the options on the Location tab, see Defining table size and location and Create new sheets for tables.

Anchor corner

Specifies a corner of the table as the anchor point for placing the table on the drawing sheet. This also is the anchor point for making the table larger.

Location

The options in this section specify table location prior to placement.

Create table on active sheet

Places the table on the current sheet using the following specifications:

Page gap

Specifies the distance between individual pages when the table is split into multiple pages. If this is set to 0, then there is no page gap.

Page gap defines the default table page placement. Individual table pages can be moved manually after the table is created.

Enable predefined origin for placement

When selected, specifies that the table is placed on the sheet at the table origin point specified in the X origin and Y origin boxes, rather than freely placed at a user-defined origin.

Note:

When this check box is selected, you cannot place the table dynamically using the mouse.

Create new sheets for table

Creates new sheets to contain the table using the following specifications:

First sheet

Specifies the background and sheet size to use for the first table page.

Additional sheets

Specifies the background and sheet size to use for additional table pages.

Show sheet backgrounds

Specifies that the background sheets are shown on the table sheets when the table is created.

When deselected, sets the sheet size to the background sheet size but does not display the background.

Maintain sheets with table size

Controls the cleanup of table sheets. When selected, specifies that when the table grows, new sheets are created; when the table gets smaller or is deleted, the unused sheets are deleted also.

When deselected, this may result in a blank sheet.

Note:

When this check box is selected, you cannot use the options in the Position section to move a table to a different sheet.

Position
Page

Lists all pages in the table. You can use this option to split table pages across drawing sheets.

For example, you can select a page, and then select a sheet from the Sheet list to move the page to that sheet.

You also can select multiple pages and move them to a single sheet.

Sheet

Lists all drawing sheets in the document. Use this and the Page control to move a page of the table or parts list to another sheet.

Note:

To create links (and to remove links) between family member variables and drawing dimensions in a family of parts table, the table and the dimensioned view must reside on the same sheet.

X origin

Specifies the X coordinate of the table origin point on the sheet.

Y origin

Specifies the Y coordinate of the table origin point on the sheet.
