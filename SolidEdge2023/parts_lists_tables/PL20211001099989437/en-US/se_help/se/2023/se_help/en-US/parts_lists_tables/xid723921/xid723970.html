Copy table contents to the clipboard

You can copy, cut, and paste some or all of the table content using shortcut commands on the Data tab in the Table Properties dialog box, or when editing a table directly on the drawing sheet.

Copy selected table content

  1. Right-click the table and choose Properties.

  2. On the Data tab in the Table Properties dialog box, select the table contents that you want to copy.

  3. Right-click to display the shortcut menu, and choose Copy.

  4. Move the cursor to another table or application where you want to copy the content, and then right-click and choose Paste.

    If you are copying the content to a table, you must select the same number of cells to paste the content.

Tip:

  • The Cut, Copy, and Paste shortcut commands also are available when editing a table directly.

  • The cell formatting is not copied with the content.

Copy the table content and the table border

To copy the table data along with the border and lines:

  1. Right-click the table on the drawing sheet, and choose Copy.

  2. Move the cursor to the sheet where you want to paste the table, and from the ribbon, choose Home tab→Clipboard group→Paste.

Tip:

To learn how you can multi-select table data, see Edit a data cell directly.
