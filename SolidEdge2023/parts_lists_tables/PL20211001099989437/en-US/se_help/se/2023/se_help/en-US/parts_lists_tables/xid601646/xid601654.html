Block Table Properties dialog box

On this tab

You can do this

General tab

Specify the table style and the maximum table height or number of data rows.

Location tab

Specify the placement location on the sheet, and whether to place it on the current sheet or a new sheet.

See the help topic, Defining table size and location.

Title tab

Specify the text, formatting, and positioning of table titles and subtitles.

See the help topic, Using the Title tab.

Columns tab

Specify the column content and the initial column formatting. You define content by selecting the properties that you want to extract to display in the columns. You can combine multiple properties in each column, and you can add simple text strings to any column.

See the help topic, Using the Columns tab.

Data tab

Add and remove columns and rows, and where you edit extracted information displayed in individual cells. See the help topic, Using the Data tab.

Options tab

Specify how you want blocks to be listed (by master block name or by block view name). Specify the table formatting (for example, a nested table with indented numbering).

Item Number tab

After the block table is created, edit the item numbers displayed in the table and in the balloons.

Sorting tab

Specify that the block table is sorted by item number, block name, block view, or quantity. This also affects the presentation of the table data.

Note:

In block tables, sorting is turned off (set to blank) by default.

Groups tab

For a block table that spans multiple table sheets, you can use the to define table subheadings to categorize table data and keep like items together. See the help topic, Grouping data in tables.

Balloon tab

Specify the appearance of the balloons and how they are numbered.
