Saving and reusing the parts list format

You can save a parts list format with a name you define, so you can easily use it again. Use the Saved Settings option on the General page of the Parts List Properties dialog box to name, save, and reapply your parts list format.

After creating a saved setting, a quick way to apply it to a parts list is to use the Saved Settings list on the Parts List command bar.

You also can define different parts list saved settings for different sizes of assembly models. Use the following options on the Drawing View Wizard tab (Solid Edge Options dialog box), under Parts List Saved Settings, to make the assignments:

  • Small Assemblies

  • Medium Assemblies

  • Large Assemblies

In this case, the appropriate saved settings are applied automatically when you select the Parts List command and then select an assembly drawing view.
