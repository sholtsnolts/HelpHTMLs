About Solid Edge 3D Publishing

Use Solid Edge 3D Publishing for technical documentation

You can use Solid Edge 3D Publishing to create technical documentation from your 3D CAD models, and to publish the documents directly to universal, easily shared formats.

You also can create technical illustrations. All of the functionality that is provided by Solid Edge Illustrations is available in Solid Edge 3D Publishing.

Here are some examples of when to use Solid Edge 3D Publishing:

  • You want to create a multiple-page, interactive 3D PDF or HTML5 document or web page.

  • You want to use one of the smart templates tailored to producing different types of documentation in different formats, for example, a click-able parts list or maintenance instructions.

  • You want to link parts list entries to a web page (for example, a vendor website) or to another page in a drill-down parts list.

For more information, start with the Solid Edge3D Publishing workflow.

Template selection

You can create or import custom templates, and you can choose from stock, single-page formats (high-definition, landscape, mobile, portrait) and from multiple-page document templates.
