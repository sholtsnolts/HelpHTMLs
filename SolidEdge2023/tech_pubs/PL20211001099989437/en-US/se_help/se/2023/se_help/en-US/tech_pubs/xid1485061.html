Solid Edge Illustrations workflow

Use Solid Edge Illustrations to create a set of illustrations. This set of illustrations is a storyboard. Once the illustrations are complete, then the storyboard is published to the desired image format or as a PDF or HTML file.

Create graphical Illustrations

To use Solid Edge Illustrations to create graphics:

  1. Open your 3D model in Solid Edge, and then choose Tools tab→Environs group→Solid Edge Illustrations .

  2. Create one or more illustrations. Illustrations capture the viewpoint, render style, exploded state, and many other properties of the 3D view.

  3. Add markups and callouts to your illustrations.

  4. Publish your illustrations as raster or vector images.

Create a 3D PDF or HTML document

To use Solid Edge Illustrations to create a single-page 3D PDF or HTML document or web page:

  1. Open your 3D model in Solid Edge, and then choose Tools tab→Environs group→Solid Edge Illustrations .

  2. Create one or more illustrations.

  3. Pick a single-page template.

  4. Customize the template, change the styles and colors, add logos.

  5. Preview the template and try the interactive buttons and 3D view.

  6. Publish your illustrations as a single page PDF or HTML document.

Create an animation

To use Solid Edge Illustrations to create an animation:

  1. Open your 3D model in Solid Edge, and then choose Tools tab→Environs group→Solid Edge Illustrations .

  2. Create a series of illustrations. Each illustration represents a key frame for your animation.

  3. Set up timing and transitions.

  4. Publish the storyboard as an animation.
