Simplified parts support for technical publications

Simplification of geometry is done at three levels: top-node assembly, subassembly, and part level.

Export options

There are various options for exporting simplified geometry:

Include Configurations

Select this check box to include the required configuration. You can select All Configurations, Active Configuration, or Select Configurations from the dropdown list. Choose Select Configurations to select the active configuration. Click Select to select the required active configuration from the Select Configurations page.

Include Simplified Parts

Select this check box to export simplification of geometry at the part level. Only the active configuration is exported here, wherever the simplified geometry at part level exists.

Include Simplified Assemblies

Select this check box to export simplification of geometry at the subassembly level. Only the active configuration is exported here, wherever the simplified geometry at subassembly level exists.

Include Top Assembly Simplification

Select this check box to export simplification of geometry at the top-node level.

Note:

You can export simplified parts and subassemblies simultaneously.

Multiple instances of same sub assembly or part in one configuration

Example:

Suppose that in an active configuration, there are two instances of the same assembly: one is simplified and another is designed. In that case, you can export simplified as well as designed bodies. But if only the designed body is selected in an active configuration, you can export only the designed body, using the Include Configurations chek box. This will be very helpful as PLMXML will contain fewer number of bodies, therefore, improving performance while exporting large assemblies to technical publications.

Section views

You can also export section views of simplified bodies to technical publications.
