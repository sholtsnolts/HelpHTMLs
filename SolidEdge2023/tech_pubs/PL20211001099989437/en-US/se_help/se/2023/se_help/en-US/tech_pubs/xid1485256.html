Publishing formats

Solid Edge Technical Publications is designed to produce real-world content that is easily distributed to anyone. The software focuses on print, PDF, and HTML, because these represent methods that are widely acceptable forms of documentation delivery.

Print

Printed documents generally refers to paper copies or static imagery. Here are some examples of using Solid Edge Illustrations for print:

  • Publish images of 3D models and include them in a Word document.

  • Publish vector graphics and incorporate them into an Adobe Illustrator project.

To:print a multiple-page document with static 3D images, use Solid Edge 3D Publishing.

PDF

PDF is an industry standard, and these files can be opened on computers with Adobe Reader X or greater installed. When viewed with Adobe Reader, the PDF file supports interactive 3D. The PDF files created with Solid Edge Technical Publications do not require a plug-in to be installed with Adobe Reader.

Example of using Solid Edge Illustrations for PDF output:

  • Publish illustrations as a single-page interactive 3D PDF based on a stock template.

Examples of using Solid Edge 3D Publishing for PDF output:

  • Publish a multiple-page document as a static PDF, complete with high-resolution imagery of 3D models.

  • Publish a multiple-page, interactive 3D PDF document with embedded 3D views and custom page design.

HTML5

HTML5 is an industry standard. Modern web browsers as installed are generally ready to open any HTML documents created with products in Solid Edge Technical Publications. This results in the confident delivery of interactive 3D web pages that are viewable on modern browsers and most modern devices, including PC, Mac, iOS and Android devices. No plug-ins are required to view the interactive 3D documentation.

Example of using Solid Edge Illustrations for HTML:

  • Publish illustrations as a single-page, interactive HMTL5 based on a pre-designed template.

Examples of using Solid Edge 3D Publishing for HTML:

  • Publish a multiple-page, interactive 3D document to a folder of HTML (and other), which can be easily added to your web site.

  • Publish a multiple-page, interactive 3D document as a single-file HTML for easy sharing by email.
