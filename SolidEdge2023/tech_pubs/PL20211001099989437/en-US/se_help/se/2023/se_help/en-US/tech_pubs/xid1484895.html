Solid Edge 3D Publishing workflow

When you want to create a finished document, or if the document requires custom layouts, use Solid Edge 3D Publishing. This application provides tools for working with 3D documents and for designing professional page layouts spanning multiple pages.

Although Solid Edge 3D Publishing is a creative documentation tool, and there is no required process for producing documents, you can use the following steps as a starting point.

Create technical documentation for print

To use Solid Edge 3D Publishing to create a printed document:

  1. From the Solid Edge ribbon, choose Tools tab→Environs group→Solid Edge 3D Publishing .

  2. Create a new document or choose a 3D Publishing template.

    For a preview of some of the stock templates that are available, see About Solid Edge 3D Publishing.

  3. Open a Solid Edge model directly, or open a Solid Edge Illustrations QSM file containing a storyboard. You also can import a 3D CAD model.

  4. Insert text, images, 3D views, and tables.

  5. Set up 3D views—Change materials, change the render style, set up viewpoints, create exploded views, orient the 3D model, capture Illustrations, and add balloons and other markups.

  6. Print the document.

Create an interactive 3D document or web page

To use Solid Edge 3D Publishing to create an interactive 3D document:

  1. From the Solid Edge ribbon, choose Tools tab→Environs group→Solid Edge 3D Publishing .

  2. Create a new document or choose a 3D Publishing template.

    For a preview of some of the stock templates that are available, see About Solid Edge 3D Publishing.

  3. Open a Solid Edge model directly, or open a Solid Edge Illustrations QSM file containing a storyboard. You also can import a 3D CAD model.

  4. Insert text, images, 3D views, and tables.

  5. Set up 3D views—Change materials, change the render style, set up viewpoints, create exploded views, orient the 3D model, capture Illustrations, and add balloons and other markups.

  6. Capture illustrations—Set up animations with order, timing, and transitions using a storyboard.

  7. Add interactivity, such as buttons and click-able parts tables as desired.

  8. Publish as an interactive 3D PDF or 3D HTML5 document.
