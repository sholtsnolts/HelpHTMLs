Frequently asked questions (Synchronous Pulleys)

  • Is it possible to select and perform strength check of the pulleys according to the various international standards?

    Yes, it is possible. From the edge bar, select the desired standard for which you want to carry out strength validation. Available standards are ISO, ANSI, DIN and JIS. Start the Synchronous Pulleys Module and select the type of section from the Design Parameters area.

    Note:

    The section types are based on the standard selected.

  • What options should I choose to create pulleys for standard belt pitch length?

    From the Input Conditions window, select the option Based on given Belt Pitch Length from Design Criteria area. This provides the required options in the Design Parameters area to input the values.

    Note:

    To access the Input Conditions window, on the Design Parameters tab, click the Options button.

  • What options should I choose to create pulleys for standard belt pitch length?

    From the Input Conditions window, select the option Based on given Belt Pitch Length from Design Criteria area. This provides the required options in the Design Parameters area to input the values.

    Note:

    To access the Input Conditions window, on the Design Parameters tab, click the Options button.

  • How can I know the failure reasons when Strength Validation fails?

    Strength Validation fail indicates that the design calculation with the inputs provided does not comply with the performance. Use the Strength Validation Failure window to review the failure reasons and the corresponding changes should make the strength validation pass.

    Note:

    To access the Strength Validation Failure window, click the ? button from Strength Validation area.

  • Is it possible to do the strength check of the synchronous belt for the set of 3 pulleys with center distances and pulley teeth numbers as input values?

    Yes, it is possible. Set the option Number of Pulleys as 3 from the Design Parameters area. Access the Input Conditions window. Select the option Based on given Center Distance from the Design Criteria area and option Based on Pulley Teeth from the Design Method area.


  • I made changes to some parameters in the Design Parameters tab. How to identify between modified and unmodified parameters?

    Editing a part created by Solid Edge Engineering Reference is allowed only in the Assembly environment. Right-clicking the part provides an option to start Engineering Reference Synchronous Pulleys Module. You can modify the parameters and click the Update button to update the parameters of the component.
