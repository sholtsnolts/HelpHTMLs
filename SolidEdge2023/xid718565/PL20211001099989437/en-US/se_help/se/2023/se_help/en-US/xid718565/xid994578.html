Engineering Reference command

Use the features and options of the Tools tab→Environs group→Engineering Reference command to assist in designing the following:

  • Shaft Designer

  • Cam Designer

  • Spur Gear Designer

  • Bevel Gear Designer

  • Worm Gear Designer

  • Rack and Pinion Gear Designer

  • Sprockets Designer

  • Compression Spring Designer

  • Extension Spring Designer

  • Synchronous Pulleys Designer

  • Pulleys Designer

  • Beams Designer

  • Columns Designer

Note:

The Solid Edge Engineering Reference dialog box changes dimensions automatically based on the dpi of high-resolution displays such as HD and 4K monitors.
