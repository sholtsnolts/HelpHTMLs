How to (Beams)

  • How to edit the parameters of the section?

    All the inputs required for a section are available below the Section Type option in the Beam Parameters area on the Design Parameters tab. Use these options to edit the parameters of the selected section.

  • How to add and delete loads?

    Use the Add button on the Loads area on the Design Parameters tab to add a load. Use the Remove button to delete the selected load.
