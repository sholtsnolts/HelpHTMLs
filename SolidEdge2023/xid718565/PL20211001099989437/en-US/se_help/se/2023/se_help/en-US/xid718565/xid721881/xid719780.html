Rack and Pinion Gears module

The Rack Gear Module allows you to design Rack & Pinion Gear Pair. It calculates dimensions and performs strength calculations for Circular & Rectangular Rack Gears with straight teeth.

Use the Design Parameters tab to input the parameters, which define design requirements of a gear. This area also has Options you intent to use to set the base parameters such as type of gear and the calculation method used to design and calculate the strength of the gear.Using the Material button you can set the material of the gears.

Use the Calculated Results tab to view the results, which provide information about the behavior of the Rack and Pinion. You can also view result of strength calculation and the load carrying capacity of the gears.

Use the Calculated Dimensions tab to view the results, which define the geometry of the gears. These parameters are used in the construction of gear pair in Solid Edge.

To design a gear pair, input the parameters of Rack and Pinion on the Design Parameters tab. Click the Calculate button to review the results in Calculated Results tab.

When you are done with the parameters and results (viewing), click the Create button to generate the Solid Edge 3D Model of Rack and Pinion.

Note:

The units of measurement for inputs are based on the units setting in the Solid Edge session. For example, if the Solid Edge session is started with inches units then all the working units of Engineering Reference are in inches only.

Gear models generated by the Solid Edge Engineering Reference tools are suitable for use as representations in machine design and analysis. The manufacture of gears, especially generation of complex tooth profiles and clearances, requires the use specialized machinery, tools and processes. You may want to consult gear manufacturing resources and handbooks before directly using the models generated in Solid Edge for gear production.
