Worm Gear module

The Worm Gear Module allows you to design worm gear and analyze their performance. It allows you to calculate dimensions and perform strength calculations for worm gears.

Use the Design Parameters tab to input the parameters, which define design requirements of a gear. This area also has Options you intent to use to select the calculation methods to design and calculate the strength of the gear. Using the Material button you can set the material of the worm and worm gear.

Use the Calculated Results tab to view the results, which provide information about the behavior of the worm and worm gear. You can also view result of strength calculation and the load carrying capacity of the gear.

Use the Calculated Dimensions tab to view the results, which define the geometry of the worm and worm wheel. These parameters are used in the construction of gear set in Solid Edge.

To design a gear, input the parameters of worm and worm gear through the options on the Design Parameters tab. Switch to Calculated Results and click on the Calculate button to review the results.

When you are completed with the parameters and results (viewing), click on the Create button to generate the 3D Model of Solid Edge worm and worm gear.

Note:

The units of measurement for inputs are based on the units setting in the Solid Edge session. For example, if the Solid Edge session is started with inches units then all the working units of Engineering Reference are in inches only.

Gear models generated by the Solid Edge Engineering Reference tools are suitable for use as representations in machine design and analysis. The manufacture of gears, especially generation of complex tooth profiles and clearances, requires the use specialized machinery, tools and processes. You may want to consult gear manufacturing resources and handbooks before directly using the models generated in Solid Edge for gear production.
