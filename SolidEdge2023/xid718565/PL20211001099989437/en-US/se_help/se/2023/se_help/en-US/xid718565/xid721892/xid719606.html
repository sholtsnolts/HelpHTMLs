Spur Gear Module

The Spur Gear Module allows you to design spur gear and analyze their performance. It allows you to calculate dimensions and perform strength calculations for external and internal gears with straight teeth. The design information of spur gear is presented in three tabs.

Use the Design Parameters tab to input the parameters, which define design requirements of a gear. This area also has options you intent to use to set the base parameters such as type of gear and the calculation method used to design and calculate the strength of the gear.Using the Material button you can set the material of the gear.

Use the Calculated Results tab to view the results, which provide information about the behavior of the gear and pinion. You can also view result of strength calculation and the load carrying capacity of the gear.

Use the Calculated Geometry tab to view the results, which define the geometry of the gear. These parameters are used in the construction of gear set in Solid Edge.

To design a gear, input the parameters of gear and pinion through the options on the Design Parameters tab. Switch to Calculated Results and click the Calculate button to review the results.

When you are completed with the parameters and results (viewing), click the Create button to generate the 3D Model of Solid Edge gear and pinion.

Note:

The units of measurement for inputs are based on the units setting in the Solid Edge session. For example, if the Solid Edge session is started with inches units then all the working units of Engineering Reference are in inches only.

Gear models generated by the Solid Edge Engineering Reference tools are suitable for use as representations in machine design and analysis. The manufacture of gears, especially generation of complex tooth profiles and clearances, requires the use specialized machinery, tools and processes. You may want to consult gear manufacturing resources and handbooks before directly using the models generated in Solid Edge for gear production.

One can save the current Inputs and Results by clicking Reports.
