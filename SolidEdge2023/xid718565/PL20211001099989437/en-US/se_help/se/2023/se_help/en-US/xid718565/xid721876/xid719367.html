How to (Cam module)

  • How to select the profile type for a segment of cam?

    The Profiles parameter in Segment Settings area on the Calculated Results tab allows you to select the profile type for each segment.

  • How to identify the current selected Segment?

    The Segment parameter on the Calculated Results tab indicates the current selected segment.
