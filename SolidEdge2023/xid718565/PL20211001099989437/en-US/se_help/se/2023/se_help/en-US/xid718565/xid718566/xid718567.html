Shaft module

The Solid Edge Shaft Designer allows you to design and create a model of shaft. It also allows you to perform a 3D force analysis of the shaft. The design information of shaft is presented on two tabs:

  • Use the Design Parameters tab to define the parameters, which gives the shape to the shaft. This area is also used to define the parameters for force analysis on the shaft. The material of the shaft can be set using the Material button available on the form.

  • Use the Calculated Results tab to view the graphical output and other related results, which provide information about the behavior of the shaft.

To design the shaft, input the number of sections and the details of each section through the parameters on the Design Parameters tab. These form the base parameters to design the shaft. Switch to Calculated Results and click the Calculate button to review the results for the current configuration.

When you are completed with the parameters and results (viewing), click the Create button to generate the 3D Model of Solid Edge shaft model.

Note:

The units of measurement for inputs are based on the units setting in the Solid Edge session. For example, if the Solid Edge session is started with inches units then all the working units of Engineering Reference are in inches only.

One can save the current Inputs and Results by clicking Reports.
