Using the Pulley module

This part of the help document provides you with the details on usage of the Pulley Module.

The design of a pulley starts with defining type of groove profile, section and pulley dimensions. Use the Options button to access the Input Conditions window. Select design criteria, design method and load calculation method based on available design parameters.

Design Criteria area provides you with the two options, based on the center distances and the belt pitch length, for constraining the pulleys' layout.

Design Method area provides you with the two options, based on the pulley pitch circle diameter and transmission ratio, for designing the pulley parameters.

In Options, Checking the Place with Assembly relations Check box will automatically add basic required Assembly Relations in Assembly.

Based on the above selection, input the parameters through the options available in Calculation Parameters , Pulley Dimensions and Assembly Dimensions area.

When you are completed with specifying the inputs, click on the Calculate button to calculate the results. The Calculated Results tab displays the calculation results. A value of Pass for Strength Validation indicates that the current design is validated.

Use the Create button to create 3D model of pulleys' set.

Use the Reports button to save all selected Options , Input parameters, Calculated Results & Calculated Geometry.
