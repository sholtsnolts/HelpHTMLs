How to (Compression Springs)

  • How to change the direction of coils for a compression spring?

    The Coil Direction option from the Spring Parameters area on the Design Parameters tab allows you to set the direction of coils.

  • How to choose the spring end type for a compression spring?

    The Spring Ends option from the Spring Parameters area on the Design Parameters tab allows you to set the end type for the spring.

  • How to check the spring for buckling?

    The Buckle check for Guided Mounting option from the Loading Type area on the Input Parameters window allows you to check the spring performance for buckling. A guide is required if the calculation suggest that the spring fails the buckling check. This is indicated by the output parameter Buckle Check - Guide Required ? in the Calculated Results tab.

    Note:

    To access the Input Conditions window, select the options button from the Design Parameters tab.
