Using the Sprockets module

This part of the help document provides you with the details on usage of the Sprocket Module.

The strength check of chain starts with defining basic input parameters like chain type, chain number and number of sprockets. Use the Options button to access the Input Conditions window. Select design criteria, design method and load calculation method based on available design parameters.

The Design Criteria area provides you with the two options, based on the center distances and the number of chain links, for constraining the sprockets layout.

The Design Method area provides you with the two options, based on the sprocket teeth number and transmission ratio, for defining the sprocket parameters.

In Options, Checking the Place with Assembly relations Check box will automatically add basic required Assembly Relations in Assembly.

Based on the above selection, input the parameters through the options available on the Design Parameters, Sprocket Dimensions and Assembly Dimensions areas.

When you are completed with specifying the inputs, click on the Calculate button to calculate the results. The Calculated Results tab displays the calculation results. A value pass for Strength Validation indicates that the current strength check is validated.

Use the Create button to create 3D model of sprockets' set.

Use the Reports button to save all selected Options , Input parameters, Calculated Results & Calculated Geometry.
