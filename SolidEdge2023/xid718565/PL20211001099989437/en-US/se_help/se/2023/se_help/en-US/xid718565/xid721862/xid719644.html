Beam module

The Beam Module allows you to design and create a model of beam. It also allows you to perform a 3D-force analysis on the beam. The design information of the beam is presented on 2 tabs:

  • Use the Design Parameters tab to define the parameters, which gives the shape to the beam. This area is also used to define the parameters for force analysis on the beam. The material of the beam can be set using the Material button available on the form.

  • Use the Calculated Results tab to view the graphical output and other results, which provide the information about the behavior of the beam.

To design the beam, input the section type and its details through the parameters on the Design Parameters tab. These are the base parameters to design the beam. Switch to Calculated Results tab or click on the Calculate button to review the results for the current configuration.

When you are completed with the parameter results (viewing), click on the Create button to generate the 3D model of Solid Edge beam model.

Note:

The units of measurement for inputs are based on the units setting in the Solid Edge session. For example, if the Solid Edge session is started with inches units then all the working units of Engineering Reference are in inches only.
