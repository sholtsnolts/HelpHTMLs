Sprocket tutorial

Welcome

This tutorial gives a step by step instructions to select the chain and perform strength validation for chains that further creates a model of sprockets' set. No prior experience of using Solid Edge or Engineering Reference is required for this tutorial. However an engineering knowledge of sprockets will be useful.

Tutorial Guide

Each step in the tutorial demonstrates the task to be performed with GUI (Graphical User Interface) of Engineering Reference. Some tasks may require interaction with Solid Edge also.

This tutorial should take about 15 minutes to complete

Introduction

In this tutorial you design and create a model of sprockets with the following specifications.

Select and check compatibility of ISO 16B - Normal type chain for the set of two sprockets and single strand for the following input parameters:

  • Power to be transmitted=10 kW

  • Transmission ratio=2.72

  • Speed of motor shaft=950 rpm

  • Teeth on driving sprocket=25

  • Center distance = 600 mm

  • Operating Conditions: Operating temperature 60-Degree Celsius, Driving conditions-Light duty, Driver shaft conditions- Heavy shocks, 16 hrs/day service with continuous lubrication

  1. Start Solid Edge.

  2. Click the File tab→New→ISO Assembly.

  3. Click the Tools tab→Environs group→Engineering Reference→Sprockets Designer.

  4. Make sure that Design Parameters tab is selected.

  5. Select the B-normal series as Chain Type from Design Parameters area. Select 16B as Chain Number from Design Parameters area.

  6. Since a set of two sprockets and single strand chain is specified, use the Number of Sprockets and Number of Chain Strands options respectively from Design Parameters area.

  7. Click the Options button to access the Input Conditions window. From the given inputs it is clear that we need to design the chain for given center distance and transmission ratio.

  8. Select the option Based on Center Distance from Design Criteria area and the option Based on Transmission Ratio from Design Method area.

  9. Select Calculate Torque option from Load Calculation area.

  10. Click the OK button to close Input Conditions window.

  11. Based on the options selected in the Input Conditions window, Design Parameters tab provides the parameters to input the data.

  12. Input the parameters through the options available on the Design Parameters tab.

  13. Click the OK button to close the Service Factor window.

  14. Click the Calculate button to calculate the results.

  15. Review the results from the Calculated Results tab and Calculated Dimensions tab.

  16. Click the Create button to create the model of sprockets' set in Solid Edge.

Note:

If the current environment is a Part Environment then a new part is created. Use File, Save As command in Solid Edge to save the file to a different location. If the current environment is Assembly Environment then an instance of the 3D model is added to the assembly and the assembly is saved.

This concludes the tutorial.

In this tutorial you have learned how to use Sprocket Module to design and create a 3D model of sprockets' set for the given inputs.
