Compression Spring tutorial

Welcome

This tutorial gives step by step instructions to design and create a model of compression spring. No prior experience of using Solid Edge or Engineering Reference is required for this tutorial. However an engineering knowledge on the design of compression spring will be useful.

Tutorial Guide

Each step in the tutorial demonstrates the task to be performed with Graphical User Interface (GUI) of Engineering Reference. Some tasks may require interaction with Solid Edge also.

This tutorial should take about 15 minutes to complet.e

Introduction

In this tutorial you design and create a model of compression spring with the following specifications.

A cold coiled compression spring is required for Preload (Fp)=350N and Applied Load (Ff)=600N for required maximum deflection of 15 mm. The mounting space has inside diameter of 37 mm. The length at applied load should be 50 mm.

  1. Start Solid Edge.

  2. Click the File tab→New→ISO Assembly.

  3. Click the Tools tab→Environs group→Engineering Reference→Compression Spring Designer.

  4. Make sure that Design Parameters tab is selected.

  5. Select the Options button to access the Input Conditions window. From the given input it is clear that we need to design the spring for given forces and a constraint on inside diameter. Select Design Spring at Given Loads option from Design Criteria area.

  6. Select Based on Static Loading option from Loading Type area.

  7. Since a limit on deflection is specified, select Limit Spring Deflection option from Dimension Limits area.

  8. Since inside diameter is input, select Specify Inside Diameter option from Diameter Selection area.

  9. Click OK to close the Input Conditions window.

  10. Select Maximum Deflection option from Deflection Constraints area and input the value of deflection using the Maximum Deflection option from Spring Parameters area. Also input the value of Assembly Dimensions as Length at Applied Load.

  11. Based on the options selected in the Input Conditions window, Design Parameters tab provides the parameters to input the data.

  12. Input the parameters through the options available in the Design Parameters tab.

  13. Since a cold coiling process is specified, use the Coiling Process option from Spring parameters area to select this option.

  14. Click the Calculate to calculate the results.

  15. Review the results from the Calculated Results tab.

  16. Click Create to create a model of compression spring in Solid Edge.

Note:

If the current environment is a Part Environment then a new part is created. Use File, Save As command in Solid Edge to save the file to a different location. If the current environment is Assembly Environment then an instance of the 3D model is added to the assembly and the assembly is saved.

This concludes the tutorial.

In this Tutorial you have learned how to use Compression Spring Module to design and create a 3D model of spring for the given inputs.
