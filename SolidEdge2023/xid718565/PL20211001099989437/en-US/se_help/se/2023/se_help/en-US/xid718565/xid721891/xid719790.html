Sprocket module

The Sprocket module allows you to select the standard roller chain sprockets and analyze the performance. It allows you to calculate dimensions and perform strength calculations for various types of chains and their sprockets. The design information of sprockets is presented in 3 tabs.

Use the Design Parameters tab to input the parameters, which define the design requirements of chain and sprocket dimensions. This area also has options you intent to use to set the base parameters such as number of chain links, number of teeth on the sprockets, center distance and other parameters.

Use the Calculated Results tab to view the results, which provide the information about the behavior of the chain and sprockets. You can also view the result of strength calculation and load carrying capacity of the chains.

Use the Calculated Dimensions tab to view the results, which define the geometry of the chain and their sprockets. These parameters are used in the construction of the sprockets in Solid Edge.

To configure the sprockets, input the parameters through the options on the Design Parameters. Switch to the Calculated Results tab and click on the Calculate button to review the results.

When you are completed with the parameters and results (viewing), click on the Create button the 3D model of Solid Edge sprockets.

Note:

The units of measurement for inputs are based on the units setting in the Solid Edge session. For example, if the Solid Edge session is started with inches units then all the working units of Engineering Reference are in inches only.

You can save the current Inputs and Results by clicking Reports.
