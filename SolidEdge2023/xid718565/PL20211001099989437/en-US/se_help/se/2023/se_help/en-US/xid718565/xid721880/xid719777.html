How to (Pulleys)

  • How to change the number of belts?

    The input Number of Belts from the Design Parameters area allows you to set the number of belts.

  • How to constrain the center distance between the pulleys for three pulley assembly?

    Select the option Adjust Center Distance Between from the Design Parameters area. The input you have set means you are allowing the change in set center distance, while the others remain unchanged.
