Rack and Pinion Gears tutorial

Welcome

This tutorial gives step by step instructions to design and create a model of Rack gear. No prior experience of using Solid Edge or Engineering Reference is required for this tutorial. However an engineering knowledge on the design of Rack gear will be useful.

Tutorial Guide

Each step in the tutorial demonstrates the task to be performed with GUI (Graphical User Interface) of Engineering Reference. Some tasks may require interaction with Solid Edge also.

This tutorial should take about 10 minutes to complete

Introduction

IIn this tutorial you design and create a model of Rack gearing with the following specifications.

  • Type of Rack is Rectangular Rack

  • Linear Travel of Rack = 90 Degree

  • Gear Module = 2 mm

  • Tangential Force = 27 N

  • Linear Speed of Rack = 0.02 m/s

  • Gear Material = Structural steel 32

  1. Start Solid Edge.

  2. Click the File tab→New→ISO Assembly.

  3. Click the Tools tab→Environs group→Engineering Reference→Rack and Pinion Gears Designer.

  4. Make sure that Design Parameters tab is selected.

  5. Select the Options button (to access the Input Conditions window) and select Rectangular Rack option from Type of Rack area.

  6. Leave the other options in this form to default and click on OK.

  7. Select the material of the Gears from the Material button & select given matrial from the available list

  8. Input the above parameters on the Design Parameters tab and click Calculate.

  9. Review the results from the Calculated Results tab and Calculated Dimensions tab.

  10. Click on Create to create a model of Rack and Pinion in Solid Edge.

Note:

If the current environment is a Part Environment then a new part is created. Use File, Save As command in Solid Edge to save the file to a different location. If the current environment is Assembly Environment then an instance of the 3D model is added to the assembly and the assembly is saved.

This concludes tutorial.

In this Tutorial you have learned how to use Rack Gear Module to design and create a 3D model of Rack and Pinion for the given inputs.
