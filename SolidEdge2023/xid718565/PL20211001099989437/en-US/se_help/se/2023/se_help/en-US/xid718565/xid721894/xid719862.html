Frequently asked questions (Worm Gears)

  • How do I design the worm gear to have the tooth profile as Form-ZN?

    Select the option Tooth Form of Worm from the Gearing Parameters area. You use this option to input the worm tooth profile as Form-ZA or Form-ZN.

    Note:

    To access the Input Conditions window, on the Design Parameters tab, click the Options button.

  • I want to input a driving member as a worm gear (worm wheel). What should I do?

    From the Input Conditions window, select the option Worm Gear from the Driver area. This will make the worm gear (worm wheel ) as a driving member in design calculations.

    Note:

    To access the Input Conditions window, on the Design Parameters tab, click the Options button.

  • How can I know the failure reasons when the Strength Validation fails?

    The Strength Validation fail indicates that the design calculation with the inputs provided does not comply with the performance. Use the Strength Validation Failure window to review the failure reasons and the corresponding changes should make the strength validation pass.

    Note:

    To access the Strength Validation Failure window, click the ? button from the Strength Validation area.

  • I want to enter bearing span as an input. What should I do?

    Use the option Bearing Span from the Design Parameters area to enter the required bearing span.


  • I have transmission ratio as an input value. How do I select the other design parameters like center distance, diametral quotients and gears teeth number as per the transmission ratio?

    From the Input Conditions window, select the option Based on Transmission Ratio from the Design Method area. This provides the required options in the Gearing Parameters and Assembly Dimensions areas. Input to the option Transmission Ratio from Gearing Parameters area. Per this input, the center distance, diametral quotients, worm and worm gear teeth number fields from Assembly Dimensions area are updated. The values in these sets are as per standard selected. You can select any one of the rows as a design parameters combination for the worm design.

    Note:

    To access the Input Conditions window, on the Design Parameters tab, click the Options button.

  • I want review the calculated worm deflection and its allowable deflection values. What should I do?

    Go to the Calculated Results tab. Use the Worm Deflection and Allowable Worm Deflection option from the Strength Results area to review the calculated and allowable worm deflection.


  • I made changes to some parameters in the Design Parameters tab. How to identify between modified and unmodified parameters?

    An underline is placed when any parameter is modified. All underlined parameters indicate that they are modified and a calculation is required to update the results.


  • I created a worm gear using Solid Edge Engineering Reference. How do I edit the part to change some of the design parameters?

    Editing a part created by Solid Edge Engineering Reference is allowed only in the Assembly environment. Right-clicking the part provides an option to start Engineering Reference Worm Gear Module. You can modify the parameters and click the Update button to update the parameters of the component.
