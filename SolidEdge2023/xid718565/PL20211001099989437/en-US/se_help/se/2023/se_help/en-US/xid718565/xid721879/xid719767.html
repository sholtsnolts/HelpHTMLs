How to (Extension Springs)

  • How to change the direction of coils for a extension spring?

    The Coil Direction option from the Spring Parameters area on the Design Parameters tab allows you to set the direction of coils.

  • How to select the Hook Type for an extension spring?

    The Hook Type option from the Spring Parameters area on the Design Parameters tab allows you to set the hook type for the spring.
