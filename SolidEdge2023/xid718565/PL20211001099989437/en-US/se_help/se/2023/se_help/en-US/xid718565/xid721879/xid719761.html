Using the Extension Spring module

This part of the help document provides you with the details on usage of the Extension Spring Module.

The design of extension spring starts with setting of options to define the type of loading and design criteria. Use the options button to specify these details. Select the Design Method based on the available design parameters.

Based on the above selection, input the parameters through the options available in Dimensions and Spring Parameters area. Select the material of the spring using the Material button.

When you are completed with specifying the inputs, click on Calculate to calculate the results. The Calculated Results tab displays the calculation results. A value of Pass for the Strength Validation indicates that the current design is validated.

Use the Create button to create 3D model of extension spring.

Use the Reports button to save all selected Options , Input parameters, Calculated Results & Calculated Geometry.
