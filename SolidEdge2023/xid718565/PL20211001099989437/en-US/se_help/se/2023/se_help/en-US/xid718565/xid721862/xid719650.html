Beam tutorial

Welcome

This tutorial gives a step by step instructions to design and create a model of beam. No prior experience of using Solid Edge or Engineering Reference is required for this tutorial. However an engineering knowledge on the design of beam will be useful.

Tutorial Guide

Each step in the tutorial demonstrates the task to be performed with GUI (Graphical User Interface) of Engineering Reference. Some tasks may require interaction with Solid Edge also.

This tutorial should take about 15 minutes to complete.

Introduction

In this tutorial you design and create a model of beam with following specifications:

  • Material = Cast Steel

  • Section Length = 5000 mm

  • Number of supports = 2

  • Number of loads= 2

  1. Start Solid Edge.

  2. Click the File tab→New→ISO Assembly.

  3. Click the Tools tab→Environs group→Engineering Reference→Beam Designer.

  4. Make sure that Design Parameters tab is selected.

  5. Click the Material button to access the Material Values window. Input the Material as Cast Steel.

  6. Input Section Length as 5000 mm from the Beam Parameters area. Also use the Section Type option to define the section type as Circular.

  7. As the circular section is specified with the inner diameter, check the Hollow option from the Design Parameters area. Input the Outer Diameter and Inner Diameter with the specified values.

  8. Select the Support Number as 1 from the Supports area.

  9. As the first support is located at 1000 mm, set the input Distance as 1000. As the first support is fixed, check the Fixed option from the Supports area.

  10. Now select the Support Number as 2.

  11. Input the Distance as 5000. By default the Axially Rigid option is unchecked for the selected support.

  12. By default the force is at the center of the beam. Use the Add button from the Loads area to create one more force. The locations of these forces need to be specified and the properties of each force need to be edited.

  13. Select the option Load Number as 1. The Distance parameter from the Loads area allows you to set the location of each force on the beam. The first force is to be located at 1500 mm. Set the value of Distance to 1500. Use the remaining parameters in this area to input the properties of the force as in the force table.

  14. Similarly set the Distance as 3500 mm for second force and input the remaining properties of the force as in the force table.

  15. Click on the Calculate button to calculate the results. The calculated results are displayed in the Calculated Results tab.

  16. Click on the Create button to create model of beam in Solid Edge.

Note:

If the current environment is a Part Environment then a new part is created. Use File, Save As command in Solid Edge to save the file to a different location. If the current environment is Assembly Environment then an instance of the 3D model is added to the assembly and the assembly is saved.

This concludes the tutorial.
