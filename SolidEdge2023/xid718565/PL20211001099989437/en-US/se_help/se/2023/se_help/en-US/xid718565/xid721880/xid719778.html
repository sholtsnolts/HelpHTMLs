Frequently asked questions (Pulleys)

  • Is it possible to select and perform strength check of the pulleys according to the various international standards?

    Yes, it is possible. Select the desired standard from the Standard option provided on the edge bar. Available standards are ISO, ANSI, DIN, and JIS. These form the base for calculations. Select the inputs Groove Profile and Section from the Design Parameters area.

    Note:

    The groove profiles are according to the standard selected while the sections are according to the selected groove profile.

  • I have belt pitch length as input value. What options should I choose to create pulleys for this value?

    From the Input Conditions window, select the option Based on Belt Pitch length from Design Criteria area. This provides the required options in the Design Parameters area to input the values.

    Note:

    To access the Input Conditions window, on the Design Parameters tab, click the Options button.

  • How can I know the failure reasons when the Strength Validation fails?

    Strength Validation fail indicates that the design calculation with the inputs provided does not comply with the performance. Use the Strength Validation Failure window to review the failure reasons and the corresponding changes should make the strength validation pass.

    Note:

    To access the Strength Validation Failure window, click the ? button from Strength Validation area.

  • Is it possible to select and do the strength check of the v-belt for three pulleys with center distances and pulley pitch circle diameters as input values?

    Yes, it is possible. Set the input Number of Pulleys from Design Parameters area as 3. Access the Input Condition window. Select the option Based on given Center Distance from the Design Criteria area and option Based on Pulley Diameters from Design Method area.

    Note:

    Based on the above selection, input the parameters through the options available in Design Parameters, Pulley Dimensions and Assembly Dimensions area.

  • I made changes to some parameters on the Design Parameters tab. How to identify between modified and unmodified parameters?

    An underline is placed when a parameter is modified. All underlined parameters indicate that they are modified and a calculation is required to update the results.


  • I created pulleys using Solid Edge Engineering Reference. How do I edit the part to change some of the design parameters?

    Editing a part created by Solid Edge Engineering Reference is allowed only in the Assembly environment. Right-clicking the part provides an option to start Engineering Reference Pulley Module. You can modify the parameters and click the Update button to update the parameters of the component.
