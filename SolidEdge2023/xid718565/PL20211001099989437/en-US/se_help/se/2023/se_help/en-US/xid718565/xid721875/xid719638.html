Bevel gear tutorial

Welcome

This tutorial gives step by step instructions to design and create a model of bevel gear. No prior experience of using Solid Edge or Engineering Reference is required for this tutorial. However an engineering knowledge on the design of bevel gear will be useful.

Tutorial Guide

Each step in the tutorial demonstrates the task to be performed with GUI (Graphical User Interface) of Engineering Reference. Some tasks may require interaction with Solid Edge also.

This tutorial should take about 10 min to complete.

Introduction

In this tutorial you design and create a model of bevel gear with the following specifications:

  • Input power = 10 KW

  • Input speed = 1150 rpm

  • Gear Ratio = 1.5

  • Gear Material = Structural steel 32

  1. Start Solid Edge.

  2. Click the File tab→New→ISO Assembly.

  3. Click the Tools tab→Environs group→Engineering Reference→Bevel Gear Designer.

  4. Make sure that Design Parameters tab is selected.

  5. Select the Options button (to access the Input Conditions window) and select Find Torque option from Load Calculation area. We select this option since we have the power and the speed values as input parameters.

  6. Select Based on Simple Bending Stress option from Strength Calculation Method area.

  7. Select Based on given Material Strength option from Strength Calculation area.

  8. Leave the other options in this form to default and click on OK.

  9. Select the material of the Gear from the Material button.

  10. Input the above parameters through the options available in the Design Parameters tab and click Calculate.

  11. Review the results from the Calculated Results tab and Calculated Geometry tab.

  12. Click the Create button to create a model of bevel gear in Solid Edge.

Note:

If the current environment is a Part Environment then a new part is created. Use File, Save As command in Solid Edge to save the file to a different location. If the current environment is Assembly Environment then an instance of the 3D model is added to the assembly and the assembly is saved.

This concludes tutorial.

In this Tutorial you have learned how to use Bevel Gear Module to design and create a 3D model of bevel gear for the given inputs.
