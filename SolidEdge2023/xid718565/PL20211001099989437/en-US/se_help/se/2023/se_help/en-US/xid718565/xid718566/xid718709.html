Using the Shaft module

This part of the help document provides you with the details on usage of the Shaft Module.

The design of a shaft starts with specifying the following details:

  • Number of Sections: The Add --> will place a new section after the current selected section and the Add <-- will place a new section before the current selected section. The Delete button will delete the current selected section. Using these three buttons you can define the number of sections in the shaft.

  • Geometric Detail of each section: Each section can take one of the following shapes:

    • Simple

    • Ring Groove

    • Keyway

    • Locknut Groove

    • Cone

You can choose the type of section from the Section Type option from the Shaft Parameters area.

Provide the dimensional details of each section through the remaining options in this area.

The above details will give the shaft, its overall shape and size. The next task is to edit the location of supports and specify the number of forces and their properties.

By default the shaft is supported on two supports. The location of a support can be controlled through options provided in Supports area.

By default the shaft is applied with a single force in the middle of the shaft. Use the Add button from the Loads area to add more number of forces. The location and the properties of a force can be controlled through options provided in Loads area.

This completes the definition of input parameters for the design of a shaft. Switch to the Calculated Results tab and click on Calculate to calculate the results.

Use the Create button to create 3D model of shaft.

Use the Reports button to save all selected Options , Input parameters, Calculated Results & Calculated Geometry.
