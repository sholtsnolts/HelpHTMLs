Using the Columns module

This part of the help document provides you with the details on usage of the Column Module.

The design of column starts with specifying following details.

Section Type: A section can take one of the following shapes:

  • Circular

  • Hexagonal

  • Square

  • Rectangular

  • Triangular

  • Trapezoidal

  • I-beam

  • L-beam

  • C-channel

Note:

When you choose the section type as circular, rectangular or square, you are provided with a Hollow option. This option allows you to define the cutout or hole in the section.

You can choose the type of section from the Section Type option from the Column Parameters area. Provide the dimensional details of each section through the remaining options in this area.

The above details will give the column its overall shape and size.

The next task is to input the maximum magnitude of axial force, end conditions of the column and required minimum safety factor from the Load area.

Use the Options button to access the Input Conditions window. The Check Calculation option from the Design Criteria area allows you to check the strength of the column for the selected material from the Material Values window. Material Design option from the same area calculates the minimum yield strength of the material for the inputs factor of safety and modulus of elasticity.

Based on the above selection, input the parameters through the options available in Load area.

This completes the definition of input parameters for the design of a column. Switch to the Calculated Results tab and click on the Calculate button to review the results.

Use the Create button to create 3D model of column.

Use the Reports button to save all selected Options, Input parameters, Calculated Results & Calculated Geometry.
