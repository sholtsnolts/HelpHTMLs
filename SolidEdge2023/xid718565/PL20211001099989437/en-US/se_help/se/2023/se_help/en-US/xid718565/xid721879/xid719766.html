Extension Spring tutorial

Welcome

This tutorial gives step by step instructions to design and create a model of extension spring. No prior experience of using Solid Edge or Engineering Reference is required for this tutorial. However an engineering knowledge on the design of extension springs will be useful.

Tutorial Guide

Each step in the tutorial demonstrates the task to be performed with Graphical User Interface (GUI) of Engineering Reference. Some tasks may require interaction with Solid Edge also.

This tutorial should take about 15 minutes to complete.

Introduction

In this tutorial you design and create a model of extension spring with the following specifications.

A extension spring is required with B Loops for a static load of 250N. The length of the spring is 160 mm and is to be designed for a deflection of 30mm. The maximum spring deflection allowed is 55 mm and the space available admits an outside diameter of coil of not more than 50 mm.

  1. Start Solid Edge.

  2. Click the File tab→New→ISO Assembly.

  3. Click the Tools tab→Environs group→Engineering Reference→Extension Spring Designer.

  4. Make sure that Design Parameters tab is selected.

  5. Select the Options button to access the Input Conditions window. From the given input it is clear that we need to design the spring for given forces and a constraint on outside diameter. Select Design Spring at Given Loads option from Design Criteria area.

  6. Since outside diameter is input, select Specify Outside Diameter option from Diameter Selection area.

  7. Since a limit on deflection and hook type is specified, select Based on Spring Diameter, Hook Type and Loads option from Design Method area.

  8. Click OK to close the Input Conditions window.

  9. Based on the options selected in the Input Conditions window, Design Parameters tab provides the parameters to input the data.

  10. Input the parameters through the options available in the Design Parameters tab.

  11. Select Maximum Deflection option from Deflection Constraints area and input the value of deflection using the Maximum Deflection option from Spring Parameters area.

  12. To specify the type of hook Select Hook Type option from the Spring Parameters area and then select Full loop from the Hook Type window. Click OK to close the window.

  13. Click on Calculate to calculate the results.

  14. Review the results from the Calculated Results tab.

  15. Click on Create to create a model of extension spring in Solid Edge.

Note:

If the current environment is a Part Environment then a new part is created. Use File, Save As command in Solid Edge to save the file to a different location. If the current environment is Assembly Environment then an instance of the 3D model is added to the assembly and the assembly is saved.

This concludes the tutorial

In this Tutorial you have learned how to use Extension Spring Module to design and create a 3D model of spring for the given inputs
