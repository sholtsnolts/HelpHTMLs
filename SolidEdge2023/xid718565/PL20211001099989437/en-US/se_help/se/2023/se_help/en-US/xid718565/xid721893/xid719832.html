Synchronous Pulley tutorial

Welcome

This tutorial gives step by step instructions to design and create a model of synchronous pulleys. No prior experience of using Solid Edge or Engineering Reference is required for this tutorial. However an engineering knowledge of synchronous pulley will be useful.

Tutorial Guide

Each step in the tutorial demonstrates the task to be performed with GUI (Graphical User Interface) of Engineering Reference. Some tasks may require interaction with Solid Edge also..

This tutorial should take about 15 minutes to complete.

Introduction

In this tutorial you design and create a model of synchronous pulley with the following specifications.

Select and check compatibility of ISO H200 type synchronous-belt for the set of two pulleys for the following input parameters:

  • Input Power = 2 kW

  • Input shaft speed = 850 rpm

  • Teeth on Pulley:

    • Pulley 1 = 20

    • Pulley 2 = 48

  • Center Distance Between:

    • Pulley 1 and Pulley 2 = 265 mm

  • Operation Conditions: Driving Conditions — Uniform, Driver shaft conditions — Light shocks

  1. Start Solid Edge.

  2. Click the File tab→New→ISO Assembly.

  3. Click the Tools tab→Environs group→Engineering Reference→Synchronous PulleysDesigner.

  4. Make sure that Design Parameters tab is selected.

  5. Select the Section as H200.

  6. Since a set of two pulleys is specified, set the input Number of Pulleys to 2.

  7. Click the Options button to access the Input Conditions window. From the given specification it is clear that we need to design the pulley for given center distance and pulley teeth numbers.

  8. Select Based on given Center Distances from the Design Criteria area and Based On Pulley Teeth from the Design Method area.

  9. Select Calculate Torque option from Load Calculation area.

  10. Click the OK button to close the Input Conditions window

  11. Based on the options selected in the Input Conditions window, the Design Parameters tab provides the parameters to input the data.

  12. Click the Service Factor option from the Design Parameters area to open the Service Factor window.

  13. Input the parameters according to the operating conditions.

  14. Click OK button to close the Service Factor window.

  15. Click Calculate button to calculate the results.

  16. Review the results from the Calculated Results tab and Calculated Dimensions tab.

  17. Click Create to create the model of synchronous pulleys' set in Solid Edge.

Note:

If the current environment is a Part Environment then a new part is created. Use File, Save As command in Solid Edge to save the file to a different location. If the current environment is Assembly Environment then an instance of the 3D model is added to the assembly and the assembly is saved.

This concludes the tutorial.

In this Tutorial you have learned how to use Synchronous Pulley Module to design and create a 3D model of synchronous pulleys' set for the given inputs.
