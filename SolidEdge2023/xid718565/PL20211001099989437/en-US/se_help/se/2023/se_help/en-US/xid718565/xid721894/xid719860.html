Worm Gear tutorial

Welcome

This tutorial gives step by step instructions to design and create a model of worm gear. No prior experience of using Solid Edge or Engineering Reference is required for this tutorial. However an engineering knowledge on the design of worm gear will be useful.

Tutorial Guide

Each step in the tutorial demonstrates the task to be performed with GUI (Graphical User Interface) of Engineering Reference. Some tasks may require interaction with Solid Edge also.

This tutorial should take about 20 minutes to complete.

Introduction

In this tutorial you design and create a model of worm gearing with the following specifications:

  • Worm tooth form = Form-ZA, Driving member =Worm

  • Input power = 2 kW

  • Input speed = 1000 rpm

  • Transmission Ratio =20, Worm teeth number=2,Tooth diametral quotient=10

  • Center distance = 200 mm

  • Worm Material = Carbon case hardened steel

  • Worm gear material = Phosphorous Bronze with centrifugally cast

  • Bearing span of 1.0 m with pitch point distance from support = 700 mm

  • Operating Conditions: Driving condition = Light shocks, Driving conditions = Moderate shocks

  • Lubrication and Cooling: Without Fan Splash Oil Lubrication

  1. Start Solid Edge.

  2. Click the File tab→New→ISO Assembly.

  3. Click the Tools tab→Environs group→Engineering Reference→Worm Gear Designer.

  4. Make sure that Design Parameters tab is selected.

  5. Select the Options button to access the Input Conditions window and select Find Torque option from Load Calculation area. Select this option since we have the power and the speed values as input parameters.

  6. From the given inputs it is clear that we need to design the worm gear for given center distance and transmission ratio.

  7. Select the option Find Module from Output Geometrical Parameters area and the option Based on Transmission Ratio from Design Method area.

  8. Select the option Worm from Driver area, as worm is specified as the driving member.

  9. Input the options from Lubrication And Cooling are as per the available parameters.

  10. Click on the OK button to close the Input Conditions window.

  11. Input the option Tooth form of Worm as Form-ZA from the Gearing Parameters area.

  12. Input the transmission ratio in the option provided in the Gearing Parameters area. For the given transmission ratio, the values in the center distance (CD), diametral quotient(q), z1, z2 fields of the Assembly Dimensions area are updated accordingly.

  13. Select the appropriate row from Assembly Dimensions area as per the available input parameters.

  14. Click on the Material button.

  15. Select the material of the worm and worm gear from the Material Values window.

  16. Click on OK button to close the Material Values window.

  17. Input the other parameters like bearing span, pitch point distance through the options available in the Design Parameters area.

  18. Click the Service Factor option from Design Parameters area to open the Service Factor window.

  19. Input the parameters according to the operating conditions.

  20. Click the OK button to close the Service Factor window.

  21. Ensure all the parameters are entered through the options available on the Design Parameters tab.

  22. Click the Calculate button to calculate the results.

  23. Review the results from the Calculated Results tab and Calculated Dimensions tab.

  24. Click the Create button to create a model of worm and worm gear in Solid Edge.

Note:

If the current environment is a Part Environment then a new part is created. Use File, Save As command in Solid Edge to save the file to a different location. If the current environment is Assembly Environment then an instance of the 3D model is added to the assembly and the assembly is saved.

This concludes the tutorial.

In this Tutorial you have learned how to use Worm Gear Module to design and create a 3D model of worm and worm gear for the given inputs.
