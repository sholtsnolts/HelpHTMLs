How to (Sprockets)

  • How to change the number of chain strands?

    The Number of Chain Strands option from the Design Parameters area allows you to set the number of chain strands.

  • How to constrain the center distance between the sprockets for three sprocket assembly?

    Select the option Adjust Center Distance Between from Design Parameters area. The input you have set means you are allowing the change in above selected option, while the others remain unchanged.
