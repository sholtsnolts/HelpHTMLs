Frequently asked questions (Cams)

  • What is the significance of the No of Steps parameter in the Calculated Results tab?

    The No of Steps parameter defines the accuracy for detail calculation and also defines the part accuracy when inserting in Solid Edge. For example, a value of 120 for a segment of 120 degrees indicates that a point is calculated for every 1 degree of the profile.


  • I cannot change the values of the starting and ending angle for segment number 3. Why?

    A cam is defined for a total 360 degrees. The third segment is defined between the end angle of second segment (start angle of third segment) and 360 degrees. Since the end angle of second segment is already given, the third segment requires no user input of start and end angle.


  • How many cam profile types are supported by Cam Module?

    Cam Module supports a total of 12 types for profiles. The Profiles parameter in Segment Settings area on the Calculated Results tab allows the user to access these profiles.


  • I made changes to some parameters in the Design Parameters tab. How to identify between modified and unmodified parameters?

    An underline is placed when a parameter is modified. All underlined parameters indicate that they are modified and a calculation is required to update the results.


  • I created a cam using Solid Edge Engineering Reference. How do I edit the part to change some of the design parameters?

    Editing a part created by Solid Edge Engineering Reference is allowed only in the Assembly environment. Right-clicking the part provides an option to start Engineering Reference Cam Module. You can modify the parameters and click the Update button to update the parameters of the component.
