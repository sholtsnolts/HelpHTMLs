Compression Spring module

The Compression Spring Module allows you to design compression spring and analyze their performance. It allows you to calculate dimensions and perform strength calculations for various types of compression spring. The design information of compression spring is presented in two tabs.

Use the Design Parameters tab to input the parameters, which define design requirements of compression spring. This area also has options you intent to use to set the base parameters such as type of loading, design criteria and the constraints in designing the spring.Use the Material button set the material of the spring.

Use the Calculated Results tab to view the results, which provide information about the behavior of the compression spring. You can also view result of strength calculation of the spring.

To design a spring, input the parameters through the options on the Design Parameters tab. Switch to Calculated Results and click the Calculate button to review the results.

When you are completed with the parameters and results (viewing), click the Create button to generate the 3D Model of Solid Edge compression spring.

Note:

The units of measurement for inputs are based on the units setting in the Solid Edge session. For example, if the Solid Edge session is started with inches units then all the working units of Engineering Reference are in inches only.

You can save the current Inputs and Results by clicking Reports.
