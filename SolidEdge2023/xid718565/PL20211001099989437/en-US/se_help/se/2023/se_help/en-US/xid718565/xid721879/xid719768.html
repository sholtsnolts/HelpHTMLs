Frequently asked questions (Extension Springs)

  • I have the Preload and Applied Load values to input. What options should I choose to create a extension spring for these values?

    From the Input Conditions window, select the options in Design Criteria area. This provides required options in the Design Parameters tab to input the values.

    Note:

    To access the Input Conditions window, on the Design Parameters tab, select the Options button.

  • I have a limit on the Spring Diameter in my design. What options should I choose to create a extension spring with this constraint?

    From the Spring Designer Dialog click the Options button to launch the Design parameters - Input Conditions dialog. Select the appropriate options in the Dimension Limits area.

    Note:

    You may have to first select the appropriate options in the Design Criteria in the Design Parameters - Input Conditions dialog to enable the Dimension Limits options. This enables the Diameter Constraint options in the Spring Designer Dialog where the spring diameter can be set.

  • I have all the geometric details of a spring. Is it possible to find out the load carrying capacity of the spring?

    Yes, It is possible to find the load carrying capacity of the spring. The Design Criteria area in the Input Conditions window provides this feature. Preload and Applied load values in the Calculated Results tab gives an idea about the load carrying capacity of the spring.


  • Will I get a preloaded view or full compressed view when I create a extension spring model?

    You can get both views. Clicking the Create button provides a window containing these options, based on which the corresponding view is generated.


  • I want to design a spring that has an initial tension. What input parameters are required to create a spring with an initial condition?

    The Spring Condition area in the Input Conditions area allows you to select the required option. Based on this option, the Design Parameters tab provides parameters to design a spring with initial tension.


  • I created a extension spring using Solid Edge Engineering Reference. How do I edit the part to change some of the design parameters?

    Editing a part created by Solid Edge Engineering Reference is allowed only in the Assembly environment. Right-clicking the part provides an option to start Engineering Reference Extension Spring Module. You can modify the parameters and click the Update button to update the parameters of the component.
