Column module

The Column module allows you to design and create a model of column. It also allows you to perform a force analysis on the column. The design information of column is presented on 2 tabs:

Use the Design Parameters tab to define the parameters, which give the shape to the column. This area is also used to define the parameters for force analysis on the column. The material of the column can be set using the Material button on the form.

Use the Calculated Results tab to view the strength validation results and other related results, which provide the information about the behavior of the column.

To design column, input the section type and its details through the parameters on the Design Parameters tab. These are the base parameters to design a column. Switch to the Calculated Results tab and click on the Calculate button to review the results.

When you are completed with the parameter results (viewing), click on the Create button to generate the 3D model of Solid Edge column.

Note:

The units of measurement for inputs are based on the units setting in the Solid Edge session. For example, if the Solid Edge session is started with inches units then all the working units of Engineering Reference are in inches only.

You can save the current Inputs and Results by clicking Reports.
