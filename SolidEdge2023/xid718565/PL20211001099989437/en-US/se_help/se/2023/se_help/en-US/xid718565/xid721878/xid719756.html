Frequently asked questions (Compression Springs)

  • I have the Preload and Applied Load values to input. What options should I choose to create a compression spring for these values?

    From the Input Conditions window, select the options in Design Criteria area. This provides required options in the Design Parameters tab to input the values.

    Note:

    To access the Input Conditions window, on the Design Parameters tab, select the Options button.

  • I have a limit on the Spring Diameter in my design. What options should I choose to create a compression spring with this constraint?

    From the Input Conditions window, select the options in Dimension Limits area. This enables Dimension Constraints area in the Design Parameters tab to select the corresponding input option.


  • I have all the geometric details of a spring. Is it possible to find out the load carrying capacity of the spring?

    Yes, It is possible to find the load carrying capacity of the spring. The Design Criteria area in the Input Conditions window provides this feature. Preload and Applied load values in the Calculated Results tab gives an idea about the load carrying capacity of the spring.


  • Will I get a preloaded view or full compressed view when I create a compression spring model?

    You can get both views. Clicking the Create button provides a window containing these options, based on which the corresponding view is generated.


  • I created a compression spring using Solid Edge Engineering Reference. How do I edit the part to change some of the design parameters?

    Editing a part created by Solid Edge Engineering Reference is allowed only in the Assembly environment. Right-clicking the part provides an option to start Engineering Reference Compression Spring Module. You can modify the parameters and click the Update button to update the parameters of the component.
