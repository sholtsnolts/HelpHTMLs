Frequently asked questions

  • When should I use the Axially Rigid option?

    From the theory of shafts it is required that one constraint of the shaft should be axially rigid. By default the shaft is supported on two supports with one of them as axially rigid. Use the Axially Rigid option to make any one of the available supports as axially rigid. You can access the Axially Rigid option from the Supports area on the Design Parameters tab.


  • I did not provide any inputs to any Loads on the shaft. I can see some output graphs being generated. Why?

    The dead weight of the shaft is taken into account for all the calculations in shaft module. So the results and graphs calculated for no input of load correspond to the dead weight of the shaft.


  • What is the use of the Left and Right options in the Shaft Parameters area?

    Some type of sections (Keyway, Ring Grove and Locknut) have slot features available in the topology of the section. These features are located either to the left or right of the section. Use Left and Right options to define the location of these features.


  • When I add and delete sections, the size of the sections appear to be reduced in length and diameter. Why?

    The image in the graphics area is a scaled representation of the original shaft. This mode is chosen to accommodate the entire shaft within the area available to display the shaft. The original dimensions of the sections are not modified during this process.


  • When I edit the Distance parameter of Loads ( or Supports ) area, often I get the warning Distance should be less than or equal to element length?

    The maximum value for Distance parameter is the length of the corresponding section on which load ( or support) is applied. So when you enter a value greater than this value, Shafts Module gives this warning to request you to review this parameter.


  • What is the difference between XY Plane, XZ Plane and Sum options in the Calculated Results ?

    The Shafts Module takes the load input in Y and Z directions. The calculation results for Y direction load are available through XY Plane option and the calculation results for Z direction load are available through XZ Plane option. The quadratic sum total of these results is available through the Sum option.


  • When I click the Calculate button, often I get the warning It is necessary to specify torque using two forces which are mutually opposite?

    This warning is because the summation of all torque values is not zero. This sum must be zero for a valid torque input.


  • I created a shaft using Solid Edge Engineering Reference. How do I edit the part to change some of the design parameters?

    Editing a part created by Solid Edge Engineering Reference is allowed only in the Assembly environment. Right-clicking the part provides an option to start Engineering Reference Shaft Module. You can modify the parameters and click the Update button to update the parameters of the component.
