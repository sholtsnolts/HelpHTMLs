Using the Worm Gear module

This part of the help document provides you with the details on usage of the Worm Gear Module.

The design of the worm gear starts with defining basic input parameters like tooth form of worm, pressure angle and bearing span. Use the Options button to access the Input Conditions window and specify the design methods. Use the Design Method, Output Geometrical Parameters, Drivers and other areas based on available design parameters.

The Design Method area provides you with two options, based on the worm diametral quotient and the transmission ratio, for constraining the worm.

The Output Geometrical Parameters area provides you with two options, based on the module and the center distance, for defining the design procedure.

The Driver area provides you with two options to select the driving member for design calculations.

The Lubrication and Cooling area provides you with two options to allow and set the lubrication and cooling types.

Based on the above selection input the parameters through the options available in Design Parameters area.

When you are completed with specifying the inputs, click on Calculate button to calculate the results. The Calculated Results tab displays the calculation results. A value of Pass for the Strength Validation indicates that the current design is validated.

Use the Create button to create 3D model of worm gear pair.

Use the Reports button to save all selected Options , Input parameters, Calculated Results & Calculated Geometry.
