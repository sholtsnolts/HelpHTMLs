How to (Shaft module)

  • How to know the Section Type for each Section?

    Use the Section Type option in the Shaft Parameters area on the Design Parameters tab to know the section type for each section of the shaft.

  • How to edit the parameters of each section?

    All the inputs required for a section are available below the Section Type option in the Shaft Parameters area on the Design Parameters tab. Use these options to edit the parameters of the selected section.

  • How to identify the current section, current Support and current Load?

    The selected section, support and load are displayed blue in color.

  • How to add and delete sections?

    Use the buttons Add<-- and Add--> in the Shaft Parameters area on the Design Parameters tab to add a section to the left or right of the current selected section. Use Delete button to delete the current selected section.
