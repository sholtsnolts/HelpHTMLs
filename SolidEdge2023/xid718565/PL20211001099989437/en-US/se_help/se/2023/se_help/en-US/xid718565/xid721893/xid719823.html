Synchronous Pulley module

The Synchronous Pulley Module allows you to select the standard synchronous belt and analyze the performance. It allows you to calculate the base dimensions and perform strength calculations for various types of standard synchronous belts.

Use the Design Parameters tab to input the parameters, which define the design requirements of synchronous pulleys. This area also has options you intent to use to set the base parameters such as section, number of pulleys, pulley dimensions and the assembly parameters.

Use the Calculated Results tab to view the results, which provide information about the behavior of the synchronous belt and its pulleys. You can also view result of strength calculation and load carrying capacity of the synchronous belt.

Use the Calculated Dimensions tab to view the results, which define the geometry of the synchronous belt and its pulleys. These parameters are used in the construction of synchronous pulleys' set in Solid Edge.

To design a synchronous pulley, input the parameters through the options on the Design Parameters tab. Switch to Calculated Results tab and click on the Calculate button to view the results.

When you are completed with the parameters and results (viewing), click on the Create button to generate the 3D Model of Solid Edge synchronous pulleys' set.

Note:

The units of measurement for inputs are based on the units setting in the Solid Edge session. For example, if the Solid Edge session is started with inches units then all the working units of Engineering Reference are in inches only.

One can save the current Inputs and Results by clicking Reports.
