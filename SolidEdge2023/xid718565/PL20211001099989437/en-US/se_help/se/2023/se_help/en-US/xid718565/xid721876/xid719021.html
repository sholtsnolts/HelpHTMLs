Cam Module

The Solid Edge Cam Designer allows you to design a cam and create a model of cam. It also allows you to analyze the performance of cams in the form of graphs. The design information of cam is presented on two tabs:

  • Use the Design Parameters tab to define the parameters, which gives the shape and performs strength calculations on cam. The material of the cam can be set using the Material button available on the form.

  • Use the Calculated Results tab to view the cam profile and other related results, which provide information about the behavior of the cam.

To design a cam, input each parameter on the Design Parameters tab. These form the base parameters to design the cam. Switch to Calculated Results to input the parameters related to each segment of the cam. Click the Calculate button to calculate and display the results for each segment.

When you are completed with the parameters and results (viewing), click the Create button to generate the 3D Model of Solid Edge cam model.

Note:

The units of measurement for inputs are based on the units setting in the session of Solid Edge. For example, if the Solid Edge session is started with inches units then all the working units of Engineering Reference are in inches only.

You can save the current Inputs and Results by clicking Reports.
