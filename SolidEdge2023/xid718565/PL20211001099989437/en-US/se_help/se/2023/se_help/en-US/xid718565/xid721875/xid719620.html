Using the bevel gear module

This part of the help document provides you with the details on usage of the Bevel Gear Module.

The design of bevel gear starts with defining the type of gear and method of strength calculation. Use the options button to access the Input Conditions window and specify these details. Use the Load Calculation area in the same window to select the calculation method based on the available design parameters.

Two strength calculation methods are available. The Simple Bending Stress method of strength calculation validates the design for Bending stress only, while the Bending and Contact stress method validates the design for Bending and Pitting stresses with the consideration of several factors given by the standards.

Use the Strength Calculation Type area to specify the calculation type for the design of bevel gear. The Given Material Strength type takes the material selected by the user and calculates the performance of the gear, while the Find Material Strength type assumes a minimum factor of safety and calculates the material to be used.

Based on the above selection, input the parameters through the options available in Gear Parameters area.

When you are completed with specifying the inputs, click on Calculate to calculate the results. The Calculated Results tab displays the calculation results. A value of Pass for the Strength Validation indicates that the current design is validated.

Use the Create button to create 3D model of gear.

Use the Reports button to save all selected Options , Input parameters, Calculated Results & Calculated Geometry.

Note:

Gear models generated by the Solid Edge Engineering Reference tools are suitable for use as representations in machine design and analysis. The manufacture of gears, especially generation of complex tooth profiles and clearances, requires the use specialized machinery, tools and processes. You may want to consult gear manufacturing resources and handbooks before directly using the models generated in Solid Edge for gear production.
