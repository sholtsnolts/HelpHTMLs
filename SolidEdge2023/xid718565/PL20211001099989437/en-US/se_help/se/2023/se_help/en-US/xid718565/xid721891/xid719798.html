Frequently asked questions (Sprockets)

  • Is it possible to select and perform strength check of the sprockets according to the various international standards?

    Yes, it is possible. Select the desired standard from the edge bar for which you would like to carry out strength validation. Available standards are ISO, ANSI, DIN and JIS. Start the Sprockets Module and select chain type and chain number from the Design Parameters area.

    Note:

    The chain types are according to the standard selected while the chain numbers are according to the selected chain type. For the ANSI standard, only the chain numbers option is provided.

  • I have to limit the number of chain links in my sprocket design. What options should I choose to create sprockets with this constraint?

    From the Input Conditions window, select the option Based on Number of Chain Links from the Design Criteria area. This provides the required inputs in the Design Parameters area.

    Note:

    To access the Input Conditions window, on Design Parameters tab, click the Options button.

  • How can I know the failure reasons when the Strength Validation fails?

    The Strength Validation fail indicates that the design calculation with the inputs provided does not comply with the performance. Use the Strength Validation Failure window to review the failure reasons and the corresponding changes should make the strength validation pass.

    Note:

    To access the Strength Validation Failure window, click on the ? button from the Strength Validation area.

  • Is it possible to select and do the strength check of the chains for three sprockets layout with center distances and sprockets teeth as input values?

    Yes, it is possible. Select the option Number of Sprockets from Design Parameters area and set it as three. Access the Input Condition window. Select the option Based on Center Distance from Design Criteria area and the option Based on Sprocket Teeth from Design Method area. This provides the required inputs on the Design Parameters tab.


  • I want to select and create sprockets of ISO Heavy series chain. What should I do?

    Select the Standard as ISO from the input provided on the edge bar. From the Design Parameters area set the option Chain Type as A-Heavy Series. Now you are provided with the variety of the heavy series chains in the option Chain Number on Design Parameters area.


  • I made changes to some parameters in the Design Parameters tab. How to identify between modified and unmodified parameters?

    An underline is placed when a parameter is modified. All underlined parameters indicate that they are modified and a calculation is required to update the results.


  • I created sprockets using Solid Edge Engineering Reference. How do I edit the part to change some of the design parameters?

    Editing a part created by Solid Edge Engineering Reference is allowed only in the Assembly environment. Right-clicking the part provides an option to start Engineering Reference Sprockets Module. You can modify the parameters and click the Update button to update the parameters of the component.
