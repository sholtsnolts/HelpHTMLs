Using the Cam module

This part of the help document provides you with the details on usage of the Cam Module.

The design of cam starts with specifying the Pitch Circle Diameter and Width of the cam. Pitch Circle Diameter is the min distance from the center of the cam to the center of the follower. The Width of the cam specifies the thickness of the cam component. This information is also used in the 3D model construction.

Specify the details of the Follower. These details are Roller Radius and Roller Width. Roller radius is the radius of the Follower part and Roller Width is the thickness of the Follower.

Specify the Eccentricity value, Cycle Time and Cycle Speed.

The above details will give the cam, its overall shape and size.

Use the Cam Loads to specify the Loads acting on the cam.

Switch to Calculated Results tab to define the properties of each segment of the cam profile. Use the Segment Settings area to specify the details of each segment. Use the Calculate button to calculate the results of each segment and move to the next segment.

The Maximum Values area displays the results of calculation for each segment.

Use the Create button to create 3D model of cam.

Use the Reports button to save all selected Options , Input parameters, Calculated Results & Calculated Geometry.
