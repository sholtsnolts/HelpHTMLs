Using the Rack and Pinion module

This part of the help document provides you with the details on usage of the Rack Gear Module.

The design of Rack gear starts with defining the Type of Rack and method of strength calculation. Use the options button to access the Input Conditions window and specify these details.

Rack Span Calculation area lets you chose whether you want to specify number of teeth on the rack or the total rack span instead. Be careful about the difference in number of teeth in gearing & number of teeth on a Circular Rack, which is basically nothing but a sector cut out from a spur gear.

The Bending and Contact stress method is used here, which validates the design for Bending and Pitting stresses with the consideration of several factors given by the standards.

In Options, Checking the Place with Assembly relations Check box will automatically add basic required Assembly Relations in Assembly.

Based on the above selection, input the parameters available in Design Parameters tab.

When you are completed with specifying the inputs, click on Calculate to calculate the results. The Calculated Results tab displays the calculation results. A value of Pass for the Strength Validation indicates that the current design is validated. Be sure to check on the Calculated Dimensions tab for the passed design.

Use the Create button to create 3D model of the gear pair. It draws first the Pinion & then the Rack.

Use the Reports button to save all selected Options , Input parameters, Calculated Results & Calculated Geometry.
