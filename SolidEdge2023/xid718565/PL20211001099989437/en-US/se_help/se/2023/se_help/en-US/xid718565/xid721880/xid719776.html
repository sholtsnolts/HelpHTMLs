Pulley tutorial

Welcome

This tutorial gives step by step instructions to design and create a model of V-Belt Pulleys. No prior experience of using Solid Edge or Engineering Reference is required for this tutorial. However an engineering knowledge on pulley will be useful.

Tutorial Guide

Each step in the tutorial demonstrates the task to be performed with GUI (Graphical User Interface) of Engineering Reference. Some tasks may require interaction with Solid Edge also.

This tutorial should take about 15 minutes to complete.

Introduction

In this tutorial you design and create a model of pulleys' set with the following specifications.

Select and perform the strength check of DIN standard Classical-P20, single number V-Belt for the set of three pulleys for the following input parameters.

  • Input Power = 4.5 kW

  • Transmission Ratio Between:

    • Pulley 1 and Pulley 2 = 2.5

    • Pulley 1 and Pulley 3 = 1.25

  • Speed of motor shaft = 1200 rpm

  • Belt slip = 4%

  • Pitch circle diameter of pulley 1 = 250 mm

  • Center Distance Between:

    • Pulley 1 and Pulley 2 = 550 mm

    • Pulley 1 and Pulley 3= 600 mm

    • Pulley 2 and Pulley 3= 625 mm

  • Operation Conditions: Driving Conditions — Light shocks, Driver shaft conditions — Moderate shock

  1. Start Solid Edge.

  2. Click the File tab→New→ISO Assembly.

  3. Click the Tools tab→Environs group→Engineering Reference→Pulleys Designer.

  4. Make sure that Design Parameters tab is selected.

  5. From the Design Parameters area select Groove Profile as Classical V-Belt and Section as P20.

  6. Since a set of three pulleys and single v-belt is specified, use the Number of Pulleys and Number of Belts options respectively from Design Parameters area.

  7. Select the Options button to access the Input Conditions window. From the given inputs it is clear that we need to design the v-belt for given center distance and pitch circle diameter of pulley.

  8. Select Based on given Center Distance from Design Criteria area and Based on Transmission Ratio from Design Method area.

  9. Select Calculate Torque option from Load Calculation area.

  10. Click the OK button to close the Input Conditions window.

  11. Based on the options selected in the Input Conditions window, the Design Parameters tab provides the parameters to input the data.

  12. Click the Service Factor option from the Design Parameters area to open the Service Factor window.

  13. Input the parameters according to the operating conditions.

  14. Click the OK button to close the Service Factor window.

  15. Click the Calculate button to calculate the results.

  16. Review the results from the Calculated Results tab and Calculated Dimensions tab

  17. Click the Create to create the model of pulleys' set in Solid Edge.

If the current environment is a Part Environment then a new part is created. Use File, Save As command in Solid Edge to save the file to a different location. If the current environment is Assembly Environment then an instance of the 3D model is added to the assembly and the assembly is saved.

This concludes the tutorial.

In this Tutorial you have learned how to use Pulley Module to configure and create a 3D model of pulleys' set for the given inputs.
