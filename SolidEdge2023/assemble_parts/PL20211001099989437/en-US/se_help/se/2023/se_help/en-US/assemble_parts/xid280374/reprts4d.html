Sort dialog box

Sets the sorting rules for the report by column, row, and item.

Sort By

Selects the column you want to sort by and assigns an ascending or descending sort order.

Ascending

Sorts the column in ascending order. The lowest number, beginning of the alphabet, or earliest date will appear first in the list.

Descending

Sorts the column in descending order. The highest number, end of the alphabet, or latest date will appear first in the list.

Then By

Specifies the order in which to sort rows if there are duplicate items in the column you specified.
