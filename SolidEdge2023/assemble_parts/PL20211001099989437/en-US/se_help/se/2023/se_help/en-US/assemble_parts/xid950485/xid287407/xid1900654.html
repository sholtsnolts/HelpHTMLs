Copy and paste assembly components

You can copy components in an assembly and then paste them, within the same assembly or within another assembly.

Copy components

  1. Select the components you want to copy.

  2. Choose Home tab→Clipboard group→Copy .

    Note:

    You can also press Ctrl+C.

Paste components

  1. Open the target assembly into which you want to paste the copied components. This may be the same assembly from which the components are copied.

    If you copy and paste a single component, the workflow is the same as if you had dragged that component from the Parts Library.

  2. Choose Home tab→Clipboard group→Paste .

    You can also press Ctrl+V.

    The components are placed temporarily in the assembly, and the Assemble Multiple Components dialog box is displayed, listing all of the selected components along with their relationships. This dialog guides you through the process of resolving all of the external and learned relationships that had existed between items in the select set and the assembly from which they were copied.

  3. Define new relationships between the copied components and the target assembly.

    As you select a relationship in the Assemble Multiple Components dialog box, the component and the reference element are highlighted in the model. Locate and select component faces or edges within the target assembly to reinstate the original relationship in the target assembly. The Assemble Multiple Components dialog box lists the status of each relationship in the pasted components, so you can track your progress.

    As you define new relationships in the target assembly, use the Paste ribbon bar to activate parts in the target assembly, define placement parts and placement part elements, define target parts and target part elements, and lock or unlock rotation of axial relationships.

    Use External relationships or Learned relationships to speed the process of defining new relatinships in the assembly. After you selected one of the options, uncheck the checkbox available in front of relationships you want to skip.

    You can maintain the internal relationships of the copied components or you can remove them using the Internal relationships checkbox.

    To skip the guidance of the Assemble Multiple Components dialog box, click Manual Assemble to exit the Paste command, leaving the copied components in the target assembly with no new relationships.

    You can use the guidance of the Assemble Multiple Components dialog box to place define some relationships, but then click Finish to exit the Paste command, leaving the copied components in the target assembly with some new relationships.

    In either case, you can then apply new relationships to the copied components using the Assemble command.
