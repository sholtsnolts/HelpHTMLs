Create a single body using Auto-Simplify

To create a single design body from an assembly using the Auto-Simplify command, follow these steps:

  1. Open an assembly.

  2. On the Tools tab→Model group, choose the Simplify Assembly command.

  3. In the Simplify group:

    • Choose the Model command .

    • Choose the Auto-Simplify command .

  4. Select the entire assembly, subassemblies, parts, or a combination of both to create a single design body.

  5. On the Auto-Simplify command bar:

    • Select the Remove Features button to remove all internal features and voids.

    • Click Accept and then click Finish.

  6. In the Close group, click Close and Return.

  7. Right-click the Simplified Assembly in Assembly PathFinder to Edit the simplified assembly or to Save the simplified assembly as a part document.
