Copy Current Display


The Copy Current Display assembly environment command can be accessed through the Explode-Render-Animate environment (Tools tab→Environs group→ERA) or from the Home tab→Configurations group→Copy Current Display.

You use this command to copy the currently highlighted components to the existing display configuration.

For more information about display configurations and their uses, see the Using display configurations and Working with large assemblies efficiently.
