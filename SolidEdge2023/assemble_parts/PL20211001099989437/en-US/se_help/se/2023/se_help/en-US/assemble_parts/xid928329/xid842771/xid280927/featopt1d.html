Feature Options Dialog Box

Specifies which type of assembly-based feature you want to construct. This dialog box is displayed only when you are constructing material removal features such as cutouts, revolved cutouts, and holes.

Note:

To use peer geometry to define assembly features, ensure the proper settings are turned on in the Inter-Part tab of Solid Edge Options.

Create Assembly Features

Specifies that the entire feature resides in the assembly document. With an assembly feature, the reference plane, profile, and surface geometry all reside in the assembly document where the feature was constructed. Assembly features affect only the parts that physically intersect the feature extent.

Create Assembly-Driven Part Features

Specifies that only the reference plane and profile reside in the assembly document. With an assembly-driven part feature, the surface geometry resides in the part documents that are modified by the feature. Assembly-driven part features affect all parts that have the same document name as the parts that physically intersect the feature extent.

Create Part Features

Part features are the same as features created using in-place activation and reside in the part document. Write access in the part document is required to create part features. Part features are created as ordered features.

Create Synchronous Cuts, Holes, Revolved Cuts if possible

When selected, attempts to create a synchronous feature, such as cut, hole, or revolved cut.

Create Synchronous treatments (Rounds and Chamfers) if possible

When selected, attempts to create a synchronous treatment, such as round or chamfer.

Remember the Setting and Do Not Display This Dialog Again During this Session.

Specifies that the dialog box will not be displayed again during the current session when set.
