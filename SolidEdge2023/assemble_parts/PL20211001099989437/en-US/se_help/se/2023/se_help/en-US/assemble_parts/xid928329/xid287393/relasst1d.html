Assembly Relationship Assistant command bar

Defines the sets of parts to which you want to apply relationships.

Options

Accesses the Relationship Assistant Options dialog box.

Select Set One

Defines the first set of parts.

Select Set Two

Defines the second set of parts.

Relationship Assistant Settings

Accesses the Relationship Assistant Settings dialog box.

Finish/Cancel

This button changes function as you move through the assembly relationship assistant process. The Finish button applies the relationships between the sets of parts you defined. The Cancel button discards all input and exits the command.

Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selected components.
