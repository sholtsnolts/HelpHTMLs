Set Parts List properties to default to Assembly item numbers

Administrators can set the Parts List properties default to use Assembly item numbers.

  1. Open a new draft document using the ISO Draft template.

  2. Select Home tab→Drawing Views group→View Wizard.

  3. In the Select Model dialog box, set Files of type to Assembly Document (*.asm).

  4. Select the assembly you want to use and then click Open.

  5. Place the view in the drawing sheet.

  6. Select Home tab→Tables group→Parts List and click a drawing view.

  7. Click the Options tab and select the Use assembly generated item numbers check box.

    Continue setting other options using the pages in the Parts List Properties dialog box.

  8. Click the General tab and type a name for your settings in the Saved settings field.

  9. Click Save and then click OK.

    Your configuration is saved to \Program Files\Siemens\Solid Edge 2023\Template\Reports\DraftList.txt

  10. Click to place parts list.

  11. Save the draft document and upload it to Teamcenter.

  12. Copy DraftList.txt to a Reports folder on the network share.

  13. In Solid Edge Options→File Locations, set the Reports location to the network share defined in the previous step.

  14. Deploy the configuration using Solid Edge Administrator.
