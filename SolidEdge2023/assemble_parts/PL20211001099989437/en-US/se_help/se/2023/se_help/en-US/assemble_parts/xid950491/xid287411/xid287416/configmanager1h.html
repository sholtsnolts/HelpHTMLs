Open Configuration Manager

  • Choose Home tab→Configurations group→Configuration Manager.

  • The components column shows components and their status in different configurations.

  • The Find Previous Component button and the Find Next Component are used to navigate to assembly components that have been recently added to the assembly. New components can be added to existing configurations if desired.

  • The Clean Up Deleted Components button can be used to purge components from configurations that no longer exist in the assembly.
