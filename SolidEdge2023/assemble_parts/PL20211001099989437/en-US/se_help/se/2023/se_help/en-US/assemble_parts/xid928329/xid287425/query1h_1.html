Find a part in an assembly using a quick query

  1. On PathFinder, click the Select Tools tab .

  2. On the Select Tools tab, right-click the Quick Query box to display the shortcut menu.

  3. On the shortcut menu, set the Property option you want.

    For example, you can set the property to Title, Subject, Category, and so on. A check mark is displayed adjacent to the active property.

  4. On the shortcut menu, set the Condition option you want. You can set the condition to Contains, Is (Exactly), or Is Not.

  5. On the shortcut menu, click the Scope button to display the Scope dialog box, then set the scope options you want.

  6. On the Select Tools tab, in the Quick Query box, enter the value property for which you are searching, and then click the Go button .

    The parts that match the query properties you defined are selected in the assembly window.

    Tip:

    • You can use the Scope dialog box to specify whether you want to search subassemblies.

    • Quick queries are not stored on the Select Tools tab.
