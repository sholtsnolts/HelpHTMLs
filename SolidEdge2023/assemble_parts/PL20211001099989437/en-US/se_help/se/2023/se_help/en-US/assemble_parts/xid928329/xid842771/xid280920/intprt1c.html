Inter-Part Copy command

Inserts an associative copy of an existing part into a part or sheet metal document. You can copy individual faces on the part or the entire part. You can only create inter-part copies in a document that has been in-place activated from the assembly. You can use the copied geometry to help you create new geometry that is associative to the copied elements.

For more information, see the Inter-Part Associativity Help topic.

Note:

The Inter-Part Copy command is not available until you set the Inter-Part Copy options you want on the Inter-Part page of the Options dialog box.
