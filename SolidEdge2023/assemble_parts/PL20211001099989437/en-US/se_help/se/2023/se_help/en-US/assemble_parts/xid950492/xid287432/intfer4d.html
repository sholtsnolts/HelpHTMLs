Report page (Interference Options dialog box)

Defines the information you want placed in a report.

File Name

Specifies the name of the report document.

Browse

Displays the Save As dialog box so you can save the report file to a name and location you want.

Include In Report

Selects the information included in a report.

Part Names

Includes the names of the parts in each select set.

Part Centers Of Gravity

Includes the centers of gravity of the parts that interfere with other parts in the assembly.

Interference Center Of Gravity

Includes the center of gravity of the volume of interference in the report.

Interference Volume

Includes the volume of the interference in the report.
