Maintain Relationships Internal to the Select Set

Specifies that the existing relationships between the parts in the select set are maintained.

Delete Relationships Internal to the Select Set

Specifies that the existing relationships between the parts in the select set are deleted. The parts are left unconstrained.

Delete Relationships Internal to the Select Set and Ground Parts

Specifies that the existing relationships between the parts in the select set are deleted and the parts are grounded.
