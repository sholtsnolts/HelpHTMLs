Display or hide specific reference elements in an assembly

To display or hide specific elements in an assembly, right-click a component and select the Show/Hide Component command from the shortcut menu.

Use the As Last Saved command to display the selected part as it was last saved in the part document. To use As Last Saved in the Show/Hide Component dialog box, point to the command and then select or clear the check boxes to turn them on and off.

Use these procedures to control the display of the following reference elements for parts in an assembly.

On the Show/Hide Component dialog box, point to

All Elements

All Types, and then select or clear the On and Off check boxes.

Turning All Types on or off does not affect As Last Saved or Design Body.

Reference planes

Reference Planes, and then select or clear the On and Off check boxes.

Sketches

Sketches, and then select or clear the On and Off check boxes.

Coordinates systems

Coordinate Systems, and then select or clear the On and Off check boxes.

Reference axes

Reference axes, and then select or clear the On and Off check boxes.

Reference axes are available only in the ordered environment.

Surfaces

Surfaces, and then select or clear the On and Off check boxes.

Curves

Curves, and then select or clear the On and Off check boxes.

Center lines

Center Lines, and then select or clear the On and Off check boxes.

Center lines are available for tubes, pipes and frames, and can be dimensioned. Center lines can be displayed on drawing sheets in the draft environment.

Live sections

Live Sections, and then select or clear the On and Off check boxes.

PMI dimensions

PMI Dimensions, and then select or clear the On and Off check boxes.

PMI annotations

PMI Annotations, and then select or clear the On and Off check boxes.

Design bodies

Design Body, and then select or clear the On and Off check boxes.

Turning All Types on or off does not affect Design Body.
