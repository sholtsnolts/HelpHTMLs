Item number status dialog box (Assembly)

Provides a report of the assemblies queried by the Item Number Status command, and gives you the opportunity to synchronize item numbers.

Queried assemblies

Displays the number of Solid Edge assemblies evaluated.

Out of date assemblies

Displays the number of assemblies that are out of date and require synchronization.

View List

Displays the report (.csv) generated by the Item Number Status command. The reports contains a list of assemblies that are candidates for synchronization.

Note:

The report's default location, C:\Users\<username>\AppData\Roaming\Siemens\Solid Edge\Version 223\Log Files, is defined on the File Locations page of the Solid Edge Options dialog box.

Check-out assemblies

When selected, all candidate assemblies are checked out for access.

Do you want to synchronize these changes?

Determines whether or not the item numbers are synchronized when you open the assembly in Solid Edge.

Yes

Processes item number synchronization.

No

Continues and does not process item number synchronization.
