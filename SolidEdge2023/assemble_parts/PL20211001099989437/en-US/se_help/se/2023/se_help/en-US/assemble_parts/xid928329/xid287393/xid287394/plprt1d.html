Create Part In-Place Options dialog box

Save To

Specifies the storage location for the new document.

Active assembly location

Saves the new part document to the folder where the active assembly is located.

Last saved location

Saves the new part document to the last saved location.

Ground Components

Specifies whether the newly created part is grounded or not.

Place the Origin

Specifies the origin of the new document.

Coincident with assembly origin

Places the new part or subassembly at the origin, or center, of the assembly.

Offset from assembly origin

Specifies that the new part or subassembly will be offset from the assembly origin. You can offset the new part or subassembly by selecting a keypoint on an existing part or by typing an offset value. The following are available when you select this option.

By graphic input

Positions the new part or subassembly using a face and an edge on an existing part.

Create component and stay in Assembly

Creates the part document only and remains in the assembly environment

In SEEC, the part file is created and has a status of Checked In.

Create component then edit in-place

Creates the new document and opens it in in-place edit mode, so you can continue creating new features within the context of the assembly.

In SEEC, the part file is created and opened for edit. The file has a status of Checked Out.

Remember my selection and do not show this dialog again

Stores the settings of the dialog box and does not display the dialog box again.
