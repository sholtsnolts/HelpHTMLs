Inserting components from 3Dfind.it

Use the Insert Components from 3Dfind.it command to browse the 3Dfind.it website and select a component to insert into the active assembly.

In unmanaged Solid Edge, when you select a model from 3Dfind.it, the selected part is downloaded to the default location for the 3DFind.it models as defined in the Solid Edge Options dialog box, on the File Locations page. If the value is undefined, you are prompted to browse for a folder.

In the Teamcenter-managed environment, when you choose to insert a model from 3Dfind.it, the system first checks to see if the part is already in Teamcenter. If the part already exists in Teamcenter, the document from Teamcenter is used. If the part does not exist in Teamcenter, the part is downloaded from 3Dfind.it to the default location, and inserted into your open assembly. The part is saved to Teamcenter when you upload the assembly to Teamcenter.

Performing a Save As on a part inserted into an assembly from 3Dfind.it saves the copy of the part to Teamcenter as a new part that is not associated with 3Dfind.it. Be sure to apply status to a dataset from 3Dfind.it by putting the status on your Item Revision in Teamcenter.

Note:

To enable 3Dfind.it functionality in the Teamcenter-managed environment, set the Teamcenter preference SEEC_Enable_3DFindit value to 1.

For a full description of the preference, see the Teamcenter Preferences section of the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.
