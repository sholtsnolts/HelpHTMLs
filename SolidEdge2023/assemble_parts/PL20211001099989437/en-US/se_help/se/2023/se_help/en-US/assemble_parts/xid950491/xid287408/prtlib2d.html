Teamcenter Parts Library tab

Displays the contents of the parts within Teamcenter, or your local cache when you are working offline. You can drag and drop managed parts into an assembly.

Note:

You can use the commands on the shortcut menu to control which types of files are listed, whether you want to maintain relationships when placing parts, and so on.

Standard Parts

Accesses the Solid Edge Standard Parts dialog box, so you can choose from a library of parts for your assembly.

View Detail

Determines the type of representation used in the display list when you work with Teamcenter-managed documents. Changing the value updates the view in the display list.

  • Full—Displays the Item, Item Revision, and Dataset in individuals rows.

  • Intermediate—Displays the Item and Item Revision property data in a single row and any dataset information is expanded underneath.

  • Basic—Default display. The Item, Item Revision, and Dataset are displayed on a single line.

Revision Filter

Control the revisions of an item you see in the display list. The default value is set to Latest.

  • All—Displays all document revisions.

  • Latest—Displays only the latest revision of the document.

  • Released—Displays only the documents with a status of released.

  • Latest Released—Displays only the most recent revision of the document.

  • Latest 3—Displays the three most recent revisions of the document.

Up One Level

Moves the folder list up one level.

Search

Displays the Search dialog box so you can define advanced search criteria for the document you want to open.

Item ID search

Quickly searches for the Item ID of the document you want to use.

Folder List

Displays a list of Teamcenter-managed folders which contain your objects. Clicking the arrow displays additional folders.

Document List

Displays all documents in the selected folder. You can use shortcut commands to control how the list of documents is displayed.

Preview Area

Displays an image of the selected Item.
