Using the FlashFit command

The following procedures describe how to use the FlashFit command (Home tab→Assemble group→FlashFit) to apply relationships between parts. For more information about FlashFit, see About the FlashFit relationship.

The FlashFit workflow can speed up the process of defining relationships when creating assemblies. For more information about part positioning workflows, see Understanding Assembly relationships.

By default the FlashFit command is automatically selected to create new relationships when a part is dragged into the assembly workspace from the Parts Library. You can modify this default behavior using the Assemble command bar Options dialog box located on the Assemble command bar.

Insert a new part

  1. To place a new part into the assembly, drag a placement part from the Parts Library into the assembly workspace.

    The system opens the Assemble command and the associated Assemble command bar, and highlights the placement part. For more information, see Assemble parts (workflow).

  2. If the FlashFit icon is not displayed in the Relationship Types box, on the Assemble command bar, use the drop-down list in the Relationship Types box to select it now.

  3. Select the offset type and enter the appropriate values for the offset that is being defined (Fixed, Float, or Range.

    For more information about offset types, see Modify the offset value for a relationship.

  4. Select the feature on the placement part that is to be used to form a relationship.

    For more information about the FlashFit features that can be used to form a relationship, see About the FlashFit relationship.

  5. Select the appropriate feature on the target part that corresponds to the feature selected on the placement part.

    The relationship definition is complete. The command bar remains open to apply the next relationship.

  6. To apply another relationship, continue with Create additional relationships.

  7. To close the command and the command bar, press the Esc key.

Additional FlashFit relationships can be formed immediately after forming a relationship or at any later time in the assembly process.

Note:

Use the Flip option on the Assemble command bar to flip the relationship. For more information, see Using the Flip command.

Create additional relationships

  1. To create an additional FlashFit relationship between two parts, the Assemble command bar must be open. If not already open, select the Assemble command (Home tab→Assemble group→Assemble) to open the Assemble command bar.

  2. Repeat the procedure described in Insert a new part.
