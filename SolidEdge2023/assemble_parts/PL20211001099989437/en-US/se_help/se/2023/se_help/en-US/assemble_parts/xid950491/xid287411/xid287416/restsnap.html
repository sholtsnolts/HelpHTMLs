Restore Snapshot


The Restore Snapshot assembly environment command can be accessed through the Explode-Render-Animate environment (Tools tab→Environs group→ERA) or from the Home tab→Configurations group→Copy Current Display.

You use this command to restore a previously saved Snapshot of show/hide display settings. See the Take Snapshot command for more information about preserving display settings created using the Show command and Hide command on the PathFinder to manage what is visible in the current display setting.

For more information about display configurations and their uses, see the Using display configurations and Working with large assemblies efficiently.
