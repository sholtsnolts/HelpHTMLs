Using the Rigid Set command bar

The Rigid Set command bar options define a Rigid Set relationship between parts or subassemblies in an assembly. For more information, see About the Rigid Set relationship.

The Rigid Set command bar has the following features:

Shared relationships options

Select the appropriate option to determine how previously created shared relationships are to be managed:

Suppress

Select this option to preserve the shared relationships but to show them as suppressed in the PathFinder.

Delete

Select this option to delete the shared relationships.

Ignore

Select this option to preserve the shared relationships in the PathFinder as an active relationships.

Accept ✓ or Deselect X

Select Accept (✓), right-click, or press Enter to form the relationship between the highlighted parts and subassemblies. Select Deselect (X) to clear the highlight of all highlighted parts.
