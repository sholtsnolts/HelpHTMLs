Save a simplified assembly as a part

  1. In PathFinder, click the Simplified Assembly entry.

  2. On the File menu, choose Save As, and then choose Save Simplified Model As .

  3. In the Save Simplified Assembly As dialog box, select an output file type.

  4. Type a file name.
