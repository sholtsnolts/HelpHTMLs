Sheet Metal Sensor command bar

Main Steps
Face Selection Step

Allows you to specify the faces you want to include in the sheet metal sensor.

Edge Selection Step

Allows you to specify the sets of edges you want to include in the sheet metal sensor. When this step is active, you can define the Edge Sets you want, or you can select one of the predefined sensor examples.

Finish

Finishes the face and edge selection steps and displays the Sheet Metal Sensor Parameters dialog box so you can finish defining the sensor.

Face Selection Step Options
Selection

Specifies whether you want to select individual faces or the entire part.

Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selected faces.

Edge Selection Step Options
Edge Set 1

Displays a list of edge types from which you can select. The edge sets you specify are compared against each other to determine whether the sensor parameters are being exceeded.

Edge Set 2

Displays a list of edge types from which you can select. The edge sets you specify are compared against each other to determine whether the sensor parameters are being exceeded.

Predefined Sensor

Displays a list of predefined sets of edges from which you can select. When you select one of the predefined sets of edges, the Edge Set 1 and Edge Set 2 boxes are automatically filled with the proper edge types.
