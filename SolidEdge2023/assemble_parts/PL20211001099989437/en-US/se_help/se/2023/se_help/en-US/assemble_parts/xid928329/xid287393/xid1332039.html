Clone Component command Options dialog box

Assembly Relationships

The Clone Component Options dialog box is located on the Clone Component command bar. It sets the behavior for the creation of the assembly relationships using the Clone Component command .

Recreate relationships with target components

This option tries to recreate relationships applied on the parent component between a clone occurrence and the target components. Faces on a parent component with assembly relationships are mapped to a clone occurrence, and then relationships are created between the clone and the target. It works best if the geometry on the target is similar. When the geometry differs, there may be offset values applied to the assembly relationships.

Create ground relationships

All relationships are removed on the clone occurrence and a ground relationship is applied.

Do not create relationships

No relationships to the cloned components are created.

Options

Group cloned component

Checked

Cloned components are organized in Assembly PathFinder as an assembly group when placed.

Unchecked

Cloned components are not organized in Assembly PathFinder as an assembly group when placed.
