Move command bar (Assembly environment)

Selection

Use the select tool to add or remove assembly components from the select set.

Move Copy

Sets the criteria to either move or copy assembly components

Select Related Faces

Selects synchronous component faces that are related to components in the select set.

Move Subassembly Components

Enables direct movement of occurrences within subassemblies.

Relationship Options

Sets the criteria for relationships that are altered due to the component being moved. For more information, see Assembly Relationship Options dialog.

Connected Faces

Specifies how the face connected to the selection adapts during the modification.

For more information, see Modifying models using the steering wheel and 2D steering wheel.

Precedence

Determines the behavior of the selected faces versus the other model faces.

For more information, see Move command bar.
