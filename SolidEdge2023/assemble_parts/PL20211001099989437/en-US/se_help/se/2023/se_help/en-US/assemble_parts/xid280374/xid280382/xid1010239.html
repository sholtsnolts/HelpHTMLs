Item number status dialog box (Draft)

Provides a report of the assemblies queried by the Item Number Status command, and gives you the opportunity to view the report generated by the command.

Queried assemblies

Displays the number of Solid Edge assemblies evaluated.

Out of date assemblies

Displays the number of assemblies that are out of date and require synchronization.

View List

Displays the report (.csv) generated by the Item Number Status command. The report contains a list of assemblies that are candidates for synchronization.

Note:

The report's default location, C:\Users\<username>\AppData\Roaming\Siemens\Solid Edge\Version 223\Log Files, is defined on the File Locations page of the Solid Edge Options dialog box.

OK

Dismisses the dialog box.

Note:

There is no option to synchronize item numbers from the draft environment. To synchronize Solid Edge Item Numbers with Teamcenter Find Numbers, save the assembly to Teamcenter.
