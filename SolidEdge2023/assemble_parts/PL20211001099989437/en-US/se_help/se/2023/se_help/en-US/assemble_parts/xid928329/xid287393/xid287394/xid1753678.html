Create Part in Place command bar

Options

Displays the Create Part In-Place Options dialog box so that you can select options for creating the part in place.

Templates

Specifies the template to be used for the new document. You can click the Browse button to select a different template.

Browse for Template

Displays the New dialog box so that you can define the document template you want to use.

Material

Specifies the material of the new document.

Ground

Specifies whether the newly created part is grounded or not.

Origin

Specifies the origin of the new document.

Coincident with assembly origin

Places the new part or subassembly at the origin, or center, of the assembly.

Offset from assembly origin

Specifies that the new part or subassembly will be offset from the assembly origin. You can offset the new part or subassembly by selecting a keypoint on an existing part or by typing an offset value. The following are available when you select this option.

X:

X coordinate value based on the active coordinate system.

Y:

Y coordinate value based on the active coordinate system.

Z:

Z coordinate value based on the active coordinate system.

By graphic input

Positions the new part or subassembly using a face and an edge on an existing part.

Keypoints

Sets the type of keypoint you can select to define a feature extent or to position a new reference plane. This allows you to define the feature extent or the location of the reference plane using a keypoint on other existing geometry. The available keypoint options are specific to the command and workflow you use.

Sets the center and end points options.

Selects any keypoint.

Selects an end point.

Selects the center point of a circle or arc.

Selects a midpoint.

Selects a silhouette point.

Selects an edit point on a curve.
Edit in Place

Creates the new document and opens it in in-place edit mode, so you can continue creating new features within the context of the assembly.

Accept (check mark)

Accepts the selection.

Cancel

Clears the selection.
