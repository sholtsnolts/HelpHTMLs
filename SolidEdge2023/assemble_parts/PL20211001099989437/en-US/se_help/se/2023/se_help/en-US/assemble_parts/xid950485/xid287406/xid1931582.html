Replace a part with a part from 3Dfind.it

  1. Choose Home tab→Modify group→Replace Part→Replace Part with 3Dfind.it Part .

    The command automatically starts the search using the currently active or selected document. If the currently active or selected model originated from 3Dfind.it, the replace functionality opens the same part in 3Dfind.it so that you can select from variations of the same part.

  2. Select the part you want to replace and right-click to accept.

  3. From the 3Dfind.it website, search for and select the replacement part. Then click Send to Solid Edge.

    When you select a model from 3Dfind.it, the selected part is downloaded to the default location for the 3DFind.it models as defined in the Solid Edge Options dialog box, on the File Locations page. If the value is undefined, you are prompted to browse for a folder.

Tip:

  • In the Teamcenter-managed environment, when you choose to insert a model from 3Dfind.it, the system first checks to see if the part is already in Teamcenter. If the part already exists in Teamcenter, the document from Teamcenter is used. If the part does not exist in Teamcenter, the part is downloaded from 3Dfind.it to the default location, and inserted into your open assembly. The part is saved to Teamcenter when you upload the assembly to Teamcenter.

  • To enable 3Dfind.it functionality in the Teamcenter-managed environment, set the Teamcenter preference SEEC_Enable_3DFindit value to 1.

    For a full description of the preference, see the Teamcenter Preferences section of the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.
