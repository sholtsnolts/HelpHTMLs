Update Simplified Assembly command

Updates a previously simplified assembly. When you make design changes to a part in an assembly that has been simplified, you must use this command to update the simplified representation of the assembly before the changes are displayed in any documents that reference the simplified version of the assembly, such as higher level assemblies or drawings.
