Inter-Part Links dialog box

Show all links

Displays all inter-part links.

Show only broken links

Displays only the broken links when set. This can make it easier to diagnose and repair assemblies with broken inter-part links.

Show only frozen links

Displays only the frozen links when set.

Show only XpresRoute links

Displays only the XpresRoute links when set.

Note:

This option is only available in the XpresRoute environment.

Sort by Children

Lists the child Inter-Part relationships.

Sort By Parents

Lists the parent Inter-Part relationships.
