Create Simplified Assembly Model command bar

Main Steps
Create Solid Geometry

The simplified assembly model consists of ordered features within the assembly. Commands such as Extrude, Cut and Revolve are available to model the simplified representation of the assembly.

The Enclosure command will enclose selected assembly components inside a volume.

Close and Return

After the simplified representation is created, the Close and Return command finishes the step.

Note:

The Simplified Assembly displays in PathFinder and can be deleted from when the Tools→Mode→Design button is selected.
