Save a display configuration

  1. Click the PathFinder tab.

  2. In PathFinder, use the commands on the shortcut menu to arrange the display of the assembly.

  3. On the Home tab, in the Configurations group, click the arrow at the bottom-right of the Configurations group to open the Display Configurations dialog box.

  4. In the Display Configurations dialog box, type the name you want to use for the configuration.

  5. Click Save to save the display configuration.

Tip:

  • You should avoid using special characters in configuration names. For example, special characters such as \ / : ! are not allowed.

  • You can also use the Show and Hide commands to show or hide an assembly component.

  • You can apply a saved display configuration later by selecting it from the Configuration list on the Home tab in the Configurations group.

  • You can save a display configuration of an exploded assembly. The exploded configuration name can be used later to create an exploded assembly drawing.
