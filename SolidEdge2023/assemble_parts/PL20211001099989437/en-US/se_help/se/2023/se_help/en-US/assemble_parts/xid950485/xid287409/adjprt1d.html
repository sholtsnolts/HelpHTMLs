Adjustable Part dialog box

Variable Table

Displays the Variable Table for the part or assembly so you can add a variable to the adjustable part parameters list. Click the variable in the Variable Table that you want to add.

Assembly Variable

The Assembly Variable in the variable table is used to control the length of the adjustable variable in the part placed in the assembly. The value of this variable controls the adjustable dimension on the part.

Measure Distance

Runs the Measure Distance command so you can define a measurement variable. When you define a measurement variable it is added to the adjustable part list. This option is available only in the Assembly environment.

Measure Minimum Distance

Runs the Measure Minimum Distance command so you can define a measurement variable. This option is available only in the Assembly environment.

Measure Normal Distance

Runs the Measure Normal Distance command so you can define a measurement variable. This option is available only in the Assembly environment.

Measure Angle

Runs the Measure Angle command so you can define a measurement variable. This option is available only in the Assembly environment.

List Area

Displays the adjustable parameters in a spreadsheet format.

Part Variable

Displays the name of the part variable.

Notes

Displays notes that provide more information about the variable. You can type helpful information in the cell.

Assembly Variable

Displays the name of the assembly variable.

Adjust like a spring

This option uses the variable created by measuring a distance to adjust the length of the corresponding variable in the part or sheet metal document. The position of the parts attached to the adjustable part determines the length of the variable defining the distance.

Adjust to fit and allow assembly relationships

This option uses the variable created by the measurement to change the length of the adjustable part and reposition parts within the assembly that are not constrained. The length of the variable defining the adjustable part length is used to position the unconstrained parts connected to the adjustable part.

Shortcut Menu Options

Shortcut menu options are available that allow you to rename and delete adjustable part variables. You can also change the font displayed in the dialog box.
