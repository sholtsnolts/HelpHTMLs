Create a Match Coordinate Systems as a single relation

  1. Insert a part into an assembly.

  2. On the Assemble command bar, set the Relationship Types to Match Coordinate Systems, and set the Single Coordinate System Constraint option .

  3. Select the source coordinate system.

  4. Select the target coordinate system.

    The coordinate systems are matched.
