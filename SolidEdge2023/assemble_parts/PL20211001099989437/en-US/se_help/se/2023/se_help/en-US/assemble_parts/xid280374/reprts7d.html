Report Output dialog box

Previews the report you created. You can also print the report, save it to a document name you specify, copy it to the Clipboard, or create another report.

Note:

The actual report output dialog box name is based on the report type you select and the assembly document name.

Save As

Displays the Save As dialog box so you can save the report to the storage location, document name, and output type (.txt or .rtf) that you want.

Print

Displays the Print dialog box so you can print the report.

Copy

Copies the report to the Clipboard in text format with tabs between the columns. The report can then be copied from the Clipboard to a spreadsheet document, text document or drawing document.

New Report

Closes the report output dialog box and returns to the Assembly Reports dialog box for the type of report being created.
