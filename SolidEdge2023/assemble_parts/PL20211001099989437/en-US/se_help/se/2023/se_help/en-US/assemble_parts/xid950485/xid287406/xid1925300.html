Replacing a part with a part from 3Dfind.it

In some cases, you may need to replace a part in an assembly with a part from a supplier catalog. Use the Replace Part from 3Dfind.it Part command to replace a part in your active assembly with a supplier part you select from 3Dfind.it.

The command automatically starts the search using the currently active or selected document. If the currently active or selected model originated from 3Dfind.it, the replace functionality opens the same part in 3Dfind.it so that you can select from variations of the same part.

In the Teamcenter-managed environment, replacing an existing part with a new part creates an entry in Teamcenter for the new part because it is the first time the replacement part is inserted into the assembly. Be sure to apply status a dataset from 3Dfind.it by putting the status on your Item Revision in Teamcenter.

Note:

To enable 3Dfind.it functionality in the Teamcenter-managed environment, set the Teamcenter preference SEEC_Enable_3DFindit value to 1.

For a full description of the preference, see the Teamcenter Preferences section of the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.
