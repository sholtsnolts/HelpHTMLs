Customize a gusset

You can use the Gusset Options dialog box to customize gussets by defining such parameters as the gusset profile dimensions, thickness, and thickness direction.

You can use the Save Settings option to save the different gusset parameter values.

Customize a gusset

  1. Specify the profile dimensions for the gusset.

    The first face you select is d1 and the second face you select is d2.

    When placing the gusset on planar faces, press the "N" key to flip the gusset along d1 and d2.

    When placing the gusset between a cylindrical face and a planar face, press the "T" key to flip the gusset to the opposite side of the cylindrical face.

  2. Specify the gusset thickness direction. The direction can be left, right, or centered.

  3. Specify the gusset thickness.
