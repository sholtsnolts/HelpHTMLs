Create virtual components

  1. Home tab→Assemble group→Component Structure Editor .

  2. In the Virtual Components Structure Editor dialog box, specify the Model Type you want.

  3. In the Name box, type the name you want, or select a name from the Name list.

  4. Click the Add Virtual Component button or press Enter.

Tip:

  • As a best practice, you should choose between using Solid Edge Virtual Components and a workflow where Teamcenter empty items are created in a PDM client. Using both in a structure is not recommended.

  • If you need several occurrences of the same virtual component, press Enter repeatedly.

  • Use Promote Component and Demote Component to reorganize the assembly structure.

  • Use Delete to delete a virtual component.
