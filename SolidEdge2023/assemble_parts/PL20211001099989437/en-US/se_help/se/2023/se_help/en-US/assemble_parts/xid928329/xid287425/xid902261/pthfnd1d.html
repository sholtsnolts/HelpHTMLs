Assembly Relationships Selection command bar

Faces Style

Lists and applies the available faces styles.

Assembly Configuration

Lists and applies the available assembly configurations. Assembly configurations are used to manage the display of the parts and subassemblies in the active assembly.

Floating Offset

Sets the offset type to floating. With a floating offset you can use another assembly relationship to define the offset value.

Fixed Offset

Sets the offset type to fixed. With a fixed offset you can enter a value in the Offset Value box. You can edit this value later.

Offset Value

Sets the distance the part is offset from the adjacent part in the assembly. This option is available when you are using a fixed offset. You can enter a positive or negative number.
