Change the component type of a virtual component

  1. Choose Home tab→Assemble group→Virtual list→Component Structure Editor . The Virtual Components Structure Editor dialog box is displayed.

  2. On the Virtual Components Structure Editor dialog box, position the cursor over the virtual component which you want to change the component type.

  3. Right-click to display the shortcut menu, then click Change Component Type.

  4. On the Change Component Type dialog box, select the new component type, then click OK.

  5. On the Virtual Components Structure Editor dialog box, click OK.
