Create Zone command

The Create Zone command creates a named zone to make it easier to select and control the display of a set of assembly components that occupy a defined volume of space.

After you create a zone, you can show and hide the zone boundary boxes using the Zones command on the View menu.

For more information, see the Using Zones in an assembly Help topic.
