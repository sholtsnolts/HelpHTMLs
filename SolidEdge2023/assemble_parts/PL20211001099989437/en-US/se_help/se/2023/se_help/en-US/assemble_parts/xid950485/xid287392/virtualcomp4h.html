Position a Virtual Component on a Sketch

You can only position a virtual component when a sketch window is open.

  1. In Assembly PathFinder, select an instance component or a predefined component that has not been positioned on the assembly sketch.

  2. On the shortcut menu, click Position Virtual Component.

  3. Do one of the following:

    • If you selected a predefined component, on the Select Component Sketch dialog box, select the component sketch or reference plane orientation you want.

    • If you selected an instance component, go to Step 4.

  4. In the sketch window, position the virtual component graphics and then click.

Tip:

  • You can press the A key one or more times to rotate the virtual component 45 degrees counterclockwise. You can press the S key one or more times to rotate the virtual component 45 degrees clockwise.

  • You can place a virtual component into the sketch by dragging it from the Assembly PathFinder tab and dropping it into the sketch window.

  • You can also use the Place Component Copy command to copy and position an instance component or a predefined component. A new entry for the virtual component is added to Assembly PathFinder and the Virtual Components Structure Editor.
