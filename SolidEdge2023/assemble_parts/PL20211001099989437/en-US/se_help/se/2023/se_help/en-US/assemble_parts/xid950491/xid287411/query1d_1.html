Select Tools page

You can use the Select Tools page on PathFinder to find a part in an assembly using queries and zones.

Quick Query

Allows you to enter text for a quick query. You can use the commands on the shortcut menu to define the query scope and criteria. Quick queries are not stored on the Select Tools page. The Scope command on the shortcut menu displays the Query Scope dialog box.

Go

Starts the quick query.

New Query

Displays the Query dialog box so you can define the query criteria you want.

Create Zone

Runs the Create Zone command so you can define a zone.

Update Assembly Structure

Loads the entire assembly structure into memory to ensure the assembly structure is up to date. Updating the assembly structure ensures that existing zones capture the latest assembly component information.

If the assembly structure is out of date, the zone contents may not be accurate.

The assembly structure can become out of date when multiple people are working on different subassemblies, and you then open the top-level assembly with some of those changed subassemblies hidden. For example, on the Open File dialog box, there are options that allow you to open an assembly using a display configuration, a zone, or with all components hidden.

Note:

Updating the assembly structure can take several minutes, depending on the size of the assembly.

Zone shortcut menu commands

The following shortcut menu commands are available when you select a named zone entry in the Select Tools tab.

Delete

Deletes the selected zone.

Rename

Allows you to rename the selected zone.

Cut

Cuts the selected zone and adds it to the Windows clipboard.

Copy

Copies the selected zone and adds it to the Windows clipboard.

Show Components

Displays the assembly components associated with the zone.

Show Only

Displays the assembly components associated with the zone, and hides all the other assembly components.

Hide Components

Hides the assembly components associated with the zone.

Select Components

Selects the assembly components associated with the zone.

Show Zone Box

Displays the zone boundary box in the graphic window.

Hide Zone Box

Hides the zone boundary box in the graphic window.

Edit Definition

Displays the Create Zone command bar so you can edit the zone definition.
