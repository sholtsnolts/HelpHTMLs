Query Dialog Box (Assembly)

Finds parts in an assembly using the search criteria you define.

Query Name

Defines a name for the query.

Query Scope

Specifies the scope for the query. For example, you can specify that the query scope includes all the parts in the assembly, only parts that are currently displayed, and so forth.

Search Subassemblies

Specifies that you also want to search subassemblies within the active assembly.

Load Subassemblies

Specifies that you want to load and search any hidden subassemblies that were unloaded from memory using the Unload Hidden Parts command.

Query Criteria

Allows you to specify the query criteria you want.

Property

Lists the part properties you can search against.

Condition

Defines the conditional argument for the property being searched. For example, you can define a query where the Condition option is set to Contains, and the Property option is set to Material, and the Value is set to Steel. All parts whose material property in the part document contains the word "Steel" would be selected.

Value

Defines the value to search for in the specified property. You can type any text you want to search for. You can search for multiple values in one operation by separating the values with a semicolon (;). For example, you could search for parts with a material property of either steel, brass or copper by typing: Steel;Brass;Copper.

Add To List

Adds the specified search criteria to the Find Items That Match These Criteria list.

Find Items That Match These Criteria

Displays the criteria to be used in the search.

Remove

Removes selected entries from the Find Items That Match These Criteria list.
