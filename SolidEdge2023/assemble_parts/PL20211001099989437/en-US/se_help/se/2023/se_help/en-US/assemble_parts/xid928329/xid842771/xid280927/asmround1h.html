Construct a round in an assembly

Note:

You cannot use the Round command to construct assembly-driven part features until you set the Assembly-Driven Part Features option on the Inter-Part tab of the Solid Edge Options dialog box.

  1. Choose Features tab→Assembly Features group→Round .

  2. In the Feature Options dialog box, specify whether you want to construct an assembly feature, an assembly-driven part feature or a part feature.

  3. Select the edges of active parts to which to apply rounds.

    Note:

    Inactive part edges are not selectable.

  4. Accept the selection.

  5. Finish the feature.

    Tip:

    • Assembly driven features must be enabled on the Inter-part tab of the Solid Edge Options dialog box.

    • Activate the parts to which you want to apply rounds. Edges cannot be located on inactive parts.

    • Variable radius rounds are not allowed for assembly rounds.

    • Soften corner is not allowed for assembly rounds.

    • Assembly driven rounds become ordered features on the solid body in the selected component.
