Define Alternate Components Dialog Box

Allows you to define or edit the list of alternate components.

Alternate Candidates (Left Pane)

Lists the documents that are candidates for the alternate components group you are defining or editing. You can use the Search and Browse buttons to define the components you want for the Alternate Candidates list.

Alternates List (Right Pane)

Lists the documents that are part of the alternate components group you are defining or editing. To add a candidate document to the Alternates list, select the candidate in the left pane, then click the Add button to add it to the Alternates List.

Add

Adds the selected documents to the Alternates List.

Remove

Removes the selected documents from the Alternates List.

Remove All

Removes all the documents from the Alternates List.

Search

Displays the Search dialog box so, you can define the search criteria you want.

Browse

Displays the Replacement Part dialog box so, you can select documents you want to add to the Alternate Candidates List.
