Create an Inter-Part Copy

  1. Choose Home tab→Clipboard group→Part Copy list→Inter-Part Copy.

  2. Select the part in the assembly which you want to associatively copy.

  3. Select the faces you want to copy.

  4. On the Inter-Part Copy command bar, click the Accept (check mark) button.

  5. On the Inter-Part Copy command bar, click the Finish button.

    Tip:

    • You can use the Inter-Part Copy command to associatively copy faces and parts only when you have in-place activated a document from an assembly and when you have set the Inter-Part Copy option on the Inter-Part tab on the Options dialog box in the assembly.

    • You can use the Inter-Part Copy command bar to specify whether you want to copy individual faces or the entire part.

    • After you have associatively copied faces from a part in the assembly, you can use them to help you construct a new feature. For example, you can then use the Include command to include edges from the copied faces to help you construct a new profile for a feature.
