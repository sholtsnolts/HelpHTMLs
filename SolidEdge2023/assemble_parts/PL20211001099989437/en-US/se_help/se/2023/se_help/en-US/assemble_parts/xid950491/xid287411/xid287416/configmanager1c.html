Configuration Manager command

Configuration Manager is used to manage changes in configurations. With the Configuration Manager command , when parts are added to or deleted from subassemblies, the configurations in the top-level assembly can easily find the changes in the subassemblies and allow the current configurations to incorporate these changes.

Configuration Manager is useful when using large assemblies, nested assemblies, or assemblies being shared by several designers.

For more information, see the Using display configurations and Working with large assemblies efficiently Help topics.

Note:

Exploded Configurations can only be edited in the ERA environment.
