Managing components in assemblies

There are several tools available for managing components in an assembly.

The primary tool used for managing the display status of assembly components is the PathFinder tab.

You can create display configurations of an assembly, which allows you to recall the show/hide status of the components in an assembly quickly.

You can use the Create Zone command and the related zone options on the Select Tools tab to define display zones in an assembly. You can then quickly display and hide all the parts that fall within the zone area.
