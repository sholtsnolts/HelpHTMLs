Edit Definition command bar

Main Steps
Select Step

Allows you to select the 2-D sketch elements that you want to associate with the virtual component.

Orientation Step

Defines the origin and x-axis orientation of the virtual component.

Finish/Cancel

This button changes function as you move through the definition process. After you define the virtual component graphics, you can edit it by selecting the appropriate step on the command bar gain. The Cancel button discards any input and exits the command.

Select Step Options
Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selection.

Orientation Step Options
Free Space Origin

Defines a nonassociative origin that can be at any location.

Define Origin

Sets the origin and x-axis of the component from two element keypoints.

Publish On

Specifies the sketch plane on which the virtual component will be published. You can select one of the three base reference planes from the list.
