Assembly Statistics dialog box

Displays the assembly statistics for the top-level assembly.

You can right-click an occupied row in the grid to display a shortcut menu to copy information to the Windows Clipboard, change the font, sort, and format columns.

You can double-click near a column divider to fit the columns to their contents. Any changes you make to the columns are stored for future sessions.

Shortcut menu commands

The following additional shortcut menu commands are available. The shortcut menu commands that are available vary based on what is selected.

Cut

Cuts the current selection and pastes it to the Clipboard. This option is not available with this command.

Copy

Copies the current selection to the Clipboard.

Paste

Inserts the Clipboard contents at the specified insertion point. This option is not available with this command.

Columns

Displays the Columns dialog box so you set the column display and order options.

Font

Displays the Font dialog box so you can select the font, font style, and font size you want.

Search

Displays the Find dialog box so you can find text strings within the dialog box. For example, you can search for a part or subassembly.

Sort

Allows you to sort the list in ascending or descending order.

Show Document Name and Location

Displays the full path name and document name when set. When cleared, only the document name is displayed.
