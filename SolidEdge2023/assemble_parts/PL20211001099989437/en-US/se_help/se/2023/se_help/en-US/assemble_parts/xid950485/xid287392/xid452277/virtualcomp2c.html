Publish Virtual Components command

Creates the part, sheet metal, and subassembly documents that virtual components represent. When you publish a virtual component, the virtual component entry in PathFinder is replaced with the corresponding new document and any graphics assigned to the source component are copied to the new document.

In the Teamcenter-managed environment, you can publish individual components by dragging them from PathFinder on to the Solid Edge graphical window, or you can choose between a full publish and a partial publish by using of the check boxes corresponding to the component(s) in the publish Virtual Components dialog box.

Note:

When a virtual assembly is selected for publishing, all dependent virtual real components are selected as dependent references.

For more information on working with virtual components, see the Creating and Publishing Virtual Components help topic.
