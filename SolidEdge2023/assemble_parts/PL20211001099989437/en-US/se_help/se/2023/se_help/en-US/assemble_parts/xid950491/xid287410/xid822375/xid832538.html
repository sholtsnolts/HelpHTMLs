Component Tracker dialog box

Update All

Updates all files and links for the assembly from the active level down. This command does not process missing files or trigger a download to the cache. If there are no out of date documents in the tree, this button is disabled.

Save All

Saves all modified files from the active level down.

Expand All

Shows the entire assembly in a linked document tree.

Note:

This is not a PathFinder tree.

Model status

Displays the status of all components of the current document. Status is displayed in a tree, with views listed under the files to which they are associative. The following symbols can appear in the Component Tracker:

Model out-of-date

Subassembly part has been modified

File not found

Model up-to-date
