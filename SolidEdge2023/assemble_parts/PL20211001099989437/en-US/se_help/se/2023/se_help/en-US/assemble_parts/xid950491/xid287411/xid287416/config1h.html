Apply a saved display configuration

  1. Choose Home tab→Configurations group→Display Configurations.

  2. On the Display Configurations dialog box, select the name of the configuration you want to apply to the assembly.

  3. Click Apply to apply the display configuration.

Tip:

  • You can also apply a display configuration using the Configuration drop list on the Home tab, in the Configurations group.

  • To apply a display configuration that was defined in a subassembly of the current assembly, select the subassembly in PathFinder first. You can then select the subassembly display configuration you want using the Configuration drop list on the Home tab, or the Display Configurations dialog box.

  • You cannot use the display configuration names you save of an exploded assembly view in a regular assembly view. To display an exploded configuration, you must be working in the Explode-Render-Animation environment.
