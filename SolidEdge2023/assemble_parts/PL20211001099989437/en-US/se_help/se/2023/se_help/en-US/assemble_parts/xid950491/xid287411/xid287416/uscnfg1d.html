Configuration

Lists the available display configuration names. This option is available when placing or opening an assembly document (.ASM). A display configuration stores the show/hide status of the parts contained in the assembly. An assembly that has had parts hidden will place or open faster.

Note:

The display configuration takes precedence over the capability to Hide All Components. Therefore, the Hide All Components command is only valid when the Configuration is set to No selection.

Apply Activation Override

Allows you to specify how you want to open or place an assembly (.ASM). When this option is set, you can specify whether the parts in the assembly are active or inactive. When this option is cleared, the active or inactive status of the parts is determined by the last save. When you open an assembly with parts inactive, the assembly will open faster.

Activate All Parts

Specifies that all parts in the assembly become active when you open the assembly.

Inactivate All Parts

Specifies that all parts in the assembly become inactive when you open the assembly.
