Large Assembly Performance

Use the Large Assembly Performance mode for the best performance when working with large assemblies.

The Home→Modes→Large Assembly Performance command sets Large Assembly Performance mode to On. This mode increases the interactive performance of displayed assemblies. When this mode is on, display settings, such as Floor Reflection and Auto-Sharpen, are turned off and disabled.

Large Assembly Or Performance Mode - A mode in which several application and display settings have been tuned to provide improved performance with large assemblies.

On the Solid Edge Options→Assembly Open As page, use the Large Assembly Performance mode threshold setting for the total number of components to automatically apply/override various user and document settings that will improve performance.

Preferences and settings that are overridden for large assemblies are not editable in the UI and will not be saved back to registry, preferences or document.

View Overrides

In large assembly mode, the following View settings are turned “Off” and disabled:

  • Floor reflections

  • High Quality

  • Cast Shadows

  • Floor Shadows

  • Ambient Shadows

  • Silhouettes

  • Depth Fading

The View Style command does not change listed settings.

The View Overrides dialog controls indicate the mode that is applied, although they are disabled.

Solid Edge Options

In large assembly mode, the following Display settings are turned “Off” and disabled:

Settings→Options→View Page

  • Display drop shadows during view operations = Off (Default is Off but prevent user from modifying)

  • Process hidden edges during view operations = Off

  • View Transitions = Off

  • Auto-sharpen = Off (Default is Off but prevent user from modifying)

  • Glow Set to 0

  • Use shading on highlight = Off (use wireframe)

  • Use shading on selection = Off (use wireframe)

Settings→Options→Assembly Page

  • Dim surrounding components when a selection is made in relationship pathfinder = Off

  • Inactivate hidden and unused components every (X) minutes = On

  • Show dynamic movement of component during FlashFit* = Off

Global Display Settings

  • Disable Floor Shadows and Floor Reflection .

  • Sharpness set 1 .

Reference Plane Display

Independent of large assembly mode, during dynamics, the reference plane text is turned off.

Note:

This applies to all visible reference planes whether in top assembly, sub-assemblies or parts.

Coordinate System Display

Independent of large assembly mode, hide the coordinate system axis labels during dynamics.

Sketch Display

Turn on Accelerated Display for all sketches in the assembly. This setting displays sketches in a single color and solid line type. This applies to all visible sketches whether in top assemblies, sub-assemblies or parts.
