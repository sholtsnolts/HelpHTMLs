Create Systems Library Documents dialog box

Dialog Box Options
Systems Library Document

Specifies the name of the new systems library document.

Feature Part Documents

Lists the documents that contain features.

New File Location

Specifies the storage location for the new systems library document. You can click the Browse button to specify a storage location in a folder on your computer or a computer in your network.

Browse

Displays the Browse For Folder dialog box so you can specify a storage location for the new document.

Template

Specifies the template to be used for the new document. Click the Browse button to select the template you want to use.

Browse

Displays the New dialog box so that you can define the document template you want to use.
