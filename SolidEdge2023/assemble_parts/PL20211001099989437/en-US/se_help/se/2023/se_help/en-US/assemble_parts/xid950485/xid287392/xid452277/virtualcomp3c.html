Edit Definition Command (Virtual Components)

Assigns 2D sketch geometry to a virtual component. You can use the options on the command bar to specify that you want to use a free space point as the origin or two keypoints to define the origin and x-axis.

For more information on working with virtual components, see the Creating and Publishing Virtual Components Help topic.
