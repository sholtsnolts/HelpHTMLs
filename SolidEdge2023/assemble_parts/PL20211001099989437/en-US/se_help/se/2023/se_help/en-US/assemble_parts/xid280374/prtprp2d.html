Format Columns dialog box (Occurrence Properties)

Column display and order

Lists the current display setting and display order for the columns in the Occurrence Properties dialog box. You can use the check box adjacent to each column name to specify whether a column is displayed in the dialog box. You can select a column and then click the Up or Down buttons to reorder the column within the dialog box.

Up

Moves the column up in the Column display and order list. As you move a column up in the list, it is moved farther to the left in the Occurrence Properties dialog box.

Down

Moves the column down in the Column display and order list. As you move a column down in the list, it is moved farther to the right in the Occurrence Properties dialog box.

Width

Specifies the column width. To specify the width for a column, select it in the Column display and order list, and then set Width options you want to use for that column. Any changes you make to the columns are stored for future sessions.

You can also define the column width by dragging the edge of a column heading to resize it.

Specific width

Enters a specific width for a column.

Best fit

Defines the width of the column based on the largest cell in the column.

Alignment

Defines the text alignment for the column. To specify the alignment for a column, select it in the Column Display and Order list, then set the alignment option you want to use for that column.

Saved column settings

Saves column order and display changes for reuse in other assemblies. When creating a new saved setting, type a name in the box before clicking the Save button. To make changes to an existing saved setting, first select its name from the list.

Save

Writes the settings to a text file, OccurrencePropertyColumns.xml, in the Solid Edge Preferences folder. All saved settings are stored as global preferences in this file. If you enter a name that is not unique, the system prompts you to overwrite the previously saved setting.

Delete

For the currently selected name in the Saved column settings box, deletes the corresponding information from the OccurrencePropertyColumns.xml file.
