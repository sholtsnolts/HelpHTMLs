Transfer command

Transfers the selected part(s) to a new or existing assembly. You can only select parts within a single assembly for transfer. If you select parts from more than one assembly, the command becomes disabled.

The command is not available when you in-place activate into a subassembly.

For Solid Edge data management users who selected the option Automatically name files via Document Number and Revision, the system displays a dialog box where you can assign common properties, such as document number and revision number, as explained in Create documents with unique properties.

Note:

If you transfer a part that occurs more than once in the source assembly, there could be a reduction in the part count for the assembly. Likewise, if you transfer a part to a subassembly that occurs more than once, there could be an increase in part count for the assembly.
