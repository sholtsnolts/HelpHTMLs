Check for interference between parts in an assembly

  1. Choose Inspect tab→Evaluate group→Check Interference.

  2. In PathFinder or the graphics window, select the parts you want in Select Set 1, and then click the Accept button.

  3. In PathFinder or the graphics window, select the parts you want in Select Set 2, and then click the Accept button.

  4. On the command bar, click Process.

    Note:

    For Solid Edge data management users who selected the option Automatically name files via Document Number and Revision, the system displays a dialog box where you can assign common properties, such as document number and revision number, as explained in Create documents with unique properties.

    If the system displays a warning message about a probable interference, it indicates that the boolean operation that detects interference and generates the interference part failed. Click the Options button and use the Highlight interfering parts option on the Interference Options dialog box to highlight the interfering parts.

Tip:

  • If no interferences are detected, a message box is displayed to let you know.

  • If interferences are detected, the volume of interference is displayed in the highlight color.

  • You can set output options using the Interference Options button on the command bar before you run the analysis.

  • You can check one set of parts against all of the other parts in the assembly by setting options on the Interference Options dialog box.

  • You can also check a set of parts against only those parts that are in the select set.
