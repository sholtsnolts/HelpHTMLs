Manage Relationships dialog box

Create new group

When clicked, creates a new group.

Edit group

When clicked, edits the selected group relationship.

Set default group

When clicked, makes the selected group the default group.

Delete group(s)/relationship(s)

When clicked, deletes the selected group(s) or relationship(s).

Relationships area

Defines relationship information for the part or subassembly.

Relationships

Defines the relationship type for the part or subassembly.

Note:

The relationships must be the same for the part or subassembly you are placing, and the target part in the assembly. Be sure to add them in the same order, with the same group name.

Polarity

Specifies the polarity for the relationship.

Note:

You can set the polarity type to N pole or S pole in the part environment.

Tip:

You can change the polarity type from Magnetic to Capture Fit and vice versa in the part environment, as well as in the assembly environment.

If you set the relationship group to Magnetic during create and decide to use it as a Capture Fit in the assembly environment, you can change the mode to Capture Fit in the assembly environment.
