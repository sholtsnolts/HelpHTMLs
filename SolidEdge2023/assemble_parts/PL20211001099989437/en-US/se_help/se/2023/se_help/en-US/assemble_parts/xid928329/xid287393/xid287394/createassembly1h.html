Create an assembly with the Create Assembly command

  1. Save the current model (assembly, part, or sheet metal) file.

  2. From the File menu, click New→Assembly of Active Model.

  3. In the Create Assembly dialog box, click OK.

    Note:

    If you want the part to remain open, deselect the Close Current Part check box.

    The assembly is created with the current file and specified template.
