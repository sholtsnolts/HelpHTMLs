Place Component Copy Command

Copies and positions an instance component or a predefined component. You can select the component in Assembly PathFinder or the sketch window while you are working in an assembly sketch.

When copying a predefined component, you can use the Select Component Sketch dialog box to specify a component sketch or reference plane orientation you want to use in the assembly sketch.

For more information on working with virtual components, see the Creating and Publishing Virtual Components Help topic.
