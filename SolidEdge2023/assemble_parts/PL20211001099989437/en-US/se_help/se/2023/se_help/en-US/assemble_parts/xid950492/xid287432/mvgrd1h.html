Move a part in an assembly

  1. Choose Home tab→Modify group→Drag Component .

  2. On the Analyze Options dialog box, set the options you want.

  3. Select the part you want to move.

  4. On the command bar, select a movement option.

  5. In the application window, do one of the following:

    • Position the cursor over the x, y, or z axis and drag the part to a new location.

    • Click the x, y, or z axis and then type a distance or angle value in the appropriate command bar box.

Tip:

  • You can use the command bar buttons to move the part along an axis, rotate the part around an axis, or move the part freely in 3D space.

  • You can move the part incrementally by typing the distance or angle value you want on the command bar, then press the Enter key repeatedly. You can then use the Back and Forward buttons on the command bar to return to each incremental position.

  • You can use the Options button on the command bar to dynamically detect collisions between the part you are moving and another set of parts you define.

Note:

In order for the Drag Component command to work, all broken assembly relationships must be repaired or deleted.
