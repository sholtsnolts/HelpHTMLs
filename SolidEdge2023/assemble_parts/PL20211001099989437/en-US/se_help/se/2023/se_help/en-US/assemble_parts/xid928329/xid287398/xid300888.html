Position two parts by matching their coordinate systems

The following procedures describe how to use the Match Coordinate Systems command (Home tab→Assemble group→Match Coordinate Systems) to match the coordinate system of the placed part with the coordinate system of a part already in the assembly. For more information, see About the Match Coordinate Systems relationship.

Insert a new part

  1. To place a new part into the assembly, drag a placement part from the Parts Library into the assembly workspace.

    The system automatically opens the Assemble command and the associated Assemble command bar, and highlights the placement part. For more information, see Assemble parts (workflow).

  2. Select the Match Coordinate Systems option from the Relationship Types list on the Assemble command bar .

    For more information, see Using the Match Coordinate Systems command bar.

  3. Select the first part that is to be used to form the relationship.

  4. On the first part, select the desired coordinate system from the available coordinate systems associated with the part.

  5. Select the second part that is to be used to form the relationship.

  6. On the second part, select the desired coordinate system from the available coordinate systems associated with the part.

  7. On the command bar, select the type of offset to apply (Fixed, Float, or Range), and supply the appropriate values.

    For more information, see Using the Match Coordinate Systems command bar.

  8. To complete the action, press OK.

    The Assemble command bar remains open to apply the next relationship.

  9. Do one of the following:

    • To apply another relationship, continue with Create additional relationships.

    • To close the Assemble command and the Assemble command bar, press the Esc key.

Create additional relationships

  1. To create additional Match Coordinate Systems relationships between two parts, you can do either of the following:

    • To create a relationship using the Assemble command bar, the Assemble command bar must be open. If not open, select the Assemble command (Home tab→Assemble group→Assemble). On the Assemble command bar, select Match Coordinate Systems from the Relationship Types list.

    • To create a relationship using the Match Coordinate Systems command bar, the Match Coordinate Systems command bar must be open. If not open, select the Match Coordinate Systems command (Home tab→Assemble group→Match Coordinate Systems ).

    The actions that can be performed are the same for each option. However, after the Match Coordinate Systems command bar is opened, only a Match Coordinate Systems relationship can be applied. The relationship type cannot be changed (see Using the Match Coordinate Systems command bar).

  2. Select the relationship parts and features as described previously.

  3. To complete the action, press OK.

  4. To close the command and the command bar, press the Esc key.
