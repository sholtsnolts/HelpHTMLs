Replace a part in an assembly

  1. Choose Home tab→Modify group→Replace Part .

  2. Select the part you want to replace.

  3. On the Replacement Part dialog box, select the replacement part and then click OK.

Tip:

  • You can select the part you want to replace in the graphics window or in PathFinder.

  • If the part you are replacing occurs more than once in your active assembly, you can replace just the selected occurrence or all occurrences of the selected part.

  • In the Teamcenter-managed environment, you should use the Revisions command to replace an item with a different revision of the same item.

  • In some cases it may be necessary to delete assembly relationships and reapply them to position the new part properly.

  • When replacing dissimilar parts, the original part and the replacement part are required to be in the same relative orientation in their respective part files.

  • Do not use the Replace command to update a link when the folder and/or the file name has changed.  Instead, use the Change Source button on the Edit Links dialog box.

  • When replacing a subassembly, if the current or replacement subassembly is a family of assemblies, you can use the Assembly Member dialog box to specify the assembly member you want.

  • If the part you are replacing is a member of a family of parts, you can use the Family of Parts Member dialog box to specify a new member as the replacement part.
