Create a one body assembly

To create a single design body from an assembly, follow these steps:

  1. On the Home tab→Clipboard group, choose the Part Copy command .

  2. On the Select Part Copy dialog box, select the assembly document to create a single body from, and then click Open.

  3. On the Part Copy command bar, select the Parameters Step.

  4. On the Part Copy Parameters dialog box, select the Configuration list.

    When a Solid Edge assembly is used in Part Copy, display configurations in the Assembly document are listed on the Part Copy dialog box. Selecting a display configuration copies only the bodies in that configuration. If no configuration is selected, the last configuration used is the default.

  5. On the Part Copy Parameters dialog box, select the Copy as Design body option.

    When an assembly document is used, the Merge solid bodies option is turned on by default. The check box control for Adjust non-manifold condition and merge option is off by default. When it is checked and there are multiple bodies created by the Part Copy feature, Solid Edge will adjust such bodies to remove the non-manifold conditions and merge them into a single design body.

  6. On the Part Copy Parameters dialog box, select the Remove voids option. This removes all voids from all the bodies created in the Part Copy.

  7. On the Part Copy command bar, click Finish.

    The one body assembly is listed in PathFinder, under Features, as a Part Copy.
