Close an in-place activated document

  • Choose Home tab→Close group→Close and Return .

    Note:

    The changes you made to the in-place activated document are saved to memory, but not to disk.

Tip:

  • When you use the Close and Return command, you are not asked to save the part document to the hard disk until you exit the Assembly environment.

  • After you return to the Assembly environment you can use the Save command to save the changes you made to the hard disk.

  • You can use the Revert command to close the document and return to the assembly document without saving your changes.
