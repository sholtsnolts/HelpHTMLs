Convert an STL document to a part document

An STL (STereoLithography) file contains the triangles required for creating a terrain model.

  1. Choose the File menu→Open tab→Open command .

  2. In the Open File dialog box, from the Files of type list, select STL documents (.stl).

  3. Click the Options button, and ensure the Make base feature option is selected.

  4. Browse for the STL document, and then click Open.

  5. Select a part template with the appropriate length readout units.

  6. Fit the view.

  7. Save the part file.

Tip:

  • Turn off the display of the reference planes and coordinate systems prior to fitting the view.

  • The length values are set in File menu→Info→File Properties→Units tab. You can create templates that contain these units.

Note:

The imported design body can only exist in the ordered environment. When attempting to import the data into a synchronous environment, a warning message is displayed, and the environment automatically changes to the ordered environment.
