Set the adjustable/rigid status of a subassembly

To specify that a subassembly is considered adjustable:

  • In PathFinder, select a subassembly, then on the shortcut menu, choose Adjustable Assembly.

To specify that a subassembly is considered rigid:

  • In PathFinder, select a subassembly, then on the shortcut menu, choose Rigid Assembly.

Tip:

  • You must have write access to the subassembly to use the adjustable assembly functionality.

  • You can also specify that a subassembly is adjustable using the Options dialog box on the Assemble command bar.

  • When you specify that a subassembly is adjustable, you cannot in-place activate the subassembly using the Edit command.
