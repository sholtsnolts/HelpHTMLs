Select Directory dialog box

Defines the paths that you want to search for assemblies that contain the selected part. This dialog box when you create a Where Used report.

Look In

Defines the drive and folder where you want to start looking. You can use the Browse button to view the folders you want to look in. The system stores the last location you browsed in the Look In box.

Note:

If you are running Windows NT and want to browse a network share, you must map the share to a network drive. After you map the network drive, you can select it from the Look In pulldown list.

Add

Adds selected folders and documents to the Look In list.

Remove

Removes the selected folder from the Look In list.

Remove All

Removes all entries in the Look In list.
