Copy Definition Command

Copies the virtual component name and type to the Name box and Component Type list in the right pane of the Virtual Component Structure Editor dialog box. You can click the Add Virtual Component button or press the ENTER key to quickly add another copy of the virtual component to the virtual assembly structure.

You can also use this command to copy the predefined component name to the Name box to add another copy of the predefined component to the virtual assembly structure.

For more information on working with virtual components, see the Creating and Publishing Virtual Components Help topic.
