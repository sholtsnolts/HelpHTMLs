Scroll To Source Command

Scrolls to the source virtual component and selects it. The Scroll to Source command is available when you select an instance of a virtual component, which is a copy of the source component.

When working with a complex assembly sketch, this command can make it easier to find the source component for editing purposes. You can select any instance component in the graphic window or PathFinder, and the source component is highlighted in PathFinder and the graphic window.
