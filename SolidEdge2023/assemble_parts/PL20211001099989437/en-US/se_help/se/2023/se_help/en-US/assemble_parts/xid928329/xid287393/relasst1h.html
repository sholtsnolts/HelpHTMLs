Add assembly relationships to parts in an assembly automatically

  1. Choose Home tab→Relate group→Assembly Relationship Assistant .

  2. On the command bar, click the Relationship Assistant Options button and define the select set option you want, and whether you want to remove existing relationships, then click OK.

  3. In PathFinder or the graphics window, select the parts and subassemblies you want in Select Set 1, then click the Accept button.

  4. Do one of the following:

    • If you set the Select Set 2 option on the Relationship Assistant Options dialog box, in PathFinder or the graphics window, select the parts and subassemblies you want in Select Set 2, then click the Accept button.

    • If you set one of the other options, the Relationship Assistant Settings dialog box is displayed.

  5. In the Relationship Assistant Settings dialog box, specify the relationship types you want to allow, then click the Process button.

    Note:

    When you click the Process button, a list of parts and relationships is displayed in the Relationship Assistant Settings dialog box. You can select parts or relationships in the list, and they highlight in the graphic window.

  6. In the Relationship Assistant Settings dialog box, deselect any parts or relationships you do not want to apply, then click the Accept button to accept the relationships, or the Reset button to clear all the relationships.

    Tip:

    Because applying an assembly relationship involves two parts, each relationship is displayed in the list twice. If you deselect a relationship in one location, it is deselected in both locations. If you deselect a part, all the relationships applied to that part are deselected.
