Steering Wheel Move in an assembly

  1. Using the select tool, select the assembly component to move or copy. Add or remove additional components to the select set by CTRL+LMB click.

  2. Click the copy button to copy rather than move the components.

    Selecting related faces finds synchronous faces that are related to the components in the select set. The move operation acts upon these faces and the Design Intent panel will control the behavior. Copy is not valid if faces are selected.

  3. To select a subassembly component, click the move subassembly component button. Otherwise the full subassembly is moved or copied.

  4. Drag the steering wheel to move or copy. Relationships between selected components are preserved. You can delete or suppress the relationships that become invalid because of the move.

  5. You can remove the relationships, repair them, or reconnect them to other components. For more information, see Steering Wheel Move command (Assembly environment).
