Create a virtual component sketch for a document

  1. Choose Home tab→Sketch group→Component .

  2. Select a reference plane or a planar face.

  3. Do one or both of the following:

    • Use the Component Image command to create a 2D image of the part on the profile plane.

    • Use the element creation commands to create precision graphics on the profile plane. For example, you can draw lines, arcs, and circles, or include elements.

  4. Click Return to close the profile view.

Tip:

  • The graphics you create with the Component Image command cannot be selected when you place the predefined component in an assembly sketch. They can only be used for viewing purposes.

  • The precision graphics you create using the element creation commands can be selected when you place the predefined component in an assembly sketch. They can be used to position the predefined component within the assembly sketch. For example, you can use dimensions and relationships to position the component sketch.

  • If you modify the component sketch graphics for a predefined component, you can use the Update Component command in PathFinder shortcut menu to update the component sketch graphics in the assembly sketch.
