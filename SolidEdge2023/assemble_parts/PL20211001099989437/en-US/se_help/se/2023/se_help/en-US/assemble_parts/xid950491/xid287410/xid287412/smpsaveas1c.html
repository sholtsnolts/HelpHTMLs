Save Simplified As command

Saves the simplified version of the assembly as a part. You can use the Save Simplified Assembly As dialog box to specific the file type you want to use. Saving the simplified representation as a separate document can make it easier to share the document with users that do not require the entire assembly, such as vendors.

It can also reduce the data management requirements when using the assembly as a subassembly in another assembly. You can place the part document in the higher level assembly instead of the entire assembly. This can be useful when working with large assemblies.

Note:

When you are working in a Teamcenter-managed environment, the file will automatically be saved as a part (.par). The Save as Simplified command does not create any links between the simplified part and the source assembly.
