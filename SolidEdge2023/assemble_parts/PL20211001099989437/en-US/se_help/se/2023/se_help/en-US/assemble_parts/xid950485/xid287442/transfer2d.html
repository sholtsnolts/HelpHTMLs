Transfer to Assembly Level Dialog Box

Specifies assemblies for transferring files.

Transfer the Selected Item(s) to the Assembly:

Displays a tree view of the subassemblies contained in the assembly. Click the subassembly to which you want to transfer the parts. You can also transfer the selected parts to a new subassembly. If you click an assembly name, the New Subassembly button becomes available, allowing you to create a new subassembly for the files you want to transfer.

New Subassembly

Accesses the Create New Subassembly dialog box that allows you to create a new subassembly for the files you want to transfer.
