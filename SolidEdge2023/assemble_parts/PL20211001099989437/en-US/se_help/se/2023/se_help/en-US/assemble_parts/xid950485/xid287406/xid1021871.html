Repair Missing Files command

The Repair Missing Files command located under the Inspect tab→Evaluate group, opens the Repair Missing Files dialog box, which contains commands to locate or replace missing files. If files are missing, the command automatically executes on Open.

The following commands are available on the Repair Missing Files dialog box:

  • Search in —Use to browse and define folder paths to locate files.

  • Scroll to—Use to scroll to a part in PathFinder.

The following commands are available on the Repair Missing Files dialog box and on the assembly Home tab→Modify group menu:

  • Replace Part

  • Replace with Standard Parts

  • Replace with New part

Note:

Missing files can only be repaired in the main Assembly environment.
