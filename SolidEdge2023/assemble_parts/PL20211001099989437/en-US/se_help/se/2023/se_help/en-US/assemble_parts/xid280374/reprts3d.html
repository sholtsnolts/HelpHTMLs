Options dialog box (Assembly Reports)

Specifies the column content you want to show in the report. You can add or remove columns and control the order in which they appear.

Available properties

Specifies the properties—column data—that you want to show in your report. Each property you select appears as a column in the report. For example, if you are creating a parts list report, you may want to add these properties:

  • Item Number—Displays the item numbers based upon the current settings for item number generation in the assembly document. Item number generation is controlled by:

    • The Use assembly generated item numbers check box (Assembly Reports dialog box)

    • The Maintain item numbers check box (Item Numbers page, Solid Edge Options dialog box)

  • Mass (Item)—Calculates the mass of a relatively unique element based on the density of the material, as defined in the Material Table.

  • Mass (Quantity)—Calculates the sum of the mass of all similar elements.

If you are creating a report for a structural frame assembly, you also may want to add these properties:

  • Miter Cut 1—Calculates the acute angle value of the first end of a frame, pipe, or tube, as it was created. Its value is always between 0 and 90 degrees.

  • Miter Cut 2—Calculates the acute angle value of the second end of a frame, pipe, or tube, as it was created. Its value is always between 0 and 90 degrees.

If you are creating a report for an assembly containing adjustable (flexible) tubing and hoses, add this property:

  • Tube Flat Length—Displays the flattened length of adjustable hoses or tubes when they are stretched along their path.

    Note:

    Tube flat length is equivalent to the tube cut length in the Bend Table dialog box in XpresRoute.

Add

Adds the selected property to the report.

Remove

Removes the selected property from the report.

Show these column headings

Lists the column headings that will be displayed in the report in the order they will appear. You can use the Move Up and Move Down buttons to arrange the column headings in the order you want.

Move Up

Moves the selected column heading up one position in the list.

Move Down

Moves the selected column heading down one position in the list.

Fit each column

Sets the column width equal to the text in the heading.
