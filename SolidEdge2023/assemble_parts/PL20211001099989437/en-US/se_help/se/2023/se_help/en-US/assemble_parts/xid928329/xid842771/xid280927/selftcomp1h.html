Select the parent elements of an assembly feature

  1. Click the PathFinder tab.

  2. On PathFinder, select an assembly feature.

  3. On the shortcut menu, click Select Feature Components.

    Tip:

    After you select the parent components of a feature, you can use the Show, Show Only, and Hide commands to change the display of the components.
