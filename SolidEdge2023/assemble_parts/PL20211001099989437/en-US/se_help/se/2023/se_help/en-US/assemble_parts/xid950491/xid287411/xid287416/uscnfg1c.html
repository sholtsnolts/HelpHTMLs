Use Configurations and Open As Settings command

Displays the Use Configuration and Open As Settings dialog box so you can select a display configuration when placing a subassembly into an assembly.

Note:

The display configuration takes precedence over the capability to Hide All Components. Therefore, the Hide All Components command is only valid when the Configuration is set to No selection.
