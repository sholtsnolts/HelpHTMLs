Place a Systems Library document

  1. Display the Parts Library tab.

  2. On the Parts Library tab, drag a systems library assembly document and drop it in the assembly window.

  3. In the assembly window, define the relationships required to position the systems library components.

    The Place Systems Library command bar and dialog boxes guide you through this process.

  4. In the assembly window, define the part and face required to position the feature you captured.

  5. On the command bar, click OK.

Tip:

When placing systems library members, you can use the Skip button on the command bar to bypass any relationships you want to apply later.
