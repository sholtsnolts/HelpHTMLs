Assign Capture Fit command

Use the Assign Capture Fit command to predefine relationships for parts in the part environment or subassemblies in the assembly environment that will be included in an assembly.

When creating predefined relationships, observe the following for both the placement and target part or subassembly:

  • Define the relationships.

  • Make sure relationship group names are the same.

  • Add relationships in the same order.

For more information, see Create predefined relationships on a part or subassembly.

Workflow for predefining relationships

  1. Select the relationship type.

  2. Specify the element on which to apply the relationship.

  3. Specify the group name.

  4. Set the polarity for the relationship.

  5. Save the part.

Adding parts or subassemblies to an assembly

After defining the relationships, you can place the source part or subassembly into an assembly that contains the target part.

For more information, see Placing parts or subassemblies with predefined relationships.
