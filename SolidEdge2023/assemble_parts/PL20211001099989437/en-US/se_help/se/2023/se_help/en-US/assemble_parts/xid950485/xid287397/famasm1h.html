Create a new family of assemblies

  1. On PathFinder, click the Family of Assemblies tab.

  2. On the Family of Assemblies tab, click the New button to display the Alternate Assemblies dialog box.

  3. Select Create a Family of Assemblies.

  4. In the Name current member box, type a name for the first family member.

  5. In the Name new member box, type a name for the second family member.

  6. Click OK.

    On the Family of Assemblies tab, notice that the name you typed in the Name new member box is now the active member.

Tip:

  • When creating a new alternate assembly, you must specify whether the alternate assembly will be a family of assemblies or an alternate position assembly.

  • You cannot delete the default member, which is the member you named using the Name current member box.

  • When defining the parts and subassemblies contained in a particular member, you can use the Tear Off command on the shortcut menu to tear off the Family of Assemblies tab. This allows you to display the Family of Assemblies tab and PathFinder simultaneously.

  • When you place an alternate assembly into a higher level assembly, you can specify which member you want to place.

  • To learn more about alternate assemblies, work through the tutorial, Building a family of assemblies.
