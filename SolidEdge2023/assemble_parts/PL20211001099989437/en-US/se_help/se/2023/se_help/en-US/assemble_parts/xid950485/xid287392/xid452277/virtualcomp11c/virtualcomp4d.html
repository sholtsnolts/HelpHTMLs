Select Component Sketch Dialog Box

Available Sketches and Orientations

Lists the component sketches and reference plane orientations you can use to position the predefined component in the assembly sketch. You create the component sketches using the Component Sketch command.

Apply to All Instances With a Defined Position

Specifies that all instances of the predefined component will have the component sketch you select associated with it. This option is only available when you are editing the a predefined component after placement.

Rotate Counter-Clockwise 45 Degrees

Rotates the component sketch 45 degrees counter-clockwise when placed. You can click the button one or more times. You can also press the A key one or more times to rotate a component sketch counter-clockwise while you are placing it.

Rotate Clockwise 45 Degrees

Rotates the component sketch 45 degrees clockwise when placed. You can also press the S key one or more times to rotate a component sketch clockwise while you are placing it.
