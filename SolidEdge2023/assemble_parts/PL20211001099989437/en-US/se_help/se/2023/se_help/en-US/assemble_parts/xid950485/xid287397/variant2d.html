Alternate Member Dialog Box

You can use the Alternate Member dialog box to specify which alternate components member you want when placing a component into an assembly or when replacing a component in an assembly.

Member

Specifies the alternate component member you want to use for the component you are placing or the component you are replacing.

Browse

Displays the Replacement Part dialog box so you can select a replacement component.

Search

Displays the Search dialog box so you can define search criteria for a replacement component.
