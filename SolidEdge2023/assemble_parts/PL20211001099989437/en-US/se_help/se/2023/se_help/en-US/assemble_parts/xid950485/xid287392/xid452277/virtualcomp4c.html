Position Virtual Component command

Positions a virtual component in the graphic window.

You can use this command on any virtual component before graphics are associated with the component, with an instance component after graphics are associated with the source component, and with predefined components.

You can use the A key on the keyboard to rotate the virtual component while you are placing it. Every time you press the A key, the virtual component is rotated 45 degrees. You can also use the Rotate command to rotate the virtual component after you place it.

When positioning predefined components, you can use the Select Component Sketch dialog box to specify a component sketch or reference plane orientation from the predefined component you want to use in the assembly sketch.

For more information on working with virtual components, see the Creating and Publishing Virtual Components Help topic.
