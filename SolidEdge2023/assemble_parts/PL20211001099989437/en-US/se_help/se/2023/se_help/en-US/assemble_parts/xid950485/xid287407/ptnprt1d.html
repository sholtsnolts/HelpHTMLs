Pattern Parts command bar

Main Steps
Select Parts Step

Specifies the parts to be patterned. You can select parts in the active assembly, entire subassemblies, and existing patterns of parts.

Define Pattern Step

Defines how the part pattern will be constructed.

Cancel/Finish

Cancels the Pattern command or finishes the part pattern.

Selecting Parts
Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selection.

Defining the Pattern
Select Part

Specifies which part in the assembly contains the feature pattern you want to use. You can also select an assembly or part sketch if the sketch contains a 2D pattern profile.

Select Pattern Feature

Specifies the feature pattern. You can select a pattern feature on the part or a pattern on an assembly sketch.

Select Reference Position

Specifies which individual feature of the pattern will be used as the starting point of the part pattern. In most cases you should select the feature where you placed the original copy of the part to be patterned.

Note:

A reference feature only needs to be identified when using an ordered “Smart” pattern as position components. Pattern components are placed relative to the original geometry defining the pattern, and the assembly component should be positioned relative to this original component for proper placement of the pattern.
