Update All Open Documents command

The Update All Open Documents command updates an assembly when you edit a part in the context of the assembly. For example, with Automatic Update on, if you in-place activate a part from an assembly and edit the value of a driving dimension, the changes are automatically applied to the part but not to the assembly. You can use this command to update the assembly.

You can also use this command to update a part copy from within the assembly.
