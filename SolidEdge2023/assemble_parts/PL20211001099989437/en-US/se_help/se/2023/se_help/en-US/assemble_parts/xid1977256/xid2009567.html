Point Cloud Options dialog box

Specifies default options for handling point clouds when they are referenced in an assembly.

On display, zoom to the point cloud

Centers and zooms a point cloud in the graphics window when the file is opened.

Note:

For an existing point cloud, you can use the Zoom To command on the shortcut menu in PathFinder.

When the command completes, display the steering wheel for repositioning

After inserting a point cloud into the document, displays the steering wheel for you to move it.

Note:

For an existing point cloud, you can use the Move Point Cloud command on the shortcut menu in PathFinder. For more information, see Move a point cloud.

Save display data for fast open

For an existing point cloud file, caches the data in a very compressed format to reduce the time required to reopen the file.

Multiple assemblies can reference the same point cloud.

Clear Display Data

Deletes the cached reference point cloud display data from the directory.

Caution:

The Clear Display Data button clears the display data of all point cloud files that have been cached, not just the point clouds files in the open assembly, and not just the selected point cloud.
