Create a minimum distance sensor

  1. On the toolbar on the left, click the Sensors tab.

  2. On the Sensors pane, click the Minimum Distance Sensor button .

  3. On the Minimum Distance command bar, from the Element Types list, select the type of element the sensor will measure between: Keypoints/Curves/Faces, Keypoints, Curves, or Surfaces.

  4. In the graphics window, select the element to measure from.

  5. In the graphics window, select the element to measure to.

    Tip:

    Define a Minimum Distance Variable—After you specify the elements to measure between, you can create a measurement variable to track minimum distance. On the command bar, click the Measurement Variable option. In the Measurement Variable dialog box, type a variable name in the Name box, or click OK to accept the system-generated variable.

  6. On the Minimum Distance command bar, click the Close button.

  7. In the Minimum Distance Sensor Parameters dialog box, define the sensor parameters, and then click OK.

    The sensor is displayed on the Sensors pane.
