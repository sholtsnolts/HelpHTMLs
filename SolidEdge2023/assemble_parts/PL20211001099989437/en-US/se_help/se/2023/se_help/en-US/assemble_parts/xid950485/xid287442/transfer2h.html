Transfer parts to a new subassembly

  1. In PathFinder, click the part(s) you want to place in the new subassembly.

    Note:

    To select more than one element, hold the Ctrl key and click each element.

  2. Choose Home tab→Modify group→Transfer .

  3. On the Transfer to Assembly Level dialog box, click the assembly where you want to create the new subassembly.

  4. On the Transfer to Assembly Level dialog box, click the New Subassembly button.

  5. On the Create New Subassembly dialog box, in the Template box, select a file to use as a template for the new subassembly.

  6. In the New File Name box, type a new name for the subassembly.

  7. In the New File Location box, type a new file location for the subassembly.

  8. Specify how you want to position the selected part(s) in the new subassembly.  

    For more information on specifying the position, see Create New Subassembly Dialog Box.

  9. On the Create New Subassembly dialog box, click OK.

  10. On the Transfer to Assembly Level dialog box, click OK.
