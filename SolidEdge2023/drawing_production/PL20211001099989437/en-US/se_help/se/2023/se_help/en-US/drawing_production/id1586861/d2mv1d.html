2D Model View command bar

After you select the 2D Model View command and specify the geometry cropping boundary, this command bar is displayed for you to set view options before you click to place it on the drawing sheet.

Cancel

Returns you to the drawing sheet and cancels the 2D Model View function.

Drawing View Style Mapping

Specifies that the drawing view uses a predefined style. The style is set by mapping the 2D Model Views element to a style on the Drawing View Style tab of the Solid Edge Options dialog box. The style format is defined in the Drawing View Style dialog box.

When the Drawing View Style Mapping button is cleared, you can select and apply individual styles. Choose the style from the Drawing View Style list.

Drawing View Style

Selects a style for the 2D Model view. This option is not available when Drawing View Style Mapping is enabled.

Properties

Opens the Drawing View Properties dialog box for you to set options for the 2D model view.

Scale

Specifies the drawing scale as a standard ratio.  The specified ratio defines the size of the drawing in relation to the size of the real-world object. For a 2:1 ratio, the "2" represents the size of the drawing and the "1" represents the size of the real-world object.
