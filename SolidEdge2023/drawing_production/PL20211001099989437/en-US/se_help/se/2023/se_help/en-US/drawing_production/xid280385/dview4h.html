Set properties on one or more drawing views

You can change the properties on one or multiple drawing views.

  1. Do one of the following:

    • To change the properties of a single drawing view, right-click the drawing view and choose Properties.

    • To change the properties of multiple drawing views:

      1. Click the first drawing view, and then Shift+click the other drawing views, or fence-select all of the drawing views at once.

      2. On the Drawing View Selection command bar, click Properties.

  2. In the Drawing View Properties dialog box, click a tab, and then set the options that you want to use.

  3. Click Apply.

    This applies all of the changes on all of the tabs.

  4. Click OK to close the dialog box.

    The selected drawing views now show a gray outline, which indicates they are out of date.

  5. Choose Home tab→Drawing Views group→Update Views to update the drawing views with the property changes.

Tip:

When you change the properties on multiple drawing views at the same time, the drawing views that you select determine which tabs are displayed in the Drawing View Properties dialog box, as well as which options on those tabs are available for making changes.
