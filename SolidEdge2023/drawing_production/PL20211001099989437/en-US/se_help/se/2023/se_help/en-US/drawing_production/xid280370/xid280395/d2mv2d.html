Drawing Area Setup dialog box

The Drawing Area Setup dialog box collects the information necessary to create a scaled work space on the 2D Model sheet. The instructions provided by the dialog box are organized into three steps:

  • Select sheet Size for final printed drawing.

  • Enter Working area width x height OR scale factor.

  • Select a block representing the drawing border.

Sheet Size list

Specifies a sheet size for the finished printed drawing.

Width, Height text boxes and Calculate Scale button

The Width and Height text boxes specify the overall physical size of the part or assembly to be created. You can enter size information in both boxes or in just one box, clearing the other. If you leave one box blank, you can press Enter to have the system calculate the second dimension from the first.

Click the Calculate Scale button to compute the scale of the drawing from the width, height, and sheet size.

The default values in the Width and Height boxes are the dimensions of the selected sheet size.

Scale text box and Calculate Height x Width button

When setting up a new drawing area, the Scale value is used to stretch or shrink the border so that it fits the intended design size of the part or assembly. Also, it is applied to the annotation text size to maintain the correct text height when new annotations are added. This enables you to annotate in 2D model space or on the working sheet using 2D drawing views.

If you clear the Width and Height boxes and type a value in the Scale text box, you can click the Calculate Width x Height button to compute the area of the required work space.

The default value in the Scale field is 1.00.

Note:

The text scale value applied to new annotations is stored per sheet.

Document

Displays the file name and path name of a source draft document for a drawing border. If the file contains multiple blocks, they are listed at the bottom of the Drawing Area Setup dialog box.

Browse

Click to browse to a different draft file containing blocks for the drawing area border.

To choose a drawing border created specifically for the AutoCAD environment, select one of the drawing border blocks from the TitleBlocks.dft file located in the \Program Files\Siemens\Solid Edge 2023\Sample Blocks folder.

Place block

When selected, places the block name selected from the list below as the border for the drawing area. Clear this option if you do not want to use a border.

Note:

The Drawing Area Setup dialog box does not retain settings between sessions. When you close the dialog box, the settings revert to the default.
