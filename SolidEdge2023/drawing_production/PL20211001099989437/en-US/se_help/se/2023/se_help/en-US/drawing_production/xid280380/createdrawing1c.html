Drawing of Active Model command

The Drawing of Active Model command creates a drawing from the currently open assembly, part, or sheet metal document using the template specified in the Create Drawing dialog box.

You can select between the following options to generate the drawing views:

  • Create standard views: To create front, top, and side views (Isometric views for assemblies) based on the specified template.

    • If the specified template is a Quicksheet template, then the drawing views and parts lists (if any) in the Quicksheet template are created using the current model file.

    • If the specified template is not a Quicksheet template and the current model file is an assembly, then an isometric view is created.

    • If the specified template is not a Quicksheet template and the current model file is a part, or sheet metal part, then top, front, and right views are created.

    If the current model file is a family of assemblies, the active family member is used to create the new drawing.

  • Run Drawing View Creation Wizard: To open the View Wizard command bar to configure the required views.

    Note:

    Users working with large assemblies to benefit from the large assembly workflow and simplified assemblies should choose to run the Drawing View Wizard.

  • Create from Model View: To create a drawing view from the selected the model view and display it in the dynamic display mode.

    When you select this option, the Model Views list becomes available to let you choose the model view.

Note:

The New→Drawing of Active Model command is available from the File menu. It also is available on the shortcut menu, when an assembly, part, or sheet metal file is selected in PathFinder or the Library pane.
