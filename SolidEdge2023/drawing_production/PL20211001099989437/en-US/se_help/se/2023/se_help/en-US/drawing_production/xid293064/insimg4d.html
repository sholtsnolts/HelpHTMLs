Border page (Insert Image dialog box)

Image border properties can be set when inserting an image or picture into the document using the Insert Image command, and they can be edited after the image is placed.

Show Border

Shows or hides the image border. Even if the border is hidden, it can be located by moving the cursor over the edge of the image.

Style

Sets the border line style when inserting images in draft documents. This list is not available when inserting images into parts created in the 3D environment.

Color

Specifies the line color of the image border.

Width

Specifies the line thickness of the image border.

Type

Specifies the line type of the image border.
