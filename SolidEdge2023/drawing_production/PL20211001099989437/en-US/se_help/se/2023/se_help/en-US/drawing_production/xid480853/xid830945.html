Create a simplified assembly drawing view and parts list

You can use simplified assemblies to create drawing views and parts lists automatically. This is especially useful when working with large assemblies and managed documents. This workflow shows how you can associate your predefined saved settings created within the View Wizard command and the Parts List command to do this.

  • Define the assembly sizes for drawing views and parts lists.

    On the Drawing View Wizard tab (Solid Edge Options dialog box), do the following:

    1. Under Assembly Drawing Views, use the Number of occurrences box to define the size of Large Assemblies.

      Example:

      You can specify that assemblies with greater than 1000 occurrences are classified as Large Assemblies.

    2. Under Saved Settings, use the Large Assemblies list to specify the name of the drawing view saved setting that you want to use to produce drawing views of simplified assemblies.

      Example:

      The saved setting to produce an assembly drawing view using simplified assembly options would include selecting the Use simplified assemblies check box on the Drawing View Creation Wizard (Drawing View Options) dialog box.

      For more information, see Saving and reusing drawing view settings by model type.

    3. Under Parts List Saved Settings, select the name of a parts lists saved setting that specifies the formatting and content for a parts list generated using simplified assemblies.

      Example:

      This is where you can assign a parts list saved setting that specifies Use simplified assemblies on the List Control tab in the Parts List Properties dialog box.

  • Produce a drawing view of a simplified assembly automatically.

    1. For a model containing one or more simplified assemblies, start the Drawing View Wizard by doing one of the following:

      1. In the assembly model, from the File menu, select New→Drawing of Active Model .

      2. In the Draft environment, drag an assembly model from the Library tab onto the active sheet.

        For more information, see Creating drawing views from a model dragged onto a sheet.

      3. In the Draft environment, select Home tab→Drawing Views group→View Wizard .

    2. Click to place the drawing view of the assembly.

  • Place the parts list using the simplified assemblies.

    1. Choose the Home tab→Tables group→Parts List command .

    2. On the drawing sheet, select the assembly drawing view.

    3. Move your cursor to the location on the drawing sheet where you want to place the parts list, and then click.

    The parts list that is generated lists the simplified assemblies and their item count in the item balloons.

Tip:

  • Your Drawing View Wizard saved settings are saved in a file named DraftWizard.txt. The default installation folder is \Program Files\Siemens\Solid Edge 2023\Template\Reports.

  • Your parts list saved settings are saved in a file named DraftList.txt. The default installation folder is \Program Files\Siemens\Solid Edge 2023\Template\Reports.
