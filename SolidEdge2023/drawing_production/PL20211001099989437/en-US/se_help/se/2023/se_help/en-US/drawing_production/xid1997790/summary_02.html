Summary

The Drafting application provides for the creation of drawings. Views and dimensions that are placed on a drawing are associative to the 3D model and update when changes are made to the model.

In this activity you:

  • Created drawings

  • Added views to a drawings

  • Dimensioned drawing views

  • Placed annotations on drawings

  • Placed a parts list on a drawing.
