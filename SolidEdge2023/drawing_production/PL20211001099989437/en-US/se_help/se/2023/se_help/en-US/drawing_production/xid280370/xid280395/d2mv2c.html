2D Model Sheet command

The 2D Model Sheet command displays a drawing sheet with the label "2D Model" in the active document. Place geometry on this sheet from which you want to create a 2D model view with the 2D Model View command.

There is one 2D Model sheet per document.

Note:

The 2D Model Sheet command is available on the View tab, as well as on the shortcut menu when you right-click a tab in the sheet tab tray.
