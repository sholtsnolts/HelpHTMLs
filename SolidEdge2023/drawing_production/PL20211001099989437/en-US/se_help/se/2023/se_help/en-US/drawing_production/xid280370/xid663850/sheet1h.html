Edit a background sheet

You can display and edit the background sheet that is associated with a working sheet.

  1. Right-click the working sheet tab in the sheet tab tray at the bottom of the drawing, and choose Edit Background.

    This selects and displays the background sheet associated with the working sheet.

  2. Make your changes to the background sheet.

  3. Click the working sheet tab when you are finished to see the changes.

Tip:

  • You can show and hide the background sheet tabs in the sheet tab tray by right-clicking any sheet tab and choosing the Background command.

  • You can delete or rename a background sheet using the shortcut menu.

  • You can save background sheets in a template for easy access. Create the background sheets in a new document, and then save the document to a template folder. Use the Solid Edge Options command on the File menu to specify the location of the template folder so that the templates are available when you create new documents.

  • You can use the Block command to create a reusable block from the background sheet graphics. To add this background to the 2D Model sheet, use the Drawing Area Setup command and select it from the Place Block list.
