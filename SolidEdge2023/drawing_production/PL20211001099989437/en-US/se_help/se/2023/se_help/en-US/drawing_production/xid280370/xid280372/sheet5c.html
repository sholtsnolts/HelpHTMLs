Sheet Setup command

Displays the Sheet Setup dialog box for you to define the properties of the working sheet.

In the Sheet Setup dialog box, you can:

  • Display and modify working sheet properties, such as name, size, scale, and margin.

  • Select the background sheet you want to use.

  • Save the changes you make on the Size tab, so that they are applied to all working sheets you create in the document.

  • Save and apply the changes you make on the Background tab. These settings are applied to all working sheets that use the background.

Note:

Use the View tab→Sheet Views group→Working Sheets command to display the working sheets in the document.
