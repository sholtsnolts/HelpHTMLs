Create a perspective drawing view

  1. Choose Home tab→Drawing Views group→View Wizard .

  2. In the Select Model dialog box, select a part, sheet metal, or assembly document, and then click Open.

  3. On the View Wizard command bar, click the Drawing View Layout button .

  4. On the Drawing View Creation Wizard (Drawing View Layout) dialog box, click Custom.

  5. In the Custom Orientation dialog box, press the left mouse button as you move the mouse to adjust the model orientation. When the model approximates the desired orientation for the perspective view, release the button.

  6. In the Custom Orientation dialog box, do one of the following:

    (Select a predefined perspective angle)

    • Click the Perspective button , and then select a value from the Perspective Angle list.

      Values are based on the focal length of a 35 mm camera.

      • Portrait (85 mm)

      • Normal (50 mm)

      • Wide (35 mm)

      • Very wide (10 mm)

    (Define a custom perspective angle)

    1. Press Ctrl+Shift while you rotate the mouse wheel to change the perspective view angle and distance.

      The model updates with each movement of the wheel.

    2. Click the Perspective button to apply the perspective angle to the view.

      This value is stored as the new Custom angle for the view.

  7. Close the Custom Orientation dialog box, and then close the Drawing View Layout dialog box.

  8. On the drawing sheet, click to place the view.

Tip:

Before you click to place the view, you can automatically size the view to the sheet by selecting the Best Fit option on the command bar.
