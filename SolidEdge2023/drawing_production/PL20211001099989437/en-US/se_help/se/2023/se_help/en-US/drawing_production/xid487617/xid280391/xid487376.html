Modifying captions

You can control caption display and formatting separately for a selected drawing view, such as an auxiliary or section view, and for the viewing plane or cutting plane used to create it.

  • Once a drawing view is placed, you can select the view and then use the Caption text box and the Show Caption list on the Drawing View Selection command bar to add a primary caption to the drawing view, or to modify it.

  • When you select a drawing view, you can modify the placement and formatting of a caption using the Caption tab (Drawing View Properties dialog box). You also can add a secondary caption to a drawing view using the Caption tab.

  • When you select a view annotation, you can use the Caption tab (Viewing Plane, Detail Envelope, Cutting Plane Properties dialog box) to modify its properties.
