Edge Painter command

The Edge Painter command changes the line display type of an element or element segment in a drawing view, 2D view, or sketch. You can change the edge display one element at a time, or you can change multiple elements at the same time. The different line types include hidden, visible, tangent, user-defined, and assembly-hidden. To learn how to use the Edge Painter command, see Change the edge display in a drawing view.

Example:

Occasionally when you create a drawing view from a model, edges that transition from visible to hidden may not give you the results you want. You can use Edge Painter to apply the desired edge display style.

Example:

For drawings of an assembly model, another edge option is available to classify an edge as assembly-hidden. Assembly-hidden edges are visible edges that are concealed behind another part in the assembly model but would otherwise normally be visible.

Note:

The appearance of each edge type is specified by the line style that defines it. For more information, see Applying formats with styles.
