Drawing View Properties dialog box

Defines the properties for a drawing view or a 2D Model view.

Not all properties are available for all types of views. For example, dependent detail views and 2D Model views have access only to the properties on the General tab and the Text and Color tab, whereas independent detail views have access to the properties on all of the tabs.
