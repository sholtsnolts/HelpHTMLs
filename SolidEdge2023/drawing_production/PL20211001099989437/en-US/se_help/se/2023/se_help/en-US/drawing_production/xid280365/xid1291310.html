Undo drawing view changes

You can use the Undo command to remove drawing views you just created with the View Wizard command, the Principal command, the Section command, and other drawing view creation commands. You also can undo an update of a drawing view, for example, when the model has changed, and when you make changes using the Drawing View Properties dialog box.

  • On the Quick Access toolbar, click the Undo command .

Tip:

  • To undo multiple changes, continue clicking the Undo command or use the Undo list to locate a specific change to undo.

  • You also can use the Redo command with drawing views.

  • You cannot undo a broken-out section view, because it involves a change in environments to draw the section profile.
