Replace Background command

Replaces the contents of one or more Solid Edge background sheets with the contents of background sheets in another file. For example, after making changes to the background in a draft template to update a company name, logo, or title block, you can use the Replace Background command to then pull those changes into existing drawing files.

All of the background sheet contents (such as geometry, text, dimensions, annotations, property text) are copied. Use the Replace Background dialog box to specify that you want to replace a single background sheet or multiple background sheets.

Background sheets are replaced based on sheet name and sheet size.

  • If the background sheet names and sizes are identical, then the contents are simply replaced.

  • If there are different background sheet names or sizes in the two documents, then you have the option of copying the background contents from one document and replacing the background contents in the other document.

  • If there are no matches between the two documents, then you have the option of inserting the background sheet into the document.

The Replace Background command is located on the shortcut menu of a selected background sheet tab.
