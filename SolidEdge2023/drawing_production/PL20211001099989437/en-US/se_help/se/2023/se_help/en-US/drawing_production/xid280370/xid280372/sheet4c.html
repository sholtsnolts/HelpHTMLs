Insert Sheet command

Inserts a new drawing sheet in the document using the default drawing sheet settings. You can change the default settings using the Sheet Setup command.

You can create either a working sheet or a background sheet, depending upon which sheet tab your cursor is on when you select the Insert Sheet command.
