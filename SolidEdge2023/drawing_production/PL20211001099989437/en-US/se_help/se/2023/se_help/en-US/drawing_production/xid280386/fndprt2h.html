Find a part in a drawing view

  1. Do one of the following:

    • Right-click an element in the drawing view.

    • Right-click the drawing view border.

  2. On the shortcut menu, click Properties.

  3. In the Drawing View Properties dialog box, click the Display tab.

  4. In the Parts List, highlight the parts you want to query.

  5. In the query list, select the query you want to use.

  6. Click the Execute Query button.

    The parts that match your query are highlighted in the Parts List.
