Working with view annotation captions

You can define independent captions for the view annotations that are used to create section, auxiliary, and detail drawing views. For example, you can define a caption for a cutting plane line that references the section view and displays the following information:

  • The alphanumeric label for the cutting plane line.

    For more information, see Assigning view annotation labels.

  • The sheet number where the section drawing view is located.

Defining view annotation captions

Both the drawing view caption and the view annotation caption are defined within the drawing view style, on the Caption tab in the Drawing View Style dialog box. The drawing view style, in turn, is associated with the document template you are using.

Use the middle portion of the Caption tab—under View Annotation—to specify a single line of property text, plain text, and symbols that you always want to display in the source view caption where the currently selected View Annotation Type—Cutting Plane, Viewing Plane, or Detail Envelope—is drawn.

You also define caption formatting within the drawing view style, using the Caption Format tab (Drawing View Style dialog box).

Whatever you define as the default caption content can be modified after a drawing view is placed by selecting the cutting plane line, the viewing plane line, or the detail envelope, and then opening the Properties dialog box.

View annotation property text

You can use property text to insert and update the label name and the sheet number into the view annotation caption. You can use the following buttons on the Caption tab in the Drawing View Style dialog box to insert the corresponding property text codes automatically.

  • View Annotation Name (%VA)

    The %VA property text code automatically displays the view annotation label (for example, A, B, C) within the caption text of the cutting plane, viewing plane, or detail envelope view.

    Note:

    • The alphanumeric label is defined in the Specify Annotation Letters dialog box, which you can open from the Annotation tab (Solid Edge Options dialog box).

    • You also use the Define Object Sequence dialog box to specify the order that you want to assign a name to view annotation objects.

    See the Help topic, Define annotation labels.

  • View Sheet Number (%VN)

    The %VN property text code automatically displays the sheet number where the section, auxiliary, or detail view is located.

    Note:

    The %VN property text code provides a sheet cross-reference between the source view where the cutting plane, viewing plane, or detail envelope is drawn, and the resultant view.

For an example of how to apply these property text codes to a caption for a detail envelope, see the help topic, Define a view annotation caption.

Referencing text in view annotations

You can use reference text to associatively reference the information in a view annotation caption from a callout or text box. You also can reference other view annotation properties, such as its label name and the sheet number location of the derived drawing view. For more information, see the help topic Using reference text.
