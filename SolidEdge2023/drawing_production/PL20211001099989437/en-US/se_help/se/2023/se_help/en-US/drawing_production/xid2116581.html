Show model views in a palette (Draft environment)

You can view, activate, and manipulate PMI model views in the draft environment from the Model View Palette pane.

Procedure

  1. Choose Home tab→Drawing Views group→Model View Palette command.

  2. From the list of the models, select the model currently linked to the draft file to display the model views.

    If the list is empty, click and select the model or assembly to retrieve its model views.

  3. Drag the model views to the drawing sheets to create the drawing views.

    Tip:

    To align the drawing views while placing them, in the top-right corner of the palette, activate the create alignment command .
