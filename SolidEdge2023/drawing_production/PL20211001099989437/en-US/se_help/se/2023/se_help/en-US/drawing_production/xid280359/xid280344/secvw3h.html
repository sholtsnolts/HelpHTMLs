Create a thin-section view


You can use the Section Only option to create a thin-section section view that displays only the geometry that intersects the cutting plane. The resulting section view does not include the geometry behind it.

  1. Choose Home tab→Drawing Views group→Section View .

  2. Click a cutting plane.

  3. On the command bar, click Section Only .

  4. On the drawing sheet, click to position the thin-section view.
