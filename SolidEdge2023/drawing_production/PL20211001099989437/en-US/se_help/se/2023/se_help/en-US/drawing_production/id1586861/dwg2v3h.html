Edit a 2D drawing view

  1. Double-click the 2D drawing view.

    This opens the view on the 2D Model sheet.

  2. Use the commands provided to edit the 2D drawing view.

  3. Click the working sheet tab to display the updated 2D drawing view.

    Only the changes made within the 2D view boundary are visible on the working sheet.

Tip:

Any elements you place on the Auto-Hide layer are automatically hidden when you close the 2D view window.
