Rename dialog box

The Rename dialog box provides a fast method to rename a drawing sheet.

New Name

Renames the active sheet when you type a new name.

Note:

Another way to rename a drawing sheet is to use the Sheet Setup command. This method has the advantage of letting you also change other sheet properties, such as sheet size, print size, and background sheet.
