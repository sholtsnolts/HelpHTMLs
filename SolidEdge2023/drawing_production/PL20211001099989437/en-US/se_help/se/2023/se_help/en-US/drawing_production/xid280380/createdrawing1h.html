Create a drawing with the Drawing of Active Model command

Automatically create a drawing of the 3D model displayed in an open model document.

  1. Save the current model (assembly, part, or sheet metal) file.

  2. On the File menu, click New→Drawing of Active Model .

    The Create Drawing dialog box is displayed.

  3. Specify a template for the drawing.

  4. Do one of the following:

    • (For automatic drawing view creation) Clear the Run Drawing View Creation Wizard check box and click OK.

      The resulting drawing views depend upon the template you selected.

      • If the specified template is not a Quicksheet template and the current model file is a part or sheet metal part, then top, front, and right views are created.

      • If the specified template is not a Quicksheet template and the current model file is an assembly, then an isometric view is created.

      • If the specified template is a Quicksheet template, then the drawing views and parts lists (if any) in the Quicksheet template are created using the current model file.

    • (For user-defined drawing views) Select the Run Drawing View Creation Wizard check box and click OK.

      The View Wizard command bar is displayed, and a live preview of the selected model view is shown in dynamic display mode on the drawing sheet. You can use the options on the command bar to configure the view you want.

      Note:

      Users working with large assemblies who want to benefit from the large assembly workflow and simplified assemblies should choose to run the Drawing View Wizard

      • To learn how to select and place user-defined drawing views, see the following help topics:

        • Create drawing views of a part with the View Wizard.

        • Create assembly part views with the View Wizard

      • For general information about creating drawing views, see Drawing view creation.

  5. Click to place the views when you are done making changes on the View Wizard command bar.

Note:

In the Teamcenter-managed environment, draft documents are automatically created in the same Item as the 3D model resulting in a single Item Revision having a 3D dataset and the corresponding draft.
