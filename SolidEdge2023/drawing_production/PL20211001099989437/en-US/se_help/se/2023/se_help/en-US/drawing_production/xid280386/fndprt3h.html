Create a new draft query

  1. On the Library tab, click the Queries tab .

  2. On the Queries tab, click the New Query button .

  3. In the Query dialog box, set the options you want to define the search criteria.

  4. Click OK.

    The new query is added to the Queries tab and to the query list in drawing view properties.

Tip:

  • You can use the commands on the shortcut menu to edit, delete, and rename a query entry on the Query tab.

  • You can create, manipulate, and execute queries from the Display page of the Drawing View Properties dialog box.
