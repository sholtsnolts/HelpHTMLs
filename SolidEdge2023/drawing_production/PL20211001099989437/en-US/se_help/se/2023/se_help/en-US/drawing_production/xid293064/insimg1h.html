Insert an image on a drawing

You can insert an image or picture in the center of the document.

  1. Choose the Insert Image command .

    The Insert Image dialog box is displayed.

  2. On the General page (Insert Image dialog box), specify an image file:

    1. Click the Browse button.

    2. In the Open a File dialog box, set the Files of type list to All Image Files (*bmp,*.png,*.jpg,*.jpeg,*.jpe,*,tif,*.tiff).

    3. Locate the image file and click Open.

    4. (Optional) To ensure that the inserted image updates if the source file changes, select the Link to file check box.

  3. (Optional) On the Border tab (Insert Image dialog box), set image border options.

  4. Click OK to close the Insert Image dialog box and insert the image.

  5. (Optional) The image is still selected. You can use the options on the Insert Image command bar to change the image height, width, angle, horizontal or vertical display, and aspect ratio.

  6. Click in free space to exit the command.

Tip:

  • To modify an image on a drawing, click the image border or drag a box around it to select it.

  • To move the image from its default location, use the Move command.

  • Click the Image Properties button on the command bar to change the image border line properties after it is inserted.

  • You also can use the shortcut menu of an image to control its horizontal or vertical display, its aspect ratio, and to display or hide the image border.
