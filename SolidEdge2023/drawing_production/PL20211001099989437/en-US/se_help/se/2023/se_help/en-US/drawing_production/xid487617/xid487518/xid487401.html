Assigning view annotation labels

Label names

The alphanumeric label that appears in the caption of a cutting plane, detail envelope, viewing plane line, and in certain annotations is defined in the Specify Annotation Letters dialog box. You can open this dialog box from the Annotation tab in the Solid Edge Options dialog box.

You can use the Specify Annotation Letters dialog box to:

  • Define the labels for view annotations, datum frames, hole callouts, and other types of annotation.

  • Specify an alphanumeric label schema (for example, AA or A1) that increments automatically with every annotation you create (for example, AB or A2).

    Alternatively, you can define a static label that is always used for a specific annotation type.

  • Define a label using four different sequences of any combination of uppercase and lowercase letters and numbers.

  • Assign a different labeling sequence to different types of annotations. The labels can be assigned automatically according to a user-defined order or in the order they are created.

Label generation sequence

You can specify the order in which labels are generated using the following options on the Annotation tab (Solid Edge Options dialog box).

Follow object creation sequence

Assigns names based on the order in which view annotation objects are generated.

Follow defined object sequence

Assigns names to view annotation objects based on a user-defined order. You can use the Define Object Sequence dialog box to specify the order.
