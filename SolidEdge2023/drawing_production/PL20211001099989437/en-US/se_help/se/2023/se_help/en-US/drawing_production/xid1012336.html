View annotation edit handles

Modification handles are available on drawing view annotations, which are the cutting plane lines, view plane lines, and detail envelopes used to create section views, auxiliary views, and detail views. You can use the handles to make the following types of edits.

Note:

The handle colors are defined by the Handle 2 and Handle 3 colors on the Colors tab in the Solid Edge Options dialog box. The default colors (light blue and dark blue) are shown in the following table.

View annotation type

Handles

Example

Cutting plane line

Light blue edit handles:

  • Move the caption text.

  • Change the length of the direction arrows.

Modifying section views

Section view and cutting plane captions

Tip:

You can change the direction of a selected cutting plane line using the Flip Direction shortcut command.

If you change the view direction, you must also update the section view derived from it.

Detail envelope

Light blue edit handle—Moves the caption text.

Detail view and detail envelope captions

View plane line

Light blue edit handles:

  • Move the caption text.

  • Change the arrow line length.

  • Single arrow—Adjust the position of the annotation along the current vector.

  • Double arrow—Adjust the length of the view plane line.

Modifying auxiliary views and view planes

Dark blue move handle—Freely moves the view plane line.
