Set detail envelope properties

  1. In the source view, click the detail envelope.

  2. On the Detail Envelope Selection command bar, click the Properties button.

  3. In the Detail Envelope Properties dialog box, set the options you want to use.

Tip:

You can also set the detail envelope properties using the Properties command on the shortcut menu of the detail envelope.
