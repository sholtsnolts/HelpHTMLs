Understanding drawing captions

You can add two types of captions to drawings:

  • Drawing view captions—Captions that you can add manually or automatically to any drawing view. You can separate the drawing view caption content into a primary caption and a secondary caption, and you can apply a horizontal divider between the two.

  • View annotation captions—Captions on cutting plane lines, viewing plane lines, and detail envelopes. View annotation captions can be defined and managed independently of drawing view captions.

Example:

A view annotation caption (B) for a detail envelope.

A drawing view caption (DETAIL B) for a detail drawing view.
