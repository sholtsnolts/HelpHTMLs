Select Block dialog box

Use the Select Block dialog box to choose a block containing graphics that you want to display as a center of mass symbol or a coordinate system symbol in a drawing view. This dialog box is displayed when you select one of the Change buttons on the Annotation tab in the Solid Edge Options dialog box.

Select block pane

Lists all block names and block view names that exist in the current draft document. Selecting a block name displays a preview of the block graphics that are associated with it.

Browse

Locates and opens a different draft document for you to select a block.

Selecting a block from another document copies the block to the current draft file.

Note:

After opening the document, you still need to select the name of the block in the Select block pane.
