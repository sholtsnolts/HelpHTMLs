Saving and reviewing drawing comparison results

You can save and review the differences found when two draft files are compared with the Compare Drawings command.

Saving the results of a drawing comparison

Draft files are compared and saved on a sheet-by-sheet basis. When you save the results of the Sheet 1 comparison, they are written to a set of three related .bmp files:

  • <drawing_name>_Differences.bmp

  • <drawing_name>_Original.bmp

  • <drawing_name>_New.bmp

You can change the sheet selection in the Compare Drawings dialog box to Sheet 2, and then save the results of that comparison to another set of .bmp files.

Reviewing drawing comparison results

You can review the results of a previous drawing comparison by selecting the Compare Drawings command, and then browsing to and selecting a previously saved differences file, <drawing_name>_Differences.bmp. Opening this file also reopens the two files that were compared.

You can use the options in the Display section of the dialog box to examine the source files and to review the differences between them in the Differences viewport.
