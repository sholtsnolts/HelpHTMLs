Inactivate Drawing Views command

Deactivates all parts lists and model-derived tables in the active draft document so they cannot be selected and edited.

Inactive mode allows limited manipulation of drawing views, dimensions, and annotations. This command sets the document to inactive mode, and displays an Inactive watermark on the 2D Model sheet and all working sheets in the document.

Use this command to change the status of a draft document from active (for editing) to review.

Note:

  • You can set a preference to open a draft document in inactive mode by setting the Inactivate Drawing Views For Review option in the Open File dialog box and then clicking the Save As Default button.

  • When the Inactivate Drawing Views For Review option is set for managed documents, only the drawing file is downloaded to cache; the linked model documents are not. This greatly reduces the time required to open the drawing.
