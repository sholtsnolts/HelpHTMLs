Delete the drawing view display depth clipping plane

You can delete the back clipping plane from a drawing view by removing the current display depth definition.

  1. Right-click the drawing view where a clipping plane was applied.

  2. On the drawing view shortcut menu, click Remove Defined Depth.

  3. Click Yes in the message box, "Are you sure you want to delete the depth definition from the drawing view?"

  4. On the drawing view shortcut menu, click Update to refresh the view display, or choose the Update Views command on the ribbon.

Note:

The Remove Defined Depth command is only available when a selected drawing view has a clipping plane applied to it.
