Modify a broken-out section view

You can modify a broken-out section view by selecting the profile used to create it.

  1. Right-click the drawing view that shows the broken-out section and choose Properties.

  2. On the General tab in the Drawing View Properties dialog box, select the Show Broken-Out Section view profiles check box.

  3. Click the profile that was used to create the broken-out section view.

  4. Do one of the following to modify the broken-out section view:

    To do this

    Do the following

    Change the shape or size of the profile.

    1. On the command bar, click Modify Profile.

    2. Drag the handles to change the profile shape, or delete individual lines and redraw the shape, ensuring the end points are connected.

    3. When you are done, click the Close Broken Out Section button on the ribbon.

    Change the depth of geometry that is removed by the broken-out section.

    1. On the command bar, click Modify Depth.

      The current clipping plane location is displayed on the view where it was defined. If it was defined associatively, the connect relationship indicator also is shown.

    2. In a drawing view that is orthogonal to the view where the profile is drawn, click a different keypoint (such as a center point, end point, or midpoint) or a centerline.

      Tip:

      To define a non-associative extent depth, you can press Shift before you click.

    3. On the command bar, click Accept.

    Remove a broken-out section.

    1. Select the broken-out section profile.

    2. Press the Delete key.

  5. Update the broken-out section view to show the changes.
