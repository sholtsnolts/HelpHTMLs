Display or hide parts in a part view

  1. Do one of the following:

    • To change the properties of a single drawing view, right-click the drawing view and choose Properties.

    • To change the properties of multiple drawing views:

      1. Click the first drawing view, and then Shift+click the other drawing views, or fence-select all of the drawing views at once.

      2. On the Drawing View Selection command bar, click Properties.

  2. In the Drawing View Properties dialog box, click the Display tab.

  3. From the Parts list, select the part or parts that you want to display or hide.

  4. Do one of the following:

    • In the Selected Parts Display section, select Show to display a part.

    • In the Selected Parts Display section, clear the Show check box to hide a part.

  5. Set the other options you want to use for the selected parts.

  6. Click Apply to update the selected drawing views with your changes.

  7. Choose Home tab→Drawing Views group→Update Views to update the drawing views with the part display changes.

Tip:

  • You can use the Properties command to set the style options for visible, hidden, and tangent lines.

  • You can select all occurrences of a part in a drawing view. To do this, in the Drawing View Properties dialog box, click the Display tab. From the Parts list, right-click the part and then click Select All Occurrences.
