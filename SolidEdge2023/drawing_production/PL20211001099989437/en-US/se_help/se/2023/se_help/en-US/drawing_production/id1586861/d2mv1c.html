2D Model (View) command




The 2D Model (View) command creates a 2D view of the 2D geometry on the 2D Model sheet. The 2D views are placed on the working sheet.

You can create multiple 2D views at a 1:1 scale, and you can customize the clipping boundary for each view. You can add annotations and dimensions to complete the drawing.
