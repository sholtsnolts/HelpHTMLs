Inherit Break Lines command

The Inherit Break Lines shortcut command copies and applies break lines from a source view to another principal, section, or auxiliary view. The inherited break lines are associative to the source drawing view.

You can use this command to copy break lines created using the Add Break Lines command.

Note:

The view you copy to must be aligned with the source view.
