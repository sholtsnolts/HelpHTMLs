Insert an image on a model

You can insert an image, such as a logo or picture, on a model face or anywhere else in the document.

  1. Choose the Sketching tab→Insert group→Insert Image command .

  2. In 3D environments, highlight and lock a sketch plane.

    To learn how to do this, see Lock or unlock a sketch plane.

  3. On the General page (Insert Image dialog box), specify the image file to insert and the insertion method.

    Tip:

    • If you select the Browse button to locate an image file using the Open a File dialog box, set the Files of type list to All Image Files (*bmp,*.png,*.jpg,*.jpeg,*.jpe,*,tif,*.tiff).

    • For information about the insertion method, see the Insert Image command.

  4. (Optional) On the Border page (Insert Image dialog box), set image border options.

  5. Click OK to close the dialog box.

  6. Click where you want to place the image.

Tip:

After you place the image, you can:

  • Change the image height, width, angle, horizontal/vertical display, and aspect ratio using the options on the Insert Image command bar.

  • Select the image, and then move or rotate it. Choose Move (or Rotate) from the Move split-button on the Sketching tab→Draw group.
