Setting Generative Design defaults

As you create and process each new generative study or apply new loads and constraints, you may find that you want to use the same settings each time. You can create your own default settings for any of the following Generative Design options using the Generative Design tab (Solid Edge Options dialog box):

  • Material

  • Color, size, and density of individual load symbols

  • Color, size, and density of all constraint symbols

  • Study quality

  • Target mass reduction
