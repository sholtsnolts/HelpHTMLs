Graphic Symbol Size dialog box

You can use the Graphic Symbol Size dialog box to change symbol size and spacing for new and previously defined load and constraint symbols.

Graphics Symbol Size slider

Changes the symbol size of the load, constraint, or other symbol. Symbol size is proportional. The default value is 50.

Symbol Spacing slider

Changes the symbol spacing pattern of the load, constraint, or other symbol. A lower value produces a denser symbol display. Conversely, a higher value produces a sparser symbol display.
