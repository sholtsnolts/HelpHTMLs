Generative Design license type

If you are using the paid license for Generative Design (Generative Design Pro), you can use the mesh model produced by the Generate command as input to other Solid Edge commands.

If you are using the free version of Generative Design included with the Solid Edge Classic and Premium licenses, the following limitations apply:

  • Only one study can be created in the document.

  • The optimized shape is a lightweight geometry cache that cannot be used for modeling, reverse engineering, or 3D printing.

  • The result cannot be published as a convergent model, so it cannot be sent to 3D printing. Also Solid Edge modeling commands cannot be used to interact with the facet model, and the results cannot be used as input to other areas of Solid Edge functionality.
