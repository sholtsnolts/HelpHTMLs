Solution Manager Colors

Advanced Design Intent Panel

Use the following features to control how and where to show the Advanced Design Intent panel:

Show panel in the document view

Select this option to display the panel displayed in the document view. The menu is fixed and displays horizontally. This is the default condition.

Show as floating panel

Select this option to allow the panel to be dragged to a location of your preference. The location is persistent.

Make panel vertical

Activate this option to display the panel vertically.
