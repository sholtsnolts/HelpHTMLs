Dynamic Edit Command

Displays the dimensions for the selected feature so you can perform the following operations:

  • Select a dimension and change the dimensional value

  • Drag a dimension handle to reposition a dimension or the dimension text

  • Drag a profile element's handle to a new position to change the size or shape of the profile

These operations are performed without rolling back the model to the selected feature, which allows you to see the effect of the changes on the completed model.

To add or delete profile elements, dimensions, and relationships, you must open the profile window using the Edit Profile command.
