Suppress a feature using the Variable Table

  1. In Feature PathFinder, select the ordered feature you want to suppress.

  2. Right-click to display the shortcut menu, then click Add Suppression Variable.

  3. Choose Tools tab→Variables group→Variables command.

  4. In the Variable Table, find the suppression variable you added. The variable name ends with the characters _SUPPRESS.

  5. In the value column, type a number other than zero (0), then press Enter to suppress the feature.

Tip:

  • A value of zero (0) for a suppression variable specifies that you want the feature unsuppressed.

  • Suppression variables allow you to suppress and unsuppress features using an external spreadsheet.

  • Suppression variables are also available for assembly relationships and assembly features.
