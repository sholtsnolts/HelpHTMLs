Coplanar command


Modifies the position and orientation of one or more selected faces so that they are coplanar related to a target face on the part. You can use the options on the command bar to specify how you want the selected face to geometrically relate with the target face.

Makes the seed face (A) coplanar with the target face (B).

The coplanar relationship is persisted by default. You can turn the persist option off while in the command.

You can choose the coplanar command first. Then select the seed face(s) and target face.

You can also create the seed select set first and then choose the coplanar command. Then select the target face.
