Compare models

To compare two 3D models, follow these steps. For more information about the comparison, display, and report options, see the Compare Models dialog box.

Note:

In the Teamcenter-managed environment, to compare revisions of the same document, set the SEEC_Enable_MultipleRevisions preference to True.

  1. Open a part, sheet metal, or assembly file.

    Note:

    You also can select this command without having a document open, and then use the Browse buttons to locate the files to compare in the Compare Models interface.

    However, opening a file before selecting the command displays any open files in the Reference model or Working model pull down list on the Compare Models dialog box.

  2. From the File menu, choose Tools→Compare Models .

  3. In the Compare Models dialog box, select the Reference Model and the Working Model.

    The working model is compared to the reference model. In the Teamcenter-managed environment, both files are loaded to your local cache.

  4. The comparison uses the base coordinate system as the default.

    • If the parts are already aligned, skip this step.

    • If the two models have different coordinate systems, select the Align Parts option.

  5. Click Compare.

    When the comparison analysis is complete, the results display in the viewport on the right. You can switch between a Geometry Comparison view or a Properties Comparison of the results.

    You can expand any single viewport by clicking the button in the top-right corner. Click to return to the original arrangement of three views.

  6. Use the options in the Volume Changes section to control the results display.

  7. To save the results, click Generate Report. Specify a report format, location, and name.

    The geometry comparison results, property comparison results, and the options used to generate the comparison are included in the report. Reports are saved to an unmanaged location.
