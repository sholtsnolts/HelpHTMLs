Break a feature into a set of faces

  1. In the graphics window or PathFinder, select one or more features that are part of a set, such as a hole or a pattern occurrence.

  2. Right-click to display the shortcut menu, then choose Break.

Tip:

  • When you break a selection, it is changed into a face set, which is a collection of faces.

  • When you break a hole instance, you can no longer edit the face set using the original hole attributes, such as diameter, depth, and so forth. You can use the steering wheel to edit the individual faces, or add dimensions to the face edges and edit the dimensions.
