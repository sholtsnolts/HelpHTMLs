Detach a feature or set of faces

  1. Select one or more faces in the graphics window, or a feature in PathFinder.

  2. Right-click to display the shortcut menu, then click Detach.

Tip:

  • You can use the steering wheel to move the detached faces or feature to another location on the model, then use the Attach command to reattach them to the model.

  • You can also detach faces using the Detach option on the Move command bar when you are moving or rotating faces or features.
