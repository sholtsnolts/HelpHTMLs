Separate a feature from a feature set

  1. In the graphics window or PathFinder, select one or more hole instances that are part of a set of holes.

  2. Right-click to display the shortcut menu, then choose Separate.

Tip:

When you separate hole instances from a larger set of holes, you can edit the hole attributes separately from the original set.
