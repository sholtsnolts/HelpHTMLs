Compare Models dialog box

Configure Comparison

Defines the two models to compare.

Note:

In the Teamcenter-managed environment, to compare revisions of the same document, set the SEEC_Enable_MultipleRevisions preference to True.

Reference Model

The working model is compared to the reference model. You can Browse to the model location.

If the part is a family of parts, The Family of Parts field is enabled so you can choose the member.

Working Model

Model to check for changes from the reference model. You can Browse to the model location.

If the part is a family of parts, the Family of Parts field is enabled so you can choose the member.

Tolerance (%)

Default tolerance value is 0.1% total volume of the reference model.

When a comparison is performed, before displaying the changes, the ratio of ‘volume added’ to ‘total volume’ is checked. This value is compared with the tolerance that is specified as the Tolerance value. The change is displayed only if it is larger than the Tolerance value.

Align Parts

Aligns models based on selected coordinate systems. If this option is not selected, the models are aligned to the Base coordinate system. If custom coordinate systems exist and you want to use them, select the Align option and select coordinate system from the list for each model.

Display Options

Defines how the comparison results display.

Background color

Sets the background color for the viewports.

Volume Changes

Defines what volume changes to display.

Volume Changes

Defines what type of volume changes to display, and the color to use for each.

Added volume

Displays the volume added to the reference model. Choose the color for added volume. The default color is blue.

Note:

For an assembly comparison, this represents components added. Volume added or removed to any component is also shown. Assembly features are not detected in an assembly comparison. See the comparison report for details.

Removed volume

Displays the volume removed from the reference model. The color is White (glass). This color cannot be changed.

Note:

For an assembly comparison, this represents components removed. Volume added or removed to any component is also shown. Assembly features are not detected in an assembly comparison. See the comparison report for details.

Volume common to both models

Displays the volume common to both models. Choose the color for common volume. The default color is green.

Changed position
Note:

For an assembly comparison, this represents components that have changed position. See the comparison report for details. The default color is red.

View Manipulations

Viewports are kept in synchronization for view manipulations (Zoom, Fit, Pan, and Rotate).

Options

Defines the reporting parameters.

Generate Report

Displays the Save Comparison Report dialog box.

Report Type

Specifies the type of report to create (HTML, 3D PDF, or Word).

Report folder path

Defines the location to save the report. The report can only be saved to an unmanaged location on disk.

Open report after saving

Use this option to automatically show the report after it is saved.

Save Report

Generates the report.

Close

Closes the dialog box.

Report Options

Displays the Report Options dialog box.

Unit

Sets the units in the report.

Linear

Sets the length units from the list.

Angular

Sets the angular units from the list.

Views

Includes selected views in the report.

  • Front View

  • Top View

  • Left Side View

  • Back View

  • Bottom View

  • Right Side View

Plot

Plots views as native.

Compare

Starts the comparison analysis.

Viewports

Displays the compared models and results.

Click to show only that viewport in the dialog box.

Click to show all three viewports in the dialog box.

Reference Model:

Displays the reference model and the document name, including file extension.

Working Model:

Displays the working model and the document name, including file extension.

Results

Displays the comparison results.

Note:

This viewport is blank until a comparison is made.

Geometry Comparison

This tab in the viewport displays the model showing added volume, removed volume, and volume common to both models. The Volume Changes options control the display.

Properties Comparison

This tab in the viewport displays the following properties for the compared models:

  • Summary Information

  • Extended Summary Information

  • Document Summary Information

  • Custom

  • Mechanical Modeling

  • Physical Properties

  • Center of Gravity

  • Center of Volume

  • Global Moments of Inertia

  • Principal Moments of Inertia

  • Principal Axes

  • Radii of Gyration
