Tangent command


Makes the seed face(s) tangent to the target face.

The tangent relationship is persisted by default. You can turn the persist option off while in the command.

You can choose the tangent command first. Then define the select the seed and target faces.

You can also create the seed select set first and choose the tangent command. Then select the target face.

Note:

The result must be a theoretical tangency. If the result produces two faces that have an actual touching tangency, an error message appears. Message: The model could not be modified.
