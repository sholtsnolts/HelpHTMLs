2D Nesting command

Use the Tools tab→Environs group→Solid Edge 2D Nesting command to reduce fabrication costs for your flat pattern and 2D part designs using the Siemens Solid Edge 2D Nesting automatic nesting software. Nesting finds the most efficient layout for two-dimensional cutting on sheet metal stock and on other materials, including plastic, wood, fabric and textiles. It optimizes the arrangement of 2D shapes along a single cutting plane to minimize partial sheets and wasted material.

Solid Edge 2D Nesting is integrated with Solid Edge CAM Pro, or you can export it to another computer-aided manufacturing (CAM) system for computer numerical control (CNC) programming. 2D Nesting shares data with Solid Edge part, sheet metal, and drafting file formats (PAR, PSM, and DFT), as well as DXF and DWG.

For information on how to use Solid Edge 2D Nesting, click the Help button in the upper right corner of the Solid Edge 2D Nesting application window.
