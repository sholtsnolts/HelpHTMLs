Publish to GrabCAD

GrabCAD helps teams of engineers, suppliers, and clients to manage and share CAD. Use the Publish to GrabCAD command to send Solid Edge models to your project folders on the GrabCAD website.

Do the following to integrate the GrabCAD addin:

  • Visit grabcad.com to create your GrabCAD account. You must have an active GrabCAD membership and login, and you must have defined a project folder on the GrabCAD website to use the integration.

  • Login to your webkey account and download the GrabCAD addin:

    https://download.industrysoftware.automation.siemens.com/solid_edge/GrabCAD/.

    Run the setup executable file to install and register the addin.

  • Load the GrabCAD addin: Load a registered Add-In.

Registered addins are displayed in the Add Ins ribbon, though you might choose to use the Customize command to add the commands in other locations.
