Place a coordinate dimension using a coordinate system

User defined coordinate systems can be created in the model using the Coordinate System command. Coordinate dimensions can reference a user defined coordinate system or the base coordinate system.

  1. Perform the following steps before you place the dimensions.

    1. Right-click the drawing view and choose Properties.

    2. On the General tab (Drawing View Properties dialog box):

      1. From the View Coordinate System list, select a coordinate system.

      2. From the Axis list, choose a dimension axis.

        When placed, the coordinate dimension lines are displayed perpendicular to this axis. Refer to the examples, below.

  2. Choose the Coordinate Dimension command .

  3. On the Dimension command bar, from the Orientation list, select Use Coordinate System.

  4. Click an element that you want to be the dimension origin.

  5. Click to place the common origin.

    In the examples below, this is the 0 origin label, which is displayed on the dimension axis selected in step 1.

  6. Click an element that you want to measure.

  7. Click to place the dimension.

Tip:

  • You can display a coordinate system or center of mass in a drawing view using the Display tab (Drawing View Properties dialog box).

  • By default, the coordinate system is displayed as point in the drawing view. You can define a custom coordinate system by creating the coordinate system graphics and saving them as a block. For more information, see Define a custom center of mass or coordinate system for a drawing.
