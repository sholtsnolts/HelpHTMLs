Change a dimension axis

You can change the orientation of Distance Between, Angle Between, Symmetric Diameter, or Coordinate Dimensions placed using a dimension axis.

  1. Select a dimension you want to change.

    The dimension axis is highlighted and displayed as a dashed line.

  2. On the Dimension command bar, click the Dimension Axis button .

  3. Click a line that you want to set as the new dimension axis.

    You can select a line on the model, a line on the drawing geometry, or a centerline annotation.

  4. A dialog box is displayed for you to confirm the change. Do one of the following:

    • Click Accept to change only the single dimension.

    • Click Accept All to change all dimensions that were placed using the highlighted dimension axis.

    • Click Cancel.
