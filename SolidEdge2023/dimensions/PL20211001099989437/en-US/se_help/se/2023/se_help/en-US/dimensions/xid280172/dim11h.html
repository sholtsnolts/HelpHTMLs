Place a dimension group

  1. Click one of the following buttons:

    • Distance Between

    • Angle Between

    • Symmetric Diameter

    • Coordinate Dimension

  2. Click an element that you want to be the dimension origin (A).

  3. Click an element that you want to measure (B).

  4. Click to place the dimension (C).

  5. Click another element that you want to measure (D).

  6. Click to place the dimension in the dimension group (E).

Tip:

  • The first dimension that you place in a dimension group is the origin dimension.

  • The Coordinate Dimension button places coordinate dimensions. Coordinate dimensions can only be placed in a group. You can place coordinate dimensions in any order and on any side of the elements that you want to dimension.
