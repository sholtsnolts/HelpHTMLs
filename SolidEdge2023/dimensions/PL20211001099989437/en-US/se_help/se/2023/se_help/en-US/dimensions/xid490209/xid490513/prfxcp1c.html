Copy Attributes command

The Copy Attributes command can be used to do the following:

  • Copy the prefix, suffix, superfix, subfix strings, dimension display type, or tolerance string from one dimension to another.

  • Copy the style properties associated with a dimension or annotation to another dimension or annotation.
