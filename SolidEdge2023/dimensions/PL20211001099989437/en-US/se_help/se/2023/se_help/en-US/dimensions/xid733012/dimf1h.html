Format a dimension or annotation

  1. Select a dimension or annotation.

  2. Do one of the following:

    • If you are placing a new dimension or annotation, set the options you want on the command bar or in the displayed dialog box before you click to place it.

    • To edit the properties of one or more existing dimensions—

      1. Right-click a dimension and choose Properties from the shortcut menu.

      2. On the Properties dialog box, set the options you want.

        For example, you can change text size, terminator style, round-off, or symbol orientation.

      3. Click Apply to see the effect of your change immediately.

        You can use the Undo command to undo the result of your property edit.

    Tip:

    You also can change a dimension style or a text style globally with the Styles command on the View tab, in the Style group.
