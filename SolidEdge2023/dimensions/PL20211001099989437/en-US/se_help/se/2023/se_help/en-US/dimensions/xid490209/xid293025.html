Round-off dialog box

The Round-off dialog box provides a single location for adjusting the round-off formatting of primary and secondary units in linear dimensions, angular dimensions, and tolerance values. You can open the Round-off dialog box using the Advanced Round-off button on the Dimension command bar and on the Smart Dimension command bar.

You also can set and modify these settings on the Units tab and the Secondary Units tab in the dimension properties and the dimension style.

Primary linear

Specifies the round-off of primary units in a linear dimension.

Primary linear tolerance

Specifies the round-off of primary units in linear dimension tolerance values.

Secondary linear

Specifies the round-off of secondary units in linear dimensions.

Secondary linear tolerance

Specifies the round-off of secondary units in linear dimension tolerance values.

Angular

Specifies the round-off for angular dimension values.

Angular tolerance

Specifies the round-off for angular dimension tolerance values.
