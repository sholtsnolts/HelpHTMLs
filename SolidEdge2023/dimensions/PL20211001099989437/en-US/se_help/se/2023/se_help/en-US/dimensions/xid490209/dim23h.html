Set dimension units

You can set the units, label, round-off, number of subunits, and delimiter options for primary dimension units to be the same as, or different from, secondary dimension units. This is useful when you want to show dimensions in both metric and English units.

  1. Select a dimension.

  2. On the shortcut menu, click Properties.

  3. Set primary dimension units—In the Dimension Properties dialog box, on the Units page, set the options you want to use for the primary dimension unit.

    Example:

    To display a unit label in the dimension text, such as mm or ft-in, you can type the label text in the Unit label box.

    Example:

    In an ANSI document with Imperial units, you can format the fractional value in a dimension using the options in the Fractional Display Settings section: Stacked, Skewed, or Linear.

  4. Set secondary dimension units—In the Dimension Properties dialog box, on the Secondary Units page, set the options you want to use for the secondary dimension unit.

    Example:

    To use a different delimiter for secondary units, clear the Match Primary Units check box, and then click the delimiter type you want to use.

Tip:

  • You can save the dimension unit settings in a dimension style using the Styles command.

  • You can use the Maximum subunits option on the Units tab (Dimension Style and Dimension Properties) to specify a cutoff value for when a dimension or measurement is displayed in inches and when it is displayed in feet-inches.
