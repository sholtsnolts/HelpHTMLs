Using the Relationship Assistant to validate 2D dimensional constraints

The Relationship Assistant command helps you finish a profile or sketch, or make it fully parametric.

After applying all critical dimensions and relationships to the geometry, you can use the Relationship Assistant command to apply any missing geometric or dimensional relationships to help fully constrain the model. It is also a good idea to use the Show Variability option to check for required degrees of freedom in the profile.

You can also use the command to show you how many additional relationships are required and how the shape can change based on the current relationships and dimensions

Showing variability

To determine how many additional relationships are needed and how the profile or sketch can change, select the Relationship Assistant command, drag a fence around the profile, and then click the Accept button on the command bar. You can then select the Show Variability button on the command bar to display the number of relationships needed. A temporary display of the profile using the highlight color is also shown to illustrate one possibility of how the profile can change. You can use the Show Variability button repeatedly to see other variations.

To learn how, see the help topic, Show how a profile or sketch can change.
