New Revision dialog box (Dimension Tracker)

The New Revision dialog box updates balloon labels associated with changed items selected in the Dimension Tracker list.

Label

Specifies a revision name or identifier for the balloon labels on the drawing. You can type any alphanumeric characters.

OK

Applies the label name to the balloons on the drawing. Deletes the selected changes from the Dimension Tracker list.

Cancel

Closes the dialog box without changing the balloon label text or removing the changes from the list.
