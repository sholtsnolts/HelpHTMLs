Copy a dimension prefix or style

  1. Choose the Copy Attributes command from the Dimension group on the Home tab or the Sketching tab.

  2. On the Copy Attributes command bar, from the Copy What? list, specify the type of data you want to copy.

    • Between dimensions, you can copy:

      • Prefix, as defined in the Dimension Prefix dialog box.

        You can copy prefix, suffix, superfix, and subfix strings.

      • Type, as selected from the Dimension Type list on the dimension command bar.

        You can copy dimension display types and tolerance strings. See the help topic, Set the dimension type.

      • Prefix and Type

    • Between dimensions and annotations, you also can copy:

      • Style properties, as defined in the Dimension Style dialog box and in the properties dialog box for the selected dimension or annotation.

  3. Click a dimension or annotation you want to copy from.

  4. Click a dimension or annotation you want to copy to.

Tip:

  • When you select the Style option from the Copy What? list, you can copy the style between dimensions, between annotations, and between a dimension and an annotation.

  • Annotations use the same style as dimensions.
