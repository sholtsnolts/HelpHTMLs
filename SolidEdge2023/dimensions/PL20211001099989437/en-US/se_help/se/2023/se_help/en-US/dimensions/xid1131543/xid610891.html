Break Alignment Set command

You can use the Break Alignment Set command to dissolve the selected dimension alignment sets (A) so that each dimension can move independently from the others (B). To use the command, right-click a dimension that is part of an alignment set and select Break Alignment Set.

Note:

This command is not enabled for dimensions that are not part of an alignment set.
