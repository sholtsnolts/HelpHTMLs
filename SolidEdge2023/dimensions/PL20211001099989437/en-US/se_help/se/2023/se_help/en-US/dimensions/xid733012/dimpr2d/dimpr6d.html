General tab (Dimension Style and Dimension Properties)

The General tab sets the general properties for dimensions in the Draft environment. It is shared by the Dimension Style dialog box and the Dimension Properties dialog box.

Color

Sets colors for a driving, driven, or error dimensions and connectors.

Driving Dimension

Sets the color used for driving dimensions.

Driven Dimension

Sets the color used for driven dimensions.

Error Dimension

Sets the color for error dimensions and annotations. Error dimensions and annotations are not attached to elements. Error dimensions sometimes occur when you insert foreign data into a document.

Connector

Sets the color for connectors used to connect blocks and other 2D elements and objects in flow diagrams and schematic diagrams.

Scale Mode

Sets the dimension scale mode to automatic or manual.

Automatic

Determines a correct dimensional value based on the scale set in the drawing view.

Manual

Scales a dimensional value. The scale value determines the dimensional value.
