Add and remove jogs on a dimension

You can add and remove jogs on the projection lines of 2D dimensions.

  1. Choose the Select command .

  2. To insert a jog, do the following:

    1. Select an existing dimension (1) to display the dimension edit handles.

    2. Position the cursor over the dimension projection line where you want to insert the jog, and Alt+click (2).

      A jog segment is inserted.

      Example:

  3. To adjust the position of a jog segment, do the following:

    1. Select a dimension with a jog segment (3).

    2. Drag the handle to reposition the jog (4).

    3. Click in space to update the dimension.

Tip:

  • You can remove a jog by doing this:

    1. Select the dimension.

    2. Alt+click the dimension jog handle.

  • You can add multiple jogs to a dimension projection line.

  • You can remove all of the jogs on an existing dimension using the Jog button on the Dimension command bar.
