Specify properties for auto dimensioning

You can specify properties for dimensions before generating them. You can also save the specified settings to quickly reuse them.

Procedure

  1. Choose the PMI tab→Dimension group→Auto Dimension command.

    The Auto Dimension Properties dialog box appears along with the Auto Dimension dialog box. If not, you can click Properties in the Auto Dimension dialog box.

    Note:

    To always display the Auto Dimension Properties dialog box when you run the command, select the check box Show this dialog when the command begins. When cleared, select Properties in the Auto Dimension dialog box.

  2. Click the General tab and specify the following:

    • Under Select Features for Location Dimensions, select the features to be considered while generating the dimensions.

    • Under Type of Location Dimensions, select the type of dimensions to be generated.

    • Under Feature Size Dimensions, specify whether to generate size dimensions. To add the quantity of similar-sized holes while generating the dimensions, select the required prefix from the list. For more information, see Define smart feature properties in the Dimension style.

      For holes having coplanar axes, a single dimension is generated. When selected, the considered holes and the datum plane are highlighted.

    You can enter a name in the Saved Settings box to name a group of specified properties and click Save to quickly reuse them in the future. While auto-dimensioning, you can access these saved settings under Properties.

  3. Click the Standard Tolerance tab and specify the following:

    • Under Distance Dimensions, select the tolerance type and the tolerance values for distance diameter dimensions. By default, the Unit Tolerance is selected with the tolerance values + 0.5 and - 0.5.

    • Under Simple Hole Diameter, select the tolerance type and the tolerance values for simple hole diameter dimensions. By default, the Unit Tolerance is selected with the tolerance values + 0.5 and - 0.5.

    • Under Counterbore Diameter, select the tolerance type and the tolerance values for counterbore diameter dimensions. By default, the Feature Callout is selected as it lets you give more information as specified in the Feature Callout tab. For more information, see Feature Callout tab.

    • Under Countersunk Diameter, select the tolerance type and the tolerance values for countersunk diameter dimensions. By default, the Feature Callout is selected as it lets you give more information as specified in the Feature Callout tab. For more information, see Feature Callout tab.

  4. To create the datum frame annotations, click the Datum tab and then select the check box Create Datum Frame annotation.

    You can select the shape in which the datum name is displayed and choose to display the dashed datum text.

    For more information, see Text and Leader tab.

  5. Click the Feature Callout tab and specify the required specifications. For more information, see Feature Callout tab.

  6. Click the Smart Depth tab and specify the required specifications. For more information, see Smart Depth tab.
