Arrange dimensions automatically on a drawing

  1. In the Draft environment, choose Home tab→Dimensions group→Arrange Dimensions .

  2. Select dimensions by:

    • Clicking individual dimensions.

    • Clicking the drawing view to select all dimensions.

    • Dragging a box around the dimensions.

    The color of the dimension line changes when selected.

  3. On the command bar, type a value in the Stack Pitch box.

    This defines the distance between the dimension lines in a stacked dimension group.

  4. Do one of the following:

    • To create a set from the dimensions that are arranged, select the Create Alignment Set button . You can modify all of the dimensions in an alignment set at once.

    • When you want to arrange the selected dimensions, but you do not want to group them into an alignment set, deselect the Create Alignment Set button .

  5. Click Accept to use the current input or Deselect to clear the selection.

Tip:

  • To set the distance between the innermost dimension line in the stack and the dimensioned geometry, use the Initial stack distance option on the Lines and Coordinate tab (Dimension Style and Dimension Properties).

  • While selected symmetric diameter dimensions are arranged, they will not be included in the alignment set.

  • Turning Create Alignment Set on or off overrides alignment sets created with the Maintain Alignment Set command in the Dimension group on the ribbon.
