Add and edit dimension text


You can add prefix, suffix, superfix, and subfix text and supplementary information to a dimension value using the options in the Dimension Prefix dialog box. You can use this dialog box while you place or modify a dimension.

Add dimension text

  1. Click the dimension command you want to use, such as Smart Dimension or Dimension Between.

  2. On the Dimension command bar, click Prefix .

    The Dimension Prefix dialog box is displayed.

  3. In the Dimension Prefix dialog box, click in the Superfix, Prefix, Suffix, Subfix, or Subfix 2 box, and then do any of the following:

    • Add plain dimension text—Type the text as you want it to appear.

    • Add a dimension symbol or derived dimension value—Click a symbol button to insert a frequently used symbol. To find other symbols or to insert a value derived from property text, click Select Symbols and Values to open the Select Symbols and Values dialog box.

    • Type one or more hole count property text codes (%QC, %QP, %QA) directly to extract the quantity of coplanar holes, parallel holes, and all holes with the same parameters in the Hole Options dialog box. For more information, see the Feature References table in Property text codes.

      Note:

      If you are extracting hole count properties for legacy draft files, you must force the drawing views to update using Ctrl+Shift+Update Views command.

    • Add other types of property text—You also can extract property text by copying and pasting a property text string, for example, %{Accuracy|GP}, or by typing the string directly into a box.

    • Format an output value—You can format the value derived from a property text string. Click a property text string in a dimension text box, such as %TS, and then select the Format button. The type of formatting you can apply depends on your cursor position when you open the Format Values dialog box. For more information, see the help topic, Format property text values.

  4. Click Apply and OK to save the dimension text and close the Dimension Prefix dialog box.

  5. On the drawing, click to place the dimension.

    6. The dimension text is displayed with the dimension.

Tip:

  • Once you define the dimension text you want to use, you can continue to place dimensions using the same text, or you can open the Dimension Prefix dialog box and modify the text before placing the next dimension.

  • To save the dimension text and formatting that you entered, type a name in the Saved settings box and then click Save. You can quickly apply this saved setting to new dimensions or to modify existing dimensions by selecting the name from the Saved Settings list on the dimension command bar.

  • To control whether dimension text is shown with a dimension or not, use the Enable Prefix button on the Dimension command bar. When the Enable Prefix button is selected, dimension text is shown.

  • You can use the Copy Attribute command to copy a dimension prefix or style.

Modify dimension text

  1. Select a dimension.

  2. On the Dimension command bar, click Prefix .

  3. In the Dimension Prefix dialog box, add or edit the content in the dimension Superfix, Prefix, Suffix, Subfix, or Subfix 2 boxes.

    For more information and examples, see the help topic, Dimension Prefix dialog box.

  4. Click Apply to change the dimension that you selected.
