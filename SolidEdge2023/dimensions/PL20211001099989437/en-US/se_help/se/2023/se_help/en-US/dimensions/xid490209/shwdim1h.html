Display dimension formulas, names, or values

To

Do this

Display the formulas for all dimensions.

Select the dimension, then on the shortcut menu, click Show All Formulas.

Display the names for all dimensions.

Select the dimension, then on the shortcut menu, click Show All Names.

Display the dimensional values for all dimensions.

Select the dimension, then on the shortcut menu, click Show All Values.
Tip:

You can also double-click a dimension to display the Edit Formula command bar. You can then rename or define a formula for a dimension.
