Update retrieved dimensions and annotations

You can use the Update Retrieved Dimensions command to identify and update all retrieved PMI dimensions and annotations in your drawing views, even when those changes do not affect the model geometry. (The Update Views command, on the other hand, only updates retrieved dimensions and annotations when they are deleted or when their size value is changed.)

  1. Open the draft document containing the retrieved dimensions and annotations.

  2. Open the Solid Edge Options dialog box.

  3. On the Annotations page, select the Maintain associativity for retrieved dimensions option.

    Note:

    Selecting this draft file preference ensures that changes to the text and properties of retrieved PMI cause the dimensions and annotations in the drawing views to be marked as out-of-date. Examples include changes to dimension type, tolerance, or prefix text. Annotation examples include changes to leaders, arrows, text, and other properties.

  4. On the working sheet, choose the Home tab (or Sketching tab)→Update Retrieved Dimensions command.

    The Dimension Tracker dialog box is displayed. This identifies the specific dimensions and annotations that changed and were updated.

    Example:

    Textual changes to PMI are flagged as Content changed in the Dimension Tracker dialog box.

    For more information, see Tracking dimension and annotation changes.
