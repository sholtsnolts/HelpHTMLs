Coordinate dimension handles

The following handles are available for manipulating coordinate dimensions:

Use this handle

To

(1) Geometry connection handle

Change the parent element that the dimension is connected to. Press Alt while you drag it.

(2) Projection line handle point (the point on the projection line that is closest to the text)

Reposition all dimensions in the alignment set by adjusting the length of the projection lines.

Note:

This produces the same result as dragging the dimension line (4), but this handle is available even when the dimension line is not shown, as per the ANSI standard.

(3) Dimension text

  • Move the selected dimension text along the projection line.

  • Use Alt+drag to move the text perpendicular to the projection line. Adjacent dimensions also move to preserve spacing. Jogs are added automatically to prevent overlap.

(4) Dimension line

  • Reposition all dimensions in the alignment set by adjusting the length of the projection lines.

  • Use Alt+drag to adjust the position of a single dimension.

(5) Dimension line handle point (the point opposite the terminator)

Change the length of the selected dimension line.

(6) Jogged projection line

  • Change the shape of the selected projection line by dragging the jog points.

  • Move the selected dimension along the dimension line. Adjacent dimensions also move to preserve spacing. Jogs are added automatically to prevent overlap.

    For more information, see Adding and removing jogs on coordinate dimensions.
