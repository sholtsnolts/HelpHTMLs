Retrieving dimensions and annotations from a model

Showing model PMI in drawing views

You can copy dimensions and annotations from a part, sheet metal, or assembly model to a drawing view using either of the following methods:

  • Use the Retrieve Dimensions command to copy dimensions and annotations from the model to an existing orthogonal or section drawing view. The Retrieve Dimensions command copies PMI dimensions and annotations as well as sketch dimensions and annotations. To learn how to do this, see the help topic, Retrieve dimensions and annotations from the model.

  • Use the Drawing View Wizard to generate a drawing view from any previously defined PMI model view created with the PMI tab→Model Views group→View command. You can use this method to place a dimensioned isometric drawing view on the drawing. To learn how to do this, see the help topic, Create a PMI drawing view.

Updating model PMI in drawing views

The Retrieve Dimensions command copies model PMI dimensions and annotations from the model to drawing views one time.

There are two ways to update model PMI in drawing views.

  • If there are changes to model geometry, then the drawing views display the gray border that indicates they are out-of-date. You can use the Update Views command to update the part view and the retrieved dimensions. For example, if you change the size of a hole in your part, the retrieved dimension for the hole in the part view will update to the new value.

  • You can set an option to maintain associativity. This causes the retrieved PMI to display as out-of-date on the drawing, even when the changes to the PMI do not affect the size of the model. When you see this visual indicator, you can use the Update Retrieved Dimensions command to identify and update all of the changes to dimensions and annotations, including changes to content and properties.

    To learn how to do this, see Update retrieved dimensions and annotations.
