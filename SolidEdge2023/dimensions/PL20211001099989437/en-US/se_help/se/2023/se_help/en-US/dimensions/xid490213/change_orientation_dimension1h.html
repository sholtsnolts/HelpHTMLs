Change the orientation of a dimension

You can change the default, automatic orientation of a linear dimension placed with the Smart Dimension command and Distance Between command.

Change the orientation of a dimension

  1. Choose Home tab→Dimension group→Smart Dimension command and select a line (1).

    A dimension will display dynamically parallel to the line you selected.

  2. Select another orientation type from the orientation list, such as Horizontal/Vertical.

    The dimension now aligns with the vertical axis (2).

    Example:

    A linear smart dimension placed using the Horizontal/Vertical orientation:

    Tip:

    The PromptBar displays the key you can use to change the orientation, and the status bar at the bottom of the application window displays the current orientation.

  3. To change the orientation during edit do the following:

    • Select the dimension

    • Choose another orientation from the list on the menu.

Apply orientation to a dimension group

  1. Select any of the dimensions that belong to the group you want to edit (1).

  2. Select the By 2 Points orientation (2).

    All the dimensions of the group move to the new orientation.
