Connector Table command

Uses harness information from the connector drawing view to create a connector table. The table contains a default set of columns, which you can customize by adding and reordering columns or rows. If you customize the table, you can use the Save Settings option on the General tab of the Connector Table Properties dialog box to save the customized setting. You can use the Save Settings option on the command bar to select settings you have saved.

The connector table is associative to the connector drawing view and assembly. If the referenced assembly is updated, the connector table becomes out-of-date.
