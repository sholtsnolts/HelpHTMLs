Assign Terminals dialog box

Library part name

Specifies the name of the component containing the terminals being assigned. This is the name that the Harness Wizard matches against the components file to find components in the harness assembly model. It is also the name that is used for harness reports to describe wire and cable terminations.

Terminals

Displays the name and status for the terminals being assigned.

New

Adds a new row to the Terminals table.

Set

Selects a circular edge to assign to the terminal.

Delete

Deletes the selected row from the Terminals table.

Clear All

Deletes the component name and the information from all rows in the Terminals table.

Create PMI

Creates PMI callouts for the component name and terminal names.
