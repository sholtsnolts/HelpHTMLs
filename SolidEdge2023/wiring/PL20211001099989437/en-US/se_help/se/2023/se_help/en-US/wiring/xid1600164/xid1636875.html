Create a board based on the data imported from an existing ECAD file

  1. In Solid Edge, open an assembly template.

  2. Click PCB→New Board Assembly.

  3. On the New Board Assembly dialog box, enter information for the PCB assembly.

    • Project Name

    • PCB Template

    • Board Template

    • Project Location

    Click the Browse button to navigate to the appropriate template of project folder location.

  4. Under Board Options, click Import a board from ECAD data.

  5. Click Finish.

    The ECAD Collaborator dialog box displays so you can import the appropriate idx file to import.
