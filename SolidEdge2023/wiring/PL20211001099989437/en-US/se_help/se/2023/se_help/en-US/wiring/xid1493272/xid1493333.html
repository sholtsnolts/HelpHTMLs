Update Harness command

Use the Update Harness command to update the harness data to match the data sent from Solid Edge Wiring and Harness Design.

The Update Harness command compares the harness and Solid Edge Wiring and Harness Design information and updates any matching connection elements, such as wires, cables and splices. Solid Edge maintains paths used to route wires, cables, and bundles. Splice positions are also retained.

Updating harness information

Use the Update Harness command to update the harness data to match the data sent from Solid Edge Wiring and Harness Design.

The Update Harness command compares the harness and Solid Edge Wiring and Harness Design information and updates any matching connection elements, such as wires, cables and splices. Solid Edge maintains paths used to route wires, cables, and bundles. Splice positions are also retained.

The system deletes any elements that exist in the harness, but not in Solid Edge Wiring and Harness Design, and adds any elements that exist in Solid Edge Wiring and Harness Design, but not in the harness.

In the following example, the Solid Edge harness contains six wires (W1-W6), while Solid Edge Wiring and Harness Design contains six wires, (W1-4W and W7and W8). When updated from Solid Edge Wiring and Harness Design, four wires (W1-W4), found in both Solid Edge harness and Solid Edge Wiring and Harness Design are updated, the two wires (W5 and W6), found only in Solid Edge harness are deleted, and the two wires (W7 and W8), found only in Solid Edge Wiring and Harness Design are added.

Assigning multiple occurrences of components

The Assign Occurrence/Component dialog box appears if you select the Update Harness command in any of the following conditions:

  • Multiple occurrences of components

  • Unassigned terminals

  • Unassigned components

  • Solid Edge Wiring and Harness Design component ID is assigned to unmatched Solid Edge component ID

You can manually assign these components the first time to create the wires. After the wires are created, they are updated and the dialog box does not appear the next time you update.
