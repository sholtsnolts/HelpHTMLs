Create a nailboard view

  1. Choose Diagram tab→Views group→Nailboard .

  2. In the Select Model dialog box, select an assembly document that contains a wire harness.

  3. Click Open.

  4. In the Nailboard Options dialog box set options as needed, and then click OK.

  5. Click to place the nailboard view.

Tip:

  • You can change the nailboard view scale by:

    • Choosing a different scale from the Scale list.

    • Setting the nailboard view scale to match the current sheet scale. To do this, click the Set Sheet Scale button.
