Filter by Height

Shorter than

Displays PCB items shorter than the specified height.

Taller than

Displays PCB items taller than the specified height.

Filter by Layer

On the Top Layer of the Board

Displays PCB items that are placed on the top of the board.

On the Bottom Layer of the Board

Displays PCB items that are placed at the bottom of the board.

Filter by PCB item owner

That are associated with MCAD

Displays PCB items that have the Owner property set to MCAD.

That are associated with ECAD

Displays PCB items that have the Owner property set to ECAD.

Filter Copper data

Top layer

Displays copper data that are placed on the top layer.

Bottom layer

Displays copper data that are placed on the bottom layer.
