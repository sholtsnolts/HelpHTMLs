Assign Occurrence/Component dialog box

The dialog box appears if you select the Update Harness command in any of the following conditions:

  • Multiple occurrences of components

  • Unassigned terminals

  • Unassigned components

You can manually assign these components the first time, to create the wires. After the wires are created, they are updated and the dialog box does not appear the next time you update.

Components

Displays information about the components used to create the wire harness.

Assign Component

Changes the assignment of a component within the assembly.

Assign Occurrence

Changes the occurrence of a part within the assembly.

Populate Options

Displays the Harness Populate Options dialog box.

Populate

Populates components into the assembly.
