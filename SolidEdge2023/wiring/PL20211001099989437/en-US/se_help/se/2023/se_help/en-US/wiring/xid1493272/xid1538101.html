Configure Solid Edge Wiring and Harness Design connection properties

  1. In Solid Edge Wiring and Harness Design, do one of the following:

    1. Open an existing project.

    2. Create a new project.

  2. Click the Workflow→Configure button .

  3. On the Connection Properties dialog box, enter the appropriate host name in the Host field.

  4. Enter the appropriate port name in the Port fields for Solid Edge Wiring and Harness Design and Solid Edge.

    Note:

    The port number should be the same for both Solid Edge Wiring and Harness Design and Solid Edge.

  5. Click OK.
