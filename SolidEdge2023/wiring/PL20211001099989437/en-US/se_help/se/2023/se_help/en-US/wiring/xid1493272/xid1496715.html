Connection Properties dialog box

Saved settings

Lists the names of properties you have saved for the current connection.

Save

Saves the current settings to the name you type.

Delete

Deletes the current saved setting.

Same Machine

Specifies that Solid Edge and Solid Edge Wiring and Harness Design reside on the same machine.

Remote Machine

Specifies that Solid Edge and Solid Edge Wiring and Harness reside on separate machines.

Solid Edge Electrical

Host

Specifies the host name for the machine on which Solid Edge Wiring and Harness Design is loaded.

Port

Specifies the port number for the machine on which Solid Edge Wiring and Harness Design resides. The port number should match the port number for the machine on which Solid Edge is loaded.

Solid Edge

Port

Specifies the port number for the machine on which Solid Edge resides. The port number should match the port number for the machine on which Solid Edge Wiring and Harness Design is loaded.

Help

Displays online help.

Reset

Resets the dialog box options back to the default value.

OK

Dismisses the dialog box, accepting all input.

Cancel

Discards all input and dismisses the dialog box.
