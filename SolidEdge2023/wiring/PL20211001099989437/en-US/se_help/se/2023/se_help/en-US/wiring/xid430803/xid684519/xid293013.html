Connector Table command bar

Table Style

Lists and applies the available table styles.

See the Help topic, Table styles.

Auto-Callout

Automatically creates callouts for the selected connector drawing view. By default, Auto-Callout and leaders with breaklines are on. You can edit a callout after placing it. If you chose not to create a callout during the command, you can go back later and use the command to create callouts.

Place Table

Places the connector table on the drawing sheet.

Properties

Displays the Connector Table Properties dialog box.

Saved Settings

Lists and applies the names of table formats you have saved.
