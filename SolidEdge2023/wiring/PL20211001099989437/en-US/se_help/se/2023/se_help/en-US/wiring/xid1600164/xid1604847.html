Use ECAD Collaborator to import PCB data

With ECAD Collaborator you can import:

  • Baseline files

  • Proposal/incremental files

  • Response files

Import baseline files

  1. Click PCB→ECAD Collabarator.

  2. On the ECAD Collaborator dialog box, in the Collaborator Directory field, specify the folder where the IDX files reside.

    The IDX files stored in the specified directory are displayed in the Files field.

  3. In the Files list, select the IDX file you want to import.

    The PCB items for the selected the IDX file are displayed in the Collaboration Data area.

  4. Select or deselect the PCB items that you want to import.

  5. Click Apply.

    6. The PCB assembly is generated in Solid Edge PCB Collaboration. After the assembly is generated, you can select a PCB item and do such things as rotate or move the item and send the item from the top layer to the bottom layer of vice versa, and more.

Import proposal/incremental files

  1. Click PCB→ECAD Collabarator.

  2. On the ECAD Collaborator dialog box, in the Collaborator Directory field, specify the folder where the incremental IDX files reside.

    The IDX files stored in the specified directory are displayed in the Files field.

  3. In the Files list, select the IDX file you want to import.

    The PCB items within the IDX file that have been added, deleted, or changed are displayed in the Collaboration Data field and the properties for those items are displayed in the Properties field.

  4. Select or deselect the PCB items that you want to import.

  5. Click Apply.

    6. The PCB assembly is modified in PCB Collaborator.

    Note:

    For modifications, such as adding or changing holes or cutouts, only sheet metal features are supported. Solid Edge PCB Collaborator does not support hole or cutout changes to assembly features.

Import a response file

  1. Click PCB→ECAD Collabarator.

  2. On the ECAD Collaborator dialog box, in the Collaborator Directory field, specify the folder where the IDX files reside.

    The IDX files stored in the specified directory are displayed in the Files field.

  3. In the Files list, select the response IDX file you want to import.

    The PCB items within the IDX file that have been added, deleted, or changed are displayed in the Collaboration Data field and the properties for those items display in the Properties field.

  4. Select or deselect the PCB items that you want to import.

  5. Do one of the following:

    • To accept the changes, click Apply.

    • To reject the changes, click Abandon.
