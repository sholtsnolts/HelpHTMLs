New Board Assembly command

The New Board Assembly command is the starting point for ECAD/MCAD collaboration. Use the command to display the New Board Assembly dialog box so you can create the PCB board.

Once displayed you can use the dialog box to:

  • Create a new board based on the information you enter in the New Board Assembly wizard.

  • Create a board based on the data imported from an existing ECAD file.

You can also use the dialog box to specify templates for creating the assembly and board, as well as location for the project folder where the files associated with the project will reside.

After specifying the information for a new board, click the Next button to display a second dialog box where you can define the board size.

Note:

If you select the Import a board from ECAD data option, click the Next button to display the ECAD Collaborator dialog box to import the IDX file.
