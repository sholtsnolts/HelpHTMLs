Creating a wire harness automatically

The Harness Wizard command, located on the Home tab→Wizard group, automatically creates a wire harness. The wizard uses information stored in an imported net list file to populate the components and conductors in an assembly.

Note:

To access the Harness Wizard command you must be in the Electrical Routing environment.

Choose Tools tab→Environs group→Electrical Routing.

When you select the command, a wizard guides you through the process of defining information about the electrical routing.

To learn how to use the wizard, see Using step 1 of the Harness Wizard.

Note:

If you are working with 3D electrical parts, see Linking 3D Electrical parts in Wiring Harness
