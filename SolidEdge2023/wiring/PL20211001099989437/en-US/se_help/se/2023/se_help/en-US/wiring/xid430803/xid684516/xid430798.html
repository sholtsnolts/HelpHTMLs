Printing nailboard views

As with other drawing views, you can print nailboard views. You can use the Print or Print Drawings commands to print a single nailboard view to a single sheet of paper. However, for larger views that require you to print a single view at a 1:1 scale to multiple sheets in sections, we suggest you save the view to PDF format and print the PDF file. For more information, see the topic Print a nailboard view from Adobe Reader.
