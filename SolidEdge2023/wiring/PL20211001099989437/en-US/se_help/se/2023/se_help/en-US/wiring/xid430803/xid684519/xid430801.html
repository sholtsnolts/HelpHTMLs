Creating connector tables

You can use the Connector Table command to create a connector table based on the harness information from the connector drawing view. The harness information includes key attributes such as component name, terminal name, and wire id.

The table contains a default set of columns, which you customize by adding and reordering columns or rows. If you do customize the table, you can use the Save Settings option on the General tab of the Connector Table Properties dialog box to save the customized setting. You can use the Save Settings option on the command bar to select settings you have saved.

The connector table is associative to the connector drawing view and assembly. If the connector part or referenced assembly is updated, the connector table becomes out-of-date.
