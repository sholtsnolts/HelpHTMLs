Using realistic 3D components in board assemblies

Realistic 3D components create a realistic 3D model of the electrical components in the imported PCB assembly, which gives a more realistic view of the components in the assembly. Components that are represented in 2.5D are replaced with supported, standard 3D models that are available in the library. If no 3D model is available, a 2.5D representation of the component is created.

Note:

Some models are available in STEP (.stp) or ACIS (.sat) format. To use these models, you must use the Batch.exe file, located in the Solid Edge Version #\Custom\Batch\Bin folder to convert them to Solid Edge part files.

Ensure that the name of the 3D part file matches the value for the Package properties for the component in collaborator properties list.

You must change the orientation to ensure it matches the orientation of the library part in the ECAD software.

Once imported, you can manually change the 3D model assignments for the components.
