Assign Terminals command

Displays the Assign Terminals dialog box so you can specify a component name and make terminal assignments to a part or sheet metal model. Once you make these assignments, you can include the information in a net list file that is used to automatically generate a harness design.

Assigning terminals consists of two steps:

  • Assigning a component name and terminal name.

  • Selecting a square or rectangular edge of a cutout, circular edge or arc, for the terminal.
