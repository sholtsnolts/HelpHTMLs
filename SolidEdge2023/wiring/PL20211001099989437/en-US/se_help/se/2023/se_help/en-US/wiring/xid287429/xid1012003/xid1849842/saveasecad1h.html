Save component and connection information to ECAD format

Note:

The Save As ECAD command is only available once you add components or connections to the wire harness.

  1. On the File menu, point to Save As, then choose Save As ECAD.

  2. On the Save as ECAD dialog box, set Document to the document format you want to use.

  3. In the Component Document box, specify a name for the component document.

  4. In the Connection Document box, specify a name for the connection document.

Tip:

You can click Save As to specify the name, location, and format for the component and connection documents.
