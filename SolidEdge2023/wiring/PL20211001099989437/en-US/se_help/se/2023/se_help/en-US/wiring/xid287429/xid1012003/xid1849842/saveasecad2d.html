Save As ECAD dialog box

ECAD Application

Specifies the export format for the document.

Component document

Displays the location and name for the component document. You can click the Save As button to display the Save as (Components or Connections) dialog box, which allows you to specify the location and document name. You can save the document to .CMP or .CMP_XML format.

Connection document

Displays the location and name for the connection document. You can click the Save As button to display the Save as (Components or Connections) dialog box, which allows you to specify the location and document name. You can save the document to .CON or .CON_XML format.

Harness Topology document

Displays the location and name for the harness topology document. You can click the Save As button to display the Save as Harness dialog box, which allows you to specify the location and document name. You can save the document to .xml format. This option is only available if the ECAD Application option value is Siemens - Solid Edge Harness Design, Siemens - Solid Edge - Harness Topology - DSI, or Siemens - Capital HarnessXC.

Wiring X2ML document

Displays the location and name for the X2ML document. You can click the Save As button to display the Save as Wiring X2ML dialog box, which allows you to specify the location and document name. You can save the document to .xml format. This option is only available if the ECAD Application option value is Siemens - Solid Edge Harness Design.
