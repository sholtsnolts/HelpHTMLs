Use Harness Wizard to create a harness design

This procedure provides the basic steps for using the Electrical Routing Wizard command to automatically generate a electrical routing design based on information defined in an imported ECAD net list file. For more information on using the Electrical Routing Wizard, see the Working with Electrical Routing Design help topic.

  1. Click Home→Wizard→Harness Wizard.

  2. On the Harness Wizard - Step 1 of 3 dialog box, specify a document format, component document, and connection document to be used in the electrical routing design.

  3. Click the Next button.

  4. On the Harness Wizard - Step 2 of 3 dialog box, use the available commands to make component and occurrence assignments, or populate the assembly.

  5. Click the Next button.

  6. On the Harness Wizard - Step 3 of 3 dialog box, use the available commands to modify connections, delete connections from the harness design, or preview the electrical routing design.

  7. Click the Finish button.

Tip:

  • Use the Assign Component command to assign a component to a part in the assembly. When you use this command the attributes are written to the part files if you have write-access.

  • Use the Assign Occurrence command to assign an occurrence to a component or swap existing occurrences.

  • Use the Populate command to add a part to the assembly that is listed in the components file, but missing from the assembly.

  • Use the Assign Terminal command to assign a terminal to a circular edge on a part in the assembly. When you use this command the attributes are written to the part files if you have write-access.

  • Use the Delete Wire From Harness command to delete a wire from the harness design.

  • Use the Preview command to preview the electrical routing design. You can preview a single connection, or use the Ctrl and Shift keys to preview multiple connections.
