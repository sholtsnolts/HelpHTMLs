Viewing PCB Collaborator log files

The PCB Collaborator log viewer displays information related to the data exchange between the ECAD and MCAD systems. This information includes the names of files being imported or exported, as well as files that fail to import or export.

The log viewer is file specific and stored in the project folder. Each message is time stamped and as new messages are generated, they are appended to the list displayed in the viewer. This allows you to view messages from the current Solid Edge session, as well as previous sessions.

Displaying the log viewer

The log viewer is automatically displayed when you select the Apply or Send button on the ECAD Collaborator dialog box. To reopen the viewer, you can click the Log viewer button on the ECAD Collaborator dialog box.

Clearing the log viewer

You can right-click in the log viewer and then click the Clear Logs button to clear the log information from the dialog box. The Clear Logs button clears only the contents of the dialog box and not the log file messages.

Closing the log viewer

The log viewer automatically closes when you close the ECAD Collaborator dialog box. You can also click the Close button on the log viewer to close it.
