ECAD Collaborator dialog box

Specifies information for importing or exporting files with ECAD Collaborator.

Files

Displays IDX files for a particular project that are shared by ECAD user in a common location.

History

Displays the history for the file exchange. You can use the Clear button to clear the entire history.

Setup

Write Options

Schema

Displays the IDX file versions. Versions 2.0 and 3.0 are available and the default version is 3.0

Schema units

Specifies the schema units. Valid unit values include Inches, Millimeters, Microns, and Thousandths.

File Ext

Displays the extension for the file format. The only extension available is idx.

Communication Type

Displays the type for communications between the ECAD and MCAD systems. File is the only available type.

Email

Option disabled for this dialog box.

Copper Options

Show All Pads/Traces/Vias

Displays copper data in the imported .idx tree structure.

3D model options

Import 3D models

Imports 3D models exported by Pads Professional and Xpedition. The 3D files are in .xtd and are automatically aligned and mapped according to the associated alignment and mapping file.

Get PCB data

Click this option to get the latest changes made to the PCB assembly.

Select file type

Specifies the types of files to import or export. You can import or export:

  • Baseline files

  • Changes files

  • Response files

Collaboration Data

Displays the items associated with the specified IDX file.

Added

Displays only the added PCB items.

Deleted

Displays only the deleted PCB items.

Modified

Displays only the modified PCB items.

Properties

Displays the properties for the specified IDX file.
