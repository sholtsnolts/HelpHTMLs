New Board Assembly dialog box (Step 1 of 2)

Project Name

Specifies the project name for the board assembly. After you specify a project name, an project assembly file and the associated sheet metal file are generated with the defined project name. For example, project Printboard1 creates Printboard1.asm and Printboard1.psm.

Note:

The name of the project must be unique within the project folder and cannot contain special characters, such as \ / : * ? " < > | and ,. The project name cannot end with : "<space><period>".

PCB Assembly Template

Specifies the template for the PCB assembly, which specifies the measurement units for the assembly. Click the Browse button to select the template you want to use.

Board Template

Specifies the template for the board, which specifies the measurement units for the board. Click the Browse button to select the template you want to use.

Project Location

Specifies the project location where the assembly file and sheet metal board file are created.
