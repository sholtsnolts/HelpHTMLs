Harness Reports dialog box

Retrieves and displays information for wire harnesses in the assembly document. Use the Format button to define the report format.

You can display the Report Output dialog box by clicking the OK button.

Note:

The actual report output dialog box name is based on the report type you select and the assembly document name.

Components

Specifies that you want the report to display information about components connected to the wire harness. If the component is not used by the wire harness, it is not displayed in the report.

Connections

Specifies that you want the report to display information about wires, cables, and bundles in the assembly file.

Harness BOM

Specifies that you want the report to display a bill of materials containing information about the components and connections in the assembly file.

Among

Defines which components or connections you want to include in the report:

  • All harness components and connections in the assembly

  • Harness components or connections currently selected

  • Harness components or connections currently shown

Format

Displays the Format Report dialog box for the report type you specified.
