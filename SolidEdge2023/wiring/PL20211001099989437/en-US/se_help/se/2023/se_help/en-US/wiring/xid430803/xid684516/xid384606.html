Print a nailboard view with Adobe Reader

  1. Save the nailboard draft file as Adobe PDF.

  2. Open the PDF file in Adobe Reader (version 10 or later).

  3. Choose File→Print.

  4. On the Print dialog box, do the following:

    1. Select the desired printer and paper size.

    2. In the Paper Size and Handling section, select Poster.

    3. Ensure the scale is set to 100%.

    4. Set the Overlap as desired.

    5. Turn on or off Cut marks, Labels, and Tile only large pages as desired.
