Outputting to DSI format

Use the Save as ECAD dialog box, while working in Solid Edge Electrical Routing, to export Harness Topology in DSI format’. After exporting to DSI format, you can import the files in Aracadia/Capital to create a harness diagram.

Defining custom properties

  • Before exporting to DSI, you must define properties for:

    • Components

    • Top-level assembly

    These custom properties are saved to the SEHarnessPropseed.txt file.

    For components, you must define the Harness Components custom property.

  • To define this property:

      1. Right-click any component and then click File Properties.

      2. On the Custom tab of the Properties dialog box, select Harness Components from the Name drop-down menu.

      3. Specify a Type and Value for the property and then click Add.

  • For the top-level assembly you must define:

    • Harness Issue

    • Harness Customer name

    • Harness Datum

    To define these properties:

      1. Right-click the top-level assembly and then click File Properties.

      2. On the Custom tab of the Properties dialog box, select the Harness Issue from the Name drop-down menu.

      3. Specify a Type and Value for the property and then click Add.

      4. Repeat steps 1 - 3 to add Harness Customer name and Harness Datum.

      5. On the File menu, point to Save As, then choose Save As ECAD.

      6. On the Save as ECAD dialog box, set Document format to Siemens - Harness Topology - DSI.

      7. On the Save as ECAD dialog box, click Save As.

      8. On the Save As - Harness Topology dialog box, in the File Name box, specify a name and location for the component document.

      9. In the Save as type box, select Harness topology document (*dsi) from the list.

      10. On the Save As - Harness Topology dialog box, click Save.

      11. On the Save as ECAD dialog box, click OK.
