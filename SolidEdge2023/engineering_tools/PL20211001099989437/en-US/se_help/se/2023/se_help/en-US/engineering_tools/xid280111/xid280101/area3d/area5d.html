Coordinate System page (Info dialog box)


For area objects, sets the show or hide state and the display properties of the centroid coordinate system and the principal moment of inertia coordinate system. The overall size of a coordinate system is controlled by the font size in the dimension style.

A keypoint is displayed at the center of the coordinate systems. You can use it to connect other objects to it, such as dimensions.

Show Center Coordinate System

Specifies that the centroid coordinate system (X, Y) is displayed. The centroid coordinate system represents the moments of inertia about the global X and Y axes. It is always parallel to the global axes.

Show Moments Coordinate System

Specifies that the principal moment of inertia coordinate system (PX, PY) is displayed.

Leader

Specifies properties of the coordinate system leader lines.

Color

Applies a color to the leader line. All system colors are selectable, as well as any custom colors you have defined.

Line Width

Specifies a value for the coordinate system line width.

Text Scale

Specifies the size of the leader terminator with respect to the text height.

Center Terminator Length

Specifies the length of the center coordinate system leader terminator with respect to the text height.

Moments Terminator Length

Specifies the length of the moments coordinate system leader terminator with respect to the text height.

Text

Specifies the text in the area object coordinate system.

Dimension Style

Specifies the dimension style for the area object coordinate system.

Font

Specifies the text font for the area object coordinate system.

Font Style

Specifies the font style to use for the area object coordinate system text.

Font Size

Specifies the text size for the area object coordinate system.

Fill Text With Background Color

Fills the coordinate system text with the current background color.

Color

Sets the text color for the area object coordinate system.

Center X

Specifies the string to display in the coordinate system for the X axis of the center coordinate system. The default is X.

Center Y

Specifies the string to display in the coordinate system for the Y axis of the center coordinate system. The default is Y.

Orientation X

Specifies the string to display in the coordinate system for the orientation X axis. The default is PX.

Orientation Y

Specifies the string to display in the coordinate system for the orientation Y axis. The default is PY.

Preview

Shows a preview of the area based on the current properties.
