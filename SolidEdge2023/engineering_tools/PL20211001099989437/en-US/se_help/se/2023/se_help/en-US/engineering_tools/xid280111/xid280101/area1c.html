Area command

The Area command computes the area, perimeter, centroid, moments of inertia, and principal moments of inertia for the 2D geometry in one or more selected, closed boundaries. There are two methods for computing this information.

  • Create an area object, which is selectable and can be used for further computations. Set this option on the Area command bar: .

  • Compute the information for temporary display and copy/paste into another document. The selected geometry is not maintained as an area after you close the dialog box. Set this option on the Area command bar: .

Tip:

You must lock a sketch plane to use this command in the 3D environments. See Lock or unlock a sketch plane.
