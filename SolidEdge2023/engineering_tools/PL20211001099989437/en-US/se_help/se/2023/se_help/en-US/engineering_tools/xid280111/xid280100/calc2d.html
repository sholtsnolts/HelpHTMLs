Goal Seek Options dialog box

The duration of the Goal Seek operation is limited by the conditions set on the Goal Seek Options dialog box. The first condition that is met ends the calculation.

Maximum Time (seconds)

Specifies the maximum amount of time the Goal Seek command should run. The default is 60 seconds.

Maximum Number Of Iterations

Specifies the maximum number of iterations the Goal Seek command should perform the calculation with the current set of variables and values. The default is 300 iterations.

Note:

After the Goal Seek command runs, the amount of time that passed and the number of iterations that it took to complete the operation are displayed in the status bar.

For example: Total Time Elapsed (secs): 0.39000 seconds; Number of Iterations: 23.
