Physical Properties Manager dialog box

Allows you to edit the physical properties for the top-level assembly or its part documents. For example, you can specify a material for a part.

You can click the right mouse button on an occupied row in the grid to display a shortcut menu that allows you to do such things as change the font, sort the properties, find and replace text, and format columns.

You can double-click near a column divider to do a best fit on the grid. Any changes you make to the columns are stored for future sessions.

Physical properties automatically update after changing material and gage properties in part and sheet metal enviornments.

The material of parts can be changed in Property Manager without opening them in Solid Edge.

Read-only cells are displayed using a different background color.

Right-mouse click options allow:

  • Update physical properties.

  • Update material properties

  • Update gage properties.

Print

Sends a copy of the active document to a specified plotter, printer, or file. Options are available for defining the printing area, range, number of copies, and other printing characteristics. Before using this command, you must install and select a printer. For help installing a printer, see the Windows documentation.

Cut

Cuts the current selection and pastes it to the Clipboard.

Copy

Copies the current selection to the Clipboard.

Paste

Inserts the Clipboard contents at the specified insertion point.

Restore

Sets all property cells back to the previous values.

Update All

Updates the physical properties for the entire assembly. When you save the assembly document, any changes you made to part documents are also saved.

Check out-of date

Checks for physical properties that out-of-date.

Update

Updates all the out-of-date properties for all parts in the assembly.

Shortcut menu commands

The following additional shortcut menu commands are available. The shortcut menu commands are context-sensitive. In other words, the commands which are available change based on what is selected.

Cut

Cuts the current selection and pastes it to the Clipboard.

Copy

Copies the current selection to the Clipboard.

Paste

Inserts the Clipboard contents at the specified insertion point.

Columns

Displays the Columns dialog box so you set the column display and order options.

Font

Displays the Font dialog box so you can select the font, font style, and font size you want.

Search and Replace

Displays the Replace dialog box so you can find and replace text within the dialog box.

Sort

Allows you to sort the list in ascending or descending order.

Update

Updates the physical properties for the selected components. When you save the assembly document, any changes you made to part documents are also saved.
