Area command bar

The options on the Area command bar can be used to display area information about selected geometry as well as to create an area object from the selection set. An area object can be used in further computations.

Dimension Style

Sets the active dimension style for the area object.

Fill Style

Lists the available fill styles. When you choose one from the list, it sets the active fill style for the area object.

Use drawing view scale

Specifies that measurements within a drawing view or between 2D geometry and a drawing view are displayed using the scale of the model geometry in the drawing view. Applying the drawing view scale is equivalent to returning the model distance.

The inverse scale of the drawing view you select is used to calculate the value.

Scale

Applies and displays a user-defined scale value. The inverse of the selected scale is used to calculate the returned values.

Note:

Scale values are defined in the Drawing View Scales section of the Custom.xml file, in the Solid Edge Preferences folder. See the Help topic, Add custom drawing view scales to Solid Edge.

Recompute

For an existing area object, recomputes the area and other information based on changes to its 2D geometry. You should select this option if you added or removed geometry from an existing area, or if the geometry in the selection set was moved. Even if the area measurement does not change, its center point and axes do.

To learn how, see the Help topic, Recompute a modified area object.

Note:

This option is not available during area creation.

Properties

Displays the tabbed Info dialog box, which contains this information:

  • The Area Properties page displays the area and perimeter computations for the geometry in the selection set.

  • The Coordinate System page is where you specify properties of the coordinate systems that are displayed at the center of area objects. Coordinate system properties include color, line, text, terminator, and leader.

  • The Fill page is where you specify area object fill characteristics such as color, pattern, and spacing.

Area Information

Calculates area information for the selected geometry and displays the Info dialog box. Temporarily fills the area so that you can verify what you have selected, but both the area fill and the area information are discarded when the command is ended.

Create Area

In addition to calculating the area information for the selected geometry, this option creates a persistent area object from the selection set. You can use an area object as the basis for further design and engineering calculations. For example, you can add dimensions and change the size of one or more dimensions, and then recompute the area.

Note:

Create Area is the default method for the Area command.

Deselect

Discards all input and restarts the command.

Accept (check mark)

Accepts the currently selected geometry and then computes the area.
