Physical Properties command

You can use the Physical Properties command to calculate physical properties for a part or assembly. You can then save the data to an ASCII file. For more information, see Physical properties of parts and assemblies.

You can access the command from:

  • Inspect tab→Physical Properties group→Properties

  • Data Management tab→Properties group→Properties
