Recompute a modified area object

You can recompute area object information to obtain updated calculations when the geometry changes. You should do this when you have:

  • Added geometry inside an area object.

  • Trimmed or removed geometry from the object.

  • Added overlapping geometry.

  • Moved the area object.

Recalculate area information

  1. In the graphic window, select the filled area object to be recalculated.

  2. On the Area command bar, click Recompute.

    The filled area object updates to reflect where 2D geometry has been added or removed.

  3. To see the updated calculations for the area object, click Properties.
