Measure Total Length command

The Measure Total Length command measures the cumulative length of a select set of 2D wireframe geometry. As you add elements to the select set, the cumulative length calculates and displays on the command bar.

You can use the Measure Total Length command to measure 2D elements in the Draft environment and when sketching in 3D models.

Tip:

You must lock a sketch plane to use this command in the 3D environments. See Lock or unlock a sketch plane.
