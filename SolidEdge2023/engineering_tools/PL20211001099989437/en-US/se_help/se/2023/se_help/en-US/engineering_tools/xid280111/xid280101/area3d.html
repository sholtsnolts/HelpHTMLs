Info dialog box

The Info dialog box displays quantitative information for 2D geometry within selected, closed boundaries. For example, you can review information computed by the Area command. You also can set display properties for area objects.

Tabbed pages

  • Area Properties page displays the area and perimeter computations for the geometry in the selection set.

  • Fill page specifies area object fill characteristics such as color, pattern, and spacing.

  • Coordinate System page specifies properties of the coordinate systems that are displayed at the center of area objects. Coordinate system properties include color, line, text, terminator, and leader.
