Measure Area command bar

Use drawing view scale

Specifies that the measurement within a drawing view or between 2D geometry and a drawing view is displayed using the scale of the model geometry in the drawing view. Applying the drawing view scale is equivalent to returning the model distance.

For example, areas may be defined that use 2D geometry and drawing view geometry. If the area boundary includes a drawing view edge, and the Use Drawing View Scale box is checked, then the scale of the drawing view is used.

The inverse scale of the drawing view is used to calculate the value.

Scale

Applies and displays a user-defined scale value. The inverse of the selected scale is used to calculate the returned values.

Note:

Scale values are defined in the Drawing View Scales section of the Custom.xml file, in the C:\Program Files\Siemens\Solid Edge 2023\Preferences folder. See the Help topic, Add custom drawing view scales to Solid Edge.
