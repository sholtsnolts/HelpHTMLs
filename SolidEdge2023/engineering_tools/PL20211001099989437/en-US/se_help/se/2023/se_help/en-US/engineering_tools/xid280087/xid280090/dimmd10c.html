Measure Distance command

In the Part, Sheet Metal, and Assembly environments, the Measure Distance command measures a linear distance between points. You can also measure an angular distance defined by three points or two faces.

In the Draft environment, or when sketching in a model, the Measure Distance command measures a linear distance between points, or it measures the cumulative linear distance along a series of points. You can measure between elements in the same drawing view or in different drawing views. Options on the Measure Distance command bar display the distance value using the scale of the geometry in a selected drawing view or using a custom scale value.
