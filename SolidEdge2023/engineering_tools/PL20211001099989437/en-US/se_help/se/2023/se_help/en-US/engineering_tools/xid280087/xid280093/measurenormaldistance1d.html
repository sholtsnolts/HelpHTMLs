Measure Normal Distance command bar

The options that are available depend upon what you are measuring and the model type.

Element types

Sets the type of elements you want to locate.

Keypoint

Sets the type of keypoint you can select when measuring the distance. The available keypoint options are:

Selects an end point.

Selects a midpoint.

Selects the center point of a circle or arc.
Coordinate System

Selects a user-defined coordinate system to which you want to measure.

Measurement Variable

Creates a measurement variable in the variable table. A measurement variable is a driven variable whose value is derived from the distance or angle between two keypoints or elements. You typically use a measurement variable to drive another variable whose value is dependant on the measurement variable. For example, you can create a measurement variable that tracks the normal distance between two part faces in an assembly. You can then use the measurement variable to drive the thickness of another part in the assembly. If the normal distance between the two parts changes, the thickness of the other part also changes.

Reset

Resets the command to its original state.

Close

Closes the Measure Normal Distance data window and restarts the command.
