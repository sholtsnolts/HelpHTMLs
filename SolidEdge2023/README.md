# Solid Edge 2023 Help

Offline/local archive of Siemens Solid Edge 2023 help pages.

## How to use

- Open `index.html` in this folder for the tabbed help index.
- Click a section card to open that section's topic index.
- Pages are stored locally, so most internal topic links and images should work without going back to Siemens.
