AutoCAD To Solid Edge Translation Wizard - Step 3 of 8

Assists in mapping an AutoCAD model space drawings to a Solid Edge 2D view. You cannot import AutoCAD drawings that consist of model and paper space view ports into a Solid Edge 2-D view.

Sheet

Defines the properties of the drawing sheet.

Show Background

Displays the background sheet graphics in the Solid Edge Draft document you are going to create.

Background Sheet

Specifies which background sheet you want to use for the working sheet.

All graphics on the background sheet are displayed on the working sheet. Changing the background sheet causes the size and margin settings of the working sheet to update to the values defined by the selected background sheet.

2D Model View
Map AutoCAD Model Space Entities to a 2D Model View

Specifies that you want to place the model space geometry defined in the AutoCAD file in a Solid Edge 2D view. Solid Edge automatically scales the 2D view based on the View Scale option you set. Solid Edge then places the 2D view, aligning the lower left hand corner of the 2D view with the draft sheet.

This option is only available if the AutoCAD file contains model space. If the AutoCAD file contains paper space, for example, view ports, the option is disabled and Solid Edge imports the file by creating 2D views to represent each view port.

When you select this option, you activate the Display as Model Space parameter on the Drawing View Properties dialog box, which allows dimensions to be rendered as model space instead of paper space. This creates the correct display for terminators and projection (extension) lines.

AutoCAD Drawing Size

Displays the actual size of the AutoCAD drawing in AutoCAD units. This option is helpful in determining what size sheet to define before you import the AutoCAD document.

View Scale
Best Fit

Specifies that the system computes the scale using the ranges of the model space data and the size of the sheet. The computed value is rounded to the next standard scale.

Select Scale

Selects the scale from the Select Scale dropdown.

Enter Scale

Allows you to type in the drawing scale. For example, a 2 represents a 2:1 ratio of drawing scale to the real world object.

Reset

Displays a dialog box where you can reset wizard options to their default values. You can reset options on the active page or you can reset options on all pages in the wizard. If you select Reset all options, the system resets options for all pages and displays the first page of the wizard.

Cancel

Discards all input and exits the command.

Back

Moves to the previous dialog box of the Import Translation Wizard.

Next

Moves to the next dialog box of the Import Translation Wizard.

Finish

Finishes the wizard and returns to the Open File dialog box.
