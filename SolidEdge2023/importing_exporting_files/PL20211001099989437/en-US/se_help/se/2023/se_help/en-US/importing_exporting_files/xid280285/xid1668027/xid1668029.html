Creo Elements Direct Data Migration Start Page dialog box

Assists in importing Creo Elements Direct part, assembly, and drawing (*.sdpc,*.sdp,*.sda, *.sdac, *.bdl and *.pkg) files into Solid Edge.

Creo Elements Direct Folder

Specifies the name of the folder that contains the Creo Elements Direct documents to migrate to Solid Edge. You can browse to search for this folder.

Solid Edge Folder

Specifies the name of the final storage folder for the Solid Edge documents. The folder must have read and write access for the user that is currently logged in. You can browse to search for this folder.

Part template

Specifies the part template to use for the imported Creo Elements Direct files. You can browse to search for a template. For more information see, Creating Documents and Using Templates.

Assembly template

Specifies the assembly template to use for the imported Creo Elements Direct files. You can browse to search for a template. For more information see, Creating Documents and Using Templates.

Draft template

Specifies the draft template to use for the imported Creo Elements Direct files. You can browse to search for a template. For more information see, Creating Documents and Using Templates.

Creo ED configuration file

Displays the name of the Inventor configuration file to use during translation. This file contains the user-defined settings, such as font and line type mapping. The configuration file used by default is CreoED2SE.ini, located in the Solid Edge 2023\Preferences\Translators folder.

You can browse to select the configuration file that you want to use.

Log file location

Specifies the folder path for the log file to generate during translation.

Cancel

Cancels the Creo Elements Direct Data Migration Wizard without saving any of the information.

Next

Moves to the next page of the Creo Elements Direct Migration Wizard.

Finish

Completes the migration.
