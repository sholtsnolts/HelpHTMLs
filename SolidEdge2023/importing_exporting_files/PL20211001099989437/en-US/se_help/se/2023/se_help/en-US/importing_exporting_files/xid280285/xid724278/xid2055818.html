Logging for SolidWorks data migration

Because SolidWorks data migration has its limitations, it is important to know which elements are successfully migrated and which are not. The migration logs will help you to find this.

The migration logs provide information related to:

  • Features not supported for migration.

  • Migration errors or termination due to an issue from SolidWorks side.

  • Migration errors or termination due to an issue from Solid Edge side.

  • Options used for migration.

  • Drawing associativity failure due to detached drawings.
