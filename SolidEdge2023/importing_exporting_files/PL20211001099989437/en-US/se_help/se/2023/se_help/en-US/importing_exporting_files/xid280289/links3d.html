Links dialog box (Part and Sheet Metal)

Displays information about links in a document including the file location, and whether the link is automatically updated.

This dialog box is available using the Edit Links command on the shortcut menu within the Variable Table when the part or sheet metal document contains a link to another document, such as a part copy.

List View

Displays the first level linked documents with the object type, the update setting, and the linked document status. The status of the document can be Loaded or Unloaded.

If the status is set to Loaded, all the document data is loaded into physical memory.

If the status is set to Unloaded, only the graphical representation of the document is loaded into physical memory. The feature history and mathematical part definition is not loaded into memory, which reduces the physical memory demands.

You can select an unloaded document and click the Update Now button to set the status to Loaded. When you click the Update Now button the selected document is loaded and the dialog box is dismissed.

Tree View

Displays all nested document links for the active document. The "+" indicates that the document's set of linked documents is collapsed. You can select the "+" to expand the collapsed set of linked documents. You can select one or more documents from the source file section. When you select a document in the list, the full path name of the source document is displayed below the list.

A document displaying a red “X” only has its graphical representation loaded into physical memory. The feature information and mathematical part definition is not loaded into memory, which reduces the physical memory demands.

You can select an unloaded document and click the Update Now button to set the status to Loaded. When you click the Update Now button the selected document is loaded and the dialog box is dismissed.

Source File

Displays the source file. The Document Name Formula is used for the source file in the Tree View Page.

Type

Displays the document type.

Update Now

Loads the selected document in the source document section, and dismisses the dialog box.

Open Source

Opens the selected link in the source application for editing.

Change Source

Accesses the Change Source dialog box.  Use the Change Source dialog box only to resolve a link in which the folder and/or the file name has changed.  Otherwise, use the Replace Part command or Design Manager.
