AutoCAD To Solid Edge Translation Wizard - Step 6 of 8

Assists in mapping an AutoCAD font to a Solid Edge font. To re-map an AutoCAD font, click on a Solid Edge font, and then select a Solid Edge font from the drop-down list.

AutoCAD Font

Displays a list of available AutoCAD fonts. The list of fonts is defined by the active configuration file. The list also includes AutoCAD fonts found in the AutoCAD file being translated. Fonts found in the AutoCAD  file precede the list of fonts found in the Configuration file.

Solid Edge Font

Defines the Solid Edge font to be mapped to the selected AutoCAD font. By default, the mapping of fonts found in the configuration file is maintained. For those fonts found in the AutoCAD file, the default true type font used is Arial.

Reset

Displays a dialog box where you can reset wizard options to their default values. You can reset options on the active page or you can reset options on all pages in the wizard. If you select Reset all options, the system resets options for all pages and displays the first page of the wizard.

Cancel

Discards all input and exits the command.

Back

Moves to the previous dialog box of the Import Translation Wizard.

Next

Moves to the next dialog box of the Import Translation Wizard.

Finish

Finishes the wizard and returns to the Open File dialog box.
