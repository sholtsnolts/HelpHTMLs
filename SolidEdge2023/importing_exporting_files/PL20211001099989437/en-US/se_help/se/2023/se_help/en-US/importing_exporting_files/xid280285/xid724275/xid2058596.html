Migrating Inventor material definitions to solid edge

When you migrate any Inventor part file, the material definition is extracted. This material is assigned to the Solid Edge document if the material name exists in the Solid Edge .mtl file. To ensure that the Inventor material name exists in the Solid Edge Material file, Solid Edge provides a tool to convert the Inventor material file(*.adsklib) to the Solid Edge Material file (.mtl) file.

Note:

Converting the Inventor Material data base is optional. However, if you do not convert the material database files or set up of either the Inventor or Solid Edge material files location improperly, no material is applied to migrated Solid Edge files.

The Material database migration tool on the Inventor Data Migration Start Page supports migration of material definitions from Inventor to Solid Edge.

Before running the utility, consider the following:

  • After successfully converting the material table ensure that the Solid Edge\Options\File Locations pointer is pointing to the appropriate file.

  • Before starting a migration ensure that Inventor is pointing to the same material database folder that was previously converted.
