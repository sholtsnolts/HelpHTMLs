Importing non-Solid Edge files as internal components

Use the Create Internal Components option on the following dialog boxes to import non-Solid Edge files as internal components:

  • Import Options for Parasolid (.x_t) dialog box

  • Import options for JT Documents (.jt) dialog box

  • Import Options for STEP (.stp) dialog box

  • Import Options for IFC (*ifc) dialog box

  • Import Options for NX (.prt) dialog box

When you import files as internal components, they are imported as embedded components in the new Solid Edge file and no files are created on the disk. Therefore, the time spent importing the file is reduced.

Note:

When importing as internal components, translation is limited to:

  • Geometry

  • Appearance

  • Assembly structure

INI parameter for importing internal components

The Import As Internal Components parameter in the STEP.ini, SEIFC.ini, PSXMT.ini, and SePvTrn.ini files controls if non-Solid Edge files are imported as internal components. By default, the parameter value is OFF, which imports in the traditional method and creates files on the disk. You can set the value to ON to import the files as internal components so that no files are created on the disk.
