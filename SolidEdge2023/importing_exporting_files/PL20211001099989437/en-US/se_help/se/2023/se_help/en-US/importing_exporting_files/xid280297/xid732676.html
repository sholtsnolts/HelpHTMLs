Import multiple bodies into a single part file

When multiple bodies are translated into a single part file, you can use the same process in both Ordered and Synchronous to activate the part.

With Solids as design bodies enabled

  1. In PathFinder, select the body to which you want to add features.

  2. Right-click and select Activate Body.

With Solids as design bodies disabled

  1. In PathFinder, select the construction body that you want to toggle to a design body.

  2. Right-click and select Toggle Design/Construction Body.

    Note:

    If the Make Base Feature parameter in the .ini file is enabled, all solids are imported as design bodies and you can skip step 2. If the Make Base Feature parameter is disabled, you must perform step 2 to toggle to the construction body to a design body.

  3. Right-click and select Activate Body.

If you clear this option, all solid bodies contained in the file you are importing are placed in PathFinder as construction solids rather than as design bodies.
