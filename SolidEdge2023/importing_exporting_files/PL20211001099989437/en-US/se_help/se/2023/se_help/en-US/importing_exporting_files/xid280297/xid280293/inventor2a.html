Importing Inventor part and assembly documents to Solid Edge

You can open Inventor part (.ipt ) or assembly (.iam) documents in Solid Edge assembly, part, or sheet metal with the Open command. On the Open File dialog box, after you select the Inventor document you want to open, you can click the Options button to use the Import Options for Inventor (.ipt, and .iam) dialog box to set the appropriate import options.

Note:

Solid Edge does not support the import of Inventor assembly features such as weld beads.

To open Inventor documents you must have Autodesk Inventor View installed on the same machine as Solid Edge. When you install Inventor, Inventor View is automatically installed. If you do not have Inventor view installed, either standalone or with Inventor, you should visit the Autodesk website for a free copy.

Note:

Solid Edge supports Autodesk Inventor View 2019.

If you encounter problems while importing the Inventor document(s), turn Enable Logging on to create a log file of the import processes.

Note:

Inventor files must be saved in Inventor V6 or greater before they can be opened in Solid Edge.

When Inventor assembly documents are imported, Solid Edge tracks instances of individual parts within the assembly. If there are multiple instances of a part within the Inventor assembly, Solid Edge creates one instance of the part and references that part rather than create multiple instances of the part.

The InventorDirect2SE.ini file

The InventorDirect2SE.ini file, in the Solid Edge 2023\Preferences\Translators folder, stores the dialog box settings. When you change a parameter in the dialog box, a new value is saved to the InventorDirect2SE.ini file. The following is a list of parameters found in the InventorDirect2SE.ini file.

Parameter

Description

Import Boolean Solids =On

Controls the inclusion of Boolean solids in the imported file.

Import Check Bodies=Off

Controls the inclusion of all curves as a single part copy in the imported file.

Import DoNotSave=On

Controls whether or not assembly components are created on import, or on the initial save of the assembly.

Import Drive=C:

Stores the drive where the documents for the imported file.

Import Enable Default Output Path=On

Controls the enabling of the default output path.

Import Healing and Sewing=Off

Controls the inclusion of heal and sew surfaces in the imported file.

Import Imprint Curves=On

Controls the inclusion of all curves as a single part copy in the imported file.

Import Make Base Feature=On

Controls if the imported features become the base features in the new file.

Import Multiple Bodies as Single Part

Controls the inclusion of multiple solid bodies as a single part in the imported file.

Import Output Path=” “

Stores the path for the output folder of the documents being imported.

Import Path=" “

Stores the value for the path where the documents are imported.

Import Simplify Geometry=Off

Controls the inclusion of simplified geometry in the imported file.

Import Stitch Surfaces=On

Controls the inclusion of stitch surfaces in the imported file.

Enable Logging=0

Turns on or off the creation of the log file.

File property translation

Inventor file properties are mapped to Solid Edge properties when the Inventor document is imported. Solid Edge does not translate the color in Inventor documents when the document is imported.
