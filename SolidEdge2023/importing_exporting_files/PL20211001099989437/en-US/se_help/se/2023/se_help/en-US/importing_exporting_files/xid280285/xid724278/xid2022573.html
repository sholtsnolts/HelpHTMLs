Specify the location of SolidWorks and Solid Edge folders

  1. On the SolidWorks Data Migration Start Page, click Folder Selection.

  2. On the Folder Selection dialog box, specify the name and location for the SolidWorks folder. Use the Browse button to change the location of the folder.

  3. On the Folder Selection dialog box, specify the name and location for the Solid Edge folder. The default location is C:\Users\<userlogin>\AppData\Local\Temp\. Use the Browse button to change the location of the folder.

  4. On the Folder Selection dialog box, specify the name and location for the migration log file.

    The default location is C:\Users\<userlogin>\AppData\Roaming\Siemens\Solid Edge\Version 223\Log Files. Use the Browse button to change the location of the folder.

  5. On the Folder Selection dialog box, click OK.
