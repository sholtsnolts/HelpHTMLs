Saving to OBJ files

You can export Solid Edge assembly, part, and sheet metal files to OBJ files. You can use the Export Options for OBJ (.obj) dialog box to specify how you want to export the Solid Edge file.

SEOBJ.ini file

The settings on the option dialog boxes used when importing or exporting OBJ files are defined in the SEOBJ.ini file, located in the Solid Edge 2023\Preferences\Translators folder. When you make a change to a parameter in the dialog box, a new value is saved to the .ini file. You can use a text editor, such as Notepad, to set these parameters. However, if you edit this file, use extreme caution in setting these parameters. Errors made to this file can adversely affect the quality of the translation.
