Templates dialog box

Specifies the type of template to use when creating Solid Edge files from the imported Inventor files. Make sure to use template with mm units when your files are in mm units. Same is applicable for CGS and ANSI inch systems.

Part template

Specifies the part template to use when creating Solid Edge part files from the Inventor files selected for migration. You can browse to search for a template. For more information see, Creating documents and using templates.

Assembly template

Specifies the assembly template to use when creating Solid Edge assembly files from the Inventor files selected for migration. You can browse to search for a template. For more information see, Creating documents and using templates.

Drawing template

Specifies the drawing template to use when creating Solid Edge drawing files from the Inventor files selected for migration. You can browse to search for a template. For more information see, Creating documents and using templates.

Reset

Resets the template values to the default values defined in Saved settings.

OK

Saves the template values and dismisses the dialog box.
