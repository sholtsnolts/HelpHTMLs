Import Options for OBJ (.obj) dialog box

File Units

Specifies the units for the file being imported.

Millimeters

Specifies millimeters as the units of measurement for the part being imported.

Centimeters

Specifies centimeters as the units of measurement for the part being imported.

Meters

Specifies meters as the units of measurement for the part being imported.

Kilometers

Specifies kilometers as the units of measurement for the part being imported.

Inches

Specifies inches as the units of measurement for the part being imported.

Feet

Specifies feet as the units of measurement for the part being imported.

Microns

Specifies microns as the units of measurement for the part being imported.

Make base feature

Specifies that you want to make the imported mesh body the design body for the Solid Edge model.

Heal mesh faults

Specifies that you want to repair any mesh faults when you open the OBJ file.

Generate log file

Produces a log file that contains warnings and information about the files involved in translation. The name of the file is the <filename>.log and it is located in the same folder as the OBJ file.

Split Group

If checked, creates multiple mesh groups when importing an OBJ file into Solid Edge assembly. When the file is opened in an assembly template, you can see the parts and subassemblies that contain mesh groups in PathFinder.

If unchecked, creates a single mesh group or minimum mesh group when importing an OBJ file into a Solid Edge assembly template.

Note:

You will see the results of setting this option only if you import an OBJ file into Solid Edge assembly template.
