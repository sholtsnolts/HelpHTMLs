Solid Edge to JT Translation Options dialog box

Save
All Documents

Saves all open documents to the JT file.

Part Documents Only

Saves only the part documents to the JT file.

Assembly Documents Only

Saves the assembly documents as a bill of materials to the JT document. The document contains a tree structure of the assembly documents, but does not contain a graphical representation of the documents.

JT Document Structure

Specifies how the JT information is stored on disk.

Single JT Document

Creates a single or monolithic JT document.

Note:

This option must be selected to use the Solid Edge Publish to 3D PDF command to generate JT attachments for machining applications (CAM/CMM), such as Solid Edge CAM Pro.

JT Assembly and Part Documents

Creates a JT document containing the assembly structure and a folder containing the JT part documents.

JT Assembly, Sub-Assembly, and Part Documents

Creates a JT document for each document in the assembly. This option is useful if you want to view data at the subassembly level.

Only Update Modified Parts

Creates JT documents only for the parts that have changed.

Part and Assembly Simplification

Specifies that, when available, a simplified body, instead of the design body, is used when creating the JT document.

Part simplification
Use Current Configuration

Uses the current configuration to create the JT document.

All Simplified

Checks each part and uses the simplified body, if available, to create the JT document.

All Designed

Checks each part and uses the design body, if available, to create the JT file.

Assembly simplification
Use Current Configuration

Uses the current assembly configuration to create the JT document.

Top Level Simplification

Checks the top level of the assembly and uses the simplified body, if available, to create the JT document.

All Simplified

Checks each subassembly found in the assembly and uses the simplified body, if available, to create the JT document.

All Designed

Checks each part found in the assembly and uses the design body, if available, to create the JT file.

Include Precise Geometry

Specifies that you want to include B-spline data in addition to the other data stored in the document.

Note:

When the Save PMI Data option is selected, precise geometry is always sent even if this option is not set.

Remove Unsafe Characters From JT Document Name

Replaces the following characters found in a JT file name with an underscore (_).

\n \r \t ` ~ ! @ # $ % ^ & * ( ) - + = \" ' : ; , . / ? { } [ ]

Translate Visible Parts Only

Exports to the JT document only parts that are currently displayed. Parts that are hidden are not exported.

Translate Visible Constructions

Exports to the JT document only construction bodies that are currently displayed. Construction bodies that are hidden are not exported.

Translate Inter-part Copies as Constructions

Exports inter-part copies as constructions. If this option is not selected, inter-part copies are not exported.

Translate Solid Edge Document Properties

Exports the Solid Edge file properties to the JT document.

Delete Unused JT Part Documents

Deletes any JT part files that no longer used are removed from the JT assembly document.

Save PMI Data

Saves all product manufacturing information (PMI) data and all model views to the JT file being created. This option must be selected to use the Solid Edge Publish to 3D PDF command to generate JT attachments for machining applications (CAM/CMM), such as Solid Edge CAM Pro.

Save Coordinate System

Saves the coordinate system to the JT file being created.

JT Version

Specifies the version of JT to which you want to export the file.

Units

Specifies the output units of measurement for the document.

Sheet Metal

Specifies how sheet metal models are translated. You can translate the design model, flatten sheet metal model, or both.
