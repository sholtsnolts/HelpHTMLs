Saving to STEP

You can save Solid Edge assembly, part, and sheet metal documents in STEP (.stp) format with the Save As command. On the Save As dialog box, after you select the Solid Edge document you want to save, click the Options button to display the Export Options for STEP (.stp) dialog box. You can use the dialog box to control how the Solid Edge document is saved in .stp format.

The STEP.ini file, found in the Solid Edge 2023\Preferences\Translators folder, stores the settings selected from the user interface. When you change a parameter in the options form, a new value is saved to the file.

The Export Part Component Name as Unicode= parameter specifies the part component names are written as unicode. If this parameter is set to Off, the part component names are written in the user’s locale setting. By default, the value is Off.

You can use a text editor, such as Notepad, to set these parameters. However, if you edit this file, use extreme caution in setting these parameters. Errors in this file can adversely affect translation quality.

Exporting PMI

You can use the Export PMI option on the Export Options for STEP (.STP) Dialog Box to specify that you want to export any part level PMI contained in the Solid Edge file to the STEP AP242 file.

There is also a corresponding Export PMI parameter in the STEP.ini file, which is set to off by default. Leaving the parameter set to off prevents part level PMI from being exported to the STEP file being created. Setting the value to on exports all part level PMI contained in the Solid Edge file to the STEP file being created.

Note:

The STEP AP242 protocol and the Export PMI option must be selected to use the Solid Edge Publish to 3D PDF command to generate STEP AP242 attachments for machining applications (CAM/CMM), such as Solid Edge CAM Pro.

Note:

The export of assembly level PMI is not supported.

Note:

Export to STEP AP242 now supports surface association for PMI dimensions and annotations.
