Search by sketch with 3Dfind.it

  1. Open a new draft document.

  2. Change the view to 2D Model by right-clicking the drawing sheet (for example, Sheet 1) and then selecting 2D Model.

    A search for components based on a sketch can only be performed while in the 2D Model view.

  3. Draw a 2D sketch representing the shape of a part you want to search for.

    A more refined sketch produces the most precise search results.

  4. Fill the closed region by selecting Home tab→Draw group→Fill command.

  5. Choose File menu→Tools tab→Search 3Dfind.it.

  6. From the 3Dfind.it website, select the model you want to use in Solid Edge.

  7. Click Send to Solid Edge. The selected model is opened in a new tab in the Solid Edge modeling window.

    The model is downloaded to the default location for the 3DFind.it models as defined in the Solid Edge Options dialog box, on the File Locations page. If the value is undefined, you are prompted to browse for a folder.

Tip:

  • In the Teamcenter-managed environment, when you choose to insert a model from 3Dfind.it, the system first checks to see if the part is already in Teamcenter. If the part already exists in Teamcenter, the document from Teamcenter is used. If the part does not exist in Teamcenter, the part is downloaded from 3Dfind.it to the default location, and inserted into your open assembly. The part is saved to Teamcenter when you upload the assembly to Teamcenter.

  • To enable 3Dfind.it functionality in the Teamcenter-managed environment, set the Teamcenter preference SEEC_Enable_3DFindit value to 1.

    For a full description of the preference, see the Teamcenter Preferences section of the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.
