Templates dialog box

Specifies the type of template to use when creating Solid Edge part files from the imported SolidWorks files.

Part template

Specifies the part template to use when creating Solid Edge part files from the SolidWorks files selected for migration. You can browse to search for a template. For more information see, Creating documents and using templates.

Assembly template

Specifies the assembly template to use when creating Solid Edge assembly files from the SolidWorks files selected for migration. You can browse to search for a template. For more information see, Creating documents and using templates.

Drawing template

Specifies the drawing template to use when creating Solid Edge drawing files from the SolidWorks files selected for migration. You can browse to search for a template. For more information see, Creating documents and using templates.

Sheet Metal template

Specifies the sheet metal template to use when creating Solid Edge sheet metal files from the SolidWorks files selected for migration. You can browse to search for a template. For more information, see Creating documents and using templates.

Reset

Resets the template values to the default values.

OK

Saves the template values and dismisses the dialog box.
