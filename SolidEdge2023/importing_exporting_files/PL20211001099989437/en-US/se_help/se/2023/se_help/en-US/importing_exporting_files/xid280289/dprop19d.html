Info tab (Object Properties dialog box)

Type

Displays the type of the selected element. You cannot edit the type.

Origin

Sets the X and Y coordinate location for the object.

X

Sets a value for the x coordinate. You can use this option by itself or with the Y option.

Y

Sets a value for the y coordinate. You can use this option by itself or with the X option.

Link

Controls whether the inserted object is available to other viewers of the document.

As Part of the Model (Public)

Specifies that the inserted object is visible when the source document is inserted into another document. For example, if you insert a Word document into Draft document A, then insert Draft document A into Draft document B, the Word document is visible when this option is set.

As Reference Only (Private)

Specifies that the inserted object is not visible when the source document is inserted into another document. For example, if you insert a Word document into Draft document A, then insert Draft document A into Draft document B, the Word document is not visible when this option is set.

Scale

Controls the scale for the inserted object.

Scale (1:1)

Sets the object scale to 1:1.

Select Scale

Sets the object scale to the value you select from the list.

Custom

Sets the object scale to the value you specify.
