Saving to PLM

When you export a single part file from Solid Edge to PLM XML, two files are created.  One of the files ends with a .plmxml extension and the other ends with an .x_t extension. The .plmxml file is an ASCII file that references the .x_t file.

When you export an assembly from Solid Edge, a file with a .plmxml extension is generated, along with an .x_t file for each unique part. In this case the .plmxml file references each body through the .x_t files and provides the correct file association. In an assembly that contains three unique parts you will have three Parasolid .x_t files.
