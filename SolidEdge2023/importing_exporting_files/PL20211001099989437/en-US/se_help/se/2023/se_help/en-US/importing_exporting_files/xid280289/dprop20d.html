Format tab (Object Properties dialog box)

Show Border

Displays a border around the object. You can set options that control the color, line width, line type, and shape of the border.

Color

Sets the border color.

Line Width

Sets the border width.

Line Type

Sets the line type for the border.

Frame Shape

Sets the shape of the border.

Rectangular

Specifies that the border is rectangular.

Elliptical

Specifies that the border is elliptical.
