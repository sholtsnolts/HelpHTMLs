Migrating SolidWorks hole definitions to Solid Edge

SolidWorks has changed its hole database from .mdb to .sldedb file format. The Hole database migration tool on the SolidWorks Data Migration Start Page supports migration of .sldedb file format.

However, both type of databases that is .mdb and .sldedb can be migrated using the DOS window. For more information, see Migrate SolidWorks hole definitions to Solid Edge using the DOS window.

After the hole database migration is completed and correct mapping is provided, the holes are recognized after the SolidWorks files migration. For more information, see Hole feature recognition.
