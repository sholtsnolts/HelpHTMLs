Solid Edge to AutoCAD Translation Wizard (Hatch Style Mapping)

Assists in mapping Solid Edge cross hatch styles to AutoCAD cross hatch styles. To re-map a Solid Edge hatch, click on a Solid Edge hatch, and then click on the AutoCAD hatch name in the pulldown list.

Export Hatch as AutoCAD Block

Specifies that the Solid Edge hatch will be exported as AutoCAD block data instead of AutoCAD hatch objects.  Prior to version 13 of AutoCAD, hatch objects were not supported. All hatches were represented by using line elements with various line types to represent the hatch. This type of hatch was often grouped together to form a single entity of block data.

If you set the value of the AutoCAD Version option on the AutoCAD Export Translation Wizard (File Mapping) to 12 you automatically enable the output of hatch objects as line elements and not as an AutoCAD block. All other versions of AutoCAD will output hatch objects unless you select this option.

If the Solid Edge hatch object has a fill color defined, the fill will not be exported if this option is selected.

Solid Edge Hatch

Specifies the Solid Edge hatch style to be assigned to the selected AutoCAD hatch style.

AutoCAD Hatch

Displays a list of available AutoCAD hatch styles found in the configuration file.

Reset

Displays a dialog box where you can reset wizard options to their default values. You can reset options on the active page or you can reset options on all pages in the wizard. If you select Reset all options, the system resets options for all pages and displays the first page of the wizard.

Cancel

Discards all input and exits the command.

Back

Moves to the previous dialog box of the Translation Wizard.

Next

Moves to the next dialog box of the Translation Wizard.

Finish

Finishes the wizard and returns to the Save As dialog box.
