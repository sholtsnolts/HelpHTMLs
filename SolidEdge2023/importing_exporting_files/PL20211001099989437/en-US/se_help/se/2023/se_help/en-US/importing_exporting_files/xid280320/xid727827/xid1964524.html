Save surface PMI to JT format

Solid Edge 3D translation exports to JT format the model PMI annotations and dimensions that are associated with surfaces and with edges. You can use this information to identify the model surfaces referenced by the PMI for visual inspection purposes. Surface information is required for creating inspection pathways in Coordinate Measurement Machine (CMM) software, such as Hexagon's PC-DMIS.

Export support is provided for tolerances, datums, and surface finish annotations, including GD&T information (Concentricity, Roundness, Cylindricity, Flatness, Parallelism, Position, Perpendicularity, Runout, Total Runout).

  1. In the JT translation files, SePvTrn.ini and SEECtoJT.ini, search for the following parameters and change their values from 0 to 1:

    • Export Surface Connections to PMI Annotations and Dimensions

    • Export Symmetric Diameter as Standard Diameter dimension

  2. In the Save As Translated dialog box, select the JT file type, and in the Solid Edge to JT Translation Options dialog box, select the check boxes, Include precise geometry and Save PMI Data.
