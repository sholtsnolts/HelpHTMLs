Saving to kinematics XML documents (.plmxml)

When you export an assembly with motion kinematics from Solid Edge to a kinematics XML document (.plmxml), two files are created. One of the files has a .plmxml extension and the other has a .jt extension.

The PLM XML file contains data related to assembly constraints.

The JT file contains geometry. The reference to the geometry is provided in the PLM XML file.
