Solid Edge to Kinematics JT Translation Options dialog box

JT Document Structure

Specifies how the JT information is stored on disk.

Single JT Document for Process Simulate

Creates a single or monolithic JT document.

Note:

We recommend that you use this option while exporting kinematics XML documents to Process Simulate, when an assembly contains one or more subassemblies.

JT Assembly and Part Documents for NX MCD

Creates a JT document containing the assembly structure and a folder containing the JT part documents.

Note:

We recommend you that use this option while exporting kinematics XML documents to NX MCD, when an assembly contains one or more subassemblies.

Only Update Modified Parts

Creates JT documents only for the parts that have changed.
