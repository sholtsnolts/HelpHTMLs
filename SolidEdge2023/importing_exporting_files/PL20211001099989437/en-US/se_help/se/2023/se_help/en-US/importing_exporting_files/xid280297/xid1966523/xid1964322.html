Inserting files using CAD Direct

Use the CAD Direct command to insert non-Solid Edge files directly into Solid Edge assemblies, without translating the files separately. This eliminates the task of translating non-Solid Edge files before inserting them into an assembly.

To insert non-Solid Edge file as a CAD Direct component in the assembly environment, choose Home tab→Components group→Insert Component→CAD Direct .

Note:

When using the CAD Direct command, you can only insert one file at a time.

To specify which geometry elements you want to insert from JT and STEP files, you can use the .ini files or options available in the options dialog box for the type of file you are inserting.

Links are maintained between the parent non-Solid Edge file and the Solid Edge geometry. Any changes made to the non-Solid Edge geometry are reflected in the Solid Edge assembly.

Links are only maintained with the top node, so you must save the top assembly of the linked non-Solid Edge file. For example, if you insert a non-Solid Edge assembly as a CAD Direct component and later change a part in that non-Solid Edge assembly, the part is saved, but the assembly is not saved. Updates will not reflect changes in the part. In this case you will get a warning about a force update which, when accepted, will re-translate everything, and reflect changes in part. We recommend that you save the linked top assembly before updating.

If you Delete or Move or Rename the parent file from its original location, only the geometry will be maintained and not the link. When you restore the file to the original location, the link will be reestablished.

You can add multiple occurrences of CAD Direct components. Any updates to the geometry are updates for all occurrences.

You can apply constraints or features between different files you have inserted or between inserted files and Solid Edge files in the assembly.

Note:

This command supports the latest version of NX, SolidWorks, Parasolid (*.x_t; *.x_b), STEP (*.step; *.stp), and JT files.

Tip:

The name of the CAD Direct component indicates the file format of the source file.

Any features created using CAD Direct geometry are modified based on the changes in the source file and the constraints are still maintained. Therefore, it is also maintaining downstream features with some limitations. Associativity is not maintained when there is a change in the position of a parent geometry.

Example:

Cutting of edges—features created on the connected faces/edges are no longer associative.

Each file is inserted as an internal component linked to non-Solid Edge files, so no files are created on disk. If you need the components on disk, use the Publish Internal Components command. For more information, see Publishing internal components.

Updating CAD Direct links

There may be cases when changes are made to the linked non-Solid Edge file. You can replace the older version of the component with a new component with the same name. To update the changes, right-click the CAD Direct component and choose Update CAD Direct Geometry.

Changes to the CAD Direct geometry include:

  • Parts added or removed from the parent non-Solid Edge assembly.

  • Subassembly added or removed from the parent non-Solid Edge assembly.

  • Part or subassembly position changed in the parent non-Solid Edge assembly.

  • Body added to the parent non-Solid Edge file.

  • Body position changed in the parent non-Solid Edge file.

Note:

To update the changes, we recommend you save the parent non-Solid Edge file.

Changing the source file

You can change the source of a CAD Direct component using the Edit Links command. The link will be replaced with the new non-Solid Edge file. It will maintain downstream features with some limitations.

While replacing source file, consider the following instances:

Selecting a different file with the same name from a different location

If the same naming is followed for parts and subassemblies in the source file, changes will be updated by maintaining downstream features. Otherwise, changes will be updated without maintaining downstream features.

Selecting a new source file with a different name

The old CAD Direct component will be replaced with a new one without maintaining downstream features.

Selecting a file of non-supported version

The old CAD geometry with downstream features will be maintained.

Translation failure

The old CAD geometry with downstream features will be maintained.

Multiple instances of the same CAD Direct component in the file

All the instances will be replaced by the new CAD Direct component.

Breaking CAD Direct links

To break the link between the parent file and any CAD Direct component(s), right-click a CAD Direct component and choose Break CAD Direct Link. This makes the Solid Edge geometry independent of the non-Solid Edge file. If you break the link, the link cannot be reestablished. To break the link of multiple instances, hold Ctrl and click the items in PathFinder or in the work area. This command is not available for subnodes.

CAD Direct and document management

Inserting non-Solid Edge files using the CAD Direct command impacts document management as follows:

  • Design Manager—Non-Solid Edge files linked to CAD Direct components are not displayed in Design Manager. For commands located on the Design Manger or Data Management tabs, such as Save As, Pack and Go, and Where Used, links for CAD Direct components are maintained with the source file at the original location. Even after executing the Pack and Go command, the link to the CAD Direct component is with the original non-Solid Edge file. It is not copied to a new location or added to a zip file.

  • SEDM—If you try to save an unmanaged file containing a CAD Direct component to a vault location, the CAD Direct source file is not copied to the vault location. The CAD Direct component keeps the link with the original source file at the old location.

  • SEEC—After uploading an unmanaged Solid Edge assembly containing a CAD Direct component, the source file is not uploaded to the SEEC server, but the CAD Direct geometry is available in the uploaded assembly.

CAD Direct impacts on assembly

For the CAD Direct component, the entire structure appears in PathFinder, but only the top node appears in the Internal Components collector.
