Migrate Inventor documents to Solid Edge

  1. From the Start menu, choose Siemens Solid Edge 2023→Inventor Data Migration.

  2. On the Inventor Data Migration Start Page, click Material database migration tool to display the Inventor Material Database Migration dialog box. Use the dialog box to migrate Inventor material definitions to Solid Edge.

  3. On the Inventor Data Migration Start Page, click Hole database migration tool to display the Inventor Hole Database Migration dialog box. Use the dialog box to migrate Inventor hole definitions to Solid Edge.

  4. On the Inventor Data Migration Start Page, click Templates to display the Templates dialog box. Use the dialog box to specify the templates to use for the Inventor data migration.

  5. On the Inventor Data Migration Start Page, click Folder Selection to display the Folder Selection dialog box. Use the dialog box to specify the location of the Inventor and Solid Edge folders.

  6. On the Inventor Data Migration Start Page, select the appropriate options for migrating the Inventor files. You can use the Saved Settings option to save the settings for the dialog box.

  7. Click Next.

  8. On the Inventor File Selection Page, use the file type filter option to specify the type of Inventor files you want to migrate.

  9. On the Inventor File Selection Page, select the documents you want to open.

  10. Click Add or Add All to add the selected folder or documents to the list of files to be migrated.

    Note:

    Click the Select file(s) from all subfolders option if you want to include files from all subfolders to the migration.

  11. On the Inventor File Selection Page click Next.

  12. On the Preview Selection Page, select any files that you do not want to migrate, and then click Remove.

  13. Click Finish.
