Migrate SolidWorks material definitions to Solid Edge using the DOS window

The SWmat2SEMat.exe utility is delivered to the Solid Edge program folder. The executable creates a new Solid Edge materials .mtl file rather than appending an existing .mtl file. The executable organizes material definitions by placing all similar materials such as steel and iron into a single category.

Before running the utility, consider the following:

  • We recommend to make a copy of the Solid Edge Material.mtl file.

  • After successfully converting the material table ensure that the Solid Edge\Options\File location pointer is pointing to the appropriate file.

  • Before starting a migration ensure that SolidWorks is pointing to the same material database folder that was previously converted.

Migrate SolidWorks material definitions to Solid Edge

  1. Open a DOS window.

  2. CD to the Solid Edge Program folder.

  3. Run SWmat2SEMat.exe and set the following:

    • Argument 1: Full path and name of the solidworks materials.sldmat (in double quotes).

      Example:

      “C:\Program Files\SolidWorks Corp\SolidWorks\lang\english\sldmaterials\solidworks materials.sldmat”

      The output folder for the converted Solid Edge material file is the Solid Edge installation folder\Preferences\Materials.

  4. After the Solid Edge materials .mtl is created, make sure the Solid Edge installation folder\Preferences\Materials folder is set properly.
