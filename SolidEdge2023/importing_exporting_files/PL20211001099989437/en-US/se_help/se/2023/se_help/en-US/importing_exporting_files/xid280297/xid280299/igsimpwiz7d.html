IGES Import Translation Wizard (Fonts)

Assists in mapping an IGES font to a Solid Edge font. To re-map a Solid Edge font, click on an IGES font, and then enter the new IGES font name.

To add a new font to the list, enter the IGES font number in the text box and click the Add Font button.

Additional IGES Fonts

Displays a list of additional IGES fonts that you can add to the list of default fonts.

Add Font

Adds the new font to the list of available IGES fonts.

Iges Font

To delete fonts from this list, click on the font you want to delete and press the DELETE key on your keyboard. You cannot select multiple rows for deletion.

Solid Edge Font

Specifies the Solid Edge font to be mapped to the selected IGES font. The configuration file supplies the default font values.

To change the default values click on the Solid Edge font, and then click a new font from the drop-down list. The list of available true type fonts is derived from the installed fonts on your machine.

To delete fonts from this list, click on the font you want to delete and press the DELETE key on your keyboard. You cannot select multiple rows for deletion.

Preview

Lists the entities, entity names, and occurrences found in the IGES file being opened. This is used to confirm the template selection, or to make informed decisions on other parameter selections.

Back

Moves to the previous dialog box of the IGES Import Translation Wizard.

Next

Moves to the next dialog box of the IGES Import Translation Wizard.

Finish

Finishes the wizard and returns to the Open File dialog box.
