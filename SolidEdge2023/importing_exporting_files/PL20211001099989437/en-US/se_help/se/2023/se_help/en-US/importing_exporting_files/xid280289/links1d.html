Links dialog box (Draft)

Displays information about links in a document, including the file location, and whether the link is automatically updated. To use the Links dialog box, select a document on the List View tab or the Tree View tab to see its associated information.

List View

Displays the location and name of the first-level linked document, its type (Part, Sheet Metal, or Assembly), and its status (Loaded or Unloaded).

  • If the status is Loaded, all the document data is loaded into physical memory.

  • If the status is Unloaded, only the graphical representation of the document is loaded into physical memory. The feature history and mathematical part definition are not loaded into memory.

You can select an unloaded document and click the Update Now button to load the document and dismiss the dialog box.

Tree View

Displays all nested (lower level) document links for the active document. The "+" indicates that the set of linked documents is collapsed. You can select the "+" to expand the collapsed set of linked documents. When you select a document in the list, the full path name of the source document is displayed below the list.

Icons indicate the status of files linked to the active document (loaded or not loaded). If a document is loaded, the document update status is also shown (manual or automatic).

Meaning

Document update status

Automatic update of links.

When a draft file is created directly from a part, sheet metal, or assembly document, the lightning bolt symbol is displayed, because the associated model is already open (loaded).

Document currently is not loaded (or open).

You can use Update Now to load the associated document into memory.

Manual update of links.

When an existing draft file is opened, the associated part, sheet metal, or assembly document is not automatically loaded. The hand symbol is displayed when you load the associated document by opening the source file, and then reopen this dialog box.
Source File

Displays the source file. The Document Name Formula is used for the source file in the Tree View page.

Type

Displays the full document type, for example, Solid Edge Assembly Document.

Update Now

Loads the selected document and dismisses the dialog box.

Open Source

Opens the selected link in the source application for editing.

Change Source

Accesses the Change Source dialog box. Use the Change Source dialog box only to resolve a link in which the folder or the file name has changed.  Otherwise, use the Replace Part command or Design Manager.

In the Teamcenter-managed environment, the Change Source dialog box displays a list of managed folder and documents. You can select another file to replace the existing one. The new file can be a new revision of the existing item, or from another item.
