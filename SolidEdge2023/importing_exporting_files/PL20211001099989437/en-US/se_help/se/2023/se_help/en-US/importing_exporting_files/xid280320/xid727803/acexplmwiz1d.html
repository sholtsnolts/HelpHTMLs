Solid Edge to AutoCAD Translation Wizard (Layer Mapping)

Assists in mapping Solid Edge layers and attributes to an AutoCAD layer. This dialog box is displayed only if you select the Enable Layer and Attribute Mapping control on the AutoCAD Translation Wizard (File Mapping) dialog box.

This dialog box allows you to map layers from Solid edge to AutoCAD while applying attributes, such as line type, color, and width, to the layer. The layer mapping dialog box displays layers from 3 locations:

  • Current document.

  • Layers saved to the seacad.ini file from previous translations.

  • A special set of layers added to control the export of 3D derived geometries.

The special set of layers added to this list are:

  • Center Lines—centerlines created from flat patterns, tube centerlines, and any centerlines that you have created on your draft sheet.

  • Flow Lines—flow lines created from an exploded view.

  • Hatch—hatch derived from a 3-D model such as a section view. This also includes any hatch that you may have created using the hatch command.

  • Hidden—hidden lines derived from a 3-D model. This does not automatically map dashed line types you may have added to a view or your working sheet.

  • Visible—visible lines created from a 3-D model. This does not automatically map continuous line types you may have added to a view or your working sheet.

  • Tangent—tangent lines derived from a 3-D model.

  • Reference—lines created from reference geometry.

Wizard Options
Delete Row

Deletes the entire row of layer information for the layers found in the seacad.ini file. You cannot delete layers found in the active document or the special set of layers added by Solid Edge. You can only delete layers that were previously defined and stored in the seacad.ini file. To delete a layer, select the layer to highlight the entire row, then click Delete Row.

Solid Edge Layer Name

Displays the list of all layers within the active document, along with any layers that were previously defined and stored in the seacad.ini file. The layers are only a list and cannot be selected for edit.

AutoCAD Layer Name

Specifies the name of the layer created in the AutoCAD file. By default, this is the same name as the Solid Edge layer name. You can click the name and type a new layer name. Multiple Solid Edge layers can be mapped to a single layer in AutoCAD. When the same AutoCAD name is used more than once the duplicated rows are linked to the first occurrence of that name. Further edits to change color, line type, or width can only occur through the master row. To unlink the rows, edit the AutoCAD name to be unique. The Hide/Show state of this layer is determined by the master row.

Line Type By Layer

Specifies the line type of the AutoCAD layer. To enable this field, click the arrow to the left of the field. Once enabled, click the pull-down arrow to display a list of line types defined in the seacad.ini file, from which you can choose. If disabled, the line type is determined by the object's assigned line type in Solid Edge and the line type mapping defined in the [Write Linestyles] section of the seacad.ini file.

Color By Layer

Specifies the color of the AutoCAD layer. To enable this field, click the arrow to the left of the field. If enabled, the color of the AutoCAD object is determined by the line width of the Solid Edge object specified in the [Write Width] section of the seacad.ini file. If disabled, the color of the AutoCAD object is determined by the color of the object in Solid Edge. For example, if the object is red in Solid Edge, it will be red in AutoCAD. The valid range for AutoCAD colors is 1 - 255.

Width By Layer

Specifies the line width of the AutoCAD layer. To enable this field, click the arrow to the left of the field. Once enabled, click the pull-down arrow to display a list of line widths defined in the seacad.ini file, from which you can choose. If disabled, the line width is determined by the actual line width of the Solid Edge object.

Reset

Displays a dialog box where you can reset wizard options to their default values. You can reset options on the active page or you can reset options on all pages in the wizard. If you select Reset all options, the system resets options for all pages and displays the first page of the wizard.

Cancel

Discards all input and exits the command.

Next

Moves to the next dialog box of the Translation Wizard.

Finish

Finishes the wizard and returns to the Save As dialog box.

Export one or all sheets to AutoCAD
