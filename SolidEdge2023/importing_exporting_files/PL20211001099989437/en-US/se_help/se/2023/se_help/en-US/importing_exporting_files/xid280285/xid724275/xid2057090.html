Inventor Hole Database Migration dialog box

Location of Inventor hole database

Specifies the name and location of the Inventor hole database. Use the Browse button to locate the folder and database. The default location is the location that is mentioned in the active project.

Location of migrated files

Specifies the location to which the files will be migrated. The default location is C:\Program Files\Siemens\Solid Edge 2023\Preferences\Holes.

Solid Edge hole database Inch template

Specifies the template for Solid Edge hole database Inch standard. The default location is C:\Program Files\Siemens\Solid Edge 2023\Preferences\Holes\ANSI Inch.xlsx.

Solid Edge hole database Metric template

Specifies the template for Solid Edge hole database Metric standard. The default location is C:\Program Files\Siemens\Solid Edge 2023\Preferences\Holes\ISO Metric.xlsx.

Hole mapping

Displays the Hole Mapping dialog box so you can define the hole mapping for the database.

Populate Default

Populates the default paths for Inventor and Solid Edge hole database, and templates. This option requires Inventor and Solid Edge, with active licenses, to be installed on your machine.

Run

Starts the hole database migration.

Note:

If the Inventor hole database migration fails, we recommend uninstalling the Microsoft access database engine. Once the migration is complete, it can be reinstalled.
