IGES Export Translation Wizard (Linetype)

Assists in mapping Solid Edge line types to IGES line types. To re-map a Solid Edge line type, click on an IGES line type, and then select a line type from the drop-down list.

Iges Line Types

Displays a list of default line types defined by the IGES configuration file. IGES line types are defined by type 304 and have a range of 1-5.

By default the five IGES line types are mapped to that of the closest corresponding Solid Edge line types.

Solid Edge Line Type

Specifies the gap sequence for the display of graphic elements used in Solid Edge Draft. The list displays the nine line types delivered with Solid Edge. User-defined line types are not permitted.

Back

Moves to the previous dialog box of the IGES Export Translation Wizard.

Next

Moves to the next dialog box of the IGES Export Translation Wizard.

Finish

Finishes the wizard and returns to the Save As dialog box.
