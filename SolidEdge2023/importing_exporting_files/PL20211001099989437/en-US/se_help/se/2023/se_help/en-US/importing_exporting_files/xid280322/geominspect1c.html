Geometry Inspector command

The Inspect tab→Evaluate group→Geometry Inspector command detects flaws in model geometry, and it provides information to help you understand and resolve the errors. You can use the Geometry Inspector to identify flaws in imported models, which may cause problems in Solid Edge. You can use it to simplify Solid Edge models for finite element analysis and also to repair a faulty mesh model.

Note:

The command can also be found here: Reverse Engineering tab→Cleanup Mesh group→Geometry Inspector command
