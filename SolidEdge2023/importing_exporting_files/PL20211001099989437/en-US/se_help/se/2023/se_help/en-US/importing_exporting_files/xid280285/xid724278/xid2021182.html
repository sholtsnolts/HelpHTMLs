SolidWorks Hole Database Migration dialog box

SolidWorks hole database to be migrated

Specifies the name and location of the SolidWorks hole database. Use the Browse button to locate the folder and database. The default location is C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS\Toolbox\data utilities\lang\English\swbrowser.sldedb.

Location of migrated files

Specifies the location to which the files will be migrated. The default location is C:\Program Files\Siemens\Solid Edge 2023\Preferences\Holes.

Hole database standards to be migrated

Specifies the hole database standards that you want to migrate.

Hole mapping

Displays the Hole Mapping dialog box so you can define the hole mapping for the database.

Populate Default

Populates the default paths for SolidWorks and Solid Edge hole database. This option requires SolidWorks and Solid Edge, with active licenses, to be installed on your machine.

Run

Starts the hole database migration.

Note:

If the SolidWorks hole database migration fails, we recommend uninstalling the Microsoft access database engine. Once the migration is complete, it can be reinstalled.
