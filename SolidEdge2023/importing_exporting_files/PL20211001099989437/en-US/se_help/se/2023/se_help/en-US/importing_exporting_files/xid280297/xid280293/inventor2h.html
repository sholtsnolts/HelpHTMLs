Open an Inventor document in Solid Edge

  1. Choose File menu→Open tab→Browse command.

  2. In the Open File dialog box set the Files of Type option to Inventor Part documents (*.ipt) or Inventor Assembly documents (*.iam).

  3. Locate to the document you want to open, and then click Open.

  4. In the Open File dialog box click the Options button.

  5. In the Import Options for Inventor (.ipt and .iam) dialog box specify the appropriate options, and then click OK.

    Note:

    For more information on these options, see the help topic for the Import Options for Inventor (.ipt and .iam) dialog box.

  6. On the Open File dialog box click the Open button.
