Saving Changes dialog box

Assists in defining the configuration file to which you want to save the remapped configuration.

To save the changes to the original configuration file, click the Save the Remappings to the Original Text File option.

To save the changes to a new configuration file, click the Create a New Configuration File option, define a folder and file name for the new configuration file, and then click the Copy To button.

Save the Remappings to the Original Text File

Specifies that you want to save any changes you made back to the original configuration file. The default configuration file is Inventor2SE.INI, located in the Solid Edge 2023\Preferences\Translators folder.

Create a New Configuration File

Specifies that you want to save any changes you made to a new configuration file. This option applies the changes made to this translation.

Copy To

Displays the Save As dialog box to allow you to define a folder and file name for the new configuration file.

Back

Moves to the previous dialog box of the Inventor Data Migration Wizard.

Finish

Finishes the wizard.
