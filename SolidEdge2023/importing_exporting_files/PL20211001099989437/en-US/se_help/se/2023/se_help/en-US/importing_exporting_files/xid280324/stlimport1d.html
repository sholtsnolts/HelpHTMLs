STL Import Options Dialog Box

STL File Units

Specifies the units for the file being imported.

Meters

Specifies meters as the units of measurement for the part being imported.

Millimeters

Specifies millimeters as the units of measurement for the part being imported.

Inches

Specifies inches as the units of measurement for the part being imported.

Make Base Feature

Specifies that you want to make the imported mesh body the design body for the Solid Edge model.

If you clear this option, all meshed bodies contained in the STL (.stl) file you are importing are placed in PathFinder as a part copy rather than a body feature. Inserting bodies as a part copy requires you to select or create a base feature.

Heal mesh faults

Specifies that you want to repair any mesh faults when you open the STL file.
