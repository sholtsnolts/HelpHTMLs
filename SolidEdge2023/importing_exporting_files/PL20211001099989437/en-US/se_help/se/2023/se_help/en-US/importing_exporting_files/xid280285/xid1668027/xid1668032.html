Creo Elements Direct Saving Changes dialog box

Save the Remappings to the Original Text File

Specifies that you want to save your changes to the original configuration file. The default configuration file is CreoED2SE.INI, which is located in the Solid Edge 2023\Preferences\Translators folder.

Copy To

Displays the Save As dialog box to allow you to define a folder and file name for the new configuration file.

Cancel

Cancels the Creo Elements Direct Data Migration Wizard without saving any of the information.

Back

Moves to the previous dialog box of the Creo Elements Direct Data Migration Wizard.

Finish

Finishes the wizard.
