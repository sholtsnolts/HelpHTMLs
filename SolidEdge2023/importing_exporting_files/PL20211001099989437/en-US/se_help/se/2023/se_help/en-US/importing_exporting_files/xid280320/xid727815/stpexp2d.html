Export Options for STEP (.STP) dialog box

Controls how STEP saves Solid Edge documents. STEP is an international standard which defines a neutral file format for representation of geometric, topological and annotation data. Solid Edge supports the STEP standards AP203, AP214, and AP242.

Export

Specifies what types of construction geometry you want to export.

Export solids (constructions)

Export any construction geometry that exists as solid bodies.

Export sheets (constructions)

Export any construction geometry that exists as sheet bodies.

Export wires and points

Export any construction geometry that exists as wire bodies.

Note:

Use this option to export Solid Edge documents that contain 3D points to STEP.

Export displayed only

Export only the active model and construction geometry that is visible in the part or assembly.

Export PMI

Exports all part level product manufacturing information (PMI) data to the STEP file being created. This option is only enabled when the STEP protocol is set to AP242. The AP242 protocol is used to generate machine-readable files.

Note:

This option must be selected to use the Solid Edge Publish to 3D PDF command to generate STEP AP242 attachments for machining applications (CAM/CMM), such as Solid Edge CAM Pro.

Part and Assembly Simplification

When it is available, use a simplified body instead of the design body when creating the STEP document.

Part simplification
Use Current Configuration

Uses the current configuration to create the STEP document.

All Simplified

Checks each part and uses the simplified body to create the STEP document.

All Designed

Checks each part and uses the design body to create the STEP file.

Assembly simplification
Use Current Configuration

Uses the current assembly configuration to create the STEP document.

Top Level Simplification

Checks the top level of the assembly and uses the simplified body to create the STEP document.

All Simplified

Checks each subassembly found in the assembly and uses the simplified body to create the STEP document.

All Designed

Checks each part found in the assembly and uses the design body to create the STEP document.

Translation Options

Various options for defining export units, STEP protocol, and generating log files during translation.

Export Units

The units of measurement for the part being exported. You can set the units to millimeters, centimeters, inches, or meters.

STEP protocol

The STEP format to which you want to export the Solid Edge document.  You can set the protocol to AP203, AP214, AP242. AP242 is intended to replace STEP AP203 and AP214 as a single protocol for translating both parts and assemblies. AP242 was released in 2014. If you plan to send STEP files defined in this format, be sure that your target system supports this protocol.

Note:

The STEP protocol must be set to AP242 to use the Solid Edge Publish to 3D PDF command to generate STEP AP242 attachments for machining applications (CAM/CMM), such as Solid Edge CAM Pro.

Generate log file

A log file that contains warnings and information about the files involved in translation. The name of the file is the <filename>.log and it is located in the same folder as the STEP file.

User Info

General user information that you want to include in the Start section of the STEP file.

Author

The name of the author of the file you want to export.

Organization

The name of the organization of the file you want to export.

Program

The name of the program for the file you want to export.

Description

A description of the file you want to export.
