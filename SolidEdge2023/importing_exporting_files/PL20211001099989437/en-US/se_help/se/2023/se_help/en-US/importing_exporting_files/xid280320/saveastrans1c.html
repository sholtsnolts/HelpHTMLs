Save As Translated command


Saves the active document as a translation file (non-Solid Edge document such as Parasolid or STEP) to a new name, location, and format you specify.

For some output formats, there are translation options you can set. The available translation-option dialog boxes are listed below.

Note:

For a list of all available output types, see the topic Saving Solid Edge documents to other formats.

Saving translations in the Teamcenter-managed environment

In the Teamcenter-managed environment, use the Teamcenter tab→Documents group→Save Translation to Teamcenter option to determine whether the Save as Translated command saves the translated file to Teamcenter or to disk (unmanaged). Select the option to save the translation to Teamcenter, or deselect to save unmanaged to disk. The Teamcenter preference, SEEC_Save_Translated defines the behavior for translated files. Use options on the Teamcenter - Save as Translated dialog box to determine how the document is saved to Teamcenter.

You can select multiple files for translation and defer the translation process from Solid Edge to a managed request in Teamcenter Dispatcher. Define the SEEC_Dispatcher_Translation preference to generate a Dispatcher request and process the translation outside of Solid Edge.

Note:

For a description of each preference, see the Teamcenter Preferences section of the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.
