Embed another document into a Solid Edge document

To create and embed a new object

  1. On the File menu, click Info→Binder.

  2. On the Binder dialog box, click Add.

  3. On the Insert Object dialog box, click Create New.

  4. Select the type of object you want to embed.

  5. Create and save the object you want to embed.

To embed an object from another document

  1. On the File menu, click Info→Binder.

  2. On the Binder dialog box, click Add.

  3. On the Insert Object dialog box, click Create from File.

  4. Do one of the following:

    • In the File Name box, type the name of the document that contains the object you want to embed.

    • Click Browse and use the dialog box to find the document that contains the object.
