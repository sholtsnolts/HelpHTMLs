Binder dialog box

Document

Specifies the name of the document you want to link or embed.

Type

Displays the type of document. The type is based on the application used to create the document.

File Name

Displays name of the document you want to insert as an embedded object in your document.

Add

Activates the Insert Object dialog box.

Edit/View

Opens the selected document so you can edit it. To select a document, click its name in the list. If you open the direct document as read-only, the control name changes to View and you can view, but not edit the selected document.
