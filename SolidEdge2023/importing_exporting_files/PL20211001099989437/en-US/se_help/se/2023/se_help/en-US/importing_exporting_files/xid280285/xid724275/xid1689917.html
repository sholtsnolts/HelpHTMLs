Migrate the Inventor hole database to Solid Edge using the DOS window

Solid Edge provides SEHolePipeFromINV.exe utility to the Solid Edge program folder. The executable reads the Inventor hole database and creates a new Excel file in a format that is recognized by Solid Edge.

To run the hole database migration tool

  1. Locate Thread.xls and Clearance.xls files that were delivered when Inventor was installed.

  2. Open a DOS window.

  3. Enter the path for the SEHolePipeFromINV.exe . For example, “C:\Program Files\Siemens\Solid Edge 2023\Program\SEHolePipeXMLFromCED.exe.”

  4. Set the following:

    • Pathname 1 — Path for the location for Clearance.xls and Threads.xls. For example, “C:\Users\Public\Documents\Autodesk\Inventor <Version>\Design Data\XLS\en-US .”

    • Pathname 2 — The path where migrated hole database files are created.

      For example, "C:\Program Files\Siemens\Solid Edge 2023\Preferences\Holes. "

    • Pathname 3 — The path from for the Solid Edge hole database template to be used. Copy ANSI Inch.xlsx located in C:\Program Files\Siemens\Solid Edge 2023\Preferences\Holes to another location and use the new location as the path for the ANSI template you want to use. The name must be ANSI Inch.xlsx.

      For example, "D:\Inventor2SE Migration\HoleDB\ANSI Inch.xlsx. "

    • Pathname 4 — The path from for the Solid Edge hole database template to be used. Copy ISO Metric.xlsx located in C:\Program Files\Siemens\Solid Edge 2023\Preferences\Holes to another location and use the new location as the path for the ISO Metric template you want to use. The name must be ISO Metric.xlsx.

      For example, "D:\Invetor2SE Migration\HoleDB\ISO Metric.xlsx. "

    Note:

    Add all arguments one after the other.

    For example,

    C:\>”C:\Program Files\Siemens\Solid Edge 2023\Program>SEHolePipeFromINV.exe” "C:\Users\Public\Documents\Autodesk\Inventor <Version>\Design Data\XLS\en-US " “C:\Program Files\Siemens\Solid Edge 2023\Preferences\Holes” “D:\Invetor2SE Migration\HoleDB\ANSI Inch.xlsx” “D:\Invetor2SE Migration\HoleDB\ISO Metric.xlsx”

  5. Press Enter.

    This creates the files, such as INV_ISO Inch.xlsx and INV_ISO Metric.xlsx in the Solid Edge Preference\Hole folder

You can use the Create Hole Features on Migration parameter in the Inventor2SE.ini file to control how Solid Edge hole features are created during migration.

The default value for the parameter is 1. When the parameter is set to 1, only simple holes are created on migration to Solid Edge.

When the parameter is set to 0, no hole or thread attributes are captured or applied during migration.
