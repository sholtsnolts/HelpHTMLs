Translate Pro/Engineer documents to Solid Edge

  1. On the Windows icon (Start menu), point to Siemens Solid Edge 2023, and then click Pro-e Data Migration.

  2. On the ProE Data Migration dialog box, in the Type of 3D Translation pulldown, select the type of translation you want to do.

  3. On the ProE Data Migration dialog box, in the ProE Folder field, specify the folder that contains the files you want to translate.

    On the ProE Data Migration dialog box, specify a Solid Edge template.

  4. In the Connecting to ProE Data Migration Start Page dialog box, in the ProE Configuration File field, enter the configuration file name.

  5. On the Connecting to ProE Data Migration Start Page dialog box, specify the log file location information.

  6. On the ProE Data Migration Start Page dialog box, click the Next button.

  7. On the ProE Data Migration Selection Page dialog box,  select the documents you want to open.

  8. Click the Add button to add the selected folder or documents to the Files To Be Migrated list.

  9. On the ProE Data Migration Selection Page dialog box, click the Next button.

  10. On the ProE Data Save Page dialog box, specify the configuration file to which you want to save the remapped configuration.

  11. On the Saving Changes dialog box, click the Finish button.
