Saving assembly documents as NX bookmark documents (.bkm)

The Save As Translated command allows you to save an assembly file as a NX bookmark (.bkm) file. The bookmark file provides a simplified definition of the Solid Edge assembly structure that can be read by NX.

Saving Solid Edge documents to Solid Edge Viewer format (.sev)

To view files on a mobile device with the Solid Edge mobile viewer, you need to save models to .sev. You can save .par, .asm and .dft to this format. You can then transfer the file to the mobile device by copying the file directly to the device while it is connected to your system, or by sending the .sev file as an attachment in an email.

Note:

Academic Student licenses cannot save .bip files.

Saving models to Solid Edge Illustrations format (.qsm)

You can use the Save As command to save your Solid Edge part, sheet metal, and assembly documents as a QSM file. The QSM file format is used in the technical publications application, Solid Edge Illustrations. For more information, see About Solid Edge Illustrations.
