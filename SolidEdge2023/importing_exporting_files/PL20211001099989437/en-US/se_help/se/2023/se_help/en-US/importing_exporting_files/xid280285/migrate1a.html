Data migration configuration

The Data migration commands refer to the following applications:

  • Inventor

  • Parametric Technologies (ProE)

  • SolidWorks

  • Creo Elements/Direct

These data migration tools provide migration for assembly, part, and draft files.

The CAD application must be on the same machine as Solid Edge.

Supported software configurations

  • Windows 10

  • Hardware — 64 bit

  • Solid Edge 64 bit

  • Solid Edge Data Migration tool 64 bit

For more information, see:

  • Pro-e data migration

  • Inventor data migration

  • SolidWorks data migration

  • Creo Elements/Direct data migration
