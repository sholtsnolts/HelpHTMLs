Opening Parasolid Documents in Solid Edge

You can open documents created with Parasolid in Solid Edge. You can directly import a Parasolid document with a single valid solid as a Solid Edge Part (.par) document. A base feature is created from the body. You can add features to this body, just as you do with a native Solid Edge part.

Note:

You must save a Parasolid document in Solid Edge part (.par) format before you can place it in a Solid Edge assembly.

Multi-body Parasolid documents can be output from NX assemblies and the data shared between Solid Edge and NX. You can open a multi-body Parasolid document in Solid Edge Assembly, and it will automatically create a set of part documents and attach them to the assembly.

If you import a Parasolid assembly into Solid Edge using a metric template for units, the assembly file imports, displaying Metric template information. However the associated part files display English units.

To correct the problem, on the Helpers page of the Solid Edge Options dialog box, click the Set Default Template option button to set the default template.

Parasolid versions supported by Solid Edge

It is very important that you run the correct version of Parasolid when working in Solid Edge. To ensure that you are running the proper version of Parasolid, see the Parasolid versions for Solid Edge section in the Program Files\SiemensSolid Edge 2023\readme.HTM file.
