ProE Data Migration Start Page dialog box

Type of 3D Translation

Specifies the type of translation you want for the part or assembly you are migrating to Solid Edge. You can choose between STEP and ProE. For ProE documents, the translator supports ProE versions 17, 18, 19, 2000, 2000i, 2001, 2000i2, and also Wildfire, versions 1, 2, 3, 4, and 5.

ProE Folder

Specifies the name of the ProE folder that contains the documents you want to migrate to Solid Edge.

Solid Edge Folder

Specifies the name of the Solid Edge folder that stores Solid Edge documents. This is the final storage folder for the Solid Edge documents. The folder must have read and write access for the user that is currently logged in.

Part Template

Specifies the part template you want to use for the imported ProE files. You can use the Browse button to search for a template.

Assembly Template

Specifies the assembly template you want to use for the imported ProE files. You can use the Browse button to search for a template.

Draft Template

Specifies the draft template you want to use for the imported ProE files. You can use the Browse button to search for a template.

ProE Configuration

Displays the name of the ProE configuration file that you want to use during translation. This file contains the user-defined settings, such as font and line type mapping. The configuration file used by default is Proe2SE.ini which is located in the Solid Edge 2023\Preferences\Translators folder.

You can click the Browse button to select the configuration file that you want to use.

Log File Location

Specifies the folder path for the log file you want to generate during translation.

Next

Moves to the next dialog box of the ProE Data Migration Wizard.

Finish

Finishes the wizard.
