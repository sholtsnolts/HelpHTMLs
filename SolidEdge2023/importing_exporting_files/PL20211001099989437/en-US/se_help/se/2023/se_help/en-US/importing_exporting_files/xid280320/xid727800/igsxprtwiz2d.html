IGES Export Translation Wizard (Main)

Assists in mapping entities in a Solid Edge file to entities in an IGES document. You can use this step in the wizard to enable command logging, specify an IGES configuration file, and enter information for the header.

Enable Logging

Produces a log file that contains warnings and information about the files involved in translation.

The log file is named in the standard convention of file_name.log. It is written to the same folder which Solid Edge reads or is currently writing the IGES file to. Solid Edge will automatically overwrite an existing log file if the same name is specified. You must have write permission for the target folder to be able to create a log file.

File Name

Displays the name of the active Solid Edge file, which is being exported to IGES. File Name is a display-only box.

Iges Configuration File

Displays the name of the IGES configuration file that you want to use during translation. This file contains the user-defined settings, such as font and line type mapping. The configuration file used by default is seiges.ini which is located in the Solid Edge 2023\Preferences\Translators folder. This file name is reserved for Solid Edge Draft and cannot be overwritten.

User-defined configuration files that are used across network drives or defined through network drives are supported.

You can click the Browse button to select the configuration file that you want to use.

Author

Specifies the name of the author of the file you want to export. This information is stored in parameter 21, Your Name, of the Global section of the IGES file.

Organization

Specifies the name of the organization of the file you want to export. This information is stored in parameter 22, Your Organization, of the Global section of the IGES file.

Originating System

Specifies the name of the originating system for the file you want to export. This information is stored in parameter 3, Name and version of the product writing the IGES file, of the Global section of the IGES file.

Receiving System

Specifies the name of the receiving system to which you want to export the file. This information is stored in parameter 12, Name of the receiving system, of the Global section.

Description

Displays a description for the new IGES file.

Next

Moves to the next dialog box of the IGES Export Translation Wizard.

Finish

Finishes the wizard and returns to the Save As dialog box.
