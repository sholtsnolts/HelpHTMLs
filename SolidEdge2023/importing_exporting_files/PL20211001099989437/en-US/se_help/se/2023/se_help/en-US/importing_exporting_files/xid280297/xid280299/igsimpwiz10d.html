IGES Import Translation Wizard (Geometry Types)

Assists in selecting which type of geometry you want to import from the IGES file being translated and how the data will be used.

Bodies (What Type of Geometry to Extract)

Specifies the type of geometry you want to import from the IGES file.

Surfaces and Solids Only

Imports only the surfaces and solid bodies found in the IGES file.

Wireframes and points

Imports only free curves (wireframe entities) from the IGES file into Solid Edge Part, Solid Edge Assembly, or Solid Edge Sheet Metal. This option does not extract or create wire edges; it only imports those free curves found within the IGES file.

Note:

You can use this option to open IGES documents that contain 3D points.

Surfaces, Solids, wire frames and points

Imports solids and surfaces, along with any freestanding curves found in the IGES file.

Note:

You can use this option to open IGES documents that contain 3D points.

Simplify Geometry (Create Planes and Cylinders When Possible)

Simplifies b-spline geometry to planes and cylinders based on the tolerance specified in the Tolerance (Meters) box. The tolerance is displayed in meters. This is an important feature since cylinders and planes are used to define mates, aligns, axial constraints, and the modification of existing bodies for reference plane definitions. If you experience problems in converting an IGES file to Solid Edge as a solid, you may not want to use this control. In general, NURB geometry is more tolerant to edge conditions than that of Analytic geometry. This option also merges half cylinders that share a common centerline.

Active

Specifies that you want to simplify the b-spline geometry to planes and cylinders.

Tolerance (Meters)

Specifies the tolerance used to simplify the b-spline geometry to planes and cylinders. This is for display only and you cannot edit the value.

Convert Blank Surfaces (Surfaces in the IGES File With Their Display Turned Off)

Specifies that you want to translate surfaces that have their display flagged as Off in the IGES file. Typically, blank surfaces are used by the exporting system as construction geometry. If you do not set this option, the blanked surfaces in the IGES file are ignored and not translated.

Preview

Lists the entities, entity names, and occurrences found in the IGES file being opened. This is used to confirm the template selection, or to make informed decisions on other parameter selections.

Back

Moves to the previous dialog box of the IGES Import Translation Wizard.

Next

Moves to the next dialog box of the IGES Import Translation Wizard.

Finish

Finishes the wizard and returns to the Open File dialog box.
