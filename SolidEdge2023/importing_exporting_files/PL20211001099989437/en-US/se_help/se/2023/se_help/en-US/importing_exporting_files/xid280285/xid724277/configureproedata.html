Pro-e data migration

The Pro-e Data Migration command assumes that Pro/Engineer version Creo Parametric 4.0 (or newer) and Solid Edge are loaded on the same computer. Solid Edge is not compatible with Pro/Engineer versions prior to Creo Parametric 4.0

Before getting started with any data translations, you must add the Creo installation folder to your system variables path (Control Panel-> System and Security-> System-> Advanced system settings-> Environment variables-> System Variables-> Path).

For more information, see:

  • Data migration Configuration

Starting Pro/Engineer

For Solid Edge to successfully start the ProE application you must modify the ProE startup procedure in the PROE2SE.ini file, located in the Solid Edge 2023\Preferences\Translators folder. To do this, modify the parameter ProeStartup=Proe2001 to the full path location for proe.exe. For example: ProeStartup=c:\Program Files\proewildfire 4.0\bin\proe.exe.

Also, to get the Proe data migration tool to work properly, you need to set the PRO_COMM_MSG_EXE Environment variable. Navigate to the Start menu Control Panel/Systems and Security/System Properties and modify the environment variable to display the full path for PRO_COMM_MSG_EXE . For example: PRO_COMM_MSG_EXE=C:/program Files/proe2001\i486_nt\obj/pro_comm_msg.exe.
