Opening OBJ files in Solid Edge

You can open OBJ files in Solid Edge. You can use the Import Options for OBJ (.obj) dialog box to specify how you want to import the OBJ document.

SEOBJ.ini file

The settings on the option dialog boxes used when importing or exporting OBJ files are defined in the SEOBJ.ini file, located in the Solid Edge 2020\Preferences\Translators folder. When you make a change to a parameter in the dialog box, a new value is saved to the .ini file. You can use a text editor, such as Notepad, to set these parameters. However, if you edit this file, use extreme caution in setting these parameters. Errors made to this file can adversely affect the quality of the translation.
