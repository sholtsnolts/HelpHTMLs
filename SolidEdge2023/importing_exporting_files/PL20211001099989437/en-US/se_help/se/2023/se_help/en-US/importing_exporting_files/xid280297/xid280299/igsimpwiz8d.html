IGES Import Translation Wizard (Linetype)

Assists in mapping IGES line types to Solid Edge line types. To re-map a Solid Edge line type, click on an IGES line type, and then select a line type from the drop-down list.

Iges Line Types

Displays a list of default line types defined by the IGES configuration file. IGES line types are defined by type 304 and have a range of 1-5.

By default the five IGES line types are mapped to that of the closest corresponding Solid Edge line types.

Solid Edge Line Type

Specifies the gap sequence for the display of graphic elements used in Solid Edge Draft. The list displays the nine line types delivered with Solid Edge. User-defined line types are not permitted.

Preview

Lists the entities, entity names, and occurrences found in the IGES file being opened. This is used to confirm the template selection, or to make informed decisions on other parameter selections.

Back

Moves to the previous dialog box of the IGES Import Translation Wizard.

Next

Moves to the next dialog box of the IGES Import Translation Wizard.

Finish

Finishes the wizard and returns to the Open File dialog box.
