Creo Elements Direct File Selection Page dialog box

Assists in identifying the Creo Elements Direct files you want to open in Solid Edge.

Available Creo Elements Direct Files

Specifies the names of the Creo Elements Direct documents, available in the Creo Elements Direct folder defined during in the first step of translation, you can import to Solid Edge.

Add

Adds the selected documents to the Creo Elements Direct files to be migrated box.

Add All

Adds all documents to the Creo Elements Direct files to be migrated box.

Remove

Removes the selected documents from the Creo Elements Direct files to be migrated box.

Remove All

Removes all documents from the Creo Elements Direct files to be migrated box.

Cancel

Cancels the Creo Elements Direct Data Migration Wizard without saving any of the information.

Back

Moves to the previous page of the Creo Elements Direct Data Migration Wizard.

Next

Moves to the next page of the Creo Elements Direct Data Migration Wizard.

Finish

Finishes the wizard.
