Translate multiple Creo Element Direct documents in Solid Edge

  1. Run $:\Program Files\Siemens\Solid Edge 2020\Program\CreoEDDataMigration.exe.

  2. On the Creo Elements Direct Data Migration Start Page dialog box, specify a Creo Elements Direct folder, a Solid Edge folder, and set any other options, as needed. You can use the default values or change them to meet your needs.

  3. On the Creo Elements Direct Data Migration Start Page dialog box, click Next.

  4. On the Creo Elements Direct File Selection Page dialog box, select the documents you want to open, click Add to add them to the Creo Elements Direct Files To Be Migrated list, then click Next.

  5. On the Creo Element Direct Migration Saving Changes dialog box, specify the configuration file to which you want to save the remapped configuration, then click Finish.
