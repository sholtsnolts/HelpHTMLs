Solid Edge to AutoCAD Translation Wizard (Line Width Mapping)

Assists in mapping a Solid Edge line width to an AutoCAD element color. To re-map a Solid Edge line width to an AutoCAD element color, click an AutoCAD element color, and then click on a standard color button.

To add more line widths, type the width in the Add Other Widths to List Below box and click the Add button.

Do Width to Color Mapping

Allows you to do line width mapping between an AutoCAD color to a Solid Edge line width. When you select the option, you can map a line width to a color. If you do not select the option, Solid Edge uses the colors in the draft file when exporting the file to AutoCAD.

Standard Colors

Displays the seven standard AutoCAD color numbers. AutoCAD supports 256 colors, which are represented by an integer value ranging from 1 - 256.

Non-Standard Colors

Allows you to set a Solid Edge line width to an AutoCAD color value greater than seven.

Set

Sets a Solid Edge line width to an AutoCAD color value greater than seven. To set the value, select the AutoCAD color adjacent to the Solid Edge line width and enter the AutoCAD value you want to set.

Additional Width

Allows you to add new width values to the list of Solid Edge line widths. This allows you to map Solid Edge line widths to an AutoCAD color value greater than seven.

Add

Adds new line widths to the list of Solid Edge line widths. When you add a new width, the default AutoCAD Color value is six.

AutoCAD Color

Displays the current mapping of Solid Edge line widths to AutoCAD element colors.

Solid Edge Line Width

Specifies the Solid Edge line width that you want to assign to the selected AutoCAD element. By default, the Solid Edge line width values are derived from the standard list of widths provided by Solid Edge Draft. In addition, this list of values contains any unique widths you have used in the file while in Solid Edge Draft.

To delete line widths, other than the default line widths, select the line width and click the DELETE key on your keyboard. You cannot select multiple rows for deletion and you cannot delete default line widths.

Reset

Displays a dialog box where you can reset wizard options to their default values. You can reset options on the active page or you can reset options on all pages in the wizard. If you select Reset all options, the system resets options for all pages and displays the first page of the wizard.

Cancel

Discards all input and exits the command.

Back

Moves to the previous dialog box of the Translation Wizard.

Next

Moves to the next dialog box of the Translation Wizard.

Finish

Finishes the wizard and returns to the Save As dialog box.
