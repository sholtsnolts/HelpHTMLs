AutoCAD To Solid Edge Translation Wizard- Step 1 of 8

Configuration File

Displays the name of the AutoCAD configuration file that you want to use for the imported file. This file contains the user-defined settings, such as font and line type mapping. The configuration file used by default is seacad.ini which is located in the Solid Edge 2023\Preferences\Translators folder. This file name and folder are reserved for Solid Edge Draft and cannot be overwritten.

User-defined configuration files that are used across network drives or defined through network drives are supported.

Import 2D Geometry

Translates the ACIS data to 2D geometry.

Note:

If you select Import 2D Geometry:

  • The Import Solid options are disabled.

  • Sketch Plane options are enabled and the Top (XY) plane is selected.

  • The Preview pane displays only the Model tab.

  • All layout tabs are disabled.

  • Import Preview Display is displayed in the Translate Sheet field and the drop list is disabled.

Draft

Specifies draft as the 2D translation target environment.

Part Sketch

Specifies part sketch as the 2D translation target environment.

Assembly Sketch

Specifies assembly sketch as the 2D translation target environment.

Sketch Plane

Defines the part of assembly sketch plane you want to use when translating 2D geometry to the sketch environment.

Import Solids

Translates the ACIS data to either a part file or an assembly file.

Note:

If you select Import Solids:

  • The Import 2D Geometry options are disabled.

  • Sketch Plane options are disabled.

  • The As Part file and As Assembly file options are enabled.

  • The SAT Translator Options are enabled.

  • All layout tabs are disabled.

  • Import Preview Display is displayed in the Translate Sheet field and the drop list is disabled.

  • Pages 3 ̵ 7 of the wizard are skipped.

As Part file

Translates ACIS data as a part file.

As Assembly file

Translates ACIS data as an assembly file.

SAT Translator Options

Displays the Import Options for ACIS (.sat) dialog box.

Show Layers

Displays a list of layers found in the AutoCAD file being imported. By selecting a layer, you can preview the layer before importing the file. This gives you an idea of how the different layers will appear in the draft file.

You can control the display of the selected layers when you select one or more layers in the Show Layers area, and then right-click and select Show or Hide.

Preview

Previews the AutoCAD file prior to translation. When you click the button, the layer information is populated and the image in the View area is updated. Only the layers selected in the Show Layers area are displayed. If you page through the wizard and return to this page, you must re-select the Preview button to refresh this information.

Refresh

Redraws the preview to reflect any changes made to the current layer settings.

Background Color

Displays a list of background colors.

Zoom Area

Zooms into an area in the window. The two points you place define the view.

Zoom

Reduces or enlarges the display of geometry around a specified point in the window.

Fit

Fits all elements to the window.

Pan

Enables you to move in any direction from a specific point on the model to see other areas of the model.

Custom View

Maps the current view orientation in the active AutoCAD file to a standard view orientation provided by AutoCAD. The drop down list contains ten orientations from which you can choose. The displayed value defaults to the orientation in the AutoCAD file. For example, if the vector for the view is 0, 0, 1, it is recognized as the top view, and Top is displayed. If the view is unknown or is not defined as one of the ten standard AutoCAD views, the displayed value defaults to Custom View.

When you select a view orientation, the preview updates to reflect the selected orientation and the orientation name is stored in the Name= parameter in the configuration file.

This option is available only for model space drawings.

Translate Sheet(s)

Translates a selected sheet or all sheets in the file.

If you select a sheet that is not listed in the AutoCAD file, all sheets are imported. If you select a paper space sheet, the corresponding graphics in the model space are also translated.

If you display the model sheet and then set the Translate Sheet Option to Import Preview Display, Solid Edge only imports the layers displayed in the model sheet. Any layers not displayed are not imported. If you do not set the option to Import Preview Display, Solid Edge imports the entire DWG file.

Write Solid Bodies to SAT File

Translates the AutoCAD solid bodies to .SAT file format.

Reset

Displays a dialog box where you can reset wizard options to their default values. You can reset options on the active page or you can reset options on all pages in the wizard. If you select Reset all options, the system resets options for all pages and displays the first page of the wizard.

Cancel

Discards all input and exits the command.

Next

Moves to the next dialog box of the Import Translation Wizard.

Finish

Finishes the wizard and returns to the Open File dialog box.
