Migrate Inventor hole definitions to Solid Edge

  1. On the Inventor Data Migration Start Page, click Hole database migration tool to display the Inventor Hole Database Migration dialog box.

  2. On the Inventor Hole Database Migration dialog box, in the Location of Inventor hole database, specify the name and location of the Inventor hole database. Use the Browse button to locate the folder and database.

  3. On the Inventor Hole Database Migration dialog box, in the Location of migrated files, specify the location to which the files will be migrated. Use the Browse button to locate the folder.

  4. On the Inventor Hole Database Migration dialog box, in the Solid Edge hole database Inch template, specify the template for Solid Edge hole database Inch standard. Use the Browse button to locate the folder.

  5. On the Inventor Hole Database Migration dialog box, in the Solid Edge hole database Metric template, specify the template for Solid Edge hole database Metric standard. Use the Browse button to locate the folder.

  6. On the Inventor Hole Database Migration dialog box, click the Hole mapping button to display the Hole Mapping dialog box. Correct mapping is added automatically after migration.

  7. On the Inventor Hole Database Migration dialog box, click Populate Default if you are not sure of the paths used for Inventor and Solid Edge hole database, and templates. This option requires Inventor and Solid Edge, with active licenses, to be installed on your machine.

  8. On the Inventor Hole Database Migration dialog box, click Run to migrate the hole definitions.
