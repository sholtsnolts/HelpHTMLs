Solid Edge to JT Translator dialog box

Solid Edge documents

Specifies the name(s) of the Solid Edge document(s) that you want to translate. You can use the Browse button to locate the documents.

Output folder

Specifies the output folder where the translated documents will be stored. You can use the Browse button to locate the output folder.

Translate

Exports the specified Solid Edge document(s) to JT format.

Options

Displays the Solid Edge to JT Translation Options dialog box.
