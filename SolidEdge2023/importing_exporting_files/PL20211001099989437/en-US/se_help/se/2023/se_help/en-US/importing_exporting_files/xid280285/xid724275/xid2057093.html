Inventor Material Database Migration dialog box

Inventor material database

Specifies the name and location of the Inventor material database. The default location is the location that is mentioned in the active project.

Location of migrated files

Specifies the location to which the files will be migrated. The default location is the location that is mentioned in the active project.

Populate Default

Populates the default paths for Inventor and Solid Edge material database. This option requires Inventor and Solid Edge, with active licenses, to be installed on your machine. The default location is the location that is mentioned in the active project.

Run

Starts the material database migration.
