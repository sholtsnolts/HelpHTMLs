IGES Import Translation Wizard (General)

Assists in defining the view orientation for the wireframe data contained in the IGES 2-D file being translated.

View Orientation

Specifies how the XYZ data found in the IGES 2-D file is applied to the Solid Edge Draft file being created. The default value is top. The optional values are Bottom, Front, Back, Right, Left, Isometric, and Right Isometric. These values are all based on the third angle projection standard.

Preview

Lists the entities, entity names, and occurrences found in the IGES file being opened. This is used to confirm the template selection, or to make informed decisions on other parameter selections.

Back

Moves to the previous dialog box of the IGES Import Translation Wizard.

Next

Moves to the next dialog box of the IGES Import Translation Wizard.

Finish

Finishes the wizard and returns to the Open File dialog box.
