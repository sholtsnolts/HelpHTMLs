Solid Edge to AutoCAD Translation Wizard (Font Mapping)

Assists in mapping a Solid Edge font to an AutoCAD font. To remap a Solid Edge font, click on an AutoCAD font, and then enter the new AutoCAD font name.

To add a new font to the list, click on a Solid Edge font from the drop-down list and click the Add button.

Remove dependency of Solid Edge Fonts

Removes the dependency of Solid Edge fonts during export. When you select this option, Solid Edge substitutes the Mechanical symbol font Solid Edge ISO\ANSI GDT Symbols.ttf file with the Autodesk mechanical symbol font GDT.ttf file.

Additional Solid Edge Fonts

Displays a list of additional Solid Edge fonts that you can add to the list of default fonts.

Add

Adds the new font to the list of available Solid Edge fonts.

AutoCAD Font

Displays a list of available AutoCAD fonts. The list of fonts is defined by the active configuration file. The list also includes AutoCAD fonts found in the AutoCAD file being translated. Fonts found in the AutoCAD  file precede the list of fonts found in the Configuration file.

Solid Edge Font

Displays a predefined set of Solid Edge fonts, plus any unique fonts that are being used in the file that you are exporting. This value is for display only and you cannot edit it.

Reset

Displays a dialog box where you can reset wizard options to their default values. You can reset options on the active page or you can reset options on all pages in the wizard. If you select Reset all options, the system resets options for all pages and displays the first page of the wizard.

Cancel

Discards all input and exits the command.

Back

Moves to the previous dialog box of the Translation Wizard.

Next

Moves to the next dialog box of the Translation Wizard.

Finish

Finishes the wizard and returns to the Save As dialog box.

How do I
Export one or all sheets to AutoCAD
