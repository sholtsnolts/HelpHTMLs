IGES Export Translation Wizard (Fonts)

Assists in mapping a Solid Edge font to an IGES font. To re-map a Solid Edge font, click on an IGES font, and then enter the new IGES font name.

To add a new font to the list, click on a Solid Edge font from the drop-down list and click the Add Font button.

Additional Solid Edge Fonts

Displays a list of additional Solid Edge fonts that you can add to the list of default fonts.

Add Font

Adds the new font to the list of available Solid Edge fonts.

Iges Font

To delete fonts from this list, click on the font you want to delete and press the DELETE key on your keyboard. You cannot select multiple rows for deletion.

Solid Edge Font

Specifies the Solid Edge font to be mapped to the selected IGES font. The configuration file supplies the default font values.

Back

Moves to the previous dialog box of the IGES Export Translation Wizard.

Next

Moves to the next dialog box of the IGES Export Translation Wizard.

Finish

Finishes the wizard and returns to the Save As dialog box.
