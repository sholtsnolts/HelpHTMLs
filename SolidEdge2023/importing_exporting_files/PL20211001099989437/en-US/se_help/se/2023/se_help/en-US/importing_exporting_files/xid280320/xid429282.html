Export dialog box

Use this dialog box to control the level of detail of the exported model. It is available when exporting Solid Edge files to the following formats:

  • Viewer Document (*.sev)

  • 3D Adobe Acrobat (*.pdf)

  • Universal 3D (*.u3d)

Moving the slider to the farthest left position yields the smallest file size with the coarsest tessellation for the Solid Edge model. Moving the slider to the right increases the fineness of the tessellated Solid Edge exported model.

As the fineness of the exported model increases, the file size will increase and the performance of the viewer decreases when loading and manipulating the model.  The slide provides the ability to control the balance between the desired fineness of the exported model, as well as the exported file size and the desired performance while in the viewer.
