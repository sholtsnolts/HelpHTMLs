Migrating Creo Elements Direct material definitions to Solid Edge

When you migrate any Creo Elements Direct part files, the material definition is extracted. This material is assigned to the Solid Edge document if the material name exists in the Solid Edge Material.mtl file. To ensure that the Creo Elements Direct material name exists in the Solid Edge Material file, Solid Edge provides a tool to convert the Creo Elements Direct material file to the Solid Edge Material.mtl file.

Note:

Converting the Creo Elements Direct Material data base is optional. However, if you do not convert the table or improperly set up either the Creo Elements Direct or Solid Edge material table location, no material is mapped to Solid Edge.

The CEDmat2SEMat.exe utility is delivered as part of the server-side application to the Solid Edge Program folder. The executable creates a new Solid Edge Material.mtl file rather than appending an existing .mtl file. The executable organizes material definitions by placing all similar materials, such as steel and iron into a single category.

Before running the utility, you should consider:

  • Make a copy of the Solid Edge Material.mtl file.

  • After successfully converting the material table ensure that the Solid Edge\Options\File location pointer is pointing to the appropriate file.

  • Material database migration must be carried out before Creo Elements Direct files migration. Also material database migration can be carried out successfully only on the machine, where both Creo Elements Direct and Solid Edge are installed.

To migrate the material database

  1. Go to your Solid Edge installation Program folder. For example, C:\Program Files\Siemens\Solid Edge 2020\Program.

  2. Shift+Right-click -> click Open PowerShell Window here.

  3. In the Windows Powershell window, type CEDmat2SEMat.exe and press the Tab button.

  4. Enter the .csl file path and press Enter.

    The material database is migrated to the location where Solid Edge material file is located. The file name of the migrated database is the same as original Creo Elements Direct database, for example, Werkst.csl.
