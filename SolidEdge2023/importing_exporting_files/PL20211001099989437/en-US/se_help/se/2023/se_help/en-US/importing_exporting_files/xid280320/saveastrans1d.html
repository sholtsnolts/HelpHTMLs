Save As Translated dialog box

Specifies the name, location, and format for a document when you save it to a non-Solid Edge file type.

Address bar

Navigates to a folder or library to which you want to save the document.

Note:

If you want to browse a network share, you must map the share to a network drive.

Search

Finds an item in the current folder or library. The search begins as soon as you start typing.

Toolbar

Contains options for changing the appearance of your files and folders, organizing, creating new folders, and accessing Windows Help and Support. The options available in the toolbar change to show only the options that are relevant.

Navigation pane

Displays libraries, folders, and disk drives on your computer.

Files list

Displays the contents of the folder or library selected.

File name

Specifies the name of the document.

Save as type

Specifies the non-Solid Edge document type to which you save the document.

Options

Displays the translation options for various file types.
