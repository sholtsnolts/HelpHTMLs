Migrate Inventor material definitions to Solid Edge using the DOS window

Solid Edge provides INVmat2SEMat.exe utility to the Solid Edge program folder. The executable creates a new Solid Edge Material (.mtl) file rather than appending materials in existing Material.mtl file. The category of the material is maintained from Inventor database to Solid Edge material database.

Before running the utility, consider the following:

  • We recommend to make a copy of the Solid Edge Material.mtl file.

  • After successfully converting the material table ensure that the Solid Edge\Options\File location pointer is pointing to the appropriate file.

  • Before starting a migration ensure that Inventor is pointing to the same material database folder that was previously converted.

To migrate the material database:

Note:

The material file which user is going to migrate, must be added in Material Library list in active Project file.

You can find the path of the Inventor material library by hovering over that library name in the Material Browser dialog box or in the Projects dialog box. Only those material libraries can be migrated that can be applied to Inventor model.

  1. Open a DOS window.

  2. Enter the path for the INVmat2SEMat.exe . For example, “C:\Program Files\Siemens\Solid Edge 2023\Program\INVmat2SEMat.exe .”

  3. Set the following:

    • Path 1 — Location and name of the Inventor materials file in double quotes.

      For example, C:\>”C:\Program Files\Solid Edge 2023\Program\INVmat2SEMat.exe” "C:\Users\Public\Documents\Autodesk\Inventor <Version>\Design Data\Materials\ InventorMaterialLibrary.adsklib".

    This creates the Inventor materials file(.mtl) and place it in C:\Program Files\Siemens\Solid Edge 2023\Preferences\Materials folder.
