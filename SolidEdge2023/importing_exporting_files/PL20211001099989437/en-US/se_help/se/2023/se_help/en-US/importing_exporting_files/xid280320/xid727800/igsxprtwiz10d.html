IGES Export Translation Wizard (Hatchstyle)

Assists in mapping Solid Edge cross hatch styles to IGES cross hatch styles.

Additional Solid Edge Hatch Styles

Displays a list of additional Solid Edge hatch styles that you can add to the list of default hatch styles.

Add Item

Adds the new hatch style to the list of available Solid Edge hatch styles.

Iges Hatch

Specifies the IGES corresponding hatch styles. You can select the IGES hatch style to enter edit mode.

Solid Edge Hatch

Specifies the default Solid Edge hatch styles. User-defined hatches not permitted.

Back

Moves to the previous dialog box of the IGES Export Translation Wizard.

Next

Moves to the next dialog box of the IGES Export Translation Wizard.

Finish

Finishes the wizard and returns to the Save As dialog box.
