AutoCAD To Solid Edge Translation Wizard - Step 4 of 8

Assists in mapping AutoCAD line types to Solid Edge line types. To re-map an AutoCAD line type, click on a Solid Edge line type, and then select a line type from the drop-down list.

AutoCAD Line Types

Displays a list of default line type names defined in the AutoCAD configuration file. This list also includes any AutoCAD line types found in the AutoCAD file being translated. The line types found in the configuration file precede the list of line types found in the AutoCAD file. Duplicate line styles are listed only once.

Solid Edge Line Type

Specifies the gap sequence for the display of graphic elements used in Solid Edge Draft. By default, the mapping of AutoCAD line types found in the configuration file is maintained. For the AutoCAD line types found in the AutoCAD file, the default line type is Continuous.

To delete unused line types, click on the AutoCAD line type and click the DELETE key on the keyboard. You cannot select multiple rows for deletion.

Solid Edge Line Type Name

Displays the name for the selected Solid Edge line type.

Reset

Displays a dialog box where you can reset wizard options to their default values. You can reset options on the active page or you can reset options on all pages in the wizard. If you select Reset all options, the system resets options for all pages and displays the first page of the wizard.

Cancel

Discards all input and exits the command.

Back

Moves to the previous dialog box of the Import Translation Wizard.

Next

Moves to the next dialog box of the Import Translation Wizard.

Finish

Finishes the wizard and returns to the Open File dialog box.
