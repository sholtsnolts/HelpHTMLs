SolidWorks File Selection page

Assists in identifying the SolidWorks documents you want to migrate to Solid Edge.

SolidWorks Files

Displays a list of SolidWorks files that you can select for migration.

Folder Navigation buttons

Use to return to the last opened folder. If you select the last opened folder, use to return to the previously selected folder.

Address Bar

Displays the path of the active folder.

Select files for Migration

Displays the list of the SolidWorks documents that are available for the migration to Solid Edge. The list contains .sldasm, .slddrw and .sldprt files.

Add

Adds the selected documents to the Files selected for Migration box.

Add All

Adds all documents to the Files selected for Migration box.

Remove

Removes the selected documents from the Files selected for Migration box.

Remove All

Removes all documents from the Files selected for Migration box.

Select file(s) from all subfolders

Selects files from all of the selected subfolders.

File type filter

Specifies the types of SolidWorks documents you can migrate to Solid Edge. The list contains only documents with a SolidWorks assembly file (.sldasm), SolidWorks part file (.sldprt), or SolidWorks drawing file (.slddrw) extension.

Help

Displays help for the dialog box.

Cancel

Discards any input and exits the wizard.

Back

Returns to the previous dialog box of the SolidWorks Data Migration Wizard.

Next

Moves to the next dialog box of the SolidWorks Data Migration Wizard.
