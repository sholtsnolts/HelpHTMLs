Migrate Inventor material definitions to Solid Edge

  1. On the Inventor Data Migration Start Page, click Material database migration tool to display the Inventor Material Database Migration dialog box.

  2. On the Inventor Material Database Migration dialog box, in the Inventor material database to be migrated specify the name and location of the Inventor material database. Use the Browse button to locate the folder and database.

  3. On the Inventor Material Database Migration dialog box, in the Location of migrated files, specify the location to which the files will be migrated.

  4. On the Inventor Material Database Migration dialog box, click Populate Default if you are not sure about the database locations. This option requires Inventor and Solid Edge, with active licenses, to be installed on your machine.

  5. On the Inventor Material Database Migration dialog box, click Run to migrate the material definitions.
