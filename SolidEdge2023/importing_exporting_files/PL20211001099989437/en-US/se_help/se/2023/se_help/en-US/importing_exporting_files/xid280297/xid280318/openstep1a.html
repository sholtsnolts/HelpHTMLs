Opening STEP Documents in Solid Edge

You can open STEP documents in Solid Edge with the Open command. After you select a STEP file to open you use the Import Options for STEP dialog box to control how the STEP file is opened in Solid Edge.

Note:

When importing assemblies containing more than 600 to 700 part files you need to set the value of the Import DoNotSave=On parameter in the translators .ini file to Off. The Import DoNotSave parameter controls whether the Solid Edge part files are created during the translation, or on the initial save of the assembly file. Setting this value to off may take longer to translate but consumes less memory.

To translate multiple STEP files in a single session, use the Batch.exe file, located in the Solid Edge Version #\Custom\Batch\Bin folder. The executable requires Solid Edge be loaded on the machine where the translation occurs.

STEP.ini file

The settings on the option dialog boxes used when importing or exporting STEP files are defined in the STEP.ini file, located in the Solid Edge 2023\Preferences\Translators folder. When you make a change to a parameter in the dialog box, a new value is saved to the .ini file. You can use a text editor, such as Notepad, to set these parameters. However, if you edit this file, use extreme caution in setting these parameters. Errors made to this file can adversely affect the quality of the translation. The following is a list of parameters found in the STEP.ini file.

Parameter

Description

Export STEP Format=ap214

Specifies the STEP format to which you want to export the Solid Edge document. You can set the format to ap203, ap214 or ap242.

Export Construction Sheets=Off

Specifies that you want to export any construction geometry that exists as sheet bodies.

Export Construction Solids=Off

Specifies that you want to export any construction geometry that exists as solid bodies.

Export Construction Wires=Off

Specifies that you want to export any construction geometry that exists as wire bodies.

Export Displayed Parts=Off

Specifies that you want to export only the active model and construction geometry that is visible in the part or assembly.

Export Author=Unknown

Specifies the name of the author of the file you want to export.

Export Description=Unknown

Specifies a description of the file you want to export.

Export Organization=Unknown

Specifies the name of the organization of the file you want to export.

Export Program Name=Unknown

Specifies the name of the program for the file you want to export.

Export Path

Stores the value for the path where the documents will be exported.

Export Units=Millimeters

Specifies the units of measurement for the part being exported. You can set the units to millimeters, centimeters, inches, or meters.

Export Part Component Names as Unicode=Off

Specifies that when exporting an assembly that double byte characters are added to the STEP file in Unicode format rather than the authoring ASCII character set.

Export Assembly Simplification=All Designed

Specifies that you want to check each part found in the assembly and use the design body, if available, to create the STEP document.

Export Simplified parts=All Designed

Specifies that you want to check each part and use the design body, if available, to create the STEP document.

Export PMI=off

Specifies that you want to export all part level product manufacturing information (PMI) data to the STEP file being created. This option is only enabled when the STEP protocol is set to AP242. The AP242 protocol is used to generate machine-readable files.

Import Boolean Solids=On

Specifies you want to boolean all solid bodies together to form a disjoint solid and insert it into PathFinder as a part copy. This option is on by default.  If the option is off, all solid bodies are added as individual part copies to PathFinder.

Import Check bodies=Off

Specifies that you want to perform a full body check on the file.

Import Curves=On

Controls the importing of curves.

Import Surfaces=On

Controls the importing of surfaces.

Import Faceted Bodies=On

Controls the importing of faceted or meshed bodies.

Import Solid Bodies=On

Controls the importing of solid bodies.

Import DoNotSave

Controls whether or not assembly components are created on import, or on the initial save of the assembly.

Import Drive=C

Stores the drive where the documents for the imported file.

Import Enable Default Output Path=On

Specifies that the translator use the defined “Import Output Path= as the location to create the assembly part component files on import.

Import Imprint Curves=On

Specifies that you want to combine all curve data into a single part copy. It may be to your advantage to identify the curves that you do not need and either hide or delete them before importing the file. This option is on by default.

Import Make Base Feature=On

Specifies that you want to import the solid body as a design body.

Import Multiple Bodies As Single Part file=On

Controls the inclusion of multiple solid bodies as a single part in the imported file.

Import Output Path

Stores the path for the imported file.

Import Simplify Geometry=On

Controls the inclusion of simplified geometry in the imported file.

Import Stitch Surfaces=On

Specifies that you want all surfaces and sheet bodies to be stitched to a tolerance of 1.0e-5 meters. It may be to your advantage not to stitch the surfaces on import, but to evaluate what needs to be stitched and perform the stitch after you import the file. This option is on by default.  If the stitching operation creates a valid volume, the volume is converted to a solid.

Import PMI=off

Specifies that you want to import all product manufacturing information (PMI) data from the STEP file into Solid Edge. The option is off by default.

Import As Internal Components=off

Specifies that you want to import the non-Solid Edge file to Solid Edge as an internal component when you open the file as an assembly. No files are created on disk and the components are embedded within the file. The option is off by default.

Enable Logging=On

Turns on or off the creation of the log file. The default is on.

Importing PMI

You can use the Import PMI option on the Import Options for STEP (.stp) dialog box to specify that you want to import any part level PMI contained in the STEP AP242 file to the Solid Edge file.

There is also a corresponding Import PMI parameter in the STEP.ini file, which is set to off by default. Leaving the parameter set to off prevents part level PMI from being imported to the Solid Edge file. Setting the value to on imports all part level PMI contained in the STEP file to the Solid Edge file being created.

Note:

The import of assembly level PMI is not supported.
