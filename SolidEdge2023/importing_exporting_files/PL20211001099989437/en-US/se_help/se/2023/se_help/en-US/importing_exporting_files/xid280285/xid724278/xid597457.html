SolidWorks data migration

The SolidWorks Data Migration command migrates SolidWorks part, assembly, sheet metal and drawing files to Solid Edge.

The command requires that SolidWorks version 2012 or later and Solid Edge are installed on the same machine.

The imported data is placed as either synchronous or ordered, depending on the settings on the Open in option on the SolidWorks Data Migration Start Page.

Hole feature recognition

To create hole feature during migration, select the Create hole features on Migration option on the SolidWorks Data Migration Start Page.

Note:

Threaded shaft features are not supported.

Holes are not recognized in the following cases:

  • The Template used has an Ordered Mode set.

  • Database is not migrated.

  • Mapping is not correct.

  • Remove thread option is selected while creating threaded hole in SolidWorks.

  • Slots are not recognized.

  • Advanced holes are not recognized.

  • Threads created from Thread command are not migrated.

  • Holes inside pattern are not recognized.

    Example:

    If a pattern has six holes, only one hole will be recognized and the remaining holes will not be recognized.

Migrating SolidWorks drawing views

When you update the migrated Solid Edge part and assembly files, drawing views also get updated in the Solid Edge draft file. During the migration process a link is established between the resulting Solid Edge parts and assembly files and the migrated Solid Edge drawing file. The Solid Edge drawing views are created based on the SolidWorks drawing view location, type, and scale.

To create drawing views during migration, select the Create drawing views on Migration option on the SolidWorks Data Migration Start Page.

The following applies when migrating SolidWork files:

  • All drawing view types created in SolidWorks are supported except relative view.

  • Cutting plane lines, auxiliary view plane lines, and detail envelopes are supported.

  • Dimensions migrated from SolidWorks are bound to Solid Edge drawing view geometry.

  • Center marks and center lines are associative to the Solid Edge drawing views.

  • Hatch and solid fills are migrated maintaining their associativity to the Solid Edge drawing view.

  • View shading is maintained as part of this drawing view creation process.

  • Multi-sheet drawings are supported through this migration process.

Note:

Annotations, such as balloons and callouts, do not migrate as Solid Edge annotations, but migrate as blocks.
