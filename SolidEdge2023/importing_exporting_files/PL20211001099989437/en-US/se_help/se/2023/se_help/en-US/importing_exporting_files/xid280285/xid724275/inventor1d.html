Inventor Data Migration Start page

Assists in migrating Inventor part, assembly, and drawing files into Solid Edge.

Tools
Material database migration tool

Displays the Inventor Material Database Migration dialog box.

Hole database migration tool

Displays the Inventor Hole Database Migration dialog box.

Migration Options
Templates

Displays the Templates dialog box.

Folder Selection

Displays the Folder Selection dialog box.

Open In

Specifies the environment used while migration for Solid Edge files creation.

Synchronous Mode

Migrates the files in synchronous mode.

Ordered Mode

Migrates the files in ordered mode.

Options
Temporary file cleanup

Specifies that you want the system to delete the intermediate files such as PLMXML, x_t, XML and DXF created during the migration. When the option is checked, the files are deleted.

Note:

Already migrated files won't get migrated only if intermediate data and migrated files are not deleted and new files are being migrated at the same location.

Tip:

Keep this option unchecked when migrating files that are commonly used in different assemblies and drawings.

Create hole features on Migration

Specifies that you want to create hole features during the migration. After migration, you can modify some of the parameters for hole features. This option only applies when you are migrating files using synchronous mode.

Create assembly constraints on Migration

Specifies that all supported constraints from the Inventor assembly are migrated.

Create drawing views on Migration

Specifies that you want to create Solid Edge drawing views that are linked to the migrated 3D model. If the option is not checked, the migrated drawing file contains dumb DXF geometry with no link to the 3D model.

Export Inventor draft files

Specifies how Inventor drawing documents (.idw and .dwg) are migrated to Solid Edge. The values for this option are:

  • PDF – Exports the Inventor draft file to .pdf format.

  • DXF – Exports the .dxf file from Inventor and imports the .dxf file into Solid Edge Draft.

  • Both – Exports both a .dxf and .pdf file.

Note:

Migrated drawings will have drawing views linked to a 3D model only when the DXF option is selected. For the other two options, the drawing will have just lines, curves, and blocks. When drawing and PDF option is selected, only drawings are migrated and not 3D models linked to inventor drawings.

Saved Settings
Save

Specifies the name of the saved setting. You can save the settings that you are going to use for the current migration by creating a profile. It will be helpful for future migrations. You can select a profile from the Save list.

By default, two sets of profiles are already available in the Save list:

  • Migration for active files

  • Migration for archival

Display this dialog for every Migration

Specifies that you want to display the dialog box for every migration.

Note:

If you often use the same set of settings, you can access the Inventor File Selection Page directly by keeping this option unchecked.

Save

Saves the current settings to the name you provide.

Delete

Deletes the current saved setting.

Help

Displays help for the dialog box.

Reset

Resets the options to the default values.

Cancel

Discards any input and exits the wizard.

Next

Moves to the next dialog box of the Inventor Data Migration Wizard.
