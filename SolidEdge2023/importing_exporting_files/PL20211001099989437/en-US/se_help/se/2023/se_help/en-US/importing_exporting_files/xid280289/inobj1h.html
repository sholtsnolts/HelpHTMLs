Insert an object

With Linking and Embedding Object (OLE) technology, you can insert foreign objects into your document. You can create and embed a new object, or you can insert an object from another file.

Create and insert a new object

  1. Choose the Sketching tab (or Home tab)→Insert group→Insert Object command .

  2. In the Object dialog box, click Create New.

  3. From the Object Type list, select the type of object you want to insert.

    The application associated with the object type is opened.

    Example:

    If you select Microsoft Office PowerPoint Slide as the object type, the PowerPoint application is opened. If you select Bitmap Image, Microsoft Paint is opened.

  4. In the application window, create the information you want to add to the drawing.

  5. Move the cursor back into the Draft document window, and then click to place the object you created.

Insert an object from another file

  1. Choose the Sketching tab (or Home tab)→Insert group→Insert Object command .

  2. In the Object dialog box, click Create From File.

  3. Do one of the following:

    • In the File Name box, type the name of the file that contains the object you want to insert.

    • Click the Browse button and use the Browse File dialog box to find the file that contains the object.

  4. Move the object where you want to place it and click.

Tip:

You can edit the object properties, such as its position on the document, scale, and border, using the Properties command on the object shortcut menu.
