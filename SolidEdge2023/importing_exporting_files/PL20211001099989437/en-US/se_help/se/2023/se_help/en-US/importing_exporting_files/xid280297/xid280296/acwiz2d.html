AutoCAD To Solid Edge Translation Wizard - Step 2 of 8

Assists in mapping an AutoCAD document to a Solid Edge document.

Use Template File Units

Specifies the units for the file being imported are retrieved from the specified template file.

Override Template File Units

Specifies the import units for the file are defined from the list rather than by the units defined in the template file.

Generate Log File

Produces a log file that contains warnings and information about the files involved in translation.

The log file is named in the standard convention of file_name.log and is named in the standard convention of file_name.log. It is written to the same folder which Solid Edge writes or is currently reading the AutoCAD file from. Solid Edge will automatically overwrite an existing log file is the same name is specified. You must have write permission for the target folder to be able to create a log file.

Import Dimensions as Dimensions

Imports the AutoCAD dimensions as Solid Edge dimensions.  If you do not select this option, AutoCAD dimensions are imported as simple graphics and text boxes. For more information, see the Opening AutoCAD Documents in Solid Edge topic.

Import Blocks as Groups

Imports the AutoCAD blocks as groups. If you do not select this option, AutoCAD blocks are imported as blocks. Selecting this option may affect the size of your Solid Edge file and have an adverse effect on performance. For more information, see the Opening AutoCAD Documents in Solid Edge topic.

Solid Edge Draft Template

Displays the Solid Edge draft template to be used as a starting template for the Solid Edge draft document. You can click the Browse button to select the template you want to use.

Solid Edge Part Template

Displays the Solid Edge part template to be used as a starting template for the Solid Edge part document during 3D data translation. You can click the Browse button to select the template you want to use.

Solid Edge Assembly Template

Displays the Solid Edge assembly template to be used as a starting template for the Solid Edge assembly document during 3D data translation. You can click the Browse button to select the template you want to use.

Reset

Displays a dialog box where you can reset wizard options to their default values. You can reset options on the active page or you can reset options on all pages in the wizard. If you select Reset all options, the system resets options for all pages and displays the first page of the wizard.

Cancel

Discards all input and exits the command.

Next

Moves to the next dialog box of the Import Translation Wizard.

Finish

Finishes the wizard and returns to the Open File dialog box.
