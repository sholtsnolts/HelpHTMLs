Migrate SolidWorks material definitions to Solid Edge

  1. On the SolidWorks Data Migration Start Page, click Material database migration tool to display the SolidWorks Material Database Migration dialog box.

  2. On the SolidWorks Material Database Migration dialog box, in the SolidWorks material database to be migrated specify the name and location of the SolidWorks material database. Use the Browse button to locate the folder and database.

  3. On the SolidWorks Material Database Migration dialog box, in the Location of migrated files, specify the location to which the files will be migrated.

  4. On the SolidWorks Material Database Migration dialog box, click Populate Default if you want to populate the default paths for SolidWorks and Solid Edge material database. This option requires SolidWorks and Solid Edge, with active licenses, to be installed on your machine.

  5. On the SolidWorks Material Database Migration dialog box, click Run to migrate the material definitions.
