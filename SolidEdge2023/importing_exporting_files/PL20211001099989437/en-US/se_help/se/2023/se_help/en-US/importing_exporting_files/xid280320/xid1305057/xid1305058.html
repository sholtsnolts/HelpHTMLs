Saving Solid Edge documents to 3MF format

3D Manufacturing Format (3MF) is an XML-based data format designed for 3D printing. 3MF format contains information such as material and color, which cannot be represented in STL format.

You can use the Export Options for 3MF dialog box to specify how you want to save Solid Edge documents to 3MF format. The dialog box specifies what you want to export, tolerance options for the data being exported, along with header information, such as the author and file description.

For more information about using 3MF files in Solid Edge, see Selecting a file format for additive manufacturing.
