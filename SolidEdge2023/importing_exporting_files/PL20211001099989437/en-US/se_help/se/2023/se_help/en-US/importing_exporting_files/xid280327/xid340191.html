Make Base Feature command

Makes a base feature from the selected construction.

The command is only available:

  • In the Ordered environment.

  • If the Make Base Feature on the Import Options dialog box is not selected when you import the model.

  • If the Make Base Feature on the Import Options dialog box is selected and the model being imported contains multiple bodies.
