Update a linked part copy

  1. In the PathFinder tab, select the Part Copy feature you want to update.

  2. Right-click and choose Update Link.

Tip:

The Update Link command is available only when a linked part copy is out of date with respect to the original part. A part copy is linked to the original part only if you used the Link To File option when you created the copy.
