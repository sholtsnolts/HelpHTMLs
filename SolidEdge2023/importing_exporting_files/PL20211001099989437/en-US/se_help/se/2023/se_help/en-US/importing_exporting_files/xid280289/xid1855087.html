Importing manufacturer parts

You can quickly find and import components or parts into your Solid Edge models, using Components by TraceParts, Components by 3Dfind.it, and Search 3Dfind.it. Each of the commands are accessible from the File menu→Tools tab.

  • Components by TraceParts. This command opens a catalog that contains millions of manufacturer parts. You can search the catalog by keywords, order number, or name.

  • Components by 3Dfind.it. This command opens a website that searches millions of manufacturer models.

    Note:

    To enable 3Dfind.it functionality in the Teamcenter-managed environment, set the Teamcenter preference SEEC_Enable_3DFindit value to 1.

    For a full description of the preference, see the Teamcenter Preferences section of the Teamcenter Integration for Solid Edge (SEEC) Guide for Users and Administrators.

    Learn about inserting components from 3Dfind.it and replacing a part with a 3Dfind.it part.

  • Search 3Dfind.it. This command, available when you have a model open in Solid Edge, searches supplier catalogs for models similar in shape to the active or selected model. You can use filters in 3Dfind.it to search using a variety of criteria.

    If the currently active or selected model originated from 3Dfind.it, the search functionality opens the same part in 3Dfind.it so that you can select from variations of the same part.

    You can also search 3Dfind.it using a 2D sketch. See Search by sketch with 3Dfind.it.
