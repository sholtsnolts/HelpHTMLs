Migrating Inventor hole definitions to solid edge

The Inventor hole database contains two Excel sheets that is formatted so that it is recognized by Inventor. The hole database migration tool reads the Inventor hole database and creates a new Excel file in a format that is recognized by Solid Edge.

The Hole database migration tool on the Inventor Data Migration Start Page supports migration of hole definitions from Inventor to Solid Edge.

After the hole database migration is completed and correct mapping is provided, the holes are recognized after the Inventor files migration. For more information, see Hole feature recognition.
