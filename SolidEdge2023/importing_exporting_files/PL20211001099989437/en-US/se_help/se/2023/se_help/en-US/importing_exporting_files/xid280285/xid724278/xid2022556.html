Migrate SolidWorks hole definitions to Solid Edge

  1. On the SolidWorks Data Migration Start Page, click Hole database migration tool to display the SolidWorks Hole Database Migration dialog box.

  2. On the SolidWorks Hole Database Migration dialog box, in the SolidWorks hole database to be migrated, specify the name and location of the SolidWorks hole database. Use the Browse button to locate the folder and database. The default location is C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS\Toolbox\data utilities\lang\English\swbrowser.sldedb. You can migrate only this database file using the Hole database migration tool.

  3. On the SolidWorks Hole Database Migration dialog box, use the Hole database standards to be migrated option to select or deselect the standards you want to migrate to Solid Edge.

  4. On the SolidWorks Hole Database Migration dialog box, click the Hole mapping button to display the Hole Mapping dialog box. Use the dialog box to map each migrated hole database standard to the corresponding Solid Edge standard.

  5. On the SolidWorks Hole Database Migration dialog box, click Populate Default if you want to populate the default paths for SolidWorks and Solid Edge hole database. This option requires SolidWorks and Solid Edge, with active licenses, to be installed on your machine.

  6. On the SolidWorks Hole Database Migration dialog box, click Run to migrate the hole definitions.
