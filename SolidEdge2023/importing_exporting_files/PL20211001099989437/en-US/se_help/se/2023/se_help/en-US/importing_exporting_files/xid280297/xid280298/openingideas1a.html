Opening I-DEAS Documents in Solid Edge

You can open I-DEAS documents in Solid Edge with the Open command. On the Open File dialog box, after you select the I-DEAS document you want to open, click the Options button to display the Import Options for I-DEAS (.XPK) dialog box. You can use the dialog box to control how the I-DEAS document is opened in Solid Edge.

Note:

Solid Edge version 15 supports I-DEAS version 10 and 10M3 PTF1, but does not support version 9. Solid Edge version 16 supports I-DEAS version 11.
