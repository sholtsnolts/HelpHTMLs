Edit Links command

Displays links to other documents.

You can use the Links dialog box to do the following:

  • Load and unload source documents.

  • Change source documents in the Draft environment.

  • Open source documents.

In a Teamcenter-managed environment, you can click Change Source to access managed folders and documents and select another file to replace the existing one. The new file can be a new revision of the existing item or from another item. Once you select a new file or revision, the new file is downloaded to cache and the link is updated to point to the new file or revision. Any drawing views or parts lists that change because of the selection are given a status of out-of-date and you should regenerate the view or parts list. This same behavior occurs if the new file is a different revision of the original file and the filenames are the same.
