block label

A block label consists of a text attribute name paired with a value. A block label can be added to the source block definition with the Block Label command. A block label can be defined with a fixed value, or it can prompt for an instance-specific value at the time the source block is placed on the drawing.

The AutoCAD equivalent: of a block label is a block attribute.
