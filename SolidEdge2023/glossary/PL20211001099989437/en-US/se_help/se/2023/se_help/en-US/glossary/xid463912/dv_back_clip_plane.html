back clipping plane

In a drawing view, a back clipping plane trims away all geometry behind it to simplify the drawing view. The back clipping plane is defined by specifying a drawing view display depth using the Set Drawing View Depth command on the selected view's shortcut menu.

Once defined, if the model grows or shrinks, the clipping plane remains in the same location relative to the 3D model origin.
