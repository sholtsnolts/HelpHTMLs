List of Values (LOV)

Predefined list of allowable entries used to populate property values as presented in common property dialog boxes within Solid Edge Embedded Client or Solid Edge Structure Editor. Teamcenter often requires a specific format for property values, rather than user-defined text entries. Lists of values are used to control data entry ensuring consistent data input into Teamcenter.

Lists of entries are created either by referencing existing data or by entering customized site data. Once created, the list of values is implemented throughout the interface by attaching the list of values to one or more properties.
