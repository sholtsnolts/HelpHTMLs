wildcard search

Wildcard expressions allow you to search a for one or more characters using a question mark or asterisk. For example, a wildcard search of pro* would find all topics that had any words beginning with the letters "pro," such as proactive, protrusion, professional, and so forth. Using a question mark as a wildcard replaces any single character, so a search for 800? would find all topics with the numbers 8001, 8002, 8003, 800a, and so forth.
