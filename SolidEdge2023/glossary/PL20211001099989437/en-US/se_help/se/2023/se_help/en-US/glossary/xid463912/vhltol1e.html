Drawing View VHL Tolerance

Generally, letting Solid Edge determine the VHL tolerance provides the best results.  You should only clear this check box and set VHL tolerance manually if you are having problems with drawing view quality.  If you have drawing view problems and want to see if changing the VHL tolerance resolves them, increase the setting one value at a time and use the lowest number that provides satisfactory results.  A higher VHL tolerance setting than necessary may degrade drawing view update performance.
