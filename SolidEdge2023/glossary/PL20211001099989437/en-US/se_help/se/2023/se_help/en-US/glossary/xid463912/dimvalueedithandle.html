dimension value edit handle

A graphical handle that you can use to modify a dimension whose value controls the size, orientation, or location of the model or of a 2D element.

The dimension value edit handle is displayed when you click the text of an existing dimension.

The handle displays different controls, depending upon whether it belongs to a model, a sketch, or a drawing.
