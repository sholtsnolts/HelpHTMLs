Reference set

Reference sets control the content from a component that is displayed in an assembly. NX component parts contain such things as construction geometry and sketches in the part that are used to generate the solid model. When you add the part to an assembly, you might only want to see the solid body without the construction objects. You can use reference set commands and options to control the display of a component or subassembly part in a higher level assembly.  For example you might create a reference set named Model that would define the active design body.
