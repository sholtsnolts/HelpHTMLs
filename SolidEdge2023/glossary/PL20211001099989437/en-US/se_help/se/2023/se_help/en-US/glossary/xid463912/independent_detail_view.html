independent detail view

Independent detail views are not updated when the source drawing view is updated, nor do changes you make in the independent detail view affect the source drawing view. An independent detail view can be used to show or hide parts, display hidden lines, or draw in the view without affecting the parent geometry or drawing view.

An independent detail view is created with the same drawing view properties as the source view, but you can change its properties after placement on the drawing.
