weldment assembly

An assembly document that has been marked as a weldment using the Weldment Assembly command. When you set the Mark This Assembly as a Weldment option on the Weldment Assembly dialog box, commands that let you construct weldment features and material addition features are activated.

For more information, see the Weldments in assemblies help topic.
