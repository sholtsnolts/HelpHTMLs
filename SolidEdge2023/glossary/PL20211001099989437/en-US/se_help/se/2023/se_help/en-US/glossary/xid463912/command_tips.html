command tip

A small text box that provides information about Solid Edge interface components (for example, reference planes) or concepts as you use them.  Each command tip is displayed once and does not display again unless you click Reset Tips on the Helpers page of the Options dialog box.  

On the command tip are the following controls:

  • Click the ? to display the related help topic.

  • Click the X to dismiss the tip immediately.
