offset

Sets the display type for symbols that specify intermittent weld operations on both sides of the joint. These options are inactive unless a weld type has been specified for both sides of the weld joint. You can specify whether the welds are to be placed offset (staggered) or aligned.

Aligned

Sets the weld symbol type to be aligned.

Offset

Sets the weld symbol type to be offset.
