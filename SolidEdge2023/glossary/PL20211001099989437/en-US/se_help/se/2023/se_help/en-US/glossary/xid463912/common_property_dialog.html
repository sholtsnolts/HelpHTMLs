common property dialog box (CPD)

Generic term for a collection of dialog boxes where you can view the number of documents in a transaction with Teamcenter, address the properties that are mapped, and upload or check in a document to the managed environment.

A common property dialog box is displayed when you choose the New, Save, Save As, Upload, Check In, Close or Revise commands within the Teamcenter-managed environment. Some examples of common property dialog boxes are the New Document and Upload dialog boxes.
