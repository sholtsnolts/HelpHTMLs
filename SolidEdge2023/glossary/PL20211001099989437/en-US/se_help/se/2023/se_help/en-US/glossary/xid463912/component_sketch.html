component sketch

A sketch that is used to define the 2-D geometry for a pre-defined component in a virtual assembly. You create component sketches using the Component Sketch command in a part, sheet metal, or assembly document. You can create the graphics manually by drawing lines, arcs, circles, and so forth, or you can create a 2-D image from the 3-D model using the Component Image command. You can also use a combination of graphics that you create manually and graphics created with the Component Image command.
