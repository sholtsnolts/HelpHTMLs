Web Publisher Wizard (Save Archive)

At this step, choose the file name and location for the web page.

Wizard Options
Save self-extracting executable to:

Enter the file name and location for the self-extracting archive file. You do not need to specify a file extension because a .exe or .ex_ file extension will be automatically added.

Example: C:\My Documents\web-pages\test\assembly - This will generate an archive named assembly.exe in the directory C:\My Documents\web-pages\test.

Include IPA WebView viewer

All web pages require the IPA WebView viewer (an ActiveX control) to be installed on the machine before viewing the page. For the convenience of the recipient of the published pages, this control can be bundled into the self-extracting archive and delivered in the e-mail message. It will be extracted and installed when the e-mail recipient opens the attachment.

Encrypt attachment to pass through firewalls

Many corporate firewalls strip executable files or archive files containing executable or JavaScript entries prior to delivering the message to the intended recipient. To avoid modification to the self-extracting archive, IPA will encrypt the file and change the extension to .ex_. This requires the user to rename the file back to .exe prior to running the self-extractor.
