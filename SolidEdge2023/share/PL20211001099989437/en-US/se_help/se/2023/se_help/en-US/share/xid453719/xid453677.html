Solid Edge Mobile Viewer application

The Solid Edge Mobile Viewer is an application for viewing Solid Edge designs on mobile viewing devices, such as a tablet device

The mobile viewer lets you rotate, zoom, and pan views of your designs, and you can selectively hide and show components of the designs. You can save and email snapshots of the views you create.

A special Solid Edge Viewer file type exists to represent your designs for the mobile viewer. The files are lightweight, containing only view information. You create them in Solid Edge with either the Save As command or the Save for Tablet command. Solid Edge Viewer file names have .sev extensions.

The Solid Edge Mobile Viewer for the iPad is available at the Apple App Store, and Solid Edge Mobile Viewer for Android devices is available at the Google Play Store
