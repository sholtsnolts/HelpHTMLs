Web Publisher Wizard (Page Layout Information)

The information page allows you to enter the header information for the web page to be generated.

Wizard Options
Web Page Title

Enter the title for the generated web page. The title will appear in the header of the web page.

Web Page Author

Enter the author of the generated web page. The author will appear in the header of the web page.

Company Name

Enter the text for the company name to appear in the header of the web page.

Date

Select the date to appear in the header of the web page. The current date will be set as the default date.
