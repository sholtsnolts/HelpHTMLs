Save a document for the Mobile Viewer

You can save Solid Edge assembly, part, sheet metal and draft documents in the Viewer Document format. This creates a lightweight representation of the model in a file with a .sev extension.

  1. On the File menu, choose one of the following:

    1. Save As tab→Save As command.

    2. Save As tab→Save for Tablet command.

  2. Select the folder for the new document.

  3. (Optional) In the File Name box, type a new name for the document.

  4. From the Save As Type list, select Viewer Document (*.sev).

  5. In the Save As dialog box, click Save.

Note:

Hidden parts or bodies are not saved to the viewer document.
