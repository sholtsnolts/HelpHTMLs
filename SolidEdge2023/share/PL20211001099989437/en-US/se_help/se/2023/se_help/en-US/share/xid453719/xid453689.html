Transfer a document to the Mobile Viewer

After you save a Solid Edge document as a viewer document, do one of the following:

  • Email the viewer document to an account accessible from the mobile viewer. Then, on the mobile device, save the attachment.

  • If your mobile device supports a file transfer system, use that file transfer system to move the file to the mobile viewer.

  • Copy and paste the document using a network share.
