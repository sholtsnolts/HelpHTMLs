Web Publisher Wizard (Save Location)

At this step, choose the file name and location for the web page.

Wizard Options
Save Your Page to This Location:

Enter the file name and location for the web page. You do not need to specify a file extension because an .html file extension will be automatically added.

Example: C:\My Documents\web-pages\test\assembly - This will generate a web page named assembly.html in the directory C:\My Documents\web-pages\test.

Browse

Select the Browse button to access a dialog box that allows you to search for the location you wish to publish your web page.
