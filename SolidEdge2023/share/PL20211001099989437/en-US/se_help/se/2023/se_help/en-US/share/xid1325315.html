Solid Edge 2D Drafting with Viewer Mode

You can use the free Solid Edge 2D Drafting Viewer Mode with Solid Edge to open and review your 2D drawings and 3D models in their native format. For example, you can use Viewer Mode to create drawings and view 3D models even when you are not on a network.

You also can use Viewer Mode to collaborate with others, by making the Solid Edge documents on a server accessible from desktop and mobile devices. Recipients do not need to have a Solid Edge license to view your files, which makes it ideal for sharing your models and drawings with other designers, reviewers, and vendors.

You do not need to export files to PCF format to share them with others.

Getting Free 2D Drafting with Viewer Mode

Solid Edge users can get the free Solid Edge 2D Drafting Viewer Mode by selecting the option from the Solid Edge License Utility dialog box. If you have a license for cloud-enabled Solid Edge, then the Viewer Mode is displayed by default when web service is not available or the license cannot be verified.

Non-Solid Edge users can install the free Solid Edge 2D Drafting application, which installs the Viewer Mode. It is also included as a cloud-enabled license option when you install Solid Edge 2D Drafting.

Viewing 2D drawings

When viewing 2D drawings, you can use the Viewer Mode to:

  • Create, modify, and save all 2D drawings.

  • Zoom and pan drawings using a mouse and touchscreen controls.

Viewing 3D models

In the modeling environments, you cannot create new files or new features, print to 3D, or create model-derived tables. However, you can use the Viewer Mode to:

  • View, measure, and inspect all Solid Edge 3D models. This includes:

    • Use PathFinder to show and hide elements.

    • Use PathFinder to show the contents of an assembly.

    • Select and deselect elements.

    • Rotate, zoom, and pan the view from multiple inputs.

    • Measure between points on the model, keypoints of the geometry, and sizes of surfaces.

    • Display the values of model PMI.

  • Alter display configurations of an assembly.

  • Section a model parallel to any of the primary planes.

  • Save model views for presentations and sharing.

  • Use all of the controls on the View tab to alter the display state of model elements.

File protections are built into the viewer to ensure that the details of your proprietary designs are not exposed. For example, when you create a part or sheet metal file with ordered or hybrid features, collaborators cannot roll back the history tree to see your design process.

However, for sheet metal parts, collaborators can access the flat pattern through the context menu. They also can review the synchronous sheet metal features.
