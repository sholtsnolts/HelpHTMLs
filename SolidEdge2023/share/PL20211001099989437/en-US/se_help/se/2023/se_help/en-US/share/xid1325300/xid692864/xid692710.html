Web Publisher Wizard (Summary Page)

The Summary Page shows the size of the self-extracting executable if the files have been archived. The pages also reports the presence of the IPA WebView viewer in the archive and the encryption status. If the published pages were not compressed in a self-extracting executable, the summary information is empty.

The Preview button launches Internet Explorer with the published page loaded. This allows the page to be previewed before sending it via e-mail. Clicking Finish will launch the default e-mail application with the message created by the Wizard if that option was selected on the earlier page.
