Remove a work item ID from a Solid Edge document

You can remove one or more work item IDs from the active Solid Edge document.

  1. With a document open in Solid Edge, open the Requirements docking pane.

  2. In the Work Items page, log into Polarion.

    The table is updated with work item property information from the Polarion server.

    Note:

    You must log in each time you interact with the Polarion server, even if Polarion is configured for single sign-on (SSO).

  3. Select the work item IDs from the table, right-click and select Remove.

    The work item IDs are removed from the active Solid Edge document and a comment is recorded in Polarion.

Tip:

You can choose to automatically remove work item IDs from new documents created by the Save As command by selecting the option on the Settings→Options→Requirements page of the Solid Edge Options dialog box.
