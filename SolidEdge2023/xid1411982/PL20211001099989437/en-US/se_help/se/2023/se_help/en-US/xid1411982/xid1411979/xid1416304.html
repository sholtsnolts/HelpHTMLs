Search for existing work items in Polarion

  1. In the Requirements docking pane, click the Search tab.

    Note:

    You are required to log in on the Work Items tab.

  2. In the text box, type the information for which you want to search and click Search .

    Work items containing the information are displayed.

  3. (Optional) Add any work item found by the search to the active Solid Edge document by selecting it and clicking Add .

    Note:

    Multi-select of work items is not supported. Only single selection is supported on the Search pane.

  4. (Optional) To publish the work item in Polarion at the same time it is added, select the Publish to Work Item check box.


For additional information regarding Search, see Polarion online Help: https://polarion.plm.automation.siemens.com/techsupport/documentation. Personal Default Query is not available in Solid Edge Requirements Management.
