View work item information in Solid Edge

  1. Open a Solid Edge document that includes requirement work item IDs.

  2. Open the Requirements docking pane.

    Until you log into Polarion, the docking pane displays only the work item IDs and project IDs associated with the Solid Edge document. Solid Edge stores the work item ID and project ID as custom properties in the document. The combination of these two pieces of information create a unique item in Polarion.

  3. In the Work Items page, log into Polarion.

    The table is updated with work item property information from the Polarion server.

    Note:

    You must log in to Polarion once per session, even if Polarion is configured for single singn-on (SSO).

  4. Select any part in the design window, and then click Refresh .

    The table updates to show any work items associated with the selected part.
