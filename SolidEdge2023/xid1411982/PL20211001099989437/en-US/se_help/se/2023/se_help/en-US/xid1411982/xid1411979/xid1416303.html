Refresh work item information

You can update work item information displayed in the Requirements docking pane by getting the latest data from Polarion.

  1. Open a Solid Edge document that includes requirement work item IDs.

  2. Open the Requirements docking pane.

  3. In the Work Items page, log into Polarion.

    Note:

    You must log in each time you interact with the Polarion server, even if Polarion is configured for single-sign on (SSO).

  4. Click Refresh .

    When you are logged in, the Refresh command gets the latest work item property information from Polarion and displays it in the table.

    Tip:

    If you are not logged in when you click Refresh, only the list of work item IDs is updated for the selected documents.
