Obtain the Solid Edge Requirements Management project template

  1. Access the Support Center at https://support.sw.siemens.com.

  2. Submit a support request for the Solid Edge Requirements Management project template.

  3. Download the .zip file to your local machine.

  4. Follow instructions provided by Polarion to install and use the template.
