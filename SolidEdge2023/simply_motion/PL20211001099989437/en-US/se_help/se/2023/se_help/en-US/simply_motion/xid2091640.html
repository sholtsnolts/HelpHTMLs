Exporting Solid Edge motion kinematics to NX MCD and Process Simulate

You can export an assembly with motion kinematics from Solid Edge via PLM XML and JT that can be imported in NX Mechatronics Concept Designer (MCD) or Tecnomatix Process Simulate for further simulation.

The following kinematics data is exported via kinematics XML document:

Note:

The workflow of exporting Solid Edge motion kinematics to NX MCD or Process Simulate is unidirectional.

While exporting kinematics data, make sure you use the active mechanism data.

Suppressed and deleted kinematic elements, and associated constraints data, will not be exported.

Grounded part constraints will not be exported. We recommend that you create fixed joints for grounded parts after importing the kinematics XML documents to NX MCD or Process Simulate.

NX MCD and Process Simulate do not support import of the mass and gravity data. This can impact the simulation.

Tip:

Both automatically and manually created kinematic mechanisms can be exported.

For more information, see Saving to kinematics XML documents (.plmxml).
