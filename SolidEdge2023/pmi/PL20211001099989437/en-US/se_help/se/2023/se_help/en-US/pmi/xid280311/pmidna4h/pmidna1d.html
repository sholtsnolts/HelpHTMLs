Copy To PMI command bar

Steps
Select Features And Sketches

Specifies the profile-based feature or sketch you want to copy 2D dimensions and annotations from. You can select the feature or sketch in the graphics window or on the PathFinder page. The selection is highlighted in the graphics window, along with the attached 2D dimensions and annotations. Only items that are valid to copy are available for selection.

Select Dimensions And Annotations

Specifies individual dimensions and annotations in the highlighted feature or sketch to be copied to the 3D model. Only 2D dimensions and annotations that are valid for copying are available for selection.

Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selected feature/sketch or dimensions and annotations.

Finish/Cancel

Finishes copying the selected elements or cancels the command without copying.
