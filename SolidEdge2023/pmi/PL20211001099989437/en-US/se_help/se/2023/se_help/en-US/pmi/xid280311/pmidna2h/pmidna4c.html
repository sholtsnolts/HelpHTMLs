Show or Hide commands (PMI elements)

In conjunction with the check box in front of each PMI element, the Show and Hide commands show and hide PMI annotations and dimensions in the 3D model graphics window or in the 3D model view being edited.

When you point to a Dimensions or Annotations node, Show or Hide acts as a gatekeeper for the individual elements in that collection.

  • If you select Hide while pointing to the Dimensions or Annotations node in any collection, then all the dimensions/annotations in the collection will be hidden in the display.

  • If you select Show while pointing to the Dimensions or Annotations node in any collection, then the dimensions/annotations in the collection can be shown, depending upon their individual show/hide settings.

If you set or clear the check box in front of an individual PMI element in any collection, then all instances of this PMI element are shown or hidden in the document.

Tip:

Elements that are hidden when you exit model view edit mode are removed from that model view's list of collected elements.
