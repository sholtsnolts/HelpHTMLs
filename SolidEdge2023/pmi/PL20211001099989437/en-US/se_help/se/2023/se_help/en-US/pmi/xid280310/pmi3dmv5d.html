Review PMI model views

You can review the 3D model views and associated data. For each model view that you review, the show or hide display of the assembly or parts and associated PMI dimensions and annotations is updated in each view.

  1. On the Assembly or Feature PathFinder tab, under Model Views, right-click a model view name and select Review.

    The Review dialog box appears.

    The view orientation, render mode, and annotations and dimensions associated with the selected model view are displayed in the graphics window.

    The name of the model view currently displayed is displayed in the Review dialog box.

  2. In the Review dialog box, do any of the following:

    • Select the name of the model view to expand the names list and select the required mode view to be reviewed.

    • Click the Next Model View to see the next PMI model view defined in the current document.

    • Click the Previous Model View to see the previous model view.

  3. To end the review session, click Close.
