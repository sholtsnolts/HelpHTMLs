Model view Edit Definition command bar

The model view Edit Definition command bar is displayed when you select the Edit Definition command from a model view's shortcut menu on PathFinder or in the graphics window.

Options

Opens the Model View Options dialog box where you can change the model view rendering mode as well as modify 3D section view display settings.

Model View Display

Accesses the model view edit environment to control the display of assembly or part occurrences, PMI elements, section views, and other items. The commands available on the ribbon depend upon the type of model.

The PMI tab contains commands for you to add dimensions and annotations, as well as to create 3D section views.

Name

Displays and edits the model view name shown on PathFinder. Model view names must be unique within the document.

Finish

Saves the current display settings and exits edit mode.
