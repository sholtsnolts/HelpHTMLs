Manipulate a PMI model view

You can access and manipulate 3D model views using these shortcut menu commands.

Delete

Deletes the model view definition corresponding to the model view name(s) selected in PathFinder. This does not delete any of the PMI elements being displayed by the view.

Rename

Selects the model view name field displayed in PathFinder so that you can type a new name for the model view.

Tip:

You also can rename a model view using the Name box on the Model View command bar.

The Edit Definition command displays the command bar.

Activate View

Activates and displays the selected 3D model view.

Edit Model View

Allows changes to be made to the model view such as add and remove PMI dimensions and annotations.

Review

Provides a graphical review of all PMI model views and associated data. Displays the PMI Model View Review command bar.

Set View Orientation

Sets the view orientation of the selected 3D model view to that of the active window. The current settings for zoom, scale, and extent are also applied.

If you change the view orientation, you cannot undo the command.

Edit Definition

Opens the selected model view with its current display and orientation settings so you can edit it in model view edit mode. The model view Edit Definition command bar is displayed for you to click the step you want to change.

To edit the definition and display of model parts and PMI elements associated with the view, as well as to add new PMI dimensions and annotations, see Edit a PMI Model View Definition.

Create Drawing

Lets you create a drawing view of the selected model view. For more information, see Create a PMI drawing view.
