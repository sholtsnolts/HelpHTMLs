3D section view editing commands

You can edit a 3D section view by right-clicking the view name on the Assembly or Feature PathFinder, and then selecting one of these commands from the section shortcut menu.

Section Display Options

Opens the Section Display Options dialog box for you to choose how cut parts are displayed and whether to show the cutting plane annotation (direction arrows and cutting plane) in the view.

Add to Model View

Adds the selected section view to an existing 3D model view listed in the Model Views collection on PathFinder. The section is not applied unless the model view is being edited or viewed.

Remove from Model View

Removes the selected section view from a 3D model view listed in the Model Views collection on PathFinder.

Edit Definition

Opens the selected section view so you can edit it. The Section command bar is displayed for you to click the step you want to change.

Edit Profile

Opens the section profile in the sketch editor. Use the drawing tools to edit the profile size and shape, or you can draw it again from scratch.

Note:

  • To edit the profile, cut side, or selected bodies, the section view must be in the applied state.

  • If the profile is closed, then no annotation text or cut direction arrows are displayed.

Delete

Deletes the section view(s) selected on PathFinder.

Properties

Opens the Section Options dialog box for you to edit default properties for the cutting plane annotation, dimension style, and caption.
