Change PMI model views on a drawing

You can replace one PMI model view displayed in a drawing view with another one.

  1. Right-click the drawing view that contains the PMI model view you want to replace, and then choose Properties.

  2. In the Drawing View Properties dialog box, click the Display tab.

  3. On the Display page, from the Configuration Or PMI Model View list, select a model view name.

  4. Click OK to apply the change.

  5. Right-click the PMI drawing view, and then choose Update View.
