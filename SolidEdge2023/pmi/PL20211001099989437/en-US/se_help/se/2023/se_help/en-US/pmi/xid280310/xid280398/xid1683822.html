Section by Plane command bar

Section by Plane Options

Opens the Section by Plane Options dialog box where you can customize options such as caption, dimension style, and whether hardware parts participate in the cut.

Cutting Plane Step

Specifies and defines one to three cutting planes used to cut the section view.

  • On the keyboard, press X, Y, or Z to change the default cutting plane orientation.

  • Double-clicking a ring reorients the plane normal to the ring clicked on. Works the same as pressing X, Y, or Z.

  • Click the arrow to dynamically adjust the cut.

  • Click on a ring to dynamically adjust the plane angle.

  • Right-click or press Enter to accept.

Add Cutting Plane

Adds a new cutting plane to the section view. A section view may contain up to three cutting planes. On the keyboard, press x, y, or z to change the default cutting plane orientation.

Cutting Plane List

Lists the specified cutting planes for the section view. Use the list to select a cutting plane to modify.

Delete Cutting Plane

Deletes the selected cutting plane from the section view. A section view must contain at least one cutting plane.

Plane Color

Specifies the color of the active cutting plane.

Accept

Accepts the cutting plane definition.

Deselect

Cancels the cutting plane definition.

Select Parts Step

Selects the parts to be included in or excluded from the section view.

Bounded

When selected, this option cuts the model using all included planes and plane intersections to create a bounded cut area; otherwise the section view cuts through the entire model using all included planes and specified parts.

Cut List

Specifies which parts will be cut by the command.

Cut all parts

All bodies in the model participate in the cut.

Cut only selected parts

Only selected bodies participate in the cut. This option allows you to select the bodies to include in the cut.

Cut only unselected parts

Only unselected bodies participate in the cut. This option allows you to select the bodies to exclude in the cut.

Associative Plane Step

Changes the section plane type from non-associative to associative. Click the button again to return to non-associative.

Note:

Non-associative is the default section type.

Plane Type

Specifies the plane type used to create the cut.

Note:

Each plane type selected displays the controls required to position the plane.
