Pixel Size PMI command

The Pixel Size PMI command sets the text size of all PMI elements to display using a fixed pixel size rather than a relative model size. This prevents text, leader lines, and break lines from changing size when you zoom in or out.

When you use Pixel Size PMI display (A), you can adjust the default pixel size of 20:

  • Use the Increase PMI Font button (B) to make the text larger, to a maximum of 100 pixels.

  • Use the Decrease PMI Font button (C) to make the text smaller, to a minimum of 6 pixels.

If you want PMI text to display using the size specified by the dimension font style, use the Model Size PMI command, instead.
