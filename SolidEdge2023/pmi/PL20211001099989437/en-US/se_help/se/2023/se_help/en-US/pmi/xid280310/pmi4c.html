Edit Definition command (PMI, sections, model views)

Displays the command bar for the selected 3D section view, 3D model view, PMI annotation, or PMI dimension so you can edit the inputs you used to construct the view or element.

When the Edit Definition command bar is displayed, you can:

  • Use the Options button on the command bar to edit the properties of the selected view or PMI element.

  • Use the Model View Display group button on the command bar to add and edit PMI elements using the commands on the ribbon. While you are in this edit mode, you cannot use the modeling commands; you can only add or edit PMI elements.

    Except for view orientation and render mode, changes made in this edit mode are WYSIWYG.

    • PMI elements and section views that are hidden when you exit model view edit mode are automatically removed from the model view.

    • PMI elements and section views that are added and shown when you exit model view edit mode are automatically added to the model view.
