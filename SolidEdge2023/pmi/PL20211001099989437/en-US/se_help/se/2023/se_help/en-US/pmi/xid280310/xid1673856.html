Upload Model Views command

Uploads to Teamcenter the defined model views from the open assembly.

Model views saved in an assembly are uploaded to a single JT file in Teamcenter. The assembly JT (monolithic or non-body JT), can be viewed using Teamcenter Visualization, Structure Manager, or Lifecycle Viewer.

Note:

The Teamcenter preference SEEC_Image_Generate_Assembly_ModelViews must be enabled.
