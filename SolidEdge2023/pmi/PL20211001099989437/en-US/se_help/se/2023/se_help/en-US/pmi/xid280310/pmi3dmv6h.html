Reorient a PMI model view

For an existing 3D model view in the document, you can change the view orientation and save and update the model view definition.

  1. On the Assembly or Feature PathFinder tab, right-click a model view name. Model view names are listed in the PMI collection, under Model Views.

  2. On the model view shortcut menu, click Edit Definition.

    The Edit Definition command bar is displayed, with the selected model view name listed in the Name box.

  3. On the command bar, click the Model View Display group button.

  4. (Change model view orientation) Use any of these commands on the View tab to change the model view orientation: Common Views, Spin About Axis, Rotate, Look At Face.

  5. (Update the model view definition) On PathFinder, point to the model view name, then on its shortcut menu, click Set View Orientation. This copies the active view orientation to the model view definition you are editing.

    Tip:

    You cannot undo the Set View Orientation command.

  6. On the ribbon, choose Home tab→Close Model View.

  7. (Update the Model View Name) On the command bar, edit the existing model view name to reflect the changed view orientation, and then press Enter on the keyboard to save the changes.
