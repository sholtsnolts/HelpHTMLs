Remove PMI elements from a drawing view

When you turn off associativity between PMI elements in a drawing view and the PMI model view they are derived from, the drawing view goes out of date. The PMI elements are removed from the drawing view the next time you update it.

Remove PMI elements from a drawing view

  1. Right-click a PMI drawing view that contains the PMI elements you want to remove and choose Properties.

  2. In the Drawing View Properties dialog box, click the General tab.

  3. On the General page, clear the Include PMI Dimensions From Model Views check box.

  4. Click OK to close the dialog box.

  5. Right-click the PMI drawing view and choose Update View.

Tip:

In the same manner, you can add PMI elements to a PMI drawing view that you have already created. Select the Include PMI Dimensions From Model Views option and then update the drawing view.
