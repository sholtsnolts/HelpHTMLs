Use a 3D section view in a PMI model view

You can add section views to model views:

  • Select Add To Model View from the section's shortcut menu.

  • Select the model view(s) to which you want to add the section on the Add to Model View dialog box.

Add a 3D section view to a model view

  1. On the Assembly or Feature PathFinder, in the Section Views collection above the PMI collection, right-click the section view name you want to add to a model view, and then choose Add To Model View from its shortcut menu.

  2. In the Add To Model View dialog box, select one or more model view names to which you want to add the section view.

  3. Click OK to add the section view to the list of PMI objects that are associated with the model view.

    Tip:

    To locate the section view in PathFinder, expand the Section Views collection under the model view name.

Conversely, you can remove section views from model views:

  • Select Remove From Model View from the section name shortcut menu.

  • Select the model view(s) from which you want to remove the section on the Remove From Model View dialog box.

Remove a 3D section view definition from a model view

  1. On PathFinder, click the section view name.

  2. From the section view name shortcut menu, choose Remove From Model View.

  3. In the Remove From Model View dialog box, select the model view name(s) from which you want to remove the section view.

  4. Click OK to remove the section view from the list of PMI objects that are associated with the model view.

Display a section view in the graphics window

  • Click the box next to a section view name in PathFinder.

    This updates the graphics window, but it does not affect the model view definition.
