Adding custom frame components to a local library

Use the Frame Origin command to define a cross section shape, origin and direction to use as a custom frame component.

Before choosing the Frame Origin command, it is important to create a valid component to work with the Frames command.

  • For guidelines, refer to the help topic, Creating a custom structural frame component: best practices.

  • For practice creating and using custom frame components, we recommend the activity, Creating a custom frame.
