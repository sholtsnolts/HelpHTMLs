Frame Origin command

Use the Frame Origin command to define frame cross section attributes in a sketch that will be used to create a structural frame. The command is located on the Sketching tab in the Draw group. You can use the command in either ordered or synchronous environments.

  • The Point step defines the origin point where the frame attaches to the frame path. You can select a keypoint, such as a point, an endpoint, or the midpoint of a sketch element.

  • The Orientation step uses an edge on the cross section to define the orientation of the frame at placement.
