Create a structural frame

  1. Choose Tools tab→Environs group→Frame Design .

  2. Choose Home tab→Frame group→Frame command .

  3. On the Frame Options dialog box, specify the options for the frame, and then click OK.

  4. On the Frame command bar, under the Select Path Step, click the Select Cross Section Components button.

    Note:

    • If you selected Standard Parts Library on the Frame Options dialog box, the Standard Parts dialog box is displayed.

    • If you selected the Browse for component option on the Frame Options dialog box, the Frame Wizard dialog box is displayed.

  5. Select the frame component from a folder or standard parts dialog box.

  6. Select the path to define where to place the frame.

  7. Click the Accept (check mark) button on the command bar.

  8. Click the Finish button on the command bar.
