Add a custom frame component

You can add a custom frame component to the frame library using the Frame Origin command.

  1. Create a part model sketch defining the shape and size of your frame cross section.

  2. If the handle point for the cross section is not a keypoint on the sketch, create a point element at the desired handle location.

  3. Choose the Frame Origin command.

  4. Click an existing point element or keypoint on the sketch to define the handle point for the cross section.

  5. Click an existing line to define the relative orientation line.

    Note:

    If the cross section is circular, you can skip this step.

  6. Save the frame component part file to a user-defined folder or to the frames folder used by Solid Edge.
