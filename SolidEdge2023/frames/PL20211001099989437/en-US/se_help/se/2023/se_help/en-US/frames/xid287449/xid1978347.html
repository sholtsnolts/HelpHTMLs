Trim/Extend command

Use the Trim/Extend command to trim

or extend a frame

to a selected face, plane, or body.

You can trim frames to other frames from the same group. Any assembly features created on frames are not impacted when you trim or extend a frame.

You can use the Subtract command in assembly to subtract a frame or assembly components from a frame body.

The cut length for frames is updated in parts lists and assembly based on the trim, extend, or subtraction operation.

Note:

This command is disabled for files containing weldment features created in pre-Solid Edge 2023 versions (legacy files).

To enable the command for pre-Solid Edge 2023 files:

  1. Delete all weldment parts and features, such as frames, pipes, and assembly features from the active assembly.

  2. Save the file.

  3. Recreate the deleted features.
