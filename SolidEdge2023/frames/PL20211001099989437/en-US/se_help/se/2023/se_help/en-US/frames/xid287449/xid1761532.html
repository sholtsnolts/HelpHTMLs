Create frames with weld gaps

Note:

In this procedure, the weld gap values are exaggerated for display purposes.

Create frames and apply a weld gap to all members.

  1. Choose the Frame command .

  2. On the Frame Options dialog box, click the Set weld gap value option. Type 35 in the value box.

  3. Click Apply, and then click OK.

  4. Select the path elements, and then right-click.

    Click Finish.

  5. Turn off the sketch element display.

Set weld gap for all frames

Change the weld gap value globally.

  1. In Assembly PathFinder, select the frames collector shown.

  2. Click the Edit Definition button.

  3. On the Frame command bar, click the Frame Options button.

  4. Type 70 in the Set weld gap value box. Click Apply, and then click OK.

Set weld gap for single frame

Change the weld gap value at one end of a single frame.

  1. Select a single frame.

    Note:

    If you click Edit End Conditions at this point, both ends (red dots) are edited.

  2. To edit a single end, click the end shown.

  3. On the Frame command bar, click the Weld Gaps option and type 150.

    Click Finish.

Clear overrides

Observe the result when Clear Overrides is selected.

  1. In Assembly PathFinder, select the frames collector shown.

  2. Click the Edit Definition button.

  3. On the Frame command bar, click the Frame Options button.

  4. On the Frame Options dialog box, click Clear Overrides.

    Notice the message:

    Cleared corner treatment overrides for 1 end(s).

    Click OK, and then click Finish.
