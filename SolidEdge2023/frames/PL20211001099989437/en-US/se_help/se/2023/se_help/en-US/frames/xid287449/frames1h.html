Edit frame end conditions

To edit a structural frame end treatment, you can:

  • Edit both end conditions for frame component

  • Edit a single end condition for a frame component

  • Edit the end condition for the entire frame

  • Extend or trim an end condition

  • Extend or trim multiple end conditions

  • Remove an end treatment

Edit both end conditions for a frame component

  1. In PathFinder or the graphics window click the frame component you want to edit.

  2. On the command bar, click the Edit End Conditions button .

    Note:

    You can also right-click the frame and choose Edit End Conditions.

    The handle points for all of the end conditions of the entire frame are displayed with the handle points for the selected frame component automatically selected.

  3. Do one of the following:

    • Right-click to accept the automatically selected handle points.

    • Select the handle point of adjacent frame components to modify the end condition between the two components and click the Accept button.

    The Edit End Condition group expands to show the available options.

  4. In the Edit End Condition group, use the options to specify the end condition. For more information on these options, see Frame command bar.

  5. Click Finish to update the edit.

  6. To finish the command, click in the graphics window.

Edit a single end conditions for a frame component

  1. In PathFinder or the graphics window click the frame component you want to edit.

  2. Select the sphere for the end condition you want to edit.

  3. In the Edit End Condition group, use the options to specify the desired end condition. For more information on these options, see Frame command bar.

  4. Click Finish to update the edit.

  5. To finish the command, click in the graphics window.

Edit all end conditions for the entire frame

  1. In PathFinder or the graphics window click the frame you want to edit.

  2. On the command bar, click the Edit End Conditions button .

  3. On the Frame Options dialog box, select Apply corner treatment.

  4. Select the corner treatment you want to apply.

  5. Click Apply to make the changes to the model.

  6. Click OK to dismiss the dialog box and save the changes.

Trim or extend an end condition

  1. In PathFinder or the graphics window click the frame component you want to extend or trim.

  2. On the command bar, click the Edit End Conditions button .

    Note:

    You can also right-click the frame and choose Edit End Conditions.

    The handle points for all of the end conditions of the entire frame are displayed with the handle points for the selected frame component automatically selected.

  3. Do one of the following:

    • Right-click to accept the automatically selected handle points.

    • Select the handle point of adjacent frame components to modify the end condition between the two components and click the Accept button.

    The Edit End Condition group expands to show the available options.

  4. In the Edit End Condition group, click the Extend/Trim button .

  5. Do one of the following:

    • Select the Auto-select All Frame Components option to edit all frame components at the selected end.

    • Select the frame component in the Select Frame Component Step.

  6. Click a plane, face, or body to which you want to extend or trim the frame component.

    The frame component extends to the defined point.

  7. Click Finish.

  8. To finish the command, click in the graphics window.

Trim or extend multiple end condition

  1. In PathFinder or the graphics window click the frame.

  2. On the command bar, click the Edit End Conditions button .

    Note:

    You can also right-click the frame and choose Edit End Conditions.

  3. Select the frame components that you want to trim or extend.

  4. Click the Accept button.

  5. In the Edit End Condition group, click the Extend/Trim button .

  6. Select the Auto-select All Frame Components option.

  7. Click a plane, face, or body to which you want to extend or trim the frame component.

    The frame is trimmed to the selected surface.

  8. Click Finish.

  9. To finish the command, click in the graphics window.

Remove the end condition

After removing an end treatment, you can apply a fillet or extend the length of the component.

  1. In PathFinder or the graphics window click the frame component you want to edit.

  2. On the command bar, click the Edit End Conditions button .

    Note:

    You can also right-click the frame and choose Edit End Conditions.

    The handle points for all of the end conditions of the entire frame are displayed with the handle points for the selected frame component automatically selected.

  3. Do one of the following:

    • Right-click to accept the automatically selected handle points.

    • Select the handle point of adjacent frame components to modify the end condition between the two components and click the Accept button.

    The Edit End Condition group expands to show the available options.

  4. On the command bar, click None .

  5. Click the new end condition for the component (Fillet, Extend by Distance, or Remove).

  6. If you click Fillet or Extend by Distance, in the Value field on the command bar, type a value for the fillet radius or extend length.

  7. Click Finish.

  8. To finish the command, click in the graphics window.
