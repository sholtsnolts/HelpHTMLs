Edit End Conditions command

Edits the end conditions of the selected frame component.

Editing the end conditions for a single frame component

If you select a single frame component before you select the command, you are placed in the Edit End Conditions step of the Frame command bar.

The handle points for all of the end conditions of the entire frame are displayed with the handle point for the selected frame component automatically selected.

You can then right-click to automatically accept the highlighted handle points or you can select the handle point of adjacent frame components to modify the end condition between the two components and click the Accept button.

Once you accept the handle points, the command bar expands to show the available options for editing the end condition.

Editing the end conditions for an entire frame

If you select an entire frame before you select the command, the Frame Options dialog box is displayed. You can use the Corner Treatment options on the dialog box to edit the end conditions for the entire frame.

Note:

You cannot use the dialog box to change end conditions that are overridden in the Modify End Condition step of the Frame command bar.
