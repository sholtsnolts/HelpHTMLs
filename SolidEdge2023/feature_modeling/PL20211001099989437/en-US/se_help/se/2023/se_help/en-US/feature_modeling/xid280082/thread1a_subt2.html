Available thread sizes with the Thread command

Because you use the Thread command to add thread information to an existing cylindrical face, the cylindrical face diameter must match a value listed in the hole database.

For external threads, the cylinder diameter (1) must match the nominal diameter value hole database. For internal threads, the cylinder diameter (2) must match the tap drill diameter value in the hole database.

For example, to add a Rd 24 x 1/8 metric straight pipe thread to an external cylinder, the cylinder diameter must be exactly 24 millimeters, which matches the nominal diameter for this thread in the hole database. To add the Rd 24 x 1/8 metric straight pipe thread to an internal cylinder, the cylinder diameter must be exactly 21.142 millimeters, which is the tap drill diameter in the hole database.

Note:

If the cylindrical face diameter does not match a value in the hole database, the Thread Type list on the Thread command bar will be blank and a warning message is displayed when you click the Finish button.
