Update Part Copies dialog box (Out-of-Date Parts)

Update Now and Prompt Me Again Next Time

Specifies that you want to update any part copies that are out-of-date and you want to be prompted the next time you open a file that contains part copies that are out-of-date.

Don't Update Now, but Prompt Me Next Time.

Specifies that you don't want to update any part copies that are out-of-date when you open the file, but you want to be prompted the next time you insert the part copy.

Update Now and Automatically in the Future

Specifies that you want to update any part copies that are out-of-date and in the future you to automatically update part copies that are out-of-date.

Don't Update Now and Don't Prompt Me Again. I'll Update Manually.

Specifies that you don't want to update any part copies that are out-of-date when you open the file and you don't want to be prompted the next time you insert the part copy.
