Construct a curve through selected keypoints

  1. Surfacing tab→Curves group→Curves list→Keypoint Curve .

  2. Specify the points you want to use to define the curve and then click the Accept (check mark) button on the command bar.

  3. Using the options on the command bar, define conditions for the endpoints on the curve.

  4. On the command bar, click the Preview button.

  5. Finish the feature.

Tip:

  • You can construct the curve using keypoints on existing elements or points in free space.

  • During the Select Points step, you can use options on the command bar to construct either an open or closed curve.
