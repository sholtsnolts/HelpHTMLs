Edit the radius of a face

  1. Choose Home group→Solids tab→Revolve list→Radiate .

  2. Click the faces or live section edges to radiate.

    Note:

    You can use the Include All Coaxial Faces option to select a face and all faces coaxial to the selected face.

  3. Click Accept .

  4. Drag the handle or type a value in the Dynamic Value box to edit the radius.

  5. Right-click to accept.
