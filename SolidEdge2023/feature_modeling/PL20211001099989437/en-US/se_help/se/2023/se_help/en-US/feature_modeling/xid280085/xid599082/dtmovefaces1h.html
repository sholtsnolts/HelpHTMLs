Move faces directly

  1. Choose Home tab→Modify group→Move Faces list→Move Faces .

  2. Select the faces you want to move, then click the Accept button.

  3. Define the move method you want to use.

  4. Define the start point.

  5. Define the destination point.

  6. Finish the feature.

Tip:

  • You can use the Select option on the command bar to specify whether you want to move individual faces you select, a set of faces that belong to an existing feature, or an entire body.

  • You can use the following methods to define how you want to move faces: along a two-point vector, along an edge, normal to a face, within a plane, and synchronous move.

  • You can select the Synchronous Move option to move part geometry using the steering wheel.
