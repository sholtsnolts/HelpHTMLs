Construct a helical protrusion in the synchronous environment

  1. Choose Home tab→Solids group→Add list→Helix .

  2. Select a sketch or region (A) for the cross section and a line (B) for the axis of revolution.

  3. Right-click to accept.

  4. Use the command bar and Helix Parameter dialog box to specify a creation method.

    Note:

    You can define the helical path based on:

    • Axis length and pitch

    • Axis length and number of turns

    • Pitch and number or turns

  5. Type the required values for the helix. The required information is dependent on the helix creation method you specify.

  6. Click to define the helix direction.

  7. Right-click to accept.

  8. Click to create the helix.
