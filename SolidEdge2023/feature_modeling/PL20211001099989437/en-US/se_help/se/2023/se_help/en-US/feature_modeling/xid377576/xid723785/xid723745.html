Hole extents

You can use several extent types when constructing holes:

  • Through All

  • Through Next

  • (ordered only option) From/To Extent

  • Finite Extent

The extents that are available depend on the type of hole you are creating. Simple holes, counter bore holes, countersink holes, and threaded holes support all extent types. Tapered holes support only the Finite extent option, but you can define a finite extent length that exceeds the thickness of the part.

With counterbore holes, if you use the Finite extent option, you define only the hole extent. The counterbore extent is defined by the Counterbore Depth value you specify on the Hole Options dialog box. (In ordered) with the From/To extent option, the counterbore extends from the profile reference plane to the "From" surface. The hole extends from the profile reference plane to the "To" surface.
