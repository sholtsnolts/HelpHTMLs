Activate Part Body command

You can only add or remove features on the active solid body. All inactive solid bodies are ignored during feature modeling. Use the Activate Body command to activate a solid body in order to perform feature editing.

Note:

You can perform a cut operation that recognizes all bodies that the cut intersects. Use the Body Selection step option on the command bar.

Note:

Even though you can only add or remove features from the active solid body, all solid body faces in the file participate in synchronous move operations.

For example, you can:

  • Right-click a design body face and choose the Activate Body command .

  • In PathFinder, right-click a design body and choose the Activate Body command .

In PathFinder, the active design body has an active body icon in front of the design body name.
