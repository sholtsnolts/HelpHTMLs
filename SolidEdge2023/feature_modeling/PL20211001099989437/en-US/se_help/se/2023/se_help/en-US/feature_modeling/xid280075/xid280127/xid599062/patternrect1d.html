Rectangular Pattern command bar (synchronous)

Available Actions

Lists alternate actions which are available for the selected element.

Rectangular

Specifies that you want to construct a rectangular pattern.

Circular

Specifies that you want to construct a circular pattern.

Along Curve

Specifies that you want to construct a pattern along a curve.

Fill Pattern

Specifies that you want to construct a fill pattern.

Reference Point

Defines which pattern occurrence is used to align the pattern rectangle with the elements being patterned. The pattern occurrence that is the active reference point is shown as a bold X. This option is available only when creating a new pattern.

Add to Pattern

Adds new elements to an existing pattern. This option is available only when you are editing an existing pattern.

Suppress Instance

Specifies which pattern occurrences you want to suppress. You can select an individual occurrence by clicking it, or a group of occurrences by clicking and dragging.

Suppress Region

Specifies a group of pattern occurrences you want to suppress by selecting an existing sketch region or model face.

Keypoints

Controls keypoint selection on the model.

Reset

Unsuppresses all suppressed occurrences. This option is available when you click the Suppress Occurrence or Suppress Region buttons.

Fill Style

Specifies how the pattern is constructed.

Fit

You specify the number of occurrences in the X and Y directions, and the overall height and width of the pattern. The software calculates the X and Y spacing.

Fixed

You specify the number of occurrences in the X and Y directions, and the X and Y spacing between individual pattern occurrences. The software calculates the overall height and width of the pattern.

Accept (check mark)

Accepts the current input.

Deselect (x)

Cancels the current input.
