Construct a BlueSurf feature

  1. Choose Surfacing tab→Surfaces group→BlueSurf .

  2. Select the first cross section. You can select an existing sketch or part edges.

  3. Select the second cross section. Continue selecting all the cross sections. When you have finished selecting cross sections, you can click the Preview/Finish button to finish the feature or click the Guide Curve button to specify the guide curves.

  4. Select one or more guide curves.

  5. Finish the feature.

    How To
Tip:

  • You can use the BlueSurf Options dialog box to define the creation options you want. For example, you can specify whether the ends of the surface are capped with faces, whether the surface is open or closes on itself, map vertices between cross sections, and reorder cross sections.

  • You can also use the Insert Sketch button on the command bar to create new cross sections.

  • When constructing a BlueSurf feature that uses guide curves, each guide curve must touch each cross section and the guide curves must be tangent continuous (there can be no sharp corners on the curve).
