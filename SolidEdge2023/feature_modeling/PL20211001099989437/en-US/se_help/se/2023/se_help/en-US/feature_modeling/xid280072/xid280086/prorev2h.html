Construct a revolved extrusion or cutout: subsequent features

You can construct revolved features using the Select tool or the Revolve command. Both workflows are explained in this topic.

Construct a revolved feature using the Select tool

  1. Choose Home tab→Selection group→Select .

  2. Position the cursor within a sketch region, and when it is highlighted, click to select it.

    The extrude handle (A) is displayed.

  3. There are two methods for invoking the Revolve command:

    • Method 1

      1. Position the cursor over the origin knob (B) on the extrude handle, and then click to select it.

      2. Position the cursor over the sketch element or model edge which you want to use as the axis of revolution.

        The handle switches to a revolve handle.

        Note:

        You can also place the axis on a handle in a space point.

      3. Click to drop the revolve handle on the sketch element or model edge.

    • Method 2

      1. On the command bar, click the Revolve option.

        The extrude handle switches to a revolve handle (A).

      2. Position the cursor over the origin knob on the revolve handle, and then click to select it.

      3. Position the cursor over the sketch element or model edge that you want to use as the axis of revolution. Click to drop the revolve handle on the sketch element or model edge.

        Note:

        You can also place the axis on a handle in a space point.

      Note:

      On the Revolve command bar, the Live Section option (1) is on by default. This creates a live section at the completion of the Revolve command.

  4. To specify the feature extent, click the torus on the revolve handle, and then do one of the following:

    • On the command bar, with the Auto option on, move the cursor in and out of the solid body to add (C) or remove (D) material.

    • On the command bar, with the Add Material option on, move the cursor to add material to the solid body. You can press the Spacebar to switch between the add and remove material options.

    • On the command bar, with the Remove Material option on, move the cursor to remove material from the solid body. You can press the spacebar to switch between the add and remove material options.

Construct a revolved feature using the Revolve command

  1. Choose Home tab→Solids group→Revolve .

  2. Do one of the following:

    • On the command bar, select the Face option, position the cursor within a sketch region, and when it highlights click to select it.

      The extrude handle switches to a revolve handle (A).

    • On the command bar, select the Chain option, position the cursor over a connected chain of sketch elements, and then click to select them.

      The extrude handle switches to a revolve handle (A).

      1. Position the cursor over the origin knob on the revolve handle, and then click to select it.

      2. Position the cursor over the sketch element or model edge that you want to use as the axis of revolution. Click to drop the revolve handle on the sketch element or model edge.

        Note:

        You can also place the axis on a handle in a space point.

        On the Revolve command bar, the Live Section option (1) is on by default. This creates a live section at the completion of the Revolve command.

  4. To specify the feature extent, click the torus on the revolve handle, and then do one of the following:

    • On the command bar, with the Auto option on, move the cursor in and out of the solid body to add (C) or remove (D) material.

    • On the command bar, with the Add Material option on, move the cursor to add material to the solid body. You can press the Spacebar to switch between the add and remove material options.

    • On the command bar, with the Remove Material option on, move the cursor to remove material from the solid body. You can press the spacebar to switch between the add and remove material options.
