Create a Teamcenter Feature Library member

  1. Click the Teamcenter Feature Library tab.

  2. Use the arrow on the Look In box to specify the location of the feature library folder.

  3. In the Solid Edge window, select the feature(s) you want to make members of the feature library.

  4. Drag the selected features into the Teamcenter Feature Library tab.

  5. Complete the New Document dialog box and click OK.

  6. Use the Refresh command on the shortcut menu to update the contents of the Look In portion of the menu.
