Edit or move a chamfer

You can select a chamfer and directly edit a chamfer to change the chamfer value or you can indirectly edit the chamfer to move a face that adjoins a chamfer.

Edit a chamfer

  1. Select a chamfer.

  2. Click the chamfer edit handle.

  3. Type a new value for the chamfer and click.

  4. Click to complete the edit.

Move a face adjoining a chamfer

  1. Select an adjoining face of a chamfer.

  2. Click the move handle.

  3. Move the cursor until the face is positioned approximately where you want it, and then click.

  4. Click to complete the move.
