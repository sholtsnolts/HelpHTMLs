Curvature Shading Settings Dialog Box

Dialog Box Options
Show Curvature Shading

Specifies whether you want to display curvature shading colors.

Tightest

Displays the smallest radius (highest curvature) in the model. You can click the Find in Model button to find the smallest radius in the model.

Find in Model

Finds the smallest radius in the model.

Bottom Range (smallest)

Defines the range of radii over which the bottom color spectrum is displayed. You can type a radius value or move the slider to set the range. Editing the value compresses or expands the color spectrum bias over the range between the entered value and the maximum radius. This value can be very small and can appear as scientific notation if the value is less than .1 mm

Top Range (largest)

Defines the top range of radii over which the top color spectrum is displayed. You can type a radius value or move the slider to set the range. Editing the value compresses or expands the color spectrum bias over the range between the entered value and the minimum radius.

Maximum

Displays an arbitrary maximum radius in the model. This value typically exceeds the largest radius in the model and is theoretically flat.

Reset

Restores the previous default values without dismissing the dialog box.

Save As Default

Specifies that the current settings are saved as the new default values.
