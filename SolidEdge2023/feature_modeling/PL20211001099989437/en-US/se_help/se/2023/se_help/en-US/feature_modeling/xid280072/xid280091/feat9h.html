Construct a lofted protrusion or cutout

  1. On the Home tab→Solids group, click one of the following:

    • Lofted Protrusion

    • Lofted Cutout

  2. On the command bar, select a method for creating the first cross section—you can draw a path, select from a sketch or region, or select part edges.

    Note:

    In synchronous modeling, the draw a path option is not available.

  3. Do one of the following:

    • If you are drawing a cross section, define the profile plane you want to draw it on, and then draw a cross section or copy a cross section into the profile window.

      Note:

      In synchronous modeling, the draw a cross section option is not available.

    • If you are using a sketch or part edges to define the cross section, select the sketch or part edge(s) you want to use and then click the Accept (check mark) button on the command bar.

  4. For non-periodic cross section curves, position the cursor near the vertex you want to use as the start point. Click when a dot appears at the correct vertex and then click the Finish button on the command bar.

  5. Create all the cross sections you need. You must define at least two cross sections to create the lofted feature.

  6. Define the guide curves. You can define guide curves for the loft to follow.

  7. Define the extent. You can extend the feature between cross sections, or make it close itself into a loop.

  8. Click the Preview button on the command bar.

  9. Finish the feature.

Note:

  • During the Extent step, rather than selecting a finite or closed extent, you can click the Vertex Mapping button to map points on the cross section to control the loft.

  • During the Extent step, you can define end conditions for the ends of the lofted feature.
