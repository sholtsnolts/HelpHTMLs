Trim Surface command bar

Main Steps
Select Target Step

Defines the surface that you want to trim.

Select Tool Step

Specifies the elements that define the trimming boundary. You can select one or more curves, edges, reference planes, or surfaces. If you select a curve, it must lie on the surface you are trimming.

Select Region Selection Step

Specifies the regions formed by tools that will be trimmed

Finish/Cancel

This button changes function as you move through the feature construction process. The Finish button constructs the feature using input provided in the other steps. Once you construct the feature, you can edit it by re-selecting the appropriate step on the command bar. The Cancel button discards any input and exits the command.

Invert

Inverts the selected regions to trim. In trim, when the preview of the regions to be removed are highlighted,you can invert the selection. Use this option when you want to remove more regions than you want to keep.

Select Surface and Select Trim Tool Options
Select

Sets the method of selecting the elements that define the trimming boundary.

  • Single—Allows you to select one or more individual elements.

  • Chain—Allows you to select an endpoint connected set of elements.

  • Body—Allows you to select a body, such as a surface or solid body.

Accept (check mark)

Accepts the selection.

Deselect (x)

Clears the selection.
