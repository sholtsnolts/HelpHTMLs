Construct a 3D intersection curve

  1. Choose Surfacing tab→Curves group→Cross Curve .

  2. Do one of the following:

    • If you want to select a profile from a sketch, on the command bar, in the Create-From Options list, set the Select From Sketch option. This option is not available if there are no sketches in the document.

    • If you want to draw a profile, on the command bar, in the Create-From Options list, set the reference plane option you want. You should also use these options if you want to copy geometry into the profile window or include geometry from part edges.

  3. Click the Accept (check mark) button.

    How To

  4. Repeat steps 2 and 3 to define the second curve set and then click the Accept (check mark) button.

  5. Finish the feature.
