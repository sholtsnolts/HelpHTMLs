Construct a slot

You can construct a slot in the synchronous environment or ordered environment.

Construct a slot in the synchronous environment

  1. Choose Home tab→Solids group→Hole list→Slot .

  2. Click the sketch to define the slot path.

    Note:

    The element you select must be a tangent and continuous.

  3. (Optional) On the Slot command bar, click the Options button to define properties for the slot as needed.

    For more information, see the Slot Options dialog box help topic.

  4. Define the extent of the slot by doing the following:

    1. On the command bar click one of the following:

      • Finite

      • Through All

      • Through Next

      • From/To

    2. Use the dynamic arrows and the options displayed on the command bar to finish defining the feature extent.

      Note:

      You can change the extent options at any time as you construct the feature.

  5. Right-click to place the slot.

Construct a slot in the ordered environment

  1. Choose Home tab→Solids group→Hole list→Slot .

  2. (Optional) On the Slot command bar, click the Options button to define properties for the slot as needed.

    For more information, see the Slot Options dialog box help topic.

  3. Do one of the following:

    • If you want to draw a profile, select a planar face or reference plane.

    • If you want to select a profile from an existing sketch, on the command bar, on the Create-From Options list, click the Select From Sketch option. This option is not available if there are no sketches in the document.

  4. Draw or select a tangent and continuous sketch.

  5. Define the extent of the slot by doing the following:

    1. On the command bar click one of the following:

      • Finite

      • Through All

      • Through Next

      • From/To

    2. Use the dynamic arrows and the options displayed on the command bar to finish defining the feature extent.

      You can change the extent options at any time as you construct the feature.

  6. Click Finish to place the slot.
