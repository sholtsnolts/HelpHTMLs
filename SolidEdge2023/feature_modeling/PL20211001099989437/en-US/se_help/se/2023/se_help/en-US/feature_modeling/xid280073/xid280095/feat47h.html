Select a profile from a sketch

  1. Select a profile from any sketch.

    Only curves that meet basic profile requirements are selectable from the displayed sketches.

  2. If you are constructing a revolved feature, define the rotation axis. The axis geometry must be part of the current sketch.

    Only curves that meet basic profile requirements are selectable from the displayed sketches.

  3. Click the Accept button on the command bar to validate the profile and continue constructing the feature.

    If the profile is invalid for constructing the current feature, you are returned to the profile window. A message describing the problem is displayed.

  4. When you have corrected the problem, click Finish again to proceed.
