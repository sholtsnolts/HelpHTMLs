Recognize Hole Patterns command

Use the Recognize Hole Patterns command to detect circular or rectangular hole patterns in an imported model.

Choose Home tab→Pattern group→Pattern list→Recognize Hole Patterns. The command is available when at least three hole features exist and these three holes must form a rectangular or circular pattern (suppressed pattern instances are accounted for during recognition).

When recognition is complete, the Hole Pattern Recognition dialog box appears with the recognition results. The dialog box contains a row for each hole pattern detected. Selected holes that are not part of a pattern display red. If the pattern check box is cleared under recognize column, the hole pattern changes to red.

If the Recognize Patterns of Hole Patterns option is checked, then detected patterns of hole patterns are included in the patterns detected column. You can uncheck the detected pattern of patterns and only the hole patterns are listed.
