Dynamically edit the clearance or thickness value in the ordered environment

  1. Select an emboss feature in either PathFinder or the graphics window to display the Edit Feature command bar.

  2. Click the Dynamic Edit button to display the auto dimensions for the clearance and thickness.

  3. Click the dimension for the clearance or thickness to display the dynamic edit box.

  4. Type a new dimension and press Enter.

  5. Click anywhere in the graphics window to complete the edit.
