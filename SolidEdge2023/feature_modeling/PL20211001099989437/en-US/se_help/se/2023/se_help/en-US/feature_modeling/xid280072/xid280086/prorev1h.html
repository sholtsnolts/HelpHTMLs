Construct a revolved extrusion: base feature

You can construct a revolved feature using the Select tool or the Revolved Extrusion command. Both workflows are explained in this topic.

Construct a revolved base feature using the Select tool

  1. Choose Home tab→Select group→Select .

  2. Position the cursor within a sketch region, and when it is highlighted, click to select it.

    The extrude handle (A) appears.

  3. You can invoke the Revolve command by either of the following methods:

    • Method 1

      1. Position the cursor over the origin knob (A) on the extrude handle, and then click to select it.

      2. Position the cursor over the sketch element (A) or model edge which you want to use as the axis of revolution.

        The handle switches to a revolve handle.

      3. Click to drop the revolve handle (B) on the sketch element or model edge.

        Note:

        You can also place the axis on a handle in a space point.

    • Method 2

      1. On the command bar, click the Revolve option.

        The revolve handle (A) appears.

      2. Position the cursor over the origin knob (A) on the revolve handle, and then click to select it.

      3. Position the cursor over the sketch element (A) or model edge which you want to use as the axis of revolution, and then click to drop the revolve handle (B) on the sketch element or model edge.

        Note:

        You can also place the axis on a handle in a space point.

      Note:

      On the Revolve command bar, the Live Section option (1) is on by default. This creates a live section at the completion of the Revolve command.

  4. To specify the feature extent, click the torus (A) on the revolve handle, and then do one of the following:

    • Move the cursor until the revolved shape is the approximate size you want, and then click to finish the feature.

    • To define the extent precisely, select a keypoint, or type an angular value in the dynamic edit box, and then press Enter. For values, less than 360 degrees, position the cursor to define the side of the sketch in which you want to construct the feature.

    • To extend the feature in both directions, click the Symmetric button on the command bar. The extent value is divided equally in both directions. You can also press the Shift key to switch between symmetric and non-symmetric extent.

Construct a revolved base feature using the Revolve command

  1. Choose Home tab→Solids group→Revolve .

  2. Do one of the following:

    • On the command bar, select the Face option, position the cursor within a sketch region, and when it highlights click to select it.

      The revolve handle (A) appears.

    • On the command bar, select the Chain option, position the cursor over one of a connected chain of sketch elements, and then click to select them.

      The revolve handle (A) appears.

  3. Position the cursor over the origin knob (A) on the revolve handle, and then click to select it.

  4. Position the cursor over the sketch element (A) or model edge which you want to use as the axis of revolution, and then click to drop the revolve handle (B) on the sketch element or model edge.

    Note:

    You can also place the axis on a handle in a space point.

    On the Revolve command bar, the Live Section option (1) is on by default. This creates a live section at the completion of the Revolve command.

  5. To specify the feature extent, click the torus (A) on the revolve handle, and then do one of the following:

    • Move the cursor until the revolved shape is the approximate size you want, and then click to finish the feature.

    • To define the extent precisely, select a keypoint, or type an angular value in the dynamic edit box, and then press Enter. For values, less than 360 degrees, position the cursor to define the side of the sketch in which you want to construct the feature.

    • To extend the feature in both directions, click the Symmetric button on the command bar. The extent value is divided equally in both directions. You can also press the Shift key to switch between symmetric and non-symmetric extent.

Tip:

  • When constructing a base feature, the sketch elements must form a closed area.
