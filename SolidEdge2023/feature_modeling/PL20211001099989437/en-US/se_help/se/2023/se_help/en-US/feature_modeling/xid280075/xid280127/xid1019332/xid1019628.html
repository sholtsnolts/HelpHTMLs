Create a pattern by table

  1. Select the feature (or features) to pattern.

  2. Choose Home tab→Pattern group→Pattern list→Pattern by Table .

  3. Select a coordinate system to define the pattern X and Y orientation.

  4. Browse for the Excel spreadsheet document.

    If the spreadsheet is to be embedded in the file, uncheck the Link box.

    If you are working in a managed environment, first, save your document, then link the Excel spreadsheet.

  5. On the Pattern by Table command bar, click the Instance Table button. In the Instance Table dialog box, select a Parent Reference Point option.

  6. If the pattern origin needs to be modified, click the Pattern Origin Point option on the Pattern by Table command bar.

    Select a pattern origin point method. Choose either Coordinate System origin or Selected keypoint, and then right-click to accept.

  7. Accept the pattern.
