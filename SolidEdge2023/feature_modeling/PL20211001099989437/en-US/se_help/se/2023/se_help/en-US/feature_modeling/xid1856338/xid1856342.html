Apply a decal to a model

When applying decals, you can:

  • Apply a planar projection decal

  • Apply a label decal

Apply a planar projection decal (ordered environment)

  1. Choose the View tab→Style group→Decal command .

  2. On the Decal Options dialog box, click Planar projection and then click OK.

  3. On the General page (Insert Image dialog box), specify the image file to insert and the insertion method.

    Tip:

    If you select the Browse button to locate an image file using the Open a File dialog box, set the Files of type list to All Image Files (*bmp,*.png,*.jpg,*.jpeg,*.jpe,*,tif,*.tiff).

  4. Click the face on which you want to project the image and then click Accept.

  5. Click a planar face or reference plane.

    If you selected a single planar face in the previous step, this step is skipped.

  6. Use relationships and dimensions to position the decal and constrain it from moving on the face.

  7. Click the Close Sketch button .

    The image is projected onto the selected face.

  8. (Optional) Click the Decal Display Step button and use the options to change settings such as the width, height, and angle.

  9. Click Finish.

    A Decals collector is added to PathFinder and the decal is projected onto the face.

Apply a planar projection decal (synchronous environment)

  1. Choose the View tab→Style group→Decal command .

  2. On the Decal Options dialog box, click Planar projection and then click OK.

  3. On the General page (Insert Image dialog box), specify the image file to insert and the insertion method.

    Tip:

    If you select the Browse button to locate an image file using the Open a File dialog box, set the Files of type list to All Image Files (*bmp,*.png,*.jpg,*.jpeg,*.jpe,*,tif,*.tiff).

  4. Click the face on which you want to project the image and click Accept.

  5. Click a planar face or reference plane.

    If you selected a single planar face in the previous step, this step is skipped.

    The image is projected onto the selected face.

  6. (Optional) Click the Decal Display Step button and use the options to change settings such as the width, height, and angle.

  7. Click Finish.

    A Decals collector is added to PathFinder and the decal is projected onto the face.

    If needed, you can reposition the decal.

Apply a label decal

  1. Choose the View tab→Style group→Decal command .

  2. On the Decal Options dialog box, click Label and then click OK.

  3. On the General page (Insert Image dialog box), specify the image file to insert and the insertion method.

    Tip:

    If you select the Browse button to locate an image file using the Open a File dialog box, set the Files of type list to All Image Files (*bmp,*.png,*.jpg,*.jpeg,*.jpe,*,tif,*.tiff).

  4. Click a face.

  5. Use the Offset options to position the label on the face.

  6. Click the Decal Display Step option and use the options to change settings such as the width, height, and angle.

  7. Click Finish.

    A Decals collector is added to PathFinder and the decal is applied onto the face.

Reposition a decal in synchronous

  1. In PathFinder, right-click the decal and then click Show Sketch.

  2. Click the decal sketch.

  3. Use relationships and dimensions to position the decal and constrain it from moving on the face.

  4. In PathFinder, right-click the decal and then click Hide Sketch.
