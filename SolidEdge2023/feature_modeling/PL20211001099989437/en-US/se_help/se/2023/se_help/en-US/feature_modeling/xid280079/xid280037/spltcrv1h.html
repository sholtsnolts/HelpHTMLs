Split a curve

  1. Choose Surfacing tab→Curves group→Split Curve .

  2. Select a construction curve which you want to split.

  3. Select one or more elements which intersect the curve you want to split.

Tip:

You can select keypoints, curves, reference planes, or surfaces as the elements which split the curve.
