Create a 2D circular pattern of sketches

This procedure is for ordered modeling.

  1. Create a new sketch by choosing the Home tab→Sketch group→Sketch command, and then selecting a sketch plane. To use an existing sketch, select the sketch in PathFinder and click Edit Profile.

  2. Choose the Home tab→Features group→Circular Pattern command.

  3. Select sketch elements to pattern, and then right-click to accept.

    You can add or remove geometry by editing the pattern. Go to pattern edit and click on the Select Step . Select the sketch elements to add or press Shift+click to delete elements from the existing pattern.

  4. Use the options on the Circular 2D Pattern command bar (ordered sketch) to define characteristics of the pattern.

  5. Click for the center point of the circular pattern.

  6. Click to specify the radius (start point) of the pattern circle.

  7. Click to specify the direction for placing the pattern along the circle.

  8. Click Close Sketch.

Deleting pattern occurrences

  • When deleting the pattern definition, all the pattern occurrences are deleted and only the master geometry remains.

  • When deleting the master, all occurrences are deleted and the pattern definition remains as is.

  • When deleting an occurrence of a pattern, that occurrence suppresses.
