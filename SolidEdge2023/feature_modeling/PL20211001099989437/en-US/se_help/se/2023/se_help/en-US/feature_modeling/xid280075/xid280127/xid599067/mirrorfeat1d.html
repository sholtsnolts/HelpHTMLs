Mirror command bar

Available Actions

Lists alternate actions which are available for the selected element. There are no alternate actions available when mirroring.

Detach

Detaches the mirrored faces from the design body. When you set this option, the mirrored faces are placed as one or more construction bodies and are displayed using the construction color.

Persists Mirror Associativity

Maintains associativity between the feature being mirrored and the mirrored copy.
