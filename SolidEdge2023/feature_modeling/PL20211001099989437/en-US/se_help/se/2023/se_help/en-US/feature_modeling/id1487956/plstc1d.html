Lip command bar

Main Steps
Select Edge Step

Defines the edges for the lip or groove.

Direction Step

Specifies the direction and size of the lip or groove.

Finish/Cancel

This button changes function as you move through the feature construction process. The Finish button constructs the feature using input provided in the other steps. Once you construct the feature, you can edit it by re-selecting the appropriate step on the command bar. The Cancel button discards any input and exits the command.

command bar Options
Select

Sets the edge selection method for the lip or groove feature. You can use any combination of selection methods to select a set of edges to create the lip or groove. Hold the CTRL key or the SHIFT key to deselect an edge.

  • Edge/Corner—Allows you to select individual edges, or to select all edges adjacent to a corner by identifying the corner.

  • Chain—Allows you to select tangentially continuous chains of edges.

Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selection.

Width

Defines the thickness of the of the lip.

Height

Defines the height of the lip. This value is measured perpendicular to the selected edges.

Name

Displays the feature name. Feature names are assigned automatically. You can edit the name by typing a new name in the box on the command bar or by selecting the feature and using the Rename Feature command on the shortcut menu.
