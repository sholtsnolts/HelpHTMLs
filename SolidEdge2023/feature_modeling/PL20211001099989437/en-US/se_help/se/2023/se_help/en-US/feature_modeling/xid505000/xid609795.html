Edit an emboss feature in the ordered environment

  1. Select an emboss feature in either PathFinder or the graphics window to display the Edit Feature command bar.

  2. Click the Edit Definition button.

  3. Use the command bar options to edit the emboss definition. For example, click the Direction button to change the direction of the emboss.

  4. Click the Accept button to update the emboss feature .

  5. Click anywhere in the graphics window to complete the edit to the emboss feature.
