Construct a chamfer with equal setbacks

  1. Choose Home tab→Solids group→Rounds list→Chamfer Equal Setbacks .

  2. On the Chamfer command bar, specify the selection method for the chamfer.

  3. Select one or more part edges.

    Tip:

    If you set the Select type to Face or Loop, you are prompted to select a face or a face that contains loops.

  4. (Optional) Type a value for the chamfer.

  5. Right-click to place the chamfer.
