Recognize Patterns command

When working with imported models, use the Recognize Patterns command to find circular or rectangular patterns for any given collection of faces. If a pattern is found in the collection of faces, a Pattern Recognition dialog box displays. You can accept the found pattern to create a synchronous pattern feature. The new pattern feature has the same edit behavior as if it were created in the model.

The command is located on the Home tab→Pattern group→Pattern list , and it is also on the Inspect tab→Evaluate group. The command is only available in the synchronous environment.

Note:

Procedural features are not recognized.
