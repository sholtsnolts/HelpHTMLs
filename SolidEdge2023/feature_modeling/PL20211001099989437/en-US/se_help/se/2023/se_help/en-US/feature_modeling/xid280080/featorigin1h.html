Display or hide feature origins


To

Do this

To display the feature origin for a feature.

In PathFinder, right-click a feature with a hidden feature origin, then click Display Feature Origin.

To hide the feature origin for a feature.

In PathFinder, right-click a feature with a displayed feature origin, then click Display Feature Origin.

To display all the feature origins for a part.

In PathFinder, right-click in empty space, point to Show All, then click Feature Origins.

To hide all the feature origins for a part.

In PathFinder, right-click in empty space, point to Hide All, then click Feature Origins.
