Thicken a part

  1. Choose Home tab→Solids group→Add list→Thicken .

  2. Select the faces you want to offset to construct the thicken feature.

  3. On the command bar, type or select an offset distance.

  4. Click the arrow to define the side to which you want to offset the faces.

    Show Me

  5. On the Thicken command bar, click Finish.

Tip:

Set the Select option on the Thicken command bar to Feature to select a stitched surface in one step.
