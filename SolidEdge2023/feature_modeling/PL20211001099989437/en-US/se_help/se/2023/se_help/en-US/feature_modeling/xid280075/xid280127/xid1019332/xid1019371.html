Instance Table dialog box

Spreadsheet document:

Displays the Excel spreadsheet name to use for creating the pattern instances. You can type the Excel document name in this field.

Browse

Browses for the Excel spreadsheet to use for creating the pattern instances. The selected Excel document displays in the Spreadsheet document: field.

Edit

Opens the Excel spreadsheet listed in the Spreadsheet document: field.

Update

Updates the pattern based on the changes made to the Excel spreadsheet.

Instance table:

Displays the values for each pattern instance. These fields are read only.

The first row (green shaded) are the coordinates of the master instance. The orientation of all pattern instances is calculated relative to that of the master instance.

Parent Reference Point
Excel file

This option uses he first row in the spreadsheet to define the position and orientation of the Parent Reference Point.

Selected keypoint

This option allows selecting a keypoint as the Parent Reference Point. This option can be used if the X and Y coordinates of the reference with respect to the origin point are not known. The Parent row in the Instance Table displays the X and Y coordinates of the selected keypoint with respect to the pattern origin.
