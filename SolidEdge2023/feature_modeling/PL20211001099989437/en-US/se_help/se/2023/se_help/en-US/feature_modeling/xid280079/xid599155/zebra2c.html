Zebra Stripes Settings command

Displays the Zebra Stripe Settings dialog box.

Show Stripes

Specifies whether zebra stripes are displayed in the active window.

Stripe Color

Specifies the stripe color.

Background Color

Specifies the background color between the stripes.

Number of Stripes

Specifies the number of stripes.

Stripe Thickness

Specifies the thickness of the stripes.

Quality

Specifies the quality.

Mapping

Specifies the mapping method you want.
