Add a thread reference to a feature

  1. Choose Home tab→Solids group→Hole list→Thread .

  2. On the command bar, click the Options button.

  3. On the Thread Options dialog box, use the controls to define the thread attributes.

  4. Click the end of the cylinder to which you want to add a thread reference.
