Create a part copy

  1. Choose Home tab→Clipboard group→Part Copy.

  2. On the Select Part Copy dialog box, select the part or assembly you want, and then click OK.

    The part or assembly is temporarily placed in the document.

  3. On the Part Copy command bar, click the Parameters Step button.

  4. On the Part Copy Parameters dialog box, set the options you want, and then click OK.

    Note:

    If you want the part copy to be associative to the original part, set the Link To File option.

  5. On the Part Copy command bar, click Finish.

Tip:

  • If you insert the part copy before other design geometry is constructed, you can place the part copy as design geometry or as construction geometry. If you insert the part copy after design geometry is constructed, you can only place the part copy as construction geometry.

  • You can also create a flat pattern of a sheet metal part with the Part Copy command. You can place the flat pattern in a Part or Sheet Metal document.

  • You must insert the part copy into a sheet metal (.PSM) file if you want to display the bend lines in a drawing view.

  • You can use PathFinder to display the document name the part copy was created from.

  • You can use the Open command on the shortcut menu to open the original document to make changes to it.
