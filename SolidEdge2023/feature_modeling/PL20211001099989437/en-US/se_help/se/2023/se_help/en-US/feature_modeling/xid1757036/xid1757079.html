Blend command

In the Subdivision Modeling environment, use the Home tab→Modify group→Blend command to sharpen or smooth the hard edges of a subdivision model, depending on the weight value applied.

Blend edges workflow

  1. Select the edges and faces whose blend values you want to edit or examine.

  2. Drag the slider between 0 and 3 to set the value of blending you want to apply.

    The subdivision body is redrawn as you move the slider to show you the effect of the blend.

  3. Right-click or enter to accept the changes.

Controlling blends between subdivision cage elements

The shape of every edge of a subdivision body is calculated by its relationship to surrounding cage elements. The blend value determines the degree of this effect. A blend value of zero produces sharp edges, and a value of 3 produces soft, round edges.

Working with blends

There are several tools you can use to review, apply, and edit blend values.

  • The Blend command does not create any new geometry, but modifies the body shape when blend values are set on cage edges and vertices.

  • Use the Show Blend Values command on the shortcut menu to review the values applied to all or selected edges and vertices on the body.

  • Use the Show/Hide shortcut command to turn off the display of cage elements and subdivision bodies.

    Note:

    The body cage is not shown outside the Subdivision Modeling environments.
