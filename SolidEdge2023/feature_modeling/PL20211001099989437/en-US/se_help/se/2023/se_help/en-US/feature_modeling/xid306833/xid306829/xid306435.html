Multi-body Publish command

Use the Multi-body Publish command to create an individual file for each design body in a multi-body file. All design bodies in the multi-body file are published. The command is only available if a file contains more than one design body.

In the part environment, the command resides on the Home tab→Solids group→Add Body list. In the sheet metal environment, the command resides on the Home tab→Sheet Metal group→Add Body list.

The command opens the Multi-body Publish dialog box. You use the dialog box to publish design bodies and also to manage the published files.

The Multi-body Publish command also allows you to create an assembly file with the published bodies as components.
