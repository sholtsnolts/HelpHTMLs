Construct a rib

Synchronous environment

  1. Create a sketch to define the rib.

  2. Choose Home tab→Solids group→Thin Wall list→Rib .

  3. Select the sketch elements that define the rib feature.

  4. On the dynamic edit box, type in the rib thickness and press the Tab key.

  5. Set the desired extent options (No Extend, Finite Depth) and alignment option.

  6. Use the steering wheel to define the side of the sketch to create the rib on.

  7. Click the Accept button to create the rib.

Ordered environment

  1. Choose Home tab→Solids group→Thin Wall list→Rib .

  2. Do one of the following:

    • If you want to draw a profile, select a planar face or reference plane.

    • If you want to select a profile from an existing sketch, on the command bar, on the Create-From Options list, click the Select From Sketch option. This option is not available if there are no sketches in the document.

  3. Draw or select an open or closed profile.

  4. Click to define the direction you want to project the profile to form the body of the rib.

    Examples: Defining the rib direction

    The rib is automatically created symmetrically around the profile. If you want the rib to offset symmetrically, skip step 5. If you do not want the rib offset symmetrically, proceed to step 5.

  5. Click the Side Step button, and then click to define the side to which you want to offset the profile to form the thickness of the rib.

    Examples: Defining the rib side

  6. Finish the feature.
