Create a duplicate pattern

Use the Duplicate command to create a pattern of selected features that are positioned relative to blocks or coordinate systems.

  1. Choose the Home→Pattern→Pattern list→Duplicate command .

  2. Select the features to be included in the pattern and then right-click or click the Accept button on the command bar.

  3. Click a block or coordinate system to define the parent reference. This defines how the selected pattern features are positioned relative to the block or coordinate system.

  4. Click a block or coordinate system to define the duplicate location. You can also click the All Matching Blocks option to expand the selection to all occurrences of parent and instance reference blocks.

  5. Click Accept or right-click.

  6. Finish the feature.
