Pattern by Table command bar (ordered)

Options

Displays the Options dialog box for setting pattern coordinate and orientation units.

Update

Updates the pattern occurrences if the Excel file which drives the pattern changes.

Select Step

Selects features or geometry to be included in the pattern.

Select Coordinate System

Selects the coordinate system to control the pattern orientation.

Pattern Origin Point

Defines the point from which X and Y coordinates of instances are placed.

Coordinate System Origin

Sets the coordinate system origin as the pattern origin point.

Selected keypoint

Sets a model keypoint as the pattern origin point.

Keypoints

Controls keypoint selection on the model.

Instance Table

Displays the Instance Table dialog box for managing pattern instances.

Suppress Instance

Suppresses instance(s) of the pattern.

Cancel/Preview/Finish

Performs the displayed action.

Smart

Creates occurrences which adjust to part geometry based upon parent feature inputs.

Fast

Creates occurrences which do not adjust to part geometry.
