Construct an intersection point

  1. Choose Surfacing tab→Surfaces group→Intersection Point .

  2. Using the selection options on the command bar, select the elements you want to include in the first selection set and then click the Accept (check mark) button.

  3. Select the elements you want to include in the second selection set and then click the Accept (check mark) button.

  4. Finish the feature.
