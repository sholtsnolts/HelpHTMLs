Recognize a hole pattern

You can do one of the following procedures:

Workflow A

  1. Select individual hole features or use a fence to select a large pattern.

  2. Choose the Recognize Hole Patterns command.

  3. In the Hole Pattern Recognition dialog box, click the OK button to accept all recognized patterns.

    Note:

    If you do not want to convert the recognized patterns, uncheck them and then click the OK button.

Workflow B

  1. Choose the Recognize Hole Patterns command.

  2. Select individual hole features or use a fence to select a large pattern.

  3. Press the Enter key or right-click to accept.

  4. In the Hole Pattern Recognition dialog box, click the OK button to accept all recognized patterns.

    Note:

    If you do not want to convert the recognized patterns, uncheck them and then click the OK button.
