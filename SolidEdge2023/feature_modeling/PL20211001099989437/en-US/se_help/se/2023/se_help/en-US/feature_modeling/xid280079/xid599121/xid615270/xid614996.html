Intersect command bar

Main Steps
Intersect Options

  • Manual Trim/Extend-Manually extends or trims surfaces to a common intersection.

  • Create Design Bodies-Creates multiple solid bodies from the intersection of solid bodies and/or the intersection of surfaces and solid bodies.

  • Auto-trim-Forms closed volumes, closed surface loops, or a continuous chain of surfaces resulting from intersecting surfaces.

Select Bodies Step

Selects the surfaces of the bodies to trim and extend.

Trim and Extend Step

Specifies the regions to trim and the edges to extend.

Finish/Cancel

This button changes function as you move through the feature construction process. The Finish button constructs the feature using input provided in the other steps. Once you construct the feature, you can edit it by re-selecting the appropriate step on the command bar. The Cancel button discards any input and exits the command.

Invert

Inverts selection of trimmed surfaces, solid bodies, or voids.

Stitch

Creates a single surface by stitching the trimmed surfaces.

Stitch Options

Specifies the healing and tolerance options for the stitch operation.

  • Heal stitched surfaces—Sets the option to close gaps between surfaces during stitching.

  • Stitch tolerance—Sets the gap tolerance between surfaces.

  • OK—Saves the changes and dismisses the dialog box.

  • Cancel—Cancels the operation.

  • Save as Defaults—Saves the current settings as the default. Saved defaults are used the next time you use the dialog box. If you do not save the current settings as defaults, the previous default settings are used.

  • Help—Invokes the Solid Edge help for the current command.

Volume Regions

Opens the Volume Regions dialog box to control which regions are to be converted to solid bodies. Unchecked regions are not converted to solid bodies.

Extent Type

Specifies the extent type.

  • Linear—Creates a linear extension tangent to input faces.

  • Curvature Continuous—Creates a curvature continuous extension to input faces.

  • Reflective—Creates an extension that reflects the original surface.

Accept (check mark)

Accepts the selection.

Deselect (x)

Clears the selection.

Other command bar Options
Name

Displays the feature name. Feature names are assigned automatically. You can edit the name by typing a new name in the box on the command bar or by selecting the feature and using the Rename command on the shortcut menu.
