Construct a chamfer feature

Note:

If the feature is not what you wanted, you can go to any step to make adjustments. You can preview and alter the feature as many times as necessary before going to the Finish step.

  1. Choose Home tab→Solids group→Rounds list→Chamfer .

  2. Click the Chamfer Options button on the command bar and use the dialog box to set the options you want to use.

  3. If you set the option to 2 Setbacks, select the part face you want to measure the chamfer setbacks from.

  4. Select one or more part edges.

  5. Click Preview to see the results of your selections.

  6. Finish the feature.

Tip:

You can set the chamfer options before or after selecting edges to chamfer.
