Draft Face Analysis command bar

Options

Displays the Draft Face Analysis dialog box so you can define the draft face analysis settings you want.

Plane Step

Specifies the reference plane or planar face that is used to calculate the normal direction from which the part is pulled from the mold or die.

Pull Direction Step

Defines the direction that the part is pulled from the mold or die.

Finish/Cancel

This button changes functions as you move through the draft face definition process.  The Finish button completes the draft face analysis definition using input provided in the other steps.  The Cancel button discards any input and exits the command.
