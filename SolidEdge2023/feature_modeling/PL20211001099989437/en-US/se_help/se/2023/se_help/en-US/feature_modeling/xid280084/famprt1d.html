Family Of Parts tab


Members

Displays the name of the active member. You can select a member from the list to make it the active member. You can then use the New, Copy, Rename, and Delete buttons to create a new member, or copy, rename, or delete the active member.

New

Displays the New Member dialog box so you can create a new family of parts member. Type the name you want in the Member Name box.

Copy

Accesses the Copy Member dialog box so you can copy the active member to a new name. Type the name you want in the Member Name box.

Rename

Accesses the Rename Member dialog box so you can rename the active member. Type the new name you want in the Member Name box.

Delete

Accesses the Delete Member dialog box so you can delete the active member.

Edit Table

Displays the Family of Parts Edit Table dialog box. You can use the Family of Parts Edit Table dialog box to create, edit, and display the characteristics of family of parts members. You can also create new member documents for individual members.

Suppressed Features

Displays the list of suppressed features for the active member. When you add and remove suppressed features, only the active member is affected.

To suppress features, select them in the application window or in Feature PathFinder and then click the Suppress Features button.

To unsuppress features, select them in the list and then click the Unsuppress Features button.

Variables

Displays the list of variables for the active member. When you add and remove variables, all members are affected. In other words, all members of a part family must have the same set of variables. You can set different variable values for each member.

To add variables, select dimensions in the application window or variables in the Variable Table and then click the Add Variables button.

To remove variables, select them in the list and then click the Remove Variables button.

Apply

Applies the current settings to the source part and displays the family member characteristics in the graphic window.
