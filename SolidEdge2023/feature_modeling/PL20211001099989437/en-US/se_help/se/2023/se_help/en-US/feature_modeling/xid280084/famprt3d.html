Family of Parts Table dialog box

New Member

Displays the New Member dialog box so you can create a new family of parts member. Type the name you want in the Member Name box.

Copy Member

Displays the Copy Member dialog box so you can copy the active member to a new name. Type the name you want in the Member Name box. This option is available when you select the column heading for one or more family members.

Rename Member

Displays the Rename Member dialog box so you can rename the active member. Type the new name you want in the Member Name box. This option is available when you select the column heading for one or more family members.

Note:

In a Teamcenter-managed environment, changing the member name does not check out the member document and it does not make the member out-of-date.

Delete Member

Deletes the currently selected member. A dialog box is displayed that allows you to conform whether you want to delete the member. This option is available when you select the column heading for one or more family members.

Apply Member

Applies the current settings for a family member to the source part and displays the member characteristics in a preview window. This option is available when you select the column heading for one or more family members.

Select All Members

Selects all the members. This option is useful when you are ready to create member documents for a newly defined family of parts. For example, to create a set of member documents, you can click the Select All Members button, then click the Populate Member(s) button. Member documents for the selected members are created in the folder location specified in the path row for each member.

Set Path

Displays the Browse for Folder dialog box so you can define the folder location for the family members. This option is available when you select the column heading for one or more family members.

Populate Member(s)

Creates part documents for the selected family members. You can use the Set Path button to define the folder location you want for each family member. The names for the part documents are the same as the member name. This option is available when you select the column heading for one or more family members.

Note:

In a Teamcenter-managed environment, the Populate Member(s) command checks out the member document and updates the property values.

Save Table Data

Saves the changes you have made to the table.

Restore All

Restores the table to the previous saved configuration. A dialog box is displayed to confirm that you want to discard the changes you have made.

Print

Prints the table to a printer you specify.

Copy

Copies the selected text and places it on the Windows Clipboard.

Paste

Pastes the contents of the Windows Clipboard to the selected cell.

Suppress Features

Suppresses the selected features for the selected member.

Unsuppress Features

Unsuppresses the selected features for the selected member.

Table Area

Displays the family of part members in a spreadsheet format. A column on the left side contains rows for Design Intent, relationships, ordered features, ordered constructions, and variables for the source part. Additional columns are displayed for each member part you define. You can specify unique characteristics for the member parts you create by suppressing or unsuppressing ordered features or constructions, and typing new values for variables, typing a description and so on. The status row lists the current status for the member part.

You can use the buttons on the top of the dialog box and the shortcut menu commands to define the member characteristics you want.

For Solid Edge data management users who selected the option Automatically name files via Document Number and Revision, the system displays a dialog box where you can assign common properties, such as document number and revision number, as explained in Create documents with unique properties.

Row Names and Functions

Description

Allows you to type a description for a family member. You can use a description to record helpful information for the family member. In the Teamcenter-managed environment, the Description is the Dataset Description and is limited to 240 characters.

Path

Displays the current folder location for the member. You can use the Set Path button or the Set Path shortcut menu command to define a folder location for each family member.

Status

Displays the current status of the documents listed in the Filename column.

The file has not been created.

The file is up to date.

The file needs updating.

The linked file cannot be found.

An error has occurred in the creation or update of the file.
Design Intent

Each member has its own set of Design Intent relationships. These control synchronous modifications from variable edits on synchronous faces.

Relationships

Lists the persisted relationships in the source. Each member can suppress or unsuppress the persisted relationship.

Features

Lists the features for the source and member parts. For a member part, you can click in a cell, then use the buttons on the dialog box and the shortcut menu to suppress or unsuppress a feature. You can select a feature in the Source part row to define the feature characteristics for all members.

Constructions

Lists the constructions for the source and member parts. For a member part, you can click in a cell, then use the buttons on the dialog box and the shortcut menu to suppress or unsuppress a feature. You can select a construction in the Source part row to define the construction characteristics for all members.

Variables

Lists the variables for the source and member parts. For a member part, you can click in a cell, then type a variable value for the member.

Shortcut Menu Commands

The following additional shortcut menu commands are available. The shortcut menu commands are context-sensitive. In other words, the commands which are available change based on what is selected in the table area.

Shortcut menu commands which perform the same function as the option buttons along the top of the Family of Parts Edit Table dialog box are not listed. Only shortcut menu commands which provide additional functionality not available elsewhere are listed below.

Columns

Displays the Columns dialog box so you format the active column.

Font

Displays the Fonts dialog box so you can format the fonts used in the dialog box.

Find

Displays the Select Part Copy dialog box so you can search for the missing family member and redefine its location. This option is available when you select the Path cell for a family member whose Status cell has the Linked File Not Found symbol displayed.

Recreate

Recreates the missing family member in the location specified in the Path cell. This option is available when you select the Path cell for a family member whose Status cell has the Linked File Not Found symbol displayed. This option is useful when the Solid Edge document for a family member is inadvertently deleted or cannot be found.

Restore Previous Value

Restores the previous value for the selected cell.

Remove Override Variable

Removes the overridden variable from all members. This option is only available when you select the entire variable row for the variable you want to remove from the overridden variable list. The variable is also removed from the Variables list in the Family of Parts tab.

Show Only Suppressed Features

Specifies that only the suppressed features are displayed in the table.

Show Only Overridden Variables

Specifies that only the overridden variables are displayed in the table.
