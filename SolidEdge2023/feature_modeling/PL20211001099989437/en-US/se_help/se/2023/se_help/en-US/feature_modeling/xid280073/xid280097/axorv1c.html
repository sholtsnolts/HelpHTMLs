Axis of Revolution command

Use the Axis of Revolution command to define the axis of revolution of a revolved feature. This command is available when you are in the Profile step for a revolved feature. You can define an element, a part edge, a reference axis, or the edge of a perpendicular reference plane as an axis of revolution.
