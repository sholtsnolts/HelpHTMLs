Sphere command in Subdivision Modeling

In the Subdivision Modeling environment, the Home tab→Shapes group→Sphere command creates a spherical shape as the basis for your subdivision feature.

Specifying creation methods

The default creation method is At Origin, as shown in the following example.

You can change the default initial dimensions and settings on the command bar,

And then right-click to accept and finish the basic shape.

If you change the creation method to By 2 Points using the Selection Type list on the command bar, then:

  1. The first click defines a point for the center of the cubical cage.

  2. The second point defines the geometric center of the square planar side of the cubical cage.

For more information, see Sphere command bar.

Specifying initial segments (faces)

You can use the Level control on the command bar to set the number of cage segments. Cage segments define the number of edges and vertices that you can manipulate on the control cage, with more faces providing more detail.

Example:

From top to bottom: Level=1, Level=2, Level=3

Note:

You can control the size and color of vertices and edges using the Home tab→Style group→Cage Style command .
