Thread Options dialog box (ordered)

Standard

Specifies the hole database to use based on the selected standard.

Type

Specifies the type of thread.

Standard

Specifies placement of a standard thread.

Straight

Specifies placement of a straight thread.

Tapered

Specifies placement of a tapered thread.

Taper Angle

Specifies the taper angle for tapered threads. This option is available when you select Tapered for the thread type.

Thread direction

Specifies the direction of the threads either left-handed or right-handed to the threaded feature.

Default thread direction is right-handed .

Nominal diameter

Match hole diameter with the nominal diameter.

Inside pipe diameter

Match hole diameter with the inside pipe diameter.
