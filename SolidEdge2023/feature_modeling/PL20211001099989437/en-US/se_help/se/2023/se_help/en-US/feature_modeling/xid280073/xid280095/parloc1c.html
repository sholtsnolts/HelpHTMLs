Component Edge Locate command

Use the Component Edge Locate command to makes edges of the active part locatable or non-locatable.

When edges of the part are locatable, you can select them to complete other operations. For example, you can use part edges to place a dimension between the current profile elements and an existing edge.

It can also be useful to make the edges of the part non-locatable. For example, when drawing a simple profile on a complex part, you may want to make the existing edges non-locatable until you draw the profile elements. You can then make the edges locatable again to add positioning dimensions or relationships between the profile and the parts edges.

Note:

If you are editing a part in the context of an assembly (you have in-place activated the part), edges of inactive parts in the assembly are not locatable regardless of the setting of this command. You can use the Include command to copy edges of assembly parts to the profile plane.
