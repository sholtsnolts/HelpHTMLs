Trim a surface

  1. Choose Surfacing tab→Modify Surfaces group→Trim command .

  2. Select a surface or multiple surfaces that you want to trim, and on the command bar click the Accept.

  3. Select the input elements that define the boundary to which you want to trim, and then click the Accept.

  4. Select the regions of the surface you want to remove.

  5. If needed, select the Invert option to invert the selected regions to trim.

  6. Click the Accept.

Tip:

You can use a curve, a reference plane, or another surface as the input elements.
