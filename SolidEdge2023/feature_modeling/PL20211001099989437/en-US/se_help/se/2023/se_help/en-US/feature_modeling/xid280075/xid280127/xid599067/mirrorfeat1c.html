Mirror command

Use the synchronous Mirror command to construct a mirror copy of selected elements about a plane. You can mirror features, faces, face sets, and design bodies.

The mirror plane can be a reference plane or a planar face.

The mirrored features can be created as associative to the parent features by using the Persist option on the Mirror command bar.

In PathFinder, the associative mirror feature has a lock icon in front of the feature name. A non-associative mirror has no lock icon.
