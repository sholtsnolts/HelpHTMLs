Output Options

Displays the Boolean Options dialog box.

Select Target Step

Selects the target body to perform the operation on.

Select Tool Step

Selects the body, surface, or plane to be used as the tool for the operation.

Cancel

Accepts the current operation and ends the command.

Accept (check mark)

Accepts the selection.

Deselect (x)

Clears the selection.
