Feature recognition

You can import models into Solid Edge. However, imported models contain no feature definitions. Imported models are a collection of faces that only describe the model shape. To take full advantage of the synchronous modeling and edit behavior when using imported models, use the Solid Edge feature recognition tools to recognize specific faces and arrangement of faces. These recognized faces convert to synchronous features. The converted features behave the same as if they were created in the synchronous modeling system. Solid Edge can recognize holes, hole patterns, pattern of hole patterns, and chamfers.

Note:

The feature recognition tools are only available in the synchronous environment. If you import a model into the ordered environment, use the Move to Synchronous command to perform feature recognition.
