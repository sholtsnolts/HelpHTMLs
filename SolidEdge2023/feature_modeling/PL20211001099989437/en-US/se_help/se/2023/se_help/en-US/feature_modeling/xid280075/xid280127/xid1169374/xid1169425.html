Duplicate command bar

Select Step

Selects features or geometry to be included in the pattern.

Select Parent Reference

Selects the parent reference block or coordinate system for the select set.

Select Instance Reference

Selects the instance reference blocks or coordinate system coordinate system for creating duplicates of the select set.

Smart

Creates occurrences which adjust to part geometry based upon parent feature inputs.

Fast

Creates occurrences which do not adjust to part geometry.

Selection Type

Sets the method used for element selection.

All Matching References

Expands the selection to all occurrences of parent and instance references. If no reference instances are selected, all occurrences of parent reference will be selected.

Accept

Accepts the current input. To accept, right-click and press Enter.

Deselect

Deselects the current input.
