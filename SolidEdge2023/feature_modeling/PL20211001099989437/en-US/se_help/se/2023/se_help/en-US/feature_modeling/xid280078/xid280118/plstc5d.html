Thicken command bar

Main Steps
Select Step

Specifies the faces you want to offset to construct the thicken feature.

Offset Step

Defines the offset distance and direction.

Finish/Cancel

This button changes function as you move through the feature construction process. The Finish button constructs the feature using input provided in the other steps. Once you construct the feature, you can edit it by re-selecting the appropriate step on the command bar. The Cancel button discards any input and exits the command.

Select Step Options
Select

Sets the face selection criteria for defining the thicken feature. Click Accept when you are finished selecting faces. You can clear selections by clicking Deselect.

  • Single—Allows you to select an individual face.

  • Chain—Allows you to select a tangentially continuous chain of faces.

  • Feature—Allows you to select a feature, particularly stitched surfaces.

  • Body—Allows you to select the entire body, such as a surface body.

Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selection.

Offset Step Options
Distance

Sets the distance to offset the faces.

Command Bar Options
Name

Displays the feature name. Feature names are assigned automatically. You can edit the name by typing a new name in the box on the command bar or by selecting the feature and using the Rename command on the shortcut menu.
