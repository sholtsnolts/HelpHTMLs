Modify a FOP member name or description in SEEC

  1. Open the Family of Parts source.

    The Family of Parts members are downloaded to your cache.

  2. In PathFinder, click the Family of Parts tab.

  3. On the Family of Parts pane, click the Edit Table button to display the Family of Parts Edit Table dialog box.

  4. Select the column of the member you want to modify.

  5. Right-click and select Rename Member.

  6. Type a new member name in the Rename Member dialog box, and then click OK.

    The Member Name corresponds to the Dataset Name in Teamcenter.

  7. Click the cell for the Description of the member you renamed and type a new description.

    The Description corresponds to the Databaset Description in Teamcenter.

    Note:

    Changing the Member Name or Description does not check out the member document and it does not make the member out-of-date.

  8. In the Family of Parts Edit Table dialog box, click the Save Table Data button to save the changes you made.

  9. Select the column of the Member you modified.

  10. On the Family of Parts tab, click the Populate Member(s) button.

    If you are prompted, save the file by clicking OK.

  11. Click Yes on the Populate Member(s) prompt.

    Note:

    The Populate Members command checks out the member document and updates the property data.

  12. Click OK to dismiss the Family of Parts Edit Table dialog box.

  13. Close the source document.

    The Upload dialog box is displayed with a row for the source and a row for each Member you modified.

    Note:

    The Dataset Name reflects the modified Member Name, and Dataset Description reflects the modified Description from the FOP table.

  14. Click OK to upload your modifications to Teamcenter.
