Add a draft angle to a part

  1. Choose Home tab→Solids group→Draft.

  2. On the Add Draft command bar, click the Draft Options button, and then use the Draft Options dialog box to specify how you want to define the pivot location for the draft feature.

    Show Me

  3. Define the draft plane.

  4. For an add draft feature defined by a part edge, parting line, or parting surface, select the parting geometry, click the Accept button on the command bar, and then click the Next button on the command bar.

  5. Select the faces you want to draft, and define the draft angle or angles.

  6. In the graphics window, position the cursor so that the draft direction handle is displayed correctly, and then click.

    For split drafts, position the cursor to specify the draft directions. If draft directions 1 and 2 are the reverse of what you want, reposition the cursor to switch the draft direction.

  7. Right-click to finish the feature.
