Delete Faces command bar

Main Steps
Select Faces Step

Allows you to select the faces you want to delete from the model.

Finish/Cancel

This button changes function as you move through the feature construction process. The Finish button constructs the feature using input provided in the other steps. Once you construct the feature, you can edit it by re-selecting the appropriate step on the command bar. The Cancel button discards any input and exits the command.

Select Faces Step Options
Heal

Closes the gap formed by deleting a face. When deleting a face from a solid body or when simplifying a part, this option is set automatically, since a solid body is required.

When you delete a face from a construction body, you can clear this option, which creates a gap in the construction body. You can close this gap later by constructing a new surface. You can then use the Stitched Surface command to stitch the surfaces together.

Select

Sets the method of selecting the faces that you want to delete.

  • Single—Allows you to select individual faces.

  • Chain—Allows you to select tangentially continuous chains of faces.

  • Feature—Allows you to select all the faces of a feature by selecting the feature.

  • Internal Voids—Allows you to select all the internal voids of a selected design body.

Deselect (x)

Clears any selected faces.

Accept (check mark)

Accepts the selection criteria.

Other command bar Options
Name

Displays the feature name. Feature names are assigned automatically. You can edit the name by typing a new name in the box on the command bar or by selecting the feature and using the Rename command on the shortcut menu.
