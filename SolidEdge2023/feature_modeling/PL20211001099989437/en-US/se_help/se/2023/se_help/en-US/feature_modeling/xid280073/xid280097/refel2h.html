Construct a revolved feature with an existing centerline axis

If you want two revolved features to have a common centerline, you can use the centerline axis from the existing feature as the axis of revolution for the feature you are currently constructing.

  1. In the graphics window, with nothing selected, right-click to display the shortcut menu.

  2. On the shortcut menu, point to Show All, and then click Reference Axes.

  3. Choose Home tab→Solids group and then click the command for the revolved feature you want to construct.

  4. Define the profile plane for the revolved feature.

  5. When the profile window is displayed, choose Home tab→Draw→Axis Of Revolution .

  6. Click the axis you displayed in step 1.

  7. Continue to draw the revolved feature as usual.

Tip:

You can use the center line axis of any revolved protrusion, revolved cutout, or hole as the axis of revolution for a new revolved feature.
