Move or edit a helix

You can move a helix, or edit the helix feature to change such things as the helix direction and creation method, or you can edit the helix profile to make changes to the sketch element used to create the helix.

Move a helix

  1. Select the helix.

  2. Click the steering wheel axis shown.

  3. Drag the helix to a new location.

  4. Click to place the helix.

Edit a helix feature

  1. Select the helix.

  2. Click the helix edit handle.

  3. Click the direction handle.

  4. Right-click to change the helix direction.

  5. Click to save the edit.

Edit a helix profile

  1. Select the helix.

  2. Click the helix edit handle.

  3. Click the edit profile handle.

  4. Click the profile.

  5. Change the diameter of the profile.

  6. Click the green Accept button .

  7. Right-click to change the helix diameter.

  8. Click to save the edit.
