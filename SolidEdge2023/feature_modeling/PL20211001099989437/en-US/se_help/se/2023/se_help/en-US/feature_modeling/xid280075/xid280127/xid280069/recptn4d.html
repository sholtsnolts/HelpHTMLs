Rectangular Pattern command bar (Sketch)

Style

Sets the active style. This option is available only when working with a sketch.

Line Color

Sets the drawing color. You can click the More option to define custom colors with the Colors dialog box.

Line Type

Sets the drawing line type and style. This option is available only when working with a sketch.

Line Width

Sets the line width. This option is available only when working with a sketch.

Stagger Options

Displays the Stagger Options dialog box.

Reference Point

Defines which pattern occurrence is used to align the pattern profile with the feature being patterned. The pattern occurrence that is the active reference point is shown as a bold x.

Suppress Occurrence

Specifies which pattern occurrences you want to suppress. You can select an individual occurrence by clicking it, or a group of occurrences by clicking and dragging to define a rectangular area.

Suppress Regions

Specifies a group of pattern occurrences you want to suppress by selecting an existing closed sketch.

Flip

Specifies whether the suppressed occurrences are inside or outside the closed sketch.

Pattern Type

Specifies how the pattern profile is constructed:

  • Fit—You specify the number of occurrences in the x and y directions, and the height and width of the pattern profile. The software calculates the x and y spacing.

  • Filled—You specify the x and y spacing, and the height and width of the pattern profile. The software calculates the number of occurrences in the x and y directions.

  • Fixed—You specify the number of occurrences in the x and y directions, and the x and y spacing. The software calculates the height and width of the pattern profile.

X Count

Sets the number of pattern members along the x axis of the pattern rectangle.

Y Count

Sets the number of pattern members along the y axis of the pattern rectangle.

X Spacing

Specifies the distance between individual pattern occurrences along the x axis.

Y Spacing

Specifies the distance between individual pattern occurrences along the y axis.

Width

Specifies the width of the pattern rectangle.

Height

Specifies the height of the pattern rectangle.
