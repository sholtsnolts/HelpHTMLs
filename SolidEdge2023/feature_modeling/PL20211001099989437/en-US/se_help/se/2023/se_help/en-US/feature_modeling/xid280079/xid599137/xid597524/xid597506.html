Reflective Planes dialog box

Top (XY), Right (YZ), Front (ZX)

Turns on the base reference plane for reflectivity. Base reference planes are used as the primary input for the reflective planes.

Distance

Defines the offset distance from the base reference plane. Offset values must be greater than zero.

Flip

Reverses the offset distance direction of any active plane.

Transparency

Turns on transparency for all active reflective planes. The slider control defines the transparency value.
