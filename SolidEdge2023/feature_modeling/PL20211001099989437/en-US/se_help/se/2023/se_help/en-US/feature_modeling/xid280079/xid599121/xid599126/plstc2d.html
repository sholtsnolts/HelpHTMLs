Divide Part dialog box

Set Path

Accesses a dialog box that allows you to specify where you want to save the new files.

Add

Returns to the Select Cutting Surface step so you can add another division. The new division is added after the last division that was created.

Remove

Removes the selected cuts from the cutting definition list. When only the last entry in the list is selected, the Remove button is not available. To remove the last cut, you must select the next to last entry in the list.

Insert

Inserts a new division above the division selected in the list.

Edit

Returns to the Side Step for the selected division so you can redefine the cutting direction. When only the last entry in the list is selected, the Edit button is not available. To edit the last cut, you must select the next to last entry in the list.

When only the last entry in the list is selected, the Edit button is not available.

Cutting Surface

Displays the name of the feature used as the cutting surface to divide the part.

Status

Displays the current status of the documents listed in the Filename column.

The file has not been created.

The file is up to date.

The file needs updating.

The linked file cannot be found.

An error has occurred in the creation or update of the file.

A common error that can occur is that a cutting surface results in no material for one side of the divide operation, which is not valid. You can fix this by selecting the proper entry in the list and use the Edit button to define another cutting surface.
Filename

Displays the path and file name for the part.

Select All

Selects all rows in the grid.

Save Selected Files

Saves the files selected in the list.
