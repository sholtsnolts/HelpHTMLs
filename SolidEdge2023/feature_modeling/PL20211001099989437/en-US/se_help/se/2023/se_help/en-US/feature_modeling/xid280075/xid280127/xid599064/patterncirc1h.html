Construct a synchronous circular pattern feature

  1. In PathFinder or the graphics window, select the elements you want to pattern.

  2. Choose Home tab→Pattern group→Circular .

  3. Position the pattern profile and the axis of rotation for the pattern.

    A default pattern is constructed and displayed.

  4. Use command bar and the dynamic edit boxes in the graphics window to define the attributes of the pattern.

  5. On the Circular Pattern command bar, right-click or click the Accept button to finish the feature.

Tip:

  • You can pattern features, faces, and face sets.

  • You can use the Suppress option on the Pattern command bar to suppress individual pattern occurrences.
