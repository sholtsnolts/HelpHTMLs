Adding draft to parts

Use the Add Draft command to add draft angles to one or more part faces.

  1. To construct a simple draft feature in the synchronous environment, you first define a draft plane.

  2. Select the faces to draft.

    The part dynamically updates to reflect the draft angle and direction.

  3. Right-click to apply the draft angle to the model.

  4. To construct a simple draft feature in the ordered environment, you first define a draft plane.

  5. Select the faces to draft and type the draft angle on the command bar.

  6. On the command bar, click Next to specify the draft direction.

  7. Right-click to apply the draft angle to the model.
