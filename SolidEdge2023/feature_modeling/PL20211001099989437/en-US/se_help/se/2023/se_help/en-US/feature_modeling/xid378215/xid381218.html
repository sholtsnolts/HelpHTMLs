Edit a slot in the ordered environment

In the ordered environment, you can:

  • Dynamically edit a slot

  • Edit the slot feature to change such things as the slot direction and extent

  • Edit the slot profile to make changes to sketch elements used to create the slot

  • Add slots to a slot group

  • Remove a slot from a group

  • Edit parameters for a slot

Dynamically edit a slot

  1. Choose Home tab→Select group→Select .

  2. Select the slot.

  3. Click the Dynamic Profile button.

  4. Drag the slot to dynamically change the slot and click to make the change.

  5. Click to save the edit.

  6. Click to complete the edit.

Edit the slot feature

  1. Choose Home tab→Select group→Select .

  2. Select the slot.

  3. Click the Edit Definition button.

  4. Use the options on the command bar to edit the slot.

  5. On the Slot command bar, click Finish.

  6. Click to save the edit.

Edit the slot profile

  1. Choose Home tab→Select group→Select .

  2. Select the slot.

  3. Click the Edit Profile button.

  4. Make changes to the existing slot profile.

  5. Click the green Accept button .

  6. On the Slot command bar, click the Finish button.

  7. Click to save the edits.

Add slots to a slot group

  1. In PathFinder or the graphics window, select a slot occurrence within the group.

  2. Click the Edit Definition button.

  3. On the command bar, click the Sketch Step button , or ensure that it is selected.

    Note:

    Make sure the Create From option is set to Select from Sketch.

  4. Click the paths for the slots you want to add.

  5. Right-click to add the slots.

  6. On the Slot command bar, click the Finish button.

  7. Click to save the edits.

    Tip:

    You can also add slots to a group by adding new sketches.

    1. Select a slot.

    2. Click the Edit Profile button.

    3. Use the sketch commands to place lines and arcs to add to existing slot paths

Remove a slot from a group

  1. In PathFinder or the graphics window, select a slot occurrence within the group.

  2. Click the Edit Definition button.

  3. On the command bar, click the Sketch Step button , or ensure that it is selected.

    Note:

    Make sure the Create From option is set to Select from Sketch.

  4. Set the Selection Type option is set to Single.

  5. Use the Ctrl key to deselect the path for the slot you want to remove.

  6. Click the green Accept button .

  7. On the Slot command bar, click the Finish button.

  8. Click to save the edits.

    Tip:

    You can also add slots to a group by adding new sketches.

    1. Select a slot.

    2. Click the Edit Profile button.

    3. Remove the segments you want.

    4. Click the Finish button.

    5. Click to save the edit.

Edit parameters for a slot group

  1. In PathFinder or the graphics window, select a slot occurrence within the group.

  2. Click the Edit Definition button.

  3. On the command bar, click the Options button, or ensure that it is selected.

  4. On the Slot Options dialog box, set the parameters you want to change and click OK.

  5. On the Slot command bar, click the Finish button.

  6. Click to save the edits.
