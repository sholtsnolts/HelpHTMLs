Editing an ordered Web Network feature

Web thickness

  1. Select the web network feature in PathFinder, with QuickPick, or by clicking the feature on the model.

  2. On the Edit box, click either Edit Definition or Dynamic Edit.

  3. Click the web network thickness handle.

  4. In the edit value box, type in the thickness value.

  5. On the command bar, click Finish.

Draft

  1. Select the web network feature in PathFinder, with QuickPick, or by clicking the feature on the model.

  2. On the Edit box, click either Edit Definition or Dynamic Edit.

  3. Click the web network draft handle.

  4. In the edit value box, type in the draft angle value.

  5. On the command bar, click Finish.

Dimensioning

In the ordered environment, a sketch drives a web network feature. Place dimensions on the sketch profile to control the web network definition.

To edit the existing sketch dimensions or to add dimensions:

  1. Select the web network feature in PathFinder, with QuickPick, or by clicking the feature on the model.

  2. On the Edit box, click Edit Profile.

  3. Edit dimensions or add dimensions.

  4. Click the Close Sketch button.

  5. On the command bar, click Finish.

Removing or adding web network sections

Since sketches drive the web network feature, deleting or adding sketch elements changes the web network definition.
