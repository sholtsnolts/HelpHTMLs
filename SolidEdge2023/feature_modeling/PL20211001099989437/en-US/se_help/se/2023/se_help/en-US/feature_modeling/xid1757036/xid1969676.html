Offset command in Subdivision Modeling

The Offset command in the Subdivision Modeling environment moves or lifts a cage face by offsetting it by a positive or negative specified value. Use the Offset command when you want to add a set of faces that are not coplanar, which is common in free-form models.

To learn how to use the command, see Offset cage faces.

Note:

The result of moving and lifting faces by offsetting them is different than using the steering wheel with the Move/Lift command, which can only move faces in a single direction.

Using offset with lift and tip

The Offset command bar provides options to Lift the offset faces or to Tip the offset faces.

Example:

  • Faces selected for offset (red).

  • Offset with Lift (1)—New faces (orange) are created as the selected faces are offset. Neighboring faces are not modified.

  • Offset with Tip (2)—No new faces are created, but the neighboring faces (yellow) are modified.
