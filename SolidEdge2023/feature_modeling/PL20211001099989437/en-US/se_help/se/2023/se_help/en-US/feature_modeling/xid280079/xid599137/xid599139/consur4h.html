Construct a revolved construction surface

  1. Choose Surfacing tab→Surface group→Revolved Surface .

  2. Do one of the following:

    • If you want to draw a profile, select a planar face or reference plane.

    • If you want to select a profile from an existing sketch,on the command bar, on the Create-From Options list, click the Select From Sketch option. This option is not available if there are no sketches in the document.

  3. Draw or select a profile and define an axis of rotation. The profile can be open or closed. The ends of an open profile are extended infinitely; an arc with open ends is extended to form a circle.

  4. Click to define the side of the profile you want to construct the surface on.

  5. Define the extent of the surface.

  6. Finish the feature.
