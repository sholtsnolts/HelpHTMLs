Recognize Holes dialog box

Recognize column

Lists the groups of similar holes found. You can deselect a group to ignore hole replacement.

Occurrence Count column

Lists the number of occurrences in a group of similar holes recognized in a model.

Feature Name column

Lists the feature names for groups of holes. You can override the system generated name with a user-defined name.

Select Alternate Type column

Provides a button to launch the Hole Options dialog box. Use the dialog box to select an alternate hole type.

Saved Settings column

If you choose a Saved Setting from the Hole Options dialog box, displays that saved setting in the column. If you alter a Hole type from Hole Options dialog box, displays Custom in the column.

Type column

Displays the hole type in the column (simple, tapered, countersink, counter bore, or threaded).

Diameter column

Lists the nominal hole diameter of the group.

Face Selection

Ignores the current holes found on the design body. Select a single face to recognize all holes found on the selected face.

OK

Accepts all settings and replaces found faces with hole features.

Cancel

Ignores all recognized holes and ends the Recognize Holes command.
