Construct a fill pattern feature

  1. Choose Home tab→Selection group→Select .

  2. In PathFinder or the graphics window, select one or more elements to pattern.

  3. Choose Home tab→Pattern group→Rectangular Pattern list→Fill .

  4. Click the region(s) to pattern fill.

  5. Use the Fill Pattern command bar and the dynamic edit boxes in the graphics window to define the pattern parameters.

  6. On the Fill Pattern command bar, click the Accept (check mark) button to finish the feature. You can also right-click to finish the feature.

  7. Click to close the Fill Pattern command.
