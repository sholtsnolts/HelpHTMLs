Construct a hole (ordered)

  1. Choose Home tab→Solids group→Hole .

    The Hole command bar is displayed.

  2. Click Hole Options, and then select the hole type and set hole parameters in the Hole Options dialog box.

    • Simple holes

    • Threaded holes

    • Tapered holes

    • Counterbore holes

    • Countersink holes

  3. Define the reference plane for the profile sketch.

  4. The Hole Circle command is started. Click to place one or more hole circles.

    Tip:

    • You can use horizontal and vertical alignment for the center points of hole circles. When you place a circle and the center point input recognizes a horizontal or vertical alignment with the center point of another circle, the horizontal or vertical alignment relationship is applied.

    • You also can align hole circles symmetrically.

  5. Choose Home tab→Close group→Close Sketch, or click the green check mark in the graphics window.

  6. Define the hole extent (depth).

  7. Finish the feature.

Tip:

  • You can set hole parameters after placing the hole circles. The displayed hole circles update automatically.

  • All holes constructed as part of the same feature have the same parameter settings. For holes with different parameters, construct another hole feature.

  • When constructing threaded holes, the thread values available on the Hole Options dialog box are based on the hole size. You can only specify threaded holes if the hole size matches a value specified in the Holes.txt file located in the C:\Program Files\Siemens\Solid Edge 2023\Preferences folder.

  • If you delete a threaded hole entry in the Holes.txt file, when you edit a hole feature that used that entry, an asterisk is placed on the thread value listed on the Hole Options dialog box.
