Thin Wall command bar

Main Steps
Common Thickness

Specifies the common wall thickness for the thin wall feature and defines the offset direction. The default offset direction is toward the inside of the part

Open Faces

Defines the open faces for the thin wall feature. Open faces are optional.

Unique Thickness

Defines selected walls to be a thickness that is different from the common wall thickness for the feature. Unique thickness is optional.

Preview/Finish/Cancel

This button changes function as you move through the feature construction process. The Preview button shows what the constructed feature will look like based on the input provided in the other steps. The Finish button constructs the feature. After previewing or finishing the feature, you can edit it by re-selecting the appropriate step on the command bar. The Cancel button discards all input and exits the command.

Common Thickness Options
Offset Outside

Applies the entire wall thickness to the outside of the solid.

Offset Inside

Applies the entire wall thickness to the inside of the solid.

Symmetrical

Applies half the wall thickness symmetrically to both the inside and outside of the solid.

Common Thickness

Sets the common wall thickness. All walls to which you do not apply a unique thickness use the common wall thickness.

Open Faces Options
Select

Sets the method of selecting the walls to which you want to apply the thickness. You can use any combination of selection methods to select a set of walls.

  • Single—Select one or more individual elements.

  • Chain—Select an endpoint connected set of elements by selecting one of the elements in the chain.

  • Feature—Select all the walls in a feature by clicking the feature.

Accept (check mark)

Accepts the selection.

Deselect (x)

Clears the selection.

Unique Thickness Options
Select

Sets the method of selecting the walls to which you want to apply the thickness. You can use any combination of selection methods to select a set of walls.

  • Single—Select one or more individual elements.

  • Chain—Select an endpoint connected set of elements by selecting one of the elements in the chain.

  • Feature—Select all the walls in a feature by clicking the feature.

Accept (check mark)

Accepts the selection.

Deselect (x)

Clears the selection.

Unique Thickness

Sets the unique wall thickness.

Other command bar Options
Name

Displays the feature name. Feature names are assigned automatically. You can edit the name by typing a new name in the box on the command bar or by selecting the feature and using the Feature command on the shortcut menu.
