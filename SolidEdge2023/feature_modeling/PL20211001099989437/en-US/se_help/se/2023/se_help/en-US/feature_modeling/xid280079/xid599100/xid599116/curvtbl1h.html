Construct a curve by table

  1. Choose Surfacing tab→Curves group→Curves→Curve By Table .

  2. On the Insert Object dialog box, set the Create New Spreadsheet option and click OK.

  3. In the Excel spreadsheet, enter the X, Y, and Z coordinates for each point.

    Note:

    You must enter a minimum of three rows of coordinates if you set the Smoothing On option, or two rows of coordinates if you set the Smoothing Off option.

    X,Y,Z coordinate data must reside in spreadsheet columns A,B,C respectively. Any data outside those columns will not be read by the command.

    Solid Edge will stop reading coordinate data when it encounters any non-numeric data, including blank cells.

  4. On the Microsoft Excel Applications menu, click Close and Return to update the curve and close Excel.

    Note:

    • You can also select an existing Microsoft Excel spreadsheet to define the curve by setting the Create From File option on the Insert Object dialog box. You can type the path of the Excel file, or you can click the Browse button to search for the file.

    • To change data points in the curve, you can click the Edit Data Points Step button on the Curve By Table command bar.

    • To set curve parameters, you can click the Parameters Step button on the Curve By Table command bar. On the Curve By Table Parameters dialog box,  you can specify whether a curve is open or closed, periodic or natural. You can also specify the curve units, a coordinate system for the curve, and the curve fit.
