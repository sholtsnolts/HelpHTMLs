Replace circular cutouts with hole features

  1. Open an imported or Solid Edge part or sheet metal file. The file needs to contain circular or conical cutouts that represent holes. These cutouts are candidates for hole replacements.

  2. Choose Home tab→Solids group→Hole list→Recognize Holes .

  3. Select a design body.

    Note:

    If the command finds no hole candidates, the following error is displayed: No recognizable holes were found. If candidates are detected, the Hole Recognition dialog box is displayed.

  4. Observe the groups of detected holes. You either accept the detected holes in the condition they were found or you can edit the detected holes and change their type.

  5. When all detected holes are set to process the way you desire, click OK.

    All detected hole faces are consumed and replaced with hole features.

    Note:

    If there are additional design bodies in the file, the command remains active and you can select another design body to process. Otherwise, the command ends.
