Construct a chamfer with unequal setbacks

  1. Choose Home tab→Solids group→Rounds list→Chamfer Unequal Setbacks .

  2. On the command bar, click the Chamfer Options button and use the dialog box to set the options you want to use.

  3. Select the face containing the edges to chamfer and click Accept.

  4. Select the edges to chamfer.

  5. Type a value for the setback and angle.

    Note:

    If you set the option to 2 Setbacks, select the part face you want to measure the chamfer setbacks from.

  6. Click Accept to see the results of your selections.

  7. Click Finish to complete the feature.
