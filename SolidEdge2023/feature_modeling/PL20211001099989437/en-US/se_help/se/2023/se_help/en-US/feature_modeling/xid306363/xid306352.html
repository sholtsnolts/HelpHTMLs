Split a selected body into individual bodies

In a multi-body file

  1. Open a multi-body file.

  2. Choose Home tab→Solids group→Add Body list→Split.

  3. Select the target body.

  4. Select the tool body. You can select a design body, a construction body, a surface, or a reference plane as the splitting tool.

  5. On the command bar, click the Accept button.

  6. On the command bar, click the Finish button.

  7. Click Cancel or press the Esc key to end the Split command.

In a single body file

  1. Open a single body file.

  2. Choose the Home tab→Solids group→Add Body list→Split.

  3. Select the target body.

  4. Select the tool body. You can select a construction body, a surface body, or a reference plane as the splitting tool.

  5. On the command bar, click the Accept button.

  6. On the command bar, click the Finish button.

  7. Click Cancel or press the Esc key to end the Split command.
