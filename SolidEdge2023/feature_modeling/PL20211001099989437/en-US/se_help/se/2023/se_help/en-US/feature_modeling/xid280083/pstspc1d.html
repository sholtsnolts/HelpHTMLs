Paste Special dialog box

Inserts the Clipboard contents into a document using a selected format.

Source

Displays the name and location of the source data. If the data was copied from an application that did not provide the name and location to the Clipboard, "Unknown" is displayed.

Format

Displays a list of formats that you can select for the style.

Embed/Link

Sets the paste type: linked or embedded.
