Cage Style command

In the Subdivision Modeling environment, use the Home tab→Style group→Cage Style command to control how vertices and edges display on cages.

Cage Style dialog box

Edges
Color

Select a color from the list to apply to cage edges.

Size

Select a pixel size from the list to apply to cage edges.

Vertices
Color

Select a color from the list to apply to cage vertices.

Size

Select a pixel size from the list to apply to cage vertices.

Reset

Return all settings to the default values.

OK

Applies cage style settings.
