Draw a rectangular pattern sketch

Note:

Drawing pattern profiles is part of the ordered pattern feature construction process. The Rectangular Pattern command is available when constructing a pattern feature or when drawing an ordered sketch.

  1. Choose Home tab→Features group→Rectangular Pattern .

  2. Click where you want to start the pattern rectangle.

  3. Click where you want to end the pattern rectangle.

  4. Use the options on the Rectangular Pattern command bar (sketch) and the Stagger Options dialog box to define characteristics of the pattern, such as the number of pattern occurrences, pattern rectangle size, and occurrence spacing.
