Create an unmanaged feature library member

  1. Click the Feature Library tab.

  2. On the Feature Library page, use the arrow button on the Look In box to specify the location of the feature library folder.

  3. In the Solid Edge window or on the PathFinder page, select the feature(s) you want to copy to the feature library.

  4. Right-click and choose Copy. Display the Feature Library tab, right-click and choose Paste.

    Note:

    You can also click Add Entry button to perform a copy and paste into the feature library.

    When adding ordered features, a feature library member document is added to the Feature Library tab, and the Feature Set Information dialog box appears.

  5. In the Feature Library Entry dialog box, type a name for the library entry.

    Note:

    For Solid Edge data management users who selected the option Automatically name files via Document Number and Revision, the system displays a dialog box where you can assign common properties such as document number and revision number, as explained in Create documents with unique properties.

Tip:

  • To store multiple features as a unit, press the Shift or Ctrl keys, and select the features.

  • You can use the Rename command on the shortcut menu to change the name of the new feature library member document on the Feature Library page.

  • For ordered features, you can define prompts and notes for the elements in the Feature Library Information dialog box.
