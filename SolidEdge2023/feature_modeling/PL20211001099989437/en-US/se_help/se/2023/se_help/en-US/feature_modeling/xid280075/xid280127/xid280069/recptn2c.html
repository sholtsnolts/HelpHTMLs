Rectangular Pattern command (ordered sketch)

The Home tab→Feature group→Rectangular Pattern command draws an ordered rectangular pattern profile, which is used to define how a selected element is patterned. A rectangular pattern profile consists of the profile rectangle and x shaped symbols that represent the positions of the patterned features. You can specify the x and y spacing, the x and y count, the pattern height and width, and any pattern occurrences you want to suppress.

For more in-depth information on working with rectangular and circular patterns, see the Pattern command (ordered) help topic.

Note:

A Pattern profile is the same as any other sketch in Solid Edge. You must apply relationships and dimensions so it will behave predictably.
