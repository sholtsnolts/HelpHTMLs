Emboss Options dialog box

Show/Hide Tool Bodies
Show tools if they are design bodies

Displays tool bodies on completion of the command if a tool body is a design body.

Show tools if they are construction bodies

Displays tool bodies on completion of the command if a tool body is a construction body.

Sheet Metal Rounding

Specifies rounding for sheet metal bodies. These options are only available when the target body is a sheet metal solid.

Include die-side rounding

Specifies that the die side edges of the emboss are rounded.

Die radius

Specifies the radius value for the die side rounding.

Include punch and punch punch-side rounding

Specifies that the punch side corners of the emboss are rounded.

Punch radius

Specifies the radius value for the punch side rounding.
