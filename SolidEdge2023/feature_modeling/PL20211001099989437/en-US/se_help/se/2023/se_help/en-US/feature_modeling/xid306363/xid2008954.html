Assembly Subtract Options dialog box

Create Assembly Features

Specifies that the entire feature resides in the assembly document. With an assembly feature, the reference plane, profile, and surface geometry all reside in the assembly document where the feature was constructed. Assembly features affect only the parts that physically intersect the feature extent.

Note:

This applies only to objects based on internal components, such as frames, pipes, fittings, and gusset plates.

Create Part Features

Part features are the same as features created using in-place activation and reside in the part document. Write access in the part document is required to create part features. Part features are created as ordered features.

Create Synchronous subtractions if possible

When selected, attempts to create synchronous subtractions.

Recognize holes

Recognize holes

When selected, finds candidates in the model and replaces them with hole features.

Recognize holes as threaded holes if the tool is threaded

Recognizes the holes as being threaded if the tool is threaded.

Match the nearest larger hole from database for non threaded cutouts (to create a clearance hole)

Matches non threaded cutouts with a hole with the nearest larger diameter so it creates a clearance hole.

Do not recognize holes greater than

Recognizes only holes less than the specified diameter.

Maintain links for Ordered operations

Maintains any links created by ordered operations.

Remember the Setting and Do Not Display This Dialog Again During this Session.

Specifies that the dialog box will not be displayed again during the current session when set.
