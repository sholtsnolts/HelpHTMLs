Replace Face command bar

Steps
Select Face Step

Defines the faces you want to replace.

Select Replacement Step

Defines the faces, surfaces, or plane that will replace the part face.

Cancel/Finish

Cancels the command or finishes the feature.

Command Bar Options
Select

Sets the face selection method for the feature.

  • Face—Allows you to select individual faces.

Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selection.

Name

Displays the feature name. Feature names are assigned automatically. You can edit the name by typing a new name in the box on the command bar or by selecting the feature and using the Rename command on the shortcut menu.
