Insert Object dialog box

Create New Spreadsheet

Specifies that you want to create a new Microsoft Excel spreadsheet where you enter the X,Y,Z coordinate values to create a curve.

Create From File

Specifies that you want to use an existing Microsoft Excel spreadsheet to create a curve by editing or entering new X,Y,Z coordinate values in the spreadsheet.

Browse

If you set the Create From File option, Browse allows you to set the path for the existing spreadsheet.
