Recreate a published multi-body part

  1. Open a published multi-body part.

  2. Assume the published files get deleted to set up the need for a recreate. Delete all of the published files.

  3. Click the Multi-body Publish command.

  4. On the Multi-body Publish dialog box, notice that all of the published files have a question mark in the status column. This means that the linked files could not be found. Right-click each published filename and choose Recreate. You can also select all of the published files at one time and choose Recreate.

  5. The status field changes to a dash. Click the Save Files button. Notice that each design body publishes again and the status changes to show a link symbol.
