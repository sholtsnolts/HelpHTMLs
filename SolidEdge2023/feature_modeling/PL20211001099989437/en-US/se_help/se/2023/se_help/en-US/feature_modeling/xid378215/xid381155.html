Move or edit a slot in the synchronous environment

In the synchronous environment, you can:

  • Move a slot along the face

  • Edit slot parametersto change such things as slot width or extent

  • Edit the slot profile to make changes to the sketch element used to create the slot

  • Move a slot group along the face

  • Add slots to an existing slot group

  • Remove a slot to an existing slot group

  • Separate a slot from a group

Move a slot

  1. Choose Home tab→Select group→Select .

  2. Select the slot you want to move.

  3. Click the steering wheel axis shown.

  4. Drag the slot to a new location.

    Tip:

    You can select the torus of the steering wheel to rotate the slot.

  5. Click to save the edit.

    Tip:

    You can select the torus of the steering wheel to rotate the slot.

Edit slot parameters

  1. Choose Home tab→Select group→Select .

  2. Select the slot you want to edit.

  3. Click the edit handle.

    The Slot command bar, dynamic edit box, feature origin, and feature profile edit handle are displayed.

    Note:

    The appearance of the dynamic edit box depends on the type of slot. For example, for counterbore slots, you can also edit the path offset and depth offset.

  4. Use the command bar and dynamic edit box to make changes to the slot.

  5. Click to save the edits.

    Tip:

    • You can right-click the feature origin and use the commands on the shortcut menu to reposition the feature or feature origin.

    • If you create a slot with a finite extent and a slot floor face is created, the extent is edited as finite. The From/To option is not available when you edit the slot.

Edit a slot profile

  1. Choose Home tab→Select group→Select .

  2. Select the slot you want to edit.

  3. Click the Slot edit handle.

  4. Click the Edit Profile edit handle.

  5. Make changes to the existing slot profile.

  6. Click the green Accept button .

  7. Right-click to save the edit.

  8. Click to finish.

Move a slot group

  1. In PathFinder, select the slot group you want to move.

  2. Use the steering wheel to move all paths associates with the slot group.

  3. Right-click to save the move.

Add slots to a slot group

  1. In PathFinder or the graphic window, select a slot occurrence within the group.

  2. Click the Slot edit handle.

  3. On the command bar, click the More Slots button .

  4. Click the paths for the slots you want to add.

  5. Right-click to add the slots.

  6. Click to exit the command.

Remove a slot from a group

  1. In PathFinder, right-click the slot occurrence within the group.

  2. On the shortcut menu, click Delete.

    The path is deleted (1) and the entry is removed from PathFinder (2).

Separate a slot from a group

  1. In PathFinder, right-click the slot occurrence within the group.

  2. On the shortcut menu, click Separate.

    The slot is separated from the slot feature and added to a new slot group in PathFinder.

    Note:

    The Detach, Attach, and Break commands are also available for slots.
