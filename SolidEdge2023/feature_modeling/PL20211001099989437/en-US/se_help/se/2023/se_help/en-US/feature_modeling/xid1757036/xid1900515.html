Scale a subdivision model

  1. In the Subdivision Modeling environment, choose the Home tab→Modify group→Scale command .

  2. Select the elements to scale. You can click any elements of the body cage: faces, edges, or vertices.

  3. On the command bar, click a scale direction button: 3 Axes Scaling, Planar Scaling, or Linear Scaling.

  4. If you selected planar or three-axis scaling, also select or clear the Uniform scaling option.

  5. Specify the origin of scale—The system places the scale origin at the center of the select set, but you can drag the center knob of the steering wheel to any keypoint in the cage body.

    If you move the steering wheel, recenter the scale origin to the center of the selection set by doing either of the following:

    • Double-click the origin (center knob) of the steering wheel.

    • On the command bar, click Recenter .

  6. To begin scaling dynamically, click+drag one of the scale axes on the steering wheel. For precision scaling, type a scale value. If you are using non-uniform scaling, press the Tab key to move between axes to type new values for each axis. When scaling dynamically, click to finalize the scale, and then select another axis if you want.

    Valid scale values are between 0.001 and 100.

  7. Right-click to complete the command.
