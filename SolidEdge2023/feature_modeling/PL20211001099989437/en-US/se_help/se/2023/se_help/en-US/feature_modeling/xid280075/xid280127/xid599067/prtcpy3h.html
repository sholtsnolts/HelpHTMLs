Create a mirror copy of a part

  1. Choose Home tab→Clipboard group→Part Copy.

  2. On the Select Part Copy dialog box, select the part or assembly you want to mirror, and then click OK. The part or assembly is temporarily placed in the document.

  3. On the Part Copy command bar, click the Part Parameters button.

  4. On the Part Copy Parameters dialog box, set the Mirror Body option, select the reference plane you want the part mirrored about, and then click OK.

    Note:

    If you want the part copy to be associative to the original part, set the Link To File option.

  5. On the Part Copy command bar, click Finish.

Tip:

  • If you insert the part copy before other geometry is constructed, you can place the part copy as a base feature or as a construction element. If you insert the part copy after other geometry is constructed, you can only place the part copy as a construction element. You cannot add features to a part copy placed as a construction element.

  • You can use PathFinder to display the document name the part copy was created from.

  • When you select the reference plane you want to mirror the part about, the reference plane also highlights in the application window.

  • When you select an assembly document, the parts in the assembly are combined into one solid body. This capability can be very useful when working with assemblies of weldments.

  • You can use the Open command on the shortcut menu to open the original document to make changes to it.
