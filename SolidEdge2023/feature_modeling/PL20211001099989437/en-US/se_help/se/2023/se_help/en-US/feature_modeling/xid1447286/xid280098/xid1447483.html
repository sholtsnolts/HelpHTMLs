Main Steps
Path Step

Defines the path along which you want to sweep the tool body.

Tool Step

Defines the tool body to be spun and swept along the path.

Axis Step

Defines the axis about which the selected tool is spun.

Lock Direction Step

Defines the orientation of the swept tool.

Place Tool on Path

Specifies whether the tool should be placed on the selected path before sweeping.
