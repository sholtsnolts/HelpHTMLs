Hole database converter

For legacy hole databases (holes.txt, threads.txt), a holes database converter is provided. These two legacy databases are written to a new hole database file and are located in the C:\Program Files\Siemens\Solid Edge 2023\Preferences\Holes folder. Solid Edge only recognizes the hole databases that are contained in this folder.

Converter utility location: C:\Program Files\Siemens\Solid Edge 2023\Custom/HoleDatabaseConverter/bin/Hole Database Converter.exe.

Note:

Make sure the decimal point symbol of the values in the text file matches the system decimal point. The values will be converted incorrectly if the symbols do not match.
