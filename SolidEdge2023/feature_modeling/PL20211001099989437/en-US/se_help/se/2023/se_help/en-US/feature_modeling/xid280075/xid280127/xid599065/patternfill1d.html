Fill Pattern command bar

  • Available Actions

    Lists alternate actions which are available for the selected element.

    • Rectangular

      Specifies that you want to construct a rectangular pattern.

    • Circular

      Specifies that you want to construct a circular pattern.

    • Along Curve

      Specifies that you want to construct a pattern along a curve.

    • Fill Pattern

      Specifies that you want to construct a fill pattern.

  • Fill Style

    Specifies the style for the fill pattern.

    • Rectangular

      Fills a region(s) with rows and columns of occurrences.

    • Stagger

      Fills a region(s) with staggered rows of occurrences.

    • Radial

      Fills a region(s) with radial rings of occurrences.

  • Fill Spacing Method

    Switches between the available fill spacing methods

    • Target Spacing

      Specifies the radial spacing of occurrences and the occurrence spacing on each ring.

    • Instance Count

      Specifies the number of occurrences per ring.

  • Suppress Instance

    Specifies which pattern occurrences you want to suppress. You can select an individual occurrence by clicking it, or a group of occurrences by clicking and dragging.

    • Use Occurrence Footprint

      Uses the footprint of the patterned object to determine whether the object is inside the fill region. This is the default setting. To use the occurrence marker method, turn this option off.

    • Allow Boundary Touching

      Allows the footprint of the occurrence to touch the boundary without suppressing.

  • Center Orient

    Controls the orientation of the radial occurrences. When this option is selected, the steering wheel changes and an arrow inside the torus displays. This arrow orients the occurrences.

    This option is available with radial fill only.

  • Add to Pattern

    Adds geometry to the pattern definition.

  • Deselect (x)

    Cancels the current input.

  • Accept (check mark)

    Accepts the current input.
