Divide Part command bar

Steps
Select Cutting Surface

Specifies the geometry you want to use for a cutting surface. You can select a reference plane or a construction surface.

Side Step

Defines which side of the geometry is copied to the new part file.

Cancel/Preview/Finish

Cancels the Divide Part command, previews the cut portion of the model that is saved as a new part, or finishes the divide part.

Other command bar Options
Next Cut

Specifies that you want to define another division in the part.
