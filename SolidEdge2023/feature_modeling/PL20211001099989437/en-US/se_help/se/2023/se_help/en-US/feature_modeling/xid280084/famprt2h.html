Create a family of parts using the Family of Parts tab

  1. In PathFinder, click the Family of Parts tab.

  2. On the Family of Parts pane, click the New button to display the New Member dialog box.

  3. Type a new member name in the New Member dialog box, then click OK.

  4. Do one or more of the following:

    • To suppress features on the new family member, select the features in the application window or in the Feature PathFinder tab, and then click the Suppress Features button.

    • To add variables to the part family, select dimensions in the application window or select dimensions or variables in the Variable Table, and then click the Add Variables button.

    • Edit the variable values for each family member.

      Note:

      Repeat steps 2 through 4 for each new family member you want to create.

  5. On the Family of Parts pane, click the Edit Table button.

  6. In the Family of Parts Edit Table dialog box, select the family members you want to save in separate documents, then click the Populate Members button.

Note:

  • When you add or remove features from the Suppressed Features list, only the active member is affected.

  • When you add or remove a variable from the Variables list, the variable is added to or removed from all members.

  • To display an individual family member in the graphic window, select the member from the list, then click the Apply button.

  • Family members are associative to the source part.
