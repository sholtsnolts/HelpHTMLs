Thread command bar (ordered)

Thread Options

Displays the Thread Options dialog box (ordered) so you can specify a standard, straight pipe, or a tapered pipe thread type.

Physical Thread

Creates physical threads on a threaded cylindrical feature. When deselected, cosmetic threads are displayed.

You can specify Thread direction (left-handed or right-handed ) to physical threads using the Thread Options dialog box (ordered).

Note:

This option is enabled by default. There is a possible performance impact when creating thread geometry. The system administrator can disable the Physical Thread option on the Solid Edge Administrator dialog box.

Note:

Physical Thread is only available for Standard Thread and is not available for Straight Pipe Thread and Tapered Pipe Thread.

Select Cylinder Step

Selects a cylinder for the thread reference.

Cylinder End Step

Selects the end of the cylinder from which the threads will be offset.

Parameters Step
Finish

Accepts the thread definition and creates the thread feature.

Offset From End

Specifies the distance the threads will be offset from the end of the cylinder.

Depth Method
To Cylinder Extent

Extends the threads the entire length of the cylinder.

Finite Value

Extends the threads a finite distance from an end of the cylinder.

Thread Depth

Specifies the depth of the threads along the cylinder.

Thread Type

Specifies the thread size from the threads listed in the hole database.
