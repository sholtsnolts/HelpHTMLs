Colors tab (Solid Edge Options dialog box)

These options are available in the Solid Edge Part, Profile, Sheet Metal, and Assembly environments.

Color Scheme

Controls the overall color scheme for the active document. The color scheme also affects all the other default color settings on this page.

You can select SE Classic from the Color Scheme list to set the color to the Solid Edge Classic default color. If the default color and the Solid Edge Classic color are the same, SE Classic will not appear in the list.

Background/View Overrides

Opens the Background tab in the View Overrides dialog box.

Highlight

Sets the highlight color.

Selected

Sets the color of selected elements.

Profile section

Sets the color of Profile section.

Guide Path

Sets the color of the guide path.

Sketch

Sets the color of the 2D elements in a profile or a sketch.

Plane

Sets the color of the reference planes. This color is used when you set the Use Shading on Reference Planes option on the View tab on the Options dialog box.

Plane Edges

Sets the color of the reference plane edges.

Handle

Sets the color of driving dimensions, which are locked dimensions, and of geometric relationships.

Driven

Sets the color of driven dimensions, which are unlocked dimensions whose values are derived by relationships to other dimensions.

Failed

Sets the failed color. This color is used for detached dimensions.

Region

Controls the color settings for regions, which are available in synchronous documents.

Opacity

Sets the opacity for the region.

Active Part

Sets the color of the active parts.

Inactive Part

Sets the color of the inactive parts.

Construction

Sets the color of the construction elements, such as construction curves, and construction surfaces.

General body

Sets the color for united bodies created by the Solid Edge Simulation Unite Bodies command in assembly documents.

Bodies created from non-manifold topology are displayed using the General Body color; bodies created from manifold topology are displayed using the Construction color.

Parent Feature

Sets the color of the parent features. You can display the parent and child features for a selected feature using the Show Parents and Children command on the PathFinder shortcut menu.

Child Feature

Sets the color of child features.

Sketch Relationship Colors

Controls the color settings for sketch elements based on whether they are fully constrained.

When you select a sketch element and the Relationship Colors command is selected on the 3D Sketching tab or on the Inspect tab, the 2D element handle shows a box filled in the color that represents the element's relationship status.

Note:

In XpresRoute, this option is available only for assembly sketches. It does not apply to 3D paths drawn in XpresRoute.

Fully defined

Sets the color of fully constrained elements. Elements are fully constrained when none of the element handles are free to move.

Under-defined

Sets the color of under-constrained elements where one or more of the element handles are free to move.

Over-defined

Sets the color of an over-constrained element.

Inconsistent

Sets the color of an inconsistent element.

Live Section Colors

Controls the color settings for Live Sections, which are available in synchronous documents.

Region

Sets the color for live section regions.

Opacity

Sets the opacity for live section regions.

Edges

Sets the color for live section edges.

Centerlines

Sets the color of live section centerlines.

Region

Sets the color for regions.

Opacity

Specifies the color transparency of the shaded region when you drag a directional fence. The lower the opacity, the easier it is to see the elements that are inside the fence.

Closed Sketch

Sets the color for closed sketches.

Opacity

Specifies the color transparency of the shaded region when you drag a directional fence. The lower the opacity, the easier it is to see the elements that are inside the fence.

Use shading on 2D Selection fence

Specifies that a shaded region is displayed when you drag a directional fence to select 2D elements. Draft and ordered sketches use this setting when the Select Tool is set to Directional Fence.

Opacity

Specifies the color transparency of the shaded region when you drag a directional fence. The lower the opacity, the easier it is to see the elements that are inside the fence.

Inside

Specifies the color of the shaded region when you drag a directional fence from left to right.

Inside/Outside and overlapping

Specifies the color of the shaded region when you drag a directional fence from right to left.
