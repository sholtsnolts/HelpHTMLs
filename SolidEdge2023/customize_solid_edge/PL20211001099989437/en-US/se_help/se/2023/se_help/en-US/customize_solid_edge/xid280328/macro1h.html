Run a macro

  1. On the File menu, click Tools→Run Macro .

  2. In the Run Macro dialog box, click the macro you want to run.

  3. Click the Open button.

Tip:

You can create macros with any programming tool that supports OLE automation, such as Microsoft Visual Basic.
