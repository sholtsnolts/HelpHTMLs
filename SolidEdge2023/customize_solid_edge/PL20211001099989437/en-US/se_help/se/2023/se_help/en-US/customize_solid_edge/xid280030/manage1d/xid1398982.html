Multi-CAD tab, Manage page (Solid Edge Options dialog box)

The Multi-CAD tab is a member of the tab set available in the Manage page of the Solid Edge Options dialog box. Use options on this page to specify options for Teamcenter-managed Multi-CAD documents.

Solid Edge Templates
Assembly

Defines the default template used for translating a non-Solid Edge assembly into Solid Edge in the multi-CAD workflow.

Part

Defines the default template used for translating a non-Solid Edge part into Solid Edge in the multi-CAD workflow.

When opening a Solid Edge assembly that uses a foreign CAD assembly
Include assembly features from the foreign assembly

Brings assembly override information from Teamcenter and integrates it into the Solid Edge assembly.

When saving an assembly to Teamcenter
Save a Solid Edge dataset to each configured revision as a driven reference

Captures your intent to save Solid Edge content back to Teamcenter when you are working in a multi-CAD environment. When selected, Solid Edge datasets are created under the source Item Revision. This option is applicable only for indirect files and not when you directly open a JT file.
