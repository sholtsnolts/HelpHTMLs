Customize the Ribbon command

Displays the Customize dialog box, where you can customize the command ribbon to fit your workflow.

  • Use the Ribbon page (Customize dialog box) to add, remove, and reorder commands on the command ribbon.

Note:

  • The changes you make are set per environment, so you can use different settings in different environments. This also means that when you change documents between Draft and Part, for example, you need to customize your settings in both places to keep the same options available.

  • Command ribbon settings are stored in a file named <ThemeName>RibbonCustom.xml; this file resides in the application data folder of the operating system that Solid Edge is being loaded on.

    Example:

    If you create a theme named Basics, the command ribbon settings is stored in the file BasicsRibbonCustom.xml.
