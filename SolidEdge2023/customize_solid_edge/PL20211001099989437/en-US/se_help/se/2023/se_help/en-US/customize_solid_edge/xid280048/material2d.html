Material Properties tab

Lists the properties for the selected material and its face and fill styles.

Material Properties

Displays the name of the current material in the Displaying properties of: field. When you select a material from the material library PathFinder, a set of associated material and mechanical properties is displayed. You can review or modify the properties directly in the table. For example, when you select the material, Brass, from the list, the Material Properties tab updates to display the density, coefficient of thermal expansion, and so forth for brass.

To learn how to create or edit a material, see Create a new material.
