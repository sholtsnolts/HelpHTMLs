Importing and exporting materials

Importing and exporting materials is performed in the Material table. The Save As and Import commands are located on the context menu when right-clicking on a material library.

Save As (export)

Use the Save As (or export) command to:

  • Save the material library to a new name.

  • Save the material library to Microsoft Office Excel Worksheet (.xls or .xlsx format).

Note:

You can edit and add custom properties in the Excel spreadsheet and then later import it to a new library.

Import

Use the Import command to select a material library (.mtl) or an Excel file (.xls or .xlsx) to import. The active library materials are overwritten by the imported materials.

Note:

If you want to merge the materials, import to a new library and then use Copy and Paste.
