Properties command

Edits the properties of a selected element or object. The different properties dialog boxes that you can access with this command depend on the environment you are working in.

Note:

You must highlight an element or object before selecting this command.

In a Teamcenter-managed environment:

  • If no files are selected in PathFinder, the action applies to the active document.

  • If a file is selected in PathFinder, the action applies to the file.

  • If multiple files are selected in PathFinder, the command is disabled.
