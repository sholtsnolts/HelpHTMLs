Edge Display tab (Solid Edge Options dialog box)

Defines and applies the styles of visible, hidden, and tangent edges for part views in a document.

If you specify display defaults for a drawing view with the Drawing View Display Defaults dialog box, those settings override the settings on the Edge Display tab.

Reference Parts
Derive "Display as Reference" From Assembly

Specifies that the occurrence properties defined in the assembly document determine whether the occurrence is displayed as a reference part. You can use the Occurrence Properties command on the PathFinder shortcut menu to specify that an assembly occurrence is displayed as a reference part in a drawing.

Show all reference parts as transparent (all occurrences in drawing view)

Displays reference parts in an assembly drawing view as transparent parts.

Visible edge style

Sets the line style for visible edges on reference parts.

Hidden edge style

Sets the line style for hidden edges on reference parts.

Tangent edge style

Sets the line style for tangent edges on reference parts.

Visible edge style

Sets the line style for visible edges. You can select from a list of all line styles available in the document.

Center of Mass and Coordinate System edge style

Sets a line style to draw the point associated with the center of mass (CM) and coordinate system (CS) when they are shown in the drawing view.

The default option is the Visible line style.

Note:

This option does not apply to block graphics displayed as the CM or CS.

Hidden edge style

Sets the line style for hidden edges. You can select from a list of all line styles available in the document.

Show hidden edges in:
Orthographic views

Displays the hidden edges on parts in orthographic part views.

Pictorial views

Displays the hidden edges on parts in pictorial part views.

Show hidden-tangent edges

Specifies that hidden-tangent edges are generated with the drawing view. The edges are displayed using the hidden edge line style.

Use this option to:

  • Show the lines between rounds and tapered faces, which do not have a sharp edge.

  • Show the external edges on the back side of a cast part.

Show edges hidden by other parts in:
Orthographic views

Displays the parts that are hidden by other parts in orthographic part views. The edges are displayed using the hidden edge line style. This option applies when working with assembly drawings.

Pictorial views

Displays the parts that are hidden by other parts in pictorial part views. The edges are displayed using the hidden edge line style. This option applies when working with assembly drawings.

Show hidden-tangent edges

Specifies that hidden-tangent edges are shown along with the edges that are hidden by other parts. The edges are displayed using the hidden edge line style.

Use this option to:

  • Show the lines between rounds and tapered faces, which do not have a sharp edge.

  • Show the external edges on the back side of a cast part.

Do not show hidden edges in section views

Prevents the display of hidden edges from source views in section views.

Tangent edge

Sets the line style for tangent edges. You can select from a list of all line styles available in the document.

Show tangent edges
Orthographic views

Displays the tangent edges on parts in orthographic part views.

Pictorial views

Displays the tangent edges on parts in pictorial part views.

Show Fill style in section view

Specifies a fill style for section views.

Derive from part

Specifies that the section view fill style should be the same fill style used in the associated part file.

Automatically alternate hatch spacing in section views

Specifies that different parts are displayed automatically with different hatch pattern spacing. This aids in distinguishing one cut face from another.

You can adjust the hatch pattern spacing in the selected section or broken-out section view using the Spacing box on the Display tab (Drawing View Properties dialog box).

Automatically alternate hatch angle in section views

Specifies that different parts are displayed automatically with different hatch pattern angles. This aids in distinguishing one cut face from another.

You can adjust the hatch pattern angle in the selected section or broken-out section view using the Angle box on the Display tab (Drawing View Properties dialog box).

Show fill style in sectioned weld beads

For weldment assemblies, automatically applies the fill style selected for section views to cut weld bead faces.

When this option is selected, the fill style shown in the Show Fill style in section view list (Edge Display tab) also is used to display the cut weld bead faces.

Solid fill sectioned weld beads

When selected, specifies that the cut weld bead faces are displayed using the solid fill style color. When deselected, the faces are displayed using the underlying hatch pattern in the fill style.

Note:

When both options are deselected, no fill is shown on the cut weld bead faces in section views.

Nailboards
Do not show hidden edges in connector views

Prevents the display of hidden edges in connector views.

Print white conductors in:

Specifies the color in which white conductors are printed.

Advanced

Displays the Advanced Edge Display Options dialog box.
