Units page (Solid Edge Options dialog box)

Controls the settings of the active file’s unit system, dimension style, base units, and derived units.

Units System

Unit System allows user to set the base and derived units for the active file based on the selected unit system. The unit system consists of the default unit of measurement for length, mass, time and angle.

  • MMKS (millimeter, kilogram, second) — Set the base unit of measurement for length, mass, time and angle.

  • IPS (inch, pound, second) — Set the base unit of measurement for length, mass, time and angle.

  • MKS (meter, kilogram, second) — Set the base unit of measurement for length, mass, time and angle.

  • FPS (foot, pound, second) — Set the base unit of measurement for length, mass, time and angle.

  • CGS (centimeter gram, second) — Set the base unit of measurement for length, mass, time and angle.

Units and Dimension Styles

Sets the active dimension style to match the length units in the current design file, this functionality is turned off when unchecked. This control is disabled when dimension style mapping is used.

Base Units

Base units are length, mass, time, angle, and temperature. which are set by default from the selected unit system. These units can be changed by selecting the drop down menu in the value and precision fields. If the value and precision fields are changed from the unit system default the background color for these field will highlight indicating a change.

The precision readout sets the number of significant figures to display. It sets the accuracy of the unit readout value. The precision setting does not alter the numbers that you type, only the display of the numbers in the box. Values ending in 5 are rounded up. For example, if the precision readout is set to .123 (three decimal places), and you draw a line that is 2.1056 inches long, then the line value length is rounded to three decimal places. The length value appears as 2.106 inches long.

Derived Units

Derived Units are all units other than the base units such as area, volume, and density. These units can be changed by selecting the drop down menu in the value and precision fields. If the value and precision fields are changed from the unit system default the background color for these field will highlight indicating a change. Changes in the base unit will still drive the derived units. Modifying a derived unit does not change the base unit.

Setting Values for derived units

You set the actual values for the mechanical properties by assigning a part material using the Material Table command. For example, to set the density value for an aluminum part, you can select from one of the available aluminum materials in the Material Table, or define a new  aluminum material that matches the working characteristics of the aluminum you need. For more information, see the Material Table command Help topic.
