Customize dialog box

You can use the Customize dialog box, for example, to restore legacy Solid Edge commands that were retired or replaced by newer commands. You can add commands to the command ribbon, the Quick Access Toolbar, the radial menu, or to all of these locations.

Tabs

Keyboard—Customizes keyboard shortcut sequences.

Quick Access—Adds and removes commands on the Quick Access Toolbar.

Ribbon—Customizes the command ribbon.

Radial Menu—Customizes the radial menu.

Layout—Customizes the arrangement of docking windows, user assistance tools, and graphics tools.
