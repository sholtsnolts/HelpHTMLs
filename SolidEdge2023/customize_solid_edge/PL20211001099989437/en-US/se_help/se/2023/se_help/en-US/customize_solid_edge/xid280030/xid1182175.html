Manage tab, Solid Edge Options dialog box

Specifies that you want to use the data management capabilities integrated into Solid Edge. Included are options for fast search vault definition, custom properties, file naming, revision rules, and workflows.

Vault Definition tab

Location of FastSearchScope.txt

Displays the location of the FastSearchScope.txt file, which defines your vault locations. The default location is \Program Files\Siemens\Solid Edge 2023\Preferences. To select a FastSearchScope.txt file from another location, use the browse button .

Vault

The vault box, which is the large box in the Vault Definition tab, displays all the locations you added to the vault.

Add

Click to add a new vault location. If not already indexed, Solid Edge automatically indexes the newly added location.

Remove

Click to remove a location from the vault. Solid Edge also removes the location from the Windows indexing list.

Document Controller (Windows user account)

Specifies the names of document controller users, separated by semicolon, exactly as they appear in the Windows user account. Document controller users have special privileges to change the status of Solid Edge documents from Released to Obsolete.

Automatically revise or copy drawings that use the selected 3D document

Automatically copies and revises drawings that use the selected 3D document. If there are multiple drafts, the system adds an increment ID between the document number and revision number. For example, test1-1.dft, test1-01-1.dft, test1-02-1.dft, and so on.

Enable distributed file access when using file replication services

If you share Solid Edge files using Xcelerator Share, Dropbox, or other file-sharing software, this option locks the files while they are being accessed by other users. The Read-Only Assistant informs you that the files are locked.

Prompt to search additional locations

Searches locations you specify, in addition to your indexed locations.

Custom Properties tab

I have custom properties in my Solid Edge file(s) that I want to perform fast search on. I have defined them in the following location(s):
Property seed file*:
User templates*:
Note:

(*) Denotes that the path to these files is defined in the File Locations page.

View Custom Properties

Opens the Registered Custom Properties dialog box, which lists the name and type of all of the properties in the propseed.txt file and user templates. These properties are registered in Windows Indexer and are used for Fast Search.

Set Custom Properties

Collects all custom properties from the default templates and property seed file.

Rebuild Index

Opens the Microsoft Indexing Options dialog box and rebuilds the index.

Document Naming Rule tab

Document Numbering Scheme

Automatically generates a unique document number when creating new files.

Prefix—The first box defines the prefix. If not needed, leave it blank. The second box is an incremented number that is saved in the NumberGenerator.dat file. The incremented number must only contain numeric values.

Number Generator File—Displays the location of the NumberGenerator.dat file. To select a NumberGenerator.dat file from another location, use the browse button . For multiple users, the NumberGenerator.dat file should be located on a mapped or shared network drive, and all users should have read and write access to it.

Automatically name files via Document Number and Revision

This option is not on by default. Selecting this option displays the following information in the Design Manager document list:

This column in the document list

Displays

New Filename

The document number, the delimiter, and the revision number. This column is read-only.

Doc Number

Displays the current document number.

Revision

Displays the current revision number. For new files, it displays the initial value, as specified in the Solid Edge Options dialog box→Manage tab→Document Naming Rule tab.
Revision Numbering Scheme

Revision delimiter—Specifies the delimiter character for use with the Revisions command to separate the document name from the revision number.

Maximum length—Specifies the maximum number of characters allowed for a revision. Valid values are 1 through 5.

Excluded characters—Specifies the characters that are not allowed in a revision value.

Document Name Formula

Edit Formula—Displays the Document Name Formula dialog box, so you can replace the display of the file name with a formula composed of document properties.

Formula box—Displays a value you compose by selecting various document properties. The list of available properties you can choose from is based on your active document. You can use the formula in place of the file name throughout Solid Edge.

Life Cycle tab

Modify

Allows you to change the location of a folder or file.

Reset

Sets the selected file location path to the default setting.

Reset All

Sets all selected file location paths to the default setting.

Managed Entity/Location

This box displays the designated location for each managed entity.

The standard parts location defines where standard parts are stored, to prevent cloning and revision in Design Manager.

Show checked statuses below in Design Manager and Property Manager

Displays the selected statuses in the SE Status column.

The following status options can be hidden from the selection, if they are not being used:

  • In Review

  • Obsolete

  • Baselined

Automatically obsolete previous revisions while releasing a file

Specifies whether or not Solid Edge automatically makes documents at the same revision level, also known as sibling documents, and documents at a lower revision level, also known as parent documents, obsolete.

You do not have to be a document controller user to set this option.

Automatically set the draft status to match the status of the part placed in the drawing

Specifies whether or not Design Manager automatically sets the status of a draft document to correspond to the status of the associated CAD file.

Workflow tab

Location of Workflowsettings.txt

Displays the location of the Workflowsettings.txt file, which stores the names of reviewers and the SMTP server information. The default location is \Program Files\Siemens\Solid Edge 2023\Preferences. To select a Workflowsettings.txt file from another location, use the browse button .

Reviewers

The reviewers' box, which is the large box in the Workflow tab, displays all the reviewers you specified for workflows.

Add

Click to add a new reviewer.

Remove

Click to remove a reviewer.

SMTP Server

Specifies the name of the SMTP server.
