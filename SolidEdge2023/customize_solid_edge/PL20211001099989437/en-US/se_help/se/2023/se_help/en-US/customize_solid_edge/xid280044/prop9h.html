Set the units of measure


Setting the units of measure affects material and physical properties applied to the model. You also can apply the unit change to the length units used in dimensions.

  1. Open the Units tab in the Solid Edge Options dialog box by doing one of the following:

    • On the File menu, select Info→File Units.

    • In the Material Table dialog box, click Units .

  2. On the Units page, select the desired Units System for the document.

    The selected unit system sets the units in the Base Units section of the dialog box. Base units affect length, mass, time, angle, and temperature.

  3. When you change the unit system, you also can change the length units used in dimensions.

    In the Units and Dimension Styles section, observe that the Change active dimension style to match length units check box is selected, and the current and new dimension style information is displayed.

    Example:

    If you change the unit system from MMKS (millimeter, kilogram, second) to MKS (meter, kilogram, second), then the dimension style units can be changed to match:

    Current style=ISO (mm)

    New active style=ISO (m)

    Clear the check box if you do not want to change the dimension units to match the base units. The current and new dimension style information resets to the original length units.

    Example:

    Current style=ISO (mm)

    New active style=blank

  4. (Optional) You can set the units for calculated properties such as area, volume, and density of a part. To do this, in the Derived Units section of the dialog box, in the Value column, click the cell for a derived unit that you want to change, and then choose a value from the list.

  5. Click Apply to save the new settings.

  6. Click OK to close the dialog box.

    If you changed the units starting from the Solid Edge Material Table, then the Material Table dialog box is displayed.

Tip:

  • If you need to specify a different delimiter for primary and secondary dimension units, or to display dual units, you can do this by updating the dimension style globally or by making changes to the dimension style locally, in the document.

    • To set dimension units globally in the Dimension style, use the Styles command. After updating the style, you can map dimension styles so that they are used automatically.

    • To set dimension units locally, see the Help topic, Set dimension units.
