Ribbon page (Customize dialog box)

Adds, removes, and organizes commands on the command ribbon bar. To learn how to use this page, see Customize the command ribbon.

Environment to customize

Specifies the environment to which you want to add a custom command ribbon bar. Select the environment first.

Choose commands from

Specifies the source of commands you want to add to the command ribbon bar. Select the source second.

To add entries from the File menu to the command ribbon, set Choose commands from:=System Menu.

Tabbed section

The list on the left specifies which command from the selected category you want to add to the command ribbon bar.

Theme

Specifies the active user interface theme. To learn more about themes, see Solid Edge themes

Save

Saves changes to the active command ribbon.

Save As

Saves changes to a new command ribbon which you name.

Delete

Deletes the active command theme.

Note:

You cannot delete the Balanced - Solid Edge Default theme.

Add

Adds the selected item from the list on the left to the active item in the list to the right, in the following order:

  • If the item from the list on the left is a tab, it is added as a tab on the list to the right.

  • If the item from the list on the left is a group, it is added to the active or selected tab on the list to the right.

  • If the item from the list on the left is a command, it is added to the active or selected group on the list to the right.

  • If the item from the list on the left is a macro, it is added to the active or selected tab or group on the list to the right. An image icon is displayed automatically with the macro.

    Note:

    If the macro contains an image icon, Solid Edge uses that image. If the macro does not contain an image, Solid Edge attempts to locate an associated image as follows:

    • If the macro is a .exe or .dll, Solid Edge looks in the same directory for an .ico, .png, or .bmp image file—in that order. It applies the first image found.

    • If no image file is found with the macro file, Solid Edge queries the operating system for an associated image and uses it. This is the same image you see when browsing to the folder using Windows Explorer, when the folder view type is set to show image icons. You can define a different icon by holding down the control key while clicking the Add button. Solid Edge displays a palette of images you can define as the icon.

    • If the macro is a website, Solid Edge checks to see if the website has a favicon available. If so, it downloads and uses the favicon image.

      For more information, see Add a macro to the command ribbon.

Note:

You can add only one instance of a command to a group.

Remove

Removes the selected command from the command ribbon bar.

New Tab

Adds a new tab and an initial group to the list on the right.

New Group

Adds a new group under the selected tab.

Split Button

Adds a node to the ribbon, consisting of a command that can be activated by clicking it, as well as an arrow pointing to a list of additional commands you define.

Example:

In the Home tab→Draw group→Line command is a split button.

Drop Button

Adds a node to the ribbon which activates a list containing the command shown in the ribbon plus additional commands you define.

Example:

In the Home tab→Planes group→More Planes command is a drop button.

Rename

Renames the selected tab, group or command.

Move Up

Moves the tab, group, or command selected in the list on the right up one row.

Move Down

Moves the tab, group, or command selected in the list on the right down one row.

Separator

Adds a vertical graphical separator between commands on the command ribbon bar. You can use separators to organize commands by group.

Note:

  • Separators are inserted after the selected command.

  • You cannot insert a separator as the first or last item in the list, but you can move it using the Move Up and Move Down buttons.

  • You can delete a separator using the Remove button.

Reset

Resets the ribbon to the default settings for that environment.
