Replace the file name with a property value

You can create a document name formula composed of document properties. The formula can be used in place of the file name throughout Solid Edge including: Drawing View Properties, Parts List Properties, QuickPick, the application title bar, and PathFinder.

For a list of properties available by document type, see the help topic, Standard and Custom properties.

  1. On the File menu, click Settings→Options.

  2. On the Helpers tab of the Solid Edge Options dialog box, click Change.

  3. In the Document Name Formula dialog box, select a property from the Property list and click Add.

  4. Continue to select and add the properties you want to use as the formula, and then click OK.

    Example:

    You can combine properties with additional characters to separate the two properties, such as [Document Number]-[Revision Number].

  5. In the Solid Edge Options dialog box, click OK.

    Note:

    When documents are open in Solid Edge, editing the document name formula in one environment will not update the document name formula for other documents already open until you close and reopen them.
