Create a new material

  1. In PathFinder, right-click the Material node and choose Material Table, or from the File menu, choose Info→Material Table.

  2. In the Material Table dialog box, right-click a folder where you want the new material to reside. From the shortcut menu, choose New Material.

  3. The default name for the new material is Material 1. Enter a name for the new material, such as Concrete or Portland Cement.

  4. The right pane in the Material Table shows default material properties that you need to define.

    1. Change the face style and fill style as appropriate for the new material.

    2. Consult an engineering handbook or an online resource to find the physical properties associated with the material (for example, density, coefficient of thermal expansion, and yield stress), and enter them in the table.

  5. Click the Apply to Model button, and click Yes in the Save Changes dialog box.

Tip:

  • To edit an existing material, select it from the Material Table, and then update the displayed properties. Click Yes in the Save Changes dialog box.

  • To delete an existing material, select it from the Material Table. Right-click and choose Delete. A Confirm Delete dialog box displays. When you exit the Material Table, a Save Changes dialog box appears.

  • The material names and property sets are stored in an external file, “libraryName”.mtl.

  • The material and mechanical properties are maintained when you place the part in an assembly, render the assembly with Advanced Rendering, create a parts list on a drawing, define a bill of materials, and so forth.
