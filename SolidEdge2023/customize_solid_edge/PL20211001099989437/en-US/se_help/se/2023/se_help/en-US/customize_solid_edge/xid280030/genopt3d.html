General page (Solid Edge Options dialog box, Part environment)

Controls settings, such as the display of 3D effects and the number of entries in the list of recently used files.

Show Units in Value Fields

Displays the units of measurement in value boxes.

Store Geometry In Part Files For Fast Open

Stores geometry display information so your documents open faster.

Reference Plane Size

Specifies the size of the base reference planes.

Feature Origin Size

Specifies the size of feature origins.

Plane Name Display

Displays a dialog box where you can define display settings for reference plane names such as color, height, and font. You can also turn the name display on or off or the plane orientation display on or off.

Prompt for Material in New Model Documents

Specifies that the Solid Edge Material Table dialog box is displayed when you create a new model document, so you can define a material for the document.

Check out-of-date properties in material table on file open.

Checks if materials are out of date, both material properties and gage properties. This is ignored when opening documents in the context of an assembly. An out-of-date indicator is also displayed on the material collector in PathFinder.

Recently Used Files list

Specifies the maximum number of entries you want to see in the Recent Files list on the File menu→Open page (up to 100).

Maximum Print File Size

Sets the maximum print file size. You can set the maximum as high as 160 MB.

Part and Assembly Undo Steps

Specifies the number of steps available in the Undo list in a part or assembly.

Revise and Increment name

Parameters for revising and incrementing filenames.

Revision Delimiter

Specifies the delimeter character for use with the Revisions command to separate the document's name from the revision number.

Starting revision value

Specifies the start character or number for revisions.

Maximum length

Specifies the maximum number of characters allowed for a revision. The value must range from 1 to 5.

Excluded characters

Specifies the characters not allowed in a revision value.

Enable Dynamic Edit of Profile/Sketches

Modifies the shape or size of the profile or relocates a dimension. When this option is cleared, you can only use the Dynamic Edit option to edit the value of a dimension. You can dynamically edit an under-constrained profile. You can also specify whether the profile is recomputed continuously as you drag the mouse, or you can specify that the model is only recomputed after you release the mouse button.

These options are not available for synchronous documents.

Recompute Continually During Edit

Specifies that the model is recomputed as you drag an element's handle to a new location. When this option is set, dynamically editing the shape or size of an under-constrained profile on a large model can be time-consuming.

Recompute After Edit

Specifies that when dragging an element's handles to a new location, the model is recomputed after you release the mouse button.

Enable Value Changes Using the Mouse Wheel

When selected, using the mouse wheel outside the value edit box changes the value in the value edit box. Using Ctrl+mouse wheel outside the value edit box zooms in and out of the model view.

When cleared, using the mouse wheel outside the value edit box zooms in and out of the model view. Using Ctrl+mouse wheel outside the value edit box changes a value in the value edit box.

Recompute Assembly During Sketch Edits

Specifies that part positions are recomputed automatically while you are editing an assembly sketch. This option is useful when parts are constrained to elements on an assembly sketch.

These options are not available for synchronous documents.

Recompute Continually During Edit

Specifies that the assembly is recomputed as you drag an element's handle to a new location. When this option is set, in a large assembly, dynamically editing the shape or size of an assembly sketch can be time-consuming.

Recompute After Edit

Specifies that when dragging an element's handles to a new location, the assembly is recomputed after you release the mouse button.

Profile Undo Steps

Specifies the number of steps available in the Undo list in the Profile environment.

This option is not available for synchronous documents.

IntelliSketch Options

Displays the IntelliSketch dialog box where you can set cursor behavior, set automatic dimensioning options, and specify which relationships are recognized as you draw.

Preferred 2D paste behavior (Draft, Ordered Sketch/Profile)

Specifies whether the Paste command pastes elements to the original position from which you cut or copied the elements, or to the position you select.

Paste to original position

Specifies that elements are pasted to the original position from which you cut or copied the elements.

Paste to selected position

Specifies that elements are pasted to the position you select.

When Creating New Sketches

Specifies region and dimension behavior.

Activate Regions within sketches

Sets display for regions before creating new sketches. The global setting does not impact existing sketches.

Migrate geometry and dimensions when features are created

  • The used sketches move to the Used Sketches collection in PathFinder.

  • Dimensions migrate to the model edges when possible.

Indicate Under-Constrained Profiles in PathFinder

Specifies that a symbol is displayed in PathFinder when a sketch or profile-based feature needs additional relationships to fully define its shape and position.

This option is not available for synchronous documents.

Enable Undo All for Profile/Sketch Modifications

Enables or disables the Undo All command.

This option is not available for synchronous documents.

When Entering Profile/Sketch

Controls whether a new window is created when working with a profile or sketch. You can also specify whether the active window is oriented parallel to the profile plane when drawing a profile or a sketch.

These options are not available for synchronous documents.

Create a New Window

Specifies that a new window is created when you draw a profile or a sketch. The new window is automatically oriented parallel to the profile plane.

Draw in the Active Model Window

Specifies that the existing window is used when you draw a profile or a sketch.

Orient The Window to the Selected Plane

Available when the option Draw in the Active Model Window is selected. Orients the window parallel to the profile plane. This can make it easier to visualize complex profiles and to add relationships. When this option is cleared, the existing window orientation is maintained when drawing a profile or sketch.

Show empty callouts and text boxes

Specifies that the empty callout symbol is displayed to identify a callout or a text box without content.

Leader lines attached to empty callout symbols are still displayed.

Property Text Error

Determines the error displayed when custom properties are undefined.

Do not show error

Prevents errors from displaying if custom properties are not defined.

Show Solid Edge error

Displays the standard Solid Edge error messages.

Show my error

Specifies the error message to display when property text and custom properties are undefined. Use the text box to type the error message string.
