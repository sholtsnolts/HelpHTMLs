Restore the command ribbon to default settings

If you customize the command ribbon, you can restore it to the original settings.

  • Click the Customize Quick Access Toolbar arrow , and from the menu, choose Ribbon Themes. From this list, select Solid Edge Default.

  • Open the Customize dialog box using one of these methods:

    1. Right-click a command on the ribbon and choose Customize the Ribbon.

    2. Click the Customize Quick Access Toolbar arrow , and from the menu, choose Customize the Ribbon.

  • On the right-hand side of the page, for Theme, select Solid Edge Default.

  • Select Close.
