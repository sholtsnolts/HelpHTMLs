Material library property file

The material names and property sets are stored in an external material database file, “libraryName”.mtl. The material file is used to populate the property set for each material in the Material Table dialog box. You can use these materials to define a material for any document on your computer, or other computers on your network.

You can add, edit, delete, move, and copy the material file. You can apply a material to a model. You can add the material library file to a favorites list.

By default, the material file is located in the C:\Program Files\Siemens\Solid Edge 2023\Preferences\Materials folder. You can instruct Solid Edge to look for the material file in a different folder, including a folder on another machine on the network. This makes it easy for all your users to work with a consistent set of materials and properties while providing the capability to customize the materials list. For more information, see Change the location of the material folder.

Note:

You can use the Prompt For Material In New Model Documents option on the General tab of the Solid Edge Options dialog box to control whether you are prompted to assign a material when creating a new document.
