Drawing View Style tab (Solid Edge Options dialog box)

The Drawing View Style tab in the Solid Edge Options dialog box displays the element-to-style mapping table for different types of drawing views and for view annotations, such as cutting planes, auxiliary view planes, and detail envelopes. The styles listed are derived from the current document.

Active drawing view style

Specifies that you want to use the active drawing view style. This style is automatically set by the option on the Unit System on the Units tab.

Use drawing view style mapping

Specifies that you want to use the drawing view style mapping table to determine the format of drawing views and view annotations.

When the Use Drawing View Style Mapping box is checked, the Drawing View Style Mapping button on the applicable command bars is also set.

When this option is cleared, a style must be selected each time a drawing view or view annotation is placed. The selected style becomes the active style until it is changed.

Set all styles to

Specifies a single style to be set for all elements in the style mapping table. For example, you can set the style that all elements will use to ISO or ANSI.

Note:

You can set a different style for a drawing view type or view annotation type by clicking the Style column in the corresponding cell and then selecting a style from the list.

Apply

Sets all elements to the style specified in the Set all styles list.

Element

Lists the elements for which you can set a default style in the Style column. The elements are:

  • Principal and Pictorial Views

  • 2D Model Views

  • Section Views

  • Auxiliary Views

  • Detail Views

  • Cutting Plane

  • Viewing Plane

  • Detail Envelope

  • Nailboard View

  • Connector View

Style

Specifies the style for the corresponding element in the Element column. To change the default style for a particular element type, click the corresponding row in the Style column, and then select a new style from the list.

Note:

  • To set all the elements listed in the Element column to use the same style, use the Set all styles list.

  • The default style for each element is the same as the current default style for the element on the Dimension Style tab (Solid Edge Options dialog box).
