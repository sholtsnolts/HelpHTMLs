Change the Material Table units

  1. In the Material Table dialog box, click Units .

    The Units page (Solid Edge Options dialog box) is displayed.

  2. Continue with the procedure, Set the units of measure.

  3. When you click OK to close the Solid Edge Options dialog box, observe that the new units are applied to the properties in the Material Table.
