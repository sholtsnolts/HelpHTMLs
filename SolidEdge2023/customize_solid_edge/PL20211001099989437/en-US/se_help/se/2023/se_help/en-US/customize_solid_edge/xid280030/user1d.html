User Profile page (Solid Edge Options dialog box)

Specifies information about the user, such as the name, mailing address, and initials used for annotation marks.

Name

Specifies the name of the current Solid Edge user.

This information can be referenced by property text to identify the name of the document creator and of anyone who modifies the document.

Initials

Specifies the initials of the current Solid Edge user.

This information can be referenced by property text to identify the initials of the document creator and of anyone who modifies the document.

Mailing address

Displays the mailing address of the current Solid Edge user.

This information can be referenced by property text to identify the mailing address of the document creator and of anyone who modifies the document.

Feedback and Spotlight
Solid Edge Product Improvement Program - Set Preference

Displays the Solid Edge Product Improvement Program dialog box so you can specify Yes to participate in the Solid Edge Product Improvement Program, or No to decline participation in the Solid Edge Product Improvement Program.

Display Solid Edge Spotlight

When selected, displays in-view, interactive navigation that connects you with resources to enhance your modeling experience.

When cleared, no in-view display occurs.

Settings and Preferences
Deploy Settings and Preferences during startup of Solid Edge

When selected, Solid Edge deploys the SolidEdgeSettingsAndPreferences.sepref file on startup. By default, the system stores the file in the My Documents folder.

Capture Settings and Preferences during exit of Solid Edge

When selected, on exiting Solid Edge, the system captures your settings and preferences in the SolidEdgeSettingsAndPreferences.sepref file. By default, the system stores the file in the My Documents folder. You use this file to deploy your settings and preferences on other machines.

By default, Solid Edge captures the registry settings and the files from the AppData folder. To capture files from the Preferences folder and template files, which are usually stored on a server, use the Settings and Preferences Wizard.

Settings and Preferences file

Specifies the location of the SolidEdgeSettingsAndPreferences.sepref file. To change the location, click Modify.

Solid Edge records this location in the Solid Edge Options dialog box→File Locations tab→File types list, in the Settings and Preferences File entry.
