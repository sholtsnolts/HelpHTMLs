Minimize The Ribbon command

Reduces the on-screen area occupied by the command ribbon by collapsing it to a set of tabs.

  • On the shortcut menu when you right-click an unoccupied area of the command ribbon.

  • On the Customize Quick Access Toolbar menu, which you display by clicking the Quick Access Toolbar arrow.
