Specify hardware parts

You can designate parts such as bolts, nuts, and washers as hardware parts. You also can specify whether hardware is cut in a 3D model section view and in a drawing section view.

Designate a model as a hardware part

  1. Open a Solid Edge part document.

  2. From the File menu, choose Info→File Properties.

  3. In the File Properties dialog box, expand the Summary group.

  4. Locate the row for Hardware Part.

  5. In the Value column, set the value to Yes.

Show cut hardware in a 3D model section view

  1. In the model document, open the Section Options dialog box by doing one of the following:

    • While defining a new 3D section view, click the Options button on the command bar.

    • For an existing 3D section, right-click the section name displayed on PathFinder, and then click Properties on the shortcut menu.

  2. In the Section Options dialog box, select or clear the Cut Hardware check box.

    Tip:

    • If you change this setting for an existing view, the section is recomputed and re-displayed automatically.

Show cut hardware in a new drawing section view

  1. In the draft document, on the File menu, click Settings→Options.

  2. In the Solid Edge Options dialog box, click the Drawing Standards tab.

  3. On the Drawing Standards page, under Cut Hardware In Section Views, select the desired drawing view generation option: Cut or Uncut.

Change hardware settings in a drawing section view

  1. In the draft document, right-click the section view and choose Properties.

  2. In the Properties dialog box, click the Display tab.

  3. On the Display page, under Selected Part(s) Display, set or clear the Cut Hardware check box.
