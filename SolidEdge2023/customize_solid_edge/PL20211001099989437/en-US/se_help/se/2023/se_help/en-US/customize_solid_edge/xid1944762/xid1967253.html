Apply filters in Dynamic Visualization

Watch the video demonstration and follow the steps below for applying filters.

  1. Open a Solid Edge assembly.

  2. From the Inspect tab→Report group, click Dynamic Visualization.

    Document properties are displayed in the property table.

  3. To display additional property columns, right-click in the property table, and then choose the Columns command.

  4. Select the check boxes associated with the properties you want to display in the table. (example: Material, Mfg Type) and then click OK.

    Entries shown inside curly brackets { } indicate system properties that can update when a document is opened. You can use the Update Status Info command to get the latest information about your document.

  5. Click the column header of the property information you want to filter.

    Unique values from the column are identified and displayed on the left side of the dialog box.

  6. Clear the check boxes that correspond to the property values you want to filter from the display.

    You can continue to refine the filtered information by combining multiple property filters. For example, you can filter on Content Type and Document Number to view all parts with a document number containing 002.

  7. To apply the filter, click .

  8. To continue filtering the property information displayed, repeat steps 3 through 7.

    The filter icon indicates that filters are applied to the column.

  9. After filtering the property information, you can create a rule using the filtered results.
