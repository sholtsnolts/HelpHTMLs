Inter-Part page (Solid Edge Options dialog box)

Defines which inter-part link options you want to use.

Update commands activate components with inter-part links (driven and driving)

Update commands activate components with inter-part links.

Allow Inter-Part Links Using:

Specifies that you want to use inter-part links when set. You can then select which of the inter-part link options you want to use.

Inter-Part Copy Command

Associatively copies surface geometry from one part file to another part file when working within the context of the assembly. The geometry you copy is associative to the parent part.

Include command in Part and Assembly 2D Sketches

Uses the Include command to associatively copy edges from a part in the assembly into the current profile or sketch.

Sketch relationships to Peer Geometry

Allows geometric relationships and driving dimensions between elements in the current profile or sketch and the edges of another part in the assembly when the Peer Edge Locate option on the Tools tab is also set.

Peer reference planes in Ordered Part features and 3D Sketches

Allows geometric relationships to peer reference planes in ordered part features and 3D sketches.

Note:

Press and hold the Shift key for this operation.

Assembly-driven part features

Constructs assembly-driven part features in an assembly. Assembly-driven part features add an associative feature to each part they cut. When you update an assembly-driven part feature in the assembly, the individual parts are also updated.

Paste Link to Variable Table

Pastes an assembly variable into the variable table of other parts and assemblies; and to link values from a spreadsheet document, such as Excel, to a design variable. When you link variables from an assembly or a spreadsheet you can control dimensions in multiple part documents simultaneously.

Assembly Relationships and Routing operations

Allows locating peer geometry to create assembly relationships and routing paths.

Note:

Press and hold the Shift key for this operation.

Peer reference planes in assembly features and 3D Sketches

Allows geometric relationships to peer reference planes in assembly features.

Note:

Press and hold the Shift key for this operation.
