Shape Search page (Solid Edge Options dialog box)

Specifies information required for using Geolus Shape Search with Solid Edge.

Java install path

Displays the installed location of JRE 8 or later as defined by the JAVA_HOME environment variable. This variable is set automatically when JRE is installed.

URL for Geolus server

Defines the location of the Geolus search server using the syntax <protocol>://<host>:<port>/geolus/

  • <protocol> is http or https

  • <host> is the name or IP address of the Geolus server

  • <port> is the port number

For more information, see Using Shape Search to locate models.
