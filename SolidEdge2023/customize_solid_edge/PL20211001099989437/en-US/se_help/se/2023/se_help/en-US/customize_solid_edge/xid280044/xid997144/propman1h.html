Edit document properties

You can edit properties with no documents open in Solid Edge or edit properties with an active document in Solid Edge.

Edit properties with no documents open in Solid Edge

  1. Choose File menu→Info tab→Property Manager.

  2. On the Select dialog box, select the document or folder that contains properties that you want to edit.

  3. To add the document or folder to the Edit Properties list, click Add.

  4. Click OK.

  5. On the Property Manager dialog box, edit a property value. The new value becomes underlined to indicate that you have modified the value.

  6. Click OK to save the changes to the document.

Edit properties with an active document in Solid Edge

  1. Choose File menu→Info tab→Property Manager.

  2. On the Property Manager dialog box, edit a property value. The new value is underlined to indicate that you have modified the value.

  3. Click OK.

  4. Save the document.
