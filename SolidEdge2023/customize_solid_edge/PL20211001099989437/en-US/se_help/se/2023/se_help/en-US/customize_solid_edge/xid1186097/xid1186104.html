Capture settings and preferences

You can capture settings and preferences using the Settings and Preferences Wizard or using the Solid Edge Options dialog box→User Profile page.

After you capture the settings and preferences, you can deploy them to other machines as explained in Deploy custom settings and preferences. You can also deploy settings and preferences programmatically, as explained in Manage settings and preferences programmatically.

  1. When Solid Edge is not running, choose Start→Siemens Solid Edge 2023→Settings and Preferences Wizard.

  2. Select Capture Settings and Preferences.

  3. Click Next.

    The Choose a location and filename box displays the default location for the settings and preferences file. You can save this file to any folder on your local machine or to a network location so you can access it from different machines.

    Solid Edge also displays this location in the Solid Edge Options dialog box→File Locations tab→File types list, in the Settings and Preferences File entry.

  4. Do one of the following:

    • To accept the default location:

      1. Select the types of settings and preferences you want to capture.

      2. Click Finish.

    • To modify the location:

      1. Click Browse and choose a new location.

      2. Select the types of settings and preferences you want to capture.

      3. Click Finish.

  5. To deploy the settings and preferences to other machines, continue with Deploy custom settings and preferences.
