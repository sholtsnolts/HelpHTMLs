Save page (Solid Edge Options dialog box)

Specifies the interval at which the system prompts you to save all open documents or create backup copies of unsaved documents and the location for the backup documents.

Automatically Preserve Documents By

Indicates that you want to preserve Solid Edge documents. When you select this option, you either choose to be prompted to save all open documents at a specific time interval or you can choose to create backup copies of modified, unsaved documents in the current Solid Edge session.

Prompt me to save all documents every <XX> minutes

Prompts you to save all open documents at the time internal you set. You can specify a time interval of 1 to 120 minutes. When this option is set and the time interval has expired, a dialog box is displayed that prompts you to save all open documents. This option does not automatically save the documents for you.

Creating Uniquely-Named Backup Copies of All Unsaved Documents

Automatically creates backup copies of all modified, unsaved Solid Edge documents in the current session at the time interval you set. You can specify a time interval of 1 to 120 minutes.

Note:

The backup file is removed from the backup folder when you save the file.

Backup Model Files

Specifies that you want to back up model files. This option is only available if you select the Creating Uniquely-Named Backup Copies of All Unsaved Documents option.

Backup Draft Files

Specifies that you want to back up draft files. This option is only available if you select the Creating Uniquely-Named Backup Copies of All Unsaved Documents option.

Prompt for File Properties on First Save

Specifies that the Properties dialog box is displayed on the initial save of a part, sheet metal file, assembly, and draft. You can define file properties when you save the document, which is helpful when defining criteria for part searches. You will not be prompted when you save tube parts or parts created in place in an assembly.
