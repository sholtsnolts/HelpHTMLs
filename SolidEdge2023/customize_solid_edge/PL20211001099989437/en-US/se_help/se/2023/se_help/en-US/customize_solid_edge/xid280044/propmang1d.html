Select dialog box

Allows you to select folders or files that contain properties that you want to edit.

BOM View For

Specifies that you want to edit properties based on a BOM view for the currently active document. The name of the active document is displayed in the control text string so you know which document you are editing. If you select this option and click the OK button, the active document is displayed in the Property Manager dialog box. This option is only available when you have a document opened.

If you use this option with a new file all of the fields on the Property Manager dialog box will be read-only.  Therefore, you must save a file before using this option in Solid Edge.

User View

Specifies that you want to edit properties based on a folder view. You can select the folders or files that contain properties you want to edit and click the Add button to add them to the Edit Properties box. This option is not available if you have a document open in Solid Edge.

Look In

Displays the drive and folder in which you want to search for the document whose properties you want to edit. The system stores the last location you browsed in the Look In box.

Note:

If you want to browse a network share, you must map the share to a network drive. After you map the network drive, you can select it from the Look In pulldown list.

Go to Last Folder Visited

Returns to the last folder that you visited.

Up One Folder

Moves the Look In folder up one level.

Views

Controls the display method for the listed documents.

Large Icons

Displays large icons for the documents.

Small Icons

Displays small icons for the documents.

List

Lists the names of the documents in columns.

Details

Displays a detailed view of the folder contents. The columns displayed include Name, Size, Type, and Modified.

Name

Lists the folders and documents you can choose to edit.

Add

Adds the selected folders and documents to the Edit Properties box.

Remove

Removes the selected folders and documents from the Edit Properties box.

Remove All

Removes all folders and documents from the Edit Properties box.

Edit Properties

Lists the folders and documents you want to edit.

Search

Displays the Search Dialog Box. You can use the search results to populate the Edit Properties box.
