Export a report in Dynamic Visualization

  1. Create at least one rule as described in Create a rule in Dynamic Visualization.

  2. In the Reports field of the Dynamic Visualization dialog box, type a name for your report, and then click Save .

    Reports are saved as part of the active assembly.

  3. To export the active report so that it can be shared, click Show More , and then click Export Report .

    The report (.dvrpt) is saved to the location your specify. There is one report per .dvrpt file.
