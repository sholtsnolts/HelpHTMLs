Manage settings and preferences programmatically

Solid Edge administrators can programmatically manage settings and preferences using the command line.

  1. At the command prompt, enter the path location for the Settings and Preferences Wizard. For example: C:\Program Files\Siemens\Solid Edge 2023\Program\SECopySettingsWizard.exe.

  2. Enter the arguments for the commands you want to run:

    Argument

    Use

    -s

    Capture settings.

    -c

    Deploy settings.

    -d

    Restore to factory defaults.

    -a

    Capture/deploy files from the AppData folder.

    -p

    Capture/deploy files from the Preferences folder.

    -t

    Capture/deploy template files.

    -r

    Capture/deploy registry settings.

  3. Enter the path to the SolidEdgeSettingsAndPreferences.sepref file. This default file is located in the My Documents folder.

    Depending what on you are trying to accomplish, your command line arguments might looks as follows:

    • To capture template files, as well as files from the AppData folder and save them to a shared location:
      C:\Program Files\Siemens\Solid Edge 2023\Program\SECopySettingsWizard.exe -s -t -a <shared_location>\MyPrefs.sepref

    • To deploy all the settings, that is AppData files, registry, preferences, and templates, from the sepref file to the machine:
      C:\Program Files\Siemens\Solid Edge 2023\Program\SECopySettingsWizard.exe -c -a -r -p -t \MyPrefs.sepref

    • To deploy just a few settings, use any of the -a -r -p -t arguments:
      C:\Program Files\Siemens\Solid Edge 2023\Program\SECopySettingsWizard.exe -s -a -p \MyPrefs.sepref

    • To restore Solid Edge to factory default:
      C:\Program Files\Siemens\Solid Edge 2023\Program\SECopySettingsWizard.exe –d

  4. Press Enter.
