Customize Quick Access Toolbar command

Displays the Customize dialog box, where you can customize the Quick Access toolbar and the keyboard to fit your workflow.

  • Use the Quick Access page (Customize dialog box) to add, remove, and reorder commands on the Quick Access toolbar.

  • Use the Keyboard page (Customize dialog box) to create custom keyboard shortcuts and change or delete default keyboard shortcuts.

  • Use the Ribbon page (Customize dialog box) to add, remove, and reorder commands on the command ribbon.

  • Use the Radial menu page (Customize dialog box) to add, remove, and reorder commands on the radial menu.

Note:

  • The changes you make are set per environment, so you can use different settings in different environments. This also means that when you change documents between Draft and Part, for example, you need to customize your settings in both places to keep the same options available.

  • Quick Access toolbar settings are stored in a file named <ThemeName>QATCustom.xml; this file resides in the application data folder of the operating system that Solid Edge is being loaded on.

    Example:

    If you create a theme named Basics, the Quick Access toolbar settings are stored in the file BasicsQATCustom.xml.

  • Keyboard shortcuts are stored in a file named <ThemeName>KBCustom.xml; this file resides in the application data folder of the operating system that Solid Edge is being loaded on.

    Example:

    If you create a theme named Basics, the keyboard settings are stored in the file BasicsKBCustom.xml.

  • Command ribbon settings are stored in a file named <ThemeName>RibbonCustom.xml; this file resides in the application data folder of the operating system that Solid Edge is being loaded on.

    Example:

    If you create a theme named Basics, the command ribbon settings are stored in the file BasicsRibbonCustom.xml.

  • This command performs the same function as the Customize command, which is available when you click the Quick Access Toolbar arrow .

  • Radial menu settings are stored in a file named <ThemeName>RadialCustom.xml; this file resides in the application data folder of the operating system that Solid Edge is being loaded on.

    Example:

    If you create a theme named Basics, the Radial menu settings are stored in the file BasicsRadialCustom.xml.

  • Multiple themes are supported such that a new path is created for each theme saved.

    Example:

    If you have customized Radial menu settings for a second theme named Advanced, these are stored in AdvancedRadialCustom.xml.

  • The ST3 customization files RibbonCustomization.xml and RadialCustomization.xml are decomposed into the number of themes present within ST3 and populating the folder structure under Customization.

  • Prior quick access toolbar customization is brought forward; the registry is read and the custom information is assigned a theme called My QAT Custom. This theme contains the defaults for the keyboard, command ribbon, and radial menu.
