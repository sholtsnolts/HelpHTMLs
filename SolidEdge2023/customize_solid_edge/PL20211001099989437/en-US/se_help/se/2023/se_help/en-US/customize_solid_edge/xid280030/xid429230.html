Assembly Open As page (Solid Edge Options dialog box, Assembly environment)

The Assembly Open As options define the criteria for selecting how an assembly opens based on the number of unique components.

You can define Small, Medium, and Large assemblies using your own criteria or the default values defined for you. Small assemblies are defined by being less than the amount entered. Large Assemblies are defined by being more than the amount entered. Medium Assemblies are defined by a range between the small and large values.

Note:

The number of unique components is defined as the number of unique parts plus unique subassemblies.

Assembly Open As

Settings are based on small, medium and large assemblies and can be set independently for each by the user.

Auto-Select

  • Determines which parameters to use based on the number of unique components.

Small Assembly

  • An assembly with less than the defined number of unique components is considered a small assembly.

Medium Assembly

  • An assembly with a number of unique components greater than a small assembly and less than a large assembly is considered a medium assembly.

Large Assembly

  • An assembly with more than the defined number of unique components is considered a large assembly.

Last Saved

  • Opens an assembly with the settings used when the assembly was last saved.

Auto-Select Criteria

Determines the number of unique components for each category of assembly: small, medium, large.

Open As Settings

Defines how an assembly is opened based on the settings defined in each of the following categories. Each setting can be set independently.

Hide all components

States whether all components are hidden using Yes or No.

Part Activation

Determines what parts are activated: Activate all, Inactivate all, Last Saved.

Part simplification
Use all designed

Uses the design body to create the JT document.

Use all simplified

Uses the simplified body to create the JT document.

Last saved

Uses the last saved configuration to create the JT file.

Assembly simplification
Use all designed

Uses the design body to create the JT document.

Use all simplified

Uses the simplified body to create the JT document.

Last saved

Uses the last saved configuration to create the JT file.

Activate changed parts on file open

Choose from Prompt or Automatic.

Open Assembly Preview for large assemblies

If checked, opens the assembly in Preview mode if the assembly is determined to be a large assembly. If unchecked, opens the assembly directly in the assembly environment.

Note:

This option is not supported when the Teamcenter integration (SEEC) is enabled.

Use Large Assembly Performance Mode when the total number of components in the assembly exceeds:

Setting for when to enter Large Assembly Performance Mode. On by default with a value of 5000.

Reset

Returns the settings to the default values.
