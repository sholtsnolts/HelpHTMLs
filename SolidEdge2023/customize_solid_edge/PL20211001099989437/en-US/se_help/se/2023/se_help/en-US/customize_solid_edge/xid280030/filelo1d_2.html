File Locations page (Solid Edge Options dialog box)

Specifies the default location for documents, templates, and other objects you create or use in Solid Edge.

Note:

Not all options are available in all environments.

Update

Updates any changes you make. The changes are saved to Options.xml. You must click Update any time you make changes to Options.xml.

Modify

Changes the location of a folder or file. If you select a folder and click the Modify button, the Browse for Folder dialog box is displayed. If you select a file and click the Modify button, the Browse for File dialog box is displayed.

Reset

Sets the selected file location path to the default setting.

Reset All

Sets all selected file location paths to the default setting.

File Types

Indicates the file type for which you want to specify a location.

Location

Displays the designated location for each file type. File types are grouped in the categories of General, Modeling & Draft, Managed, and Output.

Solid Edge Admin

Specifies the folder and name of the options configuration file which is used by the Solid Edge Administrator to define corporate policies.

Custom Setting File

Specifies the location of the custom hole settings files. You can save a custom hole setting to an external file for global use.

Link Management File

Defines the location of the ASCII text file (LinkMgmt.txt) that contains folder information and a search algorithm that defines the folders to be searched when resolving links. The search paths are processed in no particular order, so you should not assume any order when adding the search paths to the list.

Macros

Defines the location for macros that contain a sequence of actions or commands that can be named and stored. When you run the macro, the software performs the actions or runs the commands.

Property Seed File

Defines the location of the ASCII text file (Propseed.txt) used to populate values on the Properties dialog box.

Reports

Defines the folder for all reports, including draftholes.txt and draftlist.txt. When you double-click Reports, you can browse to select an unmanaged local or network share for the reports folder.

Note:

You must have write access to the folder you define.

Start up screen

Specifies the location of the Solid Edge startup screen.

User templates

Contain a default set of properties that defines what a new document will look like. You can use a template to set standards for a group of users or to define your own preferences.

Material Table

Specifies the location of material.mtl, which is used to populate the property set for each material on the Solid Edge Material Table dialog box.

Standard Parts configuration file

Specifies the location of configuration file. This option is only available in the Assembly environment and when working in XpresRoute.

Virtual Component Names File

Specifies the location of SEVCNames.txt that defines the default names for each file type. This option is only available in the Assembly environment.

Frame Local Library Folder

Specifies the location of the folder for frame Components. This option is only available in the Assembly environment.

Hole Size File

Defines the location for the ASCII text file (Holes.txt) that is used to populate values on the Hole Options dialog box.

Pipe Threads File

Specifies the location of the ASCII text file (Pipe Threads.txt) that is used to populate values on the Thread Options dialog box.

ANSI Inch Tolerance table

Specifies the location of the ANSI inch tolerance table SE-LimitsAndFitsTableANSIinch.txt. This is an ASCII text file that you can use to define the tolerance values for a Class Fit dimension.

For more information, see Class fit dimensions.

ANSI Metric Tolerance table

Specifies the location of the ANSI metric tolerance table SE-LimitsAndFitsTableANSIMetric.txt. This is an ASCII text file that you can use to define the tolerance values for a Class Fit dimension.

For more information, see Class fit dimensions.

ISO Tolerance table

Specifies the location of the ISO metric tolerance table SE-LimitsAndFitsTableISO.txt. This is an ASCII text file that you can use to define the tolerance values for a Class Fit dimension.

For more information, see Class fit dimensions.

Symbol Templates

Specifies the location of the symbol templates delivered with Solid Edge, as well as templates you create. This option is only available in the Draft environment.

SEEC Cache

Specifies the folder for the local cache which is used with Solid Edge Embedded Client.

Backup Folder Location

Specifies the folder for the backup files.

Log Files

Specifies the folder containing your log files.

Pipe Classes File

Specifies the location of the pipe classes file that contains information about the pipe fitting components. This option is only available when working in XpresRoute.

Wire Harness Configuration File

Specifies the location of the SEHarness.txt file, which defines the data format of how ECAD net list files are organized. This option is only available when working in Harness Design.

Wire Harness Conductor File

Specifies the location of the SEConductors.txt file, which contains information about the wires, cables, and bundles used in wire harness design. This option is only available when working in Harness Design.

NX Nastran Scratch Folder

Specifies the folder where the NX Nastran files will be written during solution processing. The contents of this folder may get deleted, so you should not manually add files or folders to this location.

The default scratch folder location is %Temp%\SESimulation, for example, C:\Temp\SESimulation. If you choose a new location, the contents of the previous scratch folder will be moved automatically to the new folder.

Note:

This folder location is referenced by the NX Nastran SDIR keyword. This value is not added to the study definition, because it is unique to each installation.

3D Electrical Component file

Specifies the location of SE-PreferredElectricalComponents.csv that contains the list of favorite component names. By default, this location is $:\Program Files\Siemens\Solid Edge 2023 \Preferences\SE-PreferredElectricalComponents.csv, but you can modify the path. When CMP/CON files are imported, the CMP file is directed to the preferences file first. If the component name in the CMP file matches with component name in the favorite list the relevant path is displayed in the harness wizard. To populate the part in the active assembly, click Populate in the harness wizard.

After the harness wizard operation completes, the component name and path for components which are manually browsed are written back into the csv file.

Note:

If the component is not found in the favorites list, Solid Edge searches the electrical parts folder. If the component is not found there, you must browse manually for the component.

3D Electrical Part Folder

Specifies the location where the 3D electrical components are saved. By default, this location is $:\Program Files\Siemens\Solid Edge 2023 \Training\Wiring Harness, but you can modify the path. If you specify a different folder location, that location must contain the 3D electrical components.

3Dfind.it Components

Defines the default location for downloading parts from the 3Dfind.it website. When the file location is undefined, you are prompted to browse for a folder to save the downloaded parts.
