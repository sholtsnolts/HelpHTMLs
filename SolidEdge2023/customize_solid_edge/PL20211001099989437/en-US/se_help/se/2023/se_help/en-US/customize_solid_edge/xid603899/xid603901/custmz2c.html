Show Quick Access Toolbar Below The Ribbon command

Moves the Quick Access toolbar so that it is positioned directly below the horizontal command ribbon. You might want to use this option when you have added an extensive list of commands to the Quick Access toolbar.

Note:

This command is available only when the Quick Access toolbar is in its default location at top-left of the application window.

Right-click on the Quick Access Toolbar, on a command icon, or on a command group and select Show Quick Access Toolbar Below The Ribbon.
