Deploy custom settings and preferences

You can deploy settings and preferences using the Settings and Preferences Wizard or using the Solid Edge Options dialog box→User Profile page. You can also deploy settings and preferences programmatically, as explained in Manage settings and preferences programmatically.

  1. When Solid Edge is not running, choose Start→Siemens Solid Edge 2023→Settings and Preferences Wizard.

  2. Select Deploy Settings and Preferences.

  3. Click Next.

    The Choose a location and filename box displays the default location for the settings and preferences file.

    Solid Edge displays this location in the Solid Edge Options dialog box→File Locations tab→File types list, in the Settings and Preferences File entry.

  4. (Optional) To create a backup of the system's settings, click Create a backup of the current settings for <username>.

    The wizard saves the backup file in the same location you specified in the Choose a location and filename box.

  5. Do one of the following:

    • To accept the default location:

      1. Select the types of settings and preferences you want to deploy.

      2. Click Finish.

    • To modify the location:

      1. Click Browse and choose a new location.

      2. Select the types of settings and preferences you want to deploy.

      3. Click Finish.
