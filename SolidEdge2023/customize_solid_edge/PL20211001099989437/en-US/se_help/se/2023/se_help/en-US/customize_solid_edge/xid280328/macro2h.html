Add a macro to the command ribbon

You can create macros with any programming tool that supports OLE automation, such as Microsoft Visual Basic, and then add them to the command ribbon.

Add a .exe or .dll as a macro

  1. Open the Customize dialog box using one of these methods:

    • Right-click a command on the ribbon and choose Customize the Ribbon.

    • Click the Customize arrow , and from the menu, choose Customize the Ribbon.

  2. From the Environment to customize list select the relevant environment.

  3. From the Choose commands from list, select Macros.

  4. Click the Ribbon tab.

    Note:

    The Ribbon tab displays a list of macros found in the current folder location. You can change the folder location using the Browse button at the bottom of the dialog box.

  5. From the list on the left, select a macro.

  6. From the list on the right, select the tab where you want to add the macro, expand the tab, and then select a group name.

  7. Click Add.

    Solid Edge adds the macro and its associated image icon to the command list.

    Note:

    If the macro contains an image icon, Solid Edge uses that image. If the macro does not contain an image, Solid Edge assigns an image to it. For more information, see Macros.

  8. Right-click the newly added macro and choose Button Options.

  9. In the Button Options dialog box, set the size and button text options that you want.

Tip:

You also can create a new tab to contain your macros. To create a new tab:

  1. Click an existing tab to identify a location in the tab order, and then click the New Tab button.

  2. Right-click the New Tab entry, click Rename, and then type a name, such as Macros.

  3. (Optional) To change the order of the tabs using the tabs, use the Move Up and Move Down buttons.

  4. (Optional) To add a horizontal divider within a group, use the Separator button.

Add a website as a macro

  1. From the list of macros on the left, select the option, By Keyin (user provided name and parameter).

  2. From the list on the right, select the tab where you want to add the macro, expand the tab, and then select a group name.

  3. Click Add.

  4. In the Define Macro dialog box, do the following:

    1. In the Macro box, type the web site domain name, such as www.google.com.

    2. Verify that the Parameter box is blank.

    3. Click OK.

      Solid Edge adds the website domain and the associatedfaviconare added to the command list.

  5. If the Custom Button dialog box is displayed, select one of the default image buttons, or click Browse to locate a different button. Click Assign.
