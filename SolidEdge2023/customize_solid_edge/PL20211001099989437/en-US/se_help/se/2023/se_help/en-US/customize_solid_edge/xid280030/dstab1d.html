Dimension Style page (Solid Edge Options dialog box)

The Dimension Style tab on the Options dialog box displays the element-to-style mapping table, which sets default styles for dimensions, annotations, connectors and symbols. The styles listed are derived from the current document and any template file.

Active dimension style

Sets active a dimension style selected by the user from the drop-down list, which controls how dimensions are displayed upon placement. Once a dimension style is active, any future dimensions placed either on a model or drawing will take on the active dimensions style.

Use Dimension Style Mapping

Specifies that you want to use the element-to-style mapping table to determine the format of dimensions, annotations, connectors, and symbols as you place objects that use these styles.

When the Use Dimension Style Mapping option is set, the Dimension Style Mapping button on the applicable command bars is also set.

When this option is cleared, a style must be selected each time an object is placed. The selected style becomes the active style until it is changed.

Set All Styles To

Specifies a single style to be set for all elements in the element-to-style mapping table. For example, you can set the style that all elements will use to ISO or ANSI.

Note:

To set a different style for one element type, click the Style column for the corresponding element and select a style from the list.

Apply

Sets all elements to the style specified in the Set All Styles To drop-down list.

Element

This column lists the elements for which you can set a default style in the Style column.

Style

Specifies the style for the corresponding element in the Element column. To change the default style for a particular element type, click the corresponding row in the Style column, and then select a new style from the list.

Note:

To set all the elements listed in the Element column to use the same style, use the Set All Styles To option.
