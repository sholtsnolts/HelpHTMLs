Display missing required property data

  1. Open a managed Solid Edge document.

  2. On the File menu, click Info group, click File Properties.

  3. In the Filters group, click Required Property , and then click Properties with no value .

    Values that display 0 of x indicate there are no properties that meet the filter definition.

Tip:

For information on designating required properties in Solid Edge Data Management (SEDM), see Working with registered custom properties.
