Add a favorite material

Use the Add to Favorites command to add a material to the Favorite Materials list on the Favorite and Recent Materials tab in the Material Table dialog box.

  • In the left pane of the Material Table dialog box, right-click a material and choose the Add to Favorites command.

Tip:

  • You can access the Favorite Materials list and the Recent Materials list using the shortcut menu of the Material PathFinder node in the model.

  • You can change the favorites order by selecting a material in the favorites list and then clicking Move Up or Move Down.

  • You can remove a material from the list by selecting the material and then clicking Remove.

  • You can control the number of entries in the Recent list by changing the value in the Show this number of recent materials box. You can choose a value in the range from 0–10.

  • You can delete the Recent list by clicking the Clear button.
