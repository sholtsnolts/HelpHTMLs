Create custom document properties

  1. Open a Solid Edge document.

    Tip:

    In a Teamcenter-managed environment, the document must be checked out of Teamcenter to create custom properties.

  2. On the File menu, click Info→File Properties.

  3. Do one of the following:

    • In the command ribbon, select Custom group→Add.

    • Right-click the Custom section of the table, and on the shortcut menu, click Add.

  4. In the each of the Name, Value, and Type columns, choose a value for your custom property.

    The newly added custom property is displayed with an underscore.

    Note:

    • If you do not want the name to appear in other applications, such as Microsoft Exchange, place an underscore ( _ ) in front of the name. For example, _SE_ASSEMBLY_QUANTITY_OVERRIDE.

    • The value must match the type. For example, if you select Number for the Type, you must specify a number for the Value. Values that do not match the type are stored as text.

  5. Click Apply.
