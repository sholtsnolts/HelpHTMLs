Attach Properties command bar

Command bar Options
Property Values

Accesses the Property Values dialog box .

Property Table

Displays the name of the property table.

Select Step

Specifies the elements you want to attach the property to.

Cancel/Finish

Cancels the command or attaches the property to the selected element.

Select

Sets the element selection method for element to which the property will be attached.

  • Face/Edge/Vertex—Selects an individual face, edge, or vertex.

  • Face Chain—Selects a chain of faces.

  • Feature Faces—Selects all the faces in a feature by clicking the feature.

  • Edge Chain—Selects a chain of edges.

  • Feature Edges—Selects all the edges in a feature by clicking the feature.

Deselect (x)

Clears the selection.

Accept (check mark)

Accepts the selection.

Name

Displays the feature name. Feature names are assigned automatically. You can edit the name by typing a new name in the box on the command bar or by selecting the feature and using the Rename Feature command on the shortcut menu.
