File Properties dialog box (SEEC)

Displays information about the properties of Teamcenter-managed documents. The dialog box is interactive and you can change the property values of the selected document, or documents within a select set using the commands in the ribbon. If no document is selected, the command operates on the direct document. Filters can be used in combination to assist you in displaying only the infomation you need. Asterisks (*) indicate required properties.

Right-clicking a cell in the grid displays a shortcut menu to change the font, sort the information, find and replace text, and change column display.

Restore

Returns the properties to the values shown when you opened the dialog box.

Print

Sends a copy of the data to a specified plotter, printer, or file. Options are available for defining the range, number of copies, and other printing characteristics.

Help

Accesses Solid Edge help.

Collapse

Shows or hides the command ribbon.

Command ribbon

Paste

Inserts the contents of the Clipboard at the specified location. The shortcut command is Ctrl + V.

Cut

Removes selected elements from the table and places them on the Clipboard. The selected elements replace the previous contents of the Clipboard. The shortcut command is Ctrl + X.

Copy

Copies selected elements from the table and places them on the Clipboard. This command replaces the previous contents of the Clipboard with the new contents. The shortcut command is Ctrl + C.

Find

Displays the Replace dialog box so that you can specify text to locate in the table.

Required

Displays only properties that are defined as required properties.

Editable

Displays only properties that you can modify.

Empty

Displays only the properties that have no value defined.

Mapped/Unmapped Properties

Shows only the properties that are mapped. Clicking again shows properties that are not mapped.

Text

Displays only the properties that contain the text you provide. The text filter acts on the entire table, or only on cells you highlight.

Does not contain

Displays only the properties that do not contain the text you provide.

Show/Hide Columns

Displays or hides columns within the table. This command is only active if a column is hidden using the Columns command. Property and Value columns cannot be hidden.

Show/Hide Rows

Displays or hides rows within the table. This command is only active if a row is hidden using the Columns command in common property dialog boxes.

Clear Filters

Removes any filters applied in the Filter group of the dialog box.

Check Out

Reserves access to the document in the database so that only you can modify it.

Add

Adds custom properties to the document.

Remove

Removes the selected custom properties from the document.

Settings

Provides access to display controls.

Columns

Displays the Format Columns dialog box so that you can determine the display of columns in the table.

Preview

Displays a Preview pane containing an image of the active document. The pane contains the saved view of the document along with options for displaying an interactive view, storing the image in the file, and updating the image when the document is saved.

Auto-Fit

Performs a Best Fit to resize each column based on the longest string in each column.

Apply

Saves the property changes to the file. Changes are saved to Teamcenter when you check the object in.

Close

Closes the dialog box.

File properties table

Property

Displays properties of the active object. Teamcenter properties display only after the document is saved to Teamcenter.

Value

Displays the value for the identified property. Asterisks indicate the value is required.

Type

Identifies the type of property data for the identified property - string, date, alpha, numeric, and so on.

Mapping

Displays the name of the property to which the property is mapped.

Owner

Indicates the owning application - CAD, Teamcenter, or Both.
