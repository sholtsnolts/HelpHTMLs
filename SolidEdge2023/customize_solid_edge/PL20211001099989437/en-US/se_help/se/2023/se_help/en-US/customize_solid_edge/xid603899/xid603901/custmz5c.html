Add To Quick Access Toolbar command

Adds a single command or a command group to the Quick Access toolbar.

  • Right-click a command icon and select Add to Quick Access Toolbar; the command icon is added to it.

  • Right-click a command group and select Add to Quick Access Toolbar; the group is added to it.
