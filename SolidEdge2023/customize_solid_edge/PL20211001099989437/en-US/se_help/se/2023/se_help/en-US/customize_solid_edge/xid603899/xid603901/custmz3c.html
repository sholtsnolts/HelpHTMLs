Show Quick Access Toolbar Above The Ribbon command

Restores the Quick Access toolbar to its default location at top-left of the application window.

Note:

This command is available only when the Quick Access toolbar has been moved below the command ribbon using the Show Quick Access Toolbar Below The Ribbon command.

Right-click on the Quick Access Toolbar, on a command icon, or on a command group and select Show Quick Access Toolbar Above The Ribbon.
