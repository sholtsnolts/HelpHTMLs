Create a custom material property

The Material Properties tab in the Material Table shows the default properties defined for each material. Default properties are shown in the gray shaded cells in the Property Name column. You can modify a material to add custom material properties.

  1. Do one of the following:

    • In PathFinder, right-click the Material node and choose Material Table.

    • From the File menu, choose Info→Material Table.

  2. In the left pane of the Material Table dialog box, browse to and click the material that you want to modify.

    Note that there may be different material nodes based on different standards, such as Materials-DIN and Materials-GOST.

  3. On the Material Properties tab, click Add Custom Property .

    The Add Custom Property command is also available from the shortcut menu of the selected material.

  4. In the Add/Edit Custom Property dialog box, in the Name box, type a name for the custom material property.

    Example:

    Name=Hardness

    Name=Melting Point

  5. From the Unit type list, select the units to apply to the material property. For a unit-less value or if a unit type is not shown in the list, set Unit type=Text.

    Setting a unit type displays a default value or a zero value in the Value box. This is based on the selected material and on the units in the document template.

  6. In the Value box, type a value for the property. If you assigned a unit type, take care not to type over the units.

    Example:

    Custom material property

    Unit type

    Value

    Hardness

    Text

    8.5 Mohs

    Melting Point

    Temperature

    1371.00 C

  7. To close the Add/Edit Custom Property dialog box and add the custom material property, click OK.

    The custom material property and its value are shown in blue text at the bottom of the Properties list.

  8. Do one of the following:

    • To apply the material property to an open document with an existing model, click Apply to Model.

    • Click Save.

Tip:

You can reference custom material properties on a drawing, in a part or assembly sketch, and when adding PMI in a model. For example, you can reference the Hardness and Melting Point properties in a callout or other annotation. You also can show custom material properties for individual parts as a data column in a parts list.

To reference custom material properties:

  • In annotations, use the Select Property Text dialog box and set the source of the property text to either From Graphic Connection to Part or From Graphic Connection (for an assembly). For more information, see the help topic, Create property text.

  • In a parts list, use the Columns tab in the Parts List Properties dialog box. For more information, see the help topics, Create a column for custom property content and Example—Show custom properties and materials in a parts list.

  • Anywhere you can type format codes to modify reference and property text output—The mechanism for referencing a custom material property is the /CM property text code. You can add this as a modifier for the custom material property that you want to extract, using the syntax %{CustomMaterialName/CM}.
