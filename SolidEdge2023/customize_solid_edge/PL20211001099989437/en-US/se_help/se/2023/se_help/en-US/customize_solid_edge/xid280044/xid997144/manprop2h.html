Modify the property display list

You can use the Show Properties dialog box to manage the properties that you want to display on the Property Manager dialog box.

To add properties to the display list

  1. On the Show Properties dialog box, from the Select Available Properties From pulldown menu, select the property set that contains the properties you want to add to the display list.

  2. In the Available Properties list, select the property you want to add to the list.

  3. Click the Add button to add the property to the Show These Properties in This Order List.

  4. On the Show Properties dialog box, click the OK button.

To remove properties from the display list

  1. In the Show These Properties in this Order list, select the property you want to delete from the list.

  2. Click the Remove button to delete the property from the Show These Properties in This Order List.

  3. On the Show Properties dialog box, click the OK button.

To change the order of the properties in the display list

  1. In the Show These Properties in this Order list, select the property you want to move in the list.

  2. Click the Move Up button or the Move Down to move the property the Show These Properties in This Order List. The Move Up button causes the property to appear further to the left in the list on the Property Manager dialog box and the Move Down button causes the property to appear further to the right.

  3. On the Show Properties dialog box, click the OK button.

To add a new property to the display list

  1. On the Show Properties dialog box, click the New button.

  2. On the New Property dialog box, in the Name box, type a name for the property.

  3. Use the Type pulldown menu to select the type for the property.

  4. On the New Property dialog box, click the OK button.

  5. On the Show Properties dialog box, click the OK button.
