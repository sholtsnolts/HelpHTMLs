Dynamic Visualization dialog box

Provides an interactive display of document property information used for filtering and creating rules to view property data in the model.

Commands

Icon

Command

Description

Reports

Displays a list of saved reports from which you can chose a report to be the active report.

Save Report

Retains the filters defined to create the rules that make up the report. You can use this command to save an existing report to a new name.

Show More

Display additional commands.

Help

Apply Override

Applies temporary color overrides to the model as defined by the rules you create.

Remove Override

Removes the temporary color overrides.

Refresh property data

Rereads property values and displays the results in the Up-to-Date column.

Update Status Info

Obtains the latest information about your document.

Import Report

Loads an exported report (.dvrpt) and makes it the active report.

Export Report

Saves a report out of an assembly so that it can be shared. There is one report per .dvrpt file.

Create New Report

Resets property information and starts the creation of a new report.

Delete Report

Removes the active report from the list of reports available for the active assembly.

Show Less

Display fewer commands.

Show/Hide Table

Enables or disables the display of the property table.

Select by Rule

Selects components of the assembly based on filters in the active rule.

Deactivate

Disables or enables the selected rule.

Show More

Displays additional commands.

Change Color

Specifies the color override used for the selected rule.

Reorder Rule

Changes the position of an existing rule.

Delete Rule

Removes the rule from the list of available rules.

Rename Rule

Changes the name of the active rule.

File properties table

Document Name or Object

Displays the object name of the documents displayed in the property table.

Up-to-Date

Indicates whether property values are up to date with the latest version of the document.
