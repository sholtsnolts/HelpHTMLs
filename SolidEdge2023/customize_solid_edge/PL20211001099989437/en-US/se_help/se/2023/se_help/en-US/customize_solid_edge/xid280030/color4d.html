Colors tab (Solid Edge Options dialog box, Draft)

Controls color settings for the active document. These tab options are available in the Solid Edge Draft environment.

Drawing Display

Defines a custom color scheme for the 2D Model sheet and for all other types of sheets in the draft document.

2D Model

Applies the Color Scheme selections to the 2D Model sheet.

Sheet

Applies the Color Scheme selections to the working sheets, background sheets, table sheets, and all drawing areas except the 2D Model sheet.

Color Scheme

Controls the overall color scheme for the active document.

The Preview pane shows you the result of your color scheme changes before you apply them to the sheet.

Example:

AutoCAD Model

Sets the sheet color to black, and it also sets default highlight, element selected, element disabled, and handle colors. As you draw, lines are displayed in white on the black background.

Solid Edge Default

Sets the sheet color to white, and sets the highlight, element selected, element disabled, and handles to their default colors.

Custom

Displayed automatically when you make a change to a predefined color scheme.

Sheet

Sets the default color of the active drawing sheet.

Highlight

Sets the highlight color.

Selected Element

Sets the color of selected elements.

Disabled Element

Sets the color of such elements as alignment indicators and of the empty callout symbol.

Handles

Sets the colors of different types of selection and editing handles.

Use shading on 2D Selection fence

Specifies that a shaded region is displayed when you drag a directional fence around 2D elements to select them.

This option applies in draft and in ordered sketches when the Select Tool is set to Directional Fence.

Opacity

Specifies the color transparency of the shaded region when you drag a directional fence. The lower the opacity, the easier it is to see the elements that are inside the fence.

Inside

Specifies the color of the shaded region when you drag a directional fence from left to right.

Inside/Outside and overlapping

Specifies the color of the shaded region when you drag a directional fence from right to left.

Sheet Tab Colors

Sheet tab colors are independent of the Color Scheme colors.

Sheet tab 1

Specifies the sheet tab color of the 2D Model tab, the background sheet tabs, and of alternating groups of working sheet tabs.

Sheet tabs are organized into alternating color groups in the sheet tab tray. You can use this color differentiation to help you locate a specific sheet or a group of sheets.

Sheet tab 2

Specifies the sheet tab color of the primary working sheet tabs, and for alternating groups of working sheet tabs.
