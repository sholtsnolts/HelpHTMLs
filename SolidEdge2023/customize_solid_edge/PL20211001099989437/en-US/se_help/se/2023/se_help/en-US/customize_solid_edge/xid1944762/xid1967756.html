Create a report in Dynamic Visualization

  1. Open a Solid Edge assembly.

  2. From the Inspect tab→Report group, click Dynamic Visualization.

    Document properties for the open assembly are displayed in the property table. You can display additional property columns by right-clicking in the table and choosing the Columns command.

    Entries shown inside curly brackets { } indicate system properties that can update when a document is opened. You can use the Update Status Info command to get the latest information about your document.

  3. Filter the property information as described in Apply filters in Dynamic Visualization.

    You must apply at least one filter before you can create a rule.

  4. Create at least one rule as described in Create a rule in Dynamic Visualization.

  5. Click the Reports field and type a name for your report.

  6. Click Save Report .

    The named report is displayed in the list of available reports.

Note:

Changing the name of an active report copies the active report to a new report with the new name.
