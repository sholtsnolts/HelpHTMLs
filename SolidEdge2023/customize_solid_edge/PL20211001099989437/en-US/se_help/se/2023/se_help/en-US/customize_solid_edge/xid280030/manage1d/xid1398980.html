Assembly tab, Manage page (Solid Edge Options dialog box)

The Assembly tab is a member of the tab set available in the Manage page of the Solid Edge Options dialog box. Use options on this page to specify how Teamcenter-managed assembly documents are structured.

When saving new managed Assembly documents
Product Structure - BOM Precision

Indicates how Solid Edge Assembly documents (created using the New command) generate BOM view revisions. The value displayed is based on the definition of the preference SEEC_BOM_Precision_Override. The preference value is read once per session. To update to the latest preference value, restart your session. You must be logged into Teamcenter for the value to be displayed.

Assembly structure synchronization
Check-out candidate assemblies

Checks out from the managed location assemblies that are candidates for BOM synchronization.

Show structure synchronization dialog

Determines whether the Structure Synchronization dialog box is displayed when BOM synchronization detects candidate assemblies.

Enable structure synchronization logging

When checked, structure synchronization logging is on and a log file is generated. The location of the log file is determined by the Log Files option in the File Locations page of the Solid Edge Options dialog box.

Alternate Assemblies - Alternate position of the assemblies
Synchronize alternate position assemblies as Arrangements

When checked, each Solid Edge alternate assembly member’s positional data is saved as an arrangement in Teamcenter using the member name. As a result, what you see in Solid Edge is what you see in Teamcenter Rich Client.

Alternate Assemblies - Family of Assemblies
When creating a new draft of a Family of Assembly Member
Default to saving the new draft to the Revision of the Source

When a draft is saved, the Item ID of the FOA Source is populated, and the saved draft is linked to the FOA Source. This is the preferred option.

Default to saving the new draft to the Revision of the published Member

When a draft is saved, the Item ID of the FOA Member is populated, and the saved draft is linked to the FOA Member.

Note:

Save As will not copy a draft saved to the revision of the published member. To copy the draft using Save As, save the draft to the revision of the source, or to a separate item.

Hosted Active Workspace cross highlighting
When an object is selected in Active Workspace, automatically select the content in Solid Edge

Enables automatic highlighting of the corresponding object in Solid Edge.

Note:

Selecting this option disables the Select in Solid Edge command in the Teamcenter tab of the ribbon.

Automatically expand PathFinder and scroll to the first object in the select set

When checked, PathFinder automically expands to display the first object in the selected set.

When an object is selected in Solid Edge, automatically select the object in Active Workspace

Enables automatic highlighting of the corresponding object in Active Workspace.

Note:

Selecting this option disables the Select in Active Workspace command in the Teamcenter tab of the ribbon.
