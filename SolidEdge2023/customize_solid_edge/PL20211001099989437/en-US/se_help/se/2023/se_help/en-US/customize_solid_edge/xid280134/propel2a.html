Object properties

All elements and objects in a Solid Edge document possess properties. These properties are derived initially from default or custom global styles, which can be applied to newly added elements and objects by pre-defining them in templates and by using the dimension style mapping command, where available.

Once placed, the properties for individual elements and objects can be edited. To edit individual properties, use the Properties command on the selected item's shortcut menu.

The specific types of properties vary with the item selected. For example, 2D drawing elements, such as points, lines, arcs, and circles, possess properties of color, line width, and line type, as well as other attributes particular to type of element.
