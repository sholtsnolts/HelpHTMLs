Quick Access page (Customize dialog box)

Note:

Not all options are available in all environments.

Adds, removes, and organizes commands from the Quick Access toolbar.

Environment to customize

Specifies the environment to which you want to add a custom keyboard shortcut. Select the environment first.

Choose commands from

Specifies the category that you want to customize. Select the category second.

To add entries from the File menu to the Quick Access toolbar, set Choose commands from:=System Menu.

Theme

Specifies the active user interface theme. To learn more about themes, see Solid Edge themes

Command category and command list

The bottom-left list specifies which command from the selected category you want to add to the Quick Access toolbar. Select the command after you select the category.

Save

Saves changes to the active theme.

Save As

Saves changes to a new theme which you name.

Delete

Deletes the active theme.

Note:

You cannot delete the Balanced - Solid Edge Default theme.

Add

  • Adds the selected command to the list on the right, which shows the commands available on the Quick Access toolbar.

  • For macros, you can hold down the control key while clicking the Add button and Solid Edge displays a palette of images you can define as the icon.

Remove

Removes the the Quick Access toolbar command selected in the list on the right.

Separator

Adds a vertical graphical separator between commands on the Quick Access toolbar. You can use separators to organize commands by group.

Note:

  • Separators are inserted after the selected command.

  • You cannot insert a separator as the first or last item in the list, but you can move it using the Move Up and Move Down buttons.

  • You can delete a separator using the Remove button.

Spacer

Adds a space between commands on the Quick Access toolbar.

Move Up

Moves the command selected in the list on the right up one row.

Move Down

Moves the command selected in the list on the right down one row.

Reset

Resets the Quick Access toolbar to the default set of commands.
