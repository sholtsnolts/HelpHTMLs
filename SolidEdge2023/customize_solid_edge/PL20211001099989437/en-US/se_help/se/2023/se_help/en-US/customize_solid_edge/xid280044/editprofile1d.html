Edit Profile dialog box

Displays each document being uploaded as a root node in alphabetical order, making it easier to see which documents are being uploaded into the managed library. You can set the property value for the individual documents being uploaded. You can right-click on an occupied row in the grid to display a shortcut menu to change the font, sort the properties, find and replace text, and format columns. You can double-click near a column divider to do a best fit on the grid. Any changes you make to the columns are stored for future sessions.

Files of Type

Specifies the type of files that are listed in the File Name column. The properties that are displayed are based on the files listed in the File Name column. The dialog box displays the profile properties for each file being checked in. You cannot add additional properties while you are in edit profile mode.

Save

Saves changes you have made to properties. This option is disabled if you use the Property Manager command in Solid Edge. To save your changes, use the Save command on the File menu. This control is disabled in edit profile mode.

Print

Sends a copy of the active document to a specified plotter, printer, or file. Options are available for defining the printing area, range, number of copies, and other printing characteristics.

Before using this command, you must install and select a printer.

Cut

Cuts the current selection and pastes it to the Clipboard.

Copy

Copies the current selection to the Clipboard.

Paste

Inserts the Clipboard contents at the specified insertion point.

Restore

Sets all property cells back to the previous values.

No Level

Displays the assembly structure in a list view with no levels. This control is disabled in edit profile mode.

BOM View

Displays the assembly structure in a BOM view. This control is disabled in edit profile mode.

Active Connection

Displays the folder for the currently active connection. This option is only available when working with Teamcenter.

Assign All

Assigns Teamcenter-generated values to the properties displayed in the dialog box. This option is only available when working with Teamcenter.

This dialog box is similar to the Property Manager except for the availability of some of the controls.

Customizing the View

You can click the right mouse button on an occupied row in the grid to display a shortcut menu to change the font, sort the properties, find and replace text, and format columns. You can double-click near a column divider to do a best fit on the grid. Any changes you make to the columns are stored for future sessions.

Reordering Columns

To reorder a column, select the column, hold the mouse button down, then drag the column to the appropriate place.

Formatting Columns

To format a column, click the right mouse on the column and click Columns on the shortcut menu. The Format Columns dialog box is displayed so you can change the column format.

Changing Fonts

You can change the font for a column and its contents. To change the font, click the right mouse on the column and click Font on the shortcut menu. The Font dialog box is displayed to allow you to change the font.

Setting Actions

You can upload and check in documents managed by Teamcenter using the Action column. The Upload option loads the file into the Teamcenter, but leaves it checked out to you. The Check In option loads the document into Teamcenter and makes it available for other users to check out.

The default action for a new file being saved for the first time is to Upload Document.

The default action for an existing file being uploaded when it is closed is based on the settings defined on the Manage page of the Solid Edge Options dialog box. If the option to Always upload to server but leave document checked out to me is set, the default action is to upload the document.

If the option to Always upload to server and check document in is set, the default action is to check the document in.

If neither option is set, the document is saved to disk.
