Solid Edge Software Development Kit (SDK)

The Solid Edge Software Development Kit (SDK) is a comprehensive set of tools based on the state-of-the-art object-oriented technology that helps you standardize on Solid Edge for all your computer-aided-design needs. Using Solid Edge SDK, you can easily do the following:

  • Extend and modify the existing Solid Edge functionality to suit your specific needs.

  • Develop and customize integrated applications that link design, engineering, and manufacturing processes using Solid Edge modeling geometry.

  • Create high-performance CAD/CAE/CAM programs and applications that take advantage of the solid modeling capabilities and the ease-of-use of Solid Edge.

  • Use Solid Edge as the base for all your projects to maximize productivity. All members participating in the development process including contractors, consultants, and suppliers can seamlessly share applications.

The Solid Edge SDK offers a robust set of APIs, sample code, working examples, user interface (UI) consistency guidelines, and on-line Help for programmers.
