Command options

Many command options set by the user are now stored in UserPreferences.xml, located in the application data folder of the operating system that Solid Edge is loaded on.

Example:

C:\Users\<username>\AppData\Roaming\Siemens\Solid Edge\Version 223\

Part environment

  • Part painter: Select filter setting

  • Extrude, Cut, Revolve and Revolved Cut (Ordered modeling): Extent type

  • Patterns (Ordered modeling): feature, Smart/Fast solve, occurrence values

  • Mirror feature (Ordered modeling): Smart/Fast solve

  • Boolean: Boolean type

  • Round: Round options

  • Chamfers: Chamfer options, select filter setting

Sheet Metal environment

  • Flange: Measurement point, material side

  • Bend: Material side.

  • Jog: Measurement point, material side

  • Cutout: Extent type

  • Break Corner (Synchronous modeling): Round/ chamfer setting, filter type

  • 2 Bend Closed Corner: Overlap ratio (for overlapping corner), corner treatment cutout values, gap value

Assembly environment

  • Wire path (Harness assembly): circular cutout, keypoint locate

Draft environment

  • Maintain Relationships: ON
