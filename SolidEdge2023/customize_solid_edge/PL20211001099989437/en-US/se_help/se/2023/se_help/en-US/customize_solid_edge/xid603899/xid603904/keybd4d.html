Default keyboard shortcuts

You can view and print all of the default Solid Edge keyboard shortcuts from the Keyboard page (Customize dialog box).

  1. Click the Customize Quick Access Toolbar arrow , and then choose Customize.

  2. In the Customize dialog box, click the Keyboard tab.

  3. From the Choose commands from list, select All Keyboard Assignments.

    The table updates to display all of the defaults for the environment selected in the Environment to customize list.

  4. Select the Print button.

To learn how to change the keyboard shortcuts, see the help topic, Customize the keyboard.
