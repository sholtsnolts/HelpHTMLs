Open tab, Manage page (Solid Edge Options dialog box)

The Open tab is a member of the tab set available in the Manage page of the Solid Edge Options dialog box. Use options on this page to specify how Teamcenter-managed documents are opened.

When opening managed Assembly documents
I use classic variants. Show variant rule combo box on Open dialog.

Displays options for variant rule behavior on the Open File dialog box.

I use constructions. Always apply the closure rule.

Filters out BOM lines marked as constructions and displays the resulting structure. This option impacts Solid Edge assemblies saved to Teamcenter in ST8 or earlier.

When opening managed documents from the hosted Active Workspace client
Always show the Open dialog

Determines whether a document is opened quickly, bypassing the Open dialog box, or if documents are opened using the Open dialog box. When the option is cleared, quick open is used.

Always ask before inserting the model into the active assembly

Determines whether a model is automatically inserted into the active assembly, or you are prompted to insert the model.

When opening draft documents using Open Draft
Always show the Open dialog

Displays the Open File dialog box when the Open Draft command is selected from the Teamcenter tab in the Teamcenter-managed environment.

When opening draft documents in Structure Editor
Download linked documents

Downloads to the cache, the documents that are linked to the drawing when the drawing is opened in Structure Editor.
