Create a rule in Dynamic Visualization

  1. Open a Solid Edge assembly.

  2. From the Inspect tab→Report group, click Dynamic Visualization.

  3. Filter the property information as described in Apply filters in Dynamic Visualization.

    You must apply at least one filter before you can create a rule.

  4. Create a rule comprised of the filters you applied by clicking the green check .

    A rule is created for you (for example, Rule 1). The rule is assigned a unique override color so you can view the results in the model.

  5. To continue creating rules for the property information displayed, repeat steps 3 and 4.

    Each rule you create is filtered from the results of the previous rule.

  6. To apply color overrides associated with the rules, click Apply Override .

    The corresponding components are highlighted in the model. Color overrides are temporary, for viewing purposes only.

  7. To remove the temporary color override, click Remove Override .

    Tip:

    • You can change the color associated with a rule, reorder a rule, delete or rename a rule using the commands that become available when you select a rule.

    • You can modify a rule by clicking the rule name and then editing the filters that are applied. To display the table of properties, use the Show Table command.

    • You can create a report from the list of rules you defined.
