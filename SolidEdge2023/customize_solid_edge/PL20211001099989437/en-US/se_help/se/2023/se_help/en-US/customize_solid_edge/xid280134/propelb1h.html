Modify element properties

You can modify the properties of 2D elements using the Properties command.

  1. In the graphics window or on the drawing sheet, right-click an element or object.

    Tip:

    In synchronous modeling environments, you may need to use QuickPick to select a single element that is connected to other elements. Then, use the left mouse button to select the edit handle of the 2D element. The edit handle is the filled geometric icon at the center of the element.

  2. From its shortcut menu, choose the Properties command.

    The Properties dialog box for the selected element is displayed.

  3. In the Properties dialog box, modify the settings for the properties you want to change.
