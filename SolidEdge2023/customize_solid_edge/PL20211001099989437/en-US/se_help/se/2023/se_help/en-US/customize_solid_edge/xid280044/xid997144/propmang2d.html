Property Manager dialog box

Allows you to edit the properties of a Solid Edge document. You can right-click any row in the grid to display a shortcut menu that allows you to do such things as change the font, sort the properties, find and replace text, and format columns. You can double-click near a column divider to do a best fit on the grid. Any changes you make to the columns are stored for future sessions.

Note:

To display a preview of the document in the row, click the row in the dialog box.

Files of Type

Specifies the type of files that are listed in the File Name column. The properties that are displayed are based on the files listed in the File Name column. This control is disabled in if you are in a BOM view in Solid Edge.

Save

Saves changes you made to properties.  This option is disabled if you use the Property Manager command in Solid Edge. To save your changes, use the Save command on the File menu.

Print

Sends a copy of the active document to a specified plotter, printer, or file. Options are available for defining the printing area, range, number of copies, and other printing characteristics.

Before using this command, you must install and select a printer.

Cut

Cuts the current selection and pastes it to the Clipboard.

Copy

Copies the current selection to the Clipboard.

Paste

Inserts the Clipboard contents at the specified insertion point.

Restore

Sets all property cells back to the previous values.

No Level

Displays the assembly structure in a list view with no levels.

BOM View

Displays the assembly structure in a BOM view.

Active Connection

Displays the folder for the currently active connection. This option is only available when working with Teamcenter.

Assign All (option available when working with managed data in Solid Edge)

Assigns generated values to the document and revision properties. For more information, see Assign document and revision numbers automatically.

Note:

Top-down document status is enforced for SEStatus. Each document linked to the active document must have adequate status. For example, an assembly cannot be set to Released if any component in the assembly is not at least Released. You can set document status using the Summary group, File Properties dialog box, or disable top-down document status on the File→Settings→Options→Helpers page.
