Add-In Manager Dialog Box

Loads and unloads add-in commands. Displays information about selected add-ins.

Available Add-Ins

Displays a list of the add-in commands that are currently available to you. If checked, the add-in will be automatically loaded the first time a supported environment is activated. If not checked, the add-in will not be automatically started when one of its environments is activated.

Environments

Displays a list of the environments for which the selected add-in is enabled.

Description

Displays a description of the selected add-in.

Add-In Help

Displays on-line help (if available) for the selected add-in.

Add-In Properties

Displays the Properties dialog box which contains information for the selected add-in.
