General tab, Manage page (Solid Edge Options dialog box)

The General tab is a member of the tab set available in the Manage page of the Solid Edge Options dialog box. Use options on this page to specify how Teamcenter-managed documents are closed, saved, or revised. The controls on this page are disabled when you are working offline.

When closing managed documents
Always upload to server, but leave document checked out to me

Uploads documents from your local system to Teamcenter when closing a document, but keeps the document checked out. If the document is checked out, it is not available to other users.

Always upload to server and check document in

Uploads a document from your local system to Teamcenter when closing a document. When the document is checked in, it is available to other users.

When saving existing managed documents to Teamcenter
Do not show common property dialog

Eliminates the display of common property dialog boxes when you upload or check in existing documents to Teamcenter. All direct and indirect documents must have been previously saved to Teamcenter. If any new documents are discovered, the New Document dialog box is displayed.

The Hide CPD command in the Teamcenter ribbon is synchronized with this option.

When saving managed documents with Simulation results (.ssd)
Upload the .ssd file on-demand using the Upload .ssd command

Requires you to explicitly select the Upload .ssd command to have the Simulation results file (.ssd) uploaded to Teamcenter.

Automatically upload a new .ssd file and upload modifications on-demand

Uploads a new Simulation results file (.ssd) to Teamcenter, but only uploads modified .ssd files when you explicitly select the Upload .ssd command.

Always upload the modified .ssd file when the model is saved to Teamcenter

Uploads a modified Simulation results file (.ssd) to Teamcenter each time the Solid Edge model is saved to Teamcenter.

When revising or copying managed documents
Include drawings with linked drawing views that are stored in different Item Revisions

Performs a where-used search for draft documents automatically when you revise or choose Save As for 3D documents. The draft is revised or saved along with the 3D document and the draft is displayed in the dialog box to be checked in or uploaded.

When this option is cleared, draft documents will not be revised. Draft documents with a status of Obsolete are ignored in the Where-Used search.

Note:

This option only applies to draft documents that are in separate item revisions.

When revising or copying a product structure containing multiple revisions of an item

Revise All

Create one new revision and replace the other revisions with the new revision.

Creates a new revision for the latest revision and uses it for all previous revisions.

Create one new revision and reuse the other revisions.

Retains older revisions and creates a revision for the latest revision.

Save As All

Create one new item and replace the other revisions with the new item.

Creates a new item for the latest revision and uses it for all previous revisions.

Create one new item and reuse the other revisions.

Retains older revisions and creates an item for the latest revision.

Create new item for each revision.

Keeps each unique revision of the item and creates new items.

Working Mode
Always work offline when Solid Edge starts

Enables working with Teamcenter-managed documents without a connection to Teamcenter. This assists with external partner collaboration where documents need to be shared outside the managed environment.

Working Offline Mode
Prompt for File Properties on first save

Displays the File Properties dialog box the first time you save a document created while working offline. You have the option to modify the Item Type and Item Revision properties. Mandatory properties are identified with , but can be defined in the dialog box or when you check the document into Teamcenter.

Note:

This option is only available when you are working offline from Teamcenter.

Save Translation to Teamcenter
Enable Save Translation to Teamcenter

Saves translated documents to Teamcenter. This option and the Teamcenter tab→Document group→Save Translation to Teamcenter option are synchronized. The Teamcenter preference, SEEC_Save_Translation determines the behavior for managing translated files.

View Teamcenter Preferences
Preferences

Displays a comprehensive, read-only list of Teamcenter preferences. You can sort or search for specific preferences. Some preferences may not exist in the database, and those preferences could have an empty value.

Note:

Sort and Search requires the browser to run scripts or ActiveX controls. You can temporarily allow blocked content, or you can set your system's security options to allow active content to run on the computer.
