Restore the radial menu to default settings

The radial menu can be restored to its default settings.

  1. Open the Customize dialog box using one of these methods:

    • Right-click a command on the ribbon and choose Customize Quick Access Toolbar.

    • Click the Customize arrow , and from the menu, choose Customize.

  2. Select the Radial Menu tab.

  3. Select a Solid Edge environment.

  4. Click the Reset button.

  5. Select Close.
