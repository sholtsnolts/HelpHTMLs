# HelpHTMLs

A local/offline archive of HTML help systems for software and tools I use.

This repository is meant to make old or hard-to-search help files easier to browse, search, and reference. Some vendor help systems are spread across hundreds or thousands of HTML pages, so this repo keeps them organized in one place with rebuilt index pages and local hyperlinks.

## Available Help Archives

### Solid Edge 2023 Help

Offline/local copy of Siemens Solid Edge 2023 help documentation.

Open:

[SolidEdge2023/index.html](SolidEdge2023/index.html)

Includes sections for part modeling, assemblies, sheet metal, drawings, annotations, dimensions, sketching, PMI, MBD, frames, wiring, simulation, data management, and related tools.

The Solid Edge archive has been post-processed so that:

- section indexes are easier to browse
- most internal help links point to local files
- most images and shared help assets are stored locally
- a user-friendly README is included inside the `SolidEdge2023` folder

### CANKing Help

Offline/local copy of CANKing HTML help files.

Open:

[CANKing/index.html](CANKing/index.html)

Useful for CAN/CANopen setup, periodic messages, DBC-related setup, and CANKing workspace instructions.
