.. _setup:

Setup
=====

Kvaser CanKing is compatible with both Windows and Linux and can be downloaded from our homepage, see https://kvaser.com/download/.

Before you can use Kvaser CanKing, you must, however, download and install the Kvaser Drivers for the OS you are using. Those can also be downloaded from Kvaser's homepage. Instructions for how to install the Kvaser drivers are included in the respective software and not covered by this guide. For a list of all required drivers for each supported OS, see the corresponding section below.

.. _setup_windows_x64:

Windows x64
-----------

Kvaser CanKing is distributed as a Windows installer, an .exe file.

**Prerequisites:**

* Kvaser Drivers for Windows

**Installation**

1. Install Kvaser CanKing on Windows by running the canking-7.x.y-win-x64.exe installer and follow the instructions.

.. _setup_linux_intel_x64:

Linux for Intel x64
-------------------

Kvaser CanKing is distributed as a Debian package, a .deb file.

**Prerequisites:**

* Kvaser Drivers for Linux
* Kvaser SDK for Linux

.. note::
  Kvaser CanKing does not work with SocketCAN, you must use the Kvaser Drivers.

**Installation**

1. Install Kvaser CanKing on Linux for Intel x64 by running the command:

.. code:: bash

  sudo dpkg --install canking-7.x.y-linux-amd64.deb

.. _setup_extensions:

Extensions
----------

Kvaser CanKing supports extensions, which are additional features that can be added to the software. Extensions are OS independent and can be downloaded from our homepage, see https://kvaser.com/download/.

**Prerequisites:**

* Kvaser CanKing must be installed before installing any extensions.

**Installation**

1. Install the desired extensions either by:

   * Running (double-clicking) the extension file, a .ckext file.
   * Or by selecting **More** followed by **Extensions** and **Install...** from the tool bar menu of the Kvaser CanKing application and selecting the extension file, a .ckext file, in the file dialog.
