.. cli_overview:

Overview
========

The CanKing CLI is a command line interface for Kvaser CanKing. It is installed with CanKing and can be found in the installation folder, typically `C:\\Program Files\\Kvaser CanKing\\service\\ck.exe` on Windows and `/opt/canking/service/ck` on Linux.

To get started with the CanKing CLI, open a terminal and type `ck --help` to see the available commands and options.

For more information on a specific command, type `ck <command> --help`.

.. _cli_canking_terminal:

CanKing Terminal
----------------

The CanKing CLI can be accessed from within the CanKing application. To do this, open CanKing and go to the `Tools Panel` in the lower part of the main window. From there, select `Terminal` to open a terminal view with the CanKing CLI ready to use.

If the `Tools Panel` isn't visible, then select 'More'->'View'->'Toggle Tools Panel' from the head menu.

.. _cli_windows_terminal:

Windows Terminal
----------------

On Windows, you can open a Command Prompt or PowerShell window and navigate to the CanKing CLI installation folder. You can then run the `ck.exe` executable with the desired commands and options.

To make CanKing CLI available from any folder, you can add the installation folder to your system's PATH environment variable.

**Adding CanKing CLI to PATH on Windows**

1. Open the Start menu and search for "Environment Variables"
2. Click "Edit the system environment variables"
3. In the System Properties dialog, click "Environment Variables..."
4. In the Environment Variables dialog, find "Path" in the System variables section and click "Edit..."
5. Click "New" and add the CanKing CLI installation folder path (typically `C:\\Program Files\\Kvaser CanKing\\service`)
6. Click "OK" to close all dialogs
7. Restart your Command Prompt or PowerShell window

After adding to PATH, you can run `ck` commands from any directory without specifying the full path to the executable.

.. _cli_linux_terminal:

Linux Terminal
--------------

On Linux, you can open a terminal and run the `ck` executable with the desired commands and options. A symbolic link to the `ck` executable is already added to `/usr/bin` making it available from any directory.
