# ck install extension

Installs a CanKing extension.

## Synopsis

```text
ck install extension
    <EXTENSION_FILE>
    [--language <en|zh|ja>]
    [-r|--replace]

ck install extension -h|--help
```

## Arguments

- ```EXTENSION_FILE```  
The absolute path to the WebAssembly file, .wasm-file, containing the service extension, or to the Node.js package file, .tgz-file, containing the GUI extension.

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

- ```-r|--replace```  
Replace/Update any existing extension with the same name.

## Examples

```bash
ck install extension path/to/extensionfile.wasm
ck install extension path/to/extensionfile.tgz --replace
```
