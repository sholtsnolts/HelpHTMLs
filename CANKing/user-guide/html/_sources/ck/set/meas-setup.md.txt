# ck set meas-setup

Sets a new measurement setup.

## Synopsis

```text
ck set meas-setup
    <SETUP_JSON_OR_PROJECT_FILE>
    [--language <en|zh|ja>]

ck set meas-setup -h|--help
```

## Arguments

- ```SETUP_JSON_OR_PROJECT_FILE```  
The measurement setup as JSON or path to a CanKing project file.

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

## Examples

```bash
ck set meas-setup path/to/project.ckproj
ck set meas-setup '{"sources":[],"dataProcessors":[],"targets":[]}'
```
