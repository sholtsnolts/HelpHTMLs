# ck start traffic

Starts a Traffic Generator.

## Synopsis

```text
ck start traffic
    <TRAFFIC_GENERATOR_NAME_OR_ID>
    [--language <en|zh|ja>]

ck start traffic -h|--help
```

## Arguments

- ```TRAFFIC_GENERATOR_NAME_OR_ID```  
The identifier/name of the traffic generator to be used.

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

## Examples

```bash
ck start traffic "C1 traffic generator"
ck start traffic 58e31d82-c9e2-4557-a612-4408504bdd36
```
