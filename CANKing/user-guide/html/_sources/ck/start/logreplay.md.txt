# ck start logreplay

Starts a Log Replay node.

## Synopsis

```text
ck start logreplay
    <LOGREPLAY_NAME_OR_ID>
    [--language <en|zh|ja>]

ck start logreplay -h|--help
```

## Arguments

- ```LOGREPLAY_NAME_OR_ID```  
The name or identifier of the log replay node to be started.

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

## Examples

```bash
ck start logreplay "Logreplay"
ck start logreplay 58e31d82-c9e2-4557-a612-4408504bdd36
```
