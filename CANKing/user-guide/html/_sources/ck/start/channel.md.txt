# ck start channel

Start a channel, i.e. go bus on.

## Synopsis

```text
ck start channel
    <CHANNEL_NAME_OR_ID>
    [--language <en|zh|ja>]

ck start channel -h|--help
```

## Arguments

- ```CHANNEL_NAME_OR_ID```  
The name or identifier of the node to be started.

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

## Examples

```bash
ck start channel "CAN 1"
ck start channel 58e31d82-c9e2-4557-a612-4408504bdd36
```
