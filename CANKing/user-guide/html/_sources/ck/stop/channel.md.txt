# ck stop channel

Stop a channel, i.e. go bus off.

## Synopsis

```text
ck stop channel
    <CHANNEL_NAME_OR_ID>
    [--language <en|zh|ja>]

ck stop channel -h|--help
```

## Arguments

- ```CHANNEL_NAME_OR_ID```  
The name or identifier of the node to be stopped.

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

## Examples

```bash
ck stop channel "CAN 1"
ck stop channel 58e31d82-c9e2-4557-a612-4408504bdd36
```
