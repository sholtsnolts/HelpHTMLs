# ck stop traffic

Stops a Traffic Generator.

## Synopsis

```text
ck stop traffic
    <TRAFFIC_GENERATOR_NAME_OR_ID>
    [--language <en|zh|ja>]

ck stop traffic -h|--help
```

## Arguments

- ```TRAFFIC_GENERATOR_NAME_OR_ID```  
The name or identifier of the traffic generator node to be stopped.

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

## Examples

```bash
ck stop traffic "C1 traffic generator"
ck stop traffic 58e31d82-c9e2-4557-a612-4408504bdd36
```
