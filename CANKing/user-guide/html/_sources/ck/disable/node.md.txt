# ck disable node

Disables a node in the current measurement setup.

## Synopsis

```text
ck disable node
    <NODE_NAME_OR_ID>
    [--language <en|zh|ja>]

ck disable node -h|--help
```

## Arguments

- ```NODE_NAME_OR_ID```  
The name or identifier of the node to be disabled.

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

## Examples

```bash
ck disable node "Message Logger"
ck disable node 58e31d82-c9e2-4557-a612-4408504bdd36
```
