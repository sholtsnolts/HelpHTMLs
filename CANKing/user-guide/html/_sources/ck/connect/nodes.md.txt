# ck connect nodes

Connects two nodes in the current measurement setup.

## Synopsis

```text
ck connect nodes
    <START_NODE_ID_OR_NAME>
    <END_NODE_ID_OR_NAME>
    [--language <en|zh|ja>]

ck connect nodes -h|--help
```

## Arguments

- ```START_NODE_ID_OR_NAME```  
The start node identifier/name.

- ```END_NODE_ID_OR_NAME```  
The end node identifier/name.

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

## Examples

```bash
ck connect nodes "J1939" "Message Logger"
ck connect nodes 2073f340-b9d7-a55c-cc18-b4e1c43dfc2f 58e31d82-c9e2-4557-a612-4408504bdd36
```
