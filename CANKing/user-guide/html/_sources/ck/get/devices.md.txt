# ck get devices

Gets all available devices.

## Synopsis

```text
ck get devices
    [--language <en|zh|ja>]
    [--no-virtuals]
    [--json]

ck get devices -h|--help
```

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

- ```--no-virtuals```  
Exclude virtual devices

- ```--json```  
Returns all device details in JSON format

## Examples

```bash
ck get devices
ck get devices --no-virtuals
ck get devices --json
```
