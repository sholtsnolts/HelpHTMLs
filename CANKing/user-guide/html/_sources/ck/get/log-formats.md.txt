# ck get log-formats

Gets all supported log file formats.

## Synopsis

```text
ck get log-formats
    [--language <en|zh|ja>]
    [-m|--msg-log]
    [-s|--signal-log]
    [-r|--read-support]

ck get log-formats -h|--help
```

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

- ```-m|--msg-log```  
Include message log formats.

- ```-s|--signal-log```  
Include signal log formats.

- ```-r|--read-support```  
Only include formats with support for reading.

## Examples

```bash
ck get log-formats
ck get log-formats --msg-log --signal-log
ck get log-formats --msg-log --read-support
```
