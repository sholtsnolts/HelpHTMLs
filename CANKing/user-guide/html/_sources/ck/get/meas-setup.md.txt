# ck get meas-setup

Gets the current measurement setup.

## Synopsis

```text
ck get meas-setup
    [--language <en|zh|ja>]

ck get meas-setup -h|--help
```

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

## Examples

```bash
ck get meas-setup
```
