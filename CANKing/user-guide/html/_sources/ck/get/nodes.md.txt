# ck get nodes

Gets all nodes in the measurement.

## Synopsis

```text
ck get nodes
    [--language <en|zh|ja>]
    [-s|--sources]
    [-d|--dataprocessors]
    [-t|--target]
    [--node <NODE_NAME_OR_ID>]
    [--json]

ck get nodes -h|--help
```

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

- ```-s|--sources```  
Get source nodes.

- ```-d|--dataprocessors```  
Get dataprocessor nodes.

- ```-t|--target```  
Get target nodes.

- ```--node```  
The name or identifier of the node to be looked up.

- ```--json```  
Returns all node details in JSON format

## Examples

```bash
ck get nodes
ck get nodes --sources --dataprocessors --target
ck get nodes --sources --dataprocessors
```
