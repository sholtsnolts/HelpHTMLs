# ck delete target

Deletes a target from the current measurement setup.

## Synopsis

```text
ck delete target
    <TARGET_NAME_OR_ID>
    [--language <en|zh|ja>]

ck delete target -h|--help
```

## Arguments

- ```TARGET_NAME_OR_ID```  
The name or identifier of the target to be deleted.

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

## Examples

```bash
ck delete target "Message Logger"
ck delete target 58e31d82-c9e2-4557-a612-4408504bdd36
```
