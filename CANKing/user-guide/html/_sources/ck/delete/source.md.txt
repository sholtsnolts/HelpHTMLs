# ck delete source

Deletes a source from the current measurement setup.

## Synopsis

```text
ck delete source
    <SOURCE_NAME_OR_ID>
    [--language <en|zh|ja>]

ck delete source -h|--help
```

## Arguments

- ```SOURCE_NAME_OR_ID```  
The name or identifier of the source to be deleted.

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

## Examples

```bash
ck delete source "CAN 1"
ck delete source 1875f340-a9c4-4d5b-993c-b08ce4bdfca2
```
