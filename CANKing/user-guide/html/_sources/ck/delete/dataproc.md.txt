# ck delete dataproc

Deletes a data processor from the current measurement setup.

## Synopsis

```text
ck delete dataproc
    <DATA_PROCESSOR_NAME_OR_ID>
    [--language <en|zh|ja>]

ck delete dataproc -h|--help
```

## Arguments

- ```DATA_PROCESSOR_NAME_OR_ID```  
The name or identifier of the data processor to be deleted.

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

## Examples

```bash
ck delete dataproc "J1939"
ck delete dataproc 2073f340-b9d7-a55c-cc18-b4e1c43dfc2f
```
