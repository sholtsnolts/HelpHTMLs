# ck uninstall extension

Uninstalls a CanKing extension.

## Synopsis

```text
ck uninstall extension
    <EXTENSION_NAME>
    [--language <en|zh|ja>]

ck uninstall extension -h|--help
```

## Arguments

- ```EXTENSION_NAME```  
Name of the extension to be uninstalled.

## Options

- ```--language```  
Specify the preferred language to be used in all output strings. Default value is English (en). Supported languages are English (en), Chinese (zh), and Japanese (ja).

## Examples

```bash
ck uninstall extension extensionname
```
