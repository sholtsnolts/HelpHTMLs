.. _introduction:

Introduction
============

.. include:: welcome.rst

Kvaser CanKing consists of a graphical application that is connected to a service. The service uses Kvaser libraries such as CANlib, kvaDbLib and kvmlib. As long as the service is running, it also possible to create and start measurements from the terminal using specific commands, see :ref:`terminal_view` for more information.

.. _settings:

Settings
--------

The settings menu is found in the tool bar at the top of the Kvaser CanKing application. Among other settings, it is possible to select language. Currently, English and Chinese are available.

The settings are automatically saved and kept when switching project and after restart of the application.

.. _shortcuts:

Shortcuts
---------

.. _global_shortcuts:

Global
~~~~~~

Below is a list of all shortcuts that can be used anywhere in the Kvaser CanKing application.

:New:                 Ctrl+N
:Open:                Ctrl+O
:Save:                Ctrl+S
:Save as:             Ctrl+Shift+S
:Close:               Alt+F4
:Start measurement:   F9
:Stop measurement:    F12
:Toggle full screen:  F11
:Zoom in:             Ctrl++
:Zoom out:            Ctrl+-
:Reset zoom:          Ctrl+0
:Edit settings:       Ctrl+,
