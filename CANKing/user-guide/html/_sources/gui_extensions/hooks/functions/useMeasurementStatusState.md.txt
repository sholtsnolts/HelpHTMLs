# Function: useMeasurementStatusState()

> **useMeasurementStatusState**(): [`IMeasurementStatusContext`](../interfaces/IMeasurementStatusContext.md)

A hook for measurement status state.

## Returns

[`IMeasurementStatusContext`](../interfaces/IMeasurementStatusContext.md)

The measurement status state.

## Example

```
const { nodeStatuses, isLoaded, error } = useMeasurementStatusState();

if (!isLoaded) return <Loading />;
if (error) return <ErrorView error={error} />;
// nodeStatuses value is now safe to use
```
