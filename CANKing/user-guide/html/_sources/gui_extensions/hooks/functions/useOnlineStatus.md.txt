# Function: useOnlineStatus()

> **useOnlineStatus**(): [`IOnlineStatusContext`](../interfaces/IOnlineStatusContext.md)

A hook for online status.

## Returns

[`IOnlineStatusContext`](../interfaces/IOnlineStatusContext.md)

The online status.

## Example

```
const { onlineStatus, isLoaded, error } = useOnlineStatus();

if (!isLoaded) return <Loading />;
if (error) return <ErrorView error={error} />;
// onlineStatus is now safe to use
```
