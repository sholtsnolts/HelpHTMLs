# Function: useMessageDatabases()

> **useMessageDatabases**(`nodeIdentifier?`): [`IMessageDatabase`](../interfaces/IMessageDatabase.md)[]

Hook to get message databases available in the measurement. Will update as measurement setup changes. If nodeIdentifier is
specified, only databases available on that node will be included.
