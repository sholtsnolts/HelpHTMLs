# Function: useIsOnline()

> **useIsOnline**(): `boolean`

Hook keeping track of if the measurement is online.

## Returns

`boolean`

A flag that indicates if a measurement is started.

## Example

```
const isOnline = useIsOnline();

if (isOnline) {
  // Measurement is online
} else {
  // Measurement is offline
}
```
