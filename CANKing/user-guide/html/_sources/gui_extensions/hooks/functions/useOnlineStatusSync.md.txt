# Function: useOnlineStatusSync()

> **useOnlineStatusSync**(`parentIsInitialized`, `id`, `onGoingOnline`, `onGoingOffline`): `boolean`

Custom hook to synchronize session state with online/offline status changes.

## Parameters

### parentIsInitialized

`boolean`

Whether the parent component is initialized.

### id

`number`

The unique id of the view.

### onGoingOnline

[`OnGoingOnlineCallback`](../type-aliases/OnGoingOnlineCallback.md)

Callback function to handle going online.

### onGoingOffline

[`OnGoingOfflineCallback`](../type-aliases/OnGoingOfflineCallback.md)

Callback function to handle going offline.

## Returns

`boolean`

The current online status.
