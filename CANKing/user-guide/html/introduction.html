Introduction

Kvaser CanKing is a free of charge, general-purpose CAN and LIN bus analysis software that is compatible with all Kvaser CAN/LIN interfaces and the Kvaser virtual CAN bus. It’s available on Windows (x64) and Linux (x64 and ARM64).

This guide contains descriptions of the different parts of the application as well as multiple tutorials.

For instructions on how to install the application, please refer to Setup.

Kvaser CanKing consists of a graphical application that is connected to a service. The service uses Kvaser libraries such as CANlib, kvaDbLib and kvmlib. As long as the service is running, it also possible to create and start measurements from the terminal using specific commands, see Terminal view for more information.

Settings

The settings menu is found in the tool bar at the top of the Kvaser CanKing application. Among other settings, it is possible to select language. Currently, English and Chinese are available.

The settings are automatically saved and kept when switching project and after restart of the application.
